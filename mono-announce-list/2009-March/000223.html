Hi Everyone,

This email is to announce Tomboy 0.13.6, the sixth official
development release in the 0.13.x series.  Apologies for the late email; 
tarballs were uploaded on Monday.  Next Monday marks code freeze for 
this cycle.

Windows users, the installers for both GTK# and Tomboy were
revamped in 0.13.5.  Please follow the instructions on this wiki page 
*exactly* to upgrade to Tomboy 0.13.6 from any version prior to 0.13.5:
http://live.gnome.org/Tomboy/Installing/Windows

Tomboy is a simple personal note-taking application designed to be
unobtrusive and friendly, while supporting inter-note-links similar to
a WikiWikiWeb to help you organize your notes and ideas.  It can run
either as a GNOME panel applet or a notification tray icon.

Hey,

The Mono Accessibility team is proud to announce the latest release of Mono Accessibility 1.0.

The Mono Accessibility project enables many Windows applications to be fully accessible on Linux.  Applications based upon System.Windows.Forms can take advantage of this with no extra code, and enjoy the same level of Accessibility provided by Windows on any system Mono supports. 

The 1.0 release brings full accessibility support for all System.Windows.Forms controls, and a corresponding bridge that will allow AT-SPI-based assistive technologies like Orca and GOK to access applications that use Mono's System.Windows.Forms implementation.

For further details, check out our Release Notes[1] and our homepage[2].

Cheers,

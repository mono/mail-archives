Hello,

We've released Banshee 1.0 RC 1 (0.99.3), a bit over one week after the
1.0 Beta 2 release.  It brings some great new features and lots of bug
fixes and performance improvements.

We plan on releasing Banshee 1.0 in the next few days, so:

** translators and testers: please update and test now! **

New in this release:

* Automatic cover art fetching
* Local file system queue (play songs w/o importing)
* Over 24 bugs fixed since the last release

Release notes with screenshots, ChangeLog, and more:
http://banshee-project.org/Releases/0.99.3

Download the source code:
http://banshee-project.org/files/banshee/banshee-1-0.99.3.tar.bz2
sha1sum: 49c9509f0fa517ddc49629350eebbda2a6ed44be

Enjoy!

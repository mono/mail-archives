Hi,

I'm happy to announce that Beagle has been accepted to participate in
the Google Summer of Code once again.  We participated last year for the
first time, and we got a number of very good contributions.

The Google Summer of Code program is an initiative by Google to sponsor
students, with the help of mentors, to develop open source software.

Information on Beagle's involvement, including a list of ideas for
students, can be found here:

        http://beagle-project.org/Summer_Of_Code_2007
        
Information on the Google Summer of Code program itself, along with
links to other participating projects (including GNOME and Mono), can be
found here:

        http://code.google.com/soc/
        
The application process has just begun, and students have until March
24th to submit their applications.

Thanks!
Joe

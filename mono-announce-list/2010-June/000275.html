Hello Everyone

We have just released Mono Tools for Visual Studio Beta 2.  This release fixes
the following bugs:

Create a hot key for the Mono menu https://bugzilla.novell.com/show_bug.cgi?id=616781
Can not remove multiple profiles https://bugzilla.novell.com/show_bug.cgi?id=616405
Disable UI in choose profile dialog https://bugzilla.novell.com/show_bug.cgi?id=616422
Mac Cannot run remote app with mono tools https://bugzilla.novell.com/show_bug.cgi?id=610457
Locals shown when no local variables are present https://bugzilla.novell.com/show_bug.cgi?id=606130
VS 2k10: Profile Selector and other dialogs unusable with large fonts https://bugzilla.novell.com/show_bug.cgi?id=614769
Break on exception not implemented https://bugzilla.novell.com/show_bug.cgi?id=605227

We are nearing the final release, with no major changes planned.  

Full details of the 2.0 features, as well as the Beta 2 download links,
can be found in the Mono Tools for Visual Studio 2.0 Release Notes:
http://mono-project.com/Release_Notes_MonoTools_2.0

Public forums for all Mono Tools users are available here:
http://forums.mono-tools.com/ 

Thanks
Joseph Hill

After the Beta 1 announcement, we received some feedback from users 
that they preferred the mailing list to the forums, so we're going to keep 
this list around, particularly for announcements such as this.  If you'd 
prefer to not receive these messages, you can change your settings or 
unsubscribe here:
http://lists.ximian.com/mailman/listinfo/mono-vsbeta

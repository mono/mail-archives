Hello mono-announce, I'm not sure if this is the right list for this, but I
have just released a beta version of NNTPsharp, which is currently a C#.NET
2.0/Mono library for NNTP communications, including yEnc encoding and
decoding support. I plan to create CLI and GUI clients based on WPF and
GTK#, but a roadmap for these features is still to be announced.

All this said, it's still a one-developer project. Thus, I ask the
community's help to file bug reports, test the software, and contribute,
either through patches, technical documentation, or your own examples for
using the library.

The project's home page is at http://nntpsharp.codeplex.com.

--Timothy Roberts(tr.paracyte at gmail.com)

Hello,

We've released Banshee 1.2, a month and a half after the
1.0 release.  It brings some great new features and lots of bug
fixes and performance improvements.

New in this release:

* Internet radio
* Multi-artist album support
* DAAP Client
* Equalizer
* Playlist Importing
* Amarok migration
* Over 80 bugs fixed since the last release

Release notes with screenshots, ChangeLog, and more:
http://banshee-project.org/downloads/archives/1.2.0

Download the source code:
http://download.banshee-project.org/banshee/banshee-1-1.2.0.tar.bz2
sha1sum: b7cf7e4cf5e68ede4e88f0a9338c98a84c8db53d

Enjoy!

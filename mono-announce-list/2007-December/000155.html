Hi,

I'm pleased to announce the release of Beagle-Xesam 0.2. This release
comes with several improvements making it more conformant to the Xesam
search specification.

Xesam provides API specification for desktop search and metadata
services. Visit http://xesam.org/main for more information.
Beagle is a Linux desktop-independent service which transparently and
unobtrusively indexes your data in real-time. Visit
http://beagle-project.org/About for more information about beagle.
Beagle-Xesam is an adapter that interoperates between the beagle
service and a xesam client. Visit
http://beagle-project.org/BeagleXesam for more information.

Get Beagle-Xesam 0.2 from here:
http://beagle-project.org/files/0.3/beagle-xesam-0.2.tar.bz2
[ Will upload it to Gnome ftp soon ]

DEPENDENCY
--------------------

* Beagle-0.3.2
* NDesk-DBus, NDesk-DBus-glib >= 0.3.0

CHANGES FROM 0.1 to 0.2
--------------------------------------

* Remove in-tree NDesk-DBus bindings. NDesk-DBus is now a dependency.

* Support for "inSet" queries

* Support the "content" and "sources" attributes of <query> tags

* Support more hit.fields

* Correctly populate vendor.ontology.{fields,sources,contents}

* Fix GetState() behaviour

* Add support for the StateChanged signal

Contributors to this release: Arun Raghavan, Kevin Kubasik and Anders
Rune Jensen.

Full set of changes:
http://svn.gnome.org/viewvc/beagle/tags/BEAGLE_XESAM_0_2/ChangeLog?view=markup

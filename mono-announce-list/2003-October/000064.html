Hi everyone!

There's a new release of mod_mono out there.

* What's mod_mono

        mod_mono is the mono ASP.NET apache module. It works together
        with mod-mono-server, available in xsp.
        
* Improvements since 0.4

        Many bug fixes based on our own experience. We're using it at
        http://www.go-mono.com/docs/
        
        mod_mono_unix has been renamed to mod_mono and mod_mono to
        mod_mono_old. If you had a previous installation of mod_mono,
        you'll need to read the INSTALL file and modify httpd.conf
        accordingly.
        
* Availability

        http://www.go-mono.com/archive/mod_mono-0.5.tar.gz
        
        Requires xsp 0.6, which is at
        http://www.go-mono.com/archive/xsp-0.6.tar.gz
        
* Contributors to this version

        Pedro Martínez and me.

Mono 1.2.3 has been released, some highlights: <BR>
<BR>
Thanks to everyone using <A HREF="http://www.mono-project.com.com/Moma">Moma</A> and submitting their reports, we were able to prioritize the work needed to get applications running. The top used <B>1,933 APIs</B> have been implemented in this release as well as many bugs fixed and performance increased. <BR>
<BR>
New Moma definition files. If you are using Moma to evaluate porting your software, running it again will load the new API definitions. <BR>
<BR>
Mono debuts Rolf Bjarne's <B>Visual Basic 8.0 compiler</B> written completely in Visual Basic 8. We are also shipping Seo Sanghyeon's <A HREF="http://fepy.sourceforge.net/">IronPython Community Edition</A>. <BR>
<BR>
Some highlights: ASP.NET 2.0 is almost complete (WebParts are still missing), a new System.Media implementation is included, SOAP 1.2 is now supported, HttpListener does TLS/SSL, 2.0 System.Net.Sockets is available, many Windows.Forms API fixes and improvements, improved startup time, more ports and much ADO.NET 2.0 work. <BR>
<BR>
Please read the <A HREF="http://www.go-mono.com/archive/1.2.3">release notes</A> for all the juicy details.<BR>
<BR>
The software is available from http://www.mono-project.com/Downloads<BR>

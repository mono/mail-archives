Hello!  We just released Banshee 1.8.0, the culmination of six month's
work by 67 programmers, artists, documentation writers, and
translators.

New Features Since 1.6:
  * Integrated AmazonMP3 Store
  * Miro Guide Podcast Directory
  * Improved Apple Device Support
  * Revamped Hardware Backend
  * Keybinding for Switching Sources
  * Improved Audiobook Support
  * User Manual
  * Bulk Metadata Fixer
  * MPRIS v2 support (Ubuntu Sound Menu, etc)
  * Dozens of other enhancements

And 230 bugs fixed!

More info:
* Release Notes: http://banshee.fm/download/archives/1.8.0
* Download: http://banshee.fm/download/
* Support: http://banshee.fm/support/

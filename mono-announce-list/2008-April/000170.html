Hello,

We've released Banshee 1.0 Alpha 3 (0.98.3), two weeks after our second
1.0 alpha release, which brings some great new features and lots of bug
fixes and performance improvements.

New in this release:

* Audio CD playback and ripping support
* USB Mass Storage Digital Audio Player syncing
* New shuffle playback UI and logic
* Over 40 bugs fixed since the last release

Remember, Banshee 1.0 alpha releases can be safely installed and used in
parallel with older Banshee releases, so there should be nothing
stopping you from giving it a try!

Release notes with screenshots, ChangeLog, openSUSE 10.3 1-click
install, and full alpha-release disclaimer:
http://banshee-project.org/Releases/0.98.3

Download the source code:
http://banshee-project.org/files/banshee/banshee-1-0.98.3.tar.bz2
sha1sum: 61204019315db5efa394d394d2629199bb52143a

And because I'm shameless, I'll just plug my blog, which actually has
some useful details that are not in the release notes linked above:
http://abock.org/2008/04/10/banshee-10-alpha-3/

Enjoy!

--Aaron

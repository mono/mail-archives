Hi Everyone,

This email is to announce Tomboy 0.14.1.  This is a release of the
stable 0.14.x series and coincides with GNOME 2.26.1.

Tomboy has been branched as "gnome-2-26" for stable (0.14.x)
development.  SVN trunk is now open for new feature development
(0.15.x).


What is Tomboy?
==================================================

Tomboy is a simple personal note-taking application designed to be
unobtrusive and friendly, while supporting inter-note-links similar to
a WikiWikiWeb to help you organize your notes and ideas.  It can run
either as a GNOME panel applet or a notification tray icon, and is also
supported in your Windows tray or OS X dock.

Tomboy's Website: http://www.gnome.org/projects/tomboy/

Tomboy's Wiki: http://live.gnome.org/Tomboy

What's new in Tomboy 0.14.1
==================================================

Major changes in Tomboy 0.14.x series include the following:
* Tomboy is now fully supported on Windows and Mac OS X
* Printing has been rewritten using the Gtk.Print API, fixing many bugs
* 25% reduction in memory usage and slightly faster startup
* Significant memory savings with very large note collections
* Improvements to HTML Export add-in
* Improvements in note and URL linking
* Many fixes to note synchronization and D-Bus API
* Process now named "tomboy", not "Tomboy"
* No longer writes to disk every 40 seconds

Changes since 0.14.0 (unannounced stable release) include the following:
* Translation updates: ar, ca, el, es, he, kn, nb, tr

Changes since 0.13.6 (the last development release) include the following:
* Fix buggy behavior with Tomboy icon menus in Windows (#556406, #566173).
* Translation updates: as, bg, da, de, el, et, eu, fi, gu, hi, ja, ml, mr, nb,
                       or, pt, ro, ru, ta, te

The next six months...
==================================================

Some tentative plans for the next release cycle include:
* Automatic synchronization
* Faster startup, less memory usage (we've only just begun!)
* Better OS X experience
* Brainstorming for how Tomboy should fit into gnome-shell
* A new project to have easy synchronization to a Tomboy online service!
* Tomdroid, and how to have the best sync story to have Tomboy notes on your G1
* Tomboy website revamp
* Your awesome ideas!

But we haven't had our roadmap meeting yet, so we'll see what the community
decides during this public meeting [0] on Tuesday, April 21, 2009 at
19:30 UTC in #tomboy.

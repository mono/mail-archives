Hey Everyone,

This email is to announce Tomboy 0.7.3, the fourth development release
of the 0.7.x series leading up to 0.8.0.  Like usual, please be aware
that running the development series code is considered living on the
edge (please make regular backup copies of your notes).

This release is the first one that contains the new synchronization code.
It's really intended at this point for more advanced users.  We'd love to
hear any feedback you might have on it.  We've still got to write up
some help documentation for it, but otherwise, it should be functional.
Good work Sandy in fixing and merging this all back into trunk!

The tasks addin has been removed because it's not "baked" enough to
be included and we're running out of time.  We'll add it back in after
releasing 0.8.

Tagging support has also been removed because we've run out of
time to make the UI acceptable for a stable-quality release.  Any
volunteers here will be openly welcomed.  Please see the following
bug for more information: http://bugzilla.gnome.org/show_bug.cgi?id=380634

Tomboy is a simple note-taking application designed to be unobtrusive
and friendly, while supporting inter-note links similar to a
WikiWikiWeb to help you organize your notes and ideas.  It can run
either as a Gnome panel applet or a notification tray-icon.

Tomboy's Website: http://www.gnome.org/projects/tomboy

Tomboy's Wiki: http://live.gnome.org/Tomboy

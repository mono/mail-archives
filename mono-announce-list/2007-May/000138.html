Hey Everyone,

Banter 0.1.3 has been released:

What is it?
===========
Banter is a next generation real time collaboration client focused on
the big three: text, voice and video.  It's centered on a dynamic,
innovative and easy to use contact management system.

What's new?
===========
* Groups can now be created/deleted from group windows.
* Themes/Styles now loaded from $prefix/share/banter/Themes and
~/.banter/Themes.
* Initial work for adding people to groups via evolution.
* New work so Banter can be packaged as an RPM complete.
* Translations can now use po/LINGUAS file to add new translations.

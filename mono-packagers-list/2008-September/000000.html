The Mono 2.0 release will be delayed.  We are passing all unit tests on linux intel platforms but we are failing many unit tests on other important platforms (like ppc linux, windows and mac).  The team is working hard to get as many of these fixed as possible.

The new release date is Monday October 6.  Sorry for the inconvenience this causes.  RC2 became an internal-only build because of some critical bugs still not fixed.  RC3 is being built right now and will be released to the public on Monday October 29.

I will discuss with QA the idea of releasing RC tarballs earlier so that we can get them into the hands of packagers.

`testing` represents the planned next release of Mono. The version of
Mono in Debian Testing is 4.6.x, and has been since November.

If you want to try installing a 4.2.x version, old version of the
mono-project.com packages are available - use a sources.list line along
the lines of:

deb http://download.mono-project.com/repo/debian wheezy/snapshots/4.2 main

There's also snapshots.debian.org - instructions are on there.

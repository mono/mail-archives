Mono 2.10.2 is a bugfix release from the 2.10 branch with more than 75
bugs fixed.

The Mac packages have been modified. Instead of a base runtime framework
and an add-on C SDK we now publish a complete runtime framework (MRE -
Mono Runtime Environment) and a complete development kit (MDK - Mono
Development Kit), you know, kind of like how Java does it. This means
mono embedding developers who used to download the Framework and the C
SDK packages can now download just the SDK package. At the moment the
contents of the Runtime package are the same as what the Framework
package has contained and the SDK is that plus what was in the C SDK. We
may refine what is in the Runtime package in the future but it will
always be enough to run your favorite .NET programs on Mac.

Details on the release notes: http://www.go-mono.com/archive/2.10.2
Downloads at the usual place: http://www.go-mono.com/mono-downloads

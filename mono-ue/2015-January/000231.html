If you're upgrading from Preview 6 or Preview 5, Preview 7 just adds
missing files so re-extracting the plugin should be all you need for
those cases.

If you're upgrading from Preview 4 or older I'd recommend a clean
build since a lot of things moved around. The best way to do this
would be to delete and recreate the UnrealEngine branch and do a "git
clean -xcf" to remove untracked files, then follow the build
instructions from the beginning, including creating the UE branch and
applying the patchset.

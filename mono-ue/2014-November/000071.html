Yes, the bindings assembly projects (*not* game assembly projects)
currently use some MSBuild features that Xamarin Studio doesn't
support, so you'll need to edit them with Visual Studio for now. One
of the upcoming drops should fix this. In general the only reason to
edit them is to add custom members into the generated bindings or to
edit the core bindings/interop assemblies (Runtime and MainDomain).

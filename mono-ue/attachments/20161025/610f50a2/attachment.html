<p dir="ltr">Mono is already using a fair bit of code from .NET Core. It's too early to say exactly what the long term will be but we'll definitely be sharing/converging more over time.</p>
<p dir="ltr">Today, Mono is easier to embed (especially the low level stuff we're doing) and supports a number of gaming platforms that .NET Core does not such as iOS/tvOS, Android, and PS4. Mono also has a GC Bridge (developed for Android) that allows two GCs to work together, which will be very useful for the UE integration.</p>
<p dir="ltr">- mikayla</p>

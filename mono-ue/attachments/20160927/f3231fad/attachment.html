> Hello,<br>
><br>
> Earlier this year, at the Microsoft Build Conference, Scott Guthrie<br>
> announced that we would be open sourcing Xamarin's propietary stack under<br>
> the MIT license.   This also included relicensing Mono from the current<br>
> LGPL 2 license to the MIT license.<br>
><br>
> This opened the doors again for Mono scripting on Unreal Engine but we<br>
> needed to find the time to do the work.<br>
><br>
> Over the past year or so, Unreal Engine has evolved very rapidly, so our<br>
> existing code no longer compiled.<br>
><br>
> Mikayla Hutchinson has been heroically upgrading the codebase to work with<br>
> the latest version of Unreal Engine, and recently got the test suites and<br>
> the embedding working well enough to get our rotating C# chair to show up:<br>
><br>
> <a href="https://twitter.com/mjhutchinson/status/780456948630220800">https://twitter.com/mjhutchinson/status/780456948630220800</a><br>
><br>
> There is still some work left to be done to get back up to where we were.<br>
> This time around, instead of using our home-grown "validate the user has a<br>
> license via github hashes", we will just use GitHub built-in capabilities<br>
> to share the code with anyone that has access to EpicGame's repository.<br>
><br>
> Miguel.<br>

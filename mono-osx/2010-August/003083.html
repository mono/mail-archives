A search of the web (http://www.mactipsandtricks.com/articles/Wiley_HT_appBundles2.lasso 
) reveals that NSPrincipalClass is an Info.plist key whose value is  
NSApplication or the name of the class in your application that is  
derived from NSApplication.  So try adding that to your Info.plist file.

You need to tell which IDE you are using, although on this mailing list 
**normally** this shouldn't be relevant, because mono-osx development 
requires MacOSX + (Monodevelop | Xamarin.Studio).

I said normally because it can be done in Visual Studio too with side 
effects (cannot build
apps)

 1. You need to load MonoMac or Xamarin.Mac into Visual Studio.
    For that You need
     1. ProjectType GUIDs (I have for MonoMac, but no for Xamarin.Mac) and
     2. tweaking "visual studio project flavor" ("project flaves")
        http://msdn.microsoft.com/en-us/library/bb166488.aspx
        My starting point was this one for iOS projects:
        https://github.com/follesoe/VSMonoTouch
        In order to learn and extend previous one I forked and extended:
        https://github.com/moljac/MonoVersalProjectFlavesForVisualStudio2010
        I added support for:
         1. MonoMac (1st one to learn)
         2. WindowsMobile WindowsCE and SmartDevice
            we do port old WinMo apps to iOS and Android

This way I can work with all VS gems (plugins, addins, etc)

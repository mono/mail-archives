On Sun, Sep 13, 2009 at 6:38 PM, Kevin Heeney <koheeney at gmail.com> wrote:
> I am not sure if this is the right list for this, but with MonoDevlop 2.2
> beta 1 for Mac I am not seeing a GTK# Template under the C# menu when
> creating a new solution.  I just checked the Windows version and it has this
> template (as did previous versions on my mac).  I made sure that I have the
> latest mono and GTK# installed.  Any thoughts?

The GTK# template is part of the GTK# designer addin, which is not
included in the Mac version because important drag-n-drop features are
not implemented in GTK+ on the Mac.

However, you can just create a new C# project and add references to
the GTK# assemblies.

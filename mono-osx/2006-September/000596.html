> Is it possible to get xcode to compile a c# project using mono (any  
> hints on how to do this anywhere)? And also, does anyone know of  
> any sites that have tutorials for using dumbarton as the bridge?

Yes, it's possible by adding a run script phase. See the  
DBCocoaExample in Dumbarton examples folder. Open it in Xcode, unfold  
Target and double click in the "Build sample dll" phase, the c#  
compile script should be there.

In the dumbarton folder Docs/usage.txt expains how to use Dumbarton  
as a bridge, also the DBCocoaExample should be useful as an example.

However, I would like to add that I haven't been able to compile any  
Dumbarton example yet due to linking errors with Dumbarton and Mono.  
Any ideas ?

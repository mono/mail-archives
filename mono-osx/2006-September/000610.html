Hi,

I'm building from svn on a MacBook Pro (Intel-based MacOS X 10.4.7) and get
a failure during configuration:

PKG_CHECK_MODULES(BASE_DEPENDENCIES, glib-2.0 >= $GLIB_REQUIRED_VERSION)

where GLIB_REQUIRED_VERSION= 1.3.11 in the config script. Note that

[mymachine] pkg-config --modversion glib-2.0
2.12.2

I saw some reference to this in an old 2003 discussion group, but no actual
solution was there. I just commented out the PKG_CHECK... line in the
configure script and get everything to build fine.

What is this error and why is it happening? Anyone?

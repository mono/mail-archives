Hello Mike,

I think the Xcode integration in MonoDevelop 2.8.1 is really designed 
for Xcode 4, but it does seem to work if you get Interface Builder to 
write out the Objective-C class files. If I remember rightly, the menu 
item you need is called something like File->Write all class files... Be 
sure to save them in the same directory as the Xcode project generated 
by MonoDevelop (that should be the default selected in the save dialog).

> Hi all!
>
> Is a Layoutmanager like GridBagLayout in Java available for C#?

Not really Andreas.

System.Windows.Forms relies solely on so-called absolute positioning. 
You have to specify a position in pixel coordinates for each widget. 
System.Windows.Forms version 2.0 (included in .NET 2.0) will support 
layout management similar to Java's. As far as I know that's not 
supported in Mono yet though.

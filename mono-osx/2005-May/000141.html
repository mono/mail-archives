X-develop is a multi-language IDE for C#, Visual Basic .NET, J# and 
Java. Starting with version 1.0.1 X-develop allows developing Mono and 
Java applications on Mac OS X.

X-develop offers an unprecedented level of productivity for professional
Mono developers on Windows, Linux and Mac OS X: project-wide on-the-fly
error analysis, refactorings, intelligent code navigation, excellent VCS
support and many other powerful features.

I think I can answer my own question (bad etiquette I know). Having sent the
email below I remembered I had set up a global DYLD_LIBRARY_PATH environment
variable in ~.MacOSX/environment.plist (from Cocoa# 0.1 days). Deleting this
file has solved the problem.

Apologies for the wasted bandwidth :-)

Gareth Baker

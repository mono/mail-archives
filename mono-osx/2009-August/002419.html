Hi Jude,

I don't have any problems building with leopard, although if your trying to
compile from the svn repo, you will need to run autogen.sh instead of
configure
ie, something like this

./autogen.sh --prefix=/Library/Framework/Mono.framework/Versions/Current

hope that helps...

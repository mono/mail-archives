Hi asleB,

Looks like you need to uninstall & re-build libiconv.  Note that it's not
JUST libtool and glib2 that need to be built for x64, it's ALL dependant
libraries as well.

You have a few choices: uninstall all macports manually then re-install
them, or wipe your macports and install fresh, since nothing would have been
compiled in 64 bit anyway.

Hope this helps!
Curtis.

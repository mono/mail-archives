The Mono dmg/package installer for Mac OS X symlinks the various
executables into /usr/bin following platform convention, but doesn't do
the same with the man pages, which makes it a bit harder to use them
without manually messing about with MANPATH.

I found this bug which had been filed about it, then closed as NOTABUG
without explanation. I've reopened it and added some comments:
http://bugzilla.ximian.com/show_bug.cgi?id=74845

Is this deliberately not done, or has it just not gotten addressed yet?

You cannot use a static library with MonoMac.  You have to compile the
obj-c code as a dynamic library (dylib).  The RequiredFramework attribute
goes somewhere on your code that uses your binding DLL, not in the QLRTX.cs
file.

I just posted today a binding for the Cocos2d library here
https://github.com/trustme/cocos2d-mono.  That may help guide you in what
you need to do.

Duane

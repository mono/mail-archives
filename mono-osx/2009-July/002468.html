This is an old post, but I discovered it in the Search.

Since 2005, Eclipse most definitely is a viable option for developing with
mono. You can also download a Carbon version of Eclipse which definitely
feels very native to the mac, and has a small footprint as well as loading
up very quickly. It has all the advantages of a Enterprise standard IDE.

The plug-gin I have been using to build C# code in Eclipse is called
'Emonics'. It is quite great, but so far doesn't look like there is much
debugging support - hopefully more people will get excited about Emonics and
contribute as it seems like a really good piece of software.

Cheers

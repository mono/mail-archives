Hey,

  You are closing you "pop-up" window incorrectly. Close method sends close
signal to program. So to close your window you should use code:

*NSApplication.SharedApplication.Stop(this);*

*  this.OrderOut (this);*
I am assuming this code will be put into you "pop-up" window class and
called upon cancel button press.

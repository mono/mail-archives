When using MonoMac and Xcode 4.3, the current MD beta build fails with an
ibtool error. 

The solutions is to:

/sudo ln -s /Applications/Xcode.app/Contents/Developer/usr/bin/ibtool
/usr/bin/ibtool/

This might be specific to me, but my first attempt this failed, so i had to 

/sudo rm  /usr/bin/ibtool/

and run the ln again

Thanks to Chris Hardy and Michael Hutchinson.

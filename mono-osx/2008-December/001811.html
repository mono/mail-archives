- compile sqlite from source on OSX

CFLAGS="-Os -DSQLITE_ENABLE_COLUMN_METADATA" ./configure
--prefix=/usr/local/
make
make install

- open System.Data.SQLite.dll.config and add the following:

<dllmap dll="sqlite3" target="./libsqlite3.0.dylib" os="!windows"/>

That did it for me!

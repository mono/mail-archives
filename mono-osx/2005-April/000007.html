There's also an effort underway to make XCode more mono-friendly.  Dru has some cool stuff here: http://www.druware.com/products/xcodetools.html

Also, the mono wiki states:

Additionaly, the Cocoa# project is creating a number of framework and integrating Mono within the Apple XCode development chain that allow MacOS X developers to develop applications that leverage MacOS X native UI and class libraries.

This might be a more promising path than installing the whole Gtk mess and  hacking on MonoDevelop.

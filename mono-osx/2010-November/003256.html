Over the weekend I threw together a proof of concept winform to cocoa
wrapper.  I personally dont have the time or a reason to completely finish
this project. If this is something that others in the community will use or
contribute to I can spend a few days cleaning it up and completing some of
the core UI Controls.  The idea behind this project is not to be a complete
100% translation for winforms to cocoa.  But rather to allow you use use
mono winforms with a cocoa front end on a mac without completely rewriting
your UI.  Please respond back  if you have any interest in either using or
contributing to a project like this.

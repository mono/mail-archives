Ah, yes, if you'd built from source and not included the iPhone addin
that would be an issue. The MacDev addin is built as part of the
iPhone addin - it's basically the re-usable code I split out from the
iPhone addin for the MonoMac addin to use.

In theory the MonoMac addin does work on non-Mac platforms but certain
features would be disabled - Interface Builder support, and running
and debugging. I say "in theory" because you can't install it on other
platforms yet because they don't ship the MacDev addin.

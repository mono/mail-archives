>From my point of view:
1) thanks for the kind words
2) I have no problems with it going into the SVN repository

I wrote it so that others could enhance it and 'set it free'.

I have the original .doc file if that would make changing the documentation
format any easier - let me know and I'll forward it to you.

On the other hand, I know that Xcode V3 is around and will be making a
public appearance 'some time soon' with who knows what changes have been
made to the undocumented stuff that others have tapped into and I copied.

One other thing that may or may not be a future 'gotcha'. When Mono 1.1.18
came out with the problems with redirecting the standard input an output
streams, I tried to install a different (test) version elsewhere in my
directory structure. That made me realise that the location for the
compilers and runtime are hard coded into the plugin. I've not really have
much time to play with this to see if there is some way around it. I guess
it could be "left as an exercise for the student".

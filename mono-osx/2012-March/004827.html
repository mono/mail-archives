To find out for sure... on the Mac open Terminal.app.  Type mono --version.
 You should see something like:
$ mono --version
Mono JIT compiler version 2.10.9 (tarball Tue Feb 14 20:23:29 EST 2012)
Copyright (C) 2002-2011 Novell, Inc, Xamarin, Inc and Contributors.
www.mono-project.com
TLS:           normal
SIGSEGV:       normal
Notification:  kqueue
Architecture:  x86
Disabled:      none
Misc:          debugger softdebug
LLVM:          yes(2.9svn-mono)
GC:            Included Boehm (with typed GC)

You can then type: mono yourapp.exe.  If you are lucky it will run.  If not
you will get some errors and will most likely need to ask for help from the
original developer.

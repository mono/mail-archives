You need to edit the config file in
Mono.framework/Versions/Current/etc/mono.

Change the MonoPosixHelper line to this:
<dllmap dll="MonoPosixHelper"
target="/Library/Frameworks/Mono.framework/Libraries/libMonoPosixHelper.dyli
b"/>

That should cure the problem.

Regards
Gareth Baker

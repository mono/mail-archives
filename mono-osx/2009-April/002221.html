Sadly, at this point Mono will not be of any use to you on the iPhone.

If you want to use some parts of Mono, your best bet is to use Unity's
tool chain that uses Mono for scripting, but it is aimed at gaming.

We will support the iPhone ourselves, but it will be limited to the
Silverlight API when we do that.

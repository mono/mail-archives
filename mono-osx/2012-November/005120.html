It is possible to re-sign the code after the fact, using the codesign command line tool.

	https://developer.apple.com/library/mac/#documentation/Darwin/Reference/ManPages/man1/codesign.1.html

I do something similar to sign the third-party frameworks that get distributed with my app:

	codesign -f -s 'Developer ID Application' "${BUILT_PRODUCTS_DIR}/${WRAPPER_NAME}/Contents/Frameworks/Growl.framework/Versions/A"

(That's an example from Xcode, I haven't had to do this for a C# app yet.)

Hope that helps!

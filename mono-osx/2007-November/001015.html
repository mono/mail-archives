The patches on my blog were for compilation only.  We have seen no
incompatabilities with the binary version on Leopard.  Unfortunately
this means you're going to need to file a bug with a test case so we can
solve your problem.

In terms of the next package / release.  We just branched 1.2.6 so you
can expect to see that version packaged in the coming weeks.

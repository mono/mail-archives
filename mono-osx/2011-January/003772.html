Hello all

Here is a sample that uses the Growl framework.

Read the Readme.md file that is included to build the sample.

TwoMinuteGrowler
================

Shows the use of using Growl in your applications.  It is simple 
Two-Minute countdown timer that
notifies the user by Growl notifications of specific time events.

Also included is a patch to make some of the parameters as optional 
NullAllowed as per the documentation.

Any questions let me know.  There is a detailed process here, if needed, 
to include the Growl framework: http://cocoa-mono.org/?p=254

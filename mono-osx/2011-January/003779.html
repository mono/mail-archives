I thought I saw a response to your question in the other forum, unless it was
a similar question from someone else. Anyways, the answer I remember reading
is that you can do OS X development using Mono on Windows. But if you use
anything other than straight-up Mono, e.g. MonoMac, then you will only be
able to compile the code on Windows. You will not be able to run the code.
Additionally if using MonoMac you will not be able to wire up the app's GUI
because that requires Interface Builder.

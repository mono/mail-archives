This is weird...

In the dmg is the Monodevelop.app.  I copy it to my applications folder,
launch it and get gobs of missing dependencies errors/warnings...it's
basically DOA.  I go back and run the uninstaller script, reinstall Mono and
the Monodevelop.app, and still get the same thing.  Then I start comparing

MonoDevelop.app, not Monodevelop.app.

Hi,

Just thought I'd try to save others some aggravation...

There is an issue in the build process for mod_mono.  
You can find more details here: http://trac.macports.org/ticket/17217

The workaround I found for this is to call 'make' with the -B flag.

so your build would look like this:

./configure <change what ever flags you want here>
make -B
sudo make install


hope this helps someone,

Glenn

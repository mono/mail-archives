> So if you have this file saved as "app.config":
> 
> <?xml version="1.0" encoding="utf-8" ?>
> <configuration>
>   <appSettings>
>     <add key="MySettingKey" value="My setting value"/>
>   </appSettings>
> </configuration>

>

> doesn't.

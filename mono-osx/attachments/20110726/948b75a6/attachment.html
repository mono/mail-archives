<span class="Apple-style-span" style="font-family: arial, sans-serif; font-size: 13px; background-color: rgb(255, 255, 255); ">Hello,<br><br>I am proud to announce that a new release of Monobjc bridge is available (<a href="http://www.monobjc.net/" target="_blank" style="color: rgb(0, 0, 204); ">http://www.monobjc.net/</a>). The Monobjc bridge provides the necessary tools to develop and run .NET applications (written in C#, <a href="http://vb.net/" target="_blank" style="color: rgb(0, 0, 204); ">VB.NET</a>, etc.) that interact with Objective-C frameworks and libraries under Mac OS X. Existing Objective-C classes can be used in .NET code in an almost transparent manner.<br>

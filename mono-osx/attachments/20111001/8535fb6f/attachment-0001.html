<div>Hello,</div><div><br></div><div>I am proud to announce that a new release of Monobjc bridge is available (<a href="http://www.monobjc.net/">http://www.monobjc.net/</a>). The Monobjc bridge provides the necessary tools to develop and run .NET applications (written in C#, <a href="http://VB.NET">VB.NET</a>, etc.) that interact with Objective-C frameworks and libraries under Mac OS X. Existing Objective-C classes can be used in .NET code in an almost transparent manner.</div>

<div><br></div><div>What&#39;s new in the 4.0.1552.0 release:</div><div>- First stable release of the 4.0 series</div><div>- Support for new Mac OS X 10.7 API (Foundatio, AppKit, AVFoundation, etc)</div><div>- Ability to target Mac OS X 10.5, 10.6 and 10.7.</div>

<div>- MSBuild support (tasks and targets) for building, embedding, signing and packaging application.</div><div>- Support for Lion sandboxing entitlements</div><div>- MonoDevelop support extended to support Lion.</div><div>

<br></div><div>Note: If you are using Monobjc 3.0, the migration is easy. Check this page: <a href="http://www.monobjc.net/index.php?page=tutorial-migrating-from-monobjc-3">http://www.monobjc.net/index.php?page=tutorial-migrating-from-monobjc-3</a></div>

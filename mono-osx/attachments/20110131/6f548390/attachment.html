Hi,<div><br></div><div>Did you know that you can embed the WebKit browser if you use Java SWT? or you can use Safari or Mozilla as well.  WebKit comes as a component that you can load and add as a child window so that it&#39;s basically embedded in your app.  If you want to know more about this then let me know and i&#39;ll point you in the right direction.  If you just want a browser which isn&#39;t so integrated into your app. you can just invoke it via the JVM as a separate process and even communicate with it via XML or XMLRPC or whatever method you prefer.</div>

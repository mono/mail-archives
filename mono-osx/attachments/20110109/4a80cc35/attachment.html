    I am bran new to MonoMac, I installed MonoMac from a binary that
    didn't install the documentation into monodoc.&nbsp; This took me a while
    to figure out how to install yesterday, so i'll share so you won't
    have the same headache.&nbsp; I also haven't used *nix in years so my
    shell commands may be the long way around, but they still work :-)<br>
    <br>
    <ul>
      <li>Pull the latest and greatest MonoMac from GitHub <br>
        <b><a class="moz-txt-link-freetext" href="https://github.com/mono/monomac">https://github.com/mono/monomac</a></b><br>
        <br>
      </li>
      <li>Extract this to a location of your liking.<br>
        <br>
      </li>
      <li>Open up a terminal and go to that directory<br>
        <br>
      </li>
      <li>Copy MonoMac.dll to that directory.&nbsp; if you don't know where
        the dll is on your system, type the following command to find
        it.<br>
        <b>sudo find -f / 2&gt; /dev/null | grep -i monomac.dll$</b><br>
        <br>
        <i>Mine is:
/Applications/MonoDevelop.app/Contents/MacOS/lib/monodevelop/AddIns/MonoMac.dll</i><br>
        <br>
      </li>
      <li>Now run:<br>
        <b>mdoc update -o docs/en ./MonoMac.dll</b><br>
        <br>
      </li>
      <li>Three files should now be created in the docs directory called
        monodoc*, copy these files to<br>
        <b>sudo find -f / 2&gt; /dev/null | grep -i monodoc/sources$</b><br>
        <br>
        <i>Mine is:
          /Library/Frameworks/Mono.framework/Versions/2.8.1/lib/monodoc/sources/</i><br>
        <br>
      </li>
      <li>Last step is to rebuild the index / search.&nbsp; Go to were
        monodoc lives<br>
        <b>sudo find -f / 2&gt; /dev/null | grep -i
          MonoDoc.app/Contents/MacOS$</b><br>
        <i><br>
          Mine is:
          /Applications/MonoDevelop.app/Contents/MacOS/MonoDoc.app/Contents/MacOS</i><br>
        <br>
      </li>
      <li>Execute the following commands:<br>
        <b>mono browser.exe --make-index<br>
          mono browser.exe --make-search-index</b></li>
    </ul>
    Hope this helps someone.<br>

Hello,<br><br>I am proud to announce that a new release of Monobjc bridge is available (<a href="http://www.monobjc.net/">http://www.monobjc.net/</a>). The Monobjc bridge provides the necessary tools to develop and run .NET applications (written in C#, <a href="http://VB.NET">VB.NET</a>, etc.) that interact with Objective-C frameworks and libraries under Mac OS X. Existing Objective-C classes can be used in .NET code in an almost transparent manner.<br>

<br>What&#39;s new in the 3.0.1481.0 release:<div>- Monobjc is compatible with Lion</div><div>- MonoDevelop addins supports Xcode 4.x</div><div>- The CorePlot wrapper have been updated to CorePlot 0.4</div><div>- The Security framework is now wrapped.</div>

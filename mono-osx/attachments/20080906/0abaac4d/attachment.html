<div dir="ltr">Hello,<br>
<br>We are proud to announce that a new release of Monobjc bridge is available (<a href="http://www.monobjc.net/">http://www.monobjc.net/</a>). The Monobjc bridge provides the necessary tools to develop and run .NET
applications (written in C#, <a href="http://VB.NET">VB.NET</a>, etc.) that interact with
Objective-C frameworks and libraries under Mac OS X. Existing
Objective-C classes can be used in .NET code in an almost transparent
manner.<br>
<br>This release brings:<br>- a better support for Quartz 2D API<br>
- a better support for Core Animation for Mac OS X 10.5<br>
- a better performance by optimizing the messaging native calls<br>
- the support for ISynchronizeInvoke interface (cross-thread calls) for Mac OS X 10.4<br>
- an enhancement of the redistribution NAnt tasks (arbitrary assembly and native library can be embedded)<br><br>Downloads: <a href="http://www.monobjc.net/index.php?page=downloads" target="_blank">http://www.monobjc.net/index.php?page=downloads</a><br>
Tutorials : <a href="http://www.monobjc.net/index.php?page=tutorials" target="_blank">http://www.monobjc.net/index.php?page=tutorials</a><br>

Samples : <a href="http://www.monobjc.net/index.php?page=tutorials" target="_blank">http://www.monobjc.net/index.php?page=samples</a><br>

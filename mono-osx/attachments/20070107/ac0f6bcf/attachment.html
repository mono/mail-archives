I have made an enhancement to the plugin, so that there is an additional &quot;Cocoa# Executable&quot; project template.&nbsp; When creating this type of project, in addition to the default .cs file, you get two more files:<BR>
<BR>
&nbsp;&nbsp;&nbsp; ApplicationController.cs<BR>

We are proud to announce that a new release of Monobjc bridge is available (<a href="http://www.monobjc.net/" target="_blank">http://www.monobjc.net/</a>). The Monobjc bridge provides the necessary tools to develop and run .NET applications (written in C#, <a href="http://VB.NET">VB.NET</a>, etc.) that interact with Objective-C frameworks and libraries under Mac OS X. Existing Objective-C classes can be used in .NET code in an almost transparent manner.<br>



<br>
This release brings various fixes and enhancements for the day-to-day development.<br><br>

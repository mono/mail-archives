<br>I am proud to announce that a new release of <span>Monobjc</span> bridge is available (<a href="http://www.monobjc.net/" style="color:rgb(0, 0, 204)" target="_blank">http://www.<span>monobjc</span>.net/</a>). The <span>Monobjc</span> bridge provides the necessary tools to develop and run .NET applications (written in C#, <a href="http://vb.net/" style="color:rgb(0, 0, 204)" target="_blank">VB.NET</a>, etc.) that interact with Objective-C frameworks and libraries under Mac OS X. Existing Objective-C classes can be used in .NET code in an almost transparent manner.<br>

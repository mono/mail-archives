<HTML><HEAD><META content="text/html; charset=UTF-8" http-equiv="Content-Type"></HEAD><BODY>Bart ... if you were using textedit for editing this file, it is most probably saved as Rich Text file which can't be executed from terminal. You have to use terminal for this (or it is the easiest way)!!<BR>
<BR>
1. Start the terminal<BR>
2. type: cd /Users/<youruser>    <BR>
   // where you substitute <youruser> by your actual user name<BR>
3. type: nano monouninstall.sh    <BR>
  //  this starts the terminal text editor<BR>
4. paste the uninstall text into this editor<BR>
5. press: ctrl+x<BR>
  // this saves file<BR>
--------<BR>
another option is to save the monouninstall.sh directly to the /Users/<youruser> folder and continue directly with step 6.<BR>
--------<BR>
6. type: chmod a+x monouninstall.sh<BR>
 // this gives execution priviledges to the script you have just created<BR>
7. type: sudo ./monounistall.sh<BR>
 // this executes the uninstallment script, it will also ask you for your(administrator) password<BR>
8. type: rm monouninstall.sh<BR>
 // this deletes the uninstallment script<BR>

<span class="HOEnZb"><font color="#888888"><br></font></span></blockquote><div><br></div><div>What are you trying to do?</div><div><br></div><div>Are you trying to run an application? In that case, copy the necessary dll and dll.config files (e.g. Tao.OpenGl.dll &amp; Tao.OpenGl.dll.config) into the application directory.</div>

<div><br></div><div>Are you trying to develop an application via MonoDevelop? In that case, use the Add Reference to navigate to the dll and add it to your solution. You&#39;ll need to add the dll.config files by hand and enable the &quot;copy to output directory&quot; option.</div>

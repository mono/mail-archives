<blockquote class="gmail_quote" style="margin:0 0 0 .8ex;border-left:1px #ccc solid;padding-left:1ex">Mono 2.10.2 is a bugfix release from the 2.10 branch with more than 75<br>
bugs fixed.<br>
<br>
The Mac packages have been modified. Instead of a base runtime framework<br>
and an add-on C SDK we now publish a complete runtime framework (MRE -<br>
Mono Runtime Environment) and a complete development kit (MDK - Mono<br>
Development Kit), you know, kind of like how Java does it. This means<br>
mono embedding developers who used to download the Framework and the C<br>
SDK packages can now download just the SDK package. At the moment the<br>
contents of the Runtime package are the same as what the Framework<br>
package has contained and the SDK is that plus what was in the C SDK. We<br>
may refine what is in the Runtime package in the future but it will<br>
always be enough to run your favorite .NET programs on Mac.<br>
<br>

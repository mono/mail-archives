On Jan 23, 2007, at 11:36 PM, KielW wrote:

> Is there any work being done on dumbarton to include the new  
> API's?  i.e. Core Animation ?

I think you're misunderstanding what Dumbarton does. Dumbarton  
bridges objects from C# code into Cocoa, as opposed to Cocoa# which  
provides C# bindings for Cocoa (much in the same way that Gtk# works).

As to whether Cocoa# will support Core Animation, that's something  
for another mailing list member to answer ;)

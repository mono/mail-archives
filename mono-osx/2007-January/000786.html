Am 20.01.2007 um 19:17 schrieb Gregory Donovan:

> Just bought an intel mac.  Installed latest version of mono 1.2.2.   
> Unable to locate the uninstallMono.sh script.  I am uniinstalling  
> because I couldn't get winforms to work - got a  
> DllNotFoundException - exact message is:
>
> System.DllNotFoundException: /usr/x11r6/lib/libx11.dylib  -  
> recieved this error while running X-develop.
>
> Anyway - I'm new to Mac/OS X/Unix.  After the install was complete  
> I was given some info on the install and how to uninstall - here's  
> the specific text:
>
> "A simple uninstallMono.sh script is included in the disk image.   
> This is
> shell script that must be run as root, and it will remove the
> Mono.Framework and the links in /usr/bin."
>
> Help!

First of all, the Exception looks like you haven't installed X11 from  
your Mac OS X installation disk. If you just want SWF to work you can  
probably just install X11 now without reinstalling Mono and review  
the list archive for any further difficulties (you might need to open  
the app from X11's xterm).

As for the location of uninstallMono.sh, like you quoted it is  
located inside the disk image - that's the .dmg file you downloaded.  
With the disk image mounted run in a Terminal window: sudo /Volumes/ 
volumename/packagename.pkg/Contents/Resources/uninstallMono.sh (the  
tab key is your friend!)

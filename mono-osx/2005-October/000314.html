I have written a complex chat room (and other communication protocol)
in Mono, and deployed to OSX with no changes to code. It runs on Linux, MacOSX and Win32.
I use the wx.net (wx.widgets). Am very pleased with it.
I dont' develop on the mac however. In my experiences Mono is much more reliable
then MS .Net, I use it quite a bit now on Linux for server app. I am running a large Canadian
pizza franchise, completely backended on Mono and its been perfect. 

I think the big problem with Mono for Mac is the multimedia, i have used SDL.net on linux and 
Win32, but i dont know if SDL.net does the trick on MacOSX, as I havn't tried that yet.
And I don't know if SDL.net for OSX would require X11 backend in order to run?

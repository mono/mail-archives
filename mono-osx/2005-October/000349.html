Hi Timothy.
Just wanted to reply about what you say that Java seems to run slowsly on
Mac.
I can for real say that´s not true. Before i move to mac osx, i was a long
time windows and linux user, and java programer.
Then the first i did was to test java on an iMac 700mhz with java, and the
speed was enought. Then i did bought an iBook G4 1.1ghz, and still having my
AMD Sempron 2800+ with 1gb of ram and windows/linux, i did some loop tests,
array loading and things like that, taking in mind the seconds and
miliseconds needed... and the surprice was that even having 1.1ghz on my
ibook and 2ghz on my sempron, the speed was better on my mac.
So i can say that's not true, this is my opinion.
And a some other.. IDE´s also.. Mac can run most JAVA Ides, wich are good
for most task, like C, C#, JAVA, PHP, etc. I have Eclipse, and NetBeans,
mainly used NetBeans for JAVA programing, and its very good.
But, of course, if you want to have Visual Studio.Net or SharpDevelop on
mac, you will need to wait :)
Btw, today you can develop in JAVA, C (objetive or non objetive c),
Pascal/Delphi (FPC), Phyton, and a lot of other languages ussing XCode. Mono
is still new on mac, so with time, it will becomes the most used platform
for programing, on XCode or maybe on the ported SharpDevelop :)
What can i say... today im trying to move 100% to Mono on mac, but i still
have problems running it on mac. I have an issue on tiger that makes mono to
hung and then i need to restart to make the apps works on or two times.. its
weird, and happens when i do tests with windows.forms, also i have no other
problem like that with the other apps (and i have a lot) :).
But i hope that some day i can work on Mono instead work on Java.

Well, thanks for the chance to opine :)

Yep - If you're using the MVC 4 NuGet packages, you shouldn't have to do
anything extra (as NuGet packages are automatically bin deployed). Ensure
the references to all the MVC assemblies (like System.Web.Mvc and the Razor
assemblies) have "Copy Local" set to "true" in Visual Studio, and you
should be all set :)

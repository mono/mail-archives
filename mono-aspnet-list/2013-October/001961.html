Greetings,

I hit the exact same error. The issue is the version of XSP. You want to get
tag 3.0.11 from the git repo:

Github.com/mono/xsp

Compile 3.0.11. There is no need to run the make install step as it can be
run locally. There are two XSP binaries, xsp2 and xsp4. You want to use xsp4
for a .NET 4.0/4.5 solution.

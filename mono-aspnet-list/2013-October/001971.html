You should use Mono 3.2.3. I had a lot of problems with MVC3 and/or
previous versions of Mono (I'm using 3.0.7 for a WebForms project where it
"just works"), so I switched to MVC4 and Mono 3.2.3 and it all worked
flawlessly. I used the MVC4 packages from NuGet (beware: you have to
install the "legacy" package, the "current" one is for MVC5).

<div dir="ltr">Try <a href="http://www.tinymce.com/">TinyMCE</a> or <a href="http://ckeditor.com/">CKEditor</a>. Both are purely client-side and don't require any server-side code (and thus don't care about what the server environment is). Then you just add a regular text area (Html.TextareaFor in <a href="http://ASP.NET">ASP.NET</a> MVC, or one of the many ways of doing it in <a href="http://ASP.NET">ASP.NET</a> Web Forms - <asp:HtmlTextArea>, <asp:Textbox Mode="Multiline">, or <textarea runat="server">) and tell TinyMCE to initialise for all textareas.</div>

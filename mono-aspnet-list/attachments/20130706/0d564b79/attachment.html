    <br>
    Yes, it is possible.<br>
    <br>
    If you were using XSP, all you need is run XSP with the "--root"
    option set to "/var/www/HelloWorld".<br>
    Thus, it is expected that all .aspx and .<a target="_blank" href="http://aspx.cs/">aspx.cs</a> files are in this
    folder.<br>
    <br>
    But, with Apache mod_mono, you will need to configure HelloWorld to
    be an application. More info on [1].<br>
    <br>
    [1]
    
    http://gonzalo.name/blog/files/mod_mono.html<br>

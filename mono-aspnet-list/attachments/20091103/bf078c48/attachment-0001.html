&gt;&gt; However, whether you would use SQLite for a production site/app<br>

&gt;&gt; a connection pool. So if the site/app gets a lot of traffic it<br>
&gt;&gt; probably won&#39;t stand up well because each request is taking turns with<br>
&gt;&gt; the one connection. So it can work well for a few users but it does<br>
&gt;&gt; not scale real well to heavy web traffic.<br>

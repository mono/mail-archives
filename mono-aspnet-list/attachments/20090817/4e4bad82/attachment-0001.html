Hi,<div><br></div><div>Virtual hosting on Mono is definitely possible. I serve several virtual hosts under Apache on an openSUSE 11 box, some running Mono, some just PHP.</div><div><br></div><div>Sure, there are some quirks, but I am a big supporter of what the Mono team has accomplished so far, given the enormous task and its complexity. In fact, it&#39;s nothing short of amazing, and well worth assisting in ironing out the last few oddities.</div>

<div><br></div><div>I iterated until I got to this procedure and config file structure, now working on several machines:</div><div><br></div><div>- clean minimal install of openSUSE (11).</div><div>- use a package manager (e.g. YAST) to add apache2, mono-complete, (mysql/vsftpd), apache2-mod_mono, (apache2-mod_php5).</div>
<div>- uncomment the #NameVirtualHost *:80 in /etc/apache2/listen.conf</div><div>- allow the default server to include mod_mono.conf (/etc/apache2/default-server.conf includes all file in /etc/apache2/conf.d by default)</div>
<div><br></div><div><div>&lt;VirtualHost *:80&gt;</div><div>    ServerName site.domain</div><div>    ServerAlias www.site.domain</div><div>    ServerAdmin admin@site.domain</div><div>    DocumentRoot &quot;/home/username/htdocs&quot;</div>
<div>    ErrorLog &quot;/var/log/apache2/www.site.domain-error_log&quot;</div><div>    CustomLog &quot;/var/log/apache2/www.site.domain-access_log&quot; common</div><div>    &lt;Directory &quot;/home/username/htdocs&quot;&gt;</div>
<div>        Options Indexes FollowSymLinks</div><div>        AllowOverride None</div><div>        Order allow,deny</div><div>        Allow from all</div><div>    &lt;/Directory&gt;</div><div>  &lt;Location &quot;/&quot;&gt;</div>
<div>    SetOutputFilter DEFLATE</div><div>    SetEnvIfNoCase Request_URI &quot;\.(?:gif|jpe?g|png)$&quot; no-gzip dont-vary</div><div>  &lt;/Location&gt;</div><div>  &lt;IfModule mod_deflate.c&gt;</div><div>    AddOutputFilterByType DEFLATE text/html text/plain text/xml text/javascript</div>
<div>  &lt;/IfModule&gt;</div><div>&lt;/VirtualHost&gt;</div><div><br></div><div>I had issues with using the Mono directives (can&#39;t remember which ones, sorry).</div><div><br></div></div><div>Hope this helps someone.</div>

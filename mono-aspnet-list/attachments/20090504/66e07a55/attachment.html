I was following instructions on the Mono Project web site to install Mono on SUSE 11.1, but the commandline instructions do not work.  It says that the repository is invalid even though I can go to the link and have verified that the 2.4 packages are there.  Anyone have any ideas.

    
        This distribution supports installing packages via Zypper. Add the following repository to Zypper:

    * http://ftp.novell.com/pub/mono/download-stable/openSUSE_11.1

To add the repository, execute the following commands (as root):

    zypper addrepo <URL> mono-stable
    zypper refresh --repo mono-stable
    zypper dist-upgrade --repo mono-stable

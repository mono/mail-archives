removes many of the issues previously associated with writing complex web<br>

popular.  Mono on the other hand opens this technology up to a much wider<br>
audience. The two are a great combination.<br>

needs to accommodate the ISP’s. If ISP’s can run their ASP.net web hosts on<br>

using VirtualHosts in the apache2.conf file and then setting the ServerName<br>

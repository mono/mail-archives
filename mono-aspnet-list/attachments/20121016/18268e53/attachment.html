<DIV>AddCacheItemDependency is used to clear OutputCache in Mono Apache MVC2 
application using code below.</DIV>
<DIV>This is described in  <A 
href="http://stackoverflow.com/questions/11585/clearing-page-cache-in-asp-net">http://stackoverflow.com/questions/11585/clearing-page-cache-in-asp-net</A></DIV>
<DIV>        </DIV>
<DIV>In Mono, OutputCache is not cleared.</DIV>
<DIV>Looking into source code in GitHub shows that AddCacheItemDependency  
is not implemented in Mono.</DIV>
<DIV>How to fix this so that OutputCache can cleared ?</DIV>
<DIV> </DIV>
<DIV>Andrus.</DIV>
<DIV> </DIV>
<DIV>[OutputCache(Duration = 3600, VaryByParam = "none")]</DIV>
<DIV>public ActionResult Index()</DIV>
<DIV>{</DIV>
<DIV>  HttpContext.Current.Response.AddCacheItemDependency("Pages");</DIV>
<DIV>  return View();</DIV>
<DIV>}</DIV>
<DIV> </DIV>
<DIV>public ActionResult Refresh()</DIV>
<DIV>{</DIV>
<DIV>HttpRuntime.Cache.Insert( "Pages", DateTime.Now);</DIV>
<DIV>}</DIV>

Hi Oli,

  I managed to deploy a very basic (single home page) test MVC site successfully.

  So to resolve the issue, I had to force the Mono WCF package to install with;

$rpm Uvh --nodeps mono-addon-wcf-2.10.2-5.1.i386.rpm

  Then installed Mono MVC package normally with;

$yum install mono-addon-mvc-2.10.2-5.1.i386.rpm


  I really don't like having to force the package install in this way but it saved alot of hassle!


Good luck,

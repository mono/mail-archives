I've had little experience with it, but enough to know that sometimes your
solution and project files may change a little as you add-remove projects or
references. For the most part they seem to stay the same. With the latest
MonoDevelop, you can debug web projects, which is great. However, it's not
Visual Studio and MVC3 support while there isn't tailored. So while you can
still use it, you don't get intellisense or syntax highlighting for Razor.

Our work will be loading up with MacBookPros soon for all the .NET devs;
however I think we'll either be using BootCamp or Parallels for the Windows
environment for Visual Studio.

Hope that helps!
-Nathan

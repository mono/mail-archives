Not really, This is more about a difference with the underlying OS. On
windows the Local Application data is automatically created the first
time the user logs in, however on Linux, this directory is not
automatically created. So in both Microsoft .Net, and Mono, the
combination of Environment.GetFolderPath and Environment.SpecialFolder
get a path that the application should use for a specific purpose.
Neither Mono nor Microsoft .Net need to check the existence of these
directories because it is the operating systems job to provide them.

When using overloaded methods in ASP.NET MVC on Mono I keep getting "key was
not present in the dictionary"

I'm trying to do the following in my controller:

[ActionName("Item")]
[AcceptVerbs(HttpVerbs.Post)]
public ActionResult ItemPost() {}

[ActionName("Item")]
[AcceptVerbs(HttpVerbs.Get)]
public ActionResult ItemGet() {}

[ActionName("Item")]
[AcceptVerbs(HttpVerbs.Put)]
public ActionResult ItemPost() {}

[ActionName("Item")]
[AcceptVerbs(HttpVerbs.Delete)]
public ActionResult ItemDelete() {}

After testing I was able to narrow the problem down to the
ActionNameAttribute. I started by removing the AcceptVerbs attribute which
had no effect. I then removed the ActionNameAttribute and as expected had
the missing key. When I changed the method name to "Item" it worked as
expected (interesting side note is that if the method name is Item AND I
have the attribute set to "Item" I get the same key not found error.) Any
suggestions? This of course works fine in Windows.

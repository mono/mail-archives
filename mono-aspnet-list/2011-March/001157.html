You guys were correct. Under the covers there was an HTTPS call occurring. I
want to post this solution in case anyone else using the Google Docs Api
with dot Net wants to use mono on Ubuntu. The solution was to issue these
commands in a terminal window:

	sudo mozroots --import --ask-remove --machine
	sudo /etc/init.d/apache2 restart

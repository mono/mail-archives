Right now, gnome-sharp is a burial ground.  I think the only things
which are not deprecated at this point are Gnome.Program and

well, since libpanelapplet is in the desktop release.

gnome-desktop-sharp as gnome-panel-sharp.dll, if we choose the "bite the
bullet" option.

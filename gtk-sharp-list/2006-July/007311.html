> Usually I get my Mono compiled from Gentoo, but for this reason I

Did you install it to the mono prefix directory?  If not, you need to
set the MONO_GAC_PREFIX to the prefix were it was installed.  If you
didn't provide a --prefix option to bootstrap, you probably installed
to /usr/local.

> mcs tempapplet.cs -r:/home/dotxp/dev/C#/gtk-sharp/gnome/gnome-sharp.dll
> -r:/home/dotxp/dev/C#/gtk-sharp/gtk/gtk-sharp.dll
> -r:/usr/lib/mono/gtk-sharp/glib-sharp.dll

mcs -pkg:gnome-sharp-2.0 <sources>

That assumes you have installed the .pc files and set up PKG_CONFIG_PATH
if they went to a non-standard prefix.

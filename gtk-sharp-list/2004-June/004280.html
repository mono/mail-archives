Did you install the gtk+ runtime installer?
=20
Let's say you install gtk+ to C:\gtk+
=20
You would need to prepend C:\gtk+\lib to your PATH
=20
Did you copy all the folders and files found in C:\gtk+ and copy them to
your mono folder C:\Mono-Beta3=20
=20
Did you install the gtk# installer?
=20
Did you copy gtk# DLLs to C:\Mono-Beta3
=20
Did you prepend C:\GTK# to your PATH?
=20
Did you install each GTK# DLL to the Mono GAC via gacutil -i assembly:
=20
gacutil -i atk-sharp.dll
gacutil -i glib-sharp.dll
gacutil -i gdk-sharp.dll
gacutil -i glade-sharp.dll
gacutil -i gtk-sharp.dll
gacutil -i pango.sharp.dll
=20

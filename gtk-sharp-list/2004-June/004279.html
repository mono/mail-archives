Are you subscribed to the various email lists, such as, gtk-sharp-list,
mono-devel-list, mono-list?  If not, I suggest you subscribe to these
lists.

I actually meant to prepend, not append tot he PATH.
=20
=20
You could change it via the Control Panel.  Peformance Maintenance.
System.  Advanced tab.  Environment Variables:
Look for the System environment variable named Path.  Click Edit.
=20
If the root directory of Mono for you i:s=20
=20
     C:\Program Files\Mono-Beta3
=20
and you Path has lots of paths like :
=20
     C:\Blah
=20
Prepend it like:
C:\Program Files\Mono-Beta3;C:\Blah
=20
If you want to set the PATH from the MS-DOS command-line:
SET PATH=3DC:\Program Files\Mono-Beta3;%PATH%
=20
It is best to have the same root directory for Mono, gtk#, and gtk+ (and
dependencies).
=20
Did you install the gtk# assemblies in the Mono GAC?

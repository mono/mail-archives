GTKSharp GLWidget
http://www.golem3d.com/?q=node/40

If you compile on Windows it seems to work great on Windows and Linux.
For some reason it does not seem to work if compiled on Linux.  (That was a
while back, it may work.)

It is not a wrapper around a c++ version.  It performs direct pinvokes to
the system libraries.
It does allow for a shared context between controls.

Feel free to copy/modify/use the code any way you want.

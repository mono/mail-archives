for an mkbundle'd exe I did succeed with copyinthese files from Mono:

ATK-SHARP.DLL
GDK-SHARP.DLL
GDKSHARPGLUE-2.DLL
GLIB-SHARP.DLL
GLIBSHARPGLUE-2.DLL
GTK-SHARP.DLL
GTKSHARPGLUE-2.DLL
I18N.DLL
I18N.WEST.DLL
ICSHARPCODE.SHARPZIPLIB.DLL
MONO.CAIRO.DLL
MONO.DLL
MONO.POSIX.DLL
MONOPOSIXHELPER.DLL
PANGO-SHARP.DLL
SYSTEM.DLL
SYSTEM.XML.DLL
mono/2.0/MSCORLIB.DLL

In Addition you need an GTK+ Runtime which can be found at
http://www.gtk.org/download-windows.html.

Not all of the GTK+ Runtime is really required, you can strip it down if
you like.

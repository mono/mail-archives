On Mon, 2010-07-19 at 23:23 -0700, Stifu wrote:
> Is the download supposed to be twice bigger (from 8 MB to 16 MB)? Installed
> size also went from 19 MB to 44 MB, for the record.
> Just asking because it seems weird for a minor update. Besides, the fact it
> was only 8 MB was touted not so long ago.

The size increase is due to the inclusion of the translation files for
non-english language support.  We had several bug reports regarding the
lack of this capability, and decided that it would be better to include
it despite the size increase, since it is a significant feature of Gtk+.

The installer does still tout the huge reduction in size associated with
the removal of all the C development files, which is what dramatically
reduced the size before.

>From what I have seen printing in Gtk# is a pretty difficult business
and there is hardly any documentation on it, but printing using
System.Drawing.Printing is far easier. You can find a lot of tutorials
on printing with System.Drawing.Printing. There is also a project that
you can find in the Mono Documentation called Gtk.DotNet where you can
get a System.Drawing.Graphics object from a Gdk.Drawable.

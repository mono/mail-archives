On Wed, 2005-12-14 at 10:08 -0800, Daniel Morgan wrote:


> Some questions for Mike Kestner:
> What license does GTK# fall under?  MIT X11 or LGPL?

LGPL for the library assemblies, as indicated in COPYING and in the
commentary blocks of all source and custom files in svn and all
tarballs.

> What license does the samples in gtk-sharp release tarball fall under?
> MIT X11 or LGPL?  Or are samples in the public domain?

Unless another license is specifically indicated, they would be LGPL as
well.  Most tools like the gapi generator and parser are licensed GPL as
indicated in the commentary at the top of each source file.  

We have been lax about the sample code as far as getting license
statements into the source files.  Most are so trivial that it is hard
to imagine anyone would care if the code was reused.  But I would
contact the author of the sample if you want to apply anything other
than LGPL to your extensions unless a different license is specified in
the source file(s).

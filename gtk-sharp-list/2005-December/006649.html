As the error tells you, use the following command to compile:

mono -pkg:gtk-sharp -r:System.Drawing myCodeFileHere.cs

Or if you are using SharpDevelop/MonoDevelop you need to add the
assemblies as References ( System.Drawing, gtk-dotnet, gtk-sharp,
glib-sharp, gdk-sharp, pango-sharp, glade-sharp, etc. )

Just use C# events:

public delegate void MyHandler (uint foo, OtherType bar);

public class Foo {
	public event MyHandler MyEvent;
}

You raise the event with a call like:

MyEvent (foo, bar);

and connect to it with:

obj.MyEvent += new MyHandler (MyHandlerMethodName);

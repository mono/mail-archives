Hi,

If you want to update the gui, and receive events during a loop, try adding
this inside the "for":

while (Gtk.Application.EventsPending ())
        Gtk.Application.RunIteration ();

and if not necessary, comment "Thread.Sleep(3000);".


regards,

Pedro

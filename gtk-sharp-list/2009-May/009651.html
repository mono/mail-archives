Hey all,

I would like to announce the immediate availability of my pet project,
named Selene. Summarized in one sentence: It's an easier way to create
dialogs in your program.
More information, with elaborate code samples on the "Showcase" page,
can be found on the homepage:

    http://selene.ntlabs.org

If you are in for the straight dope, you can pull with

    git clone http://github.com/TobiasKappe/Selene.git

Currently two frontends are supported, Qt/Qyoto and Gtk#, the latter
being the most finished. All ideas, feedback and comments are welcome!

Regards,

Tobias

I'm not sure what you mean by "cairo backend". Pango.Cairo allow using

http://developer.gnome.org/doc/API/2.0/pango/pango-Cairo-Rendering.html

So I think it should be wrapped.

Printing Demo, but there must be other functions that is usefull. (I'm
no pango nor cairo expert, so I could'nt say).

Announcing the release of gtk-sharp-2.9.0 and gnome-sharp-2.15.0.
Tarballs are available for download at:

http://ftp.gnome.org/pub/GNOME/sources/gtk-sharp/2.9/
http://ftp.gnome.org/pub/GNOME/sources/gnome-sharp/2.15/

This is the first public release of the Gtk+ 2.10 API and the GNOME 2.16
API and is released in source form for the testing amusement of dwellers
on the bleeding edge.  We do not recommend moving production
applications to this release.

The API has been audited for backward compatibility and much of the new
2.10 API has been audited.  There may be some slight API instability in
the printing and recent chooser APIs as we work out the final bugs
leading to 2.10.0 and 2.16.0.

This release follows the module split previously announced on this list
as requested by the GNOME community, and is expected to be included in
the GNOME 2.16 Beta2 release.

You can get a raw IntPtr to the System.Drawing.Bitmap data, using:
var data = bmp.LockBits(new Rectangle(0, 0, bmp.Width, bmp.Height),
PixelFormat.Format32bppRgba, ImageLockMode.ReadOnly);
// data.Scan0 contains the pointer
IIRC you can construct a Gtk.Image from a raw pointer (I'm pretty sure I
have used this before). Just make sure the pixel formats match and pass this
pointer along with data.Stride (if necessary).
Don't forget to unlock and dispose the bitmap when you are done:
bmp.UnlockBits(data);
bmp.Dispose();

OK, I've set up the jobs for this.

The regular job which just builds commit to the repository is here:
http://monojenkins.cloudapp.net/job/GtkSharp/

The job that builds pull requests (along with instructions on how to
use it) is here:
http://monojenkins.cloudapp.net/job/GtkSharp-Incoming/

Right now, these jobs only build master, but I can look at expanding
to other branches later.

Let me know if you have any issues or need adjustment to the configuration.

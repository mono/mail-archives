Hi,

The location of different versions of release source tarballs is...
inconsistent. I'm wondering if there is any particular reason for this, and
if so if that can be resolved. Which site is supposed to be the canonical
site for the source tarballs?

In summary, I suppose what should maybe be done is to upload the missing
releases to each site, and perhaps clean up the folder structure on the
mono site.

(Background: I'm getting tired of doing the detective work each time I need
to rebuild.)


The issues:

GTK-SHARP
Gnome FTP (http://ftp.gnome.org/pub/gnome/sources/gtk-sharp/) seems to
contain a tidy collection of most releases including 2.99.x, BUT the last
couple of versions in the 2.12.x series are missing. 2.12.11 to 2.12.26 are
only available from mono project:
http://download.mono-project.com/sources/gtk-sharp212/. This one in turn
does NOT carry the 2.99.x releases.

Also, the mono site has the following directories:
http://download.mono-project.com/sources/gtk-sharp-2.0
http://download.mono-project.com/sources/gtk-sharp-2.8
http://download.mono-project.com/sources/gtk-sharp-2.10
http://download.mono-project.com/sources/gtk-sharp
http://download.mono-project.com/sources/gtk-sharp2
http://download.mono-project.com/sources/gtk-sharp28
http://download.mono-project.com/sources/gtk-sharp210
http://download.mono-project.com/sources/gtk-sharp212

And yes, many of the same tarballs are found in several of the directories.
Is there any technical reason preventing this from being cleaned up to a
structure similar to the one on GNOME FTP?



GNOME-SHARP
Here it  appears that GNOME FTP (
http://ftp.gnome.org/pub/gnome/sources/gnome-sharp/) has all the releases
(2.15 up to 2.24.2), while the mono site contains 2.24.1 but not 2.24.2.
And again, the mono site is split in several folders with partial overlap:

http://download.mono-project.com/sources/gnome-sharp2/
  2.16.0, 2.16.1, 2.20.0, 2.24.1

http://download.mono-project.com/sources/gnome-sharp220/
  2.20.0, 2.20.1



GNOME-DESKTOP-SHARP
Here it appears that GNOME FTP (
http://ftp.gnome.org/pub/gnome/sources/gnome-desktop-sharp/) has all the
releases (2.20 up to 2.26), while the mono site (
http://download.mono-project.com/sources/gnome-desktop-sharp2/) only has
2.20.1 and 2.24.0.

Furthermore, the 2.26 release is not tagged in git.


It can also be noted that the project page (
http://www.mono-project.com/docs/gui/gtksharp/) only referenced GNOME FTP
until just a few minutes ago when my update to also link to the mono
download site was accepted.

Oskar

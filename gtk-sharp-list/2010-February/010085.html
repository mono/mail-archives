Hi Ian,

You want to use the "DeleteEvent" event:

window.DeleteEvent += delegate (object o, DeleteEventArgs args) {
// Do something
args.RetVal = trueOrFalse; // false if you still want to close the window,
true if you want to keep it open
};

-Matthew

Ah, please note that in order to run the demo; you need to put images
with the following filenames in the gtk-sharp/samples/pixmaps directory
and run it from that directory 

"background.png"
"apple-red.png"
"gnome-ccdialog.png"
"gnome-gmenu.png"
"gtk-sharp-logo.png"
"gnome-calendar.png"
"gnome-applets.png"
"gnome-color-browser.png"
"gnome-mdi.png"

I think it's a little bit stupid to send standard gnome-icons to a
mailinglist ;-) so I exluded them

You can, of course, change the filenames in the .cs-file too

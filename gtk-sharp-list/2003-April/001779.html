I do this in nunit-gtk:

	void GrayOut (TreeIter iter)
	{
		SetValue (iter, 0, grayCircle);
		TreeIter child;
		if (!IterChildren (out child, iter))
			return;
		do {
			GrayOut (child);
		} while (IterNext (out child));
	}

If the 'iter' parameter is the first node, this method traverses all of
its childrens.

Rephrasing Latexer thoughts as a code snippet, either:

          Label label = new Label("Lord_Phoenix");
          label.UseUnderline = false;
          treeview1.Columns[0].Widget = label;
          label.ShowAll();

or

          Label label = new Label(null);
          label.Text = "Lord_Phoenix";
          treeview1.Columns[0].Widget = label;
          label.ShowAll();

:)

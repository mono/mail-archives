Hi Fredrik,

> private void OnDeleteEvent(object sender, DeleteEventArgs args) {
>     this.Hide();
>     args.RetVal = true;
> }

You are entirely correct - that works! Thanks!

Martin

> I'm not sure if this will work, as I can't test this right now.

>
> private void OnDeleteEvent(object sender, DeleteEventArgs args) {
>     this.Hide();
>     args.RetVal = true;
> }

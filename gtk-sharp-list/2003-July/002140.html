If you notice the comment at the top of the xml file, we do not modify
those files directly.  These types of updates are made via rules in a
metadata file.  If you want to try to supply such a patch, take a look
at some of the "pass_as" rules in sources/*.metadata as an example of
what to add to the Pango.metadata file.

If you would prefer not to get down and dirty with the source parser,
please file a bug and I'll add the rules on my next bug frenzy.

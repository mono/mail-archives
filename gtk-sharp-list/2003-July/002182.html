On Sat, 2003-07-19 at 15:14, Alejandro Mery wrote:
> hi,
> i'm new to gtk# and i see you have perl-modules to download
> are they mandatory to build gtk#?
> can you package them as .tar.bz2 or .tar.gz in adition to current .src.rpm?

The perl module rpms are there for people who want to install the
gtk-sharp-gapi package.  The gapi package includes the source parser and
code generator used to produce bindings for C libraries.  

They are not necessary for anybody who just wants to build Gtk# since
the package provides XML API files for the libraries bound by Gtk#. 

If you want to build the perl module dependencies for the parser from
source, you can download the source tarballs for all those rpms from
cpan.

Hello,

I've been pretty quiet lately (because of work).  But I thought
I'd put out an Mandrake 9.1 RPM for Gtk# 0.10.

You can get it from:

http://download.reptile.ca/opensource/GtkSharp/gtk-sharp-0.10-1mdk.i686.rpm

This RPM does not contain GnomeDB#, Gda#, or GStreamer#.

(GnomeDB# and Gda# were not included because I wasn't able
to get the latest gnome-db and gda to compile on Mandrake 9.1.
I'll try some more later.)

If anyone finds any problems with it, let me know please.

(The RPM was made with "checkinstall", so there is no SRPM.)

For ideas, you can look at the DataGrid.cs in SQL# For GTK#
found at mcs/tools/SqlSharp/gui/gtk-sharp.

This DataGrid contains a TreeView that uses the ListStore tree model.  For
data binding, it uses the various classes and interfaces defined in
System.ComponentModel - this allows it to not be dependent upon System.Data.

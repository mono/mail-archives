If you add a widget to the window without any "layout" widgets, that
one widget will take up all the space it can by default (usually?).
 You may have to set the X Fill/Y Fill if you use any layout item
such as those deriving from Gtk.Box.


For locking the parent window size, you'd just have set Resizable to
false.
[In Glade this is readily configurable and would show you how things
look.]

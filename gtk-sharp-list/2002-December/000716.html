Actually, Cygwin does include GNU make.  The Cygwin Setup .exe does not
install the make nor other binutils by default.  When installing cygwin, you
have to manually select these options.
Just use the cygwin setup that you ran earlier, except this time, make sure
you have selected make, binutils, autoconf, automake, etc for installation.

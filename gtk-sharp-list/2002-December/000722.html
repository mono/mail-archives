Based on Daniel Morgans binaries, I put together a new gtk# release for
windows. It is thought as an Add-On to the windows setup on the mono
download page. Beside the gtk# binaries it includes all dependecies (mainly
gtk+).

Also I worked on the default theme and it now matches the look of windows
95/98/ME/NT/2000. It might look a bit outdated on Windows XP.
A screenshot is attached.

Note: You'll need to mention the complete path, when referencing the
gtk-sharp assemblies:

mcs hello.cs -r c:\mono-0.17\install\lib\gtk-sharp.dll -r
c:\mono-0.17\install\lib\glib-sharp.dll

http://mono.jroith.de/gtk-sharp-0.6-stable.exe (~ 3, 5 MB)

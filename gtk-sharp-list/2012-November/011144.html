On 04/11/2012 05:57, Nicholas Frechette wrote:
> I have a GTK# application I run on Ubuntu 12.04. It works great except for the
> scroll bars.
> I removed the overlay scrollbars package to use the classic ones instead and it
> works for all applications except my home built GTK# app. Somehow they still
> display the overlay version.
> 
> Any idea on how to fix this would be welcome! I hate those overlay scrollbars :(

There are separate gtk2 and gtk3 overlay-scrollbar packages
(overlay-scrollbar-gtk2 and overlay-scrollbar-gtk3). Check that you have both of
them removed. Specifically, the -gtk2 one should correspond to the one that Gtk#
is using, because we don't have Gtk#3 yet.

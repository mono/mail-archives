Version 1:

ulong s = 5;
s = s*1000*1000*1000;
uint seekType = (uint)SeekType.MethodSet;

And I tried the following:

long s = 5;
s = s*1000*1000*1000;
uint seekType = (uint)SeekType.MethodSet;

the bindnings. Any help or pointing me to example code would be really

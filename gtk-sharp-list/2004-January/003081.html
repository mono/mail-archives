I personally don't think "beta bindings" need to be tracking the latest
api so early in the process.  If they are able to make a solid release

to follow the schedule then.  In addition, with this being the first
bindings release and having been notified only a month or so ago it
would be nice for additional leeway.

Anyways, I think it is a great idea, and I hope we can be a part of it
soon.

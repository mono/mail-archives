It looks like you are including GtkHtml wrappers in your gtk-sharp tarball.

Please note that your GNOME Platform Bindings modules should not be wrapping
non-GNOME-Platform APIs in the same module:
http://developer.gnome.org/dotplan/bindings/rules.html#WhatShouldBeWrapped
So Gtk# really needs to change this for the release set.

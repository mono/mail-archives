Hello guys,

    The Glade.GladeWidgetAttribute has been renamed to
Glade.WidgetAttribute, if you had code that used the GladeWidget
attribute, and looked like this:

	[GladeWidget]
	Widget w;

    Change it to:

	[Glade.Widget]
	Widget w;

Miguel.

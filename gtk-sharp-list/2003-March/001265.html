	So - if you look at:

	ORBit2/include/orbit/orb-core/orbit-small.h

remote object type introspection.

	ORBit2/src/orb/orb-core/orbit-interface.idl

	It's worth doing the client side first - since that's the easiest; we
need to do some work on the server impl. IMHO.

	For working code and an outline of how it all works, you can see
gnome-python/pyorbit/README ( and the source in there ).

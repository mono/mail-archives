just posted the last comment on that Bugzilla report a couple weeks ago.

because using my 2.10 builds with any version of gtk# was causing weird
issues, but using Ivan Wong's builds seemed to work fine.  It turns out,

have been unable to run OpenVista (or any gtk# app that uses tree views)
with any build of GTK+ when using gtk# 2.10.

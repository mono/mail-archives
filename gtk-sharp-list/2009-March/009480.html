This framework already includes GTK#.

Most likely you'll have to set PKG_CONFIG_PATH to include the
directory with the GTK# and Mono .pc files.

MonoDevelop trunk (& beta 2) detects this directory automatically, so
all the libraries show up in the add references dialog, and it can
even build itself.

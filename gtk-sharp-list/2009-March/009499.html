On Mon, Mar 09, 2009 at 05:22:18PM -0500, Iggy wrote:
> Does the solo Gtk# installer for Windows install the Gtk libs in the .Net GAC?

It installs the Gtk# .dlls into the .NET GAC, yes. It does *not* install
the native gtk+ .dlls into the GAC. Those are kept in a directory that
is added to the PATH environment variable.

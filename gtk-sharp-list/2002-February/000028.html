Figured I'd send another message before the sysadmins close the list on 
me for lack of activity.

Tonight I committed a patch that completes the first pass at method 
generation.  At this point, all the object methods, constructors, and 
properties are generating cleanly-compiling C# wrapper code.  

Another aspect to tonight's commit is the removal of the xml file from 
CVS.  There was some discussion on mono-list related to this and I've 
decided to provide updates to the xml as a separate package via the 
sourceforge project.  So, if anyone is actually compiling Gtk# at this 
point, you'll need to periodically check for a new gtkapi package 
release on the gtk-sharp project page.

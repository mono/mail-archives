2006/3/3, Francisco T. Martinez <martinf at mfconsulting.com>:
> Redefined Horizons wrote:
> >Can this be done with Visual Basic .NET on Mono, or is GUI design with
> >Glade only supported for C#?
> >
> >
> Yes.  If you install the Gtk# Installer for .NET Framework SDK and the
> presence of Visual Studio .NET 2003 is detected during the install, a
> VB.NET project template to create Glade# applications will be added to
> your system.  I must point out, that VB.NET expertise is not the mono
> community specialty.

There is a reason for this, actually two:

1) Mono's VB.NET compiler is currently in a beta level, it is not
released yet. It is, however, said to be very stable and it is
included in Mono's Linux installers (I do not know if it is in the
Windows installers, but it probably is too).

2) The community dislike VB.Net (as it brings BAD memories). They aree
making a VB.Net compiler as a migration tool because Microsoft's land
programmers  like it so much. This may change whever mbas (the
compiler's name) is released, but I wouldn't count on it. Besides, we
got our own languages that we can use without falling that low: C#,
Boo, Nermele, Logo, and many more. See
http://www.mono-project.com/Languages . As a rule, if a third party
compiler is available for NET, you can install it on Mono.

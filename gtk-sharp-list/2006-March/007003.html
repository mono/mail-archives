Hi Paul,

In the same directory of the example there is a file called compile.sh.
Its a script file which compile all the examples. For this example:

mcs -r:System.Drawing -r:Mono.Cairo -pkg:gtk-sharp circles.cs sysdraw.cs

sysdraw.cs was importing libgdk-x11-2.0.so while in my computer it is
libgdk-x11-2.0.so.0. Maybe you are having the same problem.

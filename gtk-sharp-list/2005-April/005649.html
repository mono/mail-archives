GMap is released!

GMap is an opensource map software that lets you view the street map 
anywhere in the US and get directions from place to place offline.

It uses GTK#, the Tao framework, and the mysql connector for .NET.  The 
program runs under linux and windows without any modification (of 
course, it uses mono!)

http://gmap.sourceforge.net

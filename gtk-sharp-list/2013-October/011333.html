Richard,

Which version of Gtk# are you looking for?

If you want the latest stable release (2.12.21 at the time of writing) 
you can get a standalone installer from the standard mono download page 
<http://www.go-mono.com/mono-downloads/download.html>. To get to the 
Gtk# only installer:

  * In the "Previous Stable Version 2.10.x" section
  * Click the Windows icon
  * Three links will now appear in a new section "2. Download Mono for
    Windows"
  * Select "Gtk# for .NET (New version compatible with Xamarin Studio)"

Here is a direct link 
<http://download.xamarin.com/GTKforWindows/Windows/gtk-sharp-2.12.21.msi> as 
well.

Announcing the joint release of versions 2.5.5 and 1.9.5 of Gtk#.  These
releases are unstable development releases.  They are not
parallel-installable with each other, but either release can be
parallel-installed with the latest stable release.

Tarballs and binaries are available for download from:

http://www.mono-project.com/Downloads

About Gtk#: 

Gtk# is a .Net language binding for the Gtk+ Graphical User Interface
toolkit and several GNOME platform libraries. Release 1.9.5 targets the
Gtk+ 2.4 libraries.  Release 2.5.5 targets the Gtk+ 2.6 libraries.

Changes since last release:

* Version 2.5.5 is the first release of 2.6 bindings. (MK, Dan Winship)
* New working PangoAttribute binding. (Dan Winship)
* Win32 support for Gtk.DotNet.Graphics S.D interop. (Sebastian Faltoni)
* Generated glue and field bindings for objects and opaque types (DanW)
* Generate DestroyNotified delegate parameters properly. (DanW)
* Sized Pixbuf constructors and loaders. (Jose Faria)
* TreeEnumerator/IEnumerable support for ListStore (Eric Butler)
* Additional API customizations and Corrections: (John Luke, DanW, MK)
* Bugfixes (Todd Berman, Ben Maurer, DanW, Jeroen Zwartepoorte, MK)
* documentation additions (Miguel, John Luke, DanW, MK)

Discussion of Gtk# occurs on the mailing list at
gtk-sharp-list at lists.ximian.com.  Bugs can be reported to
http://bugzilla.ximian.com, module gtk#. Check the Gtk# homepage for
additional info at http://gtk-sharp.sf.net.

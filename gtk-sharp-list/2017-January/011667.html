Hi,

If you want to have the dialog with the button "yes" set by default, then
add the line:
md1.DefaultResponse = Gtk.ResponseType.Yes;

Before the md1.Run() line. Also, remove the md1.Show() line, you don't need
that.

-- Baltasar

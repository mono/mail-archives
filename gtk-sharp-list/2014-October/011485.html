Bertrand-2 wrote
> Hello,
> 
> Styling with CSS is a feature introduced in GTK+ 3.0 (the native C
> library).
> Currently, stable versions of GTK# target GTK+ 2.x, as consequence they
> don't provide the CssProvider class.
> 
> So if you want to use CSS styling in your GTK# application, you need to
> use
> GTK# 2.99.x, which is still in the beta stage:
> https://github.com/mono/gtk-sharp/releases
> 
> The Banshee source in git master has been ported to use GTK# 2.99.x,
> that's
> why it uses CssProvider.
> 
> I hope this clarifies things for you.
> 
> --

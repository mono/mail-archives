You have to add the TreeView to a parent widget and then call ShowAll() on the parent (or the top level widget). For example if the parent is an Gtk.HBox you do it like this:<br>
<br>
teamListView = new Gtk.TreeView ();<br>
hbox1.Add (teamListView);<br>
<br>
// Setting up your model and columns<br>
<br>
hbox1.ShowAll ();<br>
<br>
More info on TreeViews in C# can be found here:<br>

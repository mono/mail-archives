    <DIV><FONT face=Arial size=2>Thanks in advance</FONT></DIV></BLOCKQUOTE>
  <DIV>Here is how:<BR>// create the dialog<BR>// remember to choose correct 
  FileChooserAction from these self-explanatory values:<BR>// Open, Save, 
  SelectFolder, CreateFolder<BR>FileChooserDialog Fcd = new FileChooserDialog 
  ("Open file", null, FileChooserAction.Open);<BR><BR>// add buttons you wish to 
  see in the dialog<BR>Fcd.AddButton(Stock.Cancel, 
  ResponseType.Cancel);<BR></DIV>Fcd.AddButton(Stock.Open, 
  ResponseType.Ok);<BR><BR>// then create a filter for files. For example 
  .gif:<BR>// filter is not necessary if you wish to see all files in the 
  dialog<BR>Fcd.Filter = new 
  FileFilter();<BR>Fcd.Filter.AddPattern("*.gif");<BR><BR>// if you wish to 
  select multiple files set the do the following:<BR>Fcd.SelectMultiple = 
  true;<BR>// In this case read Filenames-property instead of 
  Filename-property<BR><BR>// run the dialog<BR>ResponseType RetVal = 
  (ResponseType)Fcd.Run();<BR><BR>// handle the dialog's exit value<BR>// Read 
  the file name from Fcd.Filename<BR>if (RetVal == ResponseType.Ok) {<BR>// do 
  something<BR>} else {<BR>// do something else<BR>}<BR><BR>// destroy the 
  dialog<BR>Fcd.Destroy();<BR><BR>PS. There is a documentation covering gtk# at 
  <A class=moz-txt-link-freetext 
  href="http://docs.gotmono.net/">http://docs.gotmono.net/</A>.<BR>

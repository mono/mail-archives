Hi. When I have to set an item in a Combo box I do the following:<br>
<br>
First I have to know which TreeIter is containing the value I want to
set. If I don't know that TreeIter, then I have to search it by hand
doing the following:<br>
<br>
<span style="font-family: courier new,monospace;">TreeIter ti;</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">comboBoxStore.GetIterFirst(out ti)</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">do</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">{</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">if ((int)comboBoxStore.GetValue(ti,1)==2 /*The value I look for*/ )</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">&nbsp;break;</span><br style="font-family: courier new,monospace;">
<br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">}while (comboBoxStore.IterNext(ref ti))</span><br style="font-family: courier new,monospace;">
<br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">//This loop doesn't check if a value has been found so I will check if the last TreeIter has the value I look for</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">if ((int)comboBoxStore.GetValue(ti,1)!=2 /*The value I look for*/ )</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">{</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">&nbsp;//Haven't found the value in the TreeStore</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">}</span><br>
<br>
once I know The TreeIter I set it as the active TreeIter in the ComboBox:<br>
<br>
<span style="font-family: courier new,monospace;">cbFiltro.SetActiveIter(ti);</span><br style="font-family: courier new,monospace;">
<br>
if I don't want to search the TreeIter I have to know it beforehand saving it somewhere:<br>
<br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">TreeIter[] ti=new TreeIter[3];<br>
</span><span style="font-family: courier new,monospace;">ti[0]=comboBoxStore.AppendValues(&quot;Option #1&quot;, &quot;1&quot;);</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">ti[1]=comboBoxStore.AppendValues(&quot;Option #2&quot;, &quot;2&quot;);</span><br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">ti[2]=comboBoxStore.AppendValues(&quot;Option #3&quot;, &quot;3&quot;);</span><br style="font-family: courier new,monospace;">
<br style="font-family: courier new,monospace;">
<span style="font-family: courier new,monospace;">(...)</span><br style="font-family: courier new,monospace;">
<br>
<span style="font-family: courier new,monospace;">cbFiltro.SetActiveIter(ti[1]);</span><br style="font-family: courier new,monospace;">
<br><br>
I suppose this should work for you too.<br>

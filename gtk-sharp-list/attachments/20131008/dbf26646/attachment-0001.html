    Which version of Gtk# are you looking for?<br>

    If you want the latest stable release (2.12.21 at the time of
    writing) you can get a standalone installer from the standard mono <a
      href="http://www.go-mono.com/mono-downloads/download.html">download
      page</a>. To get to the Gtk# only installer:<br>
    <ul>
      <li>In the "Previous Stable Version 2.10.x" section</li>
      <li>Click the Windows icon</li>
      <li>Three links will now appear in a new section "2. Download Mono
        for Windows"</li>
      <li>Select "Gtk# for .NET (New version compatible with Xamarin
        Studio)"</li>
    </ul>
    <p>Here is a direct <a

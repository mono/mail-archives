</div>I believe  the conventional way is to create a 1x1 transparent cursor<br>
and use that:<br>
<br>
using (Pixmap inv = new Pixmap (null, 1, 1, 1)) {<br>
        invisibleCursor = new Cursor (inv, inv, Gdk.Color.Zero, Gdk.Color.Zero, 0, 0);<br>
}<br>
<br>
For an example, see<br>
<a href="http://anonsvn.mono-project.com/viewvc/trunk/monodevelop/main/src/core/Mono.Texteditor/Mono.TextEditor/TextEditor.cs?view=markup" target="_blank">http://anonsvn.mono-project.com/viewvc/trunk/monodevelop/main/src/core/Mono.Texteditor/Mono.TextEditor/TextEditor.cs?view=markup</a><br>

    I'm currently working on a similar thing, however,  I'm writing on
    Linux and trying to distribute to Windows. <br>
    <br>
    You could make a new .msi file by using a separate program called
    NSIS.  You can get it from here:
    <meta http-equiv="content-type" content="text/html;
      charset=ISO-8859-1">
    <a href="http://nsis.sourceforge.net/Main_Page">http://nsis.sourceforge.net/Main_Page</a><br>
    <br>
    It is fully customizable, so as long as you can compile Gtk-sharp,
    you can package it up with NSIS.<br>
    <br>
    The good news, is that once you learn the relatively easy script
    language of NSIS, you can use that to make your distributable
    install file for your program, and you can pack in the gtk-sharp
    file along with it.<br>

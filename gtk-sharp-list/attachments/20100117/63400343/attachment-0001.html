If I need only an image without the text and I don't need all of the
features in the Gtk.Button I will normally just use a Gtk.Image in a
Gtk.Event container widget, of course that doesn't have all the
benefits a Gtk.Button does but I makes for a quick simple solution to
the button border problem.<br>

I think the padding in the button is a theme-level property, i.e. not
really supposed to be touched by app writers. Despite this,  you can
override it using a gtkrc settings file for your app, or by code, e.g.
something like
<a class="moz-txt-link-freetext" href="http://anonsvn.mono-project.com/viewvc/trunk/monodevelop/main/src/core/MonoDevelop.Ide/MonoDevelop.Ide.Gui/DragNotebook.cs?r1=135850&amp;r2=136641">http://anonsvn.mono-project.com/viewvc/trunk/monodevelop/main/src/core/MonoDevelop.Ide/MonoDevelop.Ide.Gui/DragNotebook.cs?r1=135850&amp;r2=136641</a>

You might have to poke around in the gtk+ docs to find the right property.

Hi,<br><br>as far as I know, you can only use it in your source, but not in the GUI editor.<br>To use the classes of GtkSourceView, you have to put the following line to the start of your source files:<br><br>uses GtkSourceView;<br>
<br>Don&#39;t forget to set a reference to the gtksourceview-sharp package in your project, by right clicking on &quot;References&quot; in the solution browser panel and choosing Edit references. Browse the list of installed packages and tick the package &quot;gtksourceview-sharp&quot;.<br>

Do you know about glade marko? It's a nice interface designer make for gnome
development.

As touched on, Luis is taking glade (the third version) and integrating it
into monodevelop. While this si going to be nice and more friendly
(especially with code generation to make it fully integrated), it's still ok
using glade seperatley and referencing the glade files in monodevelop.

As I understand there is also another GUI creator specifically for mono
called stetic in development (it produces glade XML stuff but is fully
written in c# and made just for mono development IIRC). As I understand the
integration with glade 3 will be the first step, then when ready stetic will
become the way it's done.

On a side note, I for one was surprised that glade 3 was being integrated
first, does that mean that stetic is just taking a long time or is it
underdeveloped right now?? I'm sure it's not easy to make a UI designer like
that so please don't take that as a jab if it is in full development right
now.

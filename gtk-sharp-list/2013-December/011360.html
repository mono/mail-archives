Hi, not much a gtk-sharp question but still..

I don't known if our how you can do it with "SQL SERVER" but I would try to
export your data as a SQL script made of CREATE TABLE and INSERT statements
and then simply run this script on the MySQL side, do your data processing,
re-export the data as a SQL script and run the script on the "SQL SERVER"
side.

Also, before all that, if possible, do a backup of the original data just
in case...

Hope that help!

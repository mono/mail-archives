Hi!

There is a tool on cvs to validate doc files for mono. It uses a schema
to validate not just wellformd validation. Wellformed testing only shows
wether it is XML or not.  So using a schema checks the semantics as well.

location:
   monodoc/doctools/valdoc

example:
   make
   valdoc ../../class/corlib/en/System.IO/TextReader.xml

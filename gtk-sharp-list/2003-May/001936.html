Hello,

There's a tutorial out on using the Glade2 GUI Builder program.
(A program much like Visual Basic, but for building GTK+ and
GNOME applications.)

http://www.kplug.org/glade_tutorial/glade2_tutorial/glade2_introduction.html

This is useful for people wanting to use Glade#.

Thus... learn the tutorial.  Make you GUI by creating
a .glade file.  (The .glade file will define the look
of your GUI.)  But do NOT use the Glade program to
do your coding!

Then use C# to write your program.  And use Glade# (from
your C# program) to use that .glade file you made.


See ya

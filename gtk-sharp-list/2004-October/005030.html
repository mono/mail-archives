This looks really cool you guys. I think a large Gtk# demo would be
great to help people learn the API - I know I certainly learn best from
example.

Orlando, I have one small recommendation to make. If you plan on
releasing this to the public as example code, might I recommend a
license change? When it comes to academic material such as this, it's
nice if the author licenses the code under a BSD-style license, or
something equivalently free. Licensing academic code under the GPL
prevents a wide range of hackers from using your example code. (For
example, BSD-licensed code can go into a GPL project, but not the other
way around.)

I have no intention of starting a slashdot-esque licensing war on this
list, so take that recommendation with a grain of salt. :) In the end,
it's your code and you're free to license it however you like.

Keep up the great work guys! I look forward to seeing what you come up
with.

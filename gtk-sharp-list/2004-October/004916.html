Gracias Everaldo:

Andrea to add to the link that Everaldo gave you, I want to mention that 
we now have a Gtk# Win32 Runtime installer.  This latest installer 
should be ideal for people who only want to run your application and not 
necessarily do further .NET Framework or Gtk# development.

As a prerequisite, the machine that will have the Gtk# Runtime installer 
run on must have the Microsoft .NET Framework 1.1 Runtime already installed.

You can find the Microsoft .NET Framework Version 1.1 Redistributable 
Package at:

http://www.microsoft.com/downloads/details.aspx?FamilyId=262D25E3-F589-4842-8157-034D1E7CF3A3&displaylang=en

You can find the Gtk# Win32 Runtime installer at:

http://forge.novell.com/modules/xfcontent/downloads.php/gtks-inst4win/Win32%20Runtime%20Installer/v1.0.2.4/

Paco

Hi Pablexxx,

Try to do this: 

[Widget] Gtk.Combo cmbURL;
ArrayList list = new ArrayList( new string[] {"one",
"two", "three"} );

cmbURL.PopdownStrings = (string[])
list.ToArray(typeof(string));


If you need to add more elements:
list.Add("four");
cmbURL.PopdownStrings = (string[])
list.ToArray(typeof(string));

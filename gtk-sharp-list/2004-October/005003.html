Hi Camilo,

Can you give a more precise example?

Meanwhile you can use ArrayList to hold all you values and keep using
one parameters. 

	ArrayList list = new ArrayList ();

	list.Add ("an string");
	list.Add (5);
	list.Add (false);

	someObject.SetTheCallback (yourStaticMethod, list);

        // then you should be careful at yourStaticMethod reading your
	// parameters becasue order is important.

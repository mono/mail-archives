Hello,

> Anyone know where some gnome-print documentation is?  A tutorial,guide, or
> simple examples for gnome-print would be nice too.

An architectural overview of the early days of gnome print is here:

	http://www.levien.com/gnome/print-arch.html

Most concepts there were implemented, but higher level code was also
added: things to drive dialog boxes, previews, settings, printer
discovery in various layers.

The latest version of gnome-print is a fairly advanced printing system. 
You might want to ask Chema (the maintainer) for more info.

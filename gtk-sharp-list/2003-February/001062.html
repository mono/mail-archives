   http://sourceforge.net/project/showfiles.php?group_id=40240

don't, you can get it from:

    http://www.go-mono.com/download.html

But there the latest Windows release seems to be for version 0.17
of Mono.  (And not for 0.19.)

As far as Gtk+ and all the other stuff, you should be able to
get something by going to:

   http://wingtk.sf.net/

The only thing is, I'm not sure if the Windows Gtk# we have will
work with just any "release" of Gtk+ on Windows.

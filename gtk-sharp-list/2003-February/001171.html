Sorry, that won't work.  Try this instead...

                Application.Init();

                Gtk.Window win = new Gtk.Window("scrolling");
                win.ShowAll();

                Gdk.Window gdkwin = win.GdkWindow;

                Application.Run();

(You have to create the Window before you can show it :-)  )

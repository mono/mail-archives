On Fri, 2002-09-06 at 06:54, Beda Kosata wrote:

> I would like to know whether Gnome libraries will be available as part 
> of Gtk# on Windows (once it is finished).

Since the "native" libraries we are wrapping are not available on
windows, Gnome# won't be available on win32.

If all you want is the canvas, I think I recall something about a
degnomeified fork of the canvas, although I don't know if it has been
ported to Gtk2.  If someone gets a Canvas fork/port building on win32,
we can probably hack the build system for win32 to expose it.

found some code here: http://home2.highway.ne.jp/mutoh/tips/progressbar.html
translated

while (Gtk.events_pending)
   Gtk.main_iteration
end

to

while (GLib.MainContext.Pending()){
   Gtk.Main.Iteration();
}

and now it works properly.

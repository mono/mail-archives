> I have gtk# installed but I can't get it to work. Seems it wants some

I hope you know that you need the core Gtk runtime before you can get
any Gtk extension running. You can get it from here:
http://gtk-win.sourceforge.net

If it's the Gtk library that's not getting identified, try adding the
directory containing Gtk runtime (the DLLs, and also "bin" is there)
into your PATH environment variable. I was also puzzled the same way
earlier with PyGtk. Strange that no documentation tells you to add the
library-dir to your PATH, I admit.

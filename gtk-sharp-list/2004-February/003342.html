I had to use the CFLAGS env variable to add a search path. I think that if you 
run configure with an explicit --prefix, you should add -I$prefix/include to 
the cflags. I build all my mono components to install under /opt/mono, so the 
mono-specific headers it was looking for were in /opt/mono/include.

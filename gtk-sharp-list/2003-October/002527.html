Any 2.2.x package can be used, but the latest will have the fewest bugs,
generally.  :) 

If you don't have gtk+ installed, you are probably going to have a large
dependency chain to work through.  You might want to consider using an
installer like red-carpet or Ximian Desktop to do the heavy lifting for
you.  If you are adventurous though, you can get rpms for many distros
at ftp.ximian.com.

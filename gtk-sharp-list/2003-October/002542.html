Hi!

I made some changes to the GnomePrint binding:
        * gnome/PrintJob.custom: Added default ctor.
        * gnome/PrintDialog.custom: Added overloaded ctor with fewer
parameters.
        * sources/Gnome.metadata: Disabled default ctor for PrintJob and

ctor.

There is also some changes to api/gdk-api.xml, im not sure why they have
arrived, I use latest CVS.

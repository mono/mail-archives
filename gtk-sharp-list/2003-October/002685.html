Hello!

If you have problems with the theme, it is clear, that you have the gtk+
files (libgtk... And stuff) in a system directory.
The get loaded instead o the gtk+ files distributed with gtk# 0.11.

This is part of the problem, known as dll-hell. I have an idea on how
this can be worked around in a future release, but it's really a problem
of windows/your installation. Just delete the gtk+ in the system dir.

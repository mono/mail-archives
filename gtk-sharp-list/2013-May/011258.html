I did some Googling, and made it work with 2.10.

I needed to set the following in my .profile:

# mono
export MONO_FRAMEWORK="/Library/Frameworks/Mono.framework/Versions/Current/"
export
PKG_CONFIG_PATH='/Library/Frameworks/Mono.framework/Versions/Current/lib/pkgconfig/'
export
DYLD_FALLBACK_LIBRARY_PATH="$MONO_FRAMEWORK/lib:/usr/local/lib:/usr/lib"

I think that Mono might need some more obvious FAQs or a config README for
new users.

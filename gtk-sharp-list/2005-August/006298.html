Sideways rendering requires GTK+ 2.8, which you can bind to if you run
`bootstrap-for-the-insane` (which I guess means GTK+ 2.8 is
unsupported :-).

Plus, no shipping distros include GTK+ 2.8 at this time, so if you want
a large user base you shouldn't depend on it.  GTK+ 2.2 (Gtk# 1.0.x) is
the recommended dependency, or GTK+ 2.4 (Gtk# 2.3.x/2.4.x) if necessary,
as these are already present on more distros.

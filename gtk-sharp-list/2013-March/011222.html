NPlot also came top of my list, but the project seems to have died since 
it transferred over to svn and SourceForge.  I've also forked it for my 
own use, and extended the Gtk# implementation to handle Interactions 
(https://github.com/hwthomas/NPlot) to the same level as the Winforms 
demo.  However, I haven't (yet) added any more documentation than in the 
original NPlot, and there is currently no integration into the Gtk# 
designer either, both of which were requirements of the original query.  
My own preference for a cross-platform toolkit is now Xwt, which already 
has a very adequate Drawing API, and I'm currently porting NPlot to Xwt  
when time allows (also on github).

Hey there,

it is great to hear that there is finally some interest in releasing the work that was done on Gtk# master during the last 4 years! I was an active developer when 2.12 was released. Afterwards, I did some generator improvements(virtual methods) and other gtk-sharp work(getting rid of the glue code etc.).

I already thought that gtk-sharp 3 would never ever be released. The interest in the binding declined, development virtually ceased and then Mike Kestner left the Mono project after it was moved to Xamarin.

I hope that there is enough interest and manpower behind this attempt to finally release something. At least we have the Banshee guys here, who stayed committed to gtk-sharp and pushed the 3.0 bindings forward.

So please, invest some time, try to build an installer and finally release the bindings. It would be worth the effort. The generator is much improved (virtual methods, generates less glue code than before). I don't know about the state of the gtk-sharp bindings, but hope that this is also better than before. I also hope that the project can stay alive, even without a regular committer.

I wish you all much success for your gtk-sharp release efforts!

Kind regards,

 -- Christian Hoff

Hi Bertrand,

As an application [1] developer I can inform you that after having 
waited over a year for any (developer) release, I eventually decided to 
pull the code form GitHub [2] and build it myself so I could start 
porting. Early this year I published a working GTK3 branch [3] for Haguichi.

What I would like to see most is a release that will be available in the 
repositories of the next major distros (i.e. Ubuntu 12.10). Then not 
only us application developers can test the bindings, but we can also 
publish GTK3 versions of our applications that end users can actually 
test without any hassle (like pulling and building GTK# 3.0 manually).

Regards,
Stephen

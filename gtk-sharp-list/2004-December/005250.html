Announcing release 1.9.1 of Gtk#.  This is an unstable development
release for the GNOME 2.6 bindings.  It is parallel installable with the
latest stable release, 1.0.4.

Tarball and RPMs for FC2 and NLD available for immediate download from:

http://sourceforge.net/project/showfiles.php?group_id=40240&package_id=135197

About Gtk#: 
Gtk# is a .Net language binding for the Gtk+ Graphical User Interface
toolkit and several GNOME platform libraries. The GNOME 2.6 platform is
currently targeted.

Changes since 1.9.1:
* NodeSelection and NodeView added: (Duncan Mak)
* Container child property restructuring: (Dan Winship)
* fixed win32 crippling gluelibname bug: (MK)
* refining of GnomeVfs api: (Jeroen Zwartepoorte)
* parser and struct support for public/private fields: (Dan Winship)
* support for ref/out params in signal handlers: (Dan Winship)
* Additional API customizations and Corrections: (Jorn Baayen, Jorge
Garcia, John Luke, Duncan Mak, Dan Winship, Jeroen Zwartepoorte, MK)
* an absurd amount of Bugfixes (Todd Berman, Tambet Ingo, Ben Maurer,
Dan Winship, MK)
* documentation additions (Shane Landrum, John Luke, Dan Winship, Jeroen
Zwartepoorte, MK)

Discussion of Gtk# occurs on the mailing list at
gtk-sharp-list@lists.ximian.com.  Bugs can be reported to
http://bugzilla.ximian.com, module gtk#. Check the Gtk# homepage for
additional info at http://gtk-sharp.sf.net.

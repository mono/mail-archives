Release 1.0 thru 1.0.4 did not have Mono.Data.dll

For some strange reason, it was removed from the build before 1.0.
I added it back after 1.0.4.  So, it should be in the latest stable and
development release.

Or, you can download the source to mcs and build it.  It is at
mcs/class/Mono.Data

The release binaries tarball for Mono SQL Query includes Mono.Data.dll for
this reason.

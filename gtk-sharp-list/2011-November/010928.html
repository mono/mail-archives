Gtk is a flow layout by default similar to HTML and Monodevelop's gui 
editor will have defaults for the AutoSize/Fill/Expand properties. I 
make heavy use of the Alignment containers and remove the Fill/Expand 
properties to attempt to force a size. Remember in a flow layout that 
the all of a Widgets parents have an effect on the appearance of a 
Widget play around but I recommend starting by putting the control you 
want to size in a Alignment container and unchecking the Fill/Expand 
properties.

As a fall back position if you really like the way WinForm's layout 
works use a Fixed Container.

I hope this helps

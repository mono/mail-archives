Hey there,

there is already an open bug report which describes this behaviour. See here: https://bugzilla.novell.com/show_bug.cgi?id=675421 . So far nobody has investigated this problem.

In general, gtk-sharp is a more or less unmaintained project by now. We a far from a stable 3.0 realease and the code in the 2.12 branch is starting to get bitrotten(especially the installer part). So you may want to consider switching to another GUI toolkit or support Xamarin and the gtk-sharp project by donating some money.

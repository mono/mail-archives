> Moritz

Ah, ok. Well, I'm not sure about the API of the first two, but you're
right, they ought to be available.

What I am more concerned about is the third need. I am pretty sure there
is some sort of .NET pattern that should be used for the
other-filesystem accesses, but I have not researched it. I can see how
one could use the Stream base class, perhaps, but not sure what to do
about directories and other things. I have cc:ed this message to
mono-list to see if anyone there has an idea of some standard
functionality the wrapper should try to conform to.

-- Rachel

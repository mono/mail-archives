GtkGL# 0.02
===========
Release 0.02 of GtkGL# is now available. This release provides a fully
managed .NET assembly and removes the previous dependency with the
native GtkGLArea widget.

About GtkGL#
============
GtkGL# is a managed .NET assembly for using the OpenGL graphics library
together with the Gtk# windowing toolkit. GtkGL# is free software, and
is licensed under a 3-clause BSD license.

GtkGL# has been tested to work on Linux. I don't have access to Windows,
so you should not expect it to work on Windows without modifications.

Download
========
You can download this release of GtkGL# from Simetron's project on 
Novell's Forge:

http://forge.novell.com/modules/xfmod/project/showfiles.php?group_id=1309&release_id=1034#selected

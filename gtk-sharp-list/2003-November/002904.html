This is a small preview of the Cairo integration patches.  We are going
to make some changes to the Cairo naming conventions, so this patch
should not be applied to CVS.

But with this, you can use the CairoSample.cs program that implements a
widget using Cairo, and no Gdk.

After we are done with the naming changes, we should get some docs for
Cairo on CVS as well.

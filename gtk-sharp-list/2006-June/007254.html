At the top of the documentation for method WaitForImage it says something like:

".. supported only in version 2.6 and above"

I think you are referencing the 2.4 or older libraries of GTK#. If you
have both installed (as I have) you need to explicitly reference the
newer ones. If you have just one set of libraries you'll need to
upgrade them. See that it means that you'll also need GTK+ 2.6/Gnome
2.10 or newer installed,  for things to work.

Hope it helps,

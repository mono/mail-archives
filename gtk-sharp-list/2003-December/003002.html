On December 14, 2003 05:55 pm, Greg Hamilton wrote:
> I'd definitely be interested in an installer.
>
> Anybody care to share their impressions of gtk in Windows?

No problems at all here.  I'm developing a "Visual Speech Interface" (for 
lack of a better term, basically to allow those using alternative input 
methods like eye trackers to synthesize speech) that runs great on windows 
using GTK#.  I developed the entire thing in Linux and never even tried it 
on windows until I had about 4000 lines of code and it worked without a 
hitch.  I think I installed the 0.8 version of GTK# (hey, it works) and 
thats all I needed.  I run using Microsofts .NET environment.  The only 
downside is that you need the GTK dlls in the same directory as your 
executable (at least I haven't gotten it to work otherwise).
Of course recent installers would be great.  But I'm just happy with having 
something that installs without problems.  Something I definetly can't say 
about the nightmare of getting it working from source on Linux.  (please 
please someone make debian packages :))

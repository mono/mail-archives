When I try to compile this code:

using Gdk;

{
  unsafe temp()
  {
    Pixbuf p = new Pixbuf (Colorspace.Rgb, false, 24, 100, 100);
    byte *pixels = p.Pixels;
  }

for both get_Pixels() and the native gdk_pixbuf_get_pixels() is an int8.

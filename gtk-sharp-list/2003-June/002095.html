Hi,

Bored by not finding an uptodate Gtk# installer for Windows I've created 
this collection of scripts. To build a Windows installer from them 
you'll need:

	- Gtk+ Runtime and Development Environment
	  (http://www.dropline.net/gtk/)
	- Cygwin (http://www.cygwin.com/)
	- NSIS (http://nsis.sf.net/)
	- gtk-sharp-0.10.tar.gz

Once you've got them you copy gtk-sharp-0.10.tar.gz into the 
"GtkSharpBuilder" directory and run "build.sh". This checks for presence 
of Gtk+ Runtime and Development Environment and builds Gtk#. During the 
build process two test programs are build to check correctness of your 
setup. Notice: The building process of "gtksharpglue.dll" appears to be 
broken since it introduces dependencies on "cygwin1.dll". Therefore 
"build.sh" uses it's own build procedure for "gtksharpglue.dll".

Well, once "build.sh" has finished it's work you feed NSIS with 
"gtksharp.nsi". This should produce the file "gtk-sharp-0.10-win32.exe". 
This installer checks for presence of the Gtk+ Runtime Environment and 
installs Gtk# into "%COMMONFILES%\Gtk#\1.0". It also writes to the 
registry "HKLM\Software\Gtk#\1.0" to allow your programs to locate the 
Gtk# Runtime and Development Environment programmatically.

A prebuilt version of the installer can be found at
http://taschenorakel.de/mathias/archive/gtk-sharp/gtk-sharp-0.10-win32.exe.

Would be nice of the scripts attached could be merged into Gtk#'s CVS to 
  allow anyone to build a Gtk# builder in future.

Ciao,
Mathias (aka tbf)

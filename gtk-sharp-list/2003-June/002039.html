On Sat, 2003-06-14 at 12:29, Lee Mallabone wrote:
> This is a bit of a guess having looked at the C docs, but... You might
> want to try messing with the CellRendererToggle.Activatable property?

Close, but not quite. The Activatable property sets whether the cell
toggle will respond to clicks. If it is set to true, it will respond and
emit the toggled signal, etc. If it is set to false, it will be "grayed
out" and will not emit toggled signals.

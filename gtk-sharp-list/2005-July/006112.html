Hi

You can do this to loop through a ListStore:

TreeIter it = new TreeIter ();
store.GetIterFirst (out it);
while (store.IterIsValid (it))
{
	string text = (string) store.GetValue (it, 0);
	Console.WriteLine (text);
			
	store.IterNext (ref it);
}


/Christian

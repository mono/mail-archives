I've been working on a tao autotools build system.  It currently tracks
subversion.  It doesn't currently build the examples or Rob's new PhysFs
tools, but the rest seems to work fine.  It produces .pc files, has a
'make install' target and all the good stuff you'd expect from a modern
autotools build system.  If anyone's interested in using such a beast,
please feel free to read and contribute to the wiki page:

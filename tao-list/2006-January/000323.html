Hi Brad,

Anyone answered you to this yet? The problem is not with Tao at all.
What you need to do is to add an entry like this:

	<dllmap dll="opengl32.dll" target="libGl.so"/>

in the mono configuration in /etc/mono/config. It needs to be inside
the configuration xml element, so minimally you config file should look
like this:

<configuration>
	<dllmap dll="opengl32.dll" target="libGl.so"/>
</configuration>

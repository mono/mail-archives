David,

You should first learn how the ARB_multisample extension works:

http://oss.sgi.com/projects/ogl-sample/registry/ARB/multisample.txt

In order to use multisampling you have to choose a pixel format that supports it. For that to work you first have to check for this extension:

http://oss.sgi.com/projects/ogl-sample/registry/ARB/wgl_extensions_string.txt

Use it to determine if this extension is supported.

http://oss.sgi.com/projects/ogl-sample/registry/ARB/wgl_pixel_format.txt

And then use that extension to choose a pixel format that supports antialiasing.

And once you have done that, you will be able have to enable antialiasing with the multisample extension.

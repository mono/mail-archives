I'm planning on writing up a port of Plane2D to C# for Tao.Ode. Plane2D (
http://www.mathematik.uni-bielefeld.de/~pserocka/ODE-Plane2D/ ) is an
extension to ODE that allows for 2D physics applications using ODE.

There is no Plane2D.dll as you actually compile Plane2D with ODE or the
application your working with. This means that Plane2D for C# would be an
actual port, not P/Invoke references like the rest of the Tao framework. I
was wondering if this should then become Tao.Ode.Plane2D or something
completely different not even associated with the Tao Framework?

In any case, I'll work on it in the next few days and inform you of my
progress.

If you're seeing NotImplementedException(), then you did not properly
build Tao.  You can't just compile Gl.cs; the resulting assembly has
to be post-processed.  Try building using the makefiles (install gnu
make from somewhere), or worst case, use the build .bat files on
windows.

     - Vlad

> > > Axiom is I guess still using an older version of the OpenGL bindings.
> > > You have two options; if your OpenGL library exports glTexImage3D (as
> > > almost all do, including Microsoft's I believe), just call it
> > > normally:
> > >
> > > Gl.glTexImage3D(Gl.GL_TEXTURE_3D_EXT, 0, ....);
> > >
> > > If you need to check for the extension explicitly:
> > >
> > > if (!GlExtensionLoader.LoadExtension("GL_EXT_texture3D")) {
> > >   Console.WriteLine("GL_EXT_texture3D not supported!");
> > > }
> > > Gl.glTexImage3DEXT(Gl.GL_TEXTURE_3D_EXT, 0, ...);
> > >
> > > (note the call to glTexImage3DEXT in the explicit extension load case.
> > >  Again, you shouldn't need to use this case.)

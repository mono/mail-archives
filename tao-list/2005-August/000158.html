hi,

if u really want to see the Tao logo in design mode in your forms change 
this code
        private void InitializeBackground() {
            Assembly assembly = Assembly.GetExecutingAssembly();
            using(Stream imageStream = 
assembly.GetManifestResourceStream("TaoButton.jpg")) {
                this.BackgroundImage = (Image) 
Bitmap.FromStream(imageStream);
            }
        }

to

private void InitializeBackground() {
	this.BackgroundImage = Tao.Platform.Windows.Properties.Resources.TaoButton;
        }

it did the trick for me, not to mention that I do not get the error message 
anymore when the form is initialized.

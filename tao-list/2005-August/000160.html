Hi all,

I have completed crteating a Windows installer script using NSIS. It can
easily create a Windows installer for Tao. I also added a 'docs' target to
the make file so that NDOC documentation can be created. The installer
copies the binaries and source into a Tao directory. It creates Shortcuts
for all of the examples, installs Win32 dependencies (SDL.dll, freglut.dll
etc) into the system dir, installs the assemblies into the GAC and also
sets VS 2003 so that Tao shows up in the references list. So when should we
do a release? Also, what should the version number be. I set it at 'Tao
1.2' for the time being.

Hi, 

I was looking at Tao and have seen that you use the functions and
constant names 1:1 from the C Libraries. This seems a bit odd to me
because in C most libraries prepend a prefix because there are no
namespaces in C (SDL prepends SDL_, OpenGL gl and GL_ for example). Now
in C# this looks odd as we have 2 "namespaces" that basically mean the
same. For Example:

	Sdl.SDL_Init(Sdl.SDL_INIT_EVERYTHING);

This would look more natural if the prefix is stripped:

	Sdl.Init(Sdl.INIT_EVERYTHING);

Are there any reasons why you don't do this?

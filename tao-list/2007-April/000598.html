You're trying to get GTK going on Mac?  You probably need to either
install the X server and configure GTK against that, or you need to try
to build the native Mac version, which I think is still a work in
progress.  The X server option is probably easier and more stable, but
if you want to read about the native version check out:

http://developer.imendio.com/projects/gtk-macosx

After a messing with the method signatures for a while I found that 
glShaderSource (int shader, int count, string[] arg_string, int[] 
length) works.  The existing signatures were trying to send an array of 
string arrays or a reference to a string array, so GL was probably 
looking in the wrong place to read the string data hence getting a 
reference exception.

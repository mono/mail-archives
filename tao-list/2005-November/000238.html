Ok, from what people are saying it sounds like there is still a need
to maintain the .NET 1.1 version; with 2.0 just recently having been
released, I'm not surprised.  However, I will probably begin work on
the 2.0 version of the Gl bindings at least, separately from the
current code.  Hopefully that work will result in a set of 2.0 OpenGL
bindings that work significantly better than the current 1.1 ones, and
can be used to spur people on towards adoption of 2.0 ;)  I believe
that the rest of Tao will only be able to take advantage of 2.0
features in a few places, so it shouldn't be affected.

(There is much in the current OpenGL bindings that doesn't quite work
right, mainly due to marshalling issues.  If people run into those,
some can be fixed -- just let me know and I'll see what I can do.)

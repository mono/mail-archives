Each library serves a different purpose.  For example, one of them is for
sound and another is for physics.  You only need OpenGL.  In fact, if you
are only doing simple 2D graphics, then OpenGL is probably overkill, and it
would be easier just to use the functionality already built into Mono.  To
use OpenGL, you'd have to read pretty much the entire redbook.

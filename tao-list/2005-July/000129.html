Before dealing with glfw/sdl/etc., copy the dll's into the examples
directory, and try one of the NeHe examples -- NeHe.Lesson03.exe for
example.  The NeHe examples use the Tao.Platform.Windows stuff, and
don't depend on an external library (they're also not cross platform
because of it, but good for testing).

If the NeHe examples work, then the Tao build is (90% most likely)
fine; you'll need to look at how you're building Glfw.  In particular,
you don't want a glfw that uses cygwin -- I'm not sure what that will
do the windows runtime stuff.  You need a fully native win32 glfw
build (or glut, etc.).

    - Vladimir

Hello,<br><br>I've completed Tao bindings to PhysicsFS ( <a href="http://icculus.org/physfs/">http://icculus.org/physfs/</a> ).<br><br>For those of you who haven't heard of it, PhysFS provides an abstract interface for various archive formats roughly based off of Quake's pak file system.&nbsp; You define a search path and then can read directories and archives in a transparent hierarchical filesystem.&nbsp; This makes for easy
       access to ZIP files in the same way as you access a file directly on
       the disk, and it makes it easy to ship a new archive that will override
       a previous archive on a per-file basis.&nbsp; For an example of its use, visit the PhysFS website ( <a href="http://icculus.org/physfs/">http://icculus.org/physfs/</a> ).<br><br>I decided to write it as it came up as a constant request from various members on 
GameDev.net.&nbsp; I also had personal interest in having the C# interface for use in projects with Tao.<br><br>All documentation has been added in from the official docs and all functionality is provided.&nbsp; I translated the simple tutorial from the website to C# and 
Tao.PhysFs and it worked perfectly.&nbsp; All required files are complete (ndoc, dll.config, Makefile, snk, etc) and I'd love to add them all to Tao, but don't have write access to the repository.&nbsp; Could you give me write access to the SVN so I could add in

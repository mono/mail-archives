El Miércoles, 25 de Mayo de 2005 08:55, Vladimir Vukicevic escribió:
> I just committed to svn "Phase 1" of the rework.  Everything's moved
> around, and the build is makefile-based... do "make net-1.1" or "make
> mono-1.1" first, and then do "make" to build everything.  You'll end
> up with stuff in the "dist" dir.  If you want to strongname the
> assemblies, "make STRONG=1".  (Come to think of it, that really should
> be the default unless you explicitly request otherwise.. I can't think
> of any reason why you'd want to not build with it.)
>
> I was able to do a full build (including Tao.Platform.Windows) on
> linux, and move the assemblies over to windows and have the dozen or
> so examples I tried still run (some Redbook, some NeHe, some Glfw,
> some NateRobins), so that's encouraging.  Tao.OpenGl is now
> Tao.OpenGl.ExtensionLoader, Tao.OpenGl, and tao.OpenGl.Glu as three
> separate assemblies.
>
> I've also updated Tao.OpenGl with the latest spec bits from Ignacio.
> I have not yet changed the generator to output names without a gl
> prefix; will get to that tomorrow.  Also still need to create Visual
> Studio files or similar for non-makefile builds.
>
> If anyone has a chance to try things out, let me know how it goes.
>

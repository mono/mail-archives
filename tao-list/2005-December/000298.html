> Hey all,
>
> I've been spending some free time over the last month putting together
> an autotooled tao package.  I am currently creating this distribution
> in parallel with Vlad's official release.  You can get packages from
> my web site here:
>
> http://colliertech.com/downloads/tao/
>
> I am using this autotooled version in order to get gtkglarea-sharp
> into the garnome build system.  Garnome requires an autotools build
> system in order to integrate correctly.  And gtkglarea-sharp requires
> tao in order to provide any useful functionality.  So I created this
> package.
>
> Some other info about the new package:
> 1) It hasn't been tested on windows, and it will not work there yet.
>   1a) I will work on this as soon as I have some time.  I have a  
> windows
> machine to test it on.
> 2) it is much smaller, as it doesn't include the examples or their
> associated test data
> 3) it provides .pc files.  You can do mcs -pkg:tao-opengl in order to
> add all references required for Tao.OpenGl
> 4) make install will place the .dll files (and their  
> associated .config
> files) into $prefix/lib/mono/tao/
> 5) make install will also place the assemblies into the GAC
> 6) The examples have not been autotooled yet.  And based on the  
> size of
> the archive with the examples in it (>90M), I may not decide to
> auto-tool-ize them.  They can always be gotten from svn, though.
>
> I've added mention of it to the Mono wiki.
>
> Cheers,
>

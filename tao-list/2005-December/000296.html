You can simply put the control in a form, then use the overrideable
OnPaint method to draw the scene... every time you need to re-render
the scene, you can Invalidate the control... anyway, to render
animated scenes, Invalidate function should be called by a main
loop... I think that is better to avoid the SimpleOpenGLControl if you
are going to render animated scenes... many NeHe examples in the
Examples folder of Tao distributions can be used to learn how to
acquire the needed device context ^^

Oh, before doing anything else, you should initialize the control
(there is a specific function, InitializeContexts, that should be
called, and then you should call DestroyContexts when you have done
with the OpenGl context... often in the OnClose method of the form)...
hope this helps... :P

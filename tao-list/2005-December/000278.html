After spending much time debugging, I've whittled some problematic
code down to this:

public void CheckLight0()
{
    bool boolVal;
    int intVal;
    bool[] boolArray = new bool[1];
    int[] intArray = new int[1];

    Gl.glGetBooleanv(Gl.GL_LIGHT0, out boolVal);
    Console.WriteLine("out boolVal: " + boolVal);

    Gl.glGetBooleanv(Gl.GL_LIGHT0, out intVal);
    Console.WriteLine("out intVal: " + intVal);

    Gl.glGetBooleanv(Gl.GL_LIGHT0, boolArray);
    Console.WriteLine("boolArray: " + boolArray[0]);

    Gl.glGetBooleanv(Gl.GL_LIGHT0, intArray);
    Console.WriteLine("intArray: " + intArray[0]);
}

Elsewhere, I call this method, once with the light disabled, and once enabled:
Console.WriteLine("Disabled");
Gl.glDisable(Gl.GL_LIGHT0);
CheckLight0();

Console.WriteLine("Enabled");
Gl.glEnable(Gl.GL_LIGHT0);
CheckLight0();


The output is this:
Disabled
out boolVal: True
out intVal: 0
boolArray: False
intArray: 0
Enabled
out boolVal: True
out intVal: 1
boolArray: True
intArray: 1

It seems that using the out bool version of glGetBooleanv always
returns true, while all other overloads seem to work correctly.

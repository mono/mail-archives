Hello,

I've completed Tao bindings to PhysicsFS ( http://icculus.org/physfs/ ).

For those of you who haven't heard of it, PhysFS provides an abstract
interface for various archive formats roughly based off of Quake's pak file
system.  You define a search path and then can read directories and archives
in a transparent hierarchical filesystem.  This makes for easy access to ZIP
files in the same way as you access a file directly on the disk, and it
makes it easy to ship a new archive that will override a previous archive on
a per-file basis.  For an example of its use, visit the PhysFS website (
http://icculus.org/physfs/ ).

I decided to write it as it came up as a constant request from various
members on GameDev.net.  I also had personal interest in having the C#
interface for use in projects with Tao.

All documentation has been added in from the official docs and all
functionality is provided.  I translated the simple tutorial from the
website to C# and Tao.PhysFs and it worked perfectly.  All required files
are complete (ndoc, dll.config, Makefile, snk, etc) and I'd love to add them
all to Tao, but don't have write access to the repository.  Could you give
me write access to the SVN so I could add in Tao.PhysFs?  If you wish to
review Tao.PhysFs before I add it into the repository, I'd be glad to upload
it to my website before hand.  I've also uploaded PhysFs.cs temporarily
here:  http://files.robloach.net/PhysFs.cs

If you need any additional information, please feel free to contact me.

Thanks again,

Just an FYI.

In the past few weeks, I've noticed some interesting activity happening 
over at the ODE project:  moving from CVS to SVN on sourceforge (this 
seems to be a popular activity), merging the unstable development branch 
into the main branch (bringing in all the unofficial improvements that 
have accumulated there), and talk of a new ODE release. 

The current incarnation of Tao.Ode is based on the latest stable ODE 
release, 0.5, which is quite old, and not recommended or used any longer 
(apparently) by the ODE development community, as far as I can tell.  
The current Tao.Ode code in SVN seems to be functional and will be part 
of the upcoming Tao release (*ahem*), so I'm inclined to turn away from 
ODE 0.5 and start tracking the main ODE svn branch for future Tao.Ode 
development.

I'll set up a development branch for the time being, but I'm likely to 
drop ODE 0.5 support and merge the development branch into the trunk as 
soon as ODE makes a release of some kind so people can take advantage of 
the ODE improvements.

I'll post updates as needed.

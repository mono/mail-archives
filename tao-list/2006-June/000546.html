It's a little difficult for a newbie to get started.  I know, because I went
through this a few months ago.  There are lots of examples in the download,
but it takes quite a bit of time to find them.  Don't give up.  Keep looking
through every folder for the source code examples.  The SimpleOpenGLControl
is available once you make a reference to Tao.Platform.Windows.dll.  I
inherited from it.  Don't forget to InitializeContexts();  And this is all
assuming that you've read the Red Book.

Compile david hudson's program with this command line:

(on one line)
mcs demo.cs -r:Tao.Sdl.dll -r:Tao.OpenGl.dll -r:Tao.OpenGl.Glu.dll
-r:Tao.OpenGl.ExtensionLoader.dll && macpack -m:2 -o:.  -r:/Library/ 
Frameworks/Mono.framework/Versions/Current/lib/li
bCocoaSharpGlue.dylib -r:Tao.Sdl.dll -r:Tao.Sdl.dll.config - 
r:Tao.OpenGl.dll -r:Tao.OpenGl.
dll.config -r:Tao.OpenGl.Glu.dll -r:Tao.OpenGl.Glu.dll.config - 
r:Tao.OpenGl.ExtensionLoader
.dll -r:Tao.OpenGl.ExtensionLoader.dll.config  -n:demo -a:demo.exe

And Console.app reports mouse motion.

I have a makefile which does this and sets the icon and packages the  
application into a dmg, if anyone is interested.

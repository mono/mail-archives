Hi!

I'm currently trying to port Opal (Open Physics Abstraction Layer, 
http://opal.sourceforge.net/) to .NET. Of course the port uses Tao.Ode 
to interface with the ODE library. This project has nothing to do with 
the proposed Opal.NET that was to be developed by some of the RealmForge 
guys.

Today I reached a state where the "simple" application that can be found 
in the samples folder of the original Opal library is fully ported to C# 
and runs on my .NET port of Opal. If anyone wants to take a closer look 
at that, go here:
http://cygon.nuclex.org/2006/06/15/porting-opal-to-net-almost-finished/

The source code for my port can be downloaded found at 
https://devel.nuclex.org/framework/svn/, thought I still need to clean 
it up, place copyright notices and acknowledge the Opal authors in there.

-Markus-

The problem with using Tao for a smart device application is that
you'd need to build the dependant libraries for that platform.  Most
mobile devices don't have graphics hardware either, so rendering would
become extremely slow.

Fortunately, modern machines are slowly getting the hardware.  You'll
see that although you can develop Managed DirectX applications for a
smart device, but you probably can't run it through either the
emulator or the device itself.

I guess we'll have to either wait, or build the dependant libraries
for the platforms ourselves.....

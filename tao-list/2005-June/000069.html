> Even if I can compile everything else, I couldn't be able to compile
> Tao.FreeGlut, with provided scripts, or by Visual studio solutions... 
> in the last case the error is that DelegateCallingConventionCdecl
> is not defined, any idea? (I am problably missing something :P)

In build.bat, change the line:

csc /nologo /target:library /D:WIN32 /unsafe
/out:..\..\dist\bin\Tao.FreeGlut.dll AssemblyInfo.cs Glut.cs
/lib:..\..\dist\bin /r:Tao.OpenGl.dll

by:

csc /nologo /target:library /D:WIN32 /unsafe
/out:..\..\dist\bin\Tao.FreeGlut.dll AssemblyInfo.cs Glut.cs
DelegateCallingConventionCdeclAttribute.cs /lib:..\..\dist\bin
/r:Tao.OpenGl.dll

That worked for me...

Also, make sure you have built and installed freeglut.dll:
http://freeglut.sourceforge.net/

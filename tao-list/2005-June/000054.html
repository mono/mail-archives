>I made some changes to the makefiles of almost all of the examples. Many of
>them were not copying their respective data files into the dist/examples
>directory after a make. Also, many had the /target:exe flag when they
>should have had /target:winexe.
>
>All of the examples using Tao.Freeglut we not working on my Windows
>machine. I compiled a debug version of freeglut.dll C lib and it reported
>the delegate problem. I modified Tao.Freeglut to use the Delegate Attribute
>used in Tao.Sdl and ran Tao.PostProcess against Tao.FreeGlut. The examples
>now work correctly.
>
>All the changes have been checked into svn.
>
>Nolan, thanks for the OSX patch. I'll test it out to make sure it does not
>affect Tao.Sdl on Windows and then I will commit it to svn.

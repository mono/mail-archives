How did you install Glade ?

If you installed Glade into an optional prefix then you'll at
least need to add the new directory to your ld cache using ldconfig.

or start Glade using the LD_LIBRARY_PATH variable.

Also note you cannot run Glade if you have not installed Glade
anywhere (you must make install, whether its the system prefix
or an optional one).

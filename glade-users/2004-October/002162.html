Hi,

I have a little app with a main window and 3-4 dialog/option-windows.
I use glade like this (an example):

main()
{
	make_main_window();
	make_option_window1();
	make_option_window2();
	make_option_window3();
	make_option_window4();
}

make_option_window1()
{
	GladeXML *xml_glade;
	GtkWidget *optwin1;
	..............

	xml_glade = glade_xml_new (GLADE_PATH"optwin1.glade", NULL, NULL);
	
	optwin1 = glade_xml_get_widget (xml_glade, "option_window_1");

	/* eventually load other widgets */

	gtk_widget_hide (optwin1);
	g_free (xml_glade);
}

App loads at start all glade files one time, and it doesn't destroy or remake option-windows, just hide and show them for every menu option called.
Is it a good use of libglade or not?

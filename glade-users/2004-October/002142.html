Application
===========

Glade 2.0.2


Description
===========

Glade is a User Interface Builder for GTK+ and GNOME.

Glade 2.0.x is for older versions of GTK+ (2.0.x/2.2.x) and GNOME
(2.0.x/2.2.x/2.4.x), and will not compile or run with the latest
versions of GTK+ (2.4.x) or GNOME (2.8.x).


Enhancements
============

Glade 2.0.2     (Oct 12 2004)
===========
 
 o Don't compile or run with GTK+ 2.4, since it isn't compatable.
 o Update docs.
 o Bug fixes.

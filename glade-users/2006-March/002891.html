I'm pretty sure this is for the old gtk+-1.x branch.

For new development, you should *really* use Gtk2-Perl and 
Gtk2-GladeXML, from:
http://gtk2-perl.sourceforge.net/

You will not 'see' anything in Glade about Perl. You simple save the 
glade project, and use Gtk2::GladeXML to create the GUI based on the 
Glade files.

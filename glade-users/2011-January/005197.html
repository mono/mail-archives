I suggest you to install/use the libpng using apt instead of using
source (libpng may be required by the other system library, though you
are not directly using them). (I am not sure libpng3 will work well with
glade).

#apt-get install libpng12-0


As far I know glade use icon theme from this 2 packages.

So install this packages:
#apt-get install gnome-icon-theme hicolor-icon-theme

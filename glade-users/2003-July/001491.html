yep, I got the same results.  Apparently these messages appear because you have set in your project that you want to use Gettext.  Gettext is a program to support multi-language software, so unless you really want that, I would recomend you to just uncheck the option (which is in the C Options folder) and rebuild the source code... That should do it. (it worked for me).

If you do want Gettext, I've read you simply need to install the package and it should work. (I didn't stick to this one long..)

BTW, sometimes there is a problem with versions when regenerating the code. What i usually do is delete the whole project directory except for the .glade file and the src directory and have glade generate the whole thing again.  The po folder, for example only exists to support Gettext.

the rest should be straight-forward.

Günther.

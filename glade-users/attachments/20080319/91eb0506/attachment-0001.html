<DIV>the tree view is not a toplevel widget ,so you have to pack it into a container to show it.In the callback of the button ,you create a new window ,and tree view,while you do not pack the tree view into the windows.</DIV>
<DIV>if you want the callback function of button to manipulate the treeview in the main window which the button is pack into,you can use the lookup_widget()function or g_object_set()and g_object_get() function,to obtain the pointer of the tree view in the main windows.</DIV>
<DIV>&nbsp;Generally there are 4 steps to use the treeview widget.</DIV>
<DIV>&nbsp;1.add data to Gtkliststore or Gtktreestore.</DIV>
<DIV>&nbsp;2.pack the GtktreeviewColumn into&nbsp; GTKtreeview</DIV>
<DIV>&nbsp;3.pack the Gtkcellrenderer into a GtktreeviewColumn</DIV>
<DIV>&nbsp;3.bind the data in Gtklist(tree)store with Gtkcellrenderer.</DIV>

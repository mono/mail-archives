<br>

<br>
You can also get Glade to load and introspect your window&#39;s properties<br>
and automatically put them in the editor; which is a matter of telling<br>
the same xml catalog where to load the library with your widget inside.<br>
<br>

<br>
Note that after the type is encoded in the glade file then GtkBuilder should<br>
be able to dynamically load your object at run time simply by name.<br>

As far as I can see, the Gtk-Critical fault says that you can not put the widget onto another parent because it already has a parent. The second parameter in the glade_xml_new() is for the root widget node. Try this:<br><br>
page1xml=glade_xml_new(&quot;./page1.glade&quot;,&quot;mainvbox&quot;,NULL);<br><br>and then<br><br>rightpanevbox= glade_xml_get_widget(page1xml,&quot;mainvbox&quot;);<br><br>Hope it works,<br>good luck.<br><br>LUK (gromot)<br>

I was able to solve my problem after reading<a
 href="http://tadeboro.blogspot.com/2009/04/as-promised-im-back-with-second-part-of.html">
GtkComboBox widget - part 2 </a>by Tadej Borovšak.  I changed my code
to do the following (simplified by removing the enum):<br>
<br>
    data-&gt;country_store = gtk_list_store_new (2, G_TYPE_STRING,
G_TYPE_STRING);<br>
<br>
    gtk_list_store_append( data-&gt;country_store, &amp;iter );<br>
    gtk_list_store_set( data-&gt;country_store, &amp;iter, 0,
"America", 1, "USA", -1 );<br>
    gtk_list_store_append( data-&gt;country_store, &amp;iter );<br>
    gtk_list_store_set( data-&gt;country_store, &amp;iter, 0, "Canada",
1, "CAN", -1 );<br>
    gtk_combo_box_set_model ( data-&gt;country, GTK_TREE_MODEL
(data-&gt;country_store));<br>

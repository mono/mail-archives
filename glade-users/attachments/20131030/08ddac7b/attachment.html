    hi Virgil , <br>
    windows is not your friend in this case :)<br>
    It is not a Glade matter but a webkit problem. You have to embed a
    webkit viewer inside a window or frame made with glade. <br>
    There is no Webkit ( better webkitgtk or pywebkitgtk) ufficial
    package at this time for win32. I have a solution build by myself
    for my app but it is for python 2.6. <br>
    For python 2.7 and above I suggest you to use the AIO from <a
      class="moz-txt-link-freetext"
      href="http://opensourcepack.blogspot.it/">http://opensourcepack.blogspot.it/</a>
    where you can find also the new  pygi + gtk3 solution. <br>
    I suggest also to use import webbroser to open firefox and point it
    to the local html help file. Less trouble. ;) <br>

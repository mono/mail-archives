      More specifically I have Python 2.7.5 installed on a Windows Vista

      information for the user of the GUI) will be displayed. It is
      important that this HTML file is displayed in a separate form (or
      window) and <b>does not need an online connection</b> to a web
      browser (e.g. FireFox). This displayed, read only HTML, will
      contain local links, embedded images and videos, etc. It should be
      treated as if it were being displayed by a web browser (e.g. the
      user should be able to move to different topics, scroll, view a
      video, etc.), but a browser is not required.<br>

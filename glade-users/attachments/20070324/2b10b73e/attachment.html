<DIV dir=ltr align=left><SPAN class=640144813-24032007><FONT face=Arial 
color=#0000ff size=2>1. Create all four images in your window using glade. Then 
set the last three images to be hidden. Then, connect a signal to the arrow key 
that hides the current image and unhides the appropriate image. You'll have to 
keep track of which image should be shown.</FONT></SPAN></DIV>
<DIV dir=ltr align=left><SPAN class=640144813-24032007><FONT face=Arial 
color=#0000ff size=2></FONT></SPAN>&nbsp;</DIV>
<DIV dir=ltr align=left><SPAN class=640144813-24032007><FONT face=Arial 
color=#0000ff size=2>2. Don't use glade to create any of the images and simply 
write the code to load the images from file and then display the appropriate 
one.</FONT></SPAN></DIV>
<DIV dir=ltr align=left><SPAN class=640144813-24032007><FONT face=Arial

color=#0000ff size=2>Also note that glade generates a couple of helpful image 
creation functions that you can use in your program to create a GtkImage object 
from an image file. Check out the support.c file that is generated by 
glade.</FONT></SPAN></DIV>

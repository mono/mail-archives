</div>You cannot control these things from Glade directly. Glade can only<br>
modify properties of objects that GtkBuilder will create.<br>
GtkSpinButton is created on-the-fly when editing is started and thus<br>
Glade cannot access it. You need to set those properties from withing<br>
GtkCellEditable::editing-started signal handler. API docs feature a<br>
sample code that shows how to set entry completion, but you should be<br>
able to modify it to adjust spin button properties with ease.<br>

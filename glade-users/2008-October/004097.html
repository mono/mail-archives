>> But this does also effect you in as much as what widgets you will

>> the better (warning: liststore in trunk is a work in progress, the rest is
>> pretty stable).
>
> I looked around, but couldn't find any docs. Which widgets should I stay away
> from if I'm using glade 3.4.1 so I don't get in trouble when the gtkbuilder
> stuff becomes available?

Just stay away from the gtk+ obsolete section (and the whole gnome
catalog), and if you are using menus/menubars, know that in gtkbuilder
they wont be supported until the 2.16 release (which is scheduled a few
months after the next gnome 2.26 release), but we *will* provide a clean
conversion for menus (Im working on that right now).

Should you need menus in builder sooner, you'll have the option to use
UIManager once I'm feature complete, which I'm expecting to be within
the month, and you'll get a devel snapshot tarball at least at that time,
again, I expect there will be bugs for the following month ;-)

> /opt/local/bin/gcc
> /opt/local/include/
> /opt/local/lib

Try typing

	cc

and see whether it responds with "no input files".  If it does, you have
a C compiler installed.  Have a read of the configure script and look
for the command it is trying to check whether the C compiler works.  It
should put the error into config.log.  Read the error and base your next
action on it.

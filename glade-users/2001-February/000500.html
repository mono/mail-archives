Deepak Gupta wrote:
> 
> hi to all
>             i'm a novice user of glade, making a gui for our
> project using libglade.
>             my need is to have a top level window without
> the maximize button. is there any way out for this?? thanx
> in advance

You could try calling this:

gboolean    gdk_window_get_decorations      (GdkWindow *window,
                                               GdkWMDecoration
*decorations);

typedef enum
  {
    GDK_DECOR_ALL         = 1 << 0,
    GDK_DECOR_BORDER      = 1 << 1,
    GDK_DECOR_RESIZEH     = 1 << 2,
    GDK_DECOR_TITLE       = 1 << 3,
    GDK_DECOR_MENU        = 1 << 4,
    GDK_DECOR_MINIMIZE    = 1 << 5,
    GDK_DECOR_MAXIMIZE    = 1 << 6
  } GdkWMDecoration;

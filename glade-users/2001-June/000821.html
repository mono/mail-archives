Yep, I got also the problem but it is because Mandrake doesn't install by
default (depending on your hard drive space) all the library for developing.
You just have to launch the Software Manager (in the menu
Configuration->Packaging), and search for "db1" (get rid of the "-l"), it
should show you a list with maybe "libdb1" that you just have to select and
click then on the install button!
For my own experience, this was not the only library missing so for all the
other ones, you just have to search for them (and don't forget to get rid of
this "-l", it is a compiler flag!). Sometimes it is a bit more tricky to
find the library as it is included with others in a whole package so the
name might be different...

Others, please correct me if I'm wrong, but Glade is a development tool
to create user interfaces. To generate HTML/CSS, you would need to write
something in another language such as C or Python. Those languages can
interface with Glade as far as loading and displaying GUI objects, but
my understanding is that one cannot generate an application in Glade
that does something programmatically other than GUI things (like tying
widgets to values of other widgets, etc.).

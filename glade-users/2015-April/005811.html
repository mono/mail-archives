On 06.04.2015 8:28, avenidagez wrote:
> Yes, mi OS, windows 7 spanish
> Glade installed and all labels are in spanish, so I did searched to use Glade in english
> And the is no option in preferences, neither in the install process to select the language?
> 
> Is there a way to select in this close minded application to change the language?

Set the LANG environment variable before running glade. You can do it in a
batch file like this:

set LANG=en_US.UTF-8
start glade.exe

You could try a List View... I think it's what you're looking for, something like a table to which you can add lines on demand..  Unfotunalelly it's a bit weird working with it in Glade. (you actually only create it, but all handling should be done manually using gtk functions).

I am currently working (in my spare time, that is) on a quick tutorial to trees/lists, but it is still going to take a while.  Meanwhile you could try taking a look at:

hope it helps.

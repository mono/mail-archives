You can edit the XML file in a text editor. The structure is fairly easy
to understand.

The only problem with editing the XML file is accidentally creating
forward references (e.g. a reference to a radio group that doesn't
appear until later in the file). But I don't think that should affect
you, since widgets in a notebook shouldn't refer to external stuff.

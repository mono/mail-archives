Terry Haimann wrote:
> 
> I am new to glade and couldn't figure out how to make a sub menu.  Is
> there a site that has some glade documantation?

You select the menubar, then select the 'Edit Menus' command in the
property editor or by right clicking on it.

Then you type in the menu items and click on 'Add' to add them to the
list.

You use the arrows at the bottom of the list to indent the items, so
you make a tree like this:

  File
     New
     Open
     Exit
  Edit
     Cut
     Copy
     Paste
  Help
     About

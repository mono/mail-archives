On Linux, use "ldd program" to find what libraries are used by a program
... for a statically linked program it lists none.

% ldd `which cow`
        statically linked (ELF)
% ldd `which glade`
        libgtk-1.2.so.0 => /usr/lib/libgtk-1.2.so.0 (0x40017000)
        libgdk-1.2.so.0 => /usr/lib/libgdk-1.2.so.0 (0x4013c000)
        libgmodule-1.2.so.0 => /usr/lib/libgmodule-1.2.so.0 (0x4016f000)
        libglib-1.2.so.0 => /usr/lib/libglib-1.2.so.0 (0x40172000)
[...]

> Hi,
> found a .gladep containing <glade-project>. Never seen one before. Glade didn't do anything when I opened it. Are these still used? Doesn't seem to have any worthwhile information.

Those are very outdated.

For what it's worth, they come from the original Glade (Glade 1)
application and were an internal format used to save the project,
this was before even libglade was developed and the only way
to use Glade was to generate code.

Glade 3 supports the standard xml generated by Glade 2 and
parsed by libglade as well as GtkBuilder xml format, by now
libglade is deprecated, please use GtkBuilder for any new
projects.

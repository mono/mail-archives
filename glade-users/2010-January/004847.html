Hello

> sorry for the silly question, but I'm not able to add any kind of widget
> to the Gtkassistant in Glade 3.6.7.
>
> In particular, I want to display on the assistant second window a simple
> text area where the final user will write his/her email address, then
> passed as a variable to the linked python script.
>
> So, which is the right way to personalize the GtkAssistant in Glade?

GtkAssistant has three pages by default when created with Glade and on
each page there is single label inside it. To make GtkAssistant with
different content, just delete that label which will give you space to
insert any widget (or widget tree) you want.

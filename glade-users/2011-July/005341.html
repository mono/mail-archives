Yes, see this bug report:

Or if you simply want to get Glade going quickly,
here's a bundle you can extract somewhere, go into
the bin directory and run glade-3.exe:
http://www.optionexplicit.be/projects/gnome-windows/20110405/glade3-bundle_3.8.0-1-20110405_win32.zip

If you need "Python Widgets Support" you can use the
PyGTK All-in-one installer (Glade 3.8.0 is included):
http://www.pygtk.org/downloads.html
or
http://ftp.gnome.org/pub/GNOME/binaries/win32/pygtk/2.24/

> dialog1 = create_dialog1();
> gtk_widget_show(dialog1);

Try replacing gtk_widget_show() call with gtk_dialog_run(). This will
start recursive main loop that will return when user clicks OK. Don't
forget to hide your dialog when you're done with it.

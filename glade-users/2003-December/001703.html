Compile without GnomeDB support, using './configure --disable-gnome-db',
or get a newer version of GnomeDB.

What versions of GnomeDB do you have? i.e. run
  pkg-config --modversion libgda
and
  pkg-config --modversion libgnomedb

Glade is supposed to only compile with GnomeDB support if the necessary
libraries are available.

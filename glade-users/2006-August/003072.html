Yes ! you have overlooked libglade :D

Glade generated code has been unmaintained for years and is generally
considered a bad idea - that considered, glade no longer generates code
(ofcourse this also means that we no longer maintain a bulk of irrelevent
code just to generate code that noone is using anyway).

You can save yourself some trouble, and probably a few hundred lines of
code in your application, by just using libglade.

On Mon, 2003-11-03 at 08:50, Alessandro Bottoni wrote:
> http://www.gnome-db.org/ says:
> 
> "libgnomedb: Database Widget Library. These widgets are integrated with the 
> latest versions of glade."
> 
> But the Glade 2 installed by my Mandrake 9.1 (Rel. 1.1.3) does not show any 
> gnome-db widget.
> 
> How much "latest" has to be Glade to support gnome-db data-aware widgets 
> without having to recompile it from source with the --enable-gnome-db option? 
> That is: which was the first release of Glade with built-in support for 
> gnome-db?
> 
> Is the gnome-db support included by default in the latest binary versions of 
> Glade or have you to recompile it from source anyway?

I think most/all distributions compile Glade without gnome-db support,
so you'll have to compile it yourself from source if you want to use
gnome-db.

(Ideally Glade support for gnome-db would be in a plugin which it only
loaded if gnome-db was present, but I doubt that will be added to
glade-2 at this point.)

Your safe.  

gtk+ and Glade use the LGPL.  The first L stands for lesser, which means 
you can use these libraries and the code generated by them how you like. 
 If you modifie the libraries themselves, then that new modification 
must then be released under the LGPL as well.  Read the licence, it's a 
lot more clear than I.

Thanks Damon.  There were absolutely no glade/gtk+ related problems at all.  
However libgda/libgnomedb needed some tweaking but nothing glade related.

The only thing to note from my experiences is that I have to manually modify 
the file configure.in (before running autorun.sh) to include the package 
"libgnomedb" when compiling with gnome-db.  Otherwise it complains about 
missing libgda, etc. files.

Anyway for other people, if someone want to add superscript/subscript to
the button label using Glade 2.12, then should follow steps:

1. Select the button label (not whole button)
2. Delete the existing label from the button
3. Add new label to the button from the glade Widgets
4. Add subscript by setting <sub>2</sub> (or superscript)
5, Set "Use markup" for the label

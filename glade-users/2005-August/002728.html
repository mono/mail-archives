Not a reporter for GTK per se, but there is a nice GPL
reporter written in C called RLIB. Free for GPL software
but you can license it for non-GPL usage. If you ask
something about it on the user support list at the
first time, you will be asked what you use RLIB for.

You can find it here: http://rlib.sicompos.com
Wiki is here: http://newrlib.sicom.com

Download:
http://sourceforge.net/project/showfiles.php?group_id=88448

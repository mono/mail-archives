It is not the code that will help you. You will also
need to know the procedure thru which you can write
the code. It is explained in this tutorial that has
been pointed out to you. If you do not find it of any
help, then let me know I shall send you code for a few
buttons and the method to put those buttons on the
screen. 

Remember that like Integrated development environment
of Visual C++, the Glade also generate a number of
files and it will become difficult for you to see,
which files have been genrated by the glade and which
files have been created by the user. So, first study
the tutorial and then you may ask further questions.

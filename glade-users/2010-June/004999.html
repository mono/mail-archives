Glade needs attention and support for the info bar simply has not been
added yet.

Support for this particular widget will be tricky as it has 'action widgets'
and 'responses' like dialogs do (and I haven't checked the gtkinfobar sources
to see if they implement the GtkBuilder semantics for response buttons
either..).

However, if anyone cares to try implementing it let me know here
or on irc (there is documentation[0] on how the plugin works too).

Cheers,
       -Tristan

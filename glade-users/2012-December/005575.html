As far as I know, Glade still builds on win32, Glade 3.8 and even I think
Glade for GTK+3 (more recent builds) also build, for GTK+3 if I recall
correctly, we still dont have access to the python plugin.

See for example the pre-built package here:
    http://lists.ximian.com/pipermail/glade-users/2011-July/005341.html

Glade also now includes a 'builds' subdirectory which contains
useful scripts for building on other platforms (currently we only
have a win32 directory there, but I plan to eventually put some
xml to use with the osx bundler, some support for the ige-mac-integration
style build).

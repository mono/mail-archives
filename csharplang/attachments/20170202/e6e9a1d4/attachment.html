<p class="MsoNormal">Copied from: https://github.com/dotnet/roslyn/issues/12836<o:p></o:p></p>
<p class="MsoNormal" style="mso-margin-top-alt:auto;background:white"><b><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">Issue:</span></b><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"><br>
The CLR supports a set of operators for the </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">System.IntPtr</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"> and </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">System.UIntPtr</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"> types
 (</span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">native int</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">). These operators can be seen in </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">III.1.5</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"> of
 the Command Language Infrastructure specification. However, these operators are not supported by the C#.<o:p></o:p></span></p>
<p class="MsoNormal" style="margin-bottom:12.0pt;background:white"><b><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">Workaround:</span></b><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"><br>
Either directly emit IL or write non-optimal code that does not directly translate to the appropriate IL instructions.<o:p></o:p></span></p>
<p class="MsoNormal" style="background:white"><b><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">Request:</span></b><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"><br>
Language support should be provided for the full set of operators supported by </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">System.IntPtr</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"> and </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">System.UIntPtr</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">.
 These operators are: </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">Add</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">Divide</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">Multiply</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">Remainder</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">Subtract</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">Negate</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">Equals</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">Compare</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">And</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">Not</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">Or</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">XOr</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">ShiftLeft</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">, </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">ShiftRight</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">.<o:p></o:p></span></p>
<p class="MsoNormal" style="background:white"><b><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">Other Thoughts:</span></b><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"><br>
C# seems to treat both </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">IntPtr</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"> and </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">UIntPtr</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"> as
 a type only used for interop and for representing pointers in a 'safe' manner, when it is really just:<o:p></o:p></span></p>
<p class="MsoNormal" style="background:white"><span style="font-size:10.5pt;font-family:-apple-system;color:#777777">an integer whose size is platform-specific.<o:p></o:p></span></p>
<p class="MsoNormal" style="background:white"><span style="font-size:10.5pt;font-family:-apple-system;color:#333333">It should be possible for a user to treat the </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">IntPtr</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"> and </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">UIntPtr</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"> types
 as an equivalent to the native </span><span style="font-size:9.0pt;font-family:Consolas;color:#333333">size_t</span><span style="font-size:10.5pt;font-family:-apple-system;color:#333333"> type. However, in order to do that, one must may have to deal with one
 or more of the following:<o:p></o:p></span></p>
<ul type="disc">
<li class="MsoNormal" style="color:#333333;mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1;background:white">
<span style="font-size:10.5pt;font-family:-apple-system">Perform multiple levels of casting<o:p></o:p></span></li><li class="MsoNormal" style="color:#333333;margin-top:3.0pt;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1;background:white">
<span style="font-size:10.5pt;font-family:-apple-system">Use integers whose size is larger than their platform's native size<o:p></o:p></span></li><li class="MsoNormal" style="color:#333333;margin-top:3.0pt;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1;background:white">
<span style="font-size:10.5pt;font-family:-apple-system">Have branching to use integers that are platform-specific<o:p></o:p></span></li><li class="MsoNormal" style="color:#333333;margin-top:3.0pt;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1;background:white">
<span style="font-size:10.5pt;font-family:-apple-system">Work with unsafe code and do pointer arithmetic<o:p></o:p></span></li></ul>

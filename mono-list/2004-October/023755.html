Today you can explore the power of mono on a new free web hosting.
monoForge, this is the name to remember, offers free web space on a Apache
+ mono + MySQL Environment.

Publishing your code on monoForge it's very simple: register on
http://www.monoforge.com and you will get a 20 MB account accessible via FTP
backed with MySQL database and the mono engine.
monoForge can be used to test your Windows .Net application in a Linux
environment, to develop the new killer app or just to say: I develop on
Windows and my projects run on Linux.

See you on http://www.monoforge.com

I'm running an ASP.NET MVC site with this stack and it's running quite well
on Mono. I'm currently on Mono 3.8:

   - ASP.NET MVC 4
   - Cassette for CSS/JS combination and minification
   - ServiceStack OrmLite with MySql.Data for the data access layer
   - SimpleInjector for IoC / Dependency injection
   - DotLESS for CSS preprocessing
   - ELMAH for error logging
   - MiniProfiler for performance profiling
   - T4MVC

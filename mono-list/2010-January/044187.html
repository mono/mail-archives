I find the mono web sites very confusing. Basic information is often hard to
find, out of date, ambiguous and/or contradictory.  Many times a single web
page will have contradictory information.  For example the page at 
http://www.mono-project.com/Mono:OSX http://www.mono-project.com/Mono:OSX 
reads "Mono's Installer for MacOS X comes with Gtk+ and Gtk# so you can
start building and running cross-platform GUI Gtk# applications that include
OSX." and a little further down also reads "There are no MacOS X packages
available at this time for Monodoc, Gtk#, XSP, mod_mono or MonoDevelop." 
I'm always left scratching my head. In my opinion the mono web sites need an
overhaul. They need to be edited for content and correctness.  Is there any
way for the mono community to contribute updates to the web site?

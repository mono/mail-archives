Hi,

> I want to install and use mono on a Fedora linux operating system
> (2.6.23.17-88.fc7).

That is an old distro and I think was the last version not have Mono in
it.

> But i could not found the mono packages to install for this distribution.
> 
> could you please point me to the steps to do that.

Personally, I'd update to f10 and use the mono packages in there.
Failing that, if you go to 

http://download.fedora.redhat.com and just navigate to the f10
directory, you can grab everything from there. You will need libgdiplus,
bytefx, ibm-data and anything that starts with mono (more or less).

If you have yum installed, you should be able to alter the yum entry to
grab from f10.

I will warn you against using the above approach as it may cause
breakages elsewhere. F7 is (in Fedora terms) an old distro and you may
be better off building from source.

For the 2.0 version of a particular assembly do e.g.

pushd ../mcs/class/Managed.Windows.Forms
make install
popd

or similar e.g. (cd ...... ; make install)

A significant speed up! :-)

(BTW for FX1.1 "make PROFILE=net_1_1 ..." )

devel@muhlesteins.com wrote:
> I'm just curious as to how mod_mono/xsp performs in a production
> environment and if anyone is doing anything significant with that

I am using mod_mono for www.govtrack.us.  The server runs Fedora Core 2, 
the latest mono RPMs, and I have no current problems (beyond the known 
issue of having to restart mod_mono after rebuilding assemblies). 
Installation was problemless, configuration with multiple virtual hosts 
(one mod_mono server) was not too bad (although a few reads through the 
man pages was necessary), and it works great.

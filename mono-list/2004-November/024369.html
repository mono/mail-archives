> Subject: [Mono-list] how to unintall mono on macosx
>
> Once I install the mono-1.0.2 on my macosx box ,Now I want to install
> the 1.0.4 version.How can I uninstall the pre-version?

Mono on Mac OS X is installed as a framework inside your startup disk's 
Library/Frameworks folder. Multiple versions can coexist without 
problems (although the 'mono' on the $PATH will be the last one you 
installed); you don't need to remove the previous one if you don't want 
to.
If you want to remove it, go to the /Library/Frameworks/Mono.framework 
folder, open the Versions folder and trash the folder with the version 
you want to remove. After that, run the 1.0.4 installer. To completely 
remove Mono, trash the Mono.framework folder, or use a tool like 
DesInstaller to remove the installed files by reading the install 
receipt.

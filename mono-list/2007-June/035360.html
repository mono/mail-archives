Hello,

An alternative to using Cocoa# is to write a "normal" Cocoa app and  
embed Mono, calling into managed code using the C API whenever you  
need it. For a Hello World app Cocoa# is probably sufficient, while  
the embedding approach is more reliable and resource-friendly for  
production use.

> My Apache config is as follows:

> "/mono/:/opt/apache/mono/:/usr/local/share/doc/xsp/test
>         /"
>     245 <Directory /opt/apache/mono>
>     246         SetHandler mono
>     247 </Directory>
>     248 <Directory /usr/local/share/doc/xsp/text>

Line 244 should be:
MonoApplications
"/mono:/opt/apache/mono,/demo:/usr/local/share/doc/xsp/test"

Lately, I've been avoiding the trailing `/' on directory names and the
like. However, I did notice that certain configurations required the
trailing `/', so you may have to experiment with that. As for the
MonoApplications format, Gonzalo described it as:

[[hostname:]port:]VPath:realpath

With commas separating multiple virtual paths.

Hope this helps!

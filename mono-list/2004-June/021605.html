We just release a beta of the Improve C# plugin for eclipse 3

FEATURES
********
Works under Windows and Linux
        - Use mono 1.0 or .net 1.1
	- C# code editing with: 
		- Syntax highlighting  
		- Auto-indentation 
		- Creation of new C# files
	- C# code compilation with: 
		- Compiler output information displayed in a console 
		- Specification of the compiler arguments via a file property page (the
arguments are persistent for each file) 

	- C# plugin parameters available via the plugin preference page: 
		- Path of Microsoft C# compiler or mono compiler
		- Automatic build on resource modification

http://www.improve-technologies.com/alpha/esharp/

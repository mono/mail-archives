Try the yum repository for Fedora Core 1 or Fedora Core 2.  They work.

If you are a Fedora Core 2 user, you can add the following lines to your
/etc/yum.conf:

[mono]
name=Mono
baseurl=http://www.go-mono.com/archive/yum-repository/fedora-2-i386/

and then run "yum install mono-complete" to obtain the whole thing.

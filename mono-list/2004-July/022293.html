What Mono 1.0 installer?

If you are talking about the Mono 1.0 Windows Installer on the
http://www.mono-project.com/downloads/
Web page.  Then, no, it does contain GTK# 1.0.

GNOME#, GnomePrint#, and GST# do not work on Windows.

GTK# 1.0 is bindings for gtk+ 2.2.4.

Paco made a combined Mono 1.0 and GTK# 1.0 Windows Installer.

Hi!

While reviewing code, I've seen in System.Environment:

	#if NET_1_1
	                static
	#endif
	                public extern bool HasShutdownStarted
	                {
	                        [MethodImplAttribute =
(MethodImplOptions.InternalCall)]
	                        get;
	                }


I would write

	#if not NET_1_0

since I expect that in 2.x this "static" won't change. But if I reason =
correctly, this "static" item would get "unstatic" with this current =
grammar in DotNet 2.x.

Regards
Jochen

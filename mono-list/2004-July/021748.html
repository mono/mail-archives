My guess is that you have 2 libgtksharpglue.so libraries installed.

What you need to do is see if you do, like one in /usr/local/lib and one 
in /usr/lib. You need to make the 1.0 install the only install on your 
system, as the wrong one is being loaded.

Also make sure all your old dlls in $prefix/lib are gone except for 
mscorlib.dll.

--Todd

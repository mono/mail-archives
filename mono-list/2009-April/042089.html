It's ok.. for the teste project i kept the test.exe in the bin subdirectory
then copied the lib and etc (large though they are) subdirectories into the
same folder as bin. Then i set the debug working directory to the parent of
all of these directories and added a Potentially Evil:

mono_set_dirs("lib\\", "etc");

above:

 mono_config_parse (NULL);

in teste.c - which looks like it's working.. thanks for your help!

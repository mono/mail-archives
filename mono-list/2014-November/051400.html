You only need to be concerned about the license terms if you plan on redistributing it.  But you're probably just writing an application that RUNS on ASP.Net.  So the consumer would install ASP.Net, and then download your application separately, and then run your application on it - meaning - you're not redistributing ASP.Net and therefore don't need to worry about its license terms.

ASP.Net is purely mono, and not related to Xamarin.

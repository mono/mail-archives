The System.Windows.Forms namespace for mono is not complete and I am not sure if mono has plans
for a third party SWF library at this point, but with that said, you can always post the code to
one of the many Windows Forms sites that take third party controls.  Here is one for instance:

http://windowsforms.net/Default.aspx?tabindex=8&tabid=50

I am sure there are others and if you would feel like contributing GPL code then I would suggest
SharpDevelop ... they might be interested, but I am not sure.

http://www.icsharpcode.net/OpenSource/SD/Default.aspx

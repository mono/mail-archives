  - Apache: 2.0.43

The process I follow for get mod_mono compiled out is:
  - cvs -z3 update -Pd mod_mono
  - cd mod_mono
  - ./autogen.sh
  - make
  - cp src/.libs/libmod_mono.so.0.0.0 \
    /home/yoros/apache/install/lib/libmod_mono.so
  - cd src; make -f makedll.mak; cp ModMono.dll /home/yoros/lib
  - export MONO_PATH=$MONO_PATH:/home/yoros/lib
  - Add the following to the config file:
      LoadModule mono_module /home/yoros/apache/install/lib/libmod_mono.so
      MonoApplication /mono /home/yoros/mono/xsp/server/test
  - cd /home/yoros/apache/install/bin
  - ./apachectl start
  - (open the URL: http://localhost:8080/mono/index.aspx)

Hello,

I have a Windows Installer for GTK# 0.8 located at
http://www.go-mono.com/archive/gtk-sharp-0.8-stable-win32-1.exe

It requires you having Mono 0.20 installed. You must install GTK# 0.8 in the
same
installation path that Mono 0.20 was installed.

If you do not have Mono 0.20, you can download it at
http://www.go-mono.com/archive/mono-0.20-stable-win32-2.exe

As always, virus check stuff you download...
Not only the archive, but also after it installing too.

Thanks goes again to Johannes for his help.

This is my first try at creating a Windows Installer for GTK#, so let me
know if you have any problems.

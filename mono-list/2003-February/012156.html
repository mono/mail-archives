In cvs, the makefile already installs mbas like mcs.
But 0.19 was released before that came into play.

Whatsoever: You can use mbas running 'mono mbas.exe ...', just be sure your 
environment settings are OK for Mono.

As with mcs, or for that matter MS's csc and vbc, we don't have direct 
support for solution files, you better run some tool (see this list archives 
please) to generate makefiles from them, using make to compile your 
projects.

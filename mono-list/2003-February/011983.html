> If you need to use a GPLed library in a non GPL program then you can
> negotiate with the copyright holders.
> 
> Sometimes GPL can protect OSS developers from being 'exploited' by
> companies.

You can only negotiate a relicensing as long as you are the sole
copyright owner.  If you accept contributions from a third party, then
the negotiations need to happen with all of the copyright holders, or
alternatively you can request that contributions assign the copyright to
you.

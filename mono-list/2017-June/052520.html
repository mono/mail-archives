This seemed to do the trick:

cat /etc/ssl/cert/*.pem > /tmp/ca-certs.pem
cert-sync /tmp/ca-certs.pem
rm /tmp/ca-certs.pem

I’m not sure if there is some other order of operations that would have worked better, like maybe installing ca-certificates before mono, but I’m glad for cert-sync.

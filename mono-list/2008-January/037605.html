Just a short notice that I've open-sourced a set of libraries for use  
with Smartcards, providing p/invoke wrappers for PC/SC and MCard APIs.  
They are mostly under MIT license.

MCard-sharp was tested on Windows XP only with MCChipdr.dll, the rest  
worked with Windows XP, Mac OS X Tiger, Linux and (as far as I could  
test) Solaris 10. The code is in SVN:

http://code.google.com/p/pcsc-sharp/

Hi,

> Can anyone comment on how to use mdtool to build a solution in release mode?
> I'm currently using it like this:
> mdtool build --project:MyProjectName

Currently, it doesn't allow specifying a configuration for building.

> Is there a way to have it generate makefiles for a release build?

The generated makefiles have support for all the configs. You can use the
configure script for selecting the configuration that you want to build. When
generating the makefiles you can specify the configuration that the generated
makefiles should use as the default one. See man mdtool.

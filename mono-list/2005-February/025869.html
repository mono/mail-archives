On Sun, 2005-02-20 at 10:34 +0000, Dave Murphy wrote:
> I've compiled monodoc from svn, but I'm missing the Mono Handbook and
> Gtk# content. Any ideas why?

In general, documentation goes in the same module as what it's
documenting.  So the Gtk# documentation is in the gtk-sharp module, and
the gtk-sharp `make install` target adds the Gtk# documentation to
monodoc.  (This isn't much of an "in general", as only one Mono module
does this, but this is what 3rd party modules should do.)

The Mono Handbook is in the monkeyguide module.

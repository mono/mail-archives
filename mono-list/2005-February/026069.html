Mono SQL# For GTK# 0.2 has been released.

SQL# is a tool for entering SQL commands and seeing the results in a 
grid.  Database tables can be browsed for their definition too.  It 
works on Linux, Mac OS X, and Windows.

This is release is to match the version which shipped with the Mono 
1.1.4 / gtk# 1.9.2 Windows Installer.

You can get it from here:
http://forge.novell.com/modules/xfmod/project/?sqlsharpgtk

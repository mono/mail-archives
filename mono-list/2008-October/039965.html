It looks like I solved the problem. You need to add the line:

#define MONO_ARCH_NOMAP32BIT

to the mini-amd64.h file in <you mono directory>/mono/mini directory.

This worked on mono 1.2.4.

Though on mono 2.0 it just brought up another error:

https://bugzilla.novell.com/show_bug.cgi?id=MONO58237

which by the looks of it has been resolved, though no solution.

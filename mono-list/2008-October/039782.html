With a command line 
`apt-cache search mono | grep mono`
then use `apt-get` to install the right ones

Or

>From Applications -> Add/Remove Apps
In the Programming section.
Show All Open  Source Applications
Search for 'mono'

Check and install MonoDevelop, Monodoc

You can start from there.

This is most likely a Wine error message. I think that wiimote.exe is a 
managed executable. If it requires Windows as well you can install Mono 
for Windows on Wine and you will be able to run that. If you want to run 
it natively on Linux you should run it using Mono.

See http://www.mono-project.com/Guide:Running_Mono_Applications

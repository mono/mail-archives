so in your user dir try to do :
mkdir ~/MonoSVNStable
cd ~/MonoSVNStable
svn co svn://anonsvn.mono-project.com/source/branches/mono-2-0/mono
svn co svn://anonsvn.mono-project.com/source/branches/mono-2-0/mcs
cd mono
./autogen.sh --prefix=/full/path/where/yu/normaly/install/with/generic/one
make
make install
echo yu normally have mono 2.0 installed, now yu just need to tweak
your .bashrc for having a good environment like with the generic
installer

This is a question for the keepass developers.  "Mono" doesn't crash.  Mono is an implementation of the C# programming language (for all intents and purposes in this conversation.)  So if the application crashes, it's due to an error in the application.

If there is a problem in mono, the application developer (or anybody, really) would need to say "The following class or method in mono is expected to perform X, but the actual result is Y, this is a bug, submit bug report, etc."

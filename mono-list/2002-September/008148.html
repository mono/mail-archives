Hello,
at least in the Mono release 0.15 there is a bug in 
mono/metadata/debug-helpers.c, mono_method_desc_new (), line 154:

  while (*end) {
      if (*end == ',')
          result->num_args++;
  }

is an endless loop. It should be:

  while (*end) {
      if (*end == ',')
          result->num_args++;
      ++end;
  }

Holger

On 09/05/02 mani gandan wrote:
> What's the advantage of mono when compare with portable.NET?

mono has a JIT compiler, so it's faster (on x86).
mono has more complete class libraries.
mono has a more complete runtime.
mono has a full-featured C# compiler.
mono is self-hosting (no MS tools required).
mono is easily embeddable in other applications.

pnet has some advantages in some areas, though I suggest discussing 
the pnet features on the pnet lists, not here.

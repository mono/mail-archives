Hi All,
 
OCL 0.5.0 has just been released.  The new release contains several
additions including:
 
*	Implementation of the Extended Numerics library as defined in the
ECMA CLI specification.
*	Support for multiple library assemblies as defined in the ECMA CLI
specification.
*	A lightweight implementation of the SOAP 1.2 specification.
*	Enhancements and bug fixes to the Compact Profile libraries.
 
For more information, or to download OCL, visit:
 
http://sourceforge.net/projects/ocl <http://sourceforge.net/projects/ocl> 
 
Regards,
Scott

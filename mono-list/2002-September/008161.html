Hi!

Are you talking about .NET 1.1 or 2.0?

1.1 will be just 1.0 + SP2+ new bugfixes, but fully compatible.
2.0 will increase speed, add DirectX 9.0 and add some features everywhere in
the classes. I guess it will be compatible, after backwards compatiblity is
the way MS follows since 20 years :-).

However since differnet .NET versions can be installed along with each
other, API breaking won't stop old apps from running.

1.1 will be available in October/November.
2.0 Beta 1 will come early 2003 and will be finished by the end of 2003.


If someone knows about additional features, please add here.

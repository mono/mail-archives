Does anybody know when groups will be implemented in regular expressions?
The code below works with CSC:

Regex q = new Regex(
    "(?<something>\\w+):(?<another>\\w+)");
Match n = q.Match("Salary:123456");
Console.WriteLine(
    "{0} = {1}", 
    n.Groups["something"].Value,
    n.Groups["another"].Value)

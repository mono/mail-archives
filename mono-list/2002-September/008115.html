    This means the executable is unable to link itself to the libraries at
the runtime.
    Just try a simple program that uses at least some library, or may be the
one that is given in "configure" file (look for the code near the lines with
something like "Checking if gcc works fine"). Compile it and execute it.

    I faced a similar problem only to find that the files required at
runtime were not been made available.

The Delphi 7 support for .NET is slightly misleading.

Delphi 7 ships with a command line "preview" compiler for .NET, which
obviously is a no go for production apps.  The other .NET support is
importing assemblies, but it's all COM interop anyways, so I can do this
with every version of Delphi except 1.0.

I prefer Delphi's IDE to Visual Studio though, but it looks like that'll
be a long way off.

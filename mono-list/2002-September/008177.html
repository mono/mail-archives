Hi!

Well, It's just the Linux way of doing a setup, and once you're used to it,
it's better than windows.
But the dependencies...

Try this:


1. Copy all 4 rpms to one directory.
2. Open a console window in that directory.
3. Type "su", and enter you're root password.
4. Type these commands.

rpm -Uhv libgc-6.1alpha5-1.i586.rpm

rpm -Uhv libgc-devel-6.1alpha5-1.i586.rpm

rpm -Uhv mono-0.15-1.i586.rpm

rpm -Uhv mono-devel-0.15-1.i586.rpm

That's it.
My suggestion is, that rpms should be shipped with a shell script,
installing all required rpms how it should be.
But that's not a problem of mono.

Johannes

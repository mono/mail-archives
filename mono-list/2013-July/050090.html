Hello,

in order to develop for Xamarin.iOS (formerly MonoTouch) or Xamarin.Android
(formerly MonoDroid) you will need an IDE called Xamarin Studio -
http://xamarin.com/studio.

Because both of platforms target different operating systems you will also
need:

[Xamarin.iOS]

- Mac system (or PC system, with networked building and deployment
http://docs.xamarin.com/guides/ios/getting_started/introduction_to_xamarin_ios_for_visual_studio
)
- XCode (optionally if you decide to design the UI)

[Xamarin.Android]

- Mac or PC system

I also would like to point you for the development center, place where you
can find more specifically information and tools to help you:
http://docs.xamarin.com/

Cheers,

I believe that DotGNU Portable.NET has a C compiler targeting the CLR. 
Perhaps this could be used to compile the existing C code.

	http://www.gnu.org/projects/dotgnu/pnet.html

Their FAQ mentions using their cscc compiler to compile C code, and the
use of "pnetC" as the libc implementation.

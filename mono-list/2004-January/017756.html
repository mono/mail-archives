Alejandro Ospina <alejandro@nuevaescuela.com> wrote:

> Hi!. I want to install Mono 0.29 on my Mandrake 9.2 machine.
> I downloaded the RedCarpet2 client y when I connect and see the list of
> avalible channels I only see two channels: redcarpet2 and Evolution... 
> How can I install Mono on that distribution?

The contrib repository of Mandrake 9.2 contains mono 0.26.
I made 9.2 packages of the current cooker versions (0.29). You can find them
at:
http://mirror.brain.org/linux/mpol/mandrake/i586/9.2/RPMS/

The easiest is to just add it as an urpmi medium by running:
"urpmi.addmedia mpol
http://mirror.brain.org/linux/mpol/mandrake/i586/9.2/RPMS/ with
../base/hdlist.cz"
After this you can run urpmi monodoc, or urpmi gtk-sharp to install packages,
or just urpmi --auto-select to update original 9.2 packages.
There are no packages for mod_mono, because I couldn't get it working yet.
Quel Qun, who did work on the packages claims it works for him, so I have to
fiddle with it some more.

I don't know if Ximian wants to make packages for Mandrake themselves. If so,
the best would be to have /usr/bin/ilasm configured as an alternative
(update-alternatives) so it doesn't conflict with Mandrake's mono and pnet
packages.

If you have problems using the Oracle ADO.NET provider on Mono on Windows,
comment the line for the oci library in the file
mono_installation_path\etc\mono\config

Before:

	<dllmap dll="oci" target="clntsh" />

After:

	<!--<dllmap dll="oci" target="clntsh" />-->

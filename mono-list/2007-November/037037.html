I use vs.net 2k5 and copy my asp.net project files to my linux server. I haven't found what I need yet in sharpdevelop or monodevelop. Though I haven't tried any of those in a year. Also I believe x-code runs on java and is out for linux too, so perhaps you can use it to develop for mono. 
 
I've also heard that other mono developers just use their favourite code editting tools like vim or emacs or just gedit, kwrite for developing. And then compile and debug using the commandline tools mono supplies. If you learn how to do that, you can debug with the latest tools without having to wait for monodevelop to implement them into their GUI.

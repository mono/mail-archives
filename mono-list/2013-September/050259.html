Why not use Xamarin.iOS?

http://docs.xamarin.com/guides/ios/advanced_topics/native_interop

Otherwise, the process is rather complicated. You can configure Mono to build a cross compiler where the host is OSX and the target is ARM. Then you have to disable features like JIT and you potentially have to customize Makefiles or manually build just the components needed to get the mini install. Building Mono for iOS requires access to some of the iOS SDK headers for locales and other bits. It doesn't seem likely to build on Windows.

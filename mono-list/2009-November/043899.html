Hi Sirmak

well, im not sure what your definition of a high loaded server is. We
run a production server very successfully at DM&ICT Service Centre.
Currently there are about 5 or 6 Mono websites running without any real
problems. Currently its the Mono 2.0.1 version, cause I still have some
issues with the 2.4.2.3. Next to this, my own server has been running
Mono for quite a while now. The only thing I had issues with, was the
Threadpool being to small. After increasing it, I hardly have had any
downtime anymore. But, I dont think these websites should be marked as
high loaded.

To who should i send a message, if we would like to be included in the
'companies using mono' page?

Hi!

The first release of mBloggy is released.=20

mBloggy is a C#/Gtk# client to send/edit/delete posts to a blog, it
supports XMLRPC (Blogger and Metaweblog) and SOAP (bloggyws, JUST for
Jaws).

Each *blog service* works as a plugin (.dll, P/Invoke) that the client
reads and use to connect with the blog service.

This is my first app in Mono, so maybe some users can find some bugs
sending posts with XMLRPC very fast.

Thanks to mono community for helping me with P/Invoke issues and some
other issues.

For next release I promise to clean the code.

Url: http://mbloggy.jaws.com.mx

Tested with: 1.0 stuff.

See ya!

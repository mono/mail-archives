Hi all,

I just updated Npgsql cvs. This update adds many bug fixes as well as 
add the geometric types support.

The following geometric types are supported:
box, point, lseg, polygon, circle, line and path.

Now there is the NpgsqlDbType enum with all the supported types which is 
used when creating new parameter objects.

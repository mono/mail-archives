Mono's SqlClient supports Integrated Security (NTLM), but you must be on a true windows domain.  A workgroup does not count. You also have to give your windows domain user id and password.  On Mono's SqlClient, you have to give your windows domain credentials because Mono is not integrated with your Windows workstation, nor SQL Server, nor the Windows server where the SQL Server instance is running.

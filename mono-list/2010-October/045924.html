Hi All,

NUnit 2.5.8 has just been released. Although it's primarily a bug fix release,
it actually represents a significant milestone for me.

This is the first release for which a significant part of the programming was
done on Linux using MonoDevelop. Prior releases were tested under Linux,
but the code was developed on Windows.

As a consequence, the Gui runner now works much better on Linux and
I have removed the warning on the download page advising Mono users to
stick with NUnit 2.4.8.

The Release Notes are here...
   http://nunit.org/index.php?p=releaseNotes&r=2.5.8

Download it here...
   https://launchpad.net/nunitv2

Charlie

Hi,

With the Linux Installer 1.2.3.1, several issues have been fixed.

There's no need to restart your desktop to use the Monodoc and
MonoDevelop launchers.

~/.bashrc will be created for you if it doesn't exist and you select to
have your environment updated during install.

A post install script will scan for missing libraries which Mono depends
on.  If found, they will be reported during install.

A script has been added:  <install dir>/bin/setup.sh that contains all
the necessary environment settings.  You can 'source' this file if you
didn't have the installer modify ~/.bashrc.

Enjoy!

Wade

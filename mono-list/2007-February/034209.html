Hi Luca,

> I'm using the novell ones, i think, installed using yum, with the
> mono.repo file downloaded from the mono website.
> To install the fedora ones I have to install the manually?

No, you just need to enable the fedora-development.repo file (su;
vi /etc/yum.repos.d/fedora-development.repo and change the enabled=0 to
enabled=1. When you update do something like yum -y install mono*
--excluderepo=fedora-core)

But you MUST yum -y remove mono* before you do the above. Once complete,
change the fedora-development.repo enabled=1 back to enabled=0.

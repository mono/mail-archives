You do understand that the point of this effort *is* to make use of mono.
The aim is to take an existing application that works on MS.NET and port it
so it works on Mono and Linux. For example if there's some killer
application that you have on windows that you'd love to see work on linux,
you'd name that. The whole point is to take something that works well on
other enviorments and make it work on Mono.

Hi greg,

The apache log you posted is missing all the [debug] info so either
you're not using the debug mod_mono.so or you must increase debug level
on apache command line args (or config file).

Once you get all the [debug] messages you should get there the exact
command mod_mono is using to launch mod-mono-server, executing that
manually on a command prompt (replacing the double \\ and / with single
\) should give a hint on why mod-mono-server is dying upon start.

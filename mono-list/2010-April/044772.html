Hi.

You can install all packages, and you will get support for 1.1, 2.0 and
several features of 3.0 and 3.5 versions of .NET framework. In my case, I
only installed packages for support of 2.0 and above versions of .NET.

You will need as minimum:

Mono core (framework, class library, compilers, git, runtime, etc).
Monodevelop (IDE for easy development, it integrates solutions, support for
projects, asp.net projects, moonlight, etc).
ASP.NET support (xsp and apache module).

Daniel.

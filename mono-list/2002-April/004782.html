As Paolo said, you can also go to:
http://mono.baselabs.org/index.php/software/

At the end of the page you'll see
mono-0.10_baselabs-20020405.i386.rpm

That RPM contains the latest binary release including the class
libraries.  Install it and you're set to go.

Note:  This file is updated daily.  The file name will be
mono-0.10_baselabs-`date %Y%m%d`.i386.rpm

Best,
Daniel.

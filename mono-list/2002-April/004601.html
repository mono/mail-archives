Has anyone had considered creating wxWindows (http://www.wxwindows.org) 
bindings for Mono?

wxWindows is a free, cross platform library that currently supports Linux 
(GTK+ or X11 with emulated widgets), Windows (native controls) and OS X 
(native controls). It's actually much more than just a GUI toolkit - for 
example, it supports cross-platform printing.

I realize the primary focus of Mono is the Linux platform, but since 
wxWindows supports Linux, and you'd basically get the other ports for free, I 
thought this might be of interest, especially now that people are starting to 
run Mono code under Windows.

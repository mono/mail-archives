Hi Johannes

I'm just guessing here, but it's probably got something to do with
unicode support on 9x.

MCS will call Directory.GetFiles(), and the runtime uses FindFirstFileW
to service this request. As far as I know, this is not supported on 9x
without the Microsoft Layer for Unicode.

If you're brave and you've got a 9x machine, you might try downloading
this and patching mono to link with it at compile time.

Dan.

Puedes usar KDevelop, pero KDevelop no conoce el syntax de C#.  Una opcion
es decirle a KDevelop que use el "syntax highlighting" de Java.  Son
parecidos.

KDevelop, esta diseñado para C is C++.

Hay un projecto llamado SharpDevelop que esta haciendo un IDE para C#,
pero actualmente solo corre en Windows.

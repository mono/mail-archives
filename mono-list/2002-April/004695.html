> It seems to be the same thing as on unix with the "*".

%* works only  under NT (and up).
Under 9x you could use the following fragment:

set ARGS=
:loop
if x%1 == x goto :done
set ARGS=%ARGS% %1
shift
goto loop
:done
:: now %ARGS% contains all command line args.

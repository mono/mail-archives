Hello there,

You can download precompiled Mono 0.11 that will run on Win 9X here:
http://mono.eurosoft.od.ua/files/mono-0.11-Win32.zip
(it includes MSLU library and wrapper).
Just unpack and run, not need to change env. vars or something, runtime libs
will be searched locally in lib dir.

Also small demo included, take a look at screenshot:
http://mono.eurosoft.od.ua/files/Gfx_NET.jpg
Type mono Gfx.exe to run.
(or mint Gfx.exe if you have really fast machine).

It runs slower under Mono than under .NET (and like Mark's JOrbis port it
uncovers some bugs in Mono), but nonetheless performance is not that bad -
it gives 1.5 FPS on 200MHz Pentium :-)
You can get the sources here:
http://mono.eurosoft.od.ua/files/Gfx_NET.zip


Have fun,
Sergey "Chanben" Chaban :-)

There are two things here that probably need to be made clear:
1) The corlib Unit Tests can be run against the mscorlib.dll to verify
the tests themselves.  This works.

2) To run the same unit tests against mono's corlib, you need to run
NUnitConsole in mono.

I'm going to write up a short how-to on this shortly and post to the
list.

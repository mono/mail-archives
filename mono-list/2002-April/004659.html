Another possibility might be using Microwindows (http://microwindows.org)
I've been considering this, since Microwindows implements 
the Win32 GDI API, it might be simpler for System.Windows.Forms.
Also, Microwindows is very small, and runs on many platforms.

Not all the built-in win32 controls have all the MS functionality,
but they're small, and this would allow Mono to run on embedded
systems, something I'm particularly interested in.

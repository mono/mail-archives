> Where can I find csc and pkg-config utils ?

The easiest thing to do is to download the script mono-build.sh
from http://www.go-mono.org/download.html

If you don't have pkg-config, this script will install it automatically.
Then it will download and install the mono sources from CVS.


You will need to have CVS however (you need it anyways if you want the
latest sources).

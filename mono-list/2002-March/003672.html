Visual Studio.NET, the product, is completely irrelevant here.  You cannot
simply take the license agreement for one product, and state that it applies
to another product.  Please "get" this point, since this is where you are
getting confused.

The only Microsoft product required for Mono development, is the *free* .NET
Framework SDK.  This SDK contains the full functionality required to compile
C# source into assemblies and subsequently execute them in the CLR
environment.

Here's a link to the download page for this product.
http://msdn.microsoft.com/downloads/default.asp?url=/downloads/sample.asp?ur
l=/msdn-files/027/000/976/msdncompositedoc.xml&frame=true

I've attached the license agreement included with this package.  (for
reference: \Program Files\Microsoft.NET\FrameworkSDK\license.txt)  You will
notice that it is significantly different than the license you dug up from
the beta release of VS.NET.

Warren.

An Addendum to the EULA 
(http://msdn.microsoft.com/library/en-us/dnnetdep/html/addendeula.asp), now 
expressly forbids running the .NET redistributables in non-Windows 
platforms, so that even if our runtime now allows it we MUST NOT copy class 
library DLLs or csc, and even sample code, to Linux.

In truth they forbid to redistribute those to run on non-windows platforms, 
but I think the border is fuzzy enough to warrant some trouble.

So I advise every mono contributor that any work with those MS resources 
have to occur on a Windows Box or VM.
DON´T COPY THINGS AROUND!

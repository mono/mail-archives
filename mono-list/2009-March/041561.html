I've found the solution to my problem. Sometimes exe files are corrupted and
will display the "program too big" message as explained here:
http://blogs.msdn.com/oldnewthing/archive/2006/01/30/519388.aspx

which must have corrupted the file, so I downloaded the installer again
using a simple download manager and it works ok.

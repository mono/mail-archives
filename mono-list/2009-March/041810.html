Hi,
<br><br>I've seen the same problem. It's a typo and easy to fix. Change file mono-profiler-logging.c line 4294
<br><br>from
<br><br>&nbsp; &nbsp; &nbsp; &nbsp; if ((stat_buf.st_mtim.tv_sec * 1000000) &lt; profiler-&gt;heap_shot_command_file_access_time)
<br>to
<br>&nbsp; &nbsp; &nbsp; &nbsp; if ((stat_buf.st_mtime * 1000000) &lt; profiler-&gt;heap_shot_command_file_access_time)
<br><br>and recompile.

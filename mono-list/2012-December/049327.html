If you're a complete newcomer, on ubuntu, I might suggest "apt-get install mono" instead of building the latest development and beta releases from source.  Ubuntu I believe distributes with a current version of stable mono.  Generally speaking, the only reason you should update more recent than what ubuntu distributes, is if there's a specific bug or feature in a later release that affects you.

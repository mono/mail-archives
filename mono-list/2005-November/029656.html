Hi,

Mono requires Windows 2000, Windows XP or Windows Server 2003.

Windows NT 4.0 is not supported because shell32.dll version 5.0 API is
required. For the same reason neither Windows 95 nor Windows 98 is
supported.

Windows 95, Windows 98 and Windows Me are not supported because Unicode API
is required.

Probably Windows Me is supported when using Microsoft Layer for Unicode
(http://www.microsoft.com/globaldev/handson/dev/mslu_announce.mspx) but
Windows 95 and Windows 98 support requires source code modification to avoid
shell32.dll version 5.0 API dependency.

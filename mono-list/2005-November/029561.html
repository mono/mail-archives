> Is it possible to make the mono server run with a diferent user+group
> when called by mod_mono in apache?

No. At least with the stable releases of apache. I think 2.1 allow
running httpd instances with different users.

You can still run mod-mono-server on your own with the user you want and
give the user running apache permissions to read/write from the unix
socket created.

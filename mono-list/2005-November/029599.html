Hi,

Try (as root) to remove /tmp/.wapi directory prior to starting apache.
Your error indicates that the apache user cannot create (or access)
that directory, which probably means it already exists (and it's owned
by some other user).

Simple rm -rf /tmp/.wapi as root should do.

alex.

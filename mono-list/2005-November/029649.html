Mono is available from official Ubuntu repositories, version 1.1.8.3
for Breezy, and (I believe) 1.1.10 for Dapper. Just apt-cache search mono
You haven't specified the version of Ubuntu that you are using - if
you're on Hoary, I believe the version included was 1.0.x, so you're
probably better off compiling it yourself.

If for some reason you can't find the packages in your repository,
take a look at:
http://archive.ubuntu.com/ubuntu/pool/main/m/mono/

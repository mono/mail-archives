Okay,
  I've commited the changes necessary to use the code generator outside
of Gtk#. There are a couple of new directories, so be sure to use
"checkout" instead of "update", and rerun autogen.

(Gtk# hackers: I separated out the getting-messy source stuff into
sources/, from the toolchain itself. You'll need to move your sources
there.)

There is now a README.generator on how to use it independently of Gtk#.
Also, I have uploaded a simple example at
http://primates.ximian.com/~tvgm/turtle-sharp.tar.gz .

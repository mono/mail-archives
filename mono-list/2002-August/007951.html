On 08/25/02 Michael Schwery wrote:
> I've figured out that the ~/.mono/config file not created in the
> install. Neither the rpms or source code compiles created it so I
> had to add it myself. It seems to me that this file should be created
> in the install. 

No, the install process should not create files in your home directory.
mono 0.15 will install a (system-wide) $prefix/etc/mono/config file.
If that file doesn't suit your needs, you need to make a copy in your
home and change that.

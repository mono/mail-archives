I think the ANDF is a pretty good idea,
I have been toying with the idea of using it instead of the gcc for the
introspector.

During my reasearch I found a funny mail from RMS about it :
http://www.geocrawler.com/archives/3/357/1991/8/0/1997036/

FROM: Richard Stallman
DATE: 08/15/1991 21:40:07
SUBJECT:  ANDF
Developers can now distribute software for all the kinds
of computers supported by GCC, by distributing source code.
The aim of the GNU project is to encourage this; the aim of
ANDF is just the opposite.  For this reason, I don`t think
it would help the GNU project to install ANDF support in GCC.

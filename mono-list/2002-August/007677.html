Help SDK is about 7M.
The second download is just a small patch for the installer script (skips check for VS.NET).
You need Help SDK because of the Help Workshop tools included (namely compiler).
Then you will use the actual help files from the Framework SDK.
Framework SDK also includes runtime for MSHelp2, as far as I know it's the only way
to get the runtime at the moment (it will be released next year, I think).

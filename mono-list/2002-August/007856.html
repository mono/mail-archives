For their format and IDs see my metadata lib:
http://mono.eurosoft.od.ua/pet.tar.gz

There is MD schema file that I use for code generation, build/srcgen/data/md-schema.xml

Not sure right now how to produce #- stream with the command-line tools (csc, vbc), except
for the older version of J# that always generates this stream instead of #~.

The following page contains some info how to install MSHelp2 SDK without VS.NET:
http://www.helpware.net/mshelp2/h2faq.htm
You can decompile help files (extract HTML etc) using compiler included in
the SDK (hxcomp) with -u option.
If you have VS.NET there is a special help project type for decompilation.

For those not afraid of navigating a Portuguese-language version of a 
sourceforge-like site can find my project to bring a Message Queue Server 
for mono (along with the System.Messaging namespace) at:

http://codigolivre.org.br/projects/monoqle/

Comments are welcome. The first tarball does not have a make file (uses a 
VS.NET solution instead), later I'll switch to a all Mono/GNU tools.

When the server matures, I'll integrate the System.Messaging classes into 
Mono class library.

The server and it's ControlPanel is GPL. The classlib is X11/MIT.

Enjoy...

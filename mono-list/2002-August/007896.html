Apologies if you already worked this out yourself (I only just caught up 2
months of traffic [1200+ messages] on this list tonight). Mozilla can
extract files from a JAR on the fly. It stores all the files it uses for its
chrome and suchlike in them. I can't remember what the URL syntax is for
accessing files in a JAR or whether you have to put anything sensible in the
manifest, but as long as the urls in the Microsoft docs are relative you
shouldn't have much trouble getting it to work.

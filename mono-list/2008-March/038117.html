Just to update everybody on this, Dave Hudson (author of SdlDotNet)
has taken up the task of making FreeTrain run on Linux/Mono/SDL.

It has come far since Atushi posted a patch to make it build on Mono
and, although it's not quite there yet, it's getting there quickly.
We now have a forum, and have also begun the process of making the
game internationalized.

Many thanks to Atsushi for helping out, it was a great catalyst.

If you want to try out the game on Mono, instructions are here:
http://freetrain.sourceforge.net/forum/viewtopic.php?f=6&t=21

The branch you need:
http://freetrain.svn.sourceforge.net/svnroot/freetrain/branches/FreeTrainSDL

Any questions should be directed to our forum:
http://freetrain.sourceforge.net/forum/

It's still not playable yet (!) so any help / patches to push it over
the line would great.

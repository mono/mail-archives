I'll name a few off the top of my head....

> What are the benefits of .NET?

- Cross language interoperability.
- Secure runtime environment.
- Detailed and complete class libraries.
- Extensibility.
- Deployment tools.
- Integrated web services.
- MS increasingly likeley to move in the direction of managed code (Longhorn).

> What are the benefits of .NET over other technologies?

Over C/C++: 
       - Security.
       - Class libraries.
       - Reflection.
       - Object orientation.
       
Over Java:
       - Multiple languages.
       - Better gui performance and appearance
       - Code Access Security.

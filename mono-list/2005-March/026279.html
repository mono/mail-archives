Hi List,

How does one run a Mono app. as a deamon?

In C/C++ I would do it this way:

pid_t   pid, sid;

/* Fork off the parent process */
pid =3D fork();
if (pid < 0) {
    exit(EXIT_FAILURE);
}
/* If we got a good PID, then
   we can exit the parent process. */
if (pid > 0) {
    exit(EXIT_SUCCESS);
}
/* Create a new SID for the child process */
sid =3D setsid();

Hi!

X-develop 1.0 beta has been released. We are particularly interested in 
bug reports/comments concerning our support for Mono on Windows and 
Linux. If you want to tell us what works, what doesn't and what can be 
improved please do so.

Introduction
---
X-develop is a commercial cross-platform multi-language IDE supporting 
the C#, VB.net, Java and Java ServerPages programming languages and the 
MS .NET, Mono and Java VMs. Our main focus are code-centric features 
such as on-the-fly project checking, refactoring, smart templates, 
on-the-fly diff, etc.

More information is available from http://www.omnicore.com.

Mono support
---
Mono is fully supported on Windows and Linux x86. Mono installations 
should be autodetected if the Windows installer is used or if installed 
in one of the usual locations on Linux. If the Mono framework is not 
found automatically it can be configured within X-develop in the 
Integration | Mono section of the Preferences Dialog.

An experimental debugger is available on Linux x86 if the Mono.Debugger 
assembly and the associated libraries are installed.

X-develop uses the same project format as Visual Studio .NET 2003/2005. 
VS .NET 2003/2005 projects should build and run unaltered under Linux 
provided that only libraries are used which are available with Mono as well.

About the license problem,

it seems quite obvious that GPL propagation is not feasible for mono, as it
would contaminate software being run on top of it. We have to find a non
propagating open source license, which would be by the way ok with
Microsoft.

On the patent side, one VERY easy solution would be to create a non profit
organization handling the development of the mono project, where Ximian
becomes one of the contributors, in France or another country where software
patents are not recognized nor legal. And badaboom, you've got an open
source library available for anyone, and not covered by any patent.

Sebastien Lambla

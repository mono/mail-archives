Hey guys!

   So we have now a public CVS server for Mono.  If you have
contributed code and would like to have an account on the CVS server,
please mail me an SSH public identity so I can setup your account.

   If you use Unix, you just need the file ~/.ssh/identity.pub.  If
you use Windows, install Cygwin (http://sources.redhat.com/cygwin),
and from a Cygwin window, type:

	ssh-keygen

   Then mail me the contents of typing:

	cat .ssh/identity.pub

   I will post instructions on how to use the CVS server for those of
you using Windows or for those of you who are new to CVS.

   For those who are not contributors, we are looking for people who
can mirror our CVS server and expose it through `AnonCVS' for allowing
anyone to browse/diff the files.   These mirrors would synchronize
their servers using rsync.

   Contact me for details.  Rsync should be ready to go this
afternoon.

best wishes,
Miguel.

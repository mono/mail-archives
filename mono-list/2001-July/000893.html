Nevermind, I figured it out.  I was using a different glib binary instead of
install glib, in order to avoid a strsignal error message.  But I went back
to the recommended source and configured and compiled it after I found out
the strsignal problem could be solved by removing the #DEFINE HAVE_STRSIGNAL
1 in the config.h file.  After taking the defines out, everything seems ok.

If this isn't the right way to do it, please let me know.

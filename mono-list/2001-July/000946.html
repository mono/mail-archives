Right now the process seems to be:

Download the current source from http://www.go-mono.com/download.html

Look through the source, find something that interests you that needs work
or needs to be implemented.

Check the class status age at http://www.go-mono.com/status.html and see if
anyone else is working on it.

Post to the list what you're planning on taking to let everyone know about
it.

When you have something completed, post again what you did or what you
changed, and work out the best way to get it commited with one of the
maintainers, which right now is just emailing it to the list and/or Miguel.
Ximian seems to be in the process of setting up the source repository, so
something will probably change in the near future.

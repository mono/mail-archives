My suggestion would be to:

	* Create a Google Project that could host the changes.

	* Import the sources

	* Create a branch based on the pristine sources,

	* Start the process of building it on Linux.

	* Evaluate which parts of the project need porting, and create
	  a plan for this, call for contributors.

	* Start replacing the code that uses Direct3D with code that
 	  uses Tao.

I would also use a Google group (as am a fan of Google Groups) instead
of a specialized forum interface, I find it easier to follow.

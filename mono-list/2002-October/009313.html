Simple guess: you wrote you app in Visual Basic.NET.

By default, VB.NET requires an additional dll to run
(Microsoft.VisualBasic.dll).  That dll is not fully implemented on Mono
at this point in time, preventing your application from running.

You have two choices: Wait until Mono implements
Microsoft.VisualBasic.dll, or remove the Microsoft.VisualBasic assembly
from your Visual Studio project and recompile the project.  If you're
using any of the functions from Microsoft.VisualBasic.dll, you'll get
compiler errors which will need to be fixed.

Once your code compiles without using Microsoft.VisualBasic.dll, it
should be able to run under Mono.

 - Jon

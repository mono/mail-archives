There is work on Visual Basic.net like support in Mono.  Does it work?  I do
not know.  I just know that a Visual Basic.net compiler and runtime has been
started.  Rafael Teixeira is heading this effort.

The vb.net compiler and runtime is written in C#, and is based on mcs (the
Mono C# compiler) and runtime.

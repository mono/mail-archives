Here's the fix...

In libodbc.cs, the OdbcReturn enum needs to be a short.

internal enum OdbcReturn : short {
Error = -1,

InvalidHandle = -2,

StillExecuting = 2,

NeedData = 99,

Success = 0,

SuccessWithInfo = 1

}

Let me know if this works for you,

Brian

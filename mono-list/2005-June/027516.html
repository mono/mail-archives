PS Make sure you do not start XSP while running under
an X tunneled session or you will have many problems
related to graphics!!!!!!!

Or if you must then do this if you have a start script
to put it into the background in say /etc/init.d

echo sudo /etc/initi.d/my_xsp_start_script | at now

One of the guys on our team fonud this. It's still
better to not use a tunneled X session if using
graphics in ASP.NET on Mono when starting the XSP
process.

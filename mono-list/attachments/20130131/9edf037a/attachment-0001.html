GLIBC is not eglib. eglib is a replacement for glib (<a href="http://developer.gnome.org/glib/">http://developer.gnome.org/glib/</a>), not GNU libc, which is what GLIBC refers to. You'll need to make sure the versions of GNU libc match between your two machines. You aren't seeing an eglib anywhere in the <font face="courier new, monospace">`make install`</font> output directory because it's statically linked into Mono.<div>

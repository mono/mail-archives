>Please tell me how I would go about incorporating hex 0x02 and 0x03, Start<br>
>Of Text and End of Text, into a string in C#.<br>
>In C i would sprintf it but have as yet not found an equivalent in C#.<br>
>In fact from the reading I have done I am no longer sure that C# can<br>

<br>
        You can use string.format. Instead of %s or %d, you use {0} or {1}, being that the position of the value in the data list. For example:<br>
<br>
        string str = "...";<br>
        string final = string.Format( "{0}{1}{2}", 0x02, str, 0x03 );<br>
<br>

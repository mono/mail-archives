Hello everyone,<br><br>The second beta version (0.1.1) of monotooth is now available in <a href="http://github.com">github.com</a> via <a href="http://github.com/locusf/monotooth/commits/0.1.1">http://github.com/locusf/monotooth/commits/0.1.1</a>. The project sources also reside in &#39;master&#39; branch. <br>
<br>If someone doesn&#39;t remember / know monotooth, it is a library that allows .NET/Mono developers use bluetooth in their projects. The main aim of the project is to provide a clean design and easy to use API for bluetooth development. Documentation is also (or at least tries) mean to be verbose, so that even beginners can use this library.<br>
<br>For more information, see <a href="http://monotooth.sourceforge.net">http://monotooth.sourceforge.net</a> for more information. This release (0.1.1) is no longer packaged since github offers it for free.<br><br>Release highlights:<br>
<br>-Inquiry works on windows (so consider this to be an Alpha release in Windows), but only with the native (ie. not via custom software) interface.<br>-Linux inquiry has been improved not to use custom library.<br><br>With Regards,<br>

</div>You&#39;re confusing namespaces with assemblies.  They are not the same.<br>
<br>
You need to reference Mono.Posix.dll, and use the Mono.Unix namespace<br>
within Mono.Posix.dll.<br>
<br>
Furthermore, UnixFileSystemInfo is the base type for UnixDirectoryInfo,<br>
UnixFileInfo, and UnixSymbolicLinkInfo.  Creating instances of any of<br>
these types will allow you to invoke UnixFileSystemInfo methods on them,<br>
or use the UnixFileSystemInfo.GetFileSystemEntry(string)[0] method to<br>
create the appropriate subclass based on the filename.<br>

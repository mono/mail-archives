<br></div><div style>Reference Assemblies are basically API maps that describe which APIs you can consume for a particular combination of runtimes you can target.</div><div style><br></div><div style>For example, if you want to target something that includes the old Xbox, Silverlight 2 and modern .NET you would be given a particular profile that gives you that.    If you want to target the Windows Store and Windows Phone 8, which have a newer version of the APIs, you would be given a different profile.</div>

I use following script for password samba/ldap password change:<br>
<br>
$ cat chpass<br>
#! /bin/sh<br>
<br>
&nbsp;&nbsp;&nbsp; {<br>
&nbsp;&nbsp;&nbsp; echo &quot;$2&quot;<br>
&nbsp;&nbsp;&nbsp; echo &quot;$2&quot;<br>
&nbsp;&nbsp;&nbsp; } | /usr/sbin/smbldap-passwd &quot;$1&quot;<br>
<br>
<br>
here is no magic, just call that chpass with &lt;username&gt; &lt;password&gt; arguments...<br>

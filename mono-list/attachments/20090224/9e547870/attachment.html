VB 2008 should produce &quot;managed code&quot; which is a set of .exe´s and .dll´s, all of them compatible with mono in linux. No need for VMware or Wine.<br><br>(unless you use something very specific to windows, such as an &quot;unmanaged&quot; third-party dll).<br>
<br>To run your app in Ubuntu you need all the same libraries that your project uses in windows; most of mono´s libraries can be found with Synaptics (or &quot;Add/Remove programs&quot; in Ubuntu), just search for mono and look for the libraries you want.<br>
<br>the next thing as Chris said, would be the right-clicking or doing &quot;mono path/to/your/app.exe&quot; in a terminal....<br><br>If you have trouble with the needed libraries, I would suggest downloading the latest binary installer (for &quot;other linux&quot;) in mono´s site. It usually works better than the mono version which is shipped by default with some Ubuntu versions...<br>

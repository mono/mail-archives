<br>
Media Browser Server:<br>
Media Browser 3 is a centralized solution. Maintain your library just once<br>
no matter how many devices you have accessing it. Manage plug-ins and<br>
everything else in one place for your whole home – all with a sophisticated<br>
but dead-easy-to-use web configuration platform.<br>
<br>
Media Browser Theater<br>
A completely new stand-alone client with embedded VLC player brings Media<br>
Browser to places it could never go before!<br>
<br>
Mobile Clients<br>
With clients for Android, Windows 8/RT and Windows Phone 7/8 you can take<br>
your media with you wherever you go.<br>
<br>
Your Favorite Plug-ins<br>
It’s a whole new world but a lot of your old familiar favorites will be<br>
there too: CoverArt, TrakT, GameBrowser, Video Backdrops, Trailers and more…<br>
*<br>
Media Browser Classic<br>
This is the new Media Center client for MB3. It looks and behaves much like<br>
the old MB 2.x and still relies on WMC for its presentation and playback. If<br>
you are coming from MB 2 and like or need WMC for its other features, this<br>
client will allow you to stay in that environment.<br>
*<br>
The Dashboard<br>
The Dashboard is the new web-based configuration interface for Media Browser<br>
Server. It is a centralized hub for configuring your library, user profiles,<br>
metadata options, plug-ins, etc. Being web-based, it is accessible from many<br>
different platforms including mobile phones and tablets.<br>
*<br>
Web Client<br>
Within the Dashboard interface there is also a fully-featured web client for<br>
MB 3. This client gives you a very rich interface into your media library<br>
including advanced views and searching capability.<br>
It is also web-based so fully mobile compatible and can be used to browse<br>
your collection and then tell some of the other clients to actually play the<br>
items.<br>
*<br>
Media Browser 3 Store<br>
Media Browser gets even better with premium features...<br>
Here you can browse all the premium plug-in features available for media<br>
browser and purchase the keys to use them beyond their trial periods.<br>
*<br>
Media Browser Free Support 100%<br>
Full support by developers of Media Browser core system and plugins no<br>
hidden fee's.<br>
Media Browser support is driven by the community, with developers actively<br>
taking part as well. If you have an issue or a question it is very likely<br>
that it has arisen before, so we ask that you please use search before<br>
asking for help.<br>

compatibility. If you see something that is wrong or out of date, we<br>

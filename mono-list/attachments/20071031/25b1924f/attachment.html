<div>There no are any problem using the connector .net of mysql on mono. The binaries of this connector are supported by mono, because it is written in C# natively.</div>
<div>&nbsp;</div>
<div>I&#39;m using the version <a href="http://5.0.8.1">5.0.8.1</a>. I downloaded the zip binaries (not installer) and unzip in a directory. The binary MySql.Data.dll must be in the bin directory of your <a href="http://asp.net">
asp.net</a> applicattion and execute gacutil -i to install in the gac. You also can compile with gmcs -r:MySql.Data.dll if you application is an executable.</div>

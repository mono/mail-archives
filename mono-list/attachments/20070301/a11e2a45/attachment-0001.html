There are many open source implementations of sql that run on linux.
You should try PostgreSql. It implements much of the same functionality
as MSSql. There is also MySql and a number of others, but I think for
stored procedures and views and triggers and everything else that MSSql
offers, you would have to go with PostgreSql...<br>

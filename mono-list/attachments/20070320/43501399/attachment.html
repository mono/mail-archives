<p class=MsoNormal style='margin-bottom:12.0pt'>Common mistakes with <a
href="http://ASP.NET">ASP.NET</a> deployment:<br>
<br>
1.- Place a &quot;Bin&quot; directory instead of &quot;bin&quot;<br>
2.- Don't place all the required DLLs in the bin dierctory (or GAC)<br>
3.- Running xsp[2], not configuring the --aplications parameter (or application
in the root dierctory) <br>
4.- Running mod-mono, not configuring the MonoApplications directive (more info
with man mod_mono)<br>

Hello,<div><br></div><div>     We are using 64-bit version of mono for Windows in a production environment. There are no packages available at this point, but it should work fine. Simply install/build the 32-bit version for Windows, and then compile the runtime using Visual Studio. Then drop the built mono.exe/dll into the 32-bit install. You need to checkout the mono directory from svn:</div>
<div><br></div><div><a href="http://anonsvn.mono-project.com/viewvc/trunk/mono/">http://anonsvn.mono-project.com/viewvc/trunk/mono/</a></div><div><br></div><div>Then open the mono\msvc\mono.sln file in Visual Studio and build. There are additional directions in mono\README.vsnet.<br>

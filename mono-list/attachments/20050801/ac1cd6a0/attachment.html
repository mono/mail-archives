<DIV>System.Data.OleDb requires libgda to work.&nbsp; libgda only works on Linux with GNOME 2.0.&nbsp; Also, OleDb is not actively maintained.</DIV>
<DIV>&nbsp;</DIV>
<DIV>I suggest using the System.Data.SqlClient or System.Data.Odbc to connect to Microsoft SQL Server.&nbsp; However for System.Data.Odbc, this will require ODBC and an ODBC driver for SQL Server.&nbsp; On Windows, this should come with MDAC.&nbsp; On Linux, use unixODBC and freetds or use iODBC and freetds.</DIV>

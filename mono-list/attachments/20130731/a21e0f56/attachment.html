</div>Close; you should prefer [DllImport("shared")], with one exception. On Windows, this will try to load SHARED.DLL, on Linux it will load libshared.so, and on OS X it will load libshared.dylib.<br>
<br>
The exception is that if the library name contains a '.', Windows LoadLibrary() will not automatically append a .DLL extension, so this would always fail:<br>
<br>
        [DllImport("gtksharp-2.0")]<br>
<br>
In these cases, you should use the full Windows filename, and provide a .dll.config file so that the file can be found on other platforms:<br>
<br>
        [DllImport("libgtksharp-2.0.dll")]<br>

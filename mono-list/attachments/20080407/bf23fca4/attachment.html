<p>PowerShell open source cross-platform&nbsp;reimplementation.<br>
<h2>About the name</h2>
<p>Pash = PoSH (PowerShell) + bash (one of the Unix shells)<br>
<h2>Goals</h2>
<p>The main goal is to provide a rich shell environment for other operating systems as well as to provide a hostable scripting engine for rich applications. The user experience should be seamless for people who are used to Windows version of PowerShell. The scrips, cmdlets and providers should runs AS-IS (<strong><u>if they are not using Windows-specific functionality</u></strong>). The rich applications that host PowerShell should run on any other operating system AS-IS. Secondary goal: the scripts should run across the machines and different OS's seamlesly (but following all the security guidelines).<br>

<h2>Environment</h2>
<p>The current implementation of Pash is written using pure .Net 2.0. It compiles on VS 2008 as well as on Mono. So all the developers can choose the environment that fits their needs and preferences. The produced assemblies can be executed "right out of the box" on Windows, <a href="http://www.go-mono.com/mono-downloads/download.html"><strong><font color="#0a8fbc">Linux, Mac (or others)</font></strong></a> without any additional recompilation. <b>Note:</b> for Windows Mobile and Windows CE the produced code should be recompiled against the .NET Compact Framework.<br>

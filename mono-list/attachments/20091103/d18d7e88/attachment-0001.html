<P><FONT SIZE=2>Dear All.<BR>
<BR>
We are currently porting Mono to Android. (and soon also to ChromeOS)<BR>
<BR>
We expect to release the first version in the beginning of next year. (2010)<BR>
<BR>
We do this because we need it for Roozz plugin. A technology layer that allow for cross platform development on 6 platforms and integrate pay-per-use, security, updating and copy-right-protection<BR>
www.3djam.com<BR>
<BR>
We hope to make it so that winform programs developed in VisualStudio will run on all platforms without recompile (in most cases).<BR>
The Roozz plugin allow these applications to run both as stand alone &quot;desktop applications&quot; and as applications embedded in a webpage.<BR>
Mono is not integrated into the Roozz technology. But Roozz uses is as an underlying layer for cross platform application development.<BR>
<BR>
Our project needs funding, so if you are willing to help, we will greatly appreciate this.<BR>
<BR>
Best regards,<BR>
<BR>
Thomas Jam Pedersen<BR>

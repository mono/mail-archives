Hello,<br>
<br>
First, try MoMA (works on Windows) on your app: <a href="http://mono-project.com/MoMA" target="_blank">http://mono-project.com/MoMA</a><br>
That&#39;ll give you an idea about whether your app will work with Mono or not.<br>
<br>
Then, install Mono on Ubuntu if it&#39;s not already there (using synaptic or<br>
apt-get for example, Google for help). Last time I tried Mono on Ubuntu, the<br>
Mono core was installed out of the box, but some components such as WinForms<br>
had to be installed separately.<br>
Keep in mind Ubuntu 9.04 ships with Mono 2.0, while the latest stable<br>
version of Mono is 2.4. So you might want to install 2.4, there are articles<br>
on the net that&#39;ll tell you what to do...<br>
<br>
Once you have Mono, you can right click your exe, and choose to run it with<br>
Mono (Mono will probably not be proposed at first, you&#39;ll have to get to the<br>
app list then select or type &quot;mono&quot;, then the option will remain<br>
available)... or command line &quot;mono yourapp.exe&quot;.<br>
<br>
Hope that helps.<br>

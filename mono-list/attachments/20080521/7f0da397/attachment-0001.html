<p class=MsoNormal>I have a problem that I wrote using mono 1.2.5, and when I
recompiled with mono 1.9.1, it started to act badly.&nbsp; I traced it down to
the Timer.Change function.<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>The program uses timer for polling.&nbsp; I don&#8217;t want
to use the Timer &#8220;period&#8221; field, since I want to wait until the
polling function is done before starting the timer again, so I use the
Timer.Change method to reset the &#8220;dueTime&#8221;.<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>I Wrote a quick test program to test the Timer.Change
function.&nbsp; It basically creates 10 objects that each create a timer, wait 3
seconds for the timer to trigger, then set the timer again.&nbsp; After only 7
iterations, all the timers failed to trigger. <o:p></o:p></p>

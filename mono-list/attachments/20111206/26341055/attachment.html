<p>The MonoDevelop team is proud to announce the release of MonoDevelop 2.8.4.2.</p>

<p>MonoDevelop is a Mono based IDE primarily designed for C# and other .NET
languages. MonoDevelop enables developers to quickly write desktop and
ASP.NET Web applications on Linux, Windows and Mac OSX. MonoDevelop
makes it easy for developers to port .NET applications created with
Visual Studio to Linux and to maintain a single code base for all
platforms.</p>

<p>This release contains several fixes and improvements. Check the
<a href="http://monodevelop.com/Download/Release_Notes/Release_Notes_for_MonoDevelop 2.8.4.2">release notes</a> to know more about the changes.</p>

<p>Sources and packages are available in the
<a href="http://www.monodevelop.com/Download">MonoDevelop Download</a> page.</p>

Enjoy!

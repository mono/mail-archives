For example, if   my.exe is compiled with:

gmcs -r:my.dll -o my.exe *.cs

To create a bundle:

mkbundle --deps  -o mybundle.exe my.exe my.dll

The resulting executable is self contained and does not need the Mono
runtime installed to run.

The biggest problem I had was getting the Web.config set up properly.<BR>
For each service you need to define an endpoint, a service behavior, and<BR>
an endpoint behavior -- .NET does a lot of this automatically so it can<BR>
be very confusing why your service doesn't work.<BR>
<BR>
Here's a trivial WCF service that runs on Mono:<BR>
<BR>
    <A HREF="https://github.com/jeffbonhag/wcfservice1">https://github.com/jeffbonhag/wcfservice1</A><BR>
<BR>
WCF works well!  You can perform AJAX requests, you can deserialize a<BR>
request to an object, and serialize a POCO by specifying it as the<BR>
return type of your WCF method -- it's all good.<BR>
<BR>
One issue I had is that UriTemplates of the form:<BR>
<BR>
    "weather/{state}/{city}"<BR>
<BR>
don't seem to work -- at least on Ubuntu, Mono 3.2.8.  Perhaps there's<BR>
additional behavior that needs to be specified.  I'll revisit the issue<BR>
someday.<BR>
<BR>
Good luck!  I've struggled with this for some time so I'm happy to<BR>
help any way I can.<BR>

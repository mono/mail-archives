<div>According to an answer that i get in this list, you cannot to debug <a href="http://asp.net">asp.net</a> pages, but you can to debug xsp itself.</div>
<div>&nbsp;</div>
<div>However, to debug an mono executable program (also a console application and a class library), you must compile it with the -debug option. For example:</div>
<div>&nbsp;</div>
<div>gmcs -debug -out:Program.exe Program.cs</div>
<div>&nbsp;</div>
<div>(gmcs is the .net 2.0 compiler)</div>
<div>&nbsp;</div>
<div>and to debug it with</div>
<div>&nbsp;</div>
<div>mdb Program.exe</div>
<div>&nbsp;</div>
<div>(mdb is the mono debugger, still under development)</div>
<div>&nbsp;</div>

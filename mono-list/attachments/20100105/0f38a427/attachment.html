<div dir="ltr">Hi all I solved the problem at last, so again:<div>The problem is:</div><div><ul><li>when compiling with &quot;--with-profile4=yes&quot;, monodoc.dll gets compiled again with dmcs and replaces the monodoc.dll that was generated first with gmcs/mcs (profile 2). When it&#39;s compiled with dmcs it won&#39;t work because it requires features from runtime v4 but it is actually being run by a v2 runtime.</li>

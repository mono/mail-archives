<p class="MsoNormal">I want to make sure that I understand this proposed GSoC task
correctly: “Using Silverlight 2.0 build the UI for video editing software”. <br>Below
I wrote my understanding of the deliverable, my questions,
bio, and risks. Please tell me if I got the idea right, thanks! </p><p class="MsoNormal"><br></p>

<p class="MsoNormal">The deliverable will be a user-friendly Silverlight
application for in-browser video editing. The app will be somewhat like <br></p><ul><li>the
GSoC 2005 Diva Video Editor <a href="http://mono-project.com/Summer2005#Diva">http://mono-project.com/Summer2005#Diva</a>, </li><li>the Top Banana MIX2007 <a href="http://silverlight.net/learn/learnvideo.aspx?video=125">http://silverlight.net/learn/learnvideo.aspx?video=125</a>
applications.</li></ul><div>The well polished GUI should resemble specialized software like Apple’s iMovie and Adobe’s After Effects. The GUI will consist of:</div><div><ul><li>preview area for the video,</li><li>timeline editor with drag-and-drop editing,
skimming, trimming, cut &amp; paste, slicing.</li><li>resource list (media library).</li><li>Tools to apply effects and transitions to the
clips.</li></ul></div><div>The sample application should work within web browsers
supporting Silverlight 2.0, be written in clean, maintainable code.</div><p></p>

<p class="MsoNormal">Questions:</p><p class="MsoNormal"></p><ol><li>The Diva Video Editor was an incredible GUI
program. We want a similar Silverlight/Moonlight alternative?<span style="mso-bidi-font-family:Calibri;mso-bidi-theme-font:minor-latin"><span style="mso-list:Ignore"><span style="font:7.0pt &quot;Times New Roman&quot;"> </span></span></span></li>

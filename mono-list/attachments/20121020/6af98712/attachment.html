The easiest thing to do is to build the Visual Studio solution mono.sln in mono/msvc. Build the x64 target in this case. It will produce the dynamic library (mono-2.0.dll) and an import lib (mono-2.0.lib). A few others and myself had the Win64 version of the runtime working pretty well. You may run into some issues, but it should simple applications reliably.<div>

<div>For serving images over the internet, using <a href="http://ASP.NET">ASP.NET</a> might be best. You can create an <a href="http://ASP.NET">ASP.NET</a> website in Visual Studio 2010 (from the "empty" template), reference the image generation library in it, and then deploy this to your Linux server. How I would approach this is to make an <a href="http://ASP.NET">ASP.NET</a> website with a handler that generates and serves the images. The code would look something like this, in a file called something like "GenerateImage.ashx" (using an .ashx extension is easiest):</div>

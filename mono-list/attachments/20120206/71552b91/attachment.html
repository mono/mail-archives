In this book, I tried to explain about the internal of the Mono C# compiler such as, how does the source files of the compiler related together, how does the Jay file get converted into the cs-parser.cs file, How does the Mono generate the tokens and using the generated tokens how does Mono parse the source code and so on.<br/>
<br/>
In addition, in the appendix section of the book I included the C# grammar used in the Mono, How to setup Cygwin in Windows operating system etc.<br/> 

<br/>The main objective to write this book was gather all the available information in regards to the Mono C# compiler in one place. As this is my first book, I would like to have suggestions and advice from all of you to improve the book for future release.<br/>

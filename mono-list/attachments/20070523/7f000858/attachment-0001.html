In <a href="http://MS.NET">MS.NET</a> the .net 1.1 Configuration assembly is System.Configuration and the .NET 2.0 Assembly is System.configuration (lowercase c)&nbsp; but in Mono it is combined into 1 assembly System.Configuration
 so when importing VS2005 projects the references don&#39;t match up and you run into build errors until you re-reference the assembly to the mono one. <br><br><br>

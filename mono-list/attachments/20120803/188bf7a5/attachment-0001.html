<br><div><div>Highlights</div><div><br></div><div>* Java / C# / <a href="http://vb.net/" style="color:rgb(17,85,204)" target="_blank">VB.NET</a> / PowerShell / Ruby are now officially supported LDTP scripting languages other than Python</div>


<div>* Approximately 130 APIs are compatible with Linux version of LDTP</div><div>* C# client is compatible with Mono .NET framework and we have tested it on Linux/Mac</div><div>* Identify object name based on automation id (window id, as per SilkTest users)</div>


<div>* i18n support</div><div>* CPU / Memory logging</div><div>* Remote test execution</div><div><br></div><div>New features:</div><div><br></div><div>* List / Tree item API's are added</div><div>* Scroll to the element if the respective pattern is enabled</div>


<div>* Added new characters in keyboard input</div><div>* Object lookup based on wildcard("?")</div><div>* Double click on allowed object's</div><div>* Added hyper link widget type under known objects</div>

<div>
<br></div><div>New APIs:</div><div><br></div><div>* getwindowsize</div><div>* simulatemousemove</div><div>* gettablerowindex</div><div>* getobjectnameatcoords</div><div>* onwindowcreate (Java/C# client)</div><div>* removecallback (Java/C# client)</div>


<div>* mouserightclick</div><div><br></div><div>Bug fixes:</div><div><br></div><div>* Taskbar is now identified as pane, rather than ukn</div><div>* generatemouseevent API now takes the optional argument, compatible with Linux</div>


<div>* Fixed a crash, if the window title has back slash</div><div>* Grabing focus on combobox element fails the object selection, removed the respective code</div><div>* Ignore special characters while searching object name</div>


<div>* Fix regexp in object lookup</div><div>* getcellvalue API now takes the optional argument, compatible with Linux</div><div>* Handle task manager menuitem, which worked slightly different than other menu</div><div>* Fixed listing sub-menus with a simplified method</div>


<div>* getcellvalue API now as the Linux version</div><div>* getchild API now returns appropriate output</div><div>* Fixed *window APIs to work with different types of window</div><div>* Fixed mouse left click on a text widget</div>

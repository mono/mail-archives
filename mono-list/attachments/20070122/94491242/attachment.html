<br>* 128 MB SDRAM<br>* 64 MB NAND Flash<br>* Bluetooth (2.0)<br><br>The OpenMoko Linux Distribution will be built with OpenEmbedded. The<br>developer devices will be based on the following software:<br><br>* Linux <a href="http://2.6.17.14">
2.6.17.14</a><br>* gcc 4.1.1<br>* binutils 2.17.50.0.5<br>* glibc 2.4<br>* Xorg 7.1<br>* glib 2.6.4<br>* gtk 2.6.10<br>* dbus 0.9<br>* eds<br>* (more)<br><br>The OpenMoko application framework adds C-APIs on top of Gtk+, Dbus,

<br>* MokoFingerWindow -- base class for finger main windows<br>* MokoFingerWheel -- rotary finger wheel<br>* MokoFingerToolBox -- finger tool box<br>* (more)<br><br>A set of core applications using this OpenMoko framework is under

Hi Enediel,<br><br>Basically you could do the deploy in visual studio 2005 to some folder... And then copy off to linux server.<br>Firstly you need mono on debian so that you could try to run this application with development server &#39;xsp2&#39; on linux box (in root folder where *aspx pages lives).
<br>If it works well, then you need to install apache and mod_mono.<br><br>Then configure apache so that web application could run.<br><br>I use following apache config:<br><br>Alias /MyWebApplication /var/www/dotnet/siers/
<br>MonoApplications xsp &quot;/MyWebApplication:/var/www/dotnet/siers/&quot;<br>MonoServerPath xsp &quot;/usr/bin/mod-mono-server2&quot;<br>&lt;Directory /var/www/dotnet/siers/&gt;<br>Allow from all<br>Order allow,deny<br>
SetHandler mono<br>MonoSetServerAlias xsp<br>&lt;/Directory&gt;<br><br>/var/www/dotnet/siers/ - path to web app root<br>MyWebApplication - name which will appear on the server,<br>(<a href="http://www.myserver.com/MyWebApplication">
www.myserver.com/MyWebApplication</a>)<br><br>Try to use latest mono versions as <a href="http://asp.net">asp.net</a> 2.0 support is improved<br><br>Regards,<br>Madars<br><br><div class="gmail_quote">On Dec 28, 2007 3:53 PM, enediel gonzalez &lt;

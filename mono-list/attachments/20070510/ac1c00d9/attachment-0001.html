solution is to use a best-guess for the approximate size of the memory stream 
and initialise the memorystream to that to start off with. For example if the 
average size of your class is 68 bytes, then initialise the memorystream to 
array.Length * 68. Or some such thing.<BR><BR>Still, a testcase may prove

<div>The override keyword indicates that the method overrides the virtual method in the base class, but this is not obligation in virtual methods. This is equivalent to Overridable and Overrides keywords in visual <a href="http://basic.net">
basic.net</a>.</div>
<div>&nbsp;</div>
<div>However, you must override the method if the method in the base class is declared as abstract. This is equivalent to MustOverride keyword in visual <a href="http://basic.net">basic.net</a>.</div>

<br>thanks<br></blockquote></div><br>Virtual or override, correct.&nbsp; You must specify this to tell the compiler &quot;yes, I meant to override this&quot; the same way you use the new keyword in that context to tell the compiler &quot;yes, I meant to hide this.&quot;
<br><br>You don&#39;t need to use override to implement an interface method since there is no implementation that you are overriding.&nbsp; (Though using this logic you could make the case that you shouldn&#39;t need the override keyword when overriding an inherited abstract method.)

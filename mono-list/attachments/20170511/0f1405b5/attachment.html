Yeah, msbuild is now a separate component so it's not included if you just build the mono repository from source.
<div class=""><br class="">
</div>
<div class="">You can get it from <a href="https://github.com/mono/msbuild/tree/xplat-2017-02" class="">https://github.com/mono/msbuild/tree/xplat-2017-02</a> (the xplat-2017-02 branch corresponds to the Mono 5.0 release).</div>
<div class=""><br class="">
</div>
<div class="">Compile with "./cibuild.sh --scope Compile --target Mono --host Mono --config Release"</div>
<div class="">Install with "./install-mono-prefix.sh <prefix>"</div>
<div class=""><br class="">
</div>
<div class="">msbuild relies on .NET Core 1.x to bootstrap so you need to make sure you have that installed from
<a href="https://dot.net/core" class="">https://dot.net/core</a>.</div>
<div class=""><br class="">
</div>
<div class="">- Alex</div>

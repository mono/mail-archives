<p>&gt; My general question is, how can I determine which .NET features are<br>
&gt; implemented in MonoDevelop?</p>
<p>To clarify, MonoDevelop is an IDE, the equivalent of Visual Studio. The runtime, which is what actually runs your app (the equivalent of .NET), is Mono. MonoDevelop on Windows even uses .NET :)</p>
<p>You can view the status of the Mono class libraries at<br>
<a href="http://go-mono.com/status/">http://go-mono.com/status/</a></p>
<p>Beware that this corresponds to the latest development version, not the release version that you&#39;ll be using, so you might just have to &#39;try and see&#39; in some cases.</p>

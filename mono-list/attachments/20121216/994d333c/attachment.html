Mono is cross-platform as in: it runs on multiple platforms.<BR>
For desktop operating systems theres GTK# which pretty much works on all the major OSs (Windows, OSX and Linux) just the same.<BR>
Mono is unnecessary on WP7/8 since they both run .Net already.<BR>
It has different bindings for Wii and Playstation because they both have different APIs, and so on.<BR>
<BR>
By cross-platform, mono guarantees to run the same code over all the systems that are supported, <BR>
but sometimes you'll have to write several GUI interfaces to counter each platform.<BR>
<BR>
You'll have the whole .Net library which ever platform you choose.<BR>

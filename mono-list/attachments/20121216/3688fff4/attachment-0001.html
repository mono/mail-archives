The mobile OSs' APIs are too different from one another, there's no way to create a wrapper API to bind them all together while keeping all the functionality.<BR>
But with proper planing, the difference between the apps should be minimal.<BR>
Create classes as generic as possible, so that they can be used with multiple platforms.<BR>

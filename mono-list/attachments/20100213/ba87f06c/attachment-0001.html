<p>I guess MonoDevelop should not provide completion for extension methods when System.Core is not referenced. MD should also reference System.Core by default in new 3.5+ projects. Please file bugs. Arguably gmcs should give a better error message too, so please file a separate bug against gmcs for that.</p>

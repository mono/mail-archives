Hi,
 
I was attempting to cross-compile Mono and failed miserably. Basically,
I have an x86 box that can run either Windows or Linux and I'd like to
build mono for a PowerPC machine that doesn't have a build environment
on it. (At some point I'd also like to try to cross-compile to a MIPS
machine and see what it takes to make that work, but I'm assuming it's
easier to start with the PowerPC.)
 
However, it appears that configuration script isn't really configured to
generate cross-compile Makefiles. At least, no matter how I try to run
configure, it doesn't generate Makefiles that seem even attempt to
cross-compile.
 
Am I doing something wrong or missing something? Is it possible to
cross-compile Mono?
 
Thanks for any tips, hints or info :-)
 
Doug Harber

Announcing release 0.6 of Gtk#, codenamed "Hot on the Monkeys' Heels."

Tarballs available for immediate download from:

http://sourceforge.net/project/showfiles.php?group_id=40240

[Note: these release notes, as well as a largely unmodified candidate
source tarball were prepared by Rachel Hestilow.  Thanks Rachel!]

About Gtk#: 
Gtk# is a .Net language binding for the Gtk+ Graphical User Interface
toolkit. The binding is fairly complete, but it is still
largely unproven and not recommended for production work. 

Changes since 0.5:
* Namespace fixes: Mike
* GStreamer support: Alp
* Configuration summary: Gonzalo
* Compilation fixes: Duncan (scribble), Gonzalo (DbClient)
* New GNOME.NET tutorial: Johannes Roith, Martin Willemoes Hansen
* New tutorial sample code: Alejandro Snchez Acosta
* Clipboard fixes: Vlad
* New .NET-style struct null handlers: Vlad
* Refcounting fixes: Vlad
* UTF8 conversion of source files: Gonzalo
* Add pango constants: Daniel Morgan
* Introspection and GValue object store fixes: Radek
* New dialog constructor: Mike
* Metadata fixes and updates: Mike
* TextBuffer convenience "Text" property: Duncan
* EventArgs dynamic construction fixes: Gonzalo

Discussion of Gtk# occurs on the mailing list at
gtk-sharp-list@ximian.com.  Bugs can be reported to bugzilla.ximian.com,
module gtk#.

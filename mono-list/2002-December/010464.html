I had similar problem, I have RedHat8.0. I had one more warning when I
started server:

** Warning **: cannot find /usr/etc/mono/machine.config

I found the file in /etc/mono/machine.conifg, so I created symbolic link
to /usr/etc/mono/ and it helped. Now it works. So this may help you,
too.

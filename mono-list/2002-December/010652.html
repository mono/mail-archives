Hello all,

I just completed another tool - cert2spc. It's a clone of the MS cert2spc
tools which combines X509 certificates and CRL into an SPC (Software
Publisher Certificate). In fact this is just a PKCS7 file containing all the
certificates. On Windows 2000 the file can be opened as a certificate store
(in MMC).

The source are commited in /mcs/tools/cert2spc.cs and ASN1.cs (shared from
X509Certificate.cs) and are part of the build process.

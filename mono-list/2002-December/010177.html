The status of the documentation is basically that we have set down on
using the ECMA XML schema for documenting.  The only change is that we
have one file per class to be documented instead of a big file with
everything.

The documentation is on module `monodoc', but we lack a complete editor
or a documentation browser.  If you do not mind entering the data
manually (which is not that difficult), you could help in the
documentation effort.

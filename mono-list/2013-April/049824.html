Ubuntu 13.04 which was released today has MonoDevelop 3.0.3.2 in it.

Best way to learn about debian packaging is to look at packages for
existing opensource mono projects. The debian package source for my
project (Wide Margin) can be found here:
https://bitbucket.org/trampster/widemargin/src/4161aa4df469c58f7dbd68308936d885070ded74/debian?at=default

For debian packaging help go to #debian-cli on IRC OFTC, All the
debian mono packagers hang out there and are very helpful.

Best of luck,
Daniel

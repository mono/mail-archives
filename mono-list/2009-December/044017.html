Hello,

For some reason I don't see the original emails but I checked them out
on nabble.

Editing in the DataGridView is broken (and unfortunately has been for
a while), because if I remember correctly there is something wrong
with how the DataTable or its View dispatch
ListChanged/PositionChanged events in the "add new row - edit - commit
/ cancel" cycle (too many events I think) which leads to DGV thinking
there is a new row added when in fact there isn't one or something
similar. I remember looking at this issue for a while digging into
System.Data but never got to fix it and unfortunately I won't be able
to spend time on that soon. It really sucks not to have this working.

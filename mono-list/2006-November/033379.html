On Tue, 2006-11-14 at 11:12 +0900, Atsushi Eno wrote:
> The closest path to support fortran would be some kind of GCC
> support for CIL backend, which practically does not exist yet.

This is not completely true... the support is there even if not
100% complete, and just targeting the C language for now.

See here:
http://gcc.gnu.org/ml/gcc/2006-09/msg00101.html

I will be working on completing support for external C libraries
in the next few weeks.

Since Fortran does not use the C++ "object model" or similar high
level things, it is likely that this work will "almost" provide a
Fortran-CIL compiler (using the gcc Fortran front end and the CIL
back end), but those who are interested should step forward and
do their share of the work ;-)

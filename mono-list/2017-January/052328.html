For Debian 8 (Jessie) you need to add a second separate repository for libdiplus as mentioned here: http://www.mono-project.com/docs/getting-started/install/linux/#libgdiplus-debian-80-and-later-not-ubuntu

> echo "deb http://download.mono-project.com/repo/debian wheezy-libjpeg62-compat main" | sudo tee -a /etc/apt/sources.list.d/mono-xamarin.list

I just tried this in a clean Docker container and it works.

Hi guys
I came across http://sourceforge.net/projects/nbt in my search for a nice 
open source c# bittorrent client library. The code in that project wasn't 
working and last updated in 2000/2001. I've managed to hack it into 
submission though and currently it's mono compatible and single file 
torrents work. I'm looking into building a web based torrent download 
service for people behind firewalls etc. POC on my website @ 
http://blog.megabike.name/downloadStatus.aspx.

If anyone is interested in the current incarnation of the code you can get 
it at http://megabike.name/NBT.Core.zip (vs.net 2003 and monodevelop 
projects included). Original author is chinese so I removed most of the 
comments and there's no documentation, but the code is well written and 
easy enough to understand. All I ask is that you send me patches if you 
make any improvements. I'm still trying to contact the original author to 
try and take over his sourceforge project, but no luck so far.

I have similar problem on my Windows 2000 machine. I couldn't successfully
solve it. I appears even if you compile simplest Hello World program.

The only solution that worked for me is NOT to install Beta2 full version,
but to install REDIST version. It contains CSC and all the libraries. What
I'm missing is MSDN Library... But at least it works.

You might experiment with installing full .NET SDK, then copying the
C:\Program Files\Microsoft.NET, uninstalling, installing REDIST version and
then running the MSDN Library. I hadn't had time to try this.

Hope this helps.

I've seen this error a few times before.  It has happened to me when
my system ran out of memory.  And it always happened while compiling
SWF.  Use "top" in a different terminal while you're running the
compile and watch to see if the mem usage goes to 100%.  If so you
either need to add more memory or hack the configure scripts to not
compile SWF.

-Abe

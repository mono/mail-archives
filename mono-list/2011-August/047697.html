On 08/28/2011 04:37 PM, BillC wrote:
>> But running programs as root is not always a good idea, so why not try
>> something like
>> su yourUserName -c "mono program.exe"
>> this should run the command under your login.
>>
>
> I've tried that already, but it's still not starting my program at boot.
> I also couldnt find any errors in messages of boot.log

Couple of things I'd look for:
- redirect stdout/stderr of your program to some file, like so:
   mono foo.exe >& /tmp/foo.log
- when using 'su' to run it, try 'su -l -c ...' so that you get a properly setup environment for 
your program to run in.

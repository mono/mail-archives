Sorry, I've missed that. You need to set PKG_CONFIG_PATH to
point to the prefix you've installed glib:

export PKG_CONFIG_PATH=/usr/local/lib/pkgconfig:$PKG_CONFIG_PATH

Adjust the /usr/local part of the path with the prefix
you've specified to `configure'. It defaults to /usr/local.
Then add the line to your profile script or create a new
script "monopaths" and source it with ". monopaths" (assuming
you're using a sh-like shell).

Moreover, you should add /usr/local/lib to your LD path.
See the docs of your system about how to do it (usually
done by editing /etc/ld.so.conf or by placing a file in
/etc/ld.so.conf.d/ or by setting LD_LIBRARY_PATH).

Hi All,

Intel Labs has implemented portions of the CLI (Common Language
Infrastructure) runtime library as defined in the draft ECMA CLI
specification, and has released it as open source, under the Intel Open
Source Software license (BSD style).  To learn about the release contents or
licensing, visit:
http://prdownloads.sourceforge.net/ocl/OCLReadme.htm

To download the source code, visit the OCL main page at:
http://sourceforge.net/projects/ocl

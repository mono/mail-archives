Hello all Mono contributors,

I am not sure this is the right place and I apologize in advance if it is
not, but I would like to congratulate all of you for the great initiative to
port CLR and C# to Unix. I believe what you are doing is going to have a
great positive impact in the software industry in the years to come. Can't
wait to see the results!

I myself would like to contribute by helping either developing the runtime
or the class libraries, but I have a couple of questions:

1) I can see that the runtime is being implemented using C. Since I have
more experience and prefer C++, and also by the fact that you are
implementing a runtime that will run object-oriented code, I was wondering
why you guys haven't chosen C++?

2) If I want to contribute by implementing some classes of the standard
library, may I do that on Windows by using cygwin? And if the answer is yes,
what kind of portability issues I should be concerned if I am
cross-developing from Windows?

3) And, finally, how can I checkin and checkout to the source code
repository? Is the access to the CVS repository publicly available (at leat
read-only), or should I have to send my code to some of you?

Thank you in advance for the answers, and, please, keep up with the good job
you are doing for the human kind ;)

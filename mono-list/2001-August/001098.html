Hello guys,

    Daily snapshots of Mono (compiler, class libraries and runtime) are
now being done thanks to Unix's super powerful, innovative and ground
breaking `cron' tool.

    You can now get your dose of daily snapshots from a single
location:  http://www.go-mono.com/snapshots, it should be useful for
those of you without a CVS account yet.

    Also, unlike the released tarballs, these snapshots contain the
sources to a number of administrivia bits (the web site for instance)
which is not yet part of the automake process.

Enjoy,
Miguel.

Apologies if this is off-topic, but I thought people on this list 
might be interested in the following announcement.

I have initiated a project to create an open implementation of 
ECMAScript (the standardized version of JavaScript) on the CLI 
platform.  The project is named JANET.  For further information, see 
the project home page:

    http://janet-js.sourceforge.net/

The project is in its early stages, but I have posted some fairly 
detailed design documents.  Comments, suggestions, and contributions 
are welcomed.

-- Steve

MonoDevelop only has one GUI designer on the Mac, the Gtk# designer, so you need to create a Gtk# project for it.

In addition, if you are using Xamarin Studio (which is just MonoDevelop, but with an Android and iOS extensions added), then you get UI designers for Android and iOS.

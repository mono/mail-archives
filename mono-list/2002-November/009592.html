GTK# runs on mono on Windows.  I would like to know if it runs on mono on
Linux.

I have an app in VB.NET ported from gtk-sharp/sample/ButtonApp.cs
and it works on mono on windows using gtk#.  It was built using vs.net with
references to Mono's System and System.Drawing and gtk# assemblies.

It does not run on .NET though.
I cheated by running the .exe in my ~/mono/install/lib path which contains
all my mono assemblies, gtk# assemblies, gtk# glue, gtk+ and dependencies.

Imports GLib
Imports Gtk
Imports GtkSharp
Imports System
Imports System.Drawing
=0A=
Public Class ButtonApp=0A=
=0A=
    Sub New()
        Application.Init()=0A=
        Dim win As Gtk.Window
        win =3D New Window("Button Tester")=0A=
        win.DefaultSize =3D New Size(200, 150)=0A=
        Dim delete As DeleteEventHandler
        delete =3D New DeleteEventHandler(AddressOf Me.Window_Delete)=0A=
=0A=
        AddHandler win.DeleteEvent, delete=0A=
=0A=
        Dim btn As Button
        btn =3D New Button("Click Me")=0A=
        Dim btnClicked =3D New EventHandler(AddressOf Me.btn_click)=0A=
        AddHandler btn.Clicked, btnClicked=0A=
=0A=
        win.Add(btn)=0A=
        win.ShowAll()=0A=
    End Sub=0A=
=0A=
    Public Sub btn_click(ByVal obj As Object, ByVal args As EventArgs)=0A=
        Console.WriteLine("Button Clicked")=0A=
    End Sub=0A=
=0A=
    Public Sub Window_Delete(ByVal obj As Object, ByVal args As =
DeleteEventArgs)=0A=
        Application.Quit()=0A=
        args.RetVal =3D True=0A=
    End Sub=0A=
End Class=0A=

Module Module1

    Sub Main()
        Application.Init()
        Dim buttonApp =3D New ButtonApp()
        Application.Run()
    End Sub

End Module

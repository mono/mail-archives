> I failed to find an archive of posts so if there is one I'd be happy
> to search through it before posting here.  I've been following Mono
> for quite some time and have played with it a bit.  I now have some
> time to actually write some programs in it and test things out.  I was
> wondering what classes are installed with the 0.16 rpm version?  For
> instance, I can include System.Collections classes but not System.Data
> classes.  If some classes are installed by default and others (i.e.
> System.Data) need to be downloaded and installed as assemblies, where
> would I get the assembly source?

The Mono C# compiler by default only references the corlib.dll and
System.dll assemblies.  Any other assemblies (like the System.Data) are
not referenced by default.

To do this, just compile your code like this:

	mcs /r:System.Data proggie.cs

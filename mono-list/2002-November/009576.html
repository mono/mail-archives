I am collecting references on mono/gtk# presentations, and I checked by

there.

somewhere, so that they can be linked from the GNOME.NET tutorial or
hosted by it.

Here is a list of the presentations that will be going on at .NET ONE

1. Keynote: Miguel de Icaza: Behind the Open Source Implementation of
the .NET Framework?

2. Writing Portable GUI Applications for Win32 and UNIX Using Gtk# (Mike
Kestner)

3. Hosting the Mono Runtime (Paolo Molaro)

4. Gtk# - Automated Library Binding Using Perl, XML, and C# (Mike
Kestner)

5. The Mono C# Compiler: Design and Implementation of a CIL Compiler 
(Miguel de Icaza)

6. The Mono JIT Compiler (Paolo Molaro)

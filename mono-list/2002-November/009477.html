i have been able to get mono to build on openbsd-current i386, but have
not had any success in getting it to run. i wanted to send everyone here a
note for their own information that mono appears to be promising on
openbsd (a major hurdle, building, has been accomplished).

my experinces were written up in a few messages to the openbsd ports list:

	http://monkey.org/openbsd/archive/ports/0211/msg00011.html
	http://monkey.org/openbsd/archive/ports/0211/msg00027.html

the second message describes where the problem occurs in mono. i dont know
enough about everything involved in it (the compiler, runtime, threading,
etc), so i'm not going to pick up the project on openbsd. however, if
someone else wants to, you have a decent base to start with.

thanks for writing reasonably portable code, by the way.

> you must do is to call its Initialize() function - this function takes an

> as first argument and we do not know what this is.

void Initialize(IntPtr emitter, string filename, bool fFullBuild);
emitter: The metadata emitter interface.

As well as the managed class library, Microsoft .NET provides an unmanaged API
for much of their execution engine's functions. This is how native programs
like csc and ildasm emit and import .NET assemblies.

You may want to take a look at the include directory in the Framework SDK,
where you'll find the COM interface IMetaDataEmit defined in cor.h. My guess is
that emitter is an instance of this interface. Not that that helps you much ;-(

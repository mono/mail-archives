This is a pointer to (unmanaged) implementation of IMetaDataEmit interface - it is supposed
to be created using COM instantiation methods, that's why it's passed using IntPtr.
SMC compiler sources contain some information about this, as well as Unmanaged Metadata
API docs (info on emitter).
Perhaps Rotor sources could provide some insights as well.

To aid in searching the Mono mailing lists I setup a simple search link 
to Google's SiteSearch.

I've placed an html search form at:
    http://www.mycgiserver.com/~gregw/pseudosearch.html

One could certainly save that document locally (or place at the bottom 
of http://www.go-mono.com/mailing-lists.html !).

Since google doesn't index lists.ximian.com on a daily basis, the search 
results are obviously not guaranteed to be up to date (but for me 
they're still useful -- beats grepping through those txt archives).

I'm not sure if multiple form inputs with the same name is valid HTML... 
but it works from mozilla. I used that as an attempt to refine the 
searches to the mailing list by simply adding a q=mono-list input.

More info on Google's WebSearch / SiteSearch service is at:
    http://www.google.com/services/free.html

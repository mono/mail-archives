Hello,

Some of you guys might already know, these days I have been writing 
Reactive Extensions (Rx) equivalents for mono, as mono-reactive project 
on github.

I got System.Reactive.dll feature complete (consider this as, no 
NotImplementationException), so I decided to tag it as version 0.1.

For the details, check out this page. I'd be highly appreciated if 
anyone who is interested in Rx tries, uses or even hacking it :)

https://github.com/atsushieno/mono-reactive/wiki/mono-reactive-v-0.1-released

I built it using mono from github master, so I'm not sure if it builds 
with our released mono versions. AFAIR Paul Betts (the ReactiveUI man) 
mentioned some compiler issue at Monospace [*1], but git master worked fine.

[*1] http://www.infoq.com/presentations/Reactive-Extensions-and-ReactiveUI

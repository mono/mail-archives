Dear All,


I have completed writing a book about the Mono C# Compiler. Since there are
only few published books so far for the Mono, I thought it would be good
idea to have another one which will help all the new starter who would love
to work with the Mono C# compiler and also others to use it as reference
book.
 

The book is published through amazon.com, if anyone would like to see the
book please click on the following link,


http://www.amazon.com/Understanding-Mono-Compiler-Mohammad-Rahman/dp/1469929066/ref=sr_1_1?ie=UTF8&qid=1328512604&sr=8-1 
http://ecx.images-amazon.com/images/I/51CaaI0AnjL._SL500_AA300_.jpg  


http://www.amazon.com/Understanding-Mono-Compiler-Mohammad-Rahman/dp/1469929066/ref=sr_1_1?ie=UTF8&qid=1328512604&sr=8-1
Understanding Mono C# Compiler 


In this book, I tried to explain about the internal of the Mono C# compiler
such as, how does the source files of the compiler related together, how
does the Jay file get converted into the cs-parser.cs file, How does the
Mono generate the tokens and using the generated tokens how does Mono parse
the source code and so on.


In addition, in the appendix section of the book I included the C# grammar
used in the Mono, How to setup Cygwin in Windows operating system etc.
 

The main objective to write this book was gather all the available
information in regards to the Mono C# compiler in one place. As this is my
first book, I would like to have suggestions and advice from all of you to
improve the book for future release.


Thank you very much all,
Mohammad A Rahman

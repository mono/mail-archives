Hi Everyone,

Just wanted to let everyone know that the open-source Eto.Forms cross
platform UI framework (https://github.com/picoe/Eto) now supports WPF, in
addition to the already-supported GTK#, MonoMac, and WinForms.

Eto.Forms is a cross platform UI where you can write one UI, and it will run
across all platforms.  It is a really great choice if you want to provide
the best experience on all platforms without having to re-write a separate
UI for each platform.

Enjoy!
Curtis.

After several fixes in the remoting infrastructure, I've been able to
run some Remoting.Corba examples in Mono (all fixes in CVS). For those
who don't know, Remoting.Corba is a project that aims to integrate
CORBA/IIOP support into the .NET Remoting architecture (more info in
http://remoting-corba.sourceforge.net/).
The client and server remoting channels seem to work fine. I haven't
been able to test with real ORBs, feedback on this would be greatly
appreciatted.

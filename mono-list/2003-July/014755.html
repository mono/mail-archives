I was running into this a week or so ago.  The problem (for me) was
twofold: (1) I'm not a Red Carpet Express subscriber, and (2) I was
using redcarpet.ximian.com as my Server URL.

(2) is the bigger problem, as I get Access Denied when I try to open
http://redcarpet.ximian.com in my browser.

Solution: see
http://support.ximian.com/cgi-bin/ximian.cfg/php/enduser/std_adp.php?p_sid=6dD7KKMg&p_lva=&p_faqid=376&p_created=1052943988&p_sp=cF9ncmlkc29ydD0mcF9yb3dfY250PTE3JnBfc2VhcmNoX3RleHQ9JnBfcHJvZF9sdmwxPTMmcF9wYWdlPTE*&p_li=

The short of it as follows:

	# killall rcd
	# /usr/sbin/rcd --no-network
	-- Start Red Carpet
	-- Click Edit->Preferences
	-- Click the Mirrors button under Server
	-- Pick a server *other* than redcarpet.ximian.com
	-- Exit Red Carpet
	# service rcd stop
	# service rcd start
	-- Re-start Red Carpet.

You may need to wait awhile after restarting rcd, as it will attempt to
update itself (get channels, etc.), so give it some time.  (I had it
hitting 100% processor usage for a couple minutes, but it went back to
normal after that.)

Hopefully this will work for you.

 - Jon

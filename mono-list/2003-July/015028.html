Yves Kurz wrote:

> Can anyone give me a link to e good C# IDE or better a link to a plugin
> for eclipse (2.1.1) ?

AFAIK, there's no C# plugin for Eclipse, at least no plug-in comparable 
to the JDT.  Improve technologies offers a simple C# editor that can 
also call the CSC or MCS and show errors in the task view.  It doesn't 
run with 2.1.1 though.  There're two editor plug-ins offering generic 
syntax highlighting which could be used for C# editing.  But they 
doesn't offer any compilation support.

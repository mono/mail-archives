I don't have my os/x box online right now, but there is a precompiler def
for socklen_t in mono/io-layer/sockets.h that needs to be removed for mono
to compile on panther.

It was a necessary define for os/x 10.2 with the december 2002 dev tools
installed.

Keep in mind that you will need to eliminate mini (and i believe profiler)
from mono/Makefile. These projects have historically failed to compile on
all versions of os/x.

For more specific information, search the mailing list archives @
http://www.go-mono.org for more info. I submitted a detailed post a while
back re: the socklen_t issue.

Good Luck!

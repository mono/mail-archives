> error to log it or do additional stuff (like MsgBox.Show etc.).
>

How about this:

try
{
    using (resource)
    {
    }
}
catch (Exception ex)
{
    // resource already freed
}

or

using (resource)
{
    try
    {
    }
    catch (Exception ex)
    {
        // can still access resource here
    }
}

Hi.

I just got the same issue recently, while trying to get Mono running on my
W7 laptop.

In the "Compiling Mono on Windows" page, we read how to run the "configure"
script in order to build Mono on Cygwin. However, the shown option
("--host=i686-pc-mingw32") doesn't work.
It doesn't provide you with the right compilation options and gcc doesn't
call the right compiler.

I then tried : "--host=i686-w64-mingw32" and it worked. gcc then calls
mingw-gcc and everything runs smoothly.

[I guess you don't need this information anymore, eight months later, but I
hope it may be useful to someone !]

Since you don't specify what development environment you are using I
will assume you are using Monodevelop in 2.0 mode.

Monodevelop (0.18) will not add the control to the designer file as soon
as you add it to you .aspx file. You will either have to build the
project after you have added the control to the aspx file, or put the
control your self to the designer file.

                                          Hi All,

  I have resurrected my coverage analysis tool for .NET. The new
version is available at:

www.nexus.hu/vargaz

It has the following improvements:
- the coverage data gathering part is implemented as a pluggable
  profiler modul, so no runtime patching is needed.
- Much improved user interface.
- Ability to export the coverage data to XML, which can be
viewed
  in any XSLT capable browser like mozilla.

The coverage data gathered while running the corlib unit
tests is
available here:

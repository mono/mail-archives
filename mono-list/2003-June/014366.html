...but monodoc needs that thing called "gtkhtml3", something that hurt
me to install for quite some time.  Also, I have on my machine a build
of that wonderful HTML parser know as "mozilla".  So, I looked for a
solution.

gtkmozembed-sharp.  It's up to the user to merge this in with the CVS
build process.  I've tested it, and it seems to work at everything I
throw at it.  Even the history seems to work (wow!).  So, if you have a
working mozilla install, then all you need for doc-love is moz-monodoc.

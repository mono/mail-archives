From: Bill Mattox

> I am wondering if there is any kind of ability for Mono to use MTS or some
kind of transaction server?

We are working on developing our own appserver infraestructure with full
support for local transactions and all the support we can gather enough
documentation to build to use distributed transactions. But we are just in
the planning phase...

Support specific for MTS is not planned as it depends on a resource that is
not available in other plataforms. But nothing holds you from developing
applications in Mono, that run inside MTS when run on windows using the MS
runtime, but I would just advise you against it for portability reasons .

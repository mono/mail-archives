wx.NET 0.3 is now available:

  http://sourceforge.net/project/showfiles.php?group_id=72362

wx.NET is a set of .NET bindings for the wxWindows cross-platform
toolkit.

New in this release:

   - XML Resource file support (thanks to Patrick Wilhelm and Achim
     Breunig),
   - wxStyledTextCtrl (Scintilla) editor bindings,
   - wxConfig support (for cross-platform configuration settings),
   - Several new input dialogs (thanks to Alexander Olk),
   - Beginning of wxGrid support (data grid control),
   - Printing framework (printing setup dialogs and classes),
   - and numerous other improvements.

For information about wx.NET, Mac OS X port screenshots, and more, visit
http://wxnet.sourceforge.net/.

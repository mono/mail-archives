My advice is to first make it work the "normal" way which means install mono-complete from package manager.

And then incrementally try getting the mkbundle stuff to work. Because you're surely going to run into issues, you'll want to deal with those one at a time.

Personally I don't think it's realistic to expect users to be able to run a mono/gtk# application without any ability to install mono/gtk# dependencies. But you may succeed - Good luck.

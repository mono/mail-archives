Hello Ivar,

If you write it; we will commit it. :-)

Seriously, yes, we would like a SAP DB data provider.

You could port their Java/JDBC provider to a C#/.NET Data Provider.

The source and binary to the JDBC provider for SAP DB is available for
download at:
http://www.sapdb.org/sap_db_jdbc.htm

The SAP DB JDBC provider is licensed under the LGPL 2.1, so if you base your
.NET data provider on this JDBC, it will be JDBC too.

Daniel

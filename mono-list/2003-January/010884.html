Hello,

> I think the question would be better worded as:
> 
> Does Mono support COM Interop?

It does not at this point, and there are no plans at this point to do so
(unless someone steps up to write the code for it).

COM has a small priority for us, as there is no standard and complete
COM implementation on Unix, which is our primary interest.  

Doing a full COM implementation is out of the scope of Mono at this
point.

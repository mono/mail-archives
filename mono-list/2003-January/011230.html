Tracy, the mod_mono apache2 module I don't believe is currently
available with the Mono RPM's, you can get the source code for mod_mono
(and/or a binary build) from http://www.apacheworld.org/modmono/ that
site also houses some very usefull installation instructions.
Optionally, the sources are available via cvs the cvs module's name is
mod_mono. Hope that helps. Enjoy!

There was a daily build server set up a while back, before Mono became
self-hosting.

Now that it's possible to build mcs using Mono itself, it seems that the
better solution is for packagers to stick to releases. mcs is easy
enough to build yourself, and working with CVS encourages enthusiasts to
contribute patches and play a more active role in debugging.

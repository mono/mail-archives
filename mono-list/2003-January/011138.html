Hello!

I've started a small configuration tool for the mono4windows distribution.
May work on Linux, too. It depends on GTK#.

Currently it can:

* Switch the Native Windows / Native Windows XP themes
* Read and show the library wrapping in /etc/config

Todo:

* Make it possible to add dlls to be wrapped
* Make it possible to remove entries
* Add machine.config configuration

Screenshot: http://www.sport-huettn.de/mono/mconf.png
Download: http://www.sport-huettn.de/mono/mconf-0.1.zip

It will be included in mono 0.18 setup, when it arrives.

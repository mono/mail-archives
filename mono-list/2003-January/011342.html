On 01/17/03 Janson, Eric wrote:
> it's not actually supposed to work on Unix.  If not, is that in the plans?
> If so, any idea of a timeline?  It's not clear (to me, anyway...not a big
> Unix expert) from the mono-FAQ...it doesn't mention IRIX, AIX or HP-UX, but
> it does seem that it's supposed to work on Solaris, anyway?!

Currently there is some support for Solaris (using the interpreter),
the build process on that platform may have been broken, though.
A detailed bug report could help us fixing it.
AIX could be supported with mostly the same code as MacOSX, but,
unless someone tries it and sends patches, I guess it won't compile
there.
HP-UX is waiting for someone to begin the port, see
http://www.go-mono.com/porting.html for instructions on starting the
port.

Judging from your list of things you want to do, it sounds like you can
help with the documentation; see:

	http://lists.ximian.com/mailman/listinfo/mono-docs-list

They are currently working on general documentation, HOWTOs, and
anything else that is needed.  I'm sure they could use additional help.

Sign up for the mailing list, check out the monkeyguide CVS module, and
help with the documentation effort.  You can see the current monkeyguide
contents at:

	http://go-mono.com/tutorial/html/en/

Finally, don't be intimidated and don't be afraid to ask questions. 
Your skills are certainly not insignificant; we need all the help we can
get.

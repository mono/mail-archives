Just a quick announcement that 0.65 of the ByteFX.Data ADO.Net provider
library is out on sourceforge.  There are three files available.  One is
a Mono compiled binary, one is a VS.Net compiled binary, and the third
is the sources.  
 
This build is mono-compilable and mono-compatible.
 
Note that in the Mono compiled package, on the ByteFX.Data assembly is
Mono compiled.
Also, note that the PostgreSQL driver in the package is just starting
and not ready to really be used.
 
Also note that this release is not very well tested but did pass some
basic tests before being released.  
 
As always, any help on development or testing greatly appreciated.  
 
The SourceForge page is http://www.sourceforge.net/projects/mysqlnet
 
Enjoy!

Reggie Burnett
ByteFX, Inc.

On 05/07/03 Jeroen Janssen wrote:
> I was wondering, I'm trying to use the precompiled mode, but I get an 
> error about 'as'. Is this a missing tool in mono/windows?

as is the GNU assembler, you need it to use the aot compiler.
Patches are welcome to make it use a different assembler if needed on
windows.
Note that loading an AOT-compile assembly won't work yet on windows
systems, some win32 programmer will need to provide a replacement for
the mprotect call in mini/aot.c and test it.

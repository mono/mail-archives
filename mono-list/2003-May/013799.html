Hi,

I run an open source project called Code Connector (GNU GPL), which is a
distributed development environment providing project management tools,
mirrored text editing and chat capabilities. I have recently made the
decision to develop Code Connector in Mono/C#/GTK# after bouncing
between Java and C/C++, and doing some testing with the different
components Code Connector will need from Mono/C#/GTK#.

I am looking for dedicated developers to join development that are
passionate about their work and communicate well. For now I am the sole
developer, but am looking to bring on at least a few more developers.

The project is in late planning stages and I have been making productive
headway with the transition to C#. 

If you are interested or have any questions please feel free to contact
me. I would love to hear from you.

Our website is here: http://codeconnector.sourceforge.net/.

Thanks!

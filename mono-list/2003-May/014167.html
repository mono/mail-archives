Finally, a question I can help someone with !

I found that the following URL's have the required RedHat8.0 RPMs that are
needed.

ftp://planetmirror.com/pub/ximian/ximian-evolution-beta/redhat-80-i386/ )
ftp://rpmfind.net/linux/ximian/evolution-devel-snapshot/redhat-80-i386/
ftp://planetmirror.com/pub/ximian/evolution-devel-snapshot/redhat-80-i386/

If you install in the following order, all the required dependancies should
be OK

[root@Powerdog evolution]# rpm -ivh bzip2-devel-1.0.2-5.i386.rpm
[root@Powerdog evolution]# rpm -ivh
libgnomeprint22-devel-2.2.1.1-1.ximian.1.i386.rpm
[root@Powerdog evolution]# rpm -ivh
libgnomeprintui22-devel-2.2.1.1-1.ximian.1.i386.rpm
[root@Powerdog evolution]# rpm -ivh
libgal2.0-devel-1.99.2.99.0.200303220701-0.snap.ximian.1.i386.rpm
[root@Powerdog evolution]# rpm -ivh
libgal2.0-modules-1.99.2.99.0.200303220701-0.snap.ximian.1.i386.rpm
[root@Powerdog evolution]# rpm -ivh
libgtkhtml3.0_0-3.0.1.0.200303220701-0.snap.ximian.1.i386.rpm
[root@Powerdog evolution]# rpm -ivh
gtkhtml3.0-3.0.1.0.200303220701-0.snap.ximian.1.i386.rpm
[root@Powerdog evolution]# rpm -ivh
gtkhtml3.0-devel-3.0.1.0.200303220701-0.snap.ximian.1.i386.rpm

Just be careful with the version numbers, there are a few floating around
that have similar version numbers ( but don't seem to provide what you
require )

Also check whether you have libgal2.0 installed already ( you may do, but
you may not )

If you are installing for RedHat 9.0, there are similar (but more up to date
RPMs) on the ftp sites listed above.  Some RPMs are also available on the
RedHat CD's as well ( or where ever you get your RedHat installation from )
so check for them there as well.

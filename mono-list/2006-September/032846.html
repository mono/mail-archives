Ok, try this:
export LD_LIBRARY_PATH=/opt/mono-1.1.17.1/lib
then run your application.
Alternative:
Add this line to your /etc/ld.so.conf file:
/opt/mono-1.1.17.1/lib
Run ldconfig and run your application again.

Hi,

I just started using C# with mono recently, and I have found a couple of
books very helpful.  The first one is Programming C# which is published by
O'Reilly, and for more mono specific applications, Mono: A Developer's
Notebook also by O'Reilly.  The latter has been exceedingly useful to me.
There are a couple of good websites too, if you haven't come across them
yet: http://msdn2.microsoft.com/en-us/library/default.aspx (The MSDN
library) and http://www.go-mono.com/docs (and the rest of the mono website -
there are various tutorials here).

Good luck!

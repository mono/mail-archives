Ahha.  That's out of date, as far as the --update command goes.  The new 
program to do that is monodocer, and I guess it hasn't yet made it into 
an RPM release.  You can either install Monodoc from SVN, or just 
compile Monodoc from SVN and run the built monodocer.exe program 
(without installing it) in the /monodoc/tools directory.  To use it:

mono monodocer.exe --assembly YourLib.dll --path path/To/Xml/Files

If you install Monodoc, you can also edit those files within Monodoc with:

monodoc --edit path/To/Xml/Files

Boo 0.5.2 was released by Rodrigo B. de Oliveira.

 From http://boo.codehaus.org/
"Boo is a new object oriented statically typed programming language for 
the Common Language Infrastructure with a python inspired syntax and a 
special focus on language and compiler extensibility."

The sources and binaries are available here: 
http://dist.codehaus.org/boo/distributions/?M=D

Changes since 0.5 include support for multidimensional arrays, 
properties with multiple indices, rawArrayIndexing for faster array 
access, and an implicit duck typing option (-ducky).  For more details 
see the boo website and this announcement from the boo mailing list:
http://groups-beta.google.com/group/boolang/messages/e9ec1450238d2780,9454f438411e5867,8e98b997f1b29dc3,b8a840c149a033ee,91ec830bc1b7244c,36903532a1a03c4e?thread_id=04aa6bf9afaeac22&mode=thread&noheader=1#doc_e9ec1450238d2780

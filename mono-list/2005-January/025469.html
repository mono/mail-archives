IClonable interface
http://msdn.microsoft.com/library/default.asp?url=/library/en-us/cpref/html/frlrfsystemicloneableclasstopic.asp

implement the interface in your class, so you can do
myClass c1 = new myClass();
myClass c2 = c1.Clone();

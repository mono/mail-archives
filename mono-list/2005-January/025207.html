I've put together a web page on how I built mono 1.0.5 from source on
Fedora Core 3.  It can be found at:

http://webpages.charter.net/mdeggers/howto-mono-fedora.html

Paul Johnson has put together an excellent web page for building mono in
various incantations (from svn, recovering from a partial rpm install,
from tarballs, etc.).  You can find his writeup here:

http://www.all-the-johnsons.co.uk/

>From Paul's writing, it looks like there are significant advantages to
running the latest dev version of mono.

I have gotten some feedback on my page that I've not had time to add.
Please note that there are dependencies between the various portions of
mono.  In particular, the gtk stuff needs to be installed before monodoc.
And monodoc needs to be installed before monodevelop.

As for finding your .pc files, I mention that in the 2.1 IMPORTANT - Build
Environment section on my page.

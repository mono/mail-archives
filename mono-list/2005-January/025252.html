Try searching for mono-core, I found that in the dag repository, I
noticed that you left out the [dag] header for the dag repo in your
yum.conf file.  I don't recall where the gpg key is on the dag server,
if you search the www.fedoraforum.org you should be able to find some
info, or you could disable the gpgcheck by setting it to 0 for dag.

[dag]
name=Dag APT Repository
baseurl=http://dag.freshrpms.net/fedora/$releasever/en/$basearch/dag/
 http://dag.atrpms.net/fedora/$releasever/en/$basearch/dag/
 http://ftp.heanet.ie/pub/freshrpms/pub/dag/fedora/$releasever/en/$basearch/dag/
gpgcheck=1

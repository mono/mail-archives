I'm running Mono on SuSE 9.2 Professional.  You should not have to download
anything from the go-mono web site.  All the packages you need are on the
SuSE 9.2 Professional disks.  If I were you, I'd simply reinstall
everything.  The only thing you need to configure is Apache.

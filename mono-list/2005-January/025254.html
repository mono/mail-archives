Hey,

   Thanks to Daniel L=F3pez from BitRock we now have an installer for Mono
1.0.5 that works on various platforms: RH9, Fedora Core 1/2/3, Suse
Enteprise 9, Novell Linux Desktop, Suse Linux 9.

   This is a test package:

http://primates.ximian.com/~miguel/tmp/monobundle-1.0.5-installer.bin

    To install, do:

	* Download the above package.

	* From a terminal window, go to the directory where the
	  file is downloaded and type:

		chmod +x monobundle-1.0.5-installer.bin

	* Run:

		./monobundle-1.0.5-installer.bin

    We are interested in collecting your feedback.

    Downside: this will not register Mono with the RPM database on the
system.

Miguel.

The F# code here:

  type A() =
    inherit TextBox()

just defines a new class "A" that inherits from TextBox and implicitly 
provides a zero-argument constructor that calls TextBox's.

The equivalent C# is something like:

  public class A : TextBox
  {
    public new A()
    {
    }
  }

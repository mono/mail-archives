Peter Hagen wrote:
> Hi
> 
> I'm testing my .net 1.1 website on the new Mono 2.0.1.

...

> I'm sure that my project (Monodevelop) is build as .net 1.1, but somehow
> the '$' is being used. Is this a bug?

You're probably running your app with xsp2/mod_mono2 under the
2.0 profile.

I'm not sure if this change has made into 2.0.1, but on SVN HEAD
all scripts were changed to run under the 2.0 profile by default:

xsp  -> .NET 2.0
xsp1 -> .NET 1.1
xsp2 -> .NET 2.0

So you might be running on the wrong profile.

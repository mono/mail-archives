allows for a type to be handled by Mono (f.ex. .SSS)

references to another assembly and/or XML files, like so:

<sss_go>
<node>
 <id>1>/id>
 <type>Some.Namespace.Class.X.1</type>
 <next_node>2</next_node>
</node>

 <type>some.xml.file</type>
 <next_node>2</next_node>

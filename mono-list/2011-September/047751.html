I'm mainly using VS 2010 as an editor with the occasional vim mixed in. I map
a drive letter to my home directory on a linux box where our repositories
are checked out via git. The main reason I work in Windows is because we
also use Unity3D.

Our build process is executed in Linux via some simple Makefiles that invoke
xbuild. It also has targets for running nunit-console for unit tests.

I really like the progress they've made in MonoDevelop 2.6 and may begin to
use it more instead of VS.

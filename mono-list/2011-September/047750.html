Our product is a "cloud platform".
We develop the web/server part of the system on Linux using Monodevelop. 
It give us the ability of debugging on linux.
The graphic interface part (some clients that runs on windows) are 
developed using Visual Studio on Windows.

We also have some exe/dll that works on both the part of the system and 
these are developed using Visual Studio on Windows and runs on linux/mac 
too.

No, SWF is not a wrapper for X widgets. All controls have been implemented 
in fully managed code, using System.Drawing for drawing the controls.

If you can test your code with our WinForms implementation and find bugs, 
please log them at bugzilla.ximian.com. Now is the perfect time, we're deep 
in bugfixing mode.

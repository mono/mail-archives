Are you talking about a way to specify another install dir ?

You can do this using the prefix option:

Unix/Lnux:

$ make install prefix=/usr/local

Windows

$ make install prefix=c:\NAnt

-or-

$ nmake -f Makefile.nmake install prefix=c:\NAnt

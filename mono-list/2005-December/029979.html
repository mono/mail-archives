Hi,

After testing Apache+Win32 in a Windows box I decided to install it in
another Wndows box. Then I experienced a problem: "The Apache service
named  reported the following error:...". It seems that mod_mono module
could not be found in Apache modules folder. But it was there. And in
event viewer there was an entry saying that http.conf had an error, but
there was any error in config file.

Then I checked dependencies for mod_mono.so (win32). mod_mono could not
found some libraries: LIBAPR.DLL, LIBHTTPD.DLL, MSVCR71.DLL. I added the
paths to those libraries to system path. The dependencies problem is solved.

But, anyway, I cannot start Apache because the problem remains:

[Event viewer entries]

The Apache service named  reported the following error:
>>> Cannot load C:/apachefriends/xampp/apache/modules/mod_mono.so into
server: No se puede encontrar el m\xf3dulo especificado.     .

The Apache service named  reported the following error:
>>> Syntax error on line 199 of
C:/apachefriends/xampp/apache/conf/httpd.conf:     .


Any idea?

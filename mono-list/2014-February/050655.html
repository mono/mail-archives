For the sake of completenes, in cas anyone needs it:
I solved the problem today (with v3.2.3) by
* adding "%make_install %{__make} install DESTDIR=%{?buildroot}" to 
~/.rpmmacros
* building with the --target switch: rpmbuild --rebuild --target i686 ~/mono-
core-3.2.3-135.46.src.rpm

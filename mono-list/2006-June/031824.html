http://www.mono-project.com/Downloads has been updated to include links
to the 1.1.13.8 (stable) release.

There are also yum/yast/opencarpet repositories for each distro for the
stable release.

** Note ** : You'll need to update your yum and opencarpet repository
information for your distro.  I'll leave the old repository location
standing until our next release, but then they will be removed.  This is
so that if someone is subscribed, they won't think their system is up to
date when actually the repository hasn't been updated.

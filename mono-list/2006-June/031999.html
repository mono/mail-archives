Well, all I know is that, as I understand it, the mono-list is for
questions relating specifically to the use of mono -- possible bugs,
configuration questions, differences with .NET -- and not so much for
generic C# questions. That said, there are regularly questions about
peripheral things and occasional posts about completely unrelated things.
It isn't uncommon for project announcements to be sent to the mono-list
simply because the code is capable of running on mono. So, don't feel bad
for having posted here :-) In any event, I don't know of a better place to
put the discussion, and you're likely to get pertinent responses here,
especially in the area of portability.

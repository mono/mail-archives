Hello Paulo,

Short said, Indigo is the upcoming communications platform to be
release with Longhorn (will be available to XP SP2 and Win2003
server).
Indigo is a unified programming model, so it unifies all other
communication options currently available in .NET like ASMX,
Transactions, etc.
Since discussing Indigo is not the goal of this list, and I suggest
you to read the plenty of information available at Microsoft's Indigo
web page:

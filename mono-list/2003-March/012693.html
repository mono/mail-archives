March 3rd, 2003
 
Qt# 0.7 - A cross-platform GUI toolkit for Mono and Portable.Net, has been 
released.
 
Download for source, binaries, tutorials and documentation:
 
http://sourceforge.net/project/showfiles.php?group_id=48999
 
Information and screenshots:
 
http://qtcsharp.sourceforge.net
 
This package contains Qt#, version 0.7. This is a maintenance release. The 
main improvements over 0.6 are an easier build system, improvements in the 
demo apps, unicode support, and support for slots that take a QString 
parameter.
  
Changes since 0.6
 
*  Add support for dynamic resize to the Mandlebrot example

*  Add overloaded version of QWidget.SetSizePolicy

*  Use LPWStr marshalling for unicode strings

*  Fix versioning and install target for libqtsharp and general improvements 
to the build system

*  Added support for slot connection in constructor in QToolButton

*  Added support for unicode QString <--> string conversion

*  Add QGL* support

*  Make SLOT and SIGNAL remove extraneous spaces

*  Add support for slots that take a single QString param

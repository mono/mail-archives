On 02/28/03 Guy wrote:
> 	im not sure if im targeting the right person with this email, im 
> 	just wondering how hard it will be to port Mono (or only what's necessary 
> to fully use C#) to BeOS.

It depends mainly on two things: does BeOS have pthread compatible
thread library? Does libgc work on BeOS? If the answer to both questions
is true, porting likely requires half an hour of work (configure.in,
mainly).
If libgc was not ported to BeOS, you can still run many programs if you
have memory available, so it's not really required to get started.

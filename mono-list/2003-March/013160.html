Assuming you have installed mono on that linux system,
and the .NET application doesn't use any classes which aren't finished =
or present in the Mono framework,
(especially System.Windows.Forms) it should work just fine with xcopy =
deployment.

Meaning you put on your hard disk just by copying them, no other =
installation required.
You can then run it by given the command: mono nameofapplication.exe

Hope this helps.
Timothy.

The web status pages are created with a makefile that is in
xxx/mono/doc/web. You need mono assemblies in xxx/mcs/class/lib and
retrieving the previous version of the makefile in mono/doc/web. Then
run make and you'll get the pages (corlib will fail. If it does not tell
us how you did it ;-).

>>Could you tell me where I can refer the code size information 
>>for all >>IL instructions?
The code size of each instruction is specified in the OPDEF 
macros. You can see this in the ECMA spec document and in the 
opcode.def file in the folder named "cil" in the mono source 
code.You will find info about the OPDEF macro args in Partiotion 
V, Annex C of the ECMA spec document

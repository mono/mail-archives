Indeed it is. To find out how special try cattting some of the files in
there, e.g.


cat /proc/ksyms (kernel symbols)
cat /proc/modules (loaded modules)
cat /proc/uptime (your uptime)

and so forth. It's a special filesystem that is used to provide
infomation from various drivers/etc back to userland. Because it is a
filesystem you can access this kind of data from anywhere in Linux e.g.
shell scripts, from Perl, etc. You can register your own data and
entries for /proc too.

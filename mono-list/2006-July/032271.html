Hello everybody,

Appomattox, a project I have been working on for a while now, has had
its first release.

About:

Appomattox is both a library and an application. As a library, it
provides a set of easy to use mapping components for Gtk# applications.
Built upon these libraries is the application called WorkBench.
WorkBench is currently a spatial data viewer, but with aims to be a
full-featured Geographic Information System (GIS).

Appomattox works on both Mono and MS.Net (Windows).  For more
information and links to download, check out the release notes:

http://www.appomattox-project.org/Release_Notes_-_0.1

And here are some pretty screenshots:

http://www.appomattox-project.org/Screenshots

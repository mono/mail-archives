	Hey,
	
	The VaryByParam="None" semantics are not implemented in Mono.
	Use a true parameter name or "*" instead.
	
	Instead of Location="Server", use "ServerAndClient".
	
	Please file a bug at http://mono-project.com/Bugs.
	
	Robert

Hello,

> I was wondering as going through the "2.0 release notes"... Does Mono 2.0
> has (or will have) a complete ASP 2.0 support, including those Webparts ?
> Does/Will it include the compacting GC ?

Mono 2.0 does not include WebParts implementation. Compacting GC is not
enabled in configure script by default (--with-gc=sgen). It in general 
works but is not stable yet. See
http://mono-project.com/Mono_Project_Roadmap

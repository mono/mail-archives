When you run your program, it does exactly what you
told it to do, i.e. { } :-) 

To run NUnit tests, you execute NUnit, not the tests.
You can run
   nunit-console mytests.dll
   nunit mytests.dll
OR nunit and then open the test assembly from the menu.

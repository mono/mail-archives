That's not a crash. ConflictOption is a .NET 2.0 feature but you're
compiling with mcs, targeting the 1.1 runtime. Gmcs is the proper
compiler.

Since mod_mono defaults to 1.1, it first has to be configured for 2.0.
See http://www.mono-project.com/Mod_mono

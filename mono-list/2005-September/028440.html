> is it possible to launch install  .Net Framework 1.1 under wine + mono,
> (I've heard it requires to add several dlls from the native windows). will

Maybe you are a bit confused over what Mono is? Mono is a
cross-platform runtime for the .NET Framework. Therefore you do not
"install .Net Framework 1.1" on any system, you would use Mono
instead. In this case, Mono replaces the .Net Framework 1.1, and it
does not require wine in any way (I think!)

Hope that helped a little...

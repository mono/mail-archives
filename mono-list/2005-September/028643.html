Joe Audette has written something about the hosting service he uses on his weblog:
http://www.joeaudette.com/BlogView.aspx?pageid=101&ItemID=152&mid=104

Right now I am in the process of getting a cheap linux root server because I want complete control (and need to open a lot of tcp/ip connections on non-http port). I will see if the machine spec is sufficient for handling my Mono application.

Generally I think Mono could be very good for web hosting in the future. .Net is built with hosting in mind and Mono is very linux-friendly. Maybe Mono will be what Java could never be. It's just that the web layer has been rewritten recently. And I don't know of any big application that is running on Mono right now. There is maybe the concern of larger companies that ASP.Net is owned by Microsoft and not an open standard.

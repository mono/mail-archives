You need to figure out what libraries your application depends on
(probably the same as the ones used to compile the app) and then
specify the packages which contain those assemblies as dependencies of
your application. Whatever tool you use to generate your debian
packages will have a way of adding those dependencies so that when you
install your app via apt (or whatever) it will automatically install
the dependencies and just the dependencies. This is the best option.

If you are not generating a debian package then you can either
recommend installing 'mono-complete' as the simple/effective option or
supply a list of the packages your users will have to install and tell
them to install them first.

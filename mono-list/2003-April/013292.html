It's fine, though mono-devel-list is a better place.
Also, please create patches with the -u option to diff and resend.
If you're using cvs diff, simply put:

==cut cut==
diff -u -p -N
==cut cut==

inside your .cvsrc file.
Diff files in ed format don't work well when the sources get
changed and they are less readable.

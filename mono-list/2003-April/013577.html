Yes, we have such a project in NAntContrib (http://nantcontrib.sf.net). Here
is a little description on the nant wiki
(http://nant.sourceforge.net/wiki/index.php/SLiNgshoT).

SLiNgshoT generates both nant, and nmake files from vs.net solution files
(not project files).

In addition we have added a solution task which takes a vs.net
solution/project and does the right thing, without using or requiring vs.net
(current in nant cvs tree, but not in any release). In addition there are
other people who have written xsl tranformation for project files.

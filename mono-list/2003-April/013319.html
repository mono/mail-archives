There existing two projects:

1. IKVM on which Andrew and Varga have pointed out.
IKVM (http://weblog.ikvm.net/) is a JVM with JITter, which starts Java-Code 
on top of .NET. The advantage of IKVM is, that you can access with any 
.NET-language Java-Classes (or was it the other way arround, that it is 
possible to use any .NET-Assembly in Java? I don know)

2. JILC, which is a native Java to .NET-Compiler
JILC (http://jilc.sourceforge.net/) was started by Gaurav Vaish. But because 
lack of time he have stopped his work on it.
The idea of JILC is, to convert Java-Binaries to .NET-Binaries.

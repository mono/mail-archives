The short answer is: not yet.

Previously (~2 weeks ago), the only part of mono that built under OS X
was the interpreter.  That had several "issues" with running proper
code, and there were also build issues, P/Invoke issues, and other
random problems.  IIRC, the interpreter couldn't build mcs, the C#
compiler.

Now, the situation is a little better, if only because `mini', the new
JIT engine, has been checked in and has some Power PC support.  However,
I don't know how complete the PPC support is (I believe it is somewhat
incomplete, though I could be wrong), and the previous issues, to the
best of my knowledge, still exist.

Check the archives for more information.

However, I'm sure any help in getting Mono to run on PPC would be
appreciated. :-)

 - Jon

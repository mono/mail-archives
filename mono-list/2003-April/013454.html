I have this portion of shell script that I run every time I upgrade wine
package, because debian's wine package updates binfmt each time wine is
installed.

#update-binfmts --install mono /usr/local/bin/mono --magic 'MZ'
update-binfmts --install mini /usr/local/bin/mini --magic 'MZ'
update-binfmts --disable wine
update-binfmts --disable mono

With this code I enable mini, uncommenting the first line and erasing
the last, mono is activated...

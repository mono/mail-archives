You can use Corba to access this information.

For .NET there is a free Corba implementation called Remoting Corba.
It's project homepage is http://remoting-corba.sourceforge.net

For access Wombat methods through Corba you have to use corba callbacks
(I think). This aren't  implemented (yet) in the actual stable version,
but are spected to be implemented in the next release.

Hello everyone,

The second beta version (0.1.1) of monotooth is now available in github.comvia
http://github.com/locusf/monotooth/commits/0.1.1. The project sources also
reside in 'master' branch.

If someone doesn't remember / know monotooth, it is a library that allows
.NET/Mono developers use bluetooth in their projects. The main aim of the
project is to provide a clean design and easy to use API for bluetooth
development. Documentation is also (or at least tries) mean to be verbose,
so that even beginners can use this library.

For more information, see http://monotooth.sourceforge.net for more
information. This release (0.1.1) is no longer packaged since github offers
it for free.

Release highlights:

-Inquiry works on windows (so consider this to be an Alpha release in
Windows), but only with the native (ie. not via custom software) interface.
-Linux inquiry has been improved not to use custom library.

With Regards,

Aleksi Suomalainen

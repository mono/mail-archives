On Sun, 2008-04-06 at 23:46 +0100, Derek Ekins wrote:
> I am trying to compile gladeui-sharp and am getting the following
> error:
> 
> checking for GLADEUI... configure: error: Package requirements
> (libgladeui-1.0 >= 2.90.0) were not met:
> No package 'libgladeui-1.0' found
> 
> I can't seem to find any info about this.
> What am I missing here?

Since gladeui-sharp depends on the C gladeui library you need to install
the C library development package for your distribution... in Debian or
Ubuntu you have libgladeui-1.1-dev for SuSE or Fedora you should have
something called libgladeui-1.0-devel.

You can always check if you have that package in this directory:

/usr/lib/pkgconfig/

There you must find(if you have it installed) a file named
libgladeui-1.0.pc, and inside that file the version of the library.

Hello:

Firebird ADO.NET Data Provider 1.5.2 available for download.


Changes in this release:

1.5.2 ( 2004-03-21 )
----- - ---- -- -- -

- Bug fixes (Please review the changelog for details) :

	- Fix for the GetBytes and GetChars methods of the FbDataReader class.

	- Commit implicit transactions of active commands when
	closing the connection.

	- Fix for delete stored proc execution using the DataAdapter.

- Varchars are now sent trimmed over the wire.

         ---------------------------------------------------

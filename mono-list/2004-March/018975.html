hmm, I think you will do not want not use glade to store the images.  At
least Glade# doesn't handle the relative paths as you noticed.  So add
an empty Gtk.Image in glade, and then in the code set the
Gtk.Image.Pixbuf property.

[Glade.Widget] Image image;
...
image.Pixbuf = new Pixbuf (null, "icon.png");

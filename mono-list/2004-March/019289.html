Hi all,

Npgsql 0.6rc2 is released.

Npgsql is a .net data provider for postgresql in 100% managed code.
More information can be found at:
http://gborg.postgresql.org/project/npgsql
http://www.go-mono.com/postgresql.html (Thanks Jaroslaw Kowalski :) )


Release notes:

This release candidate has mainly bug fixes. The most important bug fix 
is the ssl support which is working again.

The addition of SSL support added some server certificates validation 
callback functions which can be implemented by Npgsql users. Details can 
be found in the Npgsql project homepage. Npgsql provides a default 
implementation which allows users to use test certificates. Thanks 
Carlos Guzman for providing information about this callbacks.

This is probably the latest RC in the 0.6 stream.

Thanks Glen Parker (glenebob@nwlink.com) and Sami Kuhmonen (sami@iqs.fi)
for patches and tips to improve Npgsql.



Both cvs repositories: gborg and mono are updated and synched with the 
latest code.

Please, get and test it and please fill any bugs you find.

You can use both bug report system:

gborg: http://gborg.postgresql.org/project/npgsql/bugs/buglist.php
mono: http://bugzilla.ximian.com/


You can get the release at:
http://gborg.postgresql.org/project/npgsql/download/download.php?branch=devel

Also, a daily build of Npgsql can be obtained from the mono daily builds:
http://www.go-mono.com/daily/

just grab the latest monolite-xxx.tar.gz.


Thank you very much for all feedback received.

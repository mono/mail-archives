Hey everyone!

Eto.Forms (a cross platform UI toolkit) v2.2 is now released!

Some of the main highlights:

- New version of the XS and VS addins with xaml autocomplete and visual
designer
- Xaml updated to use  Portable.Xaml
<https://github.com/cwensley/Portable.Xaml>  , which fixes a bunch of bugs
with mono's System.Xaml and is now PCL.
- With an updated Xaml engine, markup extensions like {Binding MyProperty},
{Resource ..}, etc are now supported.
- New PropertyCell & CustomCell
- Many other minor enhancements and bug fixes

Read more about the new release  here
<https://github.com/picoe/Eto/releases/tag/2.2.0>  .

Get started with the XS/VS addins  here
<https://github.com/picoe/Eto/wiki/Quick-Start>  .

Enjoy!

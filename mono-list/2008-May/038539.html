Hi,

There has been a slight change in the repositories for Mono:Community.

All of the distributions available here:

http://download.opensuse.org/repositories/Mono:/Community/

used to be built against the Mono contained in the 'Mono' namespace in
OBS.

Those repositories are now built against whatever version of Mono ships
with the distro.

There are a new set of repositories labeled with "+Mono" that are built
against the "Mono" namespace from OBS.

This was done so that we could add "Mono:Community" as an optional
repository for the first boot after an install of opensuse 11.0.

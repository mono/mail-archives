Exactly like you build and install any other Linux program.

1. Download the sources.  <http://ftp.novell.com/pub/mono/sources-stable/>
2. Extract them.
3. ./configure
3a. Wait until it complains that it can't find XYZ.
3b. Go install XYZ.
3c. Go back to 3.
4. make
5. su
6. make install

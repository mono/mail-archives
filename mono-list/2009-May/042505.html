Mono 2.4 Regex doesn't seem to support character class subtraction. 

As a simple example, the Regex class '[a-z-[d-f]]' should be the same as
'[a-cg-z]'.
Mono 2.4 interprets it as the class '[a-z-[d-f]', followed by a single ']'.

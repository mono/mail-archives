It would be nice to use the System.Web implementation from Microsoft, but it will require work to isolate the Windows dependencies from the rest of the code.

The Xamarin team at Microsoft is focused on mobile, and we are not likely going to work on this area of the code, but we could assist contributors that would like to contribute to the replacement of key parts of System.Web with the Microsoft implementation.

In preparation for the next stable Banshee release, I released the latest (and first publicly available) Mono.Zeroconf, a Mono/.NET library that supports both Bonjour and Avahi on Linux, Windows, and OS X.

If you're writing a Mono/.NET application and need to perform zeroconf operations, I highly recommend trying Mono.Zeroconf. Using Mono.Zeroconf immediately makes that portion of your application cross platform.

More information: http://mono-project.com/Mono.Zeroconf

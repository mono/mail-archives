Hi,

I'm new to the list but I just want to report that I was able to render
.NET code in apache as an experiment simply by downloading the
.NET sdk and installing it on win2k, then downloading and installing
FoxServ 3.0 Beta 3 and then downloading Alpha of PHP w/ the Zend
Engine 2, unzipped it and copied the .dll into Foxserv in two places.

The Zend Engine 2 is improved for working with other languages like
.NET.

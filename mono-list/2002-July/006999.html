1.1 due out later this year, 2.0 first beta January 2003, second beta in the
summer, release in December 2003.
See quote from Miguel below!

http://www.eweek.com/article2/0,3959,373783,00.asp


Some .Net developers told eWeek that .Net still has a long way to go. ".Net
has some really good things in it, but they also have some very broken APIs
built into it. I would like to see Microsoft fix those APIs," said Miguel de
Icaza, co-founder and chief technology officer of Boston-based Ximian Inc.,
which is developing Mono, an open-source version of .Net Framework that will
run on Linux.

"The class libraries seem to have been rushed out the door before they were
tested enough, and it shows in various areas," de Icaza said. The problems
range "from simple inconsistencies in some database classes to fundamental
problems with the class libraries."

Hi guys,

MCS can now compile the corlib test suite on GNU/Linux :-)

There's a MCS command file

        mcs/class/corlib/Test/corlib_linux_test.args

to compile it so you can just say

        mcs @corlib_linux_test.args

in that directory.

I've checked the generated code with pedump but I didn't run the test
suite yet.

I've now built Mono 1.2.2 & libgdiplus for Fedora Core. This version
supersedes the 1.2.1 release made at the middle of last month and comes
with the same health warnings...

     1. This is NOT an official Fedora Core release version; I don't
        work for Red Hat. Any problems you find with this should be
        directed to the Mono bods and NOT the RedHat people.
     2. As soon as Fedora Core gets 1.2.2 into rawhide, these will
        vanish.
     3. If you are using FC 5, these RPMs **WILL** completely screw with
        anything you have from the official Fedora Core or Fedora Extras
        repositories.
     4. You should not have any problems using it with FC6 or Rawhide
     5. If there is a problem with installing these RPMs, please send
        them to me directly.

I've built them for x86_64 and i386 and have also put online the src
rpms with the modified spec files and any patches. Due to the number of
packages, I've made the mono distribution into a tarball.

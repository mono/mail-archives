Okay, problem solved.
For any of you out there that want to save some hair...

Here is the step by step:
1. download Mono Tools for Visual Studio 2010. It is a zip file.  
2. rename the download file from a *.zip file to a *.vsix file.
3. (this may not be necessary... but close all instances of Visual Studio
2010)
4. double-click the *.vsix file
5. Follow the prompts 

Volia, installed.

Hope this helps someone else.  I got the info form the following link:
http://blogs.msdn.com/b/quanto/archive/2009/05/26/what-is-a-vsix.aspx

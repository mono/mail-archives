You can disregard my original email.

I just built Mono 2.8.1 on a RHEL4 box and, much to my surprise and delight,
it built with no issues at all.  If you have to install any dependencies,
you should be able simply to download directly from Red Hat using "yum". 
This is a very safe operation (it is kind of like running Windows Update but
even less obtrusive).

Mono itself is quite unobtrusive.  It is quite self-contained, user-space
only, and it does not modify any of the system libraries.

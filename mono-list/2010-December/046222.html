Mono runs on FreeBSD 8.1.  I built and installed last night on PC-BSD 8.1
using bsd-sharp ports.
Instructions:
Install PC-BSD
Update Ports
Subversion checkout the bsd-sharp ports
cd into bsd-sharp-read-only/lang/mono
su
make install clean

references:
http://code.google.com/p/bsd-sharp/
http://www.pcbsd.org/content/view/169/11/

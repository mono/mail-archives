Sorry, I did not notice this right away but you are in fact trying to build
the Mono VisualBasic.NET compiler.  That is what "mono-basic" is.  It is not
a simple (basic) version of Mono or anything like that.

You need to have Mono installed before you can build the VB.NET compiler. 
You can find the latest version of Mono here:

http://ftp.novell.com/pub/mono/sources/mono/mono-2.8.1.tar.bz2

Yikes!!!  I see that I am the one that steered you to the basic compiler.

For what it is worth, you can install GIF support on Red Hat with the
following command:

yum install libungif libungif-devel

This should get you past the error you are encountering.

That said, I would not be trying to install libgdiplus-1.9 anywhere these
days.  Mono is up to version 2.8.1 now.  I would not recommend anything
lower than version 2.6.

I got it working - hooray!  Many thanks to everyone for your helpful advice.

In case it's useful to anyone else as a reference, these are the steps that
got me there:
- I updated to the 2.6.7 libgdiplus package
- I set the CPPFLAGS and LDFLAGS environment variables to specify an include
and library folder, respectively (I'm not root, so my libgif is not
installed in the standard place)
- I re-ran the configure script (this time it picked up the gif libraries),
then make, then make install
- I updated my LD_LIBRARY_PATH environment variable so that mono would pick
up libgdiplus.so

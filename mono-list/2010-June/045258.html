On 04.06.2010 09:54, Extreme Developer wrote:

> By deleting gcc-4 and just using gcc-3, I got a huge number of errors
> and when I don’t do that I get only one error as : The -mno-cygwin
> flag has been removed; use a mingw-targeted cross-compilerAnd by
> using /usr/bin/set-gcc-default-3.sh as suggested in that forum I get
> "access denied" !  There are some other approaches mentioned in some
> other forums but non of these advice could help unfortunately.

You may want to re-install cygwin and select only "gcc-core",
"gcc-g++", "gccmingw" because, as you've found out, gcc4 does
not support the "mingw" target.

Alternatively you could set a specific compiler using
the environment as documented in the mkbundle man page:

CC="gcc-3 -mno-cygwin" mkbundle ....

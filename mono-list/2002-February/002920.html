> The C# compiler we will get for free if someone ports the runtime.

> Porting the runtime should take a seasoned programmer a week to write
> (Radek did the PowerPC in about five days on his spare time, while not

Note that the time estimates are for getting the interpreter to work,
but some of the stuff is useful for the jit work later on as well.
There is a document that describes the process in
http://www.go-mono.com/porting.html.

Currently, besides x86, there is support for powerpc (no delegates,
though). Also, arm and sparc code generation are in the works.

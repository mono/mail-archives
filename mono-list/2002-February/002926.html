Here is the thing,

In order to compile mcs (written in c#) you need a c# compiler ;)  Luckily, 
mcs is now capable of compiling itself on windows, but notice you will still 
need a prebuilt mcs.exe to do this.  mcs is close to compiling itself under 
linux, but it is not quite there yet.  When this does happen then I am sure 
you will be able to download a prebuilt mcs binary...  So, if you want to 
compile mcs for now you'll have to use windows, and have to retrieve 
Microsoft's .net platform and either use csc or find a prebuilt copy of mcs 
lying around.

Adam Treat

ps it is true that you can use wine to compile, but you'll still need a 
windows partition and the .net sdk.  cheers.

On Thu, Feb 26, 2009 at 10:03 AM, thebluebulb <crovira220 at gmail.com> wrote:
> Does anyone know how to install Mono on shared hosting plans? I don't have
> access to the root of the server or any of that stuff. Any advice?

If by "Mono" you mean "mod_mono" then it's not possible without access
to Apache's main configuration file, as well as write access to the
directory where Apache modules live.

If you mean installing the rest of Mono, so you can run/compile apps
from the command-line then all you should need is shell access and a C
compiler.

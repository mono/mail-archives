Ive some code that allows you to access G2 library
from MONO.

G2 library allows you to do 2D graphics across
multiple
devices from one front end. more info at
http://g2.sf.net/

Get this package from: 
http://octave-gtk.sourceforge.net/mono-g2/mono-g2.tar.gz

for more information read:
http://octave-gtk.sourceforge.net/mono-g2/

Please find the latest G2-0.70 version diffs & source 
packages of the octave-g2 & mono-g2 in this tarball:
http://octave-gtk.sourceforge.net/mono-g2/g2_addons.tar.gz

RedHat 9, Apache 2.0.40-21, mod_mono-1.0.9 now work. I
hacked mod_mono.c and mod_mono.h to remove references
and code for apr_socket_send and apr_socket_recv. The
Apache APR library has only apr_send and apr_recv.

Now my site is running in cooperation with Apache.

I have issues to use ByteFx with Mono 1.1.6/Linux:
using System.Data;                              // needed?

                                                               "User 
ID=xxx;" + 
                   "Password=xxx;" + 
                                 "Database=db;";

IDbCommand dbcon = new MySqlConnection(connectionString);
Console.WriteLine("connection created");

dbcon.Open();
Console.WriteLine("connection opened");

ByteFX.Data.MySqlClient.MySqlException: Unable to connect to any of the 
specified MySQL hosts

Messages on internet point out to a bad MySQL configuration. It looks

[mysqld]
datadir=/var/lib/mysql

[mysqld_safe]
err-log=/var/log/mysqld/mysqld.log

All GNU\Linux distro can be used to develop program with mono, provided it
had mono framework installed. Major GNU\Linux distro (like openSUSE, ubuntu,
Fedora) had mono framework installed by default, but usually they don't come
with the IDE for developing it (mono). You can easily download the IDE (I
would recommend Monodevelop) from their repositories.

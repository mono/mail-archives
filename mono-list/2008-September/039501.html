IBM mainframe architecture is a little vague...

There is support for s390 and s390x architectures, so it should work  
on System z under Linux.
http://www.mono-project.com/Mono:S390

It should also run on System p under Linux, but only 32-bit currently.
http://www.mono-project.com/Mono:PowerPC

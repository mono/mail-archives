I end up using Manoli syntax highlighter which mini-csharplab earlier 
version uses.
It is simple and has small amout of code, has multi-language syntax 
description classes available and does not use any p/invokes.

In this month there was article on codeproject about using similar 
technique.

Mini-csharplap switched to csharpdevelop code editor component.
This is better since supports auto-complete and folding. However it uses 
p/invokes so cannot used with Mono.

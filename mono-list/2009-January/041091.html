Hey,

Yeah, I'm familiar with those tests, yet I don't place that much faith in
them as a cross language comparison. Some benchmarks use an optimized
version in one language and a very inefficient one in another language which
makes comparing results pretty useless.

The scimark benchmark posted here is a *lot* more useful because it's the
exact same code running on different platforms and versions of the .NET
framework.

Alan.

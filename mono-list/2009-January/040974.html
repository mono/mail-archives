Hi,

just released on Codeplex a simple, portable and easy to use key storage
framework to be used with Microsoft .Net 2.0+ and Mono 2.0+. 

http://www.codeplex.com/KeyStorage http://www.codeplex.com/KeyStorage 

The library wraps the key management solutions already available on the
target platforms.
Its aim is to relieve the application from introducing platform dependent
code to safely store passwords and other secrets. 

Currently supports:

    * DPAPI on Windows
    * GNOME-Keyring on Linux
    * Keychain Services API on Mac OS X

Please see the link above for the project documentation.

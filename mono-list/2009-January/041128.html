Andrus wrote:

> So my idea is to use single platform, Mono in all servers.

It is not practicable at present, unless someone is updating and
testing mod_mono under Windows.

Even then you'll have to deal with mono's partial instability under
Windows.

> Will Apache+mod_mono run OK in older windwses than Vista ?

No.

> Is it reasonable to use XSP instead mod_mono to run asmx web service in
> Windows production servers ?

It's probably easier to install and to set up, but you must find a
way to wrap it as a service and to deal with restarts/crashes.
Those things come for free with ISS (and apache+mod_mono).

> Which is the stable configuration to run Mono asmx web service in
> windows in production server ?

IIS + MS.NET.

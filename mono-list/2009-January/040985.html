There are recipes for mono 2.2-rc1 in OpenEmbedded.  Copy them to your
buglabs build environment as described in:

http://community.buglabs.net/finsprings/posts/19-Pulling-in-an-OE-Recipe

You can check out the OE meta data by:

git clone git://git.openembedded.net/openembedded

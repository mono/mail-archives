I managed to do this by downloading the source tarball from

http://www.hpl.hp.com/personal/Hans_Boehm/gc/
<http://www.hpl.hp.com/personal/Hans_Boehm/gc/> 

And installing it in /usr/src, typing ./configure, make and make install.
After this I needed to symbolically link the 'include' file in the GC6.0
directory to /usr/include so the mono build process could find it. I found
this easier than working with the rpm's, and had some problems getting the
'src' rpm to install at all...

> Below is the source code which is designed to demonstrate

mcs uses the System.Reflection.Emit interface to create the assemblies.
The problem is that the interface doesn't allow the separate creation of 
modules: all the modules need to be created dynamically.
We'll probably need to add a workaround for this issue in the mono
runtime.

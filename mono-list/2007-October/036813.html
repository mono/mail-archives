Hello

In my experience, 1.2.5.1 is ok if you don't need (M$) ajax working

If you need ajax, you need the SVN trunks...by this moment, it is working

In any case, from deselect, remove any "mono" reference you find...
and then set your apt source to "testing"

I also suggest you to clear your /etc/mono*, /etc/xsp* directory, and
place all
the needed configuration on one apache2 configuration file
(don't try apache-mod-mono, use apache2 with mod-mono)

Best regards

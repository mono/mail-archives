Note that there are filenames possible in linux, that are not possible 
under windows. I've got that problem when one file in linux was named 
Aux.php (in some other project). aux is a reserved filename in Windows 
(much like com, lpt and some other ones), and Tortoise cannot create it 
and checkout process breaks.

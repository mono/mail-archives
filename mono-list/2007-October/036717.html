> I would like to know if Mono has different behaviors with different
> Linux distributions ? Issues like better performance, or some
> Distro-dependent bugs (not considering installation problems) ...

Mono is usually tuned to what is available by your distribution, so
having a package built for your distribution is always a better choice
than using a generic Mono (like the one we distribute with the Universal
installer).

We tend to test with Fedora, Debian and SUSE distributions, but it
should work on most distributions.

We have historically had problems with people that are running things
like Gentoo and Slackware due to problems and bugs exposed by heavier
optimizations or enabling some uncommon features in the underlying
operating system.

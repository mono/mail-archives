As some of you have pointed out, there was text on the create account
page that said that by creating an account you were giving them
permission to contact you regarding "Novell products and services."

We agree that you should not have to accept advertisements in order to
file bugs.

Thus, we have corrected the situation. When creating your account,
please go to the following URL:

http://tinyurl.com/389ee6

This URL will automatically opt you out of all advertising.

If you've already created an account, we have set you as opt-out already.


Sorry for the confusion,
Thomas

> installation is a very old, patched, modified, buggy and unmaintained
> version of Bugzilla. The Novell Bugzilla team has offered to take over

> installation and access to various tools that will help our process. 
> 
> As part of this process we need everyone to create Novell.Com accounts. 
> 
> When creating your Novell.Com account, please make sure to use the same

> ensure that your Bugzilla configuration stays the same. 
> 
> Please create your Novell.Com account now, so that you will be able to
> immediately access the Novell Bugzilla when we do the final switchover. 
> 
> To create a Novell.Com account, please go to this URL and fill out the
> form: https://secure-www.novell.com/selfreg/jsp/createSimpleAccount.jsp 
> 
> Important: To link your existing bugs from bugzilla.ximian.com to the
> new bugzilla make sure that you use the same email address in that
> form.

> We appreciate all of the contributions that you have made to mono in the
> Ximian Bugzilla and hope that you will continue to contribute in the

Hello,

> I want to know which is better approach, XSP or mod-mono  on Apache
> server.

It is entirely up to you.

The Apache module has the advantage that you have a single web server
and you can have multiple kinds of applications running on the same host
not only Mono ones.

XSP is a very minimal web server, which is not as feature rich as
Apache.

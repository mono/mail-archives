Wide Margin my Mono based bible application has reached it's first Alpha
release and is ready for testing.

It is written in C# with a GTK# interface and a sqlite back end.

The release announcement can be found here:
http://trampster.blogspot.com/2011/07/announcing-wide-margin.html

And the code and wiki is hosted on bitbucket:
https://bitbucket.org/trampster/widemargin/wiki/Home

The code is licensed under GPL V3

You can install on ubuntu using our PPA by following the instructions found
here:
https://bitbucket.org/trampster/widemargin/wiki/Download/Install

I want to thank the mono team for providing such an awesome platform for
building open source applications.

I would also like to query what it takes to get ones blog added to
monologue?

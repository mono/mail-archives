I think run-time would be more likely (do it as one of your first start up tasks). However, I don't think that would be possible.

You could have a static member variable of one of your main classes set to the encoder you desire. You'll still need to do a run-through of your code to update all the encoder references though. This would also give you the ability to easily (run or compile time) to change the encoder.

The new installer is posted containing 1.1.9.2 and Gtk# 2.3.92.

http://www.go-mono.com/archive/1.1.9.2/windows-installer/0/mono-1.1.9.2-gtksharp-2.3.92-win32-0.exe

Some of of the issues fixed:

-Native binaries and dlls are not statically linked anymore.
-Microsoft's gdiplus is now included.  Note: this will only be installed
on Windows 2000.
-Gtk-wimp theme is included
-rsvg DllImport works
-Gtk# Docs are now included (although monodoc browser still doesn't
work)

Please file bugs for any issues.

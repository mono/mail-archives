Hi Marc,

This code behind setup below worked for me.  If you post the same
someone might spot something.   

in Dept.aspx :

 <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN" >
<%@ Page language="c#" 
	Codebehind=Dept.aspx.cs 
	AutoEventWireup="false"
	Inherits="NonIT.Dept" %>
<%@ Assembly name="Dept" %>
<HTML>
	<HEAD>
.
.
.


in Dept.aspx.cs :

using System;
using System.Data;
using System.Collections;
using System.Configuration;
using ByteFX.Data;
using System.Web;
using System.Web.UI;
using System.Web.UI.WebControls;
using System.Web.UI.HtmlControls;

namespace NonIT
{
	public class Dept : System.Web.UI.Page
	{
		protected System.Web.UI.WebControls.Literal 	 litError;
		protected System.Web.UI.WebControls.Label       lblSelectPortfolio;
.
.
.

I used MonoDevelop to build and a post build script to copy .dll
from ../bin/Debug to ../bin.  

Hope this helps!

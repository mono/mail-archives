Thank's to everybody. I've just corrected the problem.
I'l give my solution in order to avoid another one
having the same problem.

The [Glade.WidgetAttribute] <object type> <name in
Glade>  sentence need to be in the definition part of
a class, not in the implementation one of any method,
I mean, it must define a property of the class wich
may use that widget.

By this way I can develope any app wich use a GUI.

On 05/03/04 HannibAl Bundie wrote:
> After the installation of gc6.2 I don't know what I have to do. I have read 
> that I must rebuilt the mono core environment, but I have installed mono 
> with RPM packages. So, is there any other way to use Boehm collector ?

Mono comes with the Boehm GC included, you currently need to install the
Boehm GC separately only on MacOSX (because there you need a new, fixed,
version which we haven't integrated yet).

> Also, I would be very grateful if I could have an example of the use of 
> Boehm collector with a C# program.

You don't need to do anything special, if you used precompiled rpm mono
packages your are likely already using it.

Thomas,

we don't currently package specifically for RHL3. However, I am told 
the RH9 packages should work.

You'll first want to rpm install all the icu files

then rpm install on the same command line mono-core and mono-posix 
(circular dependency)

then install whichever modules you want or all of them.

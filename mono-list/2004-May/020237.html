Hi,
 
I'm running Mono 0.91 on Windows XP with SP1 installed. Used the
mono-Beta1-win32-2 package to install Mono.
 
When I run an exe created by Mono C# compiler I get the following error
message:
"helloworld.exe - Application Error
 
The application failed to initialize properly (0xc0000135). Click on OK
to terminate the application."
 
The gc.log file contains the following text:
"Needed to allocate blacklisted block at 0x224b000"
 
The application code is as follows:
"using System;
 
class helloworld {
 
 static void Main ()
 {
  Console.WriteLine ("Hello World!");
 }
}"
 
Used the following command to compile the app:
"mcs helloworld.cs"
 
Does anyone have similar problems, or maybe know how to fix them? Must I
file the error on Ximian Bugzilla for C# compiler?

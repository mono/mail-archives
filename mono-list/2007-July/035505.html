Hi,

I use following script for password samba/ldap password change:

$ cat chpass
#! /bin/sh

    {
    echo "$2"
    echo "$2"
    } | /usr/sbin/smbldap-passwd "$1"


here is no magic, just call that chpass with <username> <password>
arguments...

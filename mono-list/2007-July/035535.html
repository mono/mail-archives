Hello,

If you would like to run your site in XSP, you should be using xsp2 (not 
xsp).  For mod_mono, in your apache configuration, make sure you specify 
  the mono server path, and point it at mod-mono-server2; for example:

MonoServerPath "/usr/local/bin/mod-mono-server2"

There is more detailed info on this in the man pages.

-Mike

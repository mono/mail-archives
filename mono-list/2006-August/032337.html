I want to clarify this.

Atsushi is correct that we do not provide support for Phalanger
ourselves.   That is, we probably can not help you with answers on how
to use it.

sure that Phalanger works well with Mono.

developer and ISV both commercial and open source to get their software
running correctly on Mono.

Am personally very excited about the release of this compiler, and we
will try to fix all the issues to allow you to run it.

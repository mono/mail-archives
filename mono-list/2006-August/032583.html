It could be you are experiencing something like this:

http://www.gotmono.com/cgi-bin/yabb/YaBB.pl?board=ASPX;action=display;num=1143225634

I've had this one before. The problem is that mod_mono is using a
different version of the libraries from the one it was compiled with.
I'm not sure why this causes it to SEGFAULT. Make sure that you have
compiled XSP and mod_mono against the latest mono libraries and that
mod_mono can find the latest libraries when it runs.

If you still could not make it work, then sadly your only option is to
*downgrade* to the last stable version. Which was what I did after a
few days of trying :(

Good luck.

Hi All,

OCXT (Open CLI XML Transform), previously available only on
developer.intel.com, is now accessible through Source Forge.  OCXT is a tool
developed by Intel Labs that transforms the ECMA CLI (Common Language
Infrastructure) XML-based class library definition file into C# stubs to
guide developers in creating standards conformant libraries.  OCXT can also
be used to convert the library definition file into MS Word documents.  OCXT
was used to generate the source baseline for OCL
(https://sourceforge.net/projects/ocl).  To access the tool, visit:

http://sourceforge.net/projects/ocxt

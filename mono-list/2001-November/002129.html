Okay... first thing you need to do is learn all about .NET. Mono is based
upon the entire .NET Framework stuff created by Microsoft... so start here

http://msdn.microsoft.com/net/

A good thing to download is the Microsoft .NET Framework SDK. If you want to
learn C# and VB.net get your hands on the Visual Studio .NET release
candidate (it costs about $20) which also contains the Framework, etc.

Hope this helps.

Well, most of these projects are not Mono-related, but they are .NET 
related. Since they might be useful to some Mono developer, here's the list:

Libraries
---------
CSharp Graphics Library
About: The CSharp Graphics Library is a C# binding for OpenGL and SDL. It is 
designed to be a cross-platform graphics library for use with the .NET 
framework.
http://freshmeat.net/projects/csgl/

Project: NetCvsLib
NetCvsLib is a CVS client written entirely in C# for the .NET platform. It 
is implemented as an assembly, and thus can easily be incorporated into 
other projects.
http://sourceforge.net/projects/netcvslib

Project: OpenSmtp.Net
OpenSmtp.net is an open source SMTP component written in C#. It extend the 
BCL Mail classes, allowing developers to use SMTP servers other than MS 
SMTP. It also has an extendable web service that allows the sending of email 
via HTTP/SOAP.
http://sourceforge.net/projects/opensmtp-net/

Project: dxf2xml
Autocad Drawing Interchange Format (DXF) to XML conversion tool written plus 
XSL-sheets to output various data formats. Assembly + commandline tool 
written in C# (Microsoft.NET Beta 2).
http://sourceforge.net/projects/dxf2xml/

Project: Twain.NET
Twain.NET is a C# class library providing functionality for using TWAIN 
image capture functionality in Microsofts .NET Framework including ASP.NET. 
This library enables you capturing images from any TWAIN compatible device 
even in ASP.NET WebApplication
http://sourceforge.net/projects/twainnet/

Project: FlySolo
Client side APIs for an alternative to Microsoft's Passport.
http://sourceforge.net/projects/flysolo/

Project: DotSquare Project
DotSquare Project aims to compet and be compatible with the new Microsoft 
.Net framwork in a *nix environement. In order to achieve that, it will have 
first to implement the SOAP protocol. Second to creat a runtime and a high 
level framework for byte cod
http://sourceforge.net/projects/dotsquare/

Project: X Marks The DOT
Projects written in C# for Microsoft's .NET platform devoted to XML 
technologies. Current projects include: RDF.NET - an RDF parser, DOC.NET - 
an extensible documentation generator.
http://sourceforge.net/projects/xmarks/

Project: GTK#
(sorry, no link)

Language bindings
-----------------
Project: Ruby bindings for .NET
http://www.geocities.co.jp/SiliconValley-PaloAlto/9251/ruby/netruby.html

Project: MonoBasic
(sorry, no link)

Project: Mixal compiler for .NET framework
MixNet is a compiler of Mixal assembly language described in Knuth's "The 
Art of Computer Programming" for Microsoft.NET framework. MixNet generates 
.NET executable files. All source code is written in C#.
http://sourceforge.net/projects/mixnet/


Frameworks
----------
Project: RAZOR - Microsoft.NET Application Suite
The purpose of this project is to collect and integrate best of breed 
software modules into a cohesive, configurable application framework based 
on the Microsoft.NET (tm) platform. The purpose is not to collect random 
libraries and code snippets, but to c
http://sourceforge.net/projects/razordotnet/

Project: .FREE Enterprise
.FREE Enterprise is a Open Source alternative to Microsoft's .NET Enterprise 
Servers. .FREE Enterprise includes a Mail server, System monitoring 
software, a Multi-server/cluster administration, SQL-compliant RDBMS, 
EDI/XML transaction server, and MORE!
http://sourceforge.net/projects/free-enterprise/

Project: DotNUKE : .NET News Portal System
The DotNuke Project is about designing and building a reusable news portal 
(like PHP-Nuke) using all the powerful features of Microsoft's .NET 
platform. This project is meant to be a demonstration of the power of the 
platform.
http://sourceforge.net/projects/dotnuke/

Project: CSUnit for the Microsoft .NET Framework
CSUnit is a regression testing framework for the Microsoft .NET platform. It 
has been heavily inspired by JUnit (www.junit.org) and its most important 
classes are generally interface compatible with their corresponding JUnit 
classes.
http://sourceforge.net/projects/csunit/

Project: NAnt - A .NET Build Tool
NAnt is a Ant like build tool for .NET
http://sourceforge.net/projects/nant/

Project: ASP.Net Portal
Modular internet portal based on .Net technology. C# and VB.Net will be the 
primary languages used.
http://sourceforge.net/projects/aspdotnetportal/

Tools
-----
Project: Beautifier
About: Beautifier (previously known as PSH, the PHP Syntax Highlighter) 
highlights and indents source code using highlight configuration files which 
are similar to Ultraedit highlighting files. It supports C, C#, Java, Perl, 
PHP, Pascal, Lisp, Mumps, Eiffel, Euphoria, and x86 Assembler, amongst 
others. It handles line/block comments, keyword highlighting, correct 
indentation, string highlighting, language contexts (allowing embedded 
languages), and selection highlighting. It is available in PHP and Perl 
versions.
http://freshmeat.net/projects/beautifier/

Project: SharpDevel
SharpDevelop is a free editor for C# and VB.NET projects on Microsoft's .NET 
platform. It is open-source (GPL), and you can download both sourcecode and 
executables from this site. In addition, you can find the latest information 
and changes on SharpDevelop, as well as get in touch with its author, Mike 
Krüger.
http://icsharpcode.net/OpenSource/SD/default.asp

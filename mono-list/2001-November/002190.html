If you are not building from the recent snapshots then you will not be able 
to build without windows and cygwin...  However, if you build from the latest 
snapshots you should be able to build mcs although not the class libraries 
because Nant won't work under wine.  You will have to edit the makefile and 
of course you will still need a windows partition and csc.exe... mcs is in 
heavy development, but it is reported to work under mint now in some fashions.

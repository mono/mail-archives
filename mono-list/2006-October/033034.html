On Thu, 2006-10-19 at 09:55 +0200, enjoylife at tele2.fr wrote:
> I realized a GTK# application and i want this is binary
> compatible with Windows and Linux.
> How do i make that ?

You already did it, the resulting .exe is binary compatible.
Just make sure you have GTK# installed on windows as well...
there is an installer for that in the Mono downloads directory.

Also, make sure that your application is compatible with the
GTK# of both your Linux and Windows systems (or that they have
the save GTK# version!).

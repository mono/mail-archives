> I would like to know where I could get some material for me to learn

> with only the compiler and a editor, so I would like to learn ASP.NET

> I know HTML and think that ASP.NET tags are not so difficult to

IMHO if you want to do web development with Mono you should try 
MonoRail[1]. It's a port of RoR (Ruby on Rails) to C# and works very 
well. In fact, many people think that the WebForms methodology of 
ASP.NET is flawed[2][3], and learning the MVC model of MonoRail is quite 
better.

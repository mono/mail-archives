You could use mono -v to see the debug:

Test.cs
using System;

class Test
{
        public static void Main ()
        {
                Console.ReadLine ();
                Console.WriteLine ("Hola");
                Console.ReadLine ();
                Console.WriteLine (String.Format ("a {0}", "b"));
        }
}

Compile:
gmcs Test.cs
mono -v Test.exe

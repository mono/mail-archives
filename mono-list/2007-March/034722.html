I think you are having the same problem that I had about a month ago.  I 
asked for help on the "Fedora Forum"  You can review the thread by 
searching for User Name "DickT", thats me.

I believe the problem is that not all mono related files are installed 
by a "yum install Mono*".  I went into extras and installed anything 
with "mono" and "monodevelop" in the package name.  Once this was done I 
also had to repeat the tutorial from scratch, I think some of the 
dependencies are established when you start a monodevelop project.

I had this problem and other similars. Have a look at my solution.

The directory "C:\Program Files\Mono-1.2.3.1\lib\xsp\test" is the problem.
You may not install where the path makes use of spaces as in "Program
Files".

Install mono project and all its components at a C:\root, for example:

C:\MonoProject\Mono-1.2.3.1
C:\MonoProject\vsprj2make

Thereafter you may run XSP for testing and for your application. If you are
using the Visual Studio.NET 2003, run the xsp by using the plugin
vsprj2make.

Best Regards

Eduardo Xavier

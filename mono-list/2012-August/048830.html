A quick answer just to check a few things
-I use ',' to separate connection string elements, not ';'
-I use the "URI=" scheme because I think I remember that the "data source="
one didn't work for me.
-If I were you, I would have tried the same code on mono+Linux, just to see
if it makes any difference.
-Are there any special/weird chars inside your "inputFile" db name?

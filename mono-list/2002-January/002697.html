The make errors you sent to the list earlier are because you didn't have
Microsoft's .NET SDK installed, and that is currently required to build the
compiler and runtime library.  Hopefully soon you will be able to build it on a
Linux machine and without the .NET SDK, but for now you need to be building on a
Windows machine and have the .NET SDK installed and in your path.

Miguel,

I think this[1] is the list of missing classes (Types) from corlib.
Thanks to John Barnett's help, I got the corlib assembly to give me it's
types (had to remove Object and ValueType).

I tried to spot-check the list by hand to see if I missed anything.  I
should be able to go over it closer later, but wanted to post this list
in case I don't get a chance.  

If there are any mistakes, please let me know.  If it's close, I'll turn
my little prototype into a proper tool so anyone can generate the list.

BTW, this info would go great on the class status web page so people
easily can pick something to implement.

More later,
Nick D.

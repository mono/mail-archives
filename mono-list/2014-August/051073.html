To add my two cents, I'm also very interested in these kinds of tools.
While we already buy Xamarin licenses, we also use Mono on Linux a lot, and
have even more interest in profiling there because we do some pretty
intensive stuff. We're building large-scale interactive "smart" displays
that can be used in concert with mobile apps. Using C# for the whole
ecosystem is hugely advantageous.

We already use Macs for mobile stuff, so shipping profiler data over to a
Mac isn't a big problem for us. But I just wanted to "+1" the Linux thing
-- I think with things like MVC6 and HyperFastCgi in the works, as well as
the revitalized web site, there could be an uptick in Mono-on-Linux
usage.We look at Mono and Xamarin together as one ecosystem, each part
solving part of our needs. More people may start to recognize the potential
there.

- Brett

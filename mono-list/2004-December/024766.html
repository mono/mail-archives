It looks like nobody wanted to touch this one, so I'll answer it myself.  I
upgraded to SuSE 9.2 Professional and the problem went away by itself.
Before doing the upgrade, I removed all of the Mono 1.1 files.  SuSE 9.2
ships with the Mono 1.0 files, and, as you might guess, everything works
right out of the box.

One thing I noticed is that Red Carpet is no longer working.  The list of
available channels is empty.  I'm not sure why that is happening, but YaST
seems to be sufficient for the tools I want to use right now.

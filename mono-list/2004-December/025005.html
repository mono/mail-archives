Let me guess... you are compiling the sender.cs + message.cs into
sender.exe, and listener.cs + message.cs into listener.exe. If you are
compiling in this way, you are doing it wrong. If you want to share the
Message class, you must compile message.cs into a dll and reference the
same dll from both applications, because the assembly name is part of
the full type name, so "Message, sender" is not the same as "Message,
listener".

Lluis.

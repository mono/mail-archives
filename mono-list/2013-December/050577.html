Hello,

> Next problem that I get is when starting monodoc on Ubuntu 12.04:
> Error: did not find one of the files in
> sources//opt/mono/lib/monodoc/sources/gd2i
> libgluezilla not found. To have webbrowser support, you need
> libgluezilla installed
> libgtkhtml-3.14.so.19
>
It turns out that using gtkhtml requires xulrunner, which is not part
of Ubuntu anymore, because Mozilla updates it too fast...
Therefore it is better to remove gtkhtml again, which also makes the
problem with docbrowser go away, and make sure that libwebkit-cil-dev
is installed when compiling mono-tools.

Now my monodoc opens just fine!

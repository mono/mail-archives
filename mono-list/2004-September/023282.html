I just bought SuSE 9.1 Professional on Sunday and it comes with Mono .30 on
one of the CDs, but you have to install it using YaST2.  I ended up using
YaST2 to install Red Carpet, and I used Red Carpet to install Mono 1.0.1.
I'm still in the process of configuring the machine.  I don't consider that
to be "pre-installed", but it's still a nice Linux product.  I would highly
recommend it, but don't think it's going to be as simple as "pre-installed".

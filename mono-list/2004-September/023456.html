Presumably this is 1.0.2.  I had the same problem.  The .config file
for the .dll (eg.  /usr/lib/mono/gac/gtkhtml-sharp/1.0.0.0__35e10195dab3c99f/gtkhtml-sharp.dll.config)

refers to libgtkhtml-3.1.so.11, which isn't installed on my
bog-standard SuSe install.  libgtkhtml-3.1.so.7 is, however.  Changing
the dllmap target filename to .7 from .11 did the trick.

I'm sure you can figure out any variations required to get yours
working. ;)

Cheers,
    Rob.

> I want to install Gtk# installer to Visual Studio.2003 but when I started
to setup Gtk# installer it gave an error.It said me this;
>
> ---------------------------
> Microsft .NET Framework SDK 1.1 does not seem to be installed.

try Start/Settings/Control Panel/Add-Remove programs
and from there choose the Microsoft .NET Framework 1.1
and choose to do "Repair" (else it should have some "Change/Remove" option
and at the installer it runs you'll see options for Repair and Remove)

else just do http://windowsupdate.microsoft.com and see at the sections
other than the "critical updates" in case it shows the MS .NET Framework 1.1
Service Pack 1 or something and install that too in case it fixes your
registry

else do "Change/Remove" at the VS.net 2003 item in the add-remove programs
dialog and at the dialog opened select the "Prerequisites" and that should
re-install the VS.net2003 prerequisites (will ask for the CD/DVD) which
includes .NET framework 1.1

else just download the whole 1.1 SDK from MS and install it

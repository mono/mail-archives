You haven't installed mcs which is contained in the mono-devel package =
in red-carpet.

If you want to build mono from scratch, and intend to keep up with daily =
builds, then I would suggest getting the daily mono build from =
http://www.go-mono.com/daily/ and starting from there.

Generally, when trying to build you should either run autogen.sh or =
configure.  Both of these will accept the same command line flags.  So, =
if the docs say to run ./configure --prefix=3D/path/to/mono, you can =
also run ./autogen.sh --prefix=3D/path/to/mono

I hope that helps.

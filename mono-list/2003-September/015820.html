Hi together!

I'm missing any functionality in Microsoft's .NET framework regarding =
junction links. Linux is able to create these hard links for years, but =
NTFS volumes also support them. You can validate if it is such a hard =
link with FileAttributes.ReparsePointer. But currently MS doesn't =
provide any methods but a WIN32API call (see Junctions V1.02 from =
sysinternals.com).

It would be great if Mono could be better at this point and provide this =
feature first; I think linux users would miss it very much since it is a =
"standard feature" for years.

Regards
Jochen

I gave up trying to get XSP working under Windows, so I installed RedHat
8 under VMware and will use that for my presentation on Mono's ASP.NET
support at the upcoming Web Services Edge West conference
(http://sys-con.com/WebServicesEdge2003West/).
 
Unfortunately, the link to download an RPM of the Mono source for RedHat
8 is broken.  I see that there are links to download the source code
separately, but I don't really know enough about Linux to know what the
full ramifications of that decision might be.  I would assume that the
source code would just be in pieces - but if someone could fix the RPM
link, that would be great!

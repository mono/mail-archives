Yes, like the .NET framework we have a RijndaelManaged class implementing a
superset of AES - superset because the RijndaelManaged implementation
supports 192 and 256 bits block size unsupported by AES.

The class is available under /mcs/class/corlib/System.Security.Cryptography.

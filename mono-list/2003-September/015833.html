> I have been struggling for weeks trying to find the file ModMono.cs that is need to install MOD for Apache, is this file located in the installation file? or do I have to download it from your web site.

You cand find that file in mod_mono CVS repository. You can get it with
the following commands:

      export CVSROOT=":pserver:anonymous@anoncvs.go-mono.com:/mono"
      cvs -z3 co mod_mono

If you can't access to cvs command you can try to get the files from
mono CVSWeb. Look for the URL inside mono download page.

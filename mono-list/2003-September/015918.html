Any LDAP v2 or v3 compliant directory will be supported.  Active =
Directory
is one of them, among many others (SunOne, formerly iPlanet, OpenLDAP,
Netscape, eDirectory, etc).

You will be able to do anything you can do in the current Microsoft
implementation of the System.DirectoryServices namespace, and as well,
handle low level details if you want; for example, sending custom =
controls
to the server.

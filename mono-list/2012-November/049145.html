Mono includes a prerelease version of Entity Framework 6.0. This is the
only version that Microsoft has released as open-source. You should be able
to grab the EntityFramework.dll file from Mono's GAC and reference that in
your application.

You *might *be able to get the app running by creating an assembly binding
in the config file.

The mono-project.com website does appear to be horribly out of date in
places (including still mentioning AnonSVN).

If you click through a couple of links then you get to the Source Code
Repository page (http://mono-project.com/SourceCodeRepository), which
references the GitHub projects. The code is at
https://github.com/mono/mono (clone URL
https://github.com/mono/mono.git) and build details for Linux are at
http://mono-project.com/Compiling_Mono_From_Git (although they reference
the 2.6 tag, so things may have changed)

Hope that helps.

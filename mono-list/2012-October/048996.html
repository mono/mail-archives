I've had similar issues trying to find a precompiled library in Mono with no luck.  I tend to just grab the source file from Mono's source repo directly and compile it within my project (as long as my project is written in C#).  You can also compile the file in it's own dll if if helps you achieve your goal.  The file can be located here:
https://github.com/mono/mono/tree/master/mcs/class/Mono.Options/Mono.Options

FYI - There is also the command line parser library if you want something precompiled.  Your probably set on Mono.Options but I wanted to just point this out just in case.
http://commandline.codeplex.com/

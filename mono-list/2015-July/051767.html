If you are a beginner, and you don't have a good reason to build from
master, I would really recommend using the packages provided by
Xamarin:
http://www.mono-project.com/docs/getting-started/install/linux/#centos-fedora-and-derivatives

to install the packages you need:
yum install mono-complete xsp

You can browse the packages available at
http://download.mono-project.com/repo/centos/

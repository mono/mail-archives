> How do I write app for MeeGo using Mono (C#)?

It depends on the kind of application you want to build.

If you want to build a GUI application, just select the "Gtk# project" template 
from MonoDevelop and that will be all you need.

The MonoDevelop plugin merely helps you with the deploy stage and it is not 
released yet.

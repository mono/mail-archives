We don't ship gtk+ with the linux installer.  We rely on the gtk+ that
is installed with the distro.  This being the case, we stick with the
lowest common denominator and use gtk# 2.4.

The easiest way to use gtk# 2.8 on linux is to use SuSE 10, and use the
rpms from www.mono-project.com.

Good Day

 

I have complied and installed mod_mono on this a centos distribution. I have
written a mono website on SuSE 10 running the latest apache and it is
working. When I run it on centos I am unable to get the web page to display.
Does anyone know how to solve this? I have included the configuration for
the web page.

 

<VirtualHost 160.124.112.135>

    ServerAdmin admin at test.co.za

    ServerName test.co.za

 

    DocumentRoot /home/test/web/ecos

 

    ErrorLog /var/log/httpd/error_log.test.co.za

    CustomLog /var/log/httpd/access_log.test.co.za combined

 

    HostnameLookups Off

 

    UseCanonicalName Off

 

    ServerSignature On

 

    Include /etc/httpd/conf.d/mod_mono.conf

 

    LoadModule mono_module /usr/lib/httpd/modules/mod_mono.so

 

    AddMonoApplications default "/:/home/test/web/ecos"

 

    <Directory "/home/test/web/ecos">

                AllowOverride None

                Order allow,deny

                Allow from all

                SetHandler mono

    </Directory>

</VirtualHost>

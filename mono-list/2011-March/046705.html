Not 100% sure, PKCS7 is quite large.

Mono.Security.dll provides some (very low level) API do that support
some formats (e.g. it's used for authenticode signatures). It's
low-level enough to allow you to do anything (including a lot of
things what won't interoperate if you make mistakes ;-)

Otherwise I would suggest you look at using Bouncycastle (the .net
version of course :-)

Thanks a lot. I apologize for a delayed response as I had
been on the move for the last couple of days.

mono's VB.NET components - compiler and runtime gets
patched up on day to day basis only in the CVS HEAD branch
of mono. So you should be checking out the code from cvs 
HEAD and builidng it from there.  If you are targetting to
use mbas, we strongly discourage the use 1.0 versions of 
the mono package.

As for building the packages from CVS you can consult the 
following link.

http://www.mono-project.com/contributing/anoncvs.html

Once you checkout the code, you can consult the README
files on how to proceed further.

Hope this helps.

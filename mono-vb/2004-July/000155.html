Hi Jorge,

Currently mbas (our VB.NET compiler) isn't fully implemented, so you can try 
to run your app and help us with bug reports.

If you have a layered-design (WEB/BLL/DAL) you can compile with vbc in 
windows all of the BLL/DAL and install the compiled assemblies, but our 
runtime support also isn´t complete (specially if you don't use Option 
Strict On), so bug reports are also welcome for it.

Otherwise, if you really need it before Mono 1.2 (probably November 2004), 
that is our target release for a stable vb.net compiler/runtime, you can 
consider porting your app to C#.

Good developments,

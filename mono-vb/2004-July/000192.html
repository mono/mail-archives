It seems that you don't have cygwin installed, to simply run make in 
mcs/jay, and let gcc compile that little utility. Jay wasn't ported to C#, 
instead it was revised to support generating parsers in C#.

Without cygwin/gcc, you'll have to decode the makefile to know how build it 
with any c compiler you may have available in windows (msvc, c++builder, 
...).

To build mbas you need to have a jay.exe to be able to construct the parser 
class source (mb-parser.cs) from the grammar rules file (mb-parser.jay), 
before compiling with the c# compiler.

Hope it helps,

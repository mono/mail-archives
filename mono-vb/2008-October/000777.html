Hi,

My suspicion was confirmed, the problem is that you have the files
06+gs.aspx and 07+gs.aspx in the root directory, mono tries to compile those
automatically, but create invalid code due to the files having +'s in them. 

A bug has been filed: https://bugzilla.novell.com/show_bug.cgi?id=431622

Until that bug is fixed you can remove those files, then at least index.aspx
works (I haven't tested anything else).

Rolf

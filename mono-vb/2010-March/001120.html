> Are thee any examples VBNC.exe batch files/command lines parameters

Vbnc.exe takes the same command line parameters as vbc.exe, here are a few
samples: http://msdn.microsoft.com/en-us/library/4dehewf5.aspx - just use
vbnc instead of vbc.

To compile several files at once, just do:

vbnc file1.vb file2.vb -out:myprogram.exe

When you build a project in Visual Studio, you'll also get the command line
to vbc in the output window.

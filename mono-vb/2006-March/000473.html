Hi Scott,

MonoDevelop already uses mbas, but it's support for VB projects has
stalled some eons ago, so I'm not sure how much coding/testing is
needed to have it fully working.

Sharpdevelop, uses vbc by default, but I guess that it may be
configurable somewhere or some, hopefully small, recoding of the
language binding is in order.

As both projects are open-source it is at least feasible to make them
use any compiler. AFAIK, VS.NET VB support module has vbc hardcoded
in, and is unavailable to be changed.

The all <a href="http://VB.NET">VB.NET</a> compilers use many other things from Microsoft.VisualBasic assembly, it is the &quot;runtime&quot; library for the language, so you won&#39;t be able to remove references to it. If you don&#39;t wan&#39;t such a dependency you should use C# that AFAIK is the only language that doesn&#39;t need some other runtime library besides the 3 core .NET assemblies (mscorlib, System and System.Xml).<br>
<br>To have your program running in mono you just need to install the mono package that contains the library and the compiler (the name depends on the distro, as they package it differently, although most use the mono-basic prefix).<br>

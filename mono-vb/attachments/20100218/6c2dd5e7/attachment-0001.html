I've wanted to do that a long time ago.<br>
<br>
You can take a look at Apache Lucene, a Java search library, which you
could port to .net.<br>
Perhaps you find a way to compile the lucene library from java
source/bytecode directly to .net.<br>
<br>
Another way is to extend this codeproject project:<br>
<a class="moz-txt-link-freetext" href="http://www.codeproject.com/KB/IP/Crawler.aspx">http://www.codeproject.com/KB/IP/Crawler.aspx</a><br>
<br>
Then you need a ranking algorithm, such as Google PageRank, or perhaps
better something like Yahoo TrustRank, and a parallel computation
library, and a cluster software for computing the Eigenvectors of the
markov chains (indexing).<br>
<br>
I found this site about PageRank to be particularly useful because of
it's incredible simplicity:<br>
<a class="moz-txt-link-freetext" href="http://www.peterbe.com/PageRank-in-Python">http://www.peterbe.com/PageRank-in-Python</a><br>

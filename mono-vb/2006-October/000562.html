Hi,

I guess you created a new file that contains the new class but you don't 
pass that file on command line.

The .bat files (for Windows) use vbnc/vbnc/tests/SelfTest/bootstrap.rsp that 
references SelfCompile.files.rsp in that directory that contains the list of 
source files. You have to add the new source file to that list.

bootstrap.mono (for Linux) uses no response files it contains the list of 
source files itself. You have to add the new source file to that list as 
well.

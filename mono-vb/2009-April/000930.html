Hi,

unfortunately there is no GUI designer for Mono-VB at the moment.

Please use glade (available in every Linux distribution) for the GUI design.

Additionally you should download the Glade# Code Generator from:
http://github.com/FireRabbit/glade-sharp-code-generator/tree/master
With this tool you can easily generate the Mono-VB code to display your
windows.

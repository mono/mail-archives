Hi,

Mono currently has no VB compiler. (vbnc is not part of the release because 
it's not able to bootstrap on Mono)

Mono has a VB runtime however, so you are able to use binaries compiled 
using vbc on Windows.

You should use C# for inline code in ASP.NET. And compile code to a DLL that 
you don't want to port to C#.

Kornél

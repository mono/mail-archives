Since you use MSDE, I assume that you've got a client/server application.
If you want to keep on MSDE, you might want to continue to run MSDE on a windows server.
On your Mac, you would run the client application, which accesses the MSDE on the windows server with the native database provider for SQL Server.

--> If database server can be a separate machine, you only need to take care a little bit (e.g. Windows Forms) for your client application.

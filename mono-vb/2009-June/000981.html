Go to the project's option (right click the project in the solution pad,
select 'Options').

Now if you have a newer MonoDevelop, you should go to the Build/General page
and select WindowsForms as "My Type".

If you can't find that option, go to the Build/Compiler page and add
-define:_MYTYPE=\"WindowsForms\" to the "Additional compiler arguments"

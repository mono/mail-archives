Mono has a 1.1 Microsoft.VisualBasic.dll not a 1.0 one so your vbc 1.0
program won't execute. Please recompile with a vbc 1.1 (MS.NET 1.1)
compiler, OTHERWISE you will have to compile and install a customized
version of Mono that you will need to compile with the 1.0 profile
(not an easy task).

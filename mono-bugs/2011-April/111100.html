Forgot to explain how to create a repro:

- Create a new project
- On top directory, add the attached file whose name is "libopenal.so" (which
is a compiled binary from http://pielot.org/2010/12/14/openal-on-android/ )
using ndk r5b.
- Select the file in the project and change its build action to
"AndroidNativeLibrary"
- F5 (build and run) and select destination
- Error dialog will be shown. Select "More Details"

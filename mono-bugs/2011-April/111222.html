Mono 2.10.2 rc5

Description of Problem:
When you use the winforms print dialog, if you select 'Print to file', it
prints the output to the file you specify, but also prints to the default
printer.

Steps to reproduce the problem:
1. Install the packages wf-testapps and wf-apps.
2. Run 'PieChart'.
3. Select Print.
4. In the print dialog, make sure to check the "To file" check box.
5. Click OK.



Actual Results:
The output is sent to the file and printed to the default printer.

Expected Results:
It should only print to the file.

How often does this happen? 
Always.

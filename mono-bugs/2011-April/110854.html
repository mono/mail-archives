I'm using UIButton with custom image background placed on UINavigationBar. When
I click this button, application crashes with NullReference exception. Heppens
on MonoTouch 4.

Reproducible: Always

Steps to Reproduce:
1. Build and run sample.
2. Press "New" button on navigation bar. New navigation bar with "Back" button
with arrow background should appear. If back button has rectangular background,
something went wrong.
3. On new navigation bar, press "Back" button. Observe crash.
Actual Results:  
Application crashes with NullReference exception.

Expected Results:  
Application should not crash and clicked navigation bar should disappear (also
all other navigation bars bellow it).

+1. Download ftp://ftp.cae.net/pub/cae/testmono.zip
+2. Run the included executable with mono in a Linux system.
+3. Navigate through the different options that showcase each problem 
+individually.

+flickering in PictureBox updates.
+

+Windows-like behaviour.

Steps:
In Monodevelop 
1- new solution, name, OK
2- double click MainWindow.xib

in Interface Builder:
1- go to library->classes
2- rightclick UILabel -> new subclass...
3- type "MyLabel"
4- drag and drop MyLabel from list to window 3 times
5- save, close

back in monodevelop:
1 - try to compile

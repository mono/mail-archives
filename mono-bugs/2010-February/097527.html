openSUSE 11.2 x86_64

Automated test for BlogStarterKit fails when clicking on a story link that
already has been created in BlogStarterKit.

To reproduce:

1) Install the above environment.
2) create a new category
3) create a new story with title and summary (no story text can be entered
because the form is not editable)
4) Click "Home"
5) Click on the story name

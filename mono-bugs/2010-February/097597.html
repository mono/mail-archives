Description of Problem:

openSUSE 11.2 x86_64
mono-core-2.6.3-30.1.x86_64
apache2-mod_mono-2.6.2-30.1.x86_64

The automated test failed when testing a precompiled MVC template app.  This
app is used for all automated testing run and has worked in the past and
therefor is a regression.  This happens with both Apache and xsp2.

2) http://<test-server>/mvcTestPre/
3) Click the "About" link in the upper right hand corner

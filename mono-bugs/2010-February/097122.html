For example:

class Program
{
    static void Main()
    {
        for (int n=0; n<10; n++)
            Console.WriteLine (n);
        for (int n=0; n<10; n++)
            Console.WriteLine (n);
    }
}

The only way to get a local is using MethodMirror.GetLocal(), but this call
fails with an exception when trying to get 'n':
System.Reflection.AmbiguousMatchException: More that one local has the name
'n'.

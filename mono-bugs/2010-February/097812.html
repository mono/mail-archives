The DesignSurface implementation in System.ComponentModel.Design doesn't
support Drag&Drop of controls to it.


Reproducible: Always

Steps to Reproduce:
1. Start the attached OSControl.Editor.exe
2. Click on File -> New to create a new form
3. Drag a Button from the TreeView on the left onto the new form
Actual Results:  
The DesignSurface doesn't accept the dragged control. It's not possible to drop
any control onto the new form.

Expected Results:  
The DesignSurface should accept the dragged control and create a new control on
the form when dropped.

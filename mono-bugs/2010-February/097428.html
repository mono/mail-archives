When using System.Configuration.Save() to simply store a previously opened
Web.config file, alle comments inside the file will be stripped off as well as
some line breaks.

Reproducible: Always

Steps to Reproduce:
1. Create a Web Application using Visual Studio 2008
2. Put a Label onto the default web form
3. Enter the following code in the Page_Load() method of Default.aspx.cs
   Configuration config = WebConfigurationManager.OpenWebConfiguration("~/");
   config.Save();

Actual Results:  
The resulting Web.config file is kind of crippled and very difficult to read by
a human.

Expected Results:  
Under Windows XP / IIS the resulting Web.config is exactly the same as before!

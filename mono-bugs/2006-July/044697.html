+Summary: mono-find-provides skips files that are not in /gac/
+
+In mono 1.1.16 mono-find-provides was changed to skip files that are not
+installed in the gac. This is causing a problem in combination with
+mono-find-requires.
+
+Mandriva's rpm uses mono-find-provides and mono-find-requires to
+automatically generate deps for packages. Now all mono packages depend on
+mono(mscorlib) = 1.0.5000.0, but this isn't provided anymore as
+mscrorlib.dll is not in gac.
+
+Please revert this change as it makes mono-find-requires useless for
+automatic rpm deps.

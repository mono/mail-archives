+Summary: Bug de las sesiones en global.asax

+Please fill in this template when reporting a bug, unless you know what you
+are doing.
+Description of Problem: Solo se muestra el valor de las variables de sesion
+una vez cuando estan declaradas en el metodo Session_Start del archivo
+Global.asax.cs, como en el siguiente caso
+ 
+protected void Session_Start(Object sender, EventArgs e)
+{
+    Session["sessVar"] = "Hola";
+}
+ 
+El Problema lo tengo en XSP trabajando en Red Hat 9 y con cualquier
+navegador de estos IExplore, Mozilla y Mozilla Firefox, ya sea llamando la
+aplicacion desde la misma maquina o desde otra diferente.
+ 
+La misma aplicacion la tengo en una maquina con el sistema operativo
+Windows 2000 Professional y corre sin problemas ya sea desde la maquina
+local o desde otra diferente.
+ 
+Muestro el valor de esa variable en la parte del Load de pagina index.aspx.
+
+Steps to reproduce the problem:
+1. Copiar los archivos adjuntos a un folder de dentro de la maquina con Red
+Hat, yo los copio dentro de una carpeta que se llama ManejoSesiones que
+esta dentro de un folder que se llama pruebas que tengo dentro de /home.
+Quedando asi /home/pruebas/ManejoSesiones
+2. Matar todos los procesos de XSP con el comando killall mono.
+3. Levantar el servicio del XSP de nuevo con este comando, /usr/bin/mono
+/usr/bin/xsp.exe --root /home/pruebas/ManejoSesiones --nonstop&
+4. Abrir un navegador y escribir la direccion
+http://IPDELAMAQUINA:8080/index.aspx
+5. El Explorador muestra en la pagina index.aspx un resultado como este:
+ 
+Application Variable : Variable de Aplicacion
+Session Variable : Hola
+
+6. Cerrar el navegador y abrirlo de nuevo poniendo la misma direccion o sea
+http://IPDELAMAQUINA:8080/index.aspx esto hacerlo sin matar el proceso del XSP.
+7. El resultado en la pagina index.aspx sera uno como este:
+
+Application Variable : Variable de Aplicacion
+Session Variable : 
+
+ 
+Actual Results:
+ 
+Ya no se muestra el valor de esa variable la segunda vez que se ve la
+pagina web.
+ 
+
+Expected Results:
+ 
+Deberia de mostrarse siempre el mismo resultado, cerrando una y otra vez el
+navegador y volviendolo a abrir.
+ 
+
+How often does this happen? 
+Siempre en la maquina de Red Hat Linux 9.
+
+Additional Information:
+
+ 
+Todos los archivos estan instalados desde los rpms. que vienen dentro de la
+version 1 de Mono.

Yes this is an interesting problem as Power Server has dynamic CPU add/remove.

There are two cases where we can assume a uni:

1) the Linux kernel is not built for SMP (a uni kernel forces uniprocessor
execution independent of the number of (possible) CPUs available.

2) There is only 1 CPU in the system. (sysconf (_SC_NPROCESSORS_CONF) ==
sysconf (_SC_NPROCESSORS_ONLN) == 1)

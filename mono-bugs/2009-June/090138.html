--- Comment #1 from Andrew Jorgensen <ajorgensen at novell.com>  2009-06-23 10:46:38 MDT ---
This is not our bug, it seems to appear on any system that has linux
filesystems already, or perhaps a linux fs as the first partition, or something
to that effect.  We showed this to occur with a manually-created partition
table as well as with older versions of the mono live cd and the official 11.1
gnome live cd.

It does not occur if your partition table does not have linux filesystems (a
single windows partition, for instance).

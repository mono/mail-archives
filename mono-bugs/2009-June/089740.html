When drawing a listbox with OwnerDraw(Fixed), in the DrawItem event the
DrawItemEventArgs.ForeColor should be set to SystemColors.HighlightText for
selected Items. This leads to nearly unreadable selected text if not corrected
manually.

Reproducible: Always

Steps to Reproduce:
1.compile and run attached test.cs
2.left ListBox is draw normally, right ListBox is draw with OwnerDraw
3.see selected line in right ListBox


Expected Results:  
Should look (nearly) like left ListBox.

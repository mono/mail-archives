Description of Problem:
None of the Menu options work on a solution where the StartUp project is
located in a Solution Folder

Steps to reproduce the problem:
1. Create or open a basic, working VS project that will work with Run Locally
in Mono
2. Create a solution folder by right clicking on the solution in solution
explorer and selecting "Add".."New Solution Folder".
3. Move the startup project into the new solution folder.
4. Select "Mono".."Run Locally in Mono".

Actual Results:
"Missing Startup Project" dialog

Expected Results:
Application running in Mono.

How often does this happen? 
Every time, on all features.

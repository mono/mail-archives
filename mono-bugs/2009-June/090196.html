Description of Problem:

It is not possible to compile this valid code:

[code]
using System;

namespace TestVirtualProp
{
    class MainClass
    {
        public static void Main(string[] args)
        {
            Console.WriteLine("Hello World!");
        }
    }

    public abstract class Class1
    {
        protected string prop = null;

        public virtual string Prop
        {
            get { return prop; }
            set { prop = value; }
        }
    }

    public abstract class Class2 : Class1
    {
        public abstract override string Prop
        {
            get;
        }
    }

    public class Class3 : Class2
    {

        public override string Prop
        {
            get{ return prop; }
            set { prop = value; }
        }
    }
}
[/code]

Actual Results:
Error CS0546

Expected Results:
No Error.

Additional Information:
The same code works on VS2008.

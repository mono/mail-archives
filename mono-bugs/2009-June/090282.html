Description of Problem:

MonoVS VMware image: MonoVS-0.1.2410-vmx.zip

If a user clicks on the "Update MonoVS Server" icon on the desktop of the
VMware image, the windows which appear after the first confirmation dialog,
appear below other open windows.

Steps to reproduce the problem:
  0. Open an gnome terminal (or possibly some other application)
  1. Click on "Update MonoVS Server" icon on desktop
  2. Click "Yes" on the confirmation dialog

Actual Results:
  All subsequent windows appear below the gnome terminal.

Expected Results:
  Windows should appear above other windows.

How often does this happen?
  Always.

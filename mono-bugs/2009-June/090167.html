Description of Problem:
When I "choose remote host" in VS, it's still incredibly common for me not to
be able to see my VM in the list of hosts.  The first thing I do is drop to the
command line and type /sbin/ifconfig to get the IP, and then enter that in VS.

I suspect most Windows users won't know to / how to do this.  It would be
intuitive to be able to see the IP:port(s) that the server is listening on by
right-clicking the tray icon.

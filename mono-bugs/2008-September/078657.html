Test case:

public struct Test

{

    private int x;

    private int y;

    public Test(int x, int y)

    {

        X = x;

        Y = y;

    }



    public int X

    {

        get { return x; }

        set { x = value; }

    }



    public int Y

    {

        get { return y; }

        set { y = value; }

    }

}

Actual result:
It compiles.

Excepted result:
Compilation should fail with the error [The 'this' object cannot be used before
all of its fields are assigned to(CS0188)]. This is how csc behaves.

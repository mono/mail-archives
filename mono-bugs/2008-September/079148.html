On openSUSE 11.0 x86_64 with mono-2.0rc1 + patches to fix bug 423450

Banshee does not exit when using the window manager X button to quit the
application.  It does exit properly when using the menu "Media -> Quit".

To reproduce, start banshee-1 from a console and quit the application by
clicking on the X in the upper right hand of the window manager.  The
application hangs in the console, not returning to the command prompt until ^C
is pressed.

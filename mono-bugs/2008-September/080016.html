The properties ObjectDataSource.CacheKeyDependency and
ObjectDataSource.CacheKeyDependency are the reason why the DependencyCache hack
was added to Cache. Those properties allow the user to make the caching
mechanisms by the system (stored in HttpRuntime.InternalCache) to depend on
items stored in the user cache (stored in HttpRuntime.Cache).

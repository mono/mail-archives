This is caused by the Linux bin installer.  This cannot really by fixed because
every distro puts our libraries in different places with different names. 
Because it constantly causes problems like this, we have gotten rid of the
installer.

You will need to either:
- Install from source.
- Wait for official packages from your distro.
- Use unofficial packages from http://www.mono-project.com/Other_Downloads.

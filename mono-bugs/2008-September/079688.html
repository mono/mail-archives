Description of Problem:

On the Windows platform, the "ShowIcon" property of the Form class is not
working. An icon is displayed even if the property is set to false.

Steps to reproduce the problem:
1. Set up a new winform project with one form
2. Leave everything to defaults but set the ShowIcon property of the form to
false

The icon is displayed still.

Expected Results:

The icon should disappear.

How often does this happen? 

Always.

Test case:

using System;
using System.IO;

class Test
{
        static void Main ()
        {
                Directory.SetCurrentDirectory (@"\\192.168.0.1\robertj");
                Console.WriteLine (Path.GetFullPath ("."));
        }
}

Expected output:

\\192.168.0.1\robertj

Actual output:

\\192.168.0.1\robertj\\192.168.0.1\robertj\192.168.0.1\robertj

It only occurs if the server name is an IP.

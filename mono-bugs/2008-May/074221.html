Description of Problem:
If you try to access the xsp2 test pages in SLED 10 SP2, you just get a blank
page. No errors are logged to the screen.

This does not happen when you first install SLED 10 SP1, therefore it is a
regression in the mono shipped with SLED 10 SP2.

However, if you install SLED 10 SP1 and then run all of the updates, then you
get a blank page when accessing the xsp2 test pages.


mono-web versions:
SLED 10 SP1 no updates (works): mono-web-1.2.2-12.12
SLED 10 SP1 with updates (broken): mono-web-1.2.2-12.17
SLED 10 SP2 no updates (broken): mono-web-1.2.2-12.20


Steps to reproduce the problem:
1. Install SLED 10 SP2 i586
2. Install the XSP that ships with SLED 10 SP2
3. DO NOT UPGRADE MONO
4. cd /usr/lib/xsp/test
5. run xsp2
6. navigate to http://server:8080
7. notice that you only get a blank page


Actual Results:
Blank page


Expected Results:
The tests page should appear


How often does this happen? 
Every time

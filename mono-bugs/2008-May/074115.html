Three faults apparent from this:

1) After changing folder no item is highlighted, but if one hits the down-arrow
then the *second* item is selected.

2) Hitting space or return only gives the first item focus but does not select
it.  Thus to select it one has to do for instance: arrow-down, arrow-up.

3) If only one item is present, its not possible to select it/action it by
keyboard alone.  No effect on Space, Return, or arrow keys.

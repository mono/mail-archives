The problem is in the application code. It capture the mouse on down and
release it on up. However up is not fired(*) if the button is up outside the
plugin area. This makes it possible to move the sliders without holding the
button.

(*) the same thing happen using Windows/Firefox/Silverlight so this is a
"feature" ;-)

This one needs to be open because the fix broke menustrips for dual monitors. 
Everaldo is working on the fix, which is to have Screen.Bounds return just the
physical monitor size, not the size of all the virtual desktops.

For example, if your monitor is 800x600, Screen.Bounds needs to return 800x600,
not 3200x600 as it currently does.

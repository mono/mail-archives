I believe the Line 360 of decimal.c for the routine `div128by32' should read:

    return (a > factor || (a == factor && (c & 1) == 1)) ? 1 : 0;

instead of

    return (a >= factor || (a == factor && (c & 1) == 1)) ? 1 : 0;

as otherwise it could be simplified in just

    return (a >= factor) ? 1 : 0;

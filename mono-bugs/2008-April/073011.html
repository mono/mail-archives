--with-xen_opt=yes is the default so you usually don't need to pass it to 
configure. Make sure that the  'checking for -mno-tls-direct-seg-refs option to
gcc' configure output line says 'yes'. Also, try checking that 
'mno-tls-direct-seg-regs' is used by gcc when compiling mono, and that
MONO_XEN_OPT is defined in config.h.

In Windows, the FileDialogs allow the user to input a wildcard (like *.*) into
the File name field and after pressing <Enter> the File list is updated and
only shows files that match the wildcard. The input focus remains within the
File name field.

In Mono (1.9.3) after entering a wildcard and pressing <Enter> there will be an
error message like "The file <path>/*.* does not exist. Please verify..."

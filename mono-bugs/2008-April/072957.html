submitter adds:
"su" access required to access serial ports

NOTE: users of MonoDevelop should be notified that MonoDevelop must be launched
with admin rights (from terminal) to access serial ports.

Instructions:
1. Run Terminal
2. Type "su -"; enter admin password
3. Type "cd /usr"
4. Type "cd share"
5. Type "cd applications"
6. Type "monodevelop"

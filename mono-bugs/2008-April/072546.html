Description of Problem:

When running Windows based MDI applications, instead of the window being the
correct size, it is all 'scrunched up' and the menus are also not drawn
properly - they are all 'jammed' together.

Steps to reproduce the problem:
1. mono 'mdi-application'


Actual Results:

As can be seen from the attached screenshot, the window is not drawn correctly.
 The original application was 1024x768 resolution.  The window here is squashed
up and thus the application is un-usable.


Expected Results:

The window to be drawn at correct size with all controls in the window also
being correctly drawn.


How often does this happen? 

Every time you run an MDI application.


Additional Information:

Please note: I have also tested another non-MDI application that had menus and
was also 1024x768 resolution and the form draws almost correctly, but the menus
are as per the above description.  So it may not me MDI specific.

kCLLocationAccuracyBestForNavigation is not bound in CLLocation. Setting this
is used to make location services use additional sensor information to increase
precision of location and heading data.

Reproducible: Always

Steps to Reproduce:
See CLLocation class. Other CLLocationAccuracy constants are bound to static
fields in CLLocation
Actual Results:  
kCLLocationAccuracyBestForNavigation not bound

Expected Results:  
kCLLocationAccuracyBestForNavigation bound in CLLocation

See apple doc
https://developer.apple.com/library/ios/#documentation/CoreLocation/Reference/CLLocation_Class/CLLocation/CLLocation.html%23//apple_ref/c/data/kCLLocationAccuracyBestForNavigation

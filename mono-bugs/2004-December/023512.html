+Summary: CS0647: Error emitting 'PermissionSet' attribute for RequestOptional.
+
+Description of Problem:
+
+MCS doesn't want to encode a PermissionSet attribute for
+SecurityAction.RequestOptional. The error message says it is not a valid
+action for a non-CAS permission, however PermissionSetAttribute inherits
+from CodeAccessSecurityAttribute (which inherits from SecurityAttribute) -
+so it is a CAS permission.
+
+
+Steps to reproduce the problem:
+1. Compile the following source code using MCS (HEAD)
+
+using System;
+using System.Reflection;
+using System.Security;
+using System.Security.Permissions;
+
+[assembly: PermissionSet (SecurityAction.RequestOptional, Unrestricted=true)]
+
+public class Program {
+
+	static public int Main (string[] args)
+	{
+		Console.WriteLine ("Hello");
+	}
+}
+
+
+Actual Results:
+poupou@pollux:~/src/bugzilla> mcs bug.cs
+bug.cs(6) error CS0647: Error emitting 'PermissionSet' attribute because
+'Invalid SecurityAction for non-Code Access Security permission'
+bug.cs(10) error CS0161: Not all code paths return a value
+Compilation failed: 2 error(s), 0 warnings
+
+
+Expected Results:
+Success
+
+
+How often does this happen? 
+Always.
+
+
+Additional Information:
+poupou@pollux:~/src/bugzilla> mcs --version
+Mono C# compiler version 1.1.3.0

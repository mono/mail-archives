The monodoc paths on Mac should include the "external" directory:

/Library/Frameworks/Mono.framework/External/monodoc

The purpose of this directory is to provide a place for third-party installers
to place docs where they will not be removed by Mono upgrades. MonoDevelop
passes this to MonoDoc within MD but it would be useful to have it work outside
of MD.

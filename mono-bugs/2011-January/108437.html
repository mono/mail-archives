--- Comment #1 from Jonathan Pobst <jpobst at novell.com> 2011-01-25 19:17:25 UTC ---
This is fixed in svn.  

A workaround is to either add them manually to the project file, or go to:
C:\Program Files (x86)\Reference Assemblies\Microsoft\Framework\MonoDroid

In each one of those versions, there is a RedistList folder with a
FrameworkList.xml file in it.  Open each FrameworkList.xml file and remove the
RuntimeVersion="2.0.5" attribute.  Then restart VS and the references should
appear.

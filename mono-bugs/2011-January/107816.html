The xbuild MSBuildExtensionsPath search paths on Mac should include
the"external" directory:

/Library/Frameworks/Mono.framework/External/xbuild

The purpose of this directory is to provide a place for third-party installers
to place extensions where they will not be removed by Mono upgrades.
MonoDevelop patches this into the xbuild MSBuildExtensionsPath for builds
within MD but it would be useful to have it work outside of MD.

For the use case of hooking Gendarme into a Continuous Integration system, it
would be great to have a command line option that would log all defects to the
console *and* to HTML at the same time.

My proposal is to introduce a new flag called --console , which would be
claimed in the /help/ docs to be the default (that is, no change of behaviour
if you use it alone), but along with other options (such as --html) it would
keep sending the defects to the console.

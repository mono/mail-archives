Which Preview is this (and correspondingly which MonoDevelop version)?

With Preview 11.2, the Android SDK 9 r2, MonoDevelop 2.4.2 and the MonoDroid
development Add-in 2.4.2, I can't reproduce, as (for a major reason) there is
no longer a Properties/AndroidManifest.xml file by default, and the generated
obj/Debug/android/AndroidManifest.xml file doesn't specify an
//attribute/@android:icon attribute.

If, however, if I edit Project Options, go to the Build / MonoDroid Application
"tab", and set the Application Icon value to @drawable/Icon, then I see the
crash.

Changing the Project Options to reference @drawable/icon (note: lowercase)
allows the project to build.

When using a recent list of root certificates from Mozilla, 'mozroots' fails.
It crashes and uses more then 2 GB of memory.

Reproducible: Always

Steps to Reproduce:
1. Download a recent list of Mozilla root certificates from 
http://mxr.mozilla.org/mozilla/source/security/nss/lib/ckfw/builtins/certdata.txt?raw=1
2. Run '/usr/bin/time -v mozroots --import --sync --machine --file
certdata.txt'.
Actual Results:  
Mozroots crashes and a memory usage above 2GB is reported.

Expected Results:  
All certificates imported into the machine store.

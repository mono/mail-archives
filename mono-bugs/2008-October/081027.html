1). YAST2 is not accessible. it is therefore not possible to change the
firewall settings or change any system settings (or install other software);
for example, to install Ruby i had to use zypper.
2). many standard tools dont work - for example, opening the system monitor
results in the message "failed to execute child process - 'gnome system
monitor'". changing the system time resulted in a similar error.
3). there is no application available to read a txt or log file. if i try and
open such a file, i get "couldnt display .....(no application installed for
this file type'"
4). "vi" doesnt show color coded text. this is a feature i've used on standard
10.3 and 11 SUSE images.

Description of Problem:

My application runs with my own compilation for Ubuntu 8.04 but not with the
packages available at http://www.go-mono.com/mono-downloads/download.html for
Virtual PC and Windows. For some more problems i made a workaround in my
current version.


Steps to reproduce the problem:
1. Download a Mono version of my program at
http://sourceforge.net/projects/photoprinter/files/ (use earlier version to
show other bugs with workaround in the newest version.
2. run with "mono --debug JMGPhotoPrinterMono.exe -d" from the commandline
3. open a folder with images and try to resize the images or edit the images

Actual Results:

application crashes before showing the main form or when selecting a folder
with images.


Expected Results:

Images are shown, can be edited or resized

How often does this happen? 

always

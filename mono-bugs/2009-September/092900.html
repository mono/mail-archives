the executable on OSX 10.5.6. App is very unstable. I am able to make it crash
with the following actions. 

* Move window around the screen rapidly. 
* Move the mouse around the application window quickly. 
* Resize the window back and forth rapidly. 

Reproducible: Always

Steps to Reproduce:
1. Build default reference C# app in VS 2008. Copy to Mac. 
2. Run "mono TestApp.exe" 
3. Move the window around/move the mouse around the window/resize the window.

Application crashes.

Expected Results:  
Application should remain running.

Description of Problem:
Within MonoDevelop, one can choose the project to start when 'run' is chosen.
After restarting the IDE, this selection is forgotten.

Steps to reproduce the problem:
1. Create a solution with multiple projects
2. right-click a project (other than the first project in the tree), and choose
'set as startup'
3. save everything, and restart MonoDevelop

Actual Results:
The 'startup project' has reverted to the first project in the tree.

Expected Results:
The selected startup project should have been remembered from the previous
session.


How often does this happen? 
Consistently.

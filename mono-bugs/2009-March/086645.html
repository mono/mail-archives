Description of Problem:

While typing test input into Notepad, "Enter" seem to do nothing.  This is a
regression and worked with the mono-2.4rc3 mono on SLES 10 x86_64.  On SLES 10
x86_64, the same Notepad binary works correctly with the system mono and breaks
with the addon-cd mono-2.4rc1.

Steps to reproduce the problem:
1. On SLES 10 SP2 x86_64, install Mono Addon CD 2.4rc1 for SLES 10.
2. in a terminal, '. /opt/novell/mono/bin/mono-addon-environment.sh'
3. run 'Notepad'
4. Enter test characters including the "Enter" key to start new lines

(To test with system mono, do the above in a new terminal but exclude step 2)

Actual Results:

In the addon-CD environment, pressing "Enter" does not create a new line 90% of
the time.  Occasionally it does.

Expected Results:

A new line is created.

How often does this happen? 

Every time.

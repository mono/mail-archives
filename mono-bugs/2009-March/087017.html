Description of Problem:

Mono will not install properly. The installer gives "Installation Failed"
message. The problem occurs while running postflight. An error occurs on line
30 and line 33, the latter causing the installation to stop. The error at the
console always contains "Bad executable (or shared library)".

Steps to reproduce the problem:
1. Download the latest Mac OSX version of Mono onto a OSX 10.4 system.
2. Attempt an installation.

Actual Results:
Installation failure.

Expected Results:
Successful installation and the ability to use .NET apps in Mac OSX.

How often does this happen? 
Consistently.

Additional Information:
All .NET applications, including mono itself generate the exact same error when
they are run.

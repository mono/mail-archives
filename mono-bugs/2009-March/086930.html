Description of Problem:
If you try to view the events calendar in ClubWebSite under xsp2, it gives you
a 404 error.

However, if you do the exact same thing under mod_mono, everything works.


Steps to reproduce the problem:
1. Start xsp in the ClubWebSite directory
2. Browse to the Events Calendar
3. notice the 404
4. do the same thing while running it under apache
5. Notice there isn't a 404 and everything works properly


Actual Results:
404. Nothing pertinent is output to the console, and XSP keep running, so it's
not a crash.


Expected Results:
it should show the events calendar


How often does this happen? 
Every time under xsp, never under apache.

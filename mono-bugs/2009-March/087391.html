Description of Problem:
The mouse pointer is currently restricted to the guest os window in VMWare
Workstation 6.5 for the following openSUSE 11.0 VMWare image which includes
Mono 2.2:

http://ftp.novell.com/pub/mono/vmware/Mono-2.2_openSUSE-11.0.zip

Steps to reproduce the problem:
1. Start up guest os in VMWare Workstation 6.5
2. Click in the guest os window
3. Attempt to move mouse outside the guest os window (can't do it ... must
press CTRL+ALT to release mouse)


Actual Results:
Mouse pointer restricted to guest os window.

Expected Results:
Previous VMWare images played in VMWare Workstation did not restrict mouse
pointer to the guest os window; therefore, no restriction is expected.

How often does this happen? 
Everytime.

It appears as though everything in /Library/Frameworks/Mono.framework/Commands/
is also symlinked in /usr/bin.  This includes a lot of non-Mono stuff like
glib-gettextize.  Unfortunately, this glib-gettextize is broken and causes all
software which uses it to break:

/usr/bin/glib-gettextize: line 139: cd: /tmp/install/share/glib-2.0/gettext: No
such file or directory
gettext source directory '/tmp/install/share/glib-2.0/gettext' doesn't exist

Among the non-Mono binaries:

gdk-pixbuf-csource
gdk-pixbuf-query-loaders
glib-genmarshal
glib-gettextize
glib-mkenums
gobject-query
gtester
gtester-report
gtk-builder-convert
gtk-demo
gtk-query-immodules-2.0
gtk-update-icon-cache
libglade-convert
msgfmt
msgmerge
pango-querymodules
pkg-config
pkg-config.bin

There may be others, this isn't necessarily an exhaustive list.

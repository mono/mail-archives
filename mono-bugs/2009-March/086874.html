Description of Problem:
Sometimes while creating a post in graffiti, xsp will die with "Error: Thread
was being aborted".

This was found in Mono 2.4rc2. This didn't seem to be a problem in Mono 2.4rc1,
however the problem is sporadic, so we might just not have seen it.


Steps to reproduce the problem:
1. Unzip the graffiti zip file
2. run xsp in Graffiti/Web
3. Login to Graffiti with admin / mono
4. Click "Control Panel" in the lower right
5. Click "Write"
6. Fill out the form
7. Click "Publish"
8. If xsp doesn't die with the above error, then try steps 5-7 again. It
crashes for me about once every 5 tries.


Actual Results:
XSP dies with the error message above.


Expected Results:
XSP shouldn't die


How often does this happen? 
About once every 5 tries.

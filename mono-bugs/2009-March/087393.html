--- Comment #1 from Andrew Jorgensen <ajorgensen at novell.com>  2009-03-28 21:38:56 MDT ---
Thank you for this bug report.  It turns out this behaviour is fixed not by
installing the vmware tools but by using the vmxmouse mouse driver in your X
configuration.  Previous images were built using a different system and were
also adjusted after being built and before being published.  We've put a hack
in place on the new vmware image (Mono-2.4_openSUSE-11.1.zip in the same
location) to enable this by default under vmware.  Please try that new release
and let us know if you have any further problems.

The current builds are constructed using SUSE Studio http://susestudio.com/ and
contain VMware's open-vm-tools rather than their proprietary tools.

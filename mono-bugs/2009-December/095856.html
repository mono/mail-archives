The XDM dbus init script hacks are no longer in place in the VMware image.
Without these, gnome will fail to start randomly.

The hacks are to add dbus to the Required-Start and Required-Stop entries in
the xdm init script like so:

# Required-Start:    $remote_fs dbus
# Required-Stop:     $remote_fs dbus

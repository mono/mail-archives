Simple test for mdi child window issue

When you create a mdi application the child window cannot be moved inside its
container correctly. This issue was also tested on Mac OS X. Windows os was
Vista. The child window moves very slowly. A simple test is attached. The same
test was made using VB.NET with the same results. Executing the resulting
binary with the MS runtime results in a correct behavior. Compiling with mono
or from within VS2005 does not make a difference.

Compile the test with "mcs -reference:System.Windows.Forms Form1.cs"

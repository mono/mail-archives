Description of Problem:
When you start xsp2 and connect to it by web browser (http://localhoist:8080)
it will crash if no Default.aspx is found.

Steps to reproduce the problem:
1. Start xsp2 in empty folder
2. Type in browser address to connect to it (like http://localhost:8080)
3. It will show error that it can't find requested resource (what is right)
4. If you reload this address it will crash with "Segmentation fault".


Actual Results:
Crash during reload.

Expected Results:
Should display error message again (without crashing).

How often does this happen? 
Always.

--- Comment #4 from Zoltan Varga <vargaz at gmail.com>  2008-02-15 08:20:21 MST ---
You can get stack overflow handling by configuring mono
with the --with-sigaltstack=yes option. It is enabled by default, but it is
disabled by configure if some configure tests fail. mono --version includes the
line 
        SIGSEGV:       altstack
if it is enabled.

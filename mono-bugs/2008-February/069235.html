The following program produces incorrect output:

using System;
using System.Xml.Linq;

class Class1
{
    static void Main()
    {
        XElement a = new XElement("root",
                                  "Linux&Windows");

        Console.WriteLine (a);
        Console.WriteLine (a.Value);
    }
}


It currently displays:
<root>Linux&amp;Windows</root>
Linux&amp;Windows

It should display:
<root>Linux&Windows</root>
Linux&Windows

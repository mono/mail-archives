conditional somehow. But currently Mono is statically assigning INTL as
libintl.so on Solaris without allowing the user to override this, so I assume
it currently fails unless the user has a GNU libintl.so in LD_LIBRARY_PATH; or
would it always fail due to presence of the system libintl.so?

Repro:

using System;

struct Foo {

        int bar;
        int baz;

        public int Bar {
                get { return bar; }
        }

        public Foo (int baz)
        {
                this.baz = baz;
                bar = Bar - 1;
        }
}

Result:
repro.cs(6,13): warning CS0414: The private field `Foo.baz' is assigned but its
value is never used
Compilation succeeded - 1 warning(s)

Expected:
error CS0188: The 'this' object cannot be used before all of its fields are
assigned to.

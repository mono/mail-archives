The iPhone Simulator supports a command line to enable displaying alpha blended
layers. This helps for visualizing how much alpha blending your application is
causing.

It would be nice to have integrated into MonoDevelop rather than something
people have to do manually.

I found this via Instruments and I believe there are more options available as
well since Instruments has much more available. I have only found this one so
far:

CA_COLOR_OPAQUE=1
/Developer/Platforms/iPhoneSimulator.platform/Developer/Applications/iPhone\
Simulator.app/Contents/MacOS/iPhone\ Simulator

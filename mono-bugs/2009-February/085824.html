In the process of trying to make the bug reproducible, I managed to discover
another bug that belongs to this bug heading.  Here is the code:

using System;
using System.Linq.Expressions;

namespace MonoBug
{
   class Program
   {
      public void testFunc() {}
      static void Main()
      {
      // the commented line doesn't have any problems...
      // Expression<System.Func<Program, Action>> action = (d =>
      //   (Action)Delegate.CreateDelegate(typeof(Action), d,"testFunc" ));

         Expression<System.Func<Program, Action>> action = (d => d.testFunc);
         var t = action.Compile();
         Program p = new Program();
         t(p);
      }
   }
}

the expression/compile() pair.  But under the mono runtime, it creates this

Assigning the F1 key as a shortcut to a menu item doesn't work - i.e. the event
handler for the menu item is not fired. It seems to work for any other function
key. (And works in the same executable running under .NET)

Reproducible: Always

Steps to Reproduce:
1. Assign F1 as the shortcut key for a menu item
2. Attach an event handler to that menu item
3. Press the key - nothing happens. Activate the menu item with the mouse - it
works.

Description of Problem:

When debugging programs in monodevelop 2.0b1 on opensuse 11.1 x86_64, a warning
is printed out in the MonoDevelop External Console:

'&"warning: GDB: Failed to set controlling terminal: Operation not
permitted\n"'

Steps to reproduce the problem:
1. Create a new solution using the C Console Project template
2. place a break point on the printf line
3. click on menu items "Run -> Debug"

The console is opened with the notification.

Try to get a proper backtrace:

1) become "root"
2) stop apache and assure that apache is really stoped
3) start gdb
4) type

   -r /usr/sbin/httpd2 -X
   c

6) access a page

When httpd2 segfaults, type "t a a bt" at gdb's prompt and paste the backtrace.

Sorry for that, the managed attribute is an alias for useruntimeengine that I
introduced in CVS.

Just replace "managed" with "useruntimeengine", and do not specify any target
on the command line.  That will automatically run the tests using all available
CLRs.

If you want to explicitly target a give framework use the following syntax:

nant -t:mono-1.0 mono-1.0 clean run

(PS. I know this syntax sucks, but I don't have enough time to change all my
local build scripts ...)

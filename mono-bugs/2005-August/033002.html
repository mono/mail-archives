+When running the .bin installer for release 1.7 and 1.8, the license page

+without a mouse from using the graphical installer.  
+

+1. Run the .bin installer on a linux platform
+2. "Next" through to the License Agreeement page
+3. Attempt to use the keyboard to "Agree" to the License

+except the radio buttons for "Accept/Decline"

+The tab select should INCLUDE the "Accept/Decline" radio buttons
+
+How often does this happen? 
+On every attempt to run the installer
+
+Additional Information:
+I only ran the installer on Fedora Core 3, Core 4 Test 3, and Core 4, I 
+did not attempt to install it on any other platform though I suspect the 
+problem is NOT platform dependent.  Altering the installer to include the 
+radio buttons in the tab order should be a minor and quick change.

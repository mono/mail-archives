On .net the following sentence in an aspx page will go fine, but in mono it
fails using a substring out of index:
WebConfigurationManager.GetSection("system.web/authorization", "~")

The code in
mcs/class/System.Web/System.Web.Configuration_2.0/WebConfigurationManager.cs
should be:
               if (VirtualPathUtility.IsRooted (path)) {
                    if (path [0] == '~')
-                       relPath = path.Substring (2);
+                       relPath = path.Length > 1 ? path.Substring (2) : "";
                    else if (path [0] == '/')
                        relPath = path.Substring (1);
                    else
                        relPath = path;
                } else
                    relPath = path;

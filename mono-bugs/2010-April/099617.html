Description of Problem:

When opening a project in Xcode the Info.plist file isn't generated.

This is a huge problem when your app depends on setting in that file. Consider
Universal apps. Consider apps that need to have their bundle identifier set.

Please generate the Info.plist the same way you do when uploading to the device
- merge project settings and the project's Info.plist file.

As it stands, every time I want to test the app in Xcode (if it is crashing)
then I have to go and put all these settings in by hand.

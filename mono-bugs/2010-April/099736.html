User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; fr; rv:1.9.2.3)
Gecko/20100401 Firefox/3.6.3

I have a TabControl, with a TabPage set to AutoSize = true, so that scrollbars
appear when the content is too big.

I put a Panel inside this TabPage, which has the exact same size as the
TabPage.

Therefore, the scrollbars shouldn't appear since the content is supposed to
fit.

With .NET: no scrollbars
With Mono: vertical and horizontal scrollbars appear (they disappear if you
enlarge the window)

Note that scrollbars disappear if I set the Panel Dock property to Fill (but
setting it to Fill isn't always desirable).

Reproducible: Always

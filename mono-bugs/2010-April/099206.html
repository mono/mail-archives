Description of Problem:
When creating a new profile for a remote server, monotools does not query the
servers automatically 

Steps to reproduce the problem:
1. Open Visual Studio.
2. Create a new winforms project.
   File
   New Project
   Visual C#
   Windows
   Windows Forms Application
3. [The Linux server should be running.]
4. Mono->Run in Mono
5. Create Profile --> Select Remote Server


Actual Results:

The servers that are running are not shown.

Expected Results:

The servers that are running should show up so that the user can select from
there

How often does this happen? 
Every time

Additional Information:

if you add the IP address of the server manually it works and stays in the
profile.

After downloading and installing the monodroid SDK build 8196, all my monodroid
apps crash when starting them in the emulator. 

Reproducible: Always

Steps to Reproduce:
1. Create a new project using the build 8196
2. Press F5 in Visual Studio 2010
3. Wait for the application to start.
Actual Results:  
The application runs, but then crashes shortly after, without the user doing
anything. Visual Studio 2010 also hangs.

Expected Results:  
The software should start my application and attach the debugger.

Logcat indicates that one of the monodroid libraries: A4/libmono.so could not
be loaded.

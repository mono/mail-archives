It would be useful if the KeyEventArgs class used Keycode instead of int for
specifying the key, to help avoid having to do conversions like this:

_urlText.KeyPress += (sender, e) =>
{
    if (e.KeyCode == (int)Keycode.Enter)
    {
          // ...
    }
};

Reproducible: Always

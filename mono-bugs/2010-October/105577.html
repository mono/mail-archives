When using JavaDictionary<MyObject, MyObject> to store objects, the call will
produce the exception in the Summary.

1. Create a custom object and override ToString, Equals and GetHashCode
2. Save the object into a JavaDictionary<MyObject, MyObject> or
JavaList<MyObject>


Actual Results: Exception


Expected Results: Storage of object in Dictionary.


How often does this happen? 
Every time

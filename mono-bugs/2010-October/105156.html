--- Comment #1 from Marek Habersack <mhabersack at novell.com> 2010-10-20 13:49:05 CEST ---
Please do not abuse the Critical severity in the bug reports - it will NOT
speed the resolution up. 
Instead of bumping the severity, please provide information which may actually
help to solve the issue:

 - HTTP 500 error stack traces or debugging information from apache log
(recompile mod_mono with debugging and make sure debugging is on for the
virtual host in question)

 - a short test case which reproduces the issues you are experiencing (saying
"it happens randomly" is not really useful as the developer has no way of
reproducing the issue.

 - make sure you're using mod_mono and mod-mono-server from _the same Mono
version_

After you can provide the above information, please reopen the bug. Thanks!

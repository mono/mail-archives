(The description & repro steps below apply to "Add-in Repository Management"

I opened "Add-in Repository Management" and click "Add" to create add a new
repository. When I enter wrong URL and cancel the operation, I still get an
error dialog telling me that I entered a wrong URL (I know that, that's why I
wanted to cancel the operation).

Reproducible: Always

Steps to Reproduce:
1. Open "Add-in Repository Management" in MonoDevelop
2. Click "Add" and enter some URL that's not actual repository e.g.
http://error
3. Click "OK" and then immediately "Cancel" in the dialog that appears
Actual Results:  
It gives an Error dialog telling me that the URL doesn't contain repository.

Expected Results:  
Error message should not appear - the operation has been cancelled.

Description of Problem:
When you're working in Notepad.exe, every so often, you'll get an X11 BadWindow
error message with a stack trace (see attached).

This was found on Mono 2.2p1.

The error does _not_ crash Notepad.exe, so this bug is of minor importance.


Steps to reproduce the problem:
1. Run Notepad.exe in a terminal (so you can see the error).
2. type "test" into the window
3. Increase the font to 36
4. save the file
5. keep doing different combinations of the above until you see the error.


Actual Results:
X11 BadWindow error is output to the window.


Expected Results:
No error.


How often does this happen? 
VERY sporadically.

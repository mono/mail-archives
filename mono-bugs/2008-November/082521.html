An Ubuntu developer has isolated a bug in Mono which prevents compilation on
ARM when paired with a very recent glibc.

The patch for this can be found at
http://svn.debian.org/wsvn/pkg-mono/mono/trunk/debian/patches/armel-glibc-2.8.dpatch?op=file&rev=0&sc=0
- the bug applies to all versions of Mono, though the above patch will only
apply to Mono 2.0+ without manual tweaking.

Many Windows apps come with a way to install them silently, i.e. without user
interaction.

For instance, the .net framework from Microsoft supports this kind of
installation; see
http://blog.stealthpuppy.com/unattended/unattended-install-microsoft-net-framework

Please add a way to do this from the Mono installer.  And while you're at
it, make it easier to discover by adding a /? option that prints out the
supported commandline options.

When a user runs any Analyze for Mono Migration (MoMA)

We should check if they have referenced assemblies that are not getting
analyzed.  This should exclude a list of known dependencies that we know we
have in Mono

We can show a dialog that says something along the lines of:
The project that you are analyzing has references to libraries which are not
included as source projects in this solution.  Analyze for Mono Migration will
only scan project references for Mono compatibility.  To analyze other
references, including 3rd party libraries, consider analyzing the compiled
version of this project with the stand-alone version of the Mono Migration
Analyzer - instructions and download available at http://mono-project.com/MoMA

[list of non-analyzed assemblies]

We should include a checkbox to allow the user to not be warned again for this
project.

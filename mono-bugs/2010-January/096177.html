The server packages for Mono Tools are still called:
monovs-server
monovs-server-gui

We need to make these discoverable as:
monotools-server
monotools-server-gui

Similarly, the commands should change:
monovs-server --> monotools-server
monovs-gui-server --> monotools-server-gui

I'm fine with doing this with symbolic links and meta packages to avoid
breakage that would be caused by renaming.

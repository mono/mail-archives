Description of Problem:
multi-line selection in monodevelop with a mouse leaves entire sections
un-selected

1. open a code page with at least two pages of code
2. select and drag down so that you select code and the page has to scroll


Actual Results:

randomly selects chunks

Expected Results:
that it would work and select from your start of mousedown to end of mouseup

How often does this happen?

Description of Problem:
Currently, SUSE Studio integration only supports building an image from an RPM.
 We should enable users to skip the RPM creation step, and just upload their
compiled project to the VM.  We can upload as a zip and let SUSE Studio unzip
the project, but we'll need to let the user specify.

If we focus on web projects first, then we could really make this really
staightforward, and:
1) create the zip file
2) create the apache config
3) default the upload location and group/owner/permissions for both

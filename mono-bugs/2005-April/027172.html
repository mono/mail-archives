+Summary: mkbundle mangles arguments
+
+When mkbundle creates the new arguments to pass to mono, it does not
+allocate enough memory.  It uses the following:
+
+        char **newargs = (char **) malloc (sizeof (char *) * argc + 2);
+
+instead of:
+
+        char **newargs = (char **) malloc (sizeof (char *) * (argc + 2));

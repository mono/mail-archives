MarshalByValueComponent.Dispose doesn't raise Disposed event.
This affects objects that inherit from MarshalByValueComponent (such as
DataView).

Reproducible: Always

Steps to Reproduce:
1. Create a new instance of MarshalByValueComponent (or subclass such as
DataView)
2. Hook Disposed
3. Call Dispose()
Actual Results:  
Disposed event is not raised

Expected Results:  
Disposed event should be raised

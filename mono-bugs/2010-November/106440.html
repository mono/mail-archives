Packages are installed automatically in many linux distributions. Some
applications can rely on database driver, which is provided as separate
package. In order to install such separete package there is need to have a
command line tool to modify machine.config file(s).

I think that this new utility can be similar to gacutil in usage (install,
uninstall and list). 

It is better to place such utility into mono package, because it can be usefull
in many other distributions for scripting.

See http://bugs.gentoo.org/show_bug.cgi?id=334927 for details.

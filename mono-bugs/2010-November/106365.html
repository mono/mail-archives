This one is easy to reproduce:

using System;

namespace Test
{
    class MainClass
    {
        public static void Main (string[] args)
        {
            if (true) {
                string h = "hi";
                Console.WriteLine (h);
            }
        }
    }
}

Set a breakpoint in the "Console.WriteLine" line. When the debugger stops
there, try stepping to next line. The debugger crashes.

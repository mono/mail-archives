ListChanged not fired on DataView.AddNew()

Reproducible: Always

Steps to Reproduce:
            DataTable table = new DataTable();
            DataView view = table.DefaultView;
            ListChangedEventArgs lastListChanged = null;
            view.ListChanged += (sender, e) => { lastListChanged = e; };
            DataRowView row = view.AddNew();
            Assert.AreEqual(lastListChanged.ListChangedType,
ListChangedType.ItemAdded);

Actual Results:  
DataView.AddNew() does not fire the ListChanged event

Expected Results:  
DataView.AddNew() should fire the ListChanged event with
ListChangedType.ItemAdded

--- Comment #1 from Jonathan Pobst <jpobst at novell.com> 2010-06-23 16:54:55 UTC ---
Yeah, the deal with this is 1.1 has this feature, and it led to a lot of user
confusion.  People would check the box, and then could never get the profile
chooser to come back up.  (Probably ~15% of support issues I handled.)

The option to bring back the chooser is in our settings panel, and I documented
it in the FAQ, but both are too hard to find.

http://mono-project.com/FAQ:_MonoVS

When I redesigned the chooser for 2.0, I dropped this feature.  If we can think
of some intuitive way to fix this, I have no problem putting the option back.

The best I could do to help out this situation in 2.0 is your last used profile
will be selected by default the next time you run, so you can just hit OK each
time.

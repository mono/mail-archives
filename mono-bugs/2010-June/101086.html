Saving changes to a profile in one instance of Visual Studio will not have
those changes reflected in other copies of Visual Studio.

Steps to reproduce
1) Open VS
2) Open another copy of VS (as though you are editing a different solution)
3) Open Mono .. Debug in Mono
4) Edit and save a profile, changing its IP, Precompile settings, etc
5) In the second instance of VS, open Mono .. Debug in Mono, and choose the
recently edited profile

Expected Results:
VS will use the updated settings for the profile

Actual Results:
VS will attempt to use the old IP, Precompile settings, etc

(this is the end of the build output)

/Developer/usr/bin/dsymutil /Users/nic/code/Tweet
lightning/Tweetlightning/bin/iPhone/Release/Tweetlightning.app/Tweetlightning
-o /Users/nic/code/Tweet
lightning/Tweetlightning/bin/iPhone/Release/Tweetlightning.app.dSYM
Process exited with code 1, command:
/Developer/usr/bin/dsymutil /Users/nic/code/Tweet

Reproducible: Always

1. Make a project (or take an existing one)
2. Change any folder in the tree to have a space ("/code/FooBar" to "/code/Foo

3. Compile. Failed.
Actual Results:  
Fails:

/Developer/usr/bin/dsymutil (path info here)

it should compile. Does the call need quotes around it?

Following the bug report guidelines, you actually need to fill one bug report
per issue. So you will need to fill bug reports for the following:

1. AutoScroll does not invalidate/update client area correctly
2. Printing crashes the application
3. ToolStripComboBox ignores Handled on KeyPress
etc.

The rationale behind this is that we want to independently keep track of all
the issues, and avoid having 15 issues in the same bug report, which could
cause confusion, lack of cleaness and other similar issues ;-)

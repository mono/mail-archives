Description of Problem:
When you click "Open in Web Browser" on a remotely run web app's tray icon, it
takes 30 seconds to open.

This was found in MonoVS 0.1p4.


Steps to reproduce the problem:
1. Create a new solution based on the ASP.Net template project
2. Add "Hello World" in between the <div> tags
3. Click "Mono / Run remotely in Mono"
4. Choose the host to run it on
5. Notice the windows tray icon for the remotely running web app
6. Wait for the web browser to open
7. Close the web browser
8. Right click on the windows tray icon and select "Open in Web Browser"
9. Notice that it takes 30 seconds to open


Actual Results:
It takes 30 seconds for "Open in Web Browser" to open

Expected Results:
It should be almost immediately

How often does this happen? 
every time

PROBLEM STATEMENT

There is a combo box on the FontDialog that allows the user to select a font
color.  The problem is that there are two "Aquamarine" items in the combo box.

REPRO

1.  Run the attached (IronPython) test case application
2.  Click on the "Click me" button in the test case application
3.  Click on the color combo box to drop it down (the default selection is
"Black")
4.  Scroll through the items of the color combo box

RESULTS

There are two "Aquamarine" items in the color combo box

EXPECTED RESULTS

There should be no duplicate items

COMMENTS

For what it's worth, Windows Vista has no duplicate items, but it has many
different colors also.

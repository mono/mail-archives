Description of Problem:

WindowsXP x86_64 with monovs plugin r1829

Scanning with MoMA when no solution is open gives an error dialog with the
following error message:

---------------------------
System.NullReferenceException: Object reference not set to an instance of an
object.
   at Novell.MonoVS.BaseRunner.ClearStatusBar()
   at Novell.MonoVS.RunInMomaRunner.Run()
   at Novell.MonoVS.Package.RunInMomaCallback(Object sender, EventArgs e)
---------------------------

Steps to reproduce the problem:

0. install monovs plugin r1829
1. Open VS 2008
2. Do not open a solution
3. Scan with MoMA
4. Close first dialog which says there is no solution
5. System.NullReferenceException is present.

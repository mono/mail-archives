--- Comment #2 from Geoff Norton <gnorton at novell.com> 2011-03-18 21:30:22 UTC ---
As I mentioned on IRC.  Linking with any combination of thumb libraries in
MonoTouch 3 is completely non-deterministic since the iOS 3.0 SDK release due
to changes introduced in apples linker.

MonoTouch 4 is the only release that will support linking thumb libraries into
a MonoTouch application.

If problems persist in the MonoTouch 4 alpha, please reopen this bug.

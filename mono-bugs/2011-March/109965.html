HttpResponse.WriteFile(string) method maps files path to absolute path.
For example I have file "style.css" in root directory of my website.

In MS ASP.Net and in mono 2.6.x and older this part of code works just fine:
string filePath = Server.MapPath("/style.css");
// filePath = /srv/www/vhosts/mysite.com/style.css
Response.WriteFile(filePath);

But in Mono 2.10.1 this part of code returns error:
Could not find file

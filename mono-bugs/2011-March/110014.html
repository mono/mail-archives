Description of Problem:
Enumerating a ConcurrentBag<T> yields uninitialized elements which have not
been added to the collection.

Steps to reproduce the problem:
1. Create an instance of ConcurrentBag<T> and add some elements.
2. Enumerate the instance.

For example:
var concurrentBag = new ConcurrentBag<string>();
// add elements
int elementCounter = 0, nullCounter = 0;
foreach(string element in concurrentBag) {
    if(element == null) {
        nullCounter++;
    }
    elementCounter++;
}

Actual Results:
  concurrentBag.Count: 1000
  Total enumerated elements: 2048
  Uninitialized elements: 1048

Expected Results:
  concurrentBag.Count: 1000
  Total enumerated elements: 1000
  Uninitialized elements: 0

Additional Information:
Sample MonoDevelop project file attached.

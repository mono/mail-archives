Performing a downgrade install breaks the directory structure of the framework.

Reproducible: Always

Steps to Reproduce:
1.Install Mono 2.10.x from the OS X package
2. Install Mono 2.8.x from the OS X package
Actual Results:  
In /Library/Frameworks/Mono.framework/Versions the 'Current' link points to
2.10 but it is no longer a valid install.

Expected Results:  
Expected either 1. the package to not install the older version or 2. the
package to install the older version and move the "Current" link to point to
the actually installed lower version.

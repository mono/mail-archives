Eric, you should set LD_LIBRARY_PATH to your firefox directory. This is already
done if you start it from monodevelop, as monodevelop is based on Gecko as
well, e.g. 

export LD_LIBRARY_PATH=/usr/lib/firefox-2.0.0.16:$LD_LIBRARY_PATH
mono ./NetRadios.exe

--- Comment #4 from Sebastien Pouliot <sebastien.pouliot at gmail.com> 2011-07-25 23:42:40 UTC ---
The executable name is used as the value of CFBundleExecutable. Apple
documentation [1] states that:

<quote>The value should not include any extension on the name.</quote>

So I guess it's blind luck that this work on devices (while failing on the
simulator). Leaving the bug open until we add this fact into our MonoTouch
documentation.

[1]
http://developer.apple.com/library/mac/documentation/General/Reference/InfoPlistKeyReference/InfoPlistKeyReference.pdf

tested system with workaround:

Win 7 64 Bit ,  Mono 2.6.7
mysql .net Mono Connector version: 6.3.6.0

On Fedora 15 64 Bit it works with the same connector version 6.3.6.0 even
with the latest mono version (2.10.2). Only thing to do is changing the
filename mysql.data.dll to MySql.Data.dll

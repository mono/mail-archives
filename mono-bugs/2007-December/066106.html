--- Comment #1 from Paolo Molaro <lupus at novell.com>  2007-12-07 07:07:14 MST ---
It looks like you're running gmcs.exe with the MS runtime.
You need to run it with the mono runtime as the MS runtime is incomplete and
buggy.
As for running mono, you didn't show any exception output: please run you
program from a console and report that, from the log above it doesn't seem
you're running any managed program with it. You may also want to run mono from
a directory different from the MS CLR directory as it may try to pick up the MS
mscorlib.dll instead of its own.

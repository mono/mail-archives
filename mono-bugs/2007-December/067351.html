Modifies gacutil to match MS behavior

Modified our gacutil to:

* ignore the /nologo option for command-line compatibility with MS gacutil

* report the number of assemblies that were (un)installed in "list" mode

* report success in non-list mode when the assembly to uninstall is not in the
GAC

* allow a partial version number to be specified for the unininstall or list
command

Full changelog is available in the patch.

libmoon doesn't need cairo's PNG support

So we could reduce Moonlight surface and save about 26kb (see libgdiplus) by
configuring our own Cairo with --disable-png

However, right now, xaml2png uses the embedded Cairo to write PNG files, so
this option cannot be used (without modifying the tool).

System.Web.UI.Webcontrols.Webcontrol does not implement the protected internal
property isEnabled that is found in the .NET 2.0 class libraries. This property
is used to recursively calculate a webcontrol's enabled state from its parent
controls. This is used by a number of third party control vendors and is a
major stumbling block in there use on a mono runtime.

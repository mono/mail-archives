If you stop a program that is being debugged using the blue "stop" square
button in VS, you get a stack trace on the Linux side.

1) open Blogengine-1.5 sln
2) set a break point on line 18 of default.aspx.cs: if (Page.IsCallback)
3) debug remotely
4) hit breakpoint
5) continue
6) stop app using blue square (stop) in VS GUI.
7) stack trace on Linux appears.

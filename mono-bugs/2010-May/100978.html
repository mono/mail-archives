does not work correctly:

using System;

class Test
{
     static void Main ()
     {
         Console.WriteLine (Environment
             .GetFolderPath(Environment.SpecialFolder.Personal));
     }
}

$ HOME=foo mono test.exe

This means that g_get_home_dir() is falling back to getenv("HOME").

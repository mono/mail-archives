add an ssl webreference.

2. try to add an ssl web reference to the project



Actual Results:
Error getting response stream (Write: The authentication or decryption has
failed.): SendFailure

Expected Results:
an web reference should be added to the project

stops working. If I downgrade back to 2.2.4 and 2.2 this works again.

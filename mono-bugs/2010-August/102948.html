In a UserControl-derived class, the method call
ControlPaint.DrawReversibleLine(...) should draw a line on the screen using a
XOR raster operation, but it draws somewhere under the form, and therefore its
output appears only for a tiny fraction of a second.


Reproducible: Always

Steps to Reproduce:
1. Create a class derived from UserControl, whose method OnPaint calls
ControlPaint.DrawReversibleLine(0, 0, 1000, 1000, Color.Black)
2. Insert an instance of such control in a Form that is opened when the program
runs.
3. Run the program. To let you see the lines drawn, insert the call in a loop
repeated 10000 times.



Expected Results:  
It should draw a line

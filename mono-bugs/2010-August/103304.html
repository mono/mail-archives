When reading input from the console, the first keystroke in the line is not
always echoed on the screen

Reproducible: Sometimes

Steps to Reproduce:
1. Write code that calls Console.ReadLine()
2. Run your code
3. Input text, for example "1234"
Actual Results:  
Whatever variable you store the result will get the correct value, but only
"234" will be echoed to the screen.

Expected Results:  
Same as above, except that all keystrokes are echoed to the screen.

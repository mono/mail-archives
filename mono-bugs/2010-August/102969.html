F5 and CTRL-F5 should not require the user to select a device if the previous
device used is running.  Probably the easiest solution would be to add a
checkbox to the form:
"Always use this device if it is running."

We may have to add a MonoDroid menu at some point if we think users will want
to change this without shutting down a device.

It would be nice if everytime I hit debug I didn't have to choose the device. 
I would like to be able to right click a device and set it as the default
device.  If when debugging, the default device exists, use it, otherwise show
the list.

To deselect a device as default, there could be a hotkey to hold while
debugging is starting which would bring up the device list regardless of
defaults, or you could integrate into the visual studio tools 'Debug' menu with
an option to show the device list dialog, bypassing the default one.

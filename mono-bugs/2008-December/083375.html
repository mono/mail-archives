MAC 10.5: Column headers along with Column name is disappearing when we click
on Column name.

Note : This bug has been reproduced only on MAC.

Steps to reproduce the defect:

1. Run the application that have been attached.
2. Click on 'Root' node.
3. It will add dynamic nodes to the rootnode.

Note : Some times when we click on root node or child nodes the application
will get crash and it throws an exception in the terminal windows.
Which has been raised a bug. Please refer -
#https://bugzilla.novell.com/show_bug.cgi?id=438650

4. When root node addes dynamic nodes to it (count becomes >0), it displays the
nodes in the right side list view.
5. Now Click on list view column headers.

Actaul Result:

Listview column headers will disappear and gap will form in the listview's
column headers place.

Expected Result:
Listview should perform the column click event as expected and it should
display the column headers properly and it should sort the listview items.

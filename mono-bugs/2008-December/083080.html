--- Comment #1 from Marek Habersack <mhabersack at novell.com>  2008-12-08 05:58:40 MST ---
This is not a bug in Mono but an issue with the configuration of your
application. As there is no attachment with the app I cannot point to the exact
config file, but the issue is that your <siteMap> section is found NOT in the
top-level web.config (i.e. the application root web.config). As soon as you run
xsp2 with its --root parameter pointing to the directory in which web.config
with the <siteMap> section lives, it should work fine.

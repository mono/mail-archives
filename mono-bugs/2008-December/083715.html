That is not the default for Windows Vista.  It is controlled by:
Control Panel->Ease of Access Center->Make the keyboard easier to
use->Underline keyboard shortcuts and access keys, which defaults to off.

What this means is the underlines will not show up until you press Tab or Alt. 
Once you press Tab or Alt, it decides you are using keyboard navigation and
will then show the underlines.

If you change the setting listed above to on, then the underlines should always
show.

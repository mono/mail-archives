Repro:

Create a resx file with no resources (but valid header)

resgen empty.resx

Result: 
A empty.resources file with 0 length.

Expected:
A valid empty.resources file with 0 resources.

Note:
This breaks VS2005 support in MD, VS2005 will sometimes create resx files with
no resources, and MD will currently fail to compile those projects as is.

2. Try to compile any file with field initializer, or run compiler test suite

The critical line is this

+                       if (resolved != null)
+                               resolved.EmitStatement (ec);

because resolved is this, changing it to 

                        if (resolved != null) {
                                if (resolved == this)
                                        base.EmitStatement (ec);
                                else
                                        resolved.EmitStatement (ec);
                        }

fixes the problem

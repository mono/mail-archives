Description of Problem:

OpenSUSE 11.0 x86_64
mono-core-2.6.3-38.1
reflector-5.1.7.0 (http://reflector.red-gate.com/download.aspx)

Steps to reproduce the problem:
1. Run reflector
2. Choose mono profile from list (doesn't matter which one)
3. process hangs with no reflector window appearing


Actual Results:
Hangs

Expected Results:
Reflector should run

How often does this happen? 
Every time

desktop which is openSUSE 11.0.

newer versions, tells you about any upgrades and then exits rather than
running.

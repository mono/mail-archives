To run vbnc in Mono 2.4, you can:
1. cd "\Program Files\Mono-2.4\usr\lib\mono\2.0"
2. run vbnc

The place where the bat file is supposed to be is here (but it's not):
   C:\PROGRA~1\Mono-2.4\bin

Description of Problem:
The mojoPortal CSS is not getting sent to the client sometimes. This seems to
happen sporadically, but we've seen it on:

* Mono 2.2rc1 xsp on SLED 10 SP2 i586
* Mono 2.2rc1 mod_mono on openSUSE 11.0 x86_64
* Mono 2.2rc1 xsp on Mac OSX 10.5 i586
* Mono 2.2rc1 xsp on Mac OSX 10.4 ppc


Steps to reproduce the problem:
1. Access the mojo front page
2. hit <shift> refresh
3. If the CSS comes down properly, then try restarting apache / xsp and doing
it again. The first access almost always has CSS, but the 2nd doesn't.


Actual Results:
No CSS

Expected Results:
CSS

How often does this happen? 
Very sporadic, it'll break for awhile, and then work again. I've tried
restarting xsp and postgresql and then doing it again to see if that would make
it work, but it still broke.

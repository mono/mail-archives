Description of Problem:

Running mono-2.4p1 on opensuse 11.1 x86_64.

Creating a new ad fails with an error: "An error occurred. Please contact the 
site administrator."  In the apache2 error.log, the following line is added
when this error occurs:

[Fri Jan 23 11:11:46 2009] [error] command failed: failed to send file (file
data)

This error occurs when running under either apache2 or xsp2.  This is a
regression as it worked fine on mono-2.2.

Steps to reproduce the problem:
1. install the Mono_ASP.NET_ClassifiedsStarterKit-1.0.0/ from Mono:Community
repository
2. Configure to run on Apache by copying
/usr/share/mono/asp.net/conf/Mono_ASP.NET_ClassifiedsStarterKit-1.0.0/Mono_ASP.NET_ClassifiedsStarterKit.conf
to /etc/apache2/conf.d/
3. start apache2
4. log in and create an ad


Actual Results:
failed to create ad

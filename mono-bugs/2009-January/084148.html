Visual Studio is using a the XML generated by WebService.asmx?disco to detect
web services and the <link rel="alternate" type="text/xml"
href="WebService.asmx?disco"/> element to detect the discovery URL. It don't
needs the non-public requests. It's <link> tag processing is a bit buggy so
some modifications may have to be made to DefaultWsdlHelpGenerator.aspx, but
generally it needs no non-public stuff for this to work.

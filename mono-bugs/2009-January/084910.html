Description of Problem:

too. See the attached screenshots.

This does not happen on Mono 2.2, so this is a regression.

Relevant versions of software:
Mono 2.2:
   mono-winforms-2.2-14.1
   libgluezilla0-2.2-11.1
   mozilla-xulrunner-1.8.0.14eol-0.5

   mono-winforms-2.4-3.1
   libgluezilla0-2.4-10.1
   mozilla-xulrunner-1.8.0.14eol-0.5


Steps to reproduce the problem:
1. open UsingWebBrowser
2. Go to mono-project.com (make sure that the window is small enough to
necessitate scroll bars)
3. notice there aren't any scroll bars


Actual Results:
no scroll bars

Expected Results:
scroll bars

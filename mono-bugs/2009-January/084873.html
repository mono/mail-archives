Description of Problem:
If you click the home button in mojoPortal, it takes you to an error screen
that says:
"We're sorry but we could not find the page you are looking for. Please try the
Site Map."

This does not happen under Mono 2.2 with the same version of mojoPortal
(mojoportal-2.2.7.9-1.6).


Steps to reproduce the problem:
1. Install and configure mojo to run under mod_mono
2. Go to the main web page and click on the "Home" button at the top of the
screen.
3. Notice the error.


Actual Results:
"We're sorry but we could not find the page you are looking for. Please try the
Site Map."


Expected Results:
It should take you to the home page.


How often does this happen? 
every time.

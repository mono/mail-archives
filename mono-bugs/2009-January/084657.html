--- Comment #1 from Andrew Jorgensen <ajorgensen at novell.com>  2009-01-15 14:23:58 MST ---
We moved to using the open-vm-tools from vmware which seem to not work quite as
well as the commercial tools did.  In our own tests mouse-out did work with the
most recent versions of vmware server and vmware player but for at least one of
us it only worked when mousing out to the right or bottom.  Mousing in didn't
really work, but you can click in, so it's not that bad.

To use the commercial tools we had to run extra, manual steps after building
the image.  The open-vm-tools were good enough that we felt it was a fair trade
off.

I do appreciate the bug report though, and I'm sorry the tools aren't working
well for you on this release.  VMware is continuing to develop the tools so
they should improve in future releases.  Also one our team had much better luck
with open-vm-tools on openSUSE 11.1, which our Mono 2.4 appliance will be based
on.

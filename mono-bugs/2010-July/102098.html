--- Comment #1 from Sebastien Pouliot <spouliot at novell.com> 2010-07-12 17:44:06 UTC ---
Make sure your permissions are ok. Either:

(a) try running Apache as the current user (which should have access to the
machine store since it executed mozroots and certmgr without error).

(b) login as the apache user and try to access the certificate files (you
should be able to read them). Do "man certmgr" to see where the machine certs
are located.

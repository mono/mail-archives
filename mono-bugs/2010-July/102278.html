--- Comment #1 from Geoff Norton <gnorton at novell.com> 2010-07-17 17:46:22 UTC ---
The reason you see the difference on device is we run the linker to remove
unused code. If the code is only used by reflection we cannot determine this
and the linker will remove it. You can use the Preserve attribute to control
this behaviour on a case by case basis if needed

Description of Problem:
SLES 11 sp1 x86 64
MonoX 2.6.7
Reflector 5.1.7.0
Reflector does not display search results, and it crashes when hovering over
search buttons.

Steps to reproduce the problem:
1. Run:  'mono /mnt/mtester/test/manual/reflector/Reflector.exe'
2. Select .NET 2.0 assembly and open
3. menu: View -> Search
4. Enter 'System.Data' in the search field on the top right. Search results are
not displayed.
5. Hover over the search buttons to the right of the search box. Reflector
crashes with no error.


Actual Results:
Search results are not displayed.
Reflector crashes with no error.

Expected Results:
Search results should be displayed and Reflector should not crash.

How often does this happen? 
Always. Sometimes search locks up Reflector prior to crash.

Reproducible: Always

Steps to Reproduce:
1. Get F# standalone on Microsoft website (zip version) at www.fsharp.net
2. echo 'type Foo = A of int' > bug4.fsx
3. Run fsi.exe on this file: mono fsi.exe bug4.fsx
Actual Results:  
On Mono 2.6.4, it works fine (it displays nothing, as expected).
On Mono 2.7 (built today), I get "Operation is not supported." error.


The bug doesn't appear when using F# compiler (fsx.exe) instead of F#
interactive (fsi.exe). So, it seems to be related to dynamic compilation.

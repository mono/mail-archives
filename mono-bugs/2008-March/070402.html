NUnit has a sub-menu listing the MRU test-projects, and it prefixes each with a
number digit (1-9).  Although the number isn't (apparently) marked as an access
key with '&', it should be used as one.  When no access key is set then the
first character should be used.

This applies to both 'traditional' and ToolStrip menus.  Repro app from bug
367490.  The "1 ...", "2 ...", and "Foo bar" items should all be selectable by
the first letter i.e. 1/2/f.

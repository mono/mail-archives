Description of Problem:
The zoomfactor in the rich text box control does not work.

Steps to reproduce the problem:
1. Add a rich text control to a form
2. Set the Zoomfactor Property to 2
3. Set the text property to any text string.
4. Execute the program


Actual Results:
The text is displayed with normal size irrespective if the zoomfactor setting

Expected Results:
The text should display with a size depending on the zoomfactor

How often does this happen?

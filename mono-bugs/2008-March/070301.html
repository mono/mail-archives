Description of Problem:
After logging into the ClassifiedsStarterKit, you get a blank page. If you
delete "default.aspx" off of the end of the URL, then the right page shows up.

This did not happen in 1.2.6 and therefore is a regression.

I found this with 1.9 r96884 (in between p3 and p4 on the 1.9 branch build).


Steps to reproduce the problem:
1. Clear all cookies / authenticated sessions from firefox
2. Launch ClassifiedsStarterKit in firefox
3. Login
4. Notice the blank page
5. Remove "default.aspx" from the end of the URL
6. Notice that the normal page now shows up


Actual Results:
blank page

Expected Results:
the main page

How often does this happen? 
Every time.

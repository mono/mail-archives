Description of Problem:
When exiting from PolygonTriangulation, an error message pops up that says
"This program processes Simple Polygon only. Please check the input data."


This was found on Mono 1.9p4. This does not happen on Mono 1.9p3 and therefore
is a regression.


Steps to reproduce the problem:
1. Open PolygonTriangulation
2. Close PolygonTriangulation
3. Notice the error message box


Actual Results:
Error message.


Expected Results:
It should just close.


How often does this happen? 
Every time.

When a selection from a drop-down menu is rejected as in bug 366567 the error
dialog is presented multiple times -- if N selections have previously been made
from the menu then the dialog is displayed N times.  In fact the N is
apparently the sum of all selections for any menus in the PropertyGrid!

To repro, select a series of values from the "FooImageFormat" menu (any series
of values, but not the invalid value "MemoryBMP").  Then select the invalid
value, the error dialog will be shown multiple times.  Note that once bug
366567 is fixed, this repro will be unavailable.  It appears not to affect
other menus...

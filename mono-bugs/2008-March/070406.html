gmcs does not report CS0515 when an access modifier is defined on a static
constructor.

To reproduce, compile the following code:

public class Gen <T>
{
        public static Gen ()
        {
        }
}

Expected result:

test.cs(3,16): error CS0515: 'Gen<T>.Gen()': access modifiers are not allowed
on
static constructors

Actual result:

No error.

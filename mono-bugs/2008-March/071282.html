Currently, for code like:

--
using System;
using System.Collections.Generic;

class Test {
        static void Main (string [] args)
        {
                string s = args [0];
                foreach (char c in s)
                        Console.WriteLine (c);
        }
}
--

gmcs emits calls to System.CharEnumerator, while it should be transparent, and
use calls to IEnumerable<char>.

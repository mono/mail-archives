--- Comment #3 from Robert Jordan <robertj at gmx.net>  2008-07-07 06:04:27 MDT ---
Unix does not have APIs which allow to login by name & password. You'd have to
resort to "sudo", but even that is not always feasible, e.g. current user might
not be a sudoer or there might by no configured password authentication at all
(smartcards, etc.).

BTW, Unix programs don't need an auto-update feature because this is usually
performed by the packaging system.

You should develop a simple update notification feature that instructs to user
to update using the system packaging system.

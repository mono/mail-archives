Suggestion:

'Debug Application...' now only allows selecting a file, I'd like another
dialog where you could:

Select the executable
Add arguments to the executable
Pre-run steps

I think this would be useful for me, for the following scenario while
developping for Moonlight:

Set executable to firefox "/usr/lib/firefox/firefox"
Add arguments: "http://silverlight.net/fox"
Pre-run steps: "make install" (given that I don't use MonoDevelop to build,
only to write code).

All arguments should be saved so that to debug the same application again with
the same arguments it's just to hit the key binding and then enter.

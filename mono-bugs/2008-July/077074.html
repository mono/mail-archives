See this use case:

using System;

namespace Test
{
        public struct SomeStruct
        {
        }

        class MainClass
        {
                public static void Main(string[] args)
                {
                        SomeStruct[] arr = new SomeStruct [10];
                        Console.WriteLine ("Instance Ready");
                }
        }
}

MDB fails when trying to examine the value of an element of the array:

The links in the content window are malformed: 
The links beginning with a letter as N: T: M: etc... are in lower case ie: n:
t: m: etc... This doesn’t work on a case sensitive platform.

The links in the tree window are in uppercase and are working fine.

Reproducible: Always

Steps to Reproduce:
1. run monodoc
2. clic on a link to a Method in the content window

Actual Results:  
the text of the link appears in the content windows ( as m:xxx )

Expected Results:  
The link target should be viewed

This happens on ubuntu 11.04 with the official ubuntu natty mono packages.
This doesn’t happen on ubuntu 10.04 with the official ubuntu lucid mono
packages.

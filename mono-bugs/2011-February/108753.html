When using the device logging window there is a lot of text to dig through to
find your own little log line.  Text search would ease that burden.

It would be nice if it just supported the Visual Studio style text search, but
it would be primo ultra nice if it supported filtering.  But that may be an
enhancement too far.

To reproduce this problem :
Connect the Android Device Logging to the device.
Watch all the wonderful logging text come flying out.
Press Ctrl-F in the expectation that you will be able to search for text in the
log output.
Enter a search term.
Sigh as it search box searches your code.

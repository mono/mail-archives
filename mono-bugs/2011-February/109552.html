Test case that runs multiple web service calls in parallel

This test case doesn't reproduce the exact issue described in the report, but
it does expose several issues that occur when running web service calls in
parallel. 

In Silverlight the test can be run numerous times and it doesn't produce an
error, but in Moonlight bad things start happening quickly, including
deserialization errors and infinite loops. Even when it works correctly it is
far slower than it is in Silverlight to return results.

To run the test case, launch WcfService1.exe on a Windows machine and connect
to port 8100 from Moonlight. Press the "Get data" button multiple times to get
more results.

MonoTouch's mtouch command uses a --i18n option to specify the assemblies:

      --i18n=VALUE           List of i18n assemblies to copy to the output
                               directory, separated by commas (none,all,cjk,
                               mideast,other,rare,west)

We'll also need IDE support (VS, MD) to allow selecting the I18N assemblies.

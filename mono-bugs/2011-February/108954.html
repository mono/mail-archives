--- Comment #9 from Michael Hutchinson <mhutchinson at novell.com> 2011-02-10 19:21:40 UTC ---
We have tracked this down to a combination of issues in the debugger and the
debugger session code that only manifested when debugging apps on a Unix target
that had been built on a Windows host. The Visual Studio addin had a workaround
for the bug that allowed it to debug single projects, but the workaround broke
debugging of library projects.

We've now fixed the bug properly in MonoDevelop, and are in the process of
integrating the fixes into Visual Studio.

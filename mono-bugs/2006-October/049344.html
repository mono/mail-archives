+Summary: Mono treats browscap.ini property names as case-sensitive

+This is a problem with at least 1.1.13.8.  Haven't tried later versions yet.
+
+Description of Problem:
+Mono uses a browscap.ini file based on the one maintained at
+www.garykeith.com.  As of 9/14/2006, his files use capitalized property
+names.  Mono 1.1.13.8 (and perhaps later versions) don't recognize them so
+his updated files can't be used with Mono.  This makes supporting newer
+browsers (e.g. Firefox 2.0) difficult.

+longer works on Mono.  Something in Mono's Assembly reflection degenerated 
+and booish no longer works.  The call fails when an 
+System.Reflection.Assembly object calls a GetTypes(bool) method, where the 
+assembly reflection object is a dynamic assembly.  A trivial test case is 
+still needed.
+
+Steps to reproduce the problem:
+1. install boo
+2. run booish
+3. input:
+
+class foo:
+  a as int
+
+print "x"

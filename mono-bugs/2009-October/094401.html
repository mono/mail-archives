Right now auto login is turn on in the ASP.Net template VMs. However, since
these VMs boot to text mode, it doesn't make much sense to have them autologin.
Also, most of the time the user will want to login as root. So being logged in
as rupert doesn't help.

Please turn off auto login on the ASP.Net template VMs.

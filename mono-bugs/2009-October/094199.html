Description of Problem:
Currently precompiling a website in order to debug it is mandatory to get the
correct line numbers.

Jhill wants us to not have this hard requirement, and instead allow the user to
decide if they want their project to be precompiled or not depending on their
own needs.


To test if this is fixed, jhill suggested using these specific project types:
   * Use blog engine to test web site
   * Use a template project to test web project
   * Use the MVC template project to test MVC

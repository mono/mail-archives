When generating classes from a webservice WSDL, [Preserve (AllMembers=true)] is
needed if you do not want to add -nolinks as compile option. You can add the
attribute manually for every class, but you would need to do that each time you
update the webservice. A better solution would be to add the [Preserve
(AllMembers=true)] attribute during the generation.

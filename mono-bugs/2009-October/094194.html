--- Comment #8 from Thomas Wiest <twiest at novell.com>  2009-10-26 13:54:41 MDT ---
Here are some specifics:
If you run a winforms application remotely, and while it's still running, use
the tray icon and shut down MonoVS, all monovs processes are shut down, but the
winforms app is still running.

Also, the tray icon in windows is still active. However, you can shut it down
without problems.

Steps to reproduce:
1. Create a new solution based on the template winforms application
2. Click "Mono / Run remotely in Mono"
3. When the winforms app appears on linux, shutdown the monovs server by right
clicking on the tray icon and choosing "Quit"
4. Notice that monovs closes, but the winforms app is still running
5. Close the tray app in windows
6. Notice there were no errors.

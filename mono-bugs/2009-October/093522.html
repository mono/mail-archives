When an application on the Mac has unsaved items in it and the user requests to
quit the application, the Dock icon for that application bounces to indicate
this.  This is helpful when attempting to quit the application while not
actually in the app (either from the cmd-tab menu or right-clicking the Dock
icon to quit).

MonoDevelop does not bounce the icon when I attempt to quit and there are
unsaved items.  I don't realize this and then wonder why MonoDevelop did not
quit.

Description of Problem:
System.Management 2.0.0 isn't detected as a dependency for BlogEngine, even
though it is.

If you try to run blog engine without manually adding this as a dep, then it
fails to run (so it is clearly a dependecy).

This was found in MonoVS 0.4.


Steps to reproduce the problem:
1. Open BlogEngine in VS
2. Build the solution
3. Create a package using Mono / "Create Linux Package"
4. Go to the Deps tab
5. Click the "autogenerate deps"
6. Notice that System.Management 2.0.0 isn't in the list



Actual Results:
It doesn't add System.Management 2.0.0 as a dep.


Expected Results:
It should add System.Management 2.0.0 as a dep.


How often does this happen? 
every time.

please report the issues upstream.

Packagers, please do not add openSUSE-local patches to fix this kind of errors
but forward the bugs upstream
and close the bugs as RESOLVED UPSTREAM.

as the kernel will clean
up for you.  Manually freeing them only increases binary size and runtime.

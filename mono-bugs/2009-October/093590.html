It seems that we're going to have to create two types of RPMs. One for normal
openSUSE and one for the Mono Extension.

If the user takes an RPM that's built for the Mono Extension and tries to
install it on openSUSE, it will fail. And vice versa. This will probably lead
to a lot of confusion.

So, it would be really nice if the SuSE studio wizard detected what type of RPM
they built and only presented them with templates that will work with their
RPM.

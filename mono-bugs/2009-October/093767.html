Our current ASP.NET apps compiled for MonoX place their conf file link in
/etc/apache2/conf.d. On RHEL, it needs to be placed in /etc/httpd/conf.d.

Also, because of SELinux, the file most likely needs to be an actual file
rather than a link to somewhere outside of that directory structure.

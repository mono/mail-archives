+Gmcs refuses to compile accessing static field of nested generic class.

+1. Copile with gmcs t.cs /r:tlib.dll

+     public static Nil _N_constant_object = new Nil ();

+  }
+}
+
+3.

+PublicKeyToken=b77a5c561934e089]]'

Frederic,

I agree with what Robert said in the comment above, it's probably not a problem
with Mono, but an issue in libgamin. If it is Mono, though, the bug should be
reassigned to Mono Runtime instead, since ASP.NET is just a consumer of the
file watching utilities. 
Also, if I may give you a piece of advice, check if your server's kernel has
inotify enabled - it works much better than gamin (you can check if the
directory /proc/sys/inotify exists - if not, you probably don't have inotify in
the kernel or the relevant kernel module is not loaded or the feature isn't
compiled into the kernel at all).

I am having a similar issue with Save/Save All on Project Properties.

1) Create a new Project named 'SaveProjectTest.8980'
2) Open the Project Properties page for the 'SaveProjectTest.8980' project
3) Change the Default Namespace to 'SaveProjectTest'
4) try to save. ( CTRL-S, CTRL-SHFT-S, File->Save All ) 

Activating another editor window doesn't make a difference. Nor does building,
which normally will do a save all before the build.

5) Navigate back to the Project Properties.
6) Click the 'No AndroidManifest.xml found. Click to add one.' link.
7) Save will now succeed.

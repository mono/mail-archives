If I try to compile an anonymous method inside of a finalizer and it captures a
variable in the local scope of the finalizer, gmcs crashes.

Yes, I am aware of how weird it is to do this and how obscure a case it really
is :)

brian at brianmbp ~ $ cat t.cs                                                     
using System;
using System.Threading;

class Foo {
    ~Foo() {
        int x = 1;
        new Thread(() => Console.WriteLine("{0}", x));
    }
    public static void Main(string[] args) { }
}
brian at brianmbp ~ $ gmcs t.cs

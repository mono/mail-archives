Description of Problem:
After loading the latest version of monotouch, IB is not longer seeing the
files in the project.

Steps to reproduce the problem:
1. Create a new iPad app
2. Add a PNG file into the project
3. Start interface builder
4. Drag a UIIMageView onto the view
5. Go to the properties of the imageview that was just added
6. Navigate to the properties of the UIImageView control -> Attributes -> Image
Notice there are no images listed in the drop down.


Actual Results:
No images displayed


Expected Results:
Images that are included in the project

How often does this happen? 
Every time

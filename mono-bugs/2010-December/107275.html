Loading MonoDevelop colour schemes fails on .NET because of validation using
the schema main/src/core/Mono.Texteditor/Styles/IrBlackStyle.xml fails.
However, it works on Mono.

The error is:
The 'color' attribute is invalid - The value '#ff8000' is invalid according to
its datatype 'http://www.w3.org/2001/XMLSchema:NCName' - The '#' character,
hexadecimal value 0x23, cannot be included in a name."

Presumably Mono is allowing the '#' char in XMLSchema:NCName

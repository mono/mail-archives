There is no way to create a "background" style application that has no
application icon in the Dock on Mac OSX. The application should still be able
to create GUI windows for interacting with the program, but it should not have
a forced icon constantly in the Dock.

Reproducible: Always

Steps to Reproduce:
1. Bundle a Mono application with macpack in the command line
2. Alter Info.plist to have another entry: LSUIElement = 1
3. Run the application from the bundle
Actual Results:  
This is supposed to prevent an icon from displaying in the dock when the
application is running.

Expected Results:  
The application should run normally, but with no icon in the dock.

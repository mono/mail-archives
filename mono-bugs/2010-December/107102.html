So, when I don't have an emulator already running, I can launch it fine from
the toolset.  Then when I see the Select Device dialog, while the emulator is
booting it registers in the list.  After the emulator is almost done booting,
the emulator is removed from the Select Device dialog list and never comes
back.

To get it to come back, I have to kill the adb.exe process, then it magically
will appear in the device list again.

Reproducible: Always

Steps to Reproduce:
1. Start debugging, without an emulator running, or physical device connected
2. Choose to start an emulator, start an emulator of your choice (I'm working
with a 2.2 image)
3. Wait until you see the device show up in the list
4. Now wait until you see it disappear ;)
5. Kill adb.exe
6. Tada, it's back in the list

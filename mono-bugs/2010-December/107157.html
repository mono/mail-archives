RadInputManager does not inputify text boxes when AJAXified by RadAjaxManager.
Normally when you enter some numeric value in the textbox it should be
formatted according to the local numeric settings. For example 123456 should be
displayed with group separator like this - 123,456. 

Reproducible: Always

Steps to Reproduce:
1. Press the edit button to enter edit mode
2. Try to enter some numeric value - for example 1002003

Actual Results:  
The inputified control won't do anything. The value will remain as entered.

Expected Results:  
The inputified control will format the value (on blur) according to the local
settings.

--- Comment #2 from Jonathan Pobst <jpobst at novell.com> 2010-12-09 17:21:03 UTC ---
It looks like installing an application to an Android 2.3 emulator image is an
order of magnitude slower that 2.2:

MonoDroid 2.2:

adb -s emulator-5554 install jetboy.jetboy-Signed.apk

921 KB/s (13782284 bytes in 14.600s)
        pkg: /data/local/tmp/jetboy.jetboy-Signed.apk

MonoDroid 2.3:

adb -s emulator-5556 install jetboy.jetboy-Signed.apk

103 KB/s (13782284 bytes in 129.577s)
        pkg: /data/local/tmp/jetboy.jetboy-Signed.apk

We'll have to file a bug with Google and see if they can do anything about it.

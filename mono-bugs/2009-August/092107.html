Description of Problem:

Inputting text into text fields in Hebrew doesn't work. Instead of the Hebrew
letters types unrelated Latin characters appear.

Hebrew text displayed by the program is displayed the wrong way around, i.e.
from left to right instead from right to left.


Steps to reproduce the problem:
1. 

Start any Windows.Forms application and type, in Hebrew characters, Alef, Bet,
Gimel into a text field. Write a Windows/Forms application that displays Hebrew
text and start it under Mac OS X.


Actual Results:

See description of problem.


Expected Results:

Hebrew text should work as input and output.


How often does this happen? 

Always.

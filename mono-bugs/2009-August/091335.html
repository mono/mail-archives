MasterPageFile="~/Site.master" %>

works, but

MasterPageFile="~/Site.Master" %>

does not.  It gives the error:

Failed to create MasterPage instance for '~/Site.Master'.

Reproducible: Always

Actual Results:  
Failed to create MasterPage instance for '~/Site.Master'.

Expected Results:  
A successful load as seen with Windows / MS ASP.NET.

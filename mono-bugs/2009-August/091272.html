Description of Problem:
/usr/bin/vmware-user needs to be running for much of the vmware tools
functionality to work correctly. It runs by default on the Mono VM, but not in
the MonoVS VM.

This was found in the MonoVS 0.3p1 VM image.


Steps to reproduce the problem:
1. Boot the VM
2. pgrep /usr/bin/vmware-user
3. Notice that there are no PIDs returned (the process isn't running)


Actual Results:
it's not run by default.


Expected Results:
It should be run by default.


How often does this happen? 
Every time.

IRC that he wants the MonoVS functionality to be completely disabled.

Steps to reproduce the problem:
1. Install MonoVS and Mono for windows
2. Create a new solution based on the default C# winforms template
3. Click "Mono / Run locally"
4. Notice that it runs without a problem
5. Set the system clock to past the time bomb date (for 0.3, that date is Oct
1st, 2009)
6. Resart Visual Studio
7. Open the same project you had open before
8. Click "Mono / Run locally"
9. Notice the nag screen

Screen shot showing a time stamp on a line by itself

Description of Problem:
While watching the MonoVS Log Viewer, many times you'll just get a time stamp
on a line by itself without any text following it (see the screen shot).

This was found in MonoVS 0.3p3.


Steps to reproduce the problem:
1. Open the MonoVS Log Viewer in Linux
2. Press "Clear" so that nothing is currently in the Log Viewer
3. In Visual Stuio, create a new solution based on the default C# Winforms
template project
4. Click "Mono / Run Remotely in Mono"
5. Choose the host to run it on and click "Ok"
6. Close the winforms app on the Linux Side
7. Notice that there is now a time stamp line in the Log Viewer without any
text after the time stamp (see the screen shot).


Actual Results:
Time stamp only


Expected Results:
Time stamp and text describing the log entry


How often does this happen? 
Quite often, so I know we're missing a lot of valuable information.

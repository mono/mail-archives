***Warning***Warning***Warning***
DO NOT DO THIS UNLESS YOU KNOW WHAT YOU'RE DOING. THIS WILL MAKE YOUR VM
UNUSABLE!!!
***Warning***Warning***Warning***


To create a VM that will reboot until this problem is reproduced, do the
following:

1) Keep rebooting until the GUI finally displays

2) Once inside gnome, go into Control Center / Sessions and add a new auto
start item with the following:
Name: Reboot Until Text Mode
Command: sudo /sbin/reboot

3) Reboot the VM

4) This will cause the VM to reboot until it boots into text mode. Once it
boots to text mode instead of GUI mode, it will stop rebooting because the GUI
autostart's won't run without it booting into the GUI.

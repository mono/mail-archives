Description of Problem:
When you run "Mono / Create Linux Appliance" in VS without a solution open, an
error dialog doesn't pop up.

For all of the other entries in the Mono menu, an error dialog pops up saying:
"No Open Solution: There is currently no open solution/project to ... Please
open a solution or project."


Steps to reproduce the problem:
1. Open VS
2. Close any solution that opens automatically
3. Click "Mono / Create Linux Appliance"
4. Notice that nothing happens


Actual Results:
No error


Expected Results:
And error like the other menu items


How often does this happen? 
every time.

The attached patch adds a switch (disabled by default) to use the system copy
of libedit instead of building your own. This gets useful for distributions
like Gentoo that have a libedit package (or like Gentoo FreeBSD where libedit
is actually part of the system!).

Bundling libraries is a bad thing for most distributions!

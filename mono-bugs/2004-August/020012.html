+Summary: Type.GetElementType doesn't return null when expected
+
+Description of Problem:
+Type.GetElementType() returns not null if the current Type is not an array 
+or a pointer, or is not passed by reference, but the Type self.
+
+Steps to reproduce the problem:
+Run this:
+
+using System;
+class TestGetElementType {
+    public static void Main() {
+        TestGetElementType newMe = new TestGetElementType();
+        Type t = newMe.GetType();
+        Type t2 = t.GetElementType();
+        Console.WriteLine("The element type of {0} is {1}.", newMe, 
+t2==null? "null" : t2.ToString());
+    }
+}
+
+Actual Results:
+TestType element type is TestType
+
+Expected Results:
+TestType element type is null
+

+Testcase taken from Mono Documentation Library (slightly modified)

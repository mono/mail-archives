Description of Problem:

ICan Spy crashes upon exit on OpenSuse 11 GM x86_64 with mono-1.9.1.  This also
happens on OpenSuse 10.3 x86_64 with mono-1.9.0 so this is not a regression. 
This is a separate bug from bug #198973 since the stacktrace is different
although the behavior is similar.

Steps to reproduce the problem:
1. Run ICanSpy
2. Select /usr/lib/AnotherTetrisClone/Clone.exe as the form
3. "Select Events" -> GameForm
4. "Start Logging"
5. move mouse pointer around inside game form, click a few times as well
6. Select "Exit" from the menu

Actual Results:
Stacktrace

Expected Results:
Clean exit

How often does this happen? 
Almost always

This second issue was reported by a user testing a app created in VisualStudio.
 See the thread around
http://lists.ximian.com/pipermail/mono-list/2008-June/038767.html

This happens when the current culture of VS is one that uses a comma as the
decimal point character.  A workaround is to use only integer values.  This XML
format is also used when the form has Localized=true.

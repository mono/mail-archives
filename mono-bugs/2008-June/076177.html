Mike,

I'm sorry but (history shows that) there's very little chance that such
"meta-bugs" ever gets fixed, because (in order)
a) it lack test cases
b) it's unbounded (e.g. "several methods")
c) it spans multiple components (e.g. System.Drawing ans System.Windows.Forms)
d) it's about a incomplete feature (metafile support)

To be useful
- provide test cases (a), better yet unit tests, for all methods (b);
- split the bugs into two (or even more), one for System.Drawing, one for
System.Windows. make the later depend on the first one (if required)
- if you have working code attach it as a patch (diff -u)

I'm moving this to ENHANCEMENT so we don't loose track of the issue. Whenever
someone start working on metafile again will see it. If you need some specific
fixes then open bugs on them (with a least a test case).
Thanks

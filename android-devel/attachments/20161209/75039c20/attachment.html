<div name="messageBodySection">Today we're announcing our initial effort to make Activities in Android more friendly for .NET Developers. For now, we are calling it the ActivityController.<br>
<br>
ActivityController is meant to act as a sort of replacement/proxy for AppCompatActivity. Currently, its primary purpose is to enable the use of async/await when using StartActivityForResult in an Activity, with the ability to survive the unpredictable lifecycle
 of activities on Android.<br>
<br>
We're looking for dialog, feedback, suggestions, and contributions around this concept. The source code for the project can be found on GitHub here: <br>
https://github.com/xamarin/android-activity-controller<br>
<br>
You can read more about this on the Xamarin Forums:<br>
https://forums.xamarin.com/discussion/84221/announcing-android-activity-controller-preview

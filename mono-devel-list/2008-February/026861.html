Przemyslaw Soldacki wrote:
> I could run my tests on previous versions of Mono, but...
> Is there a simple way to make Mono downgrade? I use yast2 on OpenSUSE 
> 10.3...
>   

I don't really know of a graceful way to downgrade mono (you can always
force RPM to downgrade using --force, but again, that's not very
graceful and could cause problems).

For testing different versions of Mono, we use VMs. When I need to
downgrade a VM's mono I use smart ( http://en.opensuse.org/Smart ) and
remove mono-core (it removes everything, including banshee, tomboy,
f-spot, so be careful). Then I install the mono version I need. If you
install smart, make sure to install the smart-gui package as well. You
can then start the smart gui with "smart --gui".

It's best not to downgrade the Mono on your main linux machine. Instead,
consider using the Mono VMWare image or the Mono LiveCD from our
download page ( http://mono-project.com/Downloads ). That way your main
Linux machine continues to work well.


Hope this helps,
Thomas

Hi there Kevin,

You should recommend that users use 'sudo' rather than run things /as/
root.  Logs are made for all sudo commands, which is good for auditing.
Also, you should note how you came up with the dependency list so that
you do not personally have to update the docs every time this changes.

It may be better to host this doc on the monodevelop wiki so that others
may contribute ideas.  I'm sure someone on this list could give you an
account if they agree with me.

http://www.monodevelop.com/index.php?title=Special:Userlogin&returnto=Main_Page

Cheers,

Hey all. A few of us on the Banshee list have been kicking around the
idea of abstracting Banshee's media services into a new all-singing,
all-dancing, cross-platform library called Mono.Media. The idea is to
have a simple API that any CLI developer can use to easily a) playback
audio, b) transcode audio, c) playback video, d) transcode video. The
library is responsible for accomplishing those tasks using whatever
native frameworks are available (e.g. Gstreamer, DirectSound/Show,
Quicktime). It is similar in concept to how Mono.Zeroconf wraps both
Avahi and Bonjour in a single API. We've just started roughing out API
design and you are more than welcome to join us. If you're interested,
head over to http://banshee-project.org/Mono.Media to check out the
latest design docs and discussion. Feel free to propose new ideas and
critique existing ideas. Look forward to seeing you there!

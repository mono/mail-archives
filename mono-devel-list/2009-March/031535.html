Hi All,

As far as I can see, there is no standard, cross-platform
way to find all installed instances of mono.

It seems to me, then, that the best thing NUnit can do
to run with multiple mono installs is to have entries
in its own settings file that identifies and points to
the mono versions that the user has chosen to associate
with NUnit.

Any thoughts on this? Is selection of alternate Mono
versions from within NUnit inconsistent with how
NUnit is installed on Linux?

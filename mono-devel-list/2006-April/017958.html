Hi Henry,

You need the Connector. You can either put the dll for it in the bin folder of your app and reference it or you can install it in the GAC.
It doesn't rely on Apache that I know of but can work either with xsp or apache/mod_mono/xsp

Some versions of the connector had issues with mono, I'm not sure about the newest one, since I'm using an older version, but hopefully the newest one works with mono.

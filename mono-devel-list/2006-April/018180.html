The port of Report Manager engine to C#, mono and dot net is in beta stage,
but already working (Linux and Windows).

Basically you can design and tests your reports, using the report designer
(Windows and Linux), then you can execute your reports using the library in
your applications.

Export to pdf is working so it's useful for web applications, desktop
applications. 

For now the preview feature in Linux is using kpdf, because missing Windows
forms. I did not program a gtk# preview window because I know nothing about
gtk.

http://reportman.sourceforge.net/doc/dotnetport.html

http://reportman.sourceforge.net

There is a group in yahoo groups (reportman) to discuss, post bugs etc...

I'm interested also in your impressions, suggestions, etc.

Hi All,

NUnit 2.2.8 has been released as the latest minor release in the NUnit 2.2
series. You can download it at http://nunit.com/nunit/?p=download. No new
features are included, but several bugs are identified in 2.2.7 are fixed. 

Unless some serious problems arise, this is likely to be the last release of
NUnit 2.2. We'll be concentrating on the 2.4 release for the next month or
so. An alpha version of NUnit 2.4 will be released by the end of the month.

Charlie

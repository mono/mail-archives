> Patch to implement 2.0 parameterless Graphics.ReleaseHdc()

> - Stores IntPtr to private variable in GetHdc().
> - In ReleaseHdc(), call ReleaseHdc(IntPtr) passing stored IntPtr.
> - Set private variable back to IntPtr.Zero.

Hi,

I think a decision needs to be made as to whether Mono should continue
to ship C5.

With the release of C5 2.0.0 [1], C5 is on GitHub and NuGet. With
this, I don't think there's much of any incentive to ship it with Mono
anymore; at this point, it just complicates maintenance. Mono.C5
should probably be updated if Mono is going to continue shipping C5.

[1] http://www.itu.dk/research/c5/

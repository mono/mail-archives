Hi All,

I just released NUnitResults 1.0, a console application which reads one
or more NUnit result files and produces a set of html reports.

This is one of my "NUnitExtras" - a set of programs I have used in
my client work and which I'm slowly cleaning up and releasing.

The program creates a an index page and a set of detail pages, one
for each test assembly. Summary info to the fixture level is presented,
together with the details of any failures or ignored tests.

You can get it at http://nunit.org/?p=extras

Charlie

Have you installed the devel package for sqlite3? Probably it is
sqlite3.devel ...

In Linux a package like sqlite3 just install the library not the headers and
metadata that make you able to compile code that use that library. that
usually comes in a separate package commonly named like the libraries' one
appended with a .devel.

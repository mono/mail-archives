My experience: You get some functionality running on 2.6.4 but some
MVC features exposes bugs or missing features in mono. Currently I'm
running a patched 2.6.4, which has been patched with the following
revisions from the mono-2.6 branch (when will we get a 2.6.5?):

157188
157222
157626

With these patches applied, so far it works for me. Beyond this there
are some missing features in System.ComponentModel.DataAnnotations on
the 2.6 branch, so that seems unusable. (mono trunk seems to have
further work on this.)

However, I don't claim that my applicatiion exercises all of ASP.Net
MVC2 so YMMV.

Boehm support can't vanish soon, as there is not a few platforms that 
sgen does not support (such as Windows).

Having gc configurable at runtime is also very helpful for identifying 
GC bugs; imagine, if you cannot run mono --optimize=-all and you still 
have to determine if it is JIT issue or not. You will always have to 
reconfigure, build and install mono to examine it. That's so waste of time.

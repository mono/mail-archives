Hi All,

This is a rather small bugfix release with a few new minor framework
features:

* SetUICulture attribute
* Ignore property on TestFixture, TestCase and TestCaseData
* Several enhancements to assertions that test exceptions

For further detail and a list of bugs fixed, see the release notes at
http://nunit.org/?p=releaseNotes&r=2.5.2 

You can download it from http://nunit.org/?p=download

So *why* such a small release? Of course, it's always good to release, but
the main reason is to clearly mark the end of our use of Sourceforge for the
NUnit source code. All future work, both for NUnit 2.5 and NUnit 3.0, will
take place using Launchpad as our host.

Our new home on Launchpad:
 http://launchpad.net/nunit-xtp - our umbrella project
 http://launchpad.net/nunit-2.5 - NUnit 2.5  http://launchpad.net/nunit-3.0
- NUnit 3.0

Charlie

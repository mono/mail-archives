branches/mono-2-6 has been branched from r142636 of trunk.  mono, mcs, debugger, gluezilla, libgdiplus, mod_mono, xsp, mono-tools, and mono-basic are all open for post-2.6  work on trunk.

If you get assigned a 2.6 bug PLEASE give that bug priority over any trunk work (especially if it's a regression from 2.4).  We want to have a shorter (1 month) testing / bugfixing period for 2.6.  We want to release by the end of October.

tarballs of the current state of the branch are available from here: http://build.mono-project.com/index.aspx?lane=2.6-dist

preview builds will be published soon, probably not today (though the tarballs will probably make it into the OBS Mono:Preview repo today )

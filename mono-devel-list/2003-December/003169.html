I could be barking up the wrong tree here, but when I was first trying
to compile on Cygwin, I found that I needed to pre-build the Mono
version of resgen and put it in my path, otherwise .resx files could not
be compiled.

You need to make mcs/monoresgen before trying to build the class
libraries.

Hope this helps,

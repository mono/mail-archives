Hello Jason,

You are right, System.Drawing has no C#-glue code as GTK# has.
It Pinvokes gdiplus and it calls Cairo.
But it's possible to consider gdiplus as some sort of C-glue code
between Sys.Drawing and Cairo.

Alexandre

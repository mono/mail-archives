Hello guys,

   Today the new System.Drawing code written by Alexandre has landed on
CVS.  This adds a new build dependency on Cairo(www.cairographics.org). 
   You can either get rpms (or src rpms) from:

	http://primates.ximian.com/~duncan/cairo

   Or download it from Cairo CVS.

   I downloaded from Cairo CVS and built the following modules:

cairo/	libic/	libpixregion/  Render/	slim/  Xrender/  xsvg/

   They are standard autoconf/automake.  There is one trick though:
cairo requires libtool 1.5, delete that requirement, as it is not really
needed.

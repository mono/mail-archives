Hi,<br><br>  Statically linking the mono executable is mostly done to improve the speed of thread-local<br>access, to ease debugging etc. Note that only libmono is linked statically into the runtime,<br>the other dependencies like libglib are not. If fedora doesn&#39;t like static libs, you can omit them from the package, while still building them.<br>

Hi all,<br>In case you were interested, I've just bundled up the fruits of some serious development over the Memorial Day weekend for JaCIL.&nbsp; JaCIL can now handle about 110 instructions, approximately half of the CIL instruction set! JaCIL translated classes now can do method invocation, conversion operations, static/instance field access, array creation, array access, and branch and comparison operations.&nbsp; JaCIL also has the beginnings of its testing harness using TestNG on the Java side--the source distribution contains the test suite that shows some examples of what JaCIL can handle today.
<br><br>I am working on making the implicit conversions more sturdy (especially for &quot;ldelem&quot;/&quot;stelem&quot; instructions), and my next target is getting object creation working (i.e. &quot;newobj&quot;) amongst other things.

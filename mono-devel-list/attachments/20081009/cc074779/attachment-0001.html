<body class='hmmessage'>Hi:<BR>&nbsp;&nbsp;&nbsp; To run mono-debugger you need Mono from svn or mono 2.0 and Mono-debugger 2.0. (Which i think it is not present on Debian/Ubuntu Repository)<BR>
&nbsp;&nbsp;&nbsp; An alternative could be to run:<BR>&nbsp;&nbsp;&nbsp; ./configure --prefix=/usr --select<BR>
&nbsp;&nbsp;&nbsp; this let you select which package you will build, and you can disable the debugger Addin.<BR>
&nbsp;<BR>
&nbsp;&nbsp;&nbsp;&nbsp; My recomendation: Setup a parallel mono environment&nbsp;[*], if you have any version of mono installed, maybe some applications could depend on that installation, and if you break it you will have problem with other installed applications. (I've got this problem&nbsp;some times ago&nbsp;on Ubuntu)<BR>

You could access the static private field<br>
<br>
System.IO.FileSystemWatcher.wa<wbr>tcher<br>
<br>
via reflection and then check its type:<br>
<br>
---<br>
using System;<br>
using System.Reflection;<br>
using System.IO;<br>
<br>
class Test {<br>
        public static void Main ()<br>
        {<br>
                var fsw = new FileSystemWatcher ();<br>
                var fi = typeof (FileSystemWatcher).GetField ("watcher", BindingFlags.Static | BindingFlags.NonPublic);<br>
                Console.WriteLine (fi.GetValue (fsw).GetType ());<br>
        }<br>
}<br>
---<br>
<br>
The polling watcher's class is "System.IO.DefaultWatcher".<br>

$ man pkg_config<br>
....<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; PKG_CONFIG_PATH<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; A&nbsp; colon-separated&nbsp; (on&nbsp; Windows,&nbsp; semicolon-separated)&nbsp;
list of<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; directories to search for .pc files.&nbsp; The default
directory will<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; always be searched after searching the path; the default
is lib-<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; dir/pkgconfig where libdir is the libdir&nbsp; where&nbsp;
pkg-config&nbsp; was<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; installed.<br>
....<br>
<br>
<br>
If you have installed mono with "--prefix=&lt;something&gt;" the *.pc
files should be installed in <br>
"&lt;something&gt;/lib/pkgconfig" folder. Just execute something like
(replacing &lt;something&gt;<br>
with actual value):<br>
<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; export PKG_CONFIG_PATH=&lt;something&gt;/lib/pkgconfig<br>
<br>
before configuring csharp-studio and you should be OK.<br>

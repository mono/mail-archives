Hi all,<br>I've just released an early version of JaCIL, a .NET assembly to Java class file byte-code compiler.&nbsp; It leverages Mono.Cecil to deal with the concrete PE file format of assemblies and uses ObjectWeb ASM via <a href="http://IKVM.NET">
IKVM.NET</a> to deal with the concrete format of Java class files.<br><br>This release supports about 50 byte-codes, enough to translate simple methods with arithmetic.&nbsp; Not much, you may think, but a considerable milestone for the project.&nbsp; I plan on getting more of the basic instructions implemented for the next release including method invocation.&nbsp; Also on my list of things to do is a testing harness for the compiler/run-time so I can create a test suite.

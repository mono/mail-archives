I noticed that the mono distribution includes fsc/fsi but omits fsyacc/fslex.<div><br></div><div>These tools have been part of microsoft&#39;s F# drops for years and are part of the <a href="http://vs.net">vs.net</a> distribution as well. As a result, tons of F# software requires these tools. Any distribution which includes fsc should probably include fsyacc/fslex as well.</div>
<div><br></div><div>Because they are missing from the distribution, there is no canonical way to launch these tools on osx/linux. This makes it difficult to write build files that will work out-of-the-box for various users. Making them available in a standard location would save a ton of headache. </div>

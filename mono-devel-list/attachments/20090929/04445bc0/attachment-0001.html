<br>

<br>
/TransferRequest is suppose to kick off a new request in the application life cycle, which means that it needs to create a new request which runs all the way through from BeginRequest to EndRequest in the HttpApplication, after it has killed off the current request. /<br>

</blockquote></div>

<br>

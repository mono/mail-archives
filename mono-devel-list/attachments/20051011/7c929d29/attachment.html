<p class=MsoNormal><font size=2 face=Arial><span style='font-size:10.0pt;
font-family:Arial'>I have an application that needs to run under both .NET and
Mono and I came up with a pretty simple way to get MS DevStudio to build
both.&nbsp; I don&#8217;t know how others have solved this problem, but I
thought I would offer it up in case it helps someone else with the same
issue.&nbsp; The basic approach I took was to call the Mono compiler after the
Microsoft build has already run.&nbsp; Microsoft has a well known target
&#8220;AfterBuild&#8221; which I have hooked to invoke gmcs with pretty much
the same arguments that csc.exe takes with some minor modifications.&nbsp; I
put all the Mono build stuff in a file called Mono.targets, so it only take a
one line addition to the standard *.csproj file to get the functionality:<o:p></o:p></span></font></p>

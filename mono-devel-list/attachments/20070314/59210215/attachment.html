<h1>Getting Mono</h1>
<h2>Bootstrapping</h2>
<p>You currently need to
have the latest version of Mono installed on your computer. To build
Mono from SVN, you first need to have the Mono runtime and C# compiler
installed on your computer. </p>
<p>If you are running ubuntu, you would typically do </p>
<pre class="code"
 style="border-left: 3px solid rgb(204, 204, 204); padding: 0.5em; margin-left: 2em;">sudo apt-get install mono mono-gmcs mono-gac mono-utils</pre>
<p><strong>Note:</strong> </p>
<blockquote>Mono is currently in the universe repository. Make sure you
enable this repository. </blockquote>
<h2>Get the latest version from SVN</h2>
<p>Then, check out Mono from SVN. First you'll need to install <a
 href="http://subversion.tigris.org" rel="nofollow">Subversion</a>. If
you are running ubuntu, you would typically do </p>
<pre class="code"
 style="border-left: 3px solid rgb(204, 204, 204); padding: 0.5em; margin-left: 2em;">sudo apt-get install subversion</pre>
<p>You'll need the libgdiplus, mono and mcs modules. See <a
 href="http://www.mono-project.com/SVN" rel="nofollow">http://www.mono-project.com/SVN</a>
for more information. </p>
<p>If you are running ubuntu, you would typically do </p>
<pre class="code"
 style="border-left: 3px solid rgb(204, 204, 204); padding: 0.5em; margin-left: 2em;">svn co svn://svn.myrealbox.com/source/trunk/libgdiplus
svn co svn://svn.myrealbox.com/source/trunk/mono
svn co svn://svn.myrealbox.com/source/trunk/mcs</pre>
<p>There is more! To build mono, you need to have quite some packages
installed. Here's a partial list of Ubuntu packages. </p>
<h2>List of required packages</h2>
<p><strong>Build tools:</strong> </p>
<ul>
  <li><tt>automake1.9</tt> </li>
  <li><tt>autoconf</tt> </li>
</ul>
<p><strong>For libgdiplus:</strong> </p>
<ul>
  <li><tt>libexif-dev</tt> </li>
  <li><tt>libexpat1-dev</tt> </li>
  <li><tt>libfontconfig1-dev</tt> </li>
  <li><tt>libfreetype6-dev</tt> </li>
  <li><tt>libjpeg62-dev</tt> </li>
  <li><tt>libpng12-dev</tt> </li>
  <li><tt>libtiff4-dev</tt> </li>
  <li><tt>libtiffxx0c2</tt> </li>
  <li><tt>libungif4-dev</tt> </li>
  <li><tt>libxft-dev</tt> </li>
  <li><tt>libxrender-dev</tt> </li>
  <li><tt>x11proto-render-dev</tt> </li>
</ul>
<p><strong>For mono:</strong> </p>
<ul>
  <li><tt>autotools-dev</tt> </li>
  <li><tt>bison</tt> </li>
  <li><tt>dpatch</tt> </li>
  <li><tt>libtool</tt> </li>
  <li><tt>libxml2-dev</tt> </li>
  <li><tt>libxslt1-dev</tt> </li>
  <li><tt>m4</tt> </li>
</ul>
<h2>Building Mono</h2>
<p>To build mono, you would do this: </p>
<pre class="code"
 style="border-left: 3px solid rgb(204, 204, 204); padding: 0.5em; margin-left: 2em;">cd libgdiplus
./autogen.sh
make
cd ../mono
./autogen.sh
make</pre>
<p>If
you encounter any problem, that's most likely because of a missing
dependency. Don't proceed as long as you can't build Mono from source! </p>
<h2>Removing the pre-build version of Mono</h2>
<p>You can now remove the pre-build version of Mono. On Ubuntu: </p>
<pre class="code"
 style="border-left: 3px solid rgb(204, 204, 204); padding: 0.5em; margin-left: 2em;">sudo apt-get remove mono mono-gmcs mono-gac mono-utils</pre>
<h2>Installing Mono</h2>
<pre class="code"
 style="border-left: 3px solid rgb(204, 204, 204); padding: 0.5em; margin-left: 2em;">cd ../libgdiplus
sudo make install
cd ../mono
sudo make install</pre>
<p>You are now ready to go! </p>

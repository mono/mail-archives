These attributes are only generated as part of the moonlight build. There is no way to create them unless you modify the Makefiles to allow them to be generated in an alternate way. This is also not something which we should enable by default because it increases build time significantly and is useless unless you are also embedding mono and do all necessary function calls when loading libmono to enable coreclr. <a href="http://www.mono-project.com/CoreClrHowTo">http://www.mono-project.com/CoreClrHowTo</a><div>

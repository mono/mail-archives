<p class=MsoNormal>My machine is running the 64-bit version of Ubuntu

compatible with 64-bit Mono.<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>I have obtained the sources and managed to compile 64-bit
without any problems by just running the following commands:<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal style='text-indent:36.0pt'>./configure<o:p></o:p></p>

<p class=MsoNormal style='text-indent:36.0pt'>make<o:p></o:p></p>

<p class=MsoNormal style='text-indent:36.0pt'>make install<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>When I run &#8220;mono &#8211;V&#8221;, it says that I am
running on amd64 architecture.<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>So I assume the way I would run my programs under 32-bit is

command with the following syntax:<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal style='text-indent:36.0pt'>./configure --host=i386-pc-linux --build=x86_64-pc-linux<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>However, this fails to complete successfully, and produces
the following output (I&#8217;ve attached the &#8216;config.log&#8217;):<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal style='text-indent:36.0pt'>Checking for preceeding
underscore in symbols... configure: error: cannot run test program while cross
compiling<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>I also tried running the following command:<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ./configure

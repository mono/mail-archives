  <DIV>Hi all,<BR>I've just released an early version of JaCIL, a .NET assembly 
  to Java class file byte-code compiler.&nbsp; It leverages Mono.Cecil to deal 
  with the concrete PE file format of assemblies and uses ObjectWeb ASM via <A 
  href="http://IKVM.NET">IKVM.NET</A> to deal with the concrete format of Java 
  class files.<BR><BR>This release supports about 50 byte-codes, enough to 
  translate simple methods with arithmetic.&nbsp; Not much, you may think, but a

  style="FONT-SIZE: 10pt; FONT-FAMILY: Arial">Setting up the Environment and

  <P class=MsoNormal><FONT face=Arial size=2><SPAN 
  style="FONT-SIZE: 10pt; FONT-FAMILY: Arial">1) Install the .NET SDK V1.1 from 
  msdn.microsoft.com/downloads. I've put this step here simply because previous 
  instructions states to install it. I'm don't know if and why this step is 
  required. By the way, if you have Microsoft Visual Studio .NET 2003 installed 
  on your system then you can skip this step.<o:p></o:p></SPAN></FONT></P>
  <P class=MsoNormal><FONT face=Arial size=2><SPAN 
  style="FONT-SIZE: 10pt; FONT-FAMILY: Arial"><o:p>&nbsp;</o:p></SPAN></FONT></P>
  <P class=MsoNormal><FONT face=Arial size=2><SPAN 
  style="FONT-SIZE: 10pt; FONT-FAMILY: Arial">2) Install the latest cygwin setup 
  (about 30MB). Start choosing the default installation if you are not sure what 
  packages you need. You'll can always get the other missing packages later, if 
  mono configure script will require them. Be sure to get libiconv (not just 
  libiconv2!).&nbsp; I installed everything just to be 
  safe.<o:p></o:p></SPAN></FONT></P>
  <P class=MsoNormal><FONT face=Arial size=2><SPAN 
  style="FONT-SIZE: 10pt; FONT-FAMILY: Arial"><o:p>&nbsp;</o:p></SPAN></FONT></P>
  <P class=MsoNormal><FONT face=Arial size=2><SPAN 
  style="FONT-SIZE: 10pt; FONT-FAMILY: Arial">3) Download and install the gtk+ 
  dependencies to /usr/local:<o:p></o:p></SPAN></FONT></P>
  <P class=MsoNormal><FONT face=Arial size=2><SPAN 
  style="FONT-SIZE: 10pt; FONT-FAMILY: Arial"><o:p>&nbsp;</o:p></SPAN></FONT></P>
  <P class=MsoNormal><FONT face=Arial size=2><SPAN 
  style="FONT-SIZE: 10pt; FONT-FAMILY: Arial">http://prdownloads.sourceforge.net/gtk-win/GTK-Runtime-Environment-2.2.4.1.exe<o:p></o:p></SPAN></FONT></P>
  <P class=MsoNormal><FONT face=Arial size=2><SPAN

  installed first.&nbsp; This is optional if you do not want to compile 
  gtk-sharp.<o:p></o:p></SPAN></FONT></P>

Basically on linux ttyS* and ttyUSB* is used for serial device names, but this<br>
is not the case for *BSD, Solaris and probably OS X.<br>
On these systems tty[0-9]+ or ttyU[0-9]+ is used so the current code ony works on<br>
linux.<br>
<br>
What my diff does is that it gets the list of all tty devices from /dev and<br>
if it finds only one ttyS* device which is only the linux case it will remove all<br>
the tty[0-9]+ and ttyU[0-9]+ devices from the list. If it does not find it then<br>
we it removes everything except for tty[0-9]+ ttyU[0-9]+.<br>
<br>
So my diff will produce this list on *BSD:<br>
/dev/tty00<br>
/dev/tty01<br>
/dev/tty02<br>
/dev/tty03<br>
/dev/ttyU0<br>
/dev/ttyU1<br>
/dev/ttyU2<br>
/dev/ttyU3<br>
<br>
And this on a linux system:<br>
/dev/ttyS0<br>
/dev/ttyS1<br>
/dev/ttyS2<br>
/dev/ttyS3<br>

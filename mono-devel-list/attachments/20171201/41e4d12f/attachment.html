<div>
<p class="MsoNormal" style="margin-left:.5in"><a name="m_4821903355194776556__MailOriginalBody">This is a patch to enable mkbundle executable files to work when they are run from a path that is on the PATH environment variable.
<u></u><u></u></a></p>
<div>
<p class="MsoNormal" style="margin-left:.5in"><span><u></u> <u></u></span></p>
</div>
<div>
<p class="MsoNormal" style="margin-left:.5in"><span>Mono mini has a function probe_embedded() which looks for the program's file (using argv[0]). If the program is run from a path on the PATH environment variable then
 the open() fails and only a single parameter is sent to the mono_main() which in turn causes an if condition to print the help and exit.<u></u><u></u></span></p>
</div>
<div>
<p class="MsoNormal" style="margin-left:.5in"><span><u></u> <u></u></span></p>
</div>
<div>
<p class="MsoNormal" style="margin-left:.5in"><span>This patch searches through the directories specified in the current environment's PATH variable and tries to open() then first file found.<u></u><u></u></span></p>
</div>
<div>
<p class="MsoNormal" style="margin-left:.5in"><span><u></u> <u></u></span></p>
</div>
<div>

 to the community with the most likely or relevant owner(s) on the CC.<u></u><u></u></span></p>

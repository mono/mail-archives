just stubbed with a &quot;#if FIXME&quot; englobing all body.<br>
<br>
This patch implements SetMethodSourceRange using the new underlying<br>
MonoSymbolWriter&#39;s API so that the symbol files generated by SRE-using<br>
compiler are useful again.<br>

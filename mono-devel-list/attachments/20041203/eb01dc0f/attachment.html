<p><font size=1 face=Verdana><span lang=EN style='font-size:9.0pt;font-family:
Verdana'>Common Larceny requires the <a
href="http://www.ecma-international.org/publications/standards/Ecma-335.htm">Common
Language Infrastructure</a>. It is known to work with the <a
href="http://www.microsoft.com/net/">Microsoft .NET Framework</a>
implementation. <o:p></o:p></span></font></p>

<ul type=disc>
 <li class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;
     mso-list:l0 level1 lfo1'><font size=1 face=Verdana><span lang=EN
     style='font-size:9.0pt;font-family:Verdana'><a
     href="http://www.dotgnu.org/pnet.html">DotGNU Portable.NET</a> - Common
     Larceny does not work with version 0.6.10 of Portable.NET. Portable.NET
     cannot compile methods of the size used in Common Larceny. <o:p></o:p></span></font></li>
 <li class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;
     mso-list:l0 level1 lfo1'><font size=1 face=Verdana><span lang=EN
     style='font-size:9.0pt;font-family:Verdana'><a
     href="http://www.microsoft.com/net/">Microsoft .Net Framework 1.1</a> -
     Recommended implementation. You only need the .NET redistributable
     package, but the .NET SDK contains a much richer set of utilities and
     documentation. <o:p></o:p></span></font></li>
 <li class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;
     mso-list:l0 level1 lfo1'><font size=1 face=Verdana><span lang=EN
     style='font-size:9.0pt;font-family:Verdana'><a
     href="http://msdn.microsoft.com/netframework/">Microsoft .NET Framework
     2.0 (beta)</a> - Alternative implementation. Common Larceny has been
     tested with this implementation. <o:p></o:p></span></font></li>
 <li class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;
     mso-list:l0 level1 lfo1'><font size=1 face=Verdana><span lang=EN
     style='font-size:9.0pt;font-family:Verdana'><a
     href="http://msdn.microsoft.com/net/sscli/">Microsoft Shared Source CLI
     (Rotor)</a> - Common Larceny has not been tested with this implementation.
     <o:p></o:p></span></font></li>
 <li class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;
     mso-list:l0 level1 lfo1'><i><font size=3 face=Verdana><span lang=EN
     style='font-size:12.0pt;font-family:Verdana;font-style:italic'><a
     href="http://www.mono-project.com/downloads/">Mono Runtime</a> - Common
     Larceny does not work with version 1.0.4 or version 1.1.2 of Mono. Common
     Larceny runs briefly, but then exits without raising an error. The cause
     is unknown. <o:p></o:p></span></font></i></li>
</ul>

<p class=MsoNormal><font size=2 face=Arial><span style='font-size:10.0pt;

interoperable with other .NET languages.&nbsp; The Common Larceny project
is&nbsp;directed by Dr. Will Clinger in partnership with Dr. Matthias Felleisen,
both of <u1:place u2:st="on"><u1:PlaceName u2:st="on"><st1:place w:st="on"><st1:PlaceName
 w:st="on">Northeastern</u1:PlaceName></st1:PlaceName> <u1:PlaceType u2:st="on"><st1:PlaceType
 w:st="on">University</u1:PlaceType></u1:place></st1:PlaceType></st1:place>.&nbsp;&nbsp;It
is&nbsp; working to&nbsp;deliver a high-performance stand-alone Scheme compiler
as well as the future platform for Felleisen's <a
href="http://www.teach-scheme.org/Overview/" title=http>TeachScheme</a>
project.</span></font><o:p></o:p></p>

<DIV><BR><FONT face=Arial size=2>I have come up with a way to build using ICU 
under cygwin.<BR><BR>I downloaded the prebuilt VC6 binaries from the link on 
http://oss.software.ibm.com/icu/download/2.6.1/index.html. <BR><BR>I installed 
this under cygwin's structure (I used /usr/local/icu as the 
root).<BR><BR>Somewhere in your path put this script as icu-config (I chose 
/usr/local/bin/icu-config) having&nbsp;editted ICU_ROOT to wherever you 
installed ICU (this script is attached as well).</FONT></DIV>
<DIV><FONT face=Arial size=2><BR><FONT 
face="Courier New">#/bin/sh<BR><BR>ICU_ROOT=/usr/local/icu<BR>case 
$1<BR>in<BR>--cppflags)<BR>&nbsp;&nbsp;&nbsp; echo 
"-I$ICU_ROOT/include"<BR>&nbsp;&nbsp;&nbsp; 
;;<BR>--ldflags)<BR>&nbsp;&nbsp;&nbsp; echo "-L$ICU_ROOT/lib -licuuc 
-licuin"<BR>&nbsp;&nbsp;&nbsp; ;;<BR>--version)<BR>&nbsp;&nbsp;&nbsp; grep 
'\&lt;U_ICU_VERSION\&gt;' $ICU_ROOT/include/unicode/uversion.h | sed -e 
's/.*"\(.*\)".*/\1/'<BR>&nbsp;&nbsp;&nbsp; ;;<BR>esac</FONT></FONT></DIV><FONT 
face=Arial size=2>
<DIV><BR>now you can rerun configure and this should detect the ICU installation 
properly and use it.</DIV>

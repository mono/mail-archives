 style="font-family: -moz-fixed; font-size: 13px;" lang="x-western">Description
of Problem:
<br>
If there is an <code>EntityReference</code> inside the Text element
content, the value of this Entity referenced is omitted while building
a document.<br>
<br>
<br>
Steps to reproduce the problem :
<br>
1. Create DTD file defining some entities.
<br>
2. Create XML file referencing the DTD, with Entity <a
 href="http://www.w3.org/TR/DOM-Level-2-Core/core.html#ID-527DCFF2"
 class="noxref"><code></code></a>node inside and text element
referencing this entity.
<br>
3. Load XML document into XmlDocument<br>
4. Inspect element children structure.<br>
<br>
Actual result :
<br>
EntityReference has no children.<br>
<br>
<br>
Expected result :<br>
The subtree of the <code>EntityReference</code> node is a text element
holding Entity value. <br>
<br>
<br>
Enclosed is a small test case and a patch.

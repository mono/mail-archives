This switch refers to the capability of assembly &quot;A&quot; of running in .Net Desktop and .Net Compact frameworks.<br>
<br>
This has implications at the corlib and runtime levels.<br>
<br>
As far as I know mono does not target the Compact Framework (CF) therefore the retargetable flag does not work<br>
<br>
Since CF is a subset of Desktop Framework (DF) you *should* be able to recompile the assembly as DF and run under mono.<br>

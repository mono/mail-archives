<div>Per the commit message on <a href="https://github.com/mono/mono/commit/0542f8af71656fd318a83dac376576ff8ccab41d">
https://github.com/mono/mono/commit/0542f8af71656fd318a83dac376576ff8ccab41d</a> all of the build bots are now running Lion or Mountain Lion. As a result of being built on 10.7+ using the 10.7 (or 10.8) SDK, Mono will crash on Snow Leopard due to the lack of
 strndup. I submitted a pull request (<a href="https://github.com/mono/mono/pull/670">https://github.com/mono/mono/pull/670</a>) that avoids the detection of strndup on OS X so that strndup is not used.</div>

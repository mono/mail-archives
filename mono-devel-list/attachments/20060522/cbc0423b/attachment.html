<DIV>If you know the TCP port, use it in your connection string.</DIV>
<DIV>&nbsp;</DIV>
<DIV>Do you have the ability to connect to SQL Server using TCP/IP.&nbsp; By default, SQL Server 2005 Express has this turned off --- which is good from a security standpoint.</DIV>
<DIV>&nbsp;</DIV>
<DIV>See my two backslashes below.</DIV>
<DIV>&nbsp;</DIV>
<DIV>string ConnectionString = "Data Source=127.0.0.1\\SQLEXPRESS;<BR>Database=mydatabase;User ID=user;Password=password";<BR></DIV>
<DIV>Here, I am using TCP port to connect.</DIV>
<DIV>&nbsp;</DIV>
<DIV>string ConnectionString = "Data Source=127.0.0.1,1433;<BR>Database=mydatabase;User ID=user;Password=password";<BR></DIV>
<DIV>See <A href="http://www.mono-project.com/SQLClient">http://www.mono-project.com/SQLClient</A>&nbsp;for more info.</DIV>
<DIV>&nbsp;</DIV>
<DIV>If 127.0.0.1 does not work, try localhost.&nbsp; Or actually try your machine name.</DIV>
<DIV>Or you firewall could be blocking the port.</DIV>

<p class=MsoPlainText><font size=3 face="Courier New"><span style='font-size:
12.0pt'>Attached is a redesigned implementation of the NumberFormatter class.
The patch includes a new algorithm for float and double ToString implementation
that improves performance by a factor of 25(!) and upto 440(!!!). The patch
includes further improvements to integer <i><span style='font-style:italic'>ToString</span></i>
as well as Andreas suggestion of avoiding calling <i><span style='font-style:
italic'>NumberFormatInfo.CurrentInfo</span></i> in the case of integer types
default <i><span style='font-style:italic'>ToString()</span></i> implementation.<o:p></o:p></span></font></p>

<p class=MsoPlainText><font size=3 face="Courier New"><span style='font-size:
12.0pt'>This patch should improve Mono&#8217;s performance in important use
cases such as web-services where primitive <i><span style='font-style:italic'>ToString</span></i>
performance is an important factor.<o:p></o:p></span></font></p>

<p class=MsoPlainText><font size=3 face="Courier New"><span style='font-size:
12.0pt'><o:p>&nbsp;</o:p></span></font></p>

<div><br></div>As far as I know the final on local variables is only a hint for the java compiler (javac), not jit (hotspot) engine. Detection of loop invariants is done at runtime (during code jitting) so there&#39;s really no difference between any of the following in modern Java jits:<div>

<br></div><div>1) (final) int max = array.length;</div><div>for (int i = 0; i &lt; max; i++) {}</div><div><br></div><div><div>2) for (int i = 0; i &lt; array.length; i++) {}</div><div><br></div><div>Dawid</div><div><br></div>

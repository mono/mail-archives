This is a patch to add runtime support for generic variance. This patch lacks certain features of the spec. For example, only reference type variance is supported. Also, the runtime does not enforce the validity of variant parameters (that they only appear in interface and delegate types, that covariant parameters only appear as return types and in interface inheritance, and the contravariant parameters only appear as by-value method parameter types). This means it is possible to violate type safety will invalid CIL. The patch is contributed under the MIT/X11 license. It is attached and inline.<br>

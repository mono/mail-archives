<p>I'm in the process of converting over my OpenMetaverseTypes.dll
library (basic 3D type library) to use Mono.Simd. One thing that is very handy
to have is static members for common values, such as:</p>

<p>&nbsp;</p>

<p>public static readonly Vector4f Zero = new Vector4f();</p>

<p>public static readonly Vector4f One = new Vector4f(1f, 1f,
1f, 1f);</p>

<p>public static readonly Vector4f MinusOne = new Vector4f(-1f,
-1f, -1f, -1f);</p>

<p>&nbsp;</p>

<p>Which makes comparisons much easier:</p>

<p>&nbsp;</p>

<p>if (myvector4f == Vector4f.Zero) { … }</p>

config file <br />Mono Install Directory/etc/mono/config  <br />you will
see it there. What you will need to do is map the required dll to the
linux file. Eg for odbc connections you would normally have to add a line
in the config file like</p><p>&lt;dllmap dll=&quot;odbc32&quot;
target=&quot;libodbc.so&quot;/&gt;</p><p>In this instance you may need
(only may) something like<br />&lt;dllmapp dll=&quot;gda-2&quot;

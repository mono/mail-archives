    <div id="smartTemplate4-template">Your second call stack shows the
      Finalize() method - this means that some of your controls don't
      get disposed. Make sure all controls you create get disposed,
      either by adding them to the Forms.Controls collection (which gets
      disposed if you call Form.Dispose()) or by explicitly calling the
      control's Dispose() method for controls that aren't added to the
      collection.<br>

This is a series of patches to add support to Mono for building against<br>
the Android NDK [0].  Android runs the Linux kernel, but moves many<br>
things around compared to a &quot;normal&quot; desktop Linux distro.<br>
<br>
These patches are based against the mono-2-6 branch.<br>
<br>
This second patch patches libgc so that it will properly build and link<br>
under Android.  In particular, note that Android includes pthread_*()<br>
functions within libc, not libpthread, thus the added configure check.<br>
<br>
        * include/private/gcconfig.h: Android platforms are built atop Linux,<br>
          don&#39;t use glibc, and uses `environ` instead of `__environ`.<br>
        * <a href="http://configure.in" target="_blank">configure.in</a>: Use AC_CHECK_LIB() to check for pthread instead of<br>
          just blindly linking to -lpthread, as Android includes pthread<br>
          support within libc and doesn&#39;t provide a separate libpthread.<br>
<br>

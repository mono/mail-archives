The ComboBox is using a TreeModel to represent the rows. So look at TreeModel (TreeStore or ListStore) for info. <A HREF="http://www.go-mono.com/docs/">http://www.go-mono.com/docs/</A><BR>
<BR>
To clean up all entries in a combobox...:<BR>
<BR>
&nbsp;&nbsp;&nbsp; TreeStore store = (TreeStore)combo.Model;<BR>
&nbsp;&nbsp;&nbsp; store.Clear();<BR>
<BR>
Count the entries I don't know if there is a function for that, else you have to loop through the rows like this:<BR>
<BR>
                        <BR>
&nbsp;&nbsp;&nbsp; TreeStore model = (TreeStore)combo.Model;<BR>
&nbsp;&nbsp;&nbsp; TreeIter iter&nbsp;&nbsp; = new TreeIter();<BR>
&nbsp;&nbsp;&nbsp; <BR>
&nbsp;&nbsp;&nbsp; int count = 0;                        <BR>
&nbsp;&nbsp;&nbsp; bool res = model.GetIterFirst(out iter);<BR>
&nbsp;&nbsp;&nbsp; while(res)&nbsp; {<BR>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; count++;<BR>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; res = model.IterNext(ref iter);<BR>
&nbsp;&nbsp;&nbsp; }<BR>

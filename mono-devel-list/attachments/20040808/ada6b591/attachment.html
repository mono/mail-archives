<p class=MsoNormal><font size=2 color=navy face=Arial><span style='font-size:
10.0pt;font-family:Arial;color:navy'>Managed C++ is not supported on Mono, and
thus it is not binary compatible. As you can see in your error, it tries to
invoke some C++ code using pinvoke.<br>
So, it always will be binary compatible, unless the language you use uses
pinvoke, like managed C++. (or you use it yourself).<o:p></o:p></span></font></p>

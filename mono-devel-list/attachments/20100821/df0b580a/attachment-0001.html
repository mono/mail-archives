<div dir="ltr"><div>The using statement does not include the reference to the assembly.  It only introduces a known namespace into the global one.</div>
<div> </div>
<div>You are probably missing the actual reference to the assembly.</div>
<div>The way to add the reference depends on your dev env and the way you compile.</div>
<div> </div>

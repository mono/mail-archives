Nicholas,

It ocurs because you need to initialize a dset property.

try to use this:


sqlcon = new SqliteConnection("URI=file:sqlitetest.db");
sqlda = new SqliteDataAdapter(SELECTALL, sqlcon);
dset = new DataSet();
sqlda.Fill(dset);

I think that now it works fine.

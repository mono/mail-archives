Hola,

>     Here are the preview release notes for Mono 1.1.4, I have only
> audited the patches to the `mono/' module, which were roughly 400
> commits, it is still missing the mcs/ module which is another 1,200
> commits.


* corlib
	- Number formatter contribution by Kazuki Oikawa and a
	  bunch of testcases by akiramei.

* System.Data
	- several DataSet fixes including those from Ankit Jain.
	- some bugfixes on XML related DataSet feature.

* System.XML 2.0 is mostly implemented (except for compiled XSLT)
	- XmlSchemaValidator is implemented. And significant
	  refactoring on XML Schema validating reader is done.
	- SchemaInfo on DOM is now provided.
	- new Binary content (Base64/BinHex) reader support.

* Miscellaneous XML
	- several DOM optimization based on Ben's ideas.
	- added tiny dtd2xsd tool.
	- RELAX NG compact syntax writer.
	- experimental RELAX NG grammar inference.

Atsushi Eno

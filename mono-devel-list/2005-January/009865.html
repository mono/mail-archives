On Tue, 2005-01-18 at 09:29 +0530, Roshan Achar wrote:
> Hi All,
> 
> I am new to mono and was wondering if there is any API which allows
> querying the rpm database. Something like the "rpm -q" shell command.
> Any help is greatly appreciated.

You may want to look into libredcarpet, which is Novell's package
management library, and which works with both RPM and Debian packages.
It's a C library but it's been wrapped for mono.

Take a loot at http://cvs.gnome.org/viewcvs/libredcarpet and you can
check it out.  The mono stuff is in bindings/mono.

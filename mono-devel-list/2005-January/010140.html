Hi

I have made a new release of C#Shell, see
http://csshell.sourceforge.net/.

Release notes:

- Added configuration file. It primarily remembers window size from one
invocation of the program to the next.
- Added settings window, which handles adding of paths, dlls, and namespaces.
- Very improved auto-completion - but still some work to be done.
- Lots more unit testing.
- Added menus.
- Corrected some errors.
- Improved responsiveness.
- plus many smaller changes.

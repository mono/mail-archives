>> By the way does Mono support SOAP 1.2?
>
> It does not. I haven't found any formal documentation about the MS.NET
> 1.1 support of SOAP 1.2. Do you have an url?

.NET Framework 2.0 has SOAP 1.2 support according to this document:
http://winfx.msdn.microsoft.com/library/en-us/dv_fxintro/html/2806b1ca-5e02-49c2-b148-f2db96de115d.asp

And I think they will not release a document about things that are not
implemented until final version is released.

According to the following documentation it has no SOAP 1.2 support at all
("support for SOAP 1.2 was pulled from this version"):
http://msdn.microsoft.com/library/en-us/dnwebsrv/html/websvcwhatsnew.asp

.NET Framework 1.1 seems to have at least some SOAP 1.2 support at server
side but not at client side.

But I'm not expecienced in SOAP standard so cannot test whether it supports
1.2 or not.

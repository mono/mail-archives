Hi,

NGrid (http://ngrid.sourceforge.net) is a grid computing framework heavily
relying on many (complicated) aspects of .Net:
remoting/appdomains/assemblies/... (even a distributed GC included). One
year ago, I would never have expected Mono to support such a project.

But the last tests that I did perform with Mono 1.1.9 indicate that
everything is running smoothly. NGrid is going to be cross-platform after
all. Not too bad, especially since my lab's cluster is under linux :-).

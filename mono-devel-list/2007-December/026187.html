In general, it is a bad idea to use mkbundle.   We support it because
there are some corner cases that require it, but it is in general a bad
idea.

It makes things hard to debug, if there is a security issue with Mono,
you cant fix it by upgrading Mono, you must upgrade every app that was
bundled and lastly, if you use --static, you must obtain a license from
Novell.

On 06/30/03 Jason King wrote:
> 1. I can test that Mono runtime works by writing some new code and
> compiling.  Fine.  How can I test that I am running the latest version?  

Run mono--version, if it prints 0.25 you're running the latest version
(note that mcs --version will still print 0.24 since someone forgot to
increase the version number there).

> 2. Do I need to somehow uninstall version 0.24, and if so, how do I do that?

If you installed the new version with red-carpet you should be all set.
Running rpm -U new_mono.rpm will replace the previous mono version and
is the command you should usually use, I guess.

The Xtatic project (http://www.cis.upenn.edu/~bcpierce/xtatic) is working to
develop a lightweight extension of C# with native support for statically
typechecked XML processing.

Our compiler is now getting close to the point where it can be used on real
programs, and we are looking for a few sample applications to try it out on.
To save overhead, we'd like to re-use existing applications, rewriting just
their XML-intensive parts using our new primitives.

Specifically, we are looking for 
   * C# applications making heavy and/or interesting use of XML
   * of modest size (up to a few tens of thousands of lines)
   * with source code freely available
   * (and ideally) easily installed and portable across .NET runtimes.

We'd be most grateful for pointers and suggestions.

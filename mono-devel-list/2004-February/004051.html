Charlie,

For NAnt we use the following code to detect whether we're running on the
Mono runtime :

if (Type.GetType("Mono.Runtime", false) != null) {
    // we're on Mono
    IsMono = true;
} else {
    IsMono = false;
}

Hope this helps,

Gert

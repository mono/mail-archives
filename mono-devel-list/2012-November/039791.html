IIRC, MS.NET compiler generates .pdb files, while Mono's generates .mdb 
files. So using MonoDevelop + MS.NET won't help. Use MonoDevelop in 
Linux, or install Mono for Windows (and configure MonoDevelop to build 
with Mono, not .NET).

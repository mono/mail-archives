Hello,
> I'm trying to use MonoDev on a Mac & a Linux machines
> and I'm struggling.

There is no OSX debugger yet, and MonoDevelop on OSX does not come  
with the debugger.

The folks at Unity have just contributed the code for the OSX  
debugger, but it will take some time before we integrate and publish  
builds for everyone to use.

That should not stop you, if you really need it now, to try the  
patches, build MonoDevelop and post feedback.

Miguel.

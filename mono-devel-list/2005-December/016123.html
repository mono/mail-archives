You may need to install the devel packages.

I didn't make myself clear enough in last post.

If you installed Mono from rpm, you should get the -devel packages .rpm
and install them.

also, check if you have the folder /opt/mono-1.1.10/lib/pkgconfig

You have to find a mono.pc file inside it. Also, check /usr/lib/pkgconfig.

As a last resource, the suggestion of PKG_CONFIG_PATH is to add the
filder which has the mono.pc file in order pkgconfig can find it.
This is what I have done in my system as I use mono compiled from svn
and install it in a non-common location.

I bet you are missing the mono.pc file. Try to locate it.

I hope it helps.

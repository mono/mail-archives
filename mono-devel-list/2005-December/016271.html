Hi,

Just a quick posting to say that my updater/builder script has now been
tested an uploaded.

If you've not used it before, it's a script which will checkout whatever
modules you want from svn and build it for you - basically, it makes
life simpler.

Now, the nice thing about the script is that you can automate the build
process so that everything goes into /usr, or set the libdir prefix to
be /usr/[local/]lib64 for 64 bit compilation, do a make distclean and
build and a whole array of other things as well.

What's new
----------

The script now takes multiple options on the command line (so that you
can issue monoupdater noldconfig lib64 debug if you fancy)
The list of modules to be updated is now stored right at the start of
the script instead of dotted around the place
Quite a few new options added
monoupdater help altered to be more useful
A few optimisations
Quite a few bug fixes

Grab a copy from http://www.all-the-johnsons.co.uk/mono/monoupdater.sh

TTFN

Paul

> I'm trying to embed mono in a C++ application on Windows and I keep

jit/jit.h is missing this:

	extern "C" { ... }

or

	G_BEGIN_DECLS
	...
	G_END_DECLS

Try to include it this way until the header gets fixed:

extern "C" {

#include <mono/jit/jit.h>

}

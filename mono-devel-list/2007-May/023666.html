> There are a decent set of posix api's for symbian.  More info at:
>
> http://forum.nokia.com/main/resources/technologies/open_c/index.html

Unfortunately those appear to be for S60 3rd Edition only.

Similarly, Symbian's PIPS (PIPS Is Posix on Symbian OS) appears to be  
for 9.x only:
http://developer.symbian.com/main/academy/press/books/pdf/P.I.P.S..pdf

Some more related links follow.

This is the cross-compiler Symbian is using for 9.x:
http://www.codesourcery.com/gnu_toolchains/arm/index_html#downloads

This site has information on how to make the old tools work on Unix:
http://www.koeniglich.de/sdk2unix/symbian_sdk_on_unix.html

Slightly older information on how to make the above work on Mac OS X:
http://simonwoodside.com/dev/symbian/howto.html

A more recent alternative to the latter:
http://www.tomsci.com/xcodeplugin/


Symbian's view on additional runtime environments:
http://developer.symbian.com/main/tools/runtime/index.jsp

Other Open Source projects:
http://developer.symbian.com/main/tools/opensrc/languages/index.jsp

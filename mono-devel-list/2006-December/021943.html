Here's a quick update for those interested in the Microsoft Team 
Foundation API implementation for mono.

I've started on an implementation of the standard Team Foundation 
client, tf.exe, and scrapped my git-like Team Foundation client that I 
had previously in the tree for the time being. That's a project for later.

Documentation on the tool can be found on msdn at:
http://msdn2.microsoft.com/en-us/cc31bk2e(VS.80).aspx

Currently, work focuses on the "Informational Commands". dir, labels, 
workspaces, view, and permission commands all have pretty decent 
implementations. I plan to implement most of the "Informational 
Commands", then broaden my testing to include a codeplex Team Foundation 
server in addition to my own.

I'll add Microsoft Team Foundation API assemblies and tf.exe code to the 
olive tree, after I complete the informational command work. If anyone 
wants to review, test, or contribute to this effort now, you can 
download the code using git:

   git-clone http://joelreed.dyndns.org:8080/tfs.git

I've also put a tarball at:

   http://joelreed.dyndns.org:8080/tfs-0.1.tgz

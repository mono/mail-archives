Hello,

    Since the publishing of Mono 1.2.2, a lot of work has gone into
implementing many of the APIs that applications were missing.  So far we
have implemented 1,065 new API calls since the 1.2.2 release (22 days
ago).

    The rapid pace of development meant that the Moma Reports were
actually getting outdated.   I have now updated our scripts and the
reports themselves to reflect the current state of Mono on SVN.

    There is now a green indicator in each application listing how many
APIs that were previously reported are no longer in place, and all the
lists (sums, per-class, per-app) reflect the new implemented features.

 	http://primates.ximian.com/~miguel/momareports/

Miguel.

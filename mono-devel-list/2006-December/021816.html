Attached patch adds support for a --remote-mode to the monodoc browser.
In this mode, the browser will accept url strings from stdin. This is required
for using the browser from monodevelop as an external help viewer.

Also, it adds a "--about", which is required for monodevelop to differentiate
from the earlier browsers which don't support --remote-mode.

Please review.

Hi all,

As some of you already now, we are working on a new version control system 
(well, an SCM indeed) based on .NET / Mono. We are using it internally under 
Mono, and also some close clients, but our official Mono release will 
hopefully happen during January.

Server is already pretty stable, and the same for the command line client, 
we only have some issues with the GUI tool.

Plastic SCM is not open source, but we would like to contribute somehow to 
the community, so we would like to make it free for anybody at Mono 
development who want to use it for development.  So, if anybody is 
interested please contact me at psantosl at codicesoftware.com to get licenses, 
Linux installation instructions and so on.

Regards,

> METHOD .ctor()
> METHOD m2()

> This was from my compiler run on source like this:

> #v+
> interface I1 {
>   m1 () : void;
> }
> 
> interface I2 extends I1 {
>   m2 () : void;
> }
> 
> class C1 implements I1 {
>   m1 () : void {}
> }
> 
> class C2 extends C1 implements I2 {
>   m2 () : void {}
> }

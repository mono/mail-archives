Hello all

Attached is a reworked patch.

The patch points are: 
* Fixed order of chars in InvalidPathChars (note change caused by this
in DirectoryTest)
* Fixed null path handling in GetFullPath
* Fixed Windows-specific negative flows (':' in path and path too long)
in GetFullPath and IsPathRooted
* Fixed negative flows for wildcard chars, that are illegal in some
cases
* Fixed IsPathRooted that should be able to handle path with spaces (but
not with all whitespace chars)

$ export MONODEVELOP_SDB_TEST=1
$ monodevelop

Open the project you want to debug, Run->Run With...->Custom Command Soft
Debugger,Listen


Then run mono on your remote machine with
--debugger-agent=address=YOUR_HOST_WITH_MONODEVELOP:10000

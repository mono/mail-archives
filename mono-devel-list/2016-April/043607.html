I'd look at the latest alpha, 4.4.0.40 - you can grab that as a source
package from
http://download.mono-project.com/repo/debian/pool/main/m/mono/mono_4.4.0.40-0xamarin1.dsc

I intend to upload 4.4.x to Sid, once it's marked stable upstream.

If you want something newer, the latest git master can be grabbed from
http://download.mono-project.com/repo/debian/pool/main/m/mono/mono_4.5.0.421-0nightly1.dsc
(from last week). That'll become 4.6 eventually, and I'd definitely like
to get a 4.6 release into the next Debian, since it includes ARM64
officially

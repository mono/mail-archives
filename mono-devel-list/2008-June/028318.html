vbc, and vbnc, only add support for the My namespace (or for *all* of it at
least) when configured to do so.  It could be that MonoDevelop isn't
creating the correct command line for the compiler.  Check what Msbuild/VS
creates, and compare it with what's used in MonoDevelop.

See also "Customizing Which Objects are Available in My"
http://msdn.microsoft.com/en-us/library/ms233781.aspx for more information
on how the various aspects of My are configured.

See a sample compiler command line at
https://bugzilla.novell.com/show_bug.cgi?id=333962  Given the revision
number there, that suggests that the My namespace support was in and working
before 1.2.6.

Hi, maybe monodevelop is using another mono runtime which does not have debug
symbols and not the one you have built. Check under monodevelop Edit ->
Preferences -> .Net Runtimes that the correct runtime is being used. If not
add it manually (eg. if you built mono with "/opt/mono" prefix then point it
there).

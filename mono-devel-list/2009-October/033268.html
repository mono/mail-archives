Daniel,

I'm glad to announce you (and other members of the Mono-List) that
DeveelDB is now finally up and running: I was able to fix the issues
concerning the malfunction of file storage systems.
I'm currently writing all the NUnit tests possible to test the
remaining functions (expressions, SQL, operations, etc.) and soon I
will be able to publish a working fully copy of the project.

After your advice (and the SQL specifications), I started working on a
command-line tool and on a simple server application (which will be a
separate "pluggable" project), to dispatch the database within
networks.

I will keep you informed about the progresses, and also I would like
to involve you in the project if you'd like: it's a really challenging
one and I'm having a lot of fun working on it (consider I already
branched it to implement OpenGIS SFS and Full-Text search indices, and
many other functions can be implemented).

Cheers!
Anto

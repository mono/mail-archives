Hi, all!

Sorry for late response.

Npgsql on Mono tree is the Npgsql 1 version whose development is
stalled. We are working actively in the Npgsql2 codebase.

I remember we tested the new version and as noted we had some problems
with backward compatibility. We fixed those bugs but I think we never
imported the code back to Mono svn. In order to keep mono class
library dependency low we decided to keep the development in Npgsql
own version control and release versions bundled with major Mono
releases.

Today, in the Npgsql project you can find the latest version compiled
for Mono. I think that the last parts which need some work, which I
still have to study and any help is welcome is the creation of
distribution specific packages so the integration of Npgsql in current
mono installs is error prone.

I hope it helps.

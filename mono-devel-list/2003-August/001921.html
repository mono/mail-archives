Define "production."

There are several Linux applications using C#.  You can find some
screenshots here:

	http://www.go-mono.com/screenshots.html

Also of interest to a number of people is Nat's Dashboard:

	http://www.nat.org/dashboard/

Then there's the mono-hosted ASP.NET application hosting mono's
documentation, visible at:

	http://mono.ximian.com:8080/index.html

Some commercial apps are also using Mono:

	http://www.openlinksw.com/press/virt3beta.htm
	http://www.winfessor.com/portal/DesktopDefault.aspx?tabindex=5&tabid=10#PR13
	http://www.ximian.com/about_us/press_center/

However, mono hasn't reached 1.0 status, so if that's a concern you've
been warned.

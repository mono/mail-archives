One important issue to keep in mind is that if you want your code to be
integrated into Mono's code base, your employer must agree to do a
copyright assignment of the code to Novell, or to license the code under
the MIT X11 license (even if the code is a change to the LGPL code) as
we actively relicense the code under commercial licenses.

The PowerPC code in particular is important to us as it is used in the
Wii, PS3 and Xbox systems.

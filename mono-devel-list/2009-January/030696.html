I've been using the moq framework in a gmcs compiled project and I
encountered some strange behavior. I decided to compile moq with gmcs and
run the test suite. When I tried to compile the moq tests I got an exception
from gmcs. Ive reduced the code that causes the exception to this:

using System;
using System.Linq.Expressions;
using System.Collections.Generic;

static public class Foo
{
    static public int Helper(Expression<Predicate<int>> match)
    {
        return 0;
    }

    static public void Main()
    {
        Expression<Action<List<int>>> exp = x => x.Add( Helper(i => true) );
    }
}

helps.

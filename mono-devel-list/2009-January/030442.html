On 01/10/2009 08:31 AM, Paul wrote:
> Hi,
>
> I'm trying to get uiautomationwinforms into fedora rawhide, but it's
> complaining that mono-ui is not available. What package is mono-ui
> available from?

Unless there's a typo somewhere on our end, you mean "mono-uia".  The 
package is called mono-uia, and tarballs are available on FTP. 
uiautomationwinforms is useless without uiaatkbridge, so you'll want 
that as well.

Note that the current version of these packages is intended as a 
developer preview, and should not be used in production environments. 
For that, you'll want to wait for 1.0.

Please see the release notes and source install instructions here:

http://www.mono-project.com/Accessibility:_Release_Notes_0.9
http://www.mono-project.com/Accessibility:_Installing_From_Source

Let me know if you have any questions,
Sandy

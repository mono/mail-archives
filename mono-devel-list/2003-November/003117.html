Hi guys,
    I've got a working mono live disc (which I calling monoppix right 
now). It's intended to be a showcase for all mono has to offer. The disc 
is based off morphix 0.4.1 HeavyUI (GNOME based). Currently it includes

mono
mcs
xsp
mod_mono
monodoc
monkeyguide

from mono cvs.

Everything non-debian (mono stuff, httpd, apr, etc.) gets built from 
CVS. Also, source code for everything non-cvs is included.

For those impressed by such stuff, screen shots are at:
http://www.linuxpakistan.net/images/temp/monoknoppix10.png
http://www.linuxpakistan.net/images/temp/monoknoppix11.png
http://www.linuxpakistan.net/images/temp/monoknoppix12.png
http://www.linuxpakistan.net/images/temp/monoknoppix13.png


I need your guys' opinion on what should or shouldn't be included on the 
disc. For obvious reasons, TeX and emacs packages have been dropped to 
make space for the mono stuff. Should the disc contain the bare minimum 
needed to demo mono or should it contain everything that can fill 700 
MB? Also, what OSS .NET apps can be included on the disc that make for a 
good showcase?

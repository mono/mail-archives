NUnitLite 0.2.0 is now available for download at
http://www.codeplex.com/NUnitLite/Release/ProjectReleases.aspx?ReleaseId=656
8

NUnitLite is a lightweight version of NUnit, suitable for running in
resource-limited environments.

With this release, you can run tests under .NET 1.0, 1.1, 2.0, 3.0 and 3.5,
Mono 1.0 and 2.0 profiles and .NET CF 1.0 and 2.0. Problems around 
ExpectedExceptionAttribute on the compact framework have been resolved and
the output of a test run may be directed to the Console, to a file or as
Debug output.

Development is continuing and is currently focused on saving test results
as XML and reporting results remotely via TCP. For further details, see
the project roadmap at http://nunitlite.com/?p=roadmap

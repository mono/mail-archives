Hi Miguel.

Here are the Npgsql notes:

	- Added support for ParameterDirection.Output and
ParameterDirection.InputOutput parameters. Now you can use these types
of parameters to get results. Thanks Ivan (ivan-sun1 at mail dot ru) and
(teste at aslls dot ss) for heads up.
	- Added support for implicit parameter stored procedure call.
Parameters are added in order they were added to NpgsqlCommand
Parameters collection.
	- Added support for record return type functions. Thanks neri and
Michel for heads up and tests.
	- Added Varchar support. Thanks Peter and Gustav for heads up.
	- Npgsql doesn't depend on System.Drawing and System.Windows.Forms
(a.k.a SWF) when compiled with Mono. This dependency is only necessary
to add design time support for VS.NET and so only enabled when compiling
with csc.

	- Lot of bugfixes.


Thanks.

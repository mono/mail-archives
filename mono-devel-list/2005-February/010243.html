On 02/02/05 Aleksandar Dezelin wrote:
> Please advise me how the change this.
> 
> I've seen there's a file named 'lcids.xml' in directory
> tools/locale-builder  which allows us to change stuff mentioned above.
> How should I do that? Does Mono build scripts generate automaticly
> neccessery tables (in mono/metadat/culturetables.h) or I'll have to
> generate it manually and then send a patches?

When you have made the changes to the files there, you need to run:
make culture-table
and
install-culture-table
and then rebuild mono and test your changes.
Then you can send the patch to the tools/locale-builder dir
so we can apply it to the tree (there is no need to send the patch
of the generated files).

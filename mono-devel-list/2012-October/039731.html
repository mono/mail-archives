Hi all,

I used  Laurent Etiemble's build script
<https://github.com/Monobjc/mono-macosx-64/wiki>  , with some modifications,
to create a DMG installer for Mono 3.0 universal for OS X. Universal - that
is, the Mono binaries (Mono only, not GTK or the Carbon-dependent WinForms,
etc.) are fat (merged 32-bit and 64-bit). I have uploaded a complete image 
here <http://www.filebox.com/ptpt31n80xyv>  . I will also take the
obligation to create such an installer for each new Mono version in the
future. 
I'd like to work with the Mono team through all security and other
considerations so that the installer is uploaded at mono-project.com, with a
link next to the current 32-bit only installer. The link could say that this
is an unofficial installer, for example "by Dimitar Dobrev, the maintainer
of the C# binding for Qt", that is, that the Mono team is not responsible
for this particular installer.
Thanks for your time.

Personally, I would like to see mono-winforms finished -- the web page (http://www.mono-project.com/WinForms) suggests that support for WinForms 1.1 and 2.0 is done, and is now in a maintenance/bug-fixing state, but if my experience is any indication, it needs a LOT of bug fixing.  I used to spend my precious time-off fixing mono-winforms bugs, and hope I have the time and energy to do more in the future, but so much needs to be done.

While we're at it, I'd like to see mwf-designer (http://www.mono-project.com/WinForms_Designer) completed too.  The last time I tried to use it, it crashed left and right.

Maybe you could start by going through the bug-report database :-)

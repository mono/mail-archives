Hi Gonzalo,
"__doPostBackScript" Script is not rendered in page, if no comand Column
is Visible. Microsoft implementation emits this script, even if Command
Column is hidden.

Many developer  use __doPostBack function call and ItemCommand event to
extend DataGrid functionality. Also DataGridItem notation differs for
postback.

It will be good idea to render the Script so that applications will be
easily portable.

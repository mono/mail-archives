To handle this problem: libgdiplus0 >= 2.6
You need to get libgdiplus source probably the latest tar off of
mono-projects/downloads
You need to build it. You will then have to make a package out of it. You 
will then need to install that package. When that is installed you will have
a libgdiplus0 that meets the criteria.

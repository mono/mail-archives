Hi All,

I just uploaded the Alpha 4 release. You'll find it as a great many new
features, particularly with respect to asserts and assertion syntax. It also
incorporates an experimental feature that allows running tests in a separate
process, using a different version of the common language runtime than that
under which NUnit itself is running.

Note that this is still alpha level software, although the assert-related
features are pretty stable at this time. 

The release notes are at http://nunit.org/?p=releaseNotes&r=2.5 and you can
download it at http://nunit.org/?p=download

This page:
http://www.mono-project.com/GtkSharp

has a section called "current source release", under which:

gtk-sharp points to 2.12.10, but latest really is 2.12.11.
Except the latest is only available on
http://download.mono-project.com/sources/gtk-sharp212/ and not from
ftp.gnome.org where the download link in the above page points.

gnome-sharp points to 2.24.1, but latest really is 2.24.2.
Version 2.24.1 doesn't compile since the removal on Mono.GetOptions,
and the update (released in 2010) fixes this. Here, the situation is
reversed and the updated version is only available from
http://ftp.gnome.org/pub/GNOME/sources/gnome-sharp/2.24/, but not from
http://origin-download.mono-project.com/sources/gnome-sharp2/

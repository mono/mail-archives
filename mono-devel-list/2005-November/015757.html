Christopher,

> Can someone explain to me the difference between what these do exactly
>     Restart all mod-mono-server processes

Restart all mod-mono-server processes.

>     XXGLOBAL: Restart Server

Restart the instance that handles the new auto hosting feature.
See http://www.mono-project.com/AutoHosting

>     default: Restart Server

Restart the "default" mod-mono-server instance.
This is the instance that handles all applications not
having a special MonoSetServerAlias.

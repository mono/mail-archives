Hi folks,

  An anonymous svn server for accessing the Mono repository is now 
available.

To access via svn, use the following URL:
  svn://mono.myrealbox.com/source/trunk

"svn co svn://mono.myrealbox.com/source/trunk/mono" will checkout the mono 
sources.
"svn ls svn://mono.myrealbox.com/source/trunk" will list available modules.

To access via web browser, you may use this URL:
   http://mono.myrealbox.com/

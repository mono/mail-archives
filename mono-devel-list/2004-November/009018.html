Gtkhtml-sharp.dll is built by gtk-sharp, but only if the gtk-sharp configure
script locates a recent enough version of libgtkhtml3. That library is part
of the gtkhtml package, which is on the GNOME ftp servers :-)

    <ftp://ftp.gnome.org/pub/GNOME/sources/gtkhtml/3.5/gtkhtml-3.5.0.tar.gz>

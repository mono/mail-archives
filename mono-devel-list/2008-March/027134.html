Hi all,

With Marek's approval, I've committed a
patch<http://lists.ximian.com/pipermail/mono-patches/2008-March/112665.html>to
trunk that adds support for *.browser files and control adapters to
System.Web under the 2.0 profile.  It doesn't break any existing tests and
includes a bunch of new tests.  The *.browser processing is based on Owen
Brady's implementation<http://ocean.accesswa.net/projects/Ocean2.Web.HttpCapabilities.zip>
.

The patch doesn't add any *.browser files to the mono installation.  For
now, webapp developers will need to put those files in their app's
App_Browsers folder.

--Dean

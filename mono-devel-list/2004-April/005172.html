Alright, this is intended as a tool for the developer end to generate a 
binary distributable Mac OS X  'Mono.framework'.  This is just a script 
to run through building a Mono runtime framework with it's 
dependancies.

I suggest running it with root rights, as it does install mono into the 
/Library/Frameworks/Mono.framework path.  Also included for convenience 
are add / remove scripts to manage the /usr/bin symlinks.

At this point, the .pkg files are -not- being generated, I expect to 
have that done once I complete debugging issues that arise with this 
script.

Hello,

>     I am preparing the release notes for the Mono Beta1 release, as
> usual the volume of changes is daunting, and I would like to include a
> precise description of your contributions for this release.
> 
>     Please mail me highlights of the changes since the past release Mono
> 0.31

System.Security:
	* XML Signature classes improvements: Now it passes many of
	  the tests in W3C "merlin" and "phaos" compatibility test
	  suite (except for some missing stuff such as XPath here()),
	  including prefixed XML instance and Manifest element.

	  (Sebastien, if I missed something, add lines)

System.XML:
	* There were many bugfixes that comes from Mainsoft hackers'
	  C# port of NIST DOM test collection, Gert Driesen's NDoc
	  on Mono challenge (now NDocConsole.exe is usable on Mono),
	  and xmldsig improvements.
	* DTD parser became 2x faster and 1/2 memory usage.

System.Data:
	* Some missing parts are implemented to support full xsd.exe
	  functionality. TypedDataSetGenerator is implemented and it
	  can generate simple dataset that does not have relations.
	  With existing data loader, DataSet.InferXmlSchema() is
	  also implemented to accept .xml files in xsd.exe.

Atsushi Eno

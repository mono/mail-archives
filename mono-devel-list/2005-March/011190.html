I've found my problem, I'm using XSP and (now I know)
it uses the bin directory under the root directory
that you start XSP, I was using a subdirectory for my
application with it's own bin directory that XSP can't
see, to fix this I made a copy of 'startXSP.bat' in
the directory of my app and deleted the line 'cd /D
C:\PROGRA~1\MONO-1~1.6\share\doc\xsp\test', and now
everything works great (please try to understand I've
been a Mono developer for 2 long days:))

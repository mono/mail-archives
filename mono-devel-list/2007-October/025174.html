As of today, Mono support in OpenEmbedded (cross compiling) is working
for ARM.  You may want to start with that as there has been a lot of
work done (http://www.openembedded.org/filebrowser/org.openembedded.dev/packages/mono)
 to make Mono cross compile in a completely automated manner, and
build the following packages: http://pastebin.ca/725080

I'm setting up a feed for the neo1973 phone, and will post details shortly.

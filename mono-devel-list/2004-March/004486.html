My steps to a sucessful  monoDevelop install

1) virgin RH 9.0
2a) try to install  RedCarpet
2b) RC needed glib, gtk dependancy, which i got from www.rpmfind.com
3) installed RedCarpet
4) installed all from  mono channels and RC channel, etc.
5) installed  monoDev 1.0
6) removed  mono/mcs from /usr/bin  (as the new ones are in /usr/local/bin)
7) ran mono sucessfully

- note monodevelop doesn't have a html/asp like GUI (yet) to make it 
like MS VS  (i was kinda hoping for this, but it doesn't have it)

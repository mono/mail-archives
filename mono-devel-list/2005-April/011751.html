Hi All,

An NUnit Vision / Statement of Direction has been recently drafted and
discussed among the developers. It's now available at
http://nunit.com/devewiki.cgi?NunitVision. If you saw an earlier
version, you'll find (I hope) a bit more clarity about where we plan to
go in the future.

You can comment on the wiki, but if you see any issues of general
interest to Mono, I suggest posting something here as well.

Note that this is still not a roadmap or release plan. It just states
some principles we hope to use when discussing features to add. The
roadmap will list features we hope to have in the coming releases.

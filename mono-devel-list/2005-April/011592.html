This could be related to this bug I submitted:

http://bugzilla.ximian.com/show_bug.cgi?id=73048

XSP didn't account for the extra linefeed that IE sends.  If you use firefox, everything should work fine (if this is indeed the problem).

This is now fixed, and should be in 1.1.5 (by looking at the date it was fixed - march 16th)

Hope this helps!,
Curtis.

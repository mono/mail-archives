[This only affects those of you who're using GMCS / .NET 2.x things from
latest SVN - nothing has changed for MCS and .NET 1.x]

Hi guys,

just upgraded the metadata file format to the latest .NET 2.x version -
the one which is used by the latest Visual Studio 2005 Beta 2.

This means that if you upgrade from SVN, you have to recompile all
your .NET 2.x assemblies.

GMCS now creates files which can be used with Microsoft's latest .NET
2.x version and our runtime can now also run assemblies compiled with
it.

Sorry for any inconveniences,
Martin

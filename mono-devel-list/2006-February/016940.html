attention the error message and I didn't notice that the path

C:\cygwin\tmp\scratch\mono-1.1.13.2\mcs\class\System\System.Net\HttpWebRequest.cs:480

that path doesn't exist on my machine and it ever has been.
So, I threw away all my XDevelop file (apart from the .cs obviously), 
I rebuilt the project from scratch and everything it's okay now.

I still can't understand why that path was in the X Develop solution 
and project file, but definitely was not a Mono problem

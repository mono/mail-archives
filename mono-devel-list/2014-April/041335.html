  Hi,

  There is a programming language called Fantom that aims to make
cross-platform development easier. It is designed to run on the JVM, .NET
CLR, and JavaScript. Right now the only mature implementations are for the
JVM and JavaScript. the .NET CLR implementation has not gotten any attention
lately. I was hoping to find some Mono Developers interested in starting a
Mono Implementation for Fantom. This will help Fantom reach its goal. If you
are interested visit Fantom's website and post on the forums. Or you can
reply here. Depending on how many interested we can probably make this
happen.

 Fantom's Website : http://fantom.org/

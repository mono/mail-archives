Get the RPM from http://www.go-mono.com/download.html and type the following
commands as root in the directory where you saved the RPM files.

  rpm -ivh libgc-6.1-1.i386.rpm
  rpm -ivh mono-0.24-1.i386.rpm

I think these two packages would be sufficient, if you need to install more
packages, the procedure is the same
  rpm -ivh package_name

Does it help or you have problems at a later stage of installation?

Hello,

I had the same problem. Look like win32-exception.* are missing in mini.

May be this can be useful for you:

copy win32-exception.h from mono/jit directory to mono/mini,
add #include "win32-exception.h" to the mini.h
add content of mono/jit/win32-exception.cpp to the mono/mini/mini.c
or copy mono/jit/win32-exception.cpp and add the file to Makefile etc.

Alexandre

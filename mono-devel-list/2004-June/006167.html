Do you plan on supporting another database backend? This is where most
problems come in. Joseph did a nice job of writing postgres functions to
replace the stored procs in IBuySpy, however this wouldn't be portable
to My SQL. 

The first step of supporting running on Mono and connecting to a MS SQL
server DB running on another machine shouldn't be too much work. The
second step of supporting a database that runs on Linux will be more
difficult.

> lrwxr-xr-x  1 root  wheel  115  7 Jun 01:04 pango-sharp.dll ->

> which just point to nowhere, as the gac directory doesn't exist.

This is a known issue with an open bug report.  The gacutil /root
functionality isn't quite enough for prefixed installs, as you've
detected.

Until a proper fix lands, you either need to install Gtk# into the same
prefix as mono, or fix the links manually.

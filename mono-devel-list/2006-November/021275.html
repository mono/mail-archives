You probably won't be able to run the Microsoft Team System assemblies
under Mono.  Even if you could, I'm suspect that the VSTS license won't
allow you to do that.

However, at the Mono Meeting Charlie Poole of NUnit gave a presentation
and mentioned that NUnit might support the Team System API in the
future, but that's not the present.  I wouldn't consider using Team
System if you want to support Mono right now.

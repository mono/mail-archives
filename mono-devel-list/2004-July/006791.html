Hi monoers,
 
I've been working for some time now on a patch to allow Mono to run on the
NetBSD platform, with GC enabled. NetBSD 1.6.x does not have a decent
threading implementation, however NetBSD 2.0 has one.
 
NetBSD 2.0 is still is Beta but is pretty functional and I'm able to fully
compile the mcs tree on this platform using this patch, with the GC enabled
and a decent memory usage.
 
However, I'm still working on threading issues brought into the game by the
GC and its interactions with the libpthread implementation which basiscally
generates random crashes.
 
Here is the url for the patch : 

http://msdn.labtech.epitech.net/blogs/jaylee/archive/2004/07/02/508.aspx

Be aware that this patch has NOT been tested on other plaforms and it
*might* introduce side effects, at least in this preview state.

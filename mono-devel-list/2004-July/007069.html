Please see bug #55916 on gentoo's bugzilla
(http://bugs.gentoo.org/show_bug.cgi?id=55916)

This was a libtool problem, as confirmed by somebody in #mono a while
ago. The solution was to remove ltmain.sh, then run aclocal and
libtoolize --force --copy. Hope this helps.

Hi Gerardo,

Well it looks like a versioning problem. XSP was compiled against
System.Web 1.0, and the directory you listed is for a GAC copy of
System.Web 2.0. You need to install version 1.0 (or 1.1. with
redirection) assemblies in the GAC, or poitn with MONO_PATH to the
directory you have a copy of the 1.0 assemblies.

Hope it helps,

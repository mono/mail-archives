Hi,

This patch enables the local x86 reg allocator to allocate specific
physical regs for specific operators, like putting sreg1 into ecx for
the shift operators. My testing shows that this patch emits 20-30% less
spills for our class libs (corlib/system etc) but I havn't seen any
improvment in performance. 

This patch makes the code emited for long/shift operations a lot better.

Hello Jose,

>I know that there has been a lot of work on the server side implementation
>of SSL.

Carlos has been doing a great job on server-side SSL. Maybe he can tell us
more exactly what's the current status on this.

However server-side support isn't planned to be used with mod-mono as Apache
already has mature (and faster) SSL support. Some people have voiced
interest to have a SSL enabled XSP but I don't think anyone is working on
that currently.

> Are there any plans for security certification?

I'm not aware of any (specific) certification for TLS/SSL (maybe a CC PP
exists ?) but cryptographic algorithms can be certified under NIST FIPS
140-2. Is this the kind of certification you had in mind ? or is it
something else (more general) ?

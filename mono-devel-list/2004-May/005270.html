    The CVS tree is frozen for the Beta release, if you have changes
that you feel need to go in now for any of these modules:

	mono, mcs, gtk-sharp, *-sharp, monodoc

    You must:

	* Test your patch, post to the list and wait for approval
	  for the patch to go into CVS.

	* Get approval from the appointed maintainer on irc.

    Any features and improvements can wait for after the Beta 1 release,
which is coming shortly.

If the link does not work for you, try going to here first:

http://bugzilla.ximian.com/

Click the link named "Query".

In the "Product" list box, select "Mono Class Libraries.  While holding down
the Ctrl-key in the "Component" list box, select "Sys.Data",
"Sys.Data.OleDb", and "Sys.Data.SqlClient".  Then click the button "Submit
Query".

A list of bugs for System.Data and data providers will be shown.

Click to review any of them.

If you have a patch for any of the bugs, please submit it to the
mono-devel-list.

If there is a bug in System.Data or one of the providers, but it is not in
bugzilla, please submit the bug.

There is even a bug for implementing ADO.NET 2.0 features.

I may be answering a simpler question than is asked, but in response to techi eth’s “Do anyone have successfully in reading of serial port with Linux over Mono?” we are using Mono on a Raspberry Pi running Debian GNU/Linux to read data coming in the serial port from an Arduino sensor as part of our IoT project “ConnectTheDots”. The code is on GitHub at http://connecthedots.io , and the Wiki contains instructions on how to set up the environment (from the Arduino to the Raspberry and then up to Azure in our case). Relatively simple, but if anyone has a better way to do this, would love to hear.

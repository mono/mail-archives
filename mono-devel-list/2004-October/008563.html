Doug,
Looks like you are using an old version of GCC. If it is so,
would you mind upgrading it to 3.x ? However, we will fix
our code so that it can compile with old versions of GCC.

A side note, if you are building WINE for SWF, please be
informed that our new SWF implementation does not need
WINE. All you need is to do 'make clean' and 'make install'
inside mcs/class/Managed.Windows.Forms directory after
installing mono and mcs. This should overwrite the old
SWF dll with the new one.

- Ravi

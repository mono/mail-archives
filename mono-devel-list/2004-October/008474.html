Hi all,

Three months ago I publicated the projetc on
SourceForge, now this project has release compatible
with Mono and PNet.

http://advanced-ado.sourceforge.net/

ADP, is a transparent provider for ADO.NET, that it
has the advantage to make the dynamic load of used
providers. It run in .Net Framework, Mono and
Portable.Net, in Linux and Windows. Tests with the
following databases had been made: SQLServer, Oracle,
Firebird, Sqlite, Postgres, Mysql. But we believe that
all available providers must work.is a transparent
data provider to use with ADO. NET

Started working on an AIX 4.3 port of Mono 1.1.1. An earlier posting
reported problems
with glib on AIX, but for me the glib binary package from
      http://aixpdslib.seas.ucla.edu/allpackages.html
has no problems so far.

So far I had to manoeuvre around some minor problems (mainly include
files), but after
I switched to a GCC generated locally from a recent source tarball, things
are progressing nicely.

I only have access to AIX 4.3 (which we are stuck with because of a third
party SNA library
that has not been ported to AIX 5.x). If someone has access to AIX 5.x (and
some time to spare),
it would be nice if he/she could run my patches against that (there are
some ideosyncracies
that may be peculiar to AIX 4.x, and I would like to keep the port
generic). I have already
contacted Matthew Griffore (Matthew asked for an AIX port in a posting in
January), but he
does not have access to an AIX machine anymore.

Hi,

just a quick announcement:

we have prepared Mono 1.0.2 packages for Red Hat Enterprise Linux 3.0
AS, ES and WS.

Packages for RHEL 3.0 AS:
http://dist.best-off.org/mono/rhel-3as-i386/

Packages for RHEL 3.0 ES:
http://dist.best-off.org/mono/rhel-3es-i386/

Packages for RHEL 3.0 WS:
http://dist.best-off.org/mono/rhel-3ws-i386/

Packages for Gtk# and related packages like monodoc or monodevelop
will follow soon. Initial target was mono and mod_mono.

Other stuff: the download pages have been modified to validate as
HTML 4.01 Transitional, CSS now validates, too.

For those running Red Carpet, either running

# rug service-add http://dist.best-off.org
# rug subscribe mono

or adding http://dist.best-off.org to the list of services in the
Red Carpet client should do the trick.

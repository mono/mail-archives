On Mon, Sep 22, 2003 at 11:16:58AM -0400, Cezar Nasui wrote:
> Hi,
> 
> I want to embed XSP into an application so it can answer http requests
> after some processing. 
> I read in the xsp doc that I can use MonoApplicationHost and
> MonoWorkerRequest to embed an ASP.NET server into my application but is
> not clear for me how I should do that. 
> 
> Does anyone have some simple example / tutorials or could anyone please
> point me in the right direction?

The best place you'll find information about how to embed XSP server is
in the xsp code. You has two servers using XSP classes mod-mono-server
and xsp server. mod-mono-server is configured to talk with mod_mono_unix
through a unix socket (it's very easy to understand and it's easy to
adapt to your needs).

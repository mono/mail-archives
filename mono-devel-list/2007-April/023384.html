Hello,

I've made a patch to add the missing "ifdefs" to permit the compilation
with "--enable-minimal" flag.

I test the compilation after applied the patch with embedded and non
embedded compilation and it works, if you have some problems please
let me know.

To applied the patch copy it at the root directory of the source and type
the following command :
  patch -p0 -i MonoEmbedded.patch

After this follow the instruction on this page :
  http://www.mono-project.com/Small_footprint

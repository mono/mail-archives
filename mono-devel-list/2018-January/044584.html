> .NET Core doesn’t go. Also, Mono supports different capabilities than .NET
> Core does and so has different advantages. If you still want to be able to
> target both runtimes, you should target .NET Standard which is the common
> bits between and the .NET runtimes (.NET Fx, .NET Core, Mono and .NET
> Native).

The tool 'winetricks', which is used by many Wine users,
has options to install mono 1.1 and mono 1.2 and
make them wine's default .net runtime; see
  http://wiki.winehq.org/winetricks

It also has options to install .net 1.1 and .net 2.0.  So users can easily
try out either MS .net or mono on Wine.

What would be very helpful, however, is an unattended or silent install
mode for mono.   If the mono installer supported a /q option to not prompt
the user at all, it would really make things easier for people who need
to roll out mono many times.

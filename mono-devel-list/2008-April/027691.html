> I'm trying to run a application developed with BDS 2006. This is a web application
> connecting to the SQL Server 2005.

I don't know about the exact error you got, but I do know I tried a
simple command-line program written in Delphi.NET 2006 with mono 1.2.5
and it worked.

Delphi GUI programs use the native VCL library rather than winforms,
so they won't run on Linux.  I'm not sure what that means for Windows,
but MS.Net works fine for running Windows apps anyway, so mono isn't
much use for that purpose.

> Version information: Mono Version: 1.1.4322.2032; ASP.NET  Version: 1.1.4322.2032

Are you really still using mono 1.1?  If so, you really need to
upgrade.  It's unlikely you'll be able to get help with illegal
instruction errors on such an old version.

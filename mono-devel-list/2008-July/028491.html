Hello,

    We are doing the Mono 2.0 branch today.

    Please do not commit changes unless they are important bug fixes as
the build is not currently green on the buildbots, and developers need
to work to get the build green before branching.

    If you are a Mono core developer, please help us get the build
green, as apparently some commits from Friday broke the build badly
enough that we are back in the red.   To track the status of the build
you can go here:

	http://mono.ximian.com/monobuild/

    After branching, we would like to encourage contributors and
developers to look into reported bugs in Bugzilla that might have to be
applied to the branch and resist the urge to go and implement crazy new
features on the trunk.    

    We want to make this our best release ever, and your help in making
this happen will be appreciated.

    The actual dates for the Beta releases and the actual releases is
documented on:

	www.mono-project.com/Roadmap

Miguel.

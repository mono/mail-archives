There are probably other things that aren't going to work like the stuff in System.Net.NetworkInformation. Some of these classes access /proc and /sys to fetch network interface statistics.

Obviously not a big deal if you're not using those classes, but I wouldn't count on everything in the standard namespaces to just work.

Hi!

We'll be closing down the Mono mailing lists at lists.dot.net later today.

As you may know, the new European privacy law called GDPR (General Data Protection Regulation) goes into effect soon. Unfortunately, the mailing list software we're currently using is inadequate to deal with the new requirements.

We discussed several options and decided not to invest the significant effort required for moving the mailing lists to a different system. A major reason is we've seen a steep decline in message volume on the lists since we started using GitHub Issues and Gitter chat.

On behalf of the Mono team, I want to thank all subscribers for their involvement over the last 17 years and hope you'll stay a part of our community! :)

You can use the following systems to keep in touch with us:

- Gitter (https://gitter.im/mono/mono): Mono developers regularly hang out in the chat, perfect for asking interactive questions
- GitHub (https://github.com/mono/mono/issues): Feel free to file an issue if you want to discuss something, even if it's not a bug

Note: this doesn't affect the mailing list archives, they will stay online and existing links will keep working.

- Alex

On Sat, 2007-01-20 at 14:46 -0800, Martin Olsson wrote:
> Hi,
> 
> Will it be possible to write cross-platform application using
> Mono+WinForms and have the application look and feel *exactly* the same
> on both Linux and Windows?

It is our goal to make winforms run in the exact same manner as MS's
implementation.  That being said, it would be a little crazy to release
a piece of software for a platform you haven't even tested on.  There
will be some platform difference's that you need to take into account
(things like case sensitivity of file names) and there would be
deployment issues to take into account (what sort of installer do people
on that platform use).

I think we have done a good job of making things consistent.  There have
been many things in winforms that we have wanted to improve upon, but
have not, so that there would be a consistent platform for developers.

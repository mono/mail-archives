OracleClient is missing most of it's .Net 2.0 API. I have some work to
fix this on it's way, but I cannot declare any binding deadlines.

With the curent state, I wouldn't expect any autogenerated code from
VS2005 to work with mono OracleClient - not only many methods are
missing, even the class hierarchy is wrong - it has changed between
1.1 and 2.0, and those changes have not been introduced to  mono.

In fact I found this nice library on GITHUB, zxing.MonoTouch.
https://github.com/martinbowling/zxing.MonoTouch

Now I just needed to to generate an image on the fly...

I found this really helpful article on how to generate images from the
camera, by using screen capture.
Here is the article: MonoTouch video capture in c# code
http://blog.reinforce-lab.com/2010/02/monotouchvideocapturinghowto.html

You can find the the code here:
https://github.com/reinforce-lab/com.ReinforceLab.MonoTouch.Controls/tree/master/VideoCaptureSample/

So by putting them together you will have your own bar code reader
which will cost you nothing!!!!

when I add a MonoTouch Library Project to a solution that already contains
an iPad Window-based Project, the Library project has only the
configurations "debug" and "release", instead of the usual debug and release

project is not compiled: in MonoDevelop there is a message in the Solution

the build two times: first for library, in debug or release modality, and

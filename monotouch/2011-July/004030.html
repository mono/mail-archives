Hello,

    If you installed Lion and are having problems using MonoTouch, please
get in touch with me.

    We are looking for developers having any of these problems:

* MonoDevelop startup
* Simulator build problems
* Device deployment problems
* Debugging on device or simulator
* Activation problems
* First install, or renewed install problems

     Please contact me with the following information:

* XCode version you have installed (if multiple, specify which)
* MonoDevelop version (from the about box)
* Mono version (run mono --version to get the version number)
* MonoTouch version (run /Developer/MonoTouch/usr/bin/mtouch --version)

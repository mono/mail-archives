You have to set your first controller's NavigationItem.BackBarButtonItem property.

Let's say that TableController is your... table controller that will push other controllers when its cells are selected.
To set those controllers' back button title, do this:

this.NavigationItem.BackBarButtonItem = new UIBarButtonItem("MyCustom Back Button Title", UIBarButtonItemStyle.Bordered, null);

inside the TableController and not in the controllers that will be pushed from it.

There is no problem if you pass null for its event handler. It will have the default behavior of popping the controller and returning to the previous one.

I've not reviewed the EULA for SQL CE, but I think Matt probably has a valid
point here, and you may want to check that out.

You may also want to check with the VistaDB guys.  They have a fully managed
DB that is supposed to be a drop-in replacement for SQL CE.  I think their v3
would be the most likely to work on MonoTouch; however, it would need to 
be recompiled, so you would probably need to get them to buy into this idea.

--Joseph

Hi,

You can download mono 2.10.7 here:
http://www.go-mono.com/mono-downloads/download.html

Note that you'll also need the latest (stable) MonoDevelop version if you
want any improvements in MonoDevelop (since some updates to MD were
required too).

Also have in mind that there are known regressions in 2.10.7 (it's a beta
after all), but we're working on a 2.10.8 which should solve most of the
regressions.

MonoTouch is not using the installed mono (it has its own mono copy), so
which version of mono you have installed will not affect MonoTouch in any
way.

Rolf

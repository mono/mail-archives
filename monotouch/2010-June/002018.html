Sure. I think the easiest way is to create custom drawn UIView and place it
OVER the map. Make it identically sized to the MKMapView.

Next, use the MKMapView member functions to determine View coordinates of
the points you want to plot.

Send that data to your custom drawn view and have it render those paths.

Make sure to set InteractionsEnabled = false to your overlay view so that
touch events still get forwarded to the MKMapView.

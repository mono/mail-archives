Apple dropped the old style of developing user interfaces with XCode 4.
The concepts are similar, but the pipeline changed.

When you get MonoDevelop 2.8 Alpha 1, you are getting the version of
MonoDevelop that is designed to work with the latest version of Apple's
tools and this will require you to update your code and re-learn what you
had previously learned.   To simplify this process, we published two
tutorials, a 10 page tutorial to help those transitioning from Xcode 3 to
Xcode 4, and a 25 page tutorial for those new to .NET development on the
iPhone, aimed at getting started directly with Xcode 4.

This was part of the announcement of the Alpha release:

http://blog.xamarin.com/2011/08/17/early-adopters-ios-5-and-xcode-4-support/

My recommendation if you are only getting started and following any
published books is that you stay with the Stable releases, and do not
download any Beta or Alpha software releases.

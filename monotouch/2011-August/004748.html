There are a number of good c# books.  I'm partial to the wrox line, but that is not surprising given that I write for wrox.  
As for Monotouch, you will have to learn the iPhone-isms, so you will have to learn these even if you don't do mono touch.
As for Monotouch, there are 3 books out that I am aware of:Mine: Professional iPhone Programming with MonoTouch and .NET/C# (Wrox Programmer to Programmer) (I say mine, but Craig Dunn and Chris Hardy did a huge amount of work on it)Bryan Costanich: Developing C# Apps for iPhone and iPad using MonoTouch: iOS Apps Development for .NET DevelopersMike Bluestein: Learning MonoTouch: A Hands-On Guide to Building iOS Applications with C# and .NET
I think someone else is working on one right now, but I don't remember who.

Hi,

To try and answer your first question in short, I would go with TCP.

Now for the longer answer. UDP is a disconnected protocol and TCP is a 
connected protocol, which means bassicly that with an TCP connection you get 
the following for free:

1) You know when an connection is actually made.
2) You know when an connection gets disconnected.
3) Packets (your data) will be received in the same order as they are send.

If you are using UDP you will have to deal with all the three points 
yourself.

Hop this helps you a bit,
Richard.

The emulator is painful no matter what language/environment you're using. I've noticed what Wally did though - if you HAVE to use the emulator you're probably going to prefer doing so under OSX. The development cycle with a decent device is not bad. Device + MonoDroid + Visual Studio + Resharper is a nice setup.

The Droid folks I work with use the emulator for testing under different deployment environments but not for day-to-day development.

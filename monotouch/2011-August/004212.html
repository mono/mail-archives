We have just released Mono
2.10.3<http://mono-project.com/Release_Notes_Mono_2.10.3>,
this release of Mono fixes the reported problems that users had when running
MonoDevelop on Lion.

This fixes several Lion problems:

   - Instability when running MonoDevelop
   - Crashes while debugging
   - Problems resizing windows
   - MonoDevelop hanging

The next time you startup MonoDevelop, you will be notified that there is a
new Mono available.   Or you can head directly to our download
site<http://www.go-mono.com/mono-downloads/download.html>to get it.

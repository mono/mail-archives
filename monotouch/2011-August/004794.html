Thanks to all the Guys that have contribute to solve my Problem.

The Problem was that the sqlfile was in the app bundle so i to that i copy
it outside by invok
this method.

private void CopyDB()
		{
			var documents = Environment.GetFolderPath
(Environment.SpecialFolder.Personal);
			string db = Path.Combine (documents, "Cleantec.sqlite");
			if (!File.Exists (db)) 
			{
				File.Copy ("Cleantec.sqlite", db);
			}
		}

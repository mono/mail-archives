Alex Corrado and I tracked this down on Friday. The problem is that the
MTActivation program was linked in such a way that it is requiring
MonoFramework 2.10.4 being installed.

We think we have a fix and Alex made 4.0.7 packages last night, we just need
to test them to make sure they work. So you can probably expect a 4.0.7
which fixes this issue in the next day or so. Until then, you can install
MonoFramework 2.10.4 and that should make it work (after you install, you
can go back to 2.10.5)

Jeff

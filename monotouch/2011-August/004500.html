Hi Lennie,

Which version of XCode do you have installed? You should have no problems
using Xcode 3.x getting to both Outlets and Actions using the Library Tool
and following the video. On the Library dialog, make sure you have selected
Classes at the top, the Controller/AppDelegate you are working with in the
middle section and the select from the bottom section drop down either
Outlet or Action depending on which you want to define.

Hope this helps.

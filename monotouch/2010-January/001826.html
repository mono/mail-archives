Hello,

> Will MonoTouch support the iPad ADSK?

It supports it out of the box today, but it does not have the new APIs exposed.

We should have the new APIs ready to go in a couple of hours.

You will need to upgrade MonoDevelop and then upgrade MonoTouch to the "Alpha" release.

The "Alpha" release is only going to be available for developers that have the iPhone 3.2 SDK installed, since it is considered Apple confidential information at this point (MOnoTouch will autodetect this).

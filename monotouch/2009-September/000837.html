Hi everyone,

Thanks to some good tips from Geoff I got debugging on the iPhone 
working. It takes some time, setting the right properties on the project 
to match the Provisioning Profile, but more important: the right 
versions of all the tools.

Now I have:
Mac OSX 10.6
iPhone OS 3.0.1
XCode 3.2 for Snow Leopard and iPhone 3.1

An older version of XCode could not work with my provisioning profile 
(which was created for iPhone OS 3.0) and my updated version of the 
iPhone (3.0.1).

But now I also found out why the code ran fine on the simulator and not 
on the iPhone: it has something to do with the Linq.Xml stuff I use 
(which is based on generics). Probably I ran into some limitations as 
mentioned in the documentation.

Does this need a fix, or should I leave the idea of using XElement?

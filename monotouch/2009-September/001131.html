I'll add some real world info.  I have compiled (note not ported) my
existing Windows business layer to use MonoTouch.  The resulting .App is
4.8M.  This is with zero UI coding, just the API layer.  This is 24k lines
(wc -l), 65 classes.  Windows DLL is 360k.  So I am a little concerned about
the size.  Then again not having to rewrite the 24k lines is fantastic.

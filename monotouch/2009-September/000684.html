Is this a machine that had Leopard 10.5 installed and was upgraded to  
Snow Leopard 10.6? If so, the problem is that the iPhone SDK was  
removed as part of the upgrade. You will need to download and install  
an updated iPhone SDK from Apple.

  - Jon

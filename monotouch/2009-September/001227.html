One solution... you can add the System.Timer code manually to your MT
project.

Timer.cs
TimerDescriptionAttribute.cs
ISupportInitalize.cs
ElapsedEventArgs.cs
ElapsedEventHandler.cs

Download the source for mono, find . -name Timer.cs then copy into your
project.

Or:
       NSTimer x;
        x = NSTimer.CreateScheduledTimer (new TimeSpan (0, 0, 0, 30, 0),
delegate { Console.WriteLine ("in timer"); });

Hope that helps.

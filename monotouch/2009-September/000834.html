> Does the MonoTouch addon work in MonoDevelop - Windows version?

Not really. A few things could easily be made to work - code
completion, file/project templates - but it would be very limited.
Compiling the native code, designing interfaces, running in the
simulator, signing the app bundle, uploading to the device, etc. all
use Apple's SDK, which only exists on Mac.

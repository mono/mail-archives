The reasons are technical and legal.  The iPhone has W^X protection,  
meaning you cannot JIT.  If a page of memory has ever been marked  
writeable, it can never be marked exectable, if you attempt to do  
this, the jump to that memory page will raise an EXC_BAD_ACCESS.

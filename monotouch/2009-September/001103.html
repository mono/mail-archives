> Anybody know how to subscribe to UIKeyboardDidShowNotification.

Just do this:

var notification = NSNotificationCenter.DefaultCenter.AddObserver  
("UIKeyboardDidShowNotification", callback);
[...]

void callback ()
{
   Console.WriteLine ("Keyboard is up");
}

To remove use:
	NSNotificationCenter.DefaultServer.RemoveObserver (notification);

If you want to impress your friends at work, use a lambda:

NSNotificationCenter.DefaultCenter.AddObserver (
    "UIKeyboardDidShowNotification", n => {
        Console.WriteLine ("Keyboard is up, notification is: {0}", n);
    });

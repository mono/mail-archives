Hello Lino,

     According to the docs this means that battery monitoring is not  
enabled, so the battery state is UIDeviceBatterState.Unknown and the  
value of the property is always -1 in that case.

     To enable battery monitoring, you should set the  
BatteryMonitoringEnabled property.   The documentation advises that  
you should only turn this on when you need it.

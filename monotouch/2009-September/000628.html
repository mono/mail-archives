I was very excited to see LoadNib (the Mono cousin of loadNibNamed:)
released in Beta 8!  The return value is different than the call I've bee
using in my ObjC iPhone applications.

If you look at the iPhone documentation, you'll see that the return value
should be an NSArray (well hopefully a Mono analog).  It's a little tricky
because there is an OSX flavor that does in fact return a bool instead,
which is what LoadNib currently does.

You can see documentation for the iPhone version below:

http://developer.apple.com/IPhone/library/documentation/UIKit/Reference/NSBundle_UIKitAdditions/Introduction/Introduction.html

Is there any chance you could tweak that?

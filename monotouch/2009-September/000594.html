Phil, this isn't a "one line" solution but I have a two-step "save to file
then use FromFile"
(code<http://conceptdevelopment.net/iPhone/iSOFlair01/Main.cs.htm>
)

WebClient wc = new WebClient();
string gravatarUrl = result.Substring(startpos, endpos-startpos);
Uri uri = new Uri(gravatarUrl);
wc.DownloadFile(uri, "gravatar.png");
UIImage img = UIImage.FromFile("gravatar.png");

I did it this way because i wanted the image cached on disk anyway - but
have not tried assigning directly...

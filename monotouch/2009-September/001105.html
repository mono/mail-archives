> I have been developing/debugging etc non-visual code using the full

Based on mcs/class/Makefile, System.Json is only built for MonoTouch and
the .NET 4.0 profiles (and presumably Moonlight, though that would be
elsewhere).  So you don't have System.Json because you're not building
the .NET 4.0 profile.

If you build trunk from source, configure mono with --with-profile4=yes
to build the 4.0 profile (or just build mcs/class/System.Json by hand).

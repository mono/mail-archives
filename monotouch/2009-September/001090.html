I have an idea for a plugin to MonoDevelop to assist me in my iPhone
development, and am curious where I should start looking to make such a
thing a reality. Please allow me to give a brief overview, and maybe some
one can point me in the right direction.
I tend to write real-time apps for the iPhone - games, simulations, whatnot.
The Xcode performance tools are therefore important to me. Particularly, the
CPU monitor - I want to know what code is dictating my performance.

I have an idea for a plugin that will display performance stats for whatever
function the mouse is currently over. That is, if I have something like:

}

then MonoDevelop would show tool tips (shown as comments) - or a side
window, whatever - whenever I mouse over or press a key.

So I have the problem of adding a plugin to MonoDevelop and creating a tool
to record performance metrics and somehow integrating that tool with
monotouch.

Is this remotely possible? Can you point to where I should start looking for

any hooks mono may provide.)

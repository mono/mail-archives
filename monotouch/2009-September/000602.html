On Tue, Sep 1, 2009 at 1:29 PM, Wallace B.
McClure<wallacebmcclure at wallym.com> wrote:
> What are folks doing for json support in monotouch?  I’ve read that json.net
> works in monotouch.  Are there any other options that I should look at?

Depending on what you need it for, I happen to be a fan of Hyena.Json,
which is a much simpler (and less rigid) JSON library from Banshee:

http://git.gnome.org/cgit/banshee/tree/src/Libraries/Hyena/Hyena.Json

I used it for Tomboy and added a serializer and more specific support
for integers, so if that seems useful you can grab it here:

http://git.gnome.org/cgit/tomboy/tree/Tomboy/Addins/WebSyncService/Hyena.Json

The tests have good examples, and here's another one straight from
Tomboy (note the optional fields, which are hard to achieve with
json.net):

http://git.gnome.org/cgit/tomboy/tree/Tomboy/Addins/WebSyncService/Api/NoteInfo.cs#n35

Hope this helps,
Sandy

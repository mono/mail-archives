I'm not sure how this is a hidden behaviour.  Lets look at the code, it was basically:


public void FinishLaunching () {
  MyViewController mvc = new MyViewController ();
  window.AddSubview (mvc.View);
}


As soon as control leaves this function, mvc has no refs left to it in managed, or in native, so its going to be garbage collected.

How do you send a message to a class method when binding to a native
Objective-C library?

I don't see anything in the Apple or MonoTouch docs equivalent to
objc_msgSend<http://developer.apple.com/iphone/library/documentation/cocoa/conceptual/ObjCRuntimeGuide/Articles/ocrtHowMessagingWorks.html>that
sends a message to a class method instead of an instance method.

Desperate, I coded it up anyway with objc_msgSend, but get a SIGSEGV at the
objc_msgSend call. I'm pretty sure the selector name is right because I get
a different error (unrecognized selector sent to class) when it's wrong.

    Class nativeClassClass = new Class("NativeClass");
    return (NativeClass) Runtime.GetNSObject(
        Messaging.IntPtr_objc_msgSend(nativeClassClass.Handle, new
Selector("selector:").Handle));

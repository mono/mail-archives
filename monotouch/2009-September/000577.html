Oisin,

   -ObjC is a flag passed to gcc documented as such:

      -ObjC       Loads all members of static archive libraries that  
implement an Objective-C class or category.

Basically when you are linking against a static library, if the linker  
does not detect usage of thre oclasses, it will remove them.  You can  
use this flag or:

      -all_load   Loads all members of static archive libraries.

to prevent that.

You would need to do this if you were binding a class / consuming  
information from a static library that you consumed purely thru pinvoke.

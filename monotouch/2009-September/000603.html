I'm using Newtonsoft's JSON library.  I got it to work by compiling it for
the Mono iPhone target, and compiling it with the SILVERLIGHT and PocketPC
symbols defined. You might not even need to define those symbols now that
System.Xml is included as I think that's what tripped it up when I first
tried it.

>
> The tests have good examples, and here's another one straight from
> Tomboy (note the optional fields, which are hard to achieve with
> json.net):

>
> http://git.gnome.org/cgit/tomboy/tree/Tomboy/Addins/WebSyncService/Api/NoteInfo.cs#n35

> Unhandled Exception: System.ExecutionEngineException: Attempting to JIT

This exception always means that you are executing code that the
compiler was not able to statically compile ahead of time.

Like Geoff said, we have ways of forcing the compilation of certain
methods to make this work, but it has to be done on a case by case
basis.

Glad to hear I'm not being heretical...

On the c# front, I'm finding the object-initialisation-in-constructor  
pattern introduced in c# 3.0 to be quite useful for this sort of thing:

view.AddSubView(new UILabel(new RectangleF(222, 13, 77, 21))
{
   Font = UIFont.SystemFontOfSize(12),
   Text = "Your name:",
   TextColor = UIColor.LightGray
});

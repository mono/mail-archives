Hello,

> Is VB.Net supported by MonoTouch?

We do not currently ship a VB.NET compiler nor the runtime needed for it
for use in the Mac with MonoTouch.  

If you use Windows with Visual Studio to compile your projects (as
someone else has already pointed out) we could get it to work, but it is
not something that we support out of the box right now.

I did something similar a few weeks ago.  I know I played around with a few
ideas before I settled on this, but I can't recall if I had any problems
getting the update to happen immediately

GetCell()

...
if (cell.Tag == 1)
{
  cell.Accessory = UITableViewAccessory.Checkmark;
}
else
{
  cell.Accessory = UITableViewAccessory.None;
}
...

RowSelected()

...
cell = tableView.CellAt(indexPath);

if (cell.Tag == 1)
{
  cell.Tag = 0;
  cell.Accessory = UITableViewAccessory.None;
}
else
{
  cell.Tag = 1;
  cell.Accessory = UITableViewAccessory.Checkmark;
}

...

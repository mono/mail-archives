That is because you are adding the button on a UIImageView, which does not have user interaction enabled by default. Assign its UserInteractionEnabled property to true so that it can tunnel user interaction to its subviews. 
You do not have to set UserInteractionEnabled to a UIButton, because it is enabled by default.

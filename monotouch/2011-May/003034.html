Hi Rodney,

> Basically I am drawing lines using a CGContext and its MoveTo,
> AddLineToPoint, and StrokePath methods. This is working perfectly, however I
> would like to add a texture to the lines that are being drawn.  

One way would be to use CGPatternCreate to create a pattern that could then be set as your stroke or fill via functions such as CGContextSetStrokePattern. Patterns can do some pretty fancy things, hence the need for providing a callback to perform the rendering etc.

First hit in a google search for this technique brought up the following sample (which I haven't tested nor vouch for) 

http://blog.robertturrall.com/2010/12/03/objective-c-drawing-with-a-texture-chalk/

If your texture is a simple image you may also be able to get away with using UIColor's colorWithPatternImage: message to create a "color" that could be used as your stroke. Simple create an image with your texture, then convert it to a colour using colorWithPatternImage:.

UIColor * textureColor = [UIColor colorWithPattern:[UIImage imageNamed:@"texture.png"]];
CGContextSetStrokeColorWithColor(context, textureColor.CGColor);

Hope this helps,

I would like to suggest the following book to anyone who would like a clear
and concise description of the fundamental concepts of Cocoa:
"Cocoa Programming for Max OS X" by Hillegass

http://www.amazon.com/Cocoa-Programming-Mac-OS-3rd/dp/0321503619

I'm a long time .NET developer and used this book when I decided to start
programming the iPhone. It does not cover CocoaTouch, but has the best
explanation of Windows, Views, Delegates, Interface Builder, etc. Once you
have this knowledge, the Apple libraries start to make sense.

-Frank

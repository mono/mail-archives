I have been beginning to move my code over from a project and noticed that there would be a good opportunity hear to automate some of this.  For instance, I was converting this objective - c code:

scrollView.pagingEnabled = YES;
    scrollView.contentSize = CGSizeMake(scrollView.frame.size.width * 8, scrollView.frame.size.height);
    scrollView.showsHorizontalScrollIndicator = NO;
    scrollView.showsVerticalScrollIndicator = NO;
    scrollView.scrollsToTop = NO;
    scrollView.delegate = self;

to this C# code:

scrollView.PagingEnabled = true;
scrollView.ContentSize = new SizeF(scrollView.Frame.Size.Width * 8, scrollView.Frame.Size.Height);
scrollView.ShowsHorizontalScrollIndicator = false;
scrollView.ShowsVerticalScrollIndicator = false;
scrollView.ScrollsToTop = false;


There could be some converter that could ask in the paste context menu to paste from Objective - C.  Also some kind of code converter that will change the code to c# for some basic operations to get moving quicker on this i.e.(import from Objective - C)

Hi
it's possible to play with sqlite with javascript + HTML5 + UIWebview

and you can call jvascript from objc :

[webView stringByEvaluatingJavaScriptFromString:[NSString
stringWithFormat:@"%@%();",
handlerCallBack]];

where handlerCallBack is the javascript function you want to call


and you can pass arguments to objc from UIWebview :

when you click on a link, the event shouldStartLoadWithRequest is called
you can catch info with :

    NSURL *url = request.URL;
    NSString *urlString = url.absoluteString;

    NSArray *components = [urlString componentsSeparatedByString:@":"];

with html code like this :  <a href="ToObjc:test:test" target="_top">my
link</a><br>

or with running js code :

window.location.href='ObjC:function:params';

I think it's possible to do that with monotouch

1- put a UIWebwiew on your GUI
2- load html page with js code : (index.html is in the bundle)
- (void)viewDidLoad {

    NSString *fullPath = [[NSBundle mainBundle] pathForResource:@"index"
ofType:@"html"];
    NSString *htmlText = [NSString stringWithContentsOfFile:fullPath];

    [webView loadHTMLString:htmlText baseURL:[NSURL
fileURLWithPath:fullPath]];

}
3- call js function dealing with sqlite (see web site of webkit)
[webView stringByEvaluatingJavaScriptFromString:[NSString
stringWithFormat:@"%@%();",
handlerCallBack]];

4- when the js code handle the result, launch something like that :
window.location.href='ObjC:function:params';
then you trigg shouldStartLoadWithRequest event

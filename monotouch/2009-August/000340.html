A new drop of MonoDevelop is now available:
http://go-mono.com/archive/MonoDevelop-MonoTouch-Preview-20090818-0.dmg

The following issues have been fixed:

* The CodeBehind generator can now handle all of the xib files in
Apple's iPhone samples.
* The "View with controller" template connects the controller to the
view and includes an IntPtr constructor.
* A "Debug in Xcode" command has been added, though it does not yet
handle the Info.plist or bundle resources in subdirectories.
* Templates in the "New Project" dialog are now correctly categorized
by language.
* Bug 531110 "MonoTouch: Application Output window stops working" has been fixed

This also includes numerous fixes to MonoDevelop itself, including a
treeview crasher in the project view.

So, here it is (attached).

I have no idea if this is based on any kind of best practices but this is my
first attempt to write anything half significant in MD (and tbh, for the
iphone in general) and I am impressed with experience.

The project works exactly the same way as the out of box "Utility Project"
template in XCode. One main view with an information-style "i" button on the
bottom right, and then a flipped over modal view with a navigation bar and a
"Done" button which flips back to the main view.

Hilariously, while MD was rock solid during this process, XCode 3.1 crashed
while I was flipping back and forth looking at their project template. This
made me even more determined to continue this process. That said, I lost
intellisense today after the latest upgrade, but still managed to get it the
job done.

Great work, and please feel free to take this project and clean it up and do
what you want with it (i'd love to see this project template added to MD).

-Oisin

>> You are right. Reflection is not possible.
>
> Reflection does work, but Reflection.Emit does not.
>
> Using Activator.CreateInstance will work, as long as you do not try to
> JIT compile any new types.
>
> For example:
>
> Activator.CreateInstance (Type.GetType ("MyClass"))
>
> As long as you reference MyClass somewhere else in your code (to
> ensure that the linker does not strip it away).
>
> But the following would not work, since it involved generics, and
> would require JIT compilation:
>
> Activator.CreateInstance (Type.GetType
> ("System.Collections.Generic.IList`1<MyClass>"));

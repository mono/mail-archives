What I did was after building in monodevelop, I ran mtouch from the command
line like this:
mtouch --sim MyApp.app MyApp.exe -nolink

then you can launch the simulator from the command line like this

mtouch --launchsim=MyApp.app

After that the code with the work-around for WebClient worked fine.

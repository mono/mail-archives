On Fri, Aug 7, 2009 at 2:06 AM, Randy Edmonds<randy at randyedmonds.com> wrote:
> There appears to be a lot of documentation (in xml files) at
> /Developer/MonoTouch/usr/share/doc/MonoTouch    Are these the files that
> MonoDevelop would be using if it could? Is there anyway other way to view
> this documentation?

Yes, this is the info that the MonoDoc GUI would be using. MonoDevelop
does use *some* of it for tooltips. However, without the MonoDoc GUI
there is no easy way to view it all.

There is also an online docs viewer at http://www.go-mono.com/docs/,
but it's not as capable as the MonoDoc GUI, for example it can't
search or edit docs. Also, I'm not sure all of the docs - e.g. docs &
links merged from Apple API docs - are available online.

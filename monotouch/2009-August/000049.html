On Thu, Aug 6, 2009 at 2:07 PM, Mike Bluestein<mike.bluestein at gmail.com> wrote:
> Is there  away to set emacs keybindings in monodevelop? I see some people on
> various blogs saying it is supported. When I choose the Preferences->Key
> Bindings->Scheme I see options for Custom and Default, no emacs though. Does
> it need to be downloaded from somewhere?

Unfortunately, because the key names and availability on the Mac
differ from other systems, MD on Mac cannot use the keybinding schemes
we already had defined for MD Linux/Windows.

If you create a good custom scheme that you think would be useful to
other developers, or improve on the existing Mac scheme, I'd be happy
to merge that into MD for the next release.

> While using MonoDevelop with MonoTouch, I have noticed that it defaults, on my Mac, to Portuguese (pt) language, instead of Portuguese Brazilian (pt-br). Of course, the languages are quite similar, but there are some important differences. Besides that, some commands and menu items aren't translated at all (they are shown only in English).
>
> I was wondering how could I contribute with such translation, and maybe if somebody here could help me out.

There *is* a pt-br translation, though it's not up-to-date for MD 2.2.
I don't think any of the translations have been updated since 2.0.

MonoDevelop uses Gettext for translations, which picks up the current
language using any of the LANG/LANGUAGE/LC_MESSAGES environment
variables. Maybe the Mac is setting the wrong value for these?
Unfortunately I'm not familiar with how much things work on the Mac,
bu hopefully that'll help you to find the solution.

Regarding updating translations, we have some instructions in MonoDevelop SVN:
at http://anonsvn.mono-project.com/viewvc/trunk/monodevelop/main/po/README?view=markup
However, we're coming up to the string freeze for MonoDevelop 2.2,
currently scheduled for Sept 15
(http://monodevelop.com/Development_Roadmap), at which point I will
update all the po files and send out an email to the monodevelop
mailing with more detailed instructions for translators. So you might
want to wait for that.

Hi Thorsten,

 

since MonoTouch adds to the XCode ( the "Standard" environment for
iPhone, Cocoa and MacOS development ), you are required to have XCode
up&running.

 

XCode runs only on MacOS which again only runs on Macintoshs with > OS X
and Intel chips.

 

There are samples, where parts of iPhone apps have been developed on
Windows, using gcc, but that will only be an option if you don't plan to
use the Cocoa and other Frameworks ( e.g. if your plan is to develop
games using C++ ).

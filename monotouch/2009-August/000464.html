I don't understand.  UITabBarController does not have tabBarItem.  Are you
talking about something like this:

tabBarController.TabBar.Items[0] = new
UITabBarItem(UITabBarSystemItem.Favorites, 1);

This does not seem to work, and besides it seems wrong.  Because the tabs
are set via UITabBarController.SetViewControllers method I would expect that
each UIViewController passed into this method should specify its own tab as
it does in Obj C.

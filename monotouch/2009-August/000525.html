Hi Tim,

I came across this bug earlier and it has been fixed (just retried it with the latest beta).

Try the following:

1) Grab the latest version of MonoDevelop
2) Get the latest beta of MonoTouch (7)
3) Open up the monocatalog sample
4) Select rebuild all (if you simply select run it will not work and will still be based on the previous beta)
5) select run and do the test again

Seems to pass for me.

Cheers,

John

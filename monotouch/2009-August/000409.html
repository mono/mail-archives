I've found the offending code:

const string kCustomCellID = "MyCellID";
CustomCell cell = (CustomCell)tableView.DequeueReusableCell(kCustomCellID);

kCustomCellID should be unique.  I modifed to the following and it works:

string kCustomCellID = "MyCellID" + indexPath.Section + indexPath.Row;

Perhaps the MonoTouch samples should be updated.

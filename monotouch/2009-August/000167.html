As I'm trying to share code between MonoTouch and Windows, I ran into a
descrepency wrt OpenTK and the Math classes.
In MonoTouch, Vector2 is in the namespace OpenTK.
On Windows, Vector2 is in OpenTK.Math.

This forces me to put the following code at the top of any file that uses
OpenTK:

#if WINDOWS

using OpenTK.Math;

#else

using OpenTK;

#endif

Is there any rational behind this?

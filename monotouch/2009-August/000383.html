I mentioned this before on the list - Cocoa is sufficiently different from
.NET that I recommend you buy a well written book and read it. Or, at least,
the first few chapters. Tutorials will get you along, but you need to invest
a couple days in understand the MVC pattern employed. To that end, I
recommend:
Cocoa(R) Programming for Mac(R) OS X
http://www.amazon.com/Cocoa-Programming-Mac-OS-3rd/dp/0321503619

It does not cover Cocoa Touch, but does teach you how to think in Apple's
terms. From there, read the documentation in Xcode. It's very well written
and time spent reading their guides will pay for itself. ;-)

-Frank

> A new drop of MonoDevelop is now available:
> http://go-mono.com/archive/MonoDevelop-MonoTouch-Preview-20090826-0.dmg
>
> The following issues in the MonoTouch addin have been resolved in this
> build:
>
> * Generated Xcode projects now include the Info.plist and xibs, and
> correctly place resources in subdirectories. This fixes the "Debug in
> Xcode" command, and requires B6 of the MonoTouch SDK.
> * The codebehind generator no longer generate partial classes for
> objects from referenced xibs. This prevents conflicts on the
> [Register] attribute.
> * The "View with controller" template now includes an NSCoder
> constructor, allowing the controller to be created by a xib.
> * There are new icons for the MonoTouch templates, panels and xib files
>
> This also includes a number of fixes to the MonoDevelop core. Please
> file bug reports at for any MonoDevelop bugs you encounter so that we
> can fix them for 2.2:
> http://monodevelop.com/Developers#Reporting_Bugs. MonoDevelop is now
> feature-frozen for the 2.2 release, which is planned for the end of
> October.

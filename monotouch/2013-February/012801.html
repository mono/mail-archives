This is very easy!!

You have to get first the UUID of each iphone/ipad device and then register the device in the apple developer.

Then you have to create a Distribution profile (Adhoc) and then select all the already added devices that you want to support with the distribution.

Then you download the new distribution and you register it and thats all!!

Under Monodevelop you only have to set the distribution under the build options of your project and Voilaaaa!!

Good luck!!

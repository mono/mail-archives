I put up a rakefile for compiling F# files inside MonoTouch at
git://github.com/banshee/fsharp_rakefile.git

The idea is that you use Visual Studio (I'm running it under
Parallels) to edit your F# code and make sure it compiles.  On the
monotouch side, there's some simple rake tasks that look at the VS
project, extract the .fs files, and run fsc.exe under mono.  (There's
only trivial dependency checking; if any .fs file is more recent than
the dll, all files are compiled.)

(I still haven't had much time to use F# under monotouch, but thought
someone else might find this useful.)

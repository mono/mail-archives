Hi Dimitris,

That is a good place to start, yes. The version numbers for
dependencies would need updating, but everything else still applies.
Other articles on the MD site have not aged as well, so if you have
questions about how to do specific things, please ask on the
MonoDevelop mailing list and we can point you at examples.

You can find a few more recent tips here:
http://stackoverflow.com/questions/9669210/how-to-write-an-add-in-for-monodevelop-2-8/9670814#9670814

In particular I would recommend using a custom execution command with
the MONODEVELOP_DEV_ADDINS env var, so you can debug the addin.

-- Michael

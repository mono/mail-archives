Hi Rafael,

Try this:

void msgSelected (DialogViewController dvc, UITableView tv,
NSIndexPath path)
{
    var body = ((MessageElement) dvc.Root[path.Section][path.Row]).Body;
}

Hope that helps,

Jeff

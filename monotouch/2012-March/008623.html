> Hey,
> 
> This is supported in MonoDevelop. Right click on the solution file in the solution pane, go to the options, then go to the Build -> Configurations section. In this section you can edit the Configuration Mappings. This is where you can set project X to build with configuration Y when the solution is set to 'Debug'. For example, when the solution is set to debug you could have one project set to a 'DebugSimulatorOnly' configuration and another project set to 'DebugExtraLogging' configuration.

> Hope this helps,
> Alan.

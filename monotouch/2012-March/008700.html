The code is on the Mac (and I'm on Windows) but it's not a code problem per se. I've tested, and it seems size changes for a UISegmentedControl will not animate, but position changes will. So when the code first executes, it animates because I'm moving the control from one side of the screen to another (although I did notice it shrunk to the new size immediately, and not slowly over the animation). Next execution of the same code only changes the size of the control, hence no animation is visible. I changed it to move it back to the other side, and it is animating then.

The remaining controls didn't animate based on their Hidden values, but using Alpha instead of Hidden works.

Yes I am using the recommend naming conventions stated in Apple
documentation.  While it also states you can name and put them anywhere you
want as long as you update the Info.plist CFBundleIconFiles array with the
location/name.  This currently doesn't seem to be supported in the latest
MonoTouch 4.0.

You seem to have to use the default names and they are case sensitive, and
they apparently have to be in the root of the project.

Apple has an official Technical Q&A artical QA1686 that goes into the deails
of guideline for the icons for anyone who needs to know the filenaming
convention and the specifications of the image for each file.

http://developer.apple.com/library/ios/#qa/qa1686/_index.html

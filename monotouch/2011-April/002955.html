If anyone wants to quote for this please send me an email.

Requirement
A controller which will show a page of a PDF file on the hard disk.
Must be able to specify which page to start showing from.
Must have pinch to zoom in/out.
Like the photo viewer app on the iPhone if the user is zoomed out and 
drags the page more than 50% out of view and releases the next (or 
previous) page will slide on instead.
If the user holds their finger down I want a popup menu to appear with 
one option "Location", selecting this menu option will output to the 
console the page number + x,y position (percentage I presume?) relative 
to the page (not the view, which may be zoomed or unzoomed.)
I'd also like to have a visible indicator animate momentarily at this 
x,y position on the page if I later bring up the controller specifying 
the page number *and* x,y position on the page.


There will probably be paid additions later too.

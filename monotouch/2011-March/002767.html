All the MonoTouch tools are native Mac. You use the same interface
builder (if you want to) to build UI, and the same API's if you want
to do it in code.

was CoreData, but the Novell chaps fixed that recently, so as far as
know, there is nothing you can't do in MT that Obj-C/XC does.

Hi,

Exactly what is failing? Depending on the failure, there might be things
you can do (if it's a linker failure, you can turn on linking for simulator
builds, if you're targetting a specific iOS version, you can select the
version of the simulator to run against, etc.) There are however a lot of
cases that will still be different (for instance there is no way to detect
Ahead-of-Time compilation issues with the simulator)

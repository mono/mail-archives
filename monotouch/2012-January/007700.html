iOS does not have native epub support.  You'll either have to write that
yourself, or find some existing library you can port to MT

iOS does have native PDF support, but I don't think there is a simple way
to write a view to a PDF file.  An overview of the iOS PDF system is here:

http://developer.apple.com/library/ios/#documentation/GraphicsImaging/Conceptual/drawingwithquartz2d/dq_pdf/dq_pdf.html

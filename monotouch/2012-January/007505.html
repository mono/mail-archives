The < > syntax is different between Obj-C and C#.

You'll need to implement it as UIViewController in C#.

public partial class printerTestMonoTouchViewController : UIViewController

If you need to implement the NSStreamDelegate, you'll probably want to
subclass NSStreamDelegate separately.

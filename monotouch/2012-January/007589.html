Hi,

MonoDevelop doesn't actually use MSBuild (or our equivalent, xbuild) to
build MonoTouch projects. The support for the file format is inside
MonoDevelop itself, and only just enough to support the features of the
file format we actually use (which is why your WildcardContent doesn't
work).

So I think the best (if not only) option for now is to have your own
standalone tool you run during your build process.

Rolf

                        We have just released <a title="Mono 2.10.3" href="http://mono-project.com/Release_Notes_Mono_2.10.3" style="text-decoration:none;color:#0088cc" target="_blank">Mono 2.10.3</a>, this release of Mono fixes the reported problems that users had when running MonoDevelop on Lion.<BR>


This fixes several Lion problems:<BR>
<ul style="padding:0;margin-left:1em">
<li style="line-height:1.6;margin-left:1em">Instability when running MonoDevelop</li>
<li style="line-height:1.6;margin-left:1em">Crashes while debugging</li>
<li style="line-height:1.6;margin-left:1em">Problems resizing windows</li>
<li style="line-height:1.6;margin-left:1em">MonoDevelop hanging</li>
</ul>
The next time you startup MonoDevelop, you will be notified that there is a new Mono available. &nbsp; Or you can head directly to our <a href="http://www.go-mono.com/mono-downloads/download.html" style="text-decoration:none;color:#0088cc" target="_blank">download site</a> to get it.<BR>

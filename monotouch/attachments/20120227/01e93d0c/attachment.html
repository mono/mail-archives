<br></div><div>The MT app can be in the same solution, btw. You just need <a href="https://github.com/jamiebriant/VsMono">something </a>to fool VS into loading it, and a hacked version of monotouch.dll that claims to be .NET 4. Then you can use R# Solution Analysis to catch errors as you type: if R# says there are no errors, MonoDevelop will compile it on the Mac 99.99% of the time. I edit my MT apps on windows.</div>

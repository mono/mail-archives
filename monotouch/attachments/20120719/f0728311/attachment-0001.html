You can install different MonoTouch versions in separate directories, e.g.<br>
<br>
/Developer/MonoTouch-5.2.12/<br>
/Developer/MonoTouch-5.2.12-hoxfix/<br>
/Developer/MonoTouch-5.3.3/<br>
<br>
Then tell MonoDevelop which one to use with:<br>
- MonoDevelop menu<br>
- Preferences...<br>
- SDK Locations<br>
- MonoTouch SDK<br>
<br>
You might have to restart MonoDevelop for the change to be applied.<br>

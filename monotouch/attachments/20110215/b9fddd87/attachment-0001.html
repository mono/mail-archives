The export line for onSuccess is slightly wrong.  Do not forget the trailing &quot;:&quot;.<div>[Export(&quot;onSucesss:result:&quot;)]</div><div><br><div>Try this extra arguments line:</div><div><meta charset="utf-8"><span class="Apple-style-span" style="font-family: Calibri; font-size: 16px; "><div>

<div><br></div><div>The &quot;-v -v -v&quot; will help isolate which frameworks you need to add.  Check the build output.</div><div><br></div><div>For the libz.1.2.3.dylib library you will most likely need to manually call dlopen.</div>

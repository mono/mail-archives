</div>The default ABMultiValue&lt;T&gt; value as returned by e.g.<br>
ABPerson.GetPhones() is immutable.  You need to instead use a modifiable<br>
version.  There are two ways to do this:<br>
<br>
 1. Via ABMultiValue&lt;T&gt;.ToMutableMultiValue() [0]<br>
 2. By explicitly creating a mutable subclass [1, 2, 3], and adding<br>
    values to the ABMutableMultiValue&lt;T&gt; [4].<br>
<br>
Once you have an ABMutableMultiValue&lt;T&gt;, you can add new values, then<br>
use e.g ABPerson.SetPhones(), passing the ABMutableMultiValue&lt;T&gt;<br>
instance.<br>

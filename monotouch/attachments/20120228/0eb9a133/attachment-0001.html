          over by making it so that projects cannot target multiple
          platforms. Since its impossible to do, people then figure,
          "Well if I'm fucked anyway, I might as well start doing other

          specific code". This is not sound.</div>

        MTch specific in MTch project <br>
        SomeClass is linked either with ProjectLinker or manually</li>
      <li>SomeClass.M4A.cs<br>
        M4A specific in M4A project </li>
      <li>repeat that for other targeting platforms <br>
      </li>
      <ul>
        <li>SL, <br>
        </li>
        <li>WF, WPF <br>
          those share the same codebase in BL<br>
        </li>
      </ul>
    </ul>
    <p>Sometimes we split classes into files when working in parallel on

<p>You shouldn&#39;t compile with an older Apple SDK unless you have a problem with the current default. Newer SDKs can target older versions of iOS just fine - set your app&#39;s &quot;deployment version&quot; (aka minosversion) to the oldest OS you want to support, and ensure you guard any uses of newer APIs with runtime version checks as Jeff described.</p>

<div>Hi John,</div><div><br></div>DismissModalViewController() now clears the parent&#39;s ModalViewController property, which, if that is the only reference that still exists for your ppvc, will be lost (cleared to null). What you need to do is make sure you hold a reference to that object elsewhere.<div>
<br></div><div>This is one of the fixes in the 4.1 beta branch which is needed to more accurately reflect the state of the underlying objects. The reason it happened to work because is because we wrongly kept the ModalViewController property cached which prevented expected garbage collection when the modal view was dismissed.</div>

    Byproduct of Patterns and Practices project called Prism where MS
    guys explain <br>
    code sharing on WPF, SL and lately  WP7 platform.<br>
    Project linker is just a tool VS plugin that helps batch linking of
    files. Defines <br>
    project dependencies in sense of code sharing (copy as link + paste
    or copy + paste as link)<br>

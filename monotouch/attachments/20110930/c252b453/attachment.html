Hey jowi,<div><br></div><div>The easiest solution I can think of is to extend an UIView and override the TouchesBegan and TouchesEnded methods. Using UIButton (and their Touch* events) is for buttons with text only AFAIK. So, here&#39;s what you do:</div>

<div><br></div><div><div><div>public class RemoteControlButton : UIView {</div><div><br></div><div>  public RemoteControlButton(UIImage up, UIImage down) : base() {</div><div>    AddSubview(new UIImageView(up));</div><div>

    AddSubview(new UIImageView(down));</div><div>    // Assume both images are the same size:</div><div>    Frame = new RectangleF(new PointF(0, 0), up.Size);</div><div>  }</div><div>  </div><div>  public override void TouchesBegan(NSSet touches, UIEvent evt) {</div>

<div>    base.TouchesBegan(touches, evt);</div><div>    ExchangeSubview(0, 1);</div><div>  }</div><div>  </div><div>  public override void TouchesEnded(NSSet touches, UIEvent evt) {</div><div>    base.TouchesEnded(touches, evt);</div>

<div>    ExchangeSubview(0, 1);</div><div>  }</div><div>}</div></div></div><div><br></div><div>And then add it to your UIView or UIViewController (override ViewDidLoad) like this:</div><div>







<p>View.AddSubview(<span>new</span> RemoteControlButton(UIImage.FromFile(<span>&quot;PATH1&quot;</span>), UIImage.FromFile(<span>&quot;PATH2&quot;</span>)));</p><p>And to answer you second question; Give <a href="http://iconfinder.com" target="_blank">http://iconfinder.com</a> a go.</p>

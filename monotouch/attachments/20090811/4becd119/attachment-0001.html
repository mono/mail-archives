Additions in Beta 2:<br>
<br>
      * Added a -linksdkonly parameter to mtouch to link only the sdk<br>
        and preserve the user assemblies.<br>
<br>
Fixes in Beta 2:<br>
<br>
      * Microsoft.Win32 namespace was too aggressively linked.<br>
      * HttpRequestCreator and friends (Ftp, File) were missing their<br>
        constructors.<br>
      * mtouch was not properly reporting missing assemblies.<br>
      * mtouch was not resolving references recursively.<br>
      * mtouch now takes only one assembly, the others have to be<br>
        specified by -r.<br>
      * Fixed WebRequest authentication.<br>
      * mtouch now signs the SDK assemblies.<br>
      * Fixed a linker bug wrt virtual methods implementing interfaces<br>
        and override a method defined in a base class.<br>
      * Fixed a linker bg wrt virtual methods implementing generic<br>
        interfaces.<br>
      * Fixed a mtouch bug where arguments were not properly quoted when<br>
        aot compiling the assemblies.<br>
      * Fixed -framework linking when using things .<br>
      * Improved imported documentation (properties now imported, etc.).<br>
<br>

</div>The pattern is to override not the Dispose () method but the Dispose<br>
(bool disposing) method.   The flag that you get (disposing) tells you<br>
whether the invocation is happening because the user invoked the<br>
Dispose() method directly, or if this is being called as a Finalizer<br>
by the runtime (in a separate thread).<br>

Hi jgsacris,<div><br></div><div>I just did this today, so you are in luck!</div><div><br></div><div>First, install Mac OS X Lion. Nothing scary here. It went smoothly for me, at least.</div><div><br></div><div>Then, once Lion is installed, you can install Xcode 4.3.</div>
<div><br></div><div>If you plan to have Xcode 4.3 delete Xcode 4.2 (like I did) after it installs, you'll want to move /Developer/MonoTouch and /Developer/MonoAndroid (if you also have that) to a temporary location while you install Xcode 4.3:</div>
<div><br></div><div>In a terminal, run the following command:</div><div><br></div><div>sudo mv /Developer/MonoTouch ~/</div><div><br></div><div>That will move your MonoTouch install into your home directory.</div><div><br>
</div><div>Then download & install Xcode 4.3. Once that is complete, launch Xcode and it will ask if you'd like to delete your own Xcode in /Developer. Feel free to do so.</div><div><br></div><div>At this point, you can safely move your MonoTouch install back into /Developer.</div>
<div><br></div><div>To do so, type the following commands:</div><div><br></div><div>sudo mkdir /Developer</div><div>sudo mv ~/MonoTouch /Developer</div><div><br></div><div>Now you'll need to install some addons for Xcode (such as the iOS Simulators and the command-line tools):</div>
<div><br></div><div>Go to the Xcode menu and select "Preferences..."</div><div><br></div><div>Next, click on the "Downloads" tab</div><div><br></div><div>Now click the "Install" button next to "Command Line Tools" and any iOS Simulators that you want (I installed the 4.3 and 5.0 Sims).</div>

<br>
Additions in 1.1.1<br>
<br>
        • Added GameKit bindings<br>
        • Added MessageUI bindings<br>
        • Added NSMutableData<br>
        • Added some additional NSData overloads<br>
        • Added new mtouch flag -linkskip.  If you are attempting to use a<br>
third party dll, you may need -linkskip to avoid missing references<br>
while linking.<br>
Fixes in 1.1.1<br>
<br>
        • Fixed over-aggressive linker optimization (Guid.NewGuid () issue)<br>
        • NSObservationCenter will now explicitly call Dispose() on the<br>
NSNotification posted when using the Action&lt;T&gt; overrides.  Fixes the<br>
issue where it was impossible to play 2 videos in the MediaPlayer sample<br>
You can get the update for your version of MonoTouch by either running<br>
MonoDevelop-&gt;Check for Updates, or restarting MonoDevelop.  If you do<br>
not have these options, you are running an older version of<br>
MonoDevelop and should upgrade from <a href="http://monodevelop.com/Download/Mac_MonoTouch" target="_blank">http://monodevelop.com/Download/Mac_MonoTouch</a><br>

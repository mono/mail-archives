Hi Mittchel,<div><br></div><div>To uninstall MonoTouch:</div><div><br></div><div>Delete the following files &amp; directories:</div><div><br></div><div>/Developer/MonoTouch</div><div>/Library/Frameworks/Mono.framework/External/xbuild-frameworks/MonoTouch</div>
<div>/Library/Frameworks/Mono.framework/External/xbuild/Xamarin/Xamarin.ObjcBinding.*</div><div><br></div><div>Then, from a command prompt, run the following command:</div><div><br></div><div>sudo pkgutil --forget com.xamarin.monotouch.pkg</div>
<div><br></div><div><br></div><div>To uninstall MonoDevelop:</div><div><br></div><div>Delete MonoDevelop.app like you did before and then delete the following directories (and their contents):</div><div><br></div><div>~/Library/Application Support/MonoDevelop-2.8</div>
<div>~/Library/Caches/MonoDevelop-2.8</div><div>~/Library/Logs/MonoDevelop-2.8</div><div>~/Library/MonoDevelop-2.8</div><div>~/Library/Preferences/MonoDevelop-2.8</div><div><br></div><div><br></div><div>To uninstall Mono framework:</div>
<div><br></div><div>Delete /Library/Frameworks/Mono.framework and then issue the following command:</div><div><br></div><div>sudo pkgutil --forget com.ximian.mono-2.10.6-csdk</div><div><br></div><div>(replace 2.10.6 with whatever version you actually have installed)</div>

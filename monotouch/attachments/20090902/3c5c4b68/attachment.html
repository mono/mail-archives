<div>  -ObjC is a flag passed to gcc documented as such:</div>
<div><br></div>
<div>
<div>     -ObjC       Loads all members of static archive libraries that implement an Objective-C class or category.</div>
<div><br></div>
<div>Basically when you are linking against a static library, if the linker does not detect usage of thre oclasses, it will remove them.  You can use this flag or:</div>
<div><br></div>
<div>
<div>     -all_load   Loads all members of static archive libraries.</div>
<div><br></div>
<div>to prevent that.</div>
<div><br></div>
<div>You would need to do this if you were binding a class / consuming information from a static library that you consumed purely thru pinvoke.</div>

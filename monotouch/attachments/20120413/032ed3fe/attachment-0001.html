<div><font color="#222222" face="arial, sans-serif">In ObjC, like C and even C#, all typedef does is alias a type. Function pointers in C and Blocks in ObjC need to be typedef'd if you are going to use them like "named delegates" (as opposed to anonymous delegates).</font></div>
<div><font color="#222222" face="arial, sans-serif"><br></font></div><div><font color="#222222" face="arial, sans-serif">So, our revised syntax for named blocks is this:</font></div><div><font color="#222222" face="arial, sans-serif"><br>

<DIV>It looks like you've got something like a mix of C# and Objective-C code 
there. In C# it would look more like:</DIV>
<DIV><BR></DIV>

<DIV><BR></DIV>
<DIV>As far as the file path, (the ... above), I haven't tried using the NS 
resource stuff at all; I just mark my html files with the "content" build action 
and use paths relative to the project root.</DIV>
<DIV><BR></DIV>
<DIV>But, if the resource stuff works, the method you're trying to call would 
look like:</DIV>
<DIV><BR></DIV>
<DIV>
<DIV><SPAN style="WHITE-SPACE: pre" 
class=Apple-tab-span></SPAN>NSBundle.MainBundle.PathForResource("<SPAN 
style="BORDER-COLLAPSE: collapse; FONT-FAMILY: arial, sans-serif; FONT-SIZE: 13px"

In some cases the ObjectiveC exception code plays tricks on MonoTouch and
the managed exception handling. An exception handler in Main should catch
all exceptions in the main thread (but as you've seen yourself it doesn't
quite work in these cases).

In this case pretty much the only thing left to do is to try to get the
device log.

You define this in the ShouldAutoRotateToInterfaceOrientation for that view controller.  You have to decide which orientation you want, LandscapeLeft or LandscapeRight (or both).
So…

public override bool ShouldAutorotateToInterfaceOrientation (UIInterfaceOrientation toInterfaceOrientation)
{
if (toInterfaceOrientation == UIInterfaceOrientation.LandscapeLeft)
{
return true;
}
else
{
return false;
}
}

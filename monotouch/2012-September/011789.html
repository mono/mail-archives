>> Well, you make a uitableviewcell, as normal, and into it's
>> contentview, you add a uiscrollview which takes up the whole space of
>> the cell (you may want to remove any labels which are already in the

>>
>> And then configure it to be however big you need (ie, the scrolling
>> area - the visible area is based on the cell), and add in whatever
>> other views you want. Then the user can scroll left/right, as well as
>> up/down.

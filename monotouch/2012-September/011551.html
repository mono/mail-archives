I would think it is an incompatible API call you are making that was introduced new in ios5 and not available in ios4. For example, I believe the NavigationItem.LeftBarButtons (plural version) property is new to ios5 while ios4 only has the NavigationItem.LeftBarButton (singular version).  It would be nice to catch this at compile time somehow by I couldn't figure out how. 

You should be able to reproduce the crash by running your app in an ios4 simulator.

hard to say without looking at what "comes with" the theme, however i'm
going to guess they're using UIAppearance (if it is for iOS5+)

http://docs.xamarin.com/ios/tutorials/Introduction_to_the_Appearance_API

using the static Appearance properties lets you globally assign certain
visual properties to different UIKit objects, so it's a good fit for
"theming". there's samples around that let the user choose/adjust a theme
at runtime, although i think that's a bit extreme...

IIRC Apple added even more UIAppearance properties in 5.1 (and possibly
6.0) so if you use it, test carefully so that you only reference valid
properties for the iOS you're targetting.

HTH

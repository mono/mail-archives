Hi All
Based on the fact that we want to support new iOS releases as soon as 
possible, but that we need to have some room to be responsive to 
changes that are outside of our control, we prefer to avoid naming a 
release date in advance.

That said, we have pretty consistently delivered support for new releases
within a few days of Apple's release.

Hope that helps

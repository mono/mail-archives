Apple has a nice "Printer Simulator" application with the iOS 4.2 SDK that
allows you to simulate different types of printer. So you don't really need a
printer to integrate printing.


You can still print to your USB printer via the following ways
1. Third party solutions likePrintopia
http://www.ecamm.com/mac/printopia/
 
2. Unsupported - Run the Air print service on Windows
http://forums.macrumors.com/showthread.php?t=1047779

Mike,

  This is a side effect of you enabling -gcc_flags.  We have a fastpath when you dont have GCC flags to run a speciaized launcher, which we cannot use with -gcc_flags.  When you use -gcc_flags you need to enable the linker to remove the frameworks which you aren't using otherwise our tools will try to include them

-g

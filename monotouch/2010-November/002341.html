Hi Geoff,
 
Here is a more detailed scenario for this request.
 
Let's say I want to integrate Urban Airship's Storefront Library project
http://urbanairship.com/docs/storefront.html#overview
 
What I would like to do is to add an XCode project to my MonoDevelop Solution
and add their code files to that XCode project and the bTouch files get
generated automatically.
 
I would like to call their ViewController classes just like native .NET classes
from my code.
 
So, I guess I am asking for support for Objective-C projects in MonoDevelop and
doing the bTouch magic behind the scenes.

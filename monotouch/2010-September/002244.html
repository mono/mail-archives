Hello Mike,

We have a requirement to store certificates securely on the iPhone.  Are
> there any plans for Monotouch support for the keychain APIs, or is this
> something we would need to hand craft?
>

We will look at those in the future, but this wont happen soon.

For now, you can call into those routines using P/Invoke, just do something
like:

[DllImport (MonoTouch.Constants.SecurityLibrary)]
extern static IntPtr SecPolicyCreateSSL (int server, IntPtr
cfStringHostname);

Mono currently probes the security store on the iPhone, but does not support
storing anything there.   You can use this code as a starting point:

http://github.com/mono/mono/blob/master/mcs/class/System/System.Security.Cryptography.X509Certificates/OSX509Certificates.cs

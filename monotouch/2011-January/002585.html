On Jan 12, 2011, at 12:09 PM, David Soto wrote:
> How can i search on a 
> 
> ABMutableMultiValue<string> var??
> 
> Can i use linq on it?? And id possible how to do it??

ABMutableMultiValue<T> and ABMultiValue<T> implement IEnumerable<ABMultiValueEntry<T>>, so Yes, LINQ should work:

	ABMultiValue<string> names = ...;
	IEnumerable<ABMultiValueEntry<string>> homeNames = 
		names.Where (e => e.Label == ABLabel.Home);

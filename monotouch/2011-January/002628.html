It works for me, I tend to do this:

read:

using (NSUserDefaults defaults = NSUserDefaults.StandardUserDefaults)
{
  string expenseTypeMRU = defaults.StringForKey("expensemru");
}

write:

using (NSUserDefaults defaults = NSUserDefaults.StandardUserDefaults)
			{
				
				defaults.SetString(expenseTypeMRU, "expensemru");
				defaults.Synchronize();
				
			}

dont forget the synchonize after a write.

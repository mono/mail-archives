> Hello René,
>
> You can install different MonoTouch versions in separate directories, e.g.
>
> /Developer/MonoTouch-5.2.12/
> /Developer/MonoTouch-5.2.12-hoxfix/
> /Developer/MonoTouch-5.3.3/
>
> Then tell MonoDevelop which one to use with:
> - MonoDevelop menu
> - Preferences...
> - SDK Locations
> - MonoTouch SDK
>
> You might have to restart MonoDevelop for the change to be applied.
>

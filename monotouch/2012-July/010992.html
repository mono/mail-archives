you could pass the view controller into the delegate (or if you need
to reuse it, make it use an interface and implement that interface in
your viewcontroller). When you create the delegate, pass the VC you
want to use in.

Then you can just call the viewcontroller (or whatever it is you
passed in) and pass the image in. After that, you are in the context
of your VC.

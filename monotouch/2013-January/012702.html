I belive this error is because the dll you have is built against a full .net framework, and MonoTouch is a reduced set of it, so this class may not exist or may be in other namespace.

As I see, in the rabbitmq page you have the source code for this dll, so you can try to download the code, import it in a MonoTouch project and  adapt it to MonoTouch.

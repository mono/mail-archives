This is how you use NSAttributedString:

var str = new NSAttributedString ("MyString");
str.AddAttribute (CTStringAttributeKey.ForegroundColor,
UIColor.Red.CGColor, new NSRange (0, 2));

or:

var str = new NSAttributedString ("MyString", new CTStringAttributes () {
ForegroundColor = UIColor.Blue.CGColor });

I am trying to use coreplot monotouch bindings from github to plot a simple
scatter plot.  I want the x and y axis labels to always be in the lower left
corner of the graph.  Therefore, I believe I need to use either
IsFloatingAxis = true or to set the axisConstraints.  However, there are
issues with both methods:

1. IsFloatingAxis was removed before September 25, 2011 in the official ObjC
coreplot library.  If the monotouch bindings are based on the 0.9 release on
September 25, 2011, then they should not have a reference to IsFloatingAxis. 
And sure enough, if you try to use it, you get an ObjC unrecognized selector
runtime crash.

2. axisConstraints are commented out of the monotouch bindings so they are
unavailable.

What are my options?  Has some updated the bindings based on the latest ObjC
coreplot official library?

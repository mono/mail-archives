If you purchase an iOS developer membership from Apple, you can create a
certificate by opening XCode while your device is connected. XCode's devices
screen should contain a button to use the device for development. Click that
and a certificate will be created (Note: If you have MonoDevelop open, I've
found you need restart after doing that before it will recognize the
certificate). Now you should be able to deploy to your device.

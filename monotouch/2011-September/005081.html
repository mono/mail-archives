Thats a gotcha for new players (and old ones)

Anything in Cocoa which isn't NSMutableXXXX is NOT Mutable (it can't
be mutated - changed, altered etc)

And, generally, if there is a NSXXXX, there is an NSMutableXXXX as
well, if it makes sense to change it.

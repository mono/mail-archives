I have the exact same issue with 2.6 RC2 - had it with 2.4 too. MD 
always used to open projects in default view - now it always opens in 
solution view. It's not a huge issue to be honest - just a minor annoyance.

When the project is opened the view option is always set to "Solution" 
(from the View menu) - this view style does not show the solution 
explorer. Selecting "Default" from the view menu does show the solution 
explorer. Despite constantly changing the view style to Default each 
time a project is opened, MD never seems to remember this choice and 
always reverts back to Solution.

I have some screenshots if you need them.

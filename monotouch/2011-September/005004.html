oh, and BTW, from memory (sorry, bad pun), the simulator uses the
machine's ram, so you have a 4GB (well, unlimited GB's if you have a
big disk for swap) iPad with a quad core 2.5ghz CPU. Then you put it
on the device which has 512meg ram, ~1ghz cpu and _no_ swap.

... which is why I still have a 2nd gen ipod touch (same as a iphone
3g) as my base test device. it has only 128meg ram.... makes for some
fun design decisions.

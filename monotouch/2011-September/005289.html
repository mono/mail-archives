I have not been able to get around installing Xcode in order to get the simulator, but after that you can follow the instructions here: http://stackoverflow.com/questions/4411390/install-ios-app-into-xcode-simulator

Basically, you just find the entire application directory on the machine that does have the code, etc., zip it up and send it over to the machine that does not have the source code. They need to unzip it and put it in the same location and run the simulator.

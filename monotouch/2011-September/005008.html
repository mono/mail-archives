Hello,

Simply change:

public override CLLocationCoordinate2D Coordinate {
     get { return _coordinate; }
  }

.. to:

public override CLLocationCoordinate2D Coordinate {
     get { return _coordinate; }
     set { _coordinate = value; }
  }

Hi Nic,

You shouldn't need to remove the ~/Library/MonoTouch/License.v2 file (I'd
recommend not removing it, in fact ;-)

The only way I know of to uninstall MonoTouch (and I'm kinda new to MacOSX,
so there might be a better/cleaner way... but)

sudo rm -rf /Developer/MonoTouch
sudo pkgutil --forget `sudo pkgutil --pkgs / | grep monotouch`

The second line probably isn't necessary, but it does clean up the package
list :-)

Hope that helps,

Jeff

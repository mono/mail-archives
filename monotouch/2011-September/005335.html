 If the output is an .app file then the Xcode project is the wrong project type. It should be Xcode > File > New > New Project > iOS > Framework & Library > Cocoa Touch Static Library. You can use that project template to build yourself a static library. Just include all the files you need for the MGSplitViewController class and build.

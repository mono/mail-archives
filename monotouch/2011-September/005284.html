XCode 3.2.6 is officially declared obsolete by Apple for use with Lion, and
it won't install correctly using the normal installer. 

But, there is a way to install Xcode 3.2.6 on Lion, and still use
Monodevelop 2.6:
http://anatomicwax.tumblr.com/post/8064949186/installing-xcode-3-2-6-on-lion-redux

If you want to use Xcode 4/Interface Builder, you should use Monodevelop
2.8beta. Monodevelop 2.6 does not work with Xcode 4.1.

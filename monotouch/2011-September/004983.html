Derik,

I have seen similar before where code was being called after disposal. Try add this:

private bool disposed;
protected override void Dispose(bool disposing)
{
    If (disposing)
        this.disposed = true;
}

And then in your ShouldReturn, simply exit if the delegate is disposed.

Dino

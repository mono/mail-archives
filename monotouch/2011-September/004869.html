On Sep 2, 2011, at 2:28 PM, vasya khrenov wrote:
> 1) Do I need a Mac or can I use a Windows based or Linux based PC?

Yes, MonoTouch only runs on Mac OS X. MonoTouch depends upon Apple's iOS SDK, which doesn't run on Windows.

> 2) What editor do I need to use, Visual Studio or does one come with the software?

MonoDevelop is used.

> 3)If I buy the Mono for Android can I use it on a Mac and copy the code that I made for MonoTouch to the Mono for Android?

Mono for Android runs on both Mac OS X and Windows.

How much code you can share depends upon how you structure your code: many types and assemblies are available on both platforms (e.g. mscorlib.dll, System.dll). However, UI code is platform-specific -- monotouch.dll is specific to MonoTouch, while Mono.Android.dll is specific to Mono for Android. Consequently, any types that depend upon e.g. monotouch.dll won't be usable on Mono for Android (and vice versa).

Using design patterns such as Model-View-Controller can be used structure your code to increase what can be shared.

You need to go to the vici project website and download the monotouch 
version, that is Emit code free, as emitting is not supported by the 
Monotouch (no code generation and self code update is allowed on the 
appstore, that is why).

Same goes for newtonsofts JSON, you can download it in source code 
version and create your own Monotouch library project and compile it 
yourself.

BTW this is a good practise as almost any lib can be therefore 
transformed to monotouch to use. I did that with Farseer and XNA and 
have now full Box2D .NET port working in my apps for physics. It is a 
bit more work to tranasfer the libs, on the other hand you learn a lot 
about what's under the hood.

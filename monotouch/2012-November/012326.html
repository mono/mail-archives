Echoing Jason, I'd recommend either Subversion (self hosted or via
somewhere like Beanstalk - http://beanstalkapp.com/) or git (via
Bitbucket or GitHub)

Git takes a little learning, but SO SO worth it. I use BitBucket for
my personal stuff, as I tend to have lots of repo's, and thats
expensive on github - well, for 1 person it is. BB is free for 5
users. If this is a commercial venture, start with github and move if
it doesn't work for you.

And if you come from a VSS background: Trust The Merge. VSS was always
awful at it, so I'd avoid it as much as possible. Git is awsome at it
(SVN is "ok" at it), so you can work in a totally different way.

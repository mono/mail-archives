> I agree in that I doubt it had to do with the update. The update did

The update does not remove ARMv6 support. If you were seeing ARMv6
that's because you were building with a SDK earlier than 6.0.

IOW if you use SDK 6.0 then you'll see ARMv7 and ARMv7s (ARMv6 will be
hidden). For older SDK you'll see ARMv6 and ARMv7 (ARMv7s) will be
hidden.

This behavior is recent (6.x series) but not new to the 6.0.7 update
(in fact the UI logic is inside MonoDevelop's addin, not MonoTouch).

To default to the iPad simulator for debugging, change the project setting Target devices option under iPhone Application to iPad only.
Just to run it on the iPad simulator, select Run -> Run With... -> iPad Simulator 4.x.

Remember to set the Target devices back to iPhone and iPad before deploying.

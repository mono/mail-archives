That is down to your file system. It's possible to initialise HFS to a number of flavours, but by default, Apple ships with Case insensitive and non journaled. Some apps assume this and fail to run on case sensitive file systems (Autodesk's Sketchbook free app in Mac AppStore initially failed to run for this reason.) The simulator is just using what it sees.

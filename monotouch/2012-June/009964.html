Thank you for the quick reply!
We proceed as follows:

We create an exe on monotouch.
Then we use the following command from the terminal:
"/Developer/MonoTouch/usr/bin/mtouch
--xcode=/Users/pos/Projects/iOSCompactController/XcodeExport
/Users/pos/Projects/iOSCompactController/iOSCompactController/bin/iPhone/Release/iOSCompactController.exe
--sdk="5.1" --fat --armv7"

Then there are some dll.7.s files and a main file in the XcodeExport folder.

These we copy and paste it into our GUI XCode project.

In the Main from MonoTouch, under the invocation of mono_jit_exec we add a
call to start the objective c GUI. (UIApplicationMain (....))

Another very important information that I have forgotten: This procedure has
worked with the IOS version 4.3.5 without problems. Since the update to 5.1,
the app crashes after the install with ITunes.

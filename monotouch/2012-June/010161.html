Hi,

Which MonoTouch version are you using? This looks like a pretty old issue
where we'd upload stale assemblies in addition to the correct ones.

If you're not using the latest MonoTouch, you might consider upgrading.

Otherwise you can try deleting your app from the device / resetting the
simulator contents, delete the bin and obj files in your project and try
again.

Rolf

Hello,

Try to pinpoint where the issue is. The NRE is a consequence, not the
real issue.

E.g. Add a breakpoint on

string clientcode=txtClientCode.Text;

then look if the file exists (easy, at least on the simulator) and is
non-empty. That will tell you if you were able to serialize correctly
your data.

If it file exists then look at the data to see if your XPath query
makes sense in case the data does not match what you expect (which
would return you a null value).

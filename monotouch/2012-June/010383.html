Have a look at the ATMHud stuff, which is in the monotouch-bindings
project on github. That wraps a nice HUD which you can use to block
the user interacting with the app until it's all finished.

Also, that method takes a completion block - ie, code to call when
it's all done - so you could throw up the HUD, call the save, and in
the completion block, hide the hud and keep going.

https://github.com/mono/monotouch-bindings

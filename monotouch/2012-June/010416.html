A LUT is a table that helps to translate values to other values. For example if you are getting 6-8 bits of data but want to have 16 bits of data you apply a LUT. It could be a linear one and just multiply it to get the values. But the strength is where you have a curve in the LUT so that part of it is adjusted differently than other parts. For example values 0-256 might get more adjustment than 257-512.

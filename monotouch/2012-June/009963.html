Have in mind that you can't add a MonoTouch app as a library to an
Xcode project - you need to do it the other way if you want to use
both native and managed code: you have to create native libraries
using Xcode and then access those libraries from MonoTouch.

Here is the relevant documentation from our web site:
http://docs.xamarin.com/ios/advanced_topics/binding_objective-c_types

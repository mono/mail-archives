Hi again

Followup! OK. I have solved a problem.

Bunch of leads/links compiled together read twice forward and backward 
and  the devil is in the detail...

Most of the stuff can be done according to:
http://docs.xamarin.com/ios/Guides/Advanced_Topics/Native_Interop

Only in the part for Linking:
http://docs.xamarin.com/ios/Guides/Advanced_Topics/Native_Interop#Linking_Your_Library

it is important to add that configuration of mtouch tool for linking 
with extra arguments for gcc

[-cxx] -gcc_flags "-L${ProjectDir} -lMylibrary -lSystemLibrary 
-framework CFNetwork -force_load ${ProjectDir}/libMyLibrary.a"

[-cxx] optional - for c++ only

******* makes exported symbols internal **************, so DllImport 
should look like:

[DllImport("__Internal")]

happy hollidays

Mel

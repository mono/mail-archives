This can happen when there's a mismatch between the libraries used to build
your application and the one used to deploy it (e.g. using the debugger). Is
it possible that the solution was not cleaned/rebuilt after the update ?

If a rebuild does not solve this can you open a bug report and attach the
full build log ?

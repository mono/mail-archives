Xcode 4.3 is available from the App Store, I got it there yesterday. They
also hadn't updated developer.apple.com at the time, so the latest version
available there was still one of the preview releases. Note that we do not
officially support the preview releases of Xcode with MonoTouch 5.2.5 as
there were last minute changes to the Xcode package which are incompatible
with the way the preview releases worked. If it works that's great, but if
it doesn't just ensure you update to Xcode 4.3 final release.

Alan

Hi John,

Actually ConsoleColor exists to support colors in consoles (most commonly
cmd.exe in Windows, any terminal such as xterm in Linux and Terminal in
MacOSX).

The ConsoleColor enum shouldn't exist in MonoTouch (if you check the
Console class, you'll find that there is no way to set the
 foreground/background colors), the only reason it's there is because
nobody noticed before.

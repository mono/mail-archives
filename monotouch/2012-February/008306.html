I have class which overrides RowSelect edfrom a couple of levels deep from
UITableViewSource.

So it's:

class aSource: UITableViewSource
{
   public override void RowSelected (UITableView tbl, NSIndexPath ind )
   {
   Console.Write("aaa"); // breakpoint a
   }
}

class bSource<T> : aSource where T : someT, new ()
{
  public override void RowSelected (UITableView tbl, NSIndexPath ind )
   {
   Console.Write("bbb"); // breakpoint b
   base.RowSelected(tbl, ind);
   }
}

If I put the two break points in the debugger the bSource.RowSelect is
actioned when my table cell is selected and it prints "bbb" but the break
point doesn't work, the "aaa" prints and the breakpoint works.

Strange? Any ideas?

Mojo,

I've done a few different apps that push a view onto the stack that has a UITabBar that I then end up managing the views for the tabs myself.

As for handling the "More" button that just needs to display a table view with your other views.

To display different view controllers on the tabs just handle the ItemSelected event on the tab bar.  You can then remove the current view from the superview and add the current view as a subview.

Hello Everyone - 

 

I wanted to make the community aware of Professional Cross-Platform Mobile
Development in C#, a new book from Wrox that will be released next week.  It
discusses how the business logic and data integration layers can be
separated from the presentation layer of your application, so that you have
code sharing not only across platforms via MonoTouch, Mono for Android, and
Windows Phone 7, but also across architectures with your shared code running
on the server powering a presentation layer of HTML5/JavaScript/CSS.

 

It was written by Scott Olson, John Hunter, Ben Horgen, and Kenny Goers of
ITR Mobility, and shares a lot of our enterprise cross-platform mobile
development experience.  Hopefully the content will be valuable to many
others in the community.

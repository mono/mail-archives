Hey Mark,

Out of curiosity, could you send over the Info.plist file that gets created
in the build that gets sent over to Apple? If you could send that to Jeff
and myself (at chrisntr at xamarin.com) then that would be wonderful. Could you
possibly tell us what Application Loader version you're using too?

You can use a hotfixed MonoDevelop here -
http://files.xamarin.com/~alan/MonoDevelop-debug.dmg - to re-compile that
will fix this issue. It's not a full version of MonoDevelop so it's worth
installing this under a different name to the main MonoDevelop so that you
can install both side by side.

Hope this helps,

Chris Hardy

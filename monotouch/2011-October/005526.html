Hi Matt,

MonoDevelop 2.8 is meant to be used with Xcode4 (which changed the way the
interface builder works) and not with Xcode 3.2.6 anymore.

Xcode4 should still be able to load your Xcode3 xibs, though.

If you still have problems connecting outlets and actions with MonoDevelop
2.8 and Xcode 4, let us know.

Hope that helps,

Jeff

I really do like the concept of Mono Tools for Visual Studio and would be
happy to pay for it if I could use it for development with MonoTouch and
MonoMac. Note, I'm not trying to avoid the need for a Mac, I'd just rather
code all day in Visual Studio (I really need my ReSharper, CodeMap,
PowerTools, and so on). MonoDevelop is nice, but just not up to snuff with
VS and especially it's 3rd party plugins.

Is there any ongoing development of Mono Tools for VS?  Is there anything I
can do to help?

My process right now is:
1) develop on VS using Jonas Follesø's great 
https://github.com/follesoe/VSMonoTouch plugin  for VS2010 to open and edit
MonoTouch projects
2) push my change to Mercurial
3) pull changes on Mac
4) launch project on Mac in MonoDevelop
5) debug
6) edit back on PC as needed, then go to step 1 and repeat

This is fairly laborious and MonoTools for VS for MonoTouch would really
make that process much smoother. I'd just like to know if we might see this
happen in the near future. If not, then I may hack my own solution.

Thanks,
Brad

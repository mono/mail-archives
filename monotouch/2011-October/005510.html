Thanx for the update. But unfortunately, same result with your updated

This is what happens:

1. When the program starts the first time, the result of
RedLaserSDK.CheckReadyStatus() is RLState_LicensedModeReady.
2. When it starts scanning it shows the "Unregistered SDK Limit Reached"
message box.
3. When the program is restarted (or when I call CheckReadyStatus() again in

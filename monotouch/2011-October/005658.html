Remember that the underlying API's are the same. So MT is CocoaTouch +
.NET framework + C# (in descending order of size). XCODE etc is
CocoaTouch + Cocoa + Obj-C, so if you learn CocoaTouch, with either
tool, it's the same in either case.And it's a huge API.

So going with MT for the moment, and switching later if you have to,
is a valid option. So is going the other way.

You could use the simulator etc (which is free) until you work it out
a bit more....

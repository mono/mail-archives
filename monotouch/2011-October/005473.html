I have created the monotouch bindings for the RedLaser SDK v3.2.0 and build
the sample MonoTouch app from Chris Branson (
https://github.com/chrisbranson/RedLaserSample
https://github.com/chrisbranson/RedLaserSample ). Barcode scanning worked
fine.

After the 25 barcodes evaluation limit, I bought the Starter Package and
downloaded the RedLaser_License.xml file. Barcode scanning is working fine
with this license with the xcode example from RedLaser. But it doesn't work
with the sample Monotouch app (Unregistered SDK Limit Reached message).

I checked everything I could think about:
•	The device id is correct
•	The BundleId of the app is the same as in the RedLaser_License.xml file
•	The RedLaser_License.xml file in the .app file is binary identical to the
one downloaded from my RedLaser developer dashboard.
•	The RedLaser_License.xml file is in the root of the .app file (I included
it as Content in the root of the project)

Does anybody has this working and/or know what can be wrong?

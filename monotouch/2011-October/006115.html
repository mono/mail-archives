I forgot one important detail regarding the background color in the header
view: the view itself cannot have a background color: instead, the
individual labels that comprise the header view "columns" need to have their
background set to the desired color. 

In my implementation, the View's Opaque property is set to false and no
background color is applied while the label's Opaque property is set to true
and a background color is set.

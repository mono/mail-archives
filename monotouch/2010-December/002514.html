Right click on your project, choose options. Then iPhone Build on the left. Then in the Extra Arguments field add "-v -v -v" that'll turn on super verbose mode for the build.  (make sure you've got the right configuration set above – e.g. Debug iPhoneSim or whatever you're building for).

Then, in the error output, click the Build output button and you should see the build output.

>From there, you should either see an error you can diagnose, or one you can't, in which case you should copy and pasta here.

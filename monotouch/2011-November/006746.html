I'm not sure if this will help you, but I was getting crashes that only showed up on the device, as well.

Adding the following lines to the iPhone Build - "Additional mtouch arguments" section, fixed my problem:


-aot "ntrampolines=2048" -aot "nrgctx-trampolines=2048" -aot "nimt-trampolines=512"

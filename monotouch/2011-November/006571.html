Typical - I post and then it clicks seconds later. I'm now using the
following:

		public static void BeginImageContext (SizeF size)
		{
			if (IsRetinaDisplay)
				UIGraphics.BeginImageContextWithOptions (size, false, 0);
			else
				UIGraphics.BeginImageContext (size);
		}

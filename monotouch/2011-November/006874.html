Hi René,

It's not easy to have a general rule for differences between iOS versions
or differences between SDK versions. The logical assumption is that
installing an app built with 4.3 on an iOS5 device should still work - but
there are of course exceptions to this assumption (which can be both bugs
in iOS5, MonoTouch or actually documented behavior). This means that an app
built with the 4.3 SDK will generally work identically on iOS 4.3 and 5.

HoweverI find that the easiest way to figure it out is to try it yourself:
install the 4.3 SDK and build with it to see what happens. My guess is that
you'll see the same behavior on both iOS5 and 4.3 devices, but it's still
just a guess..

I hope this helps,
Rolf

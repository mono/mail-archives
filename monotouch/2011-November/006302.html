If your program's target is iOS 5.0, then yes, it is possible. You would override ShouldHideViewController and return false for the left hand view controller. I found that I had to set the WeakDelegate in my UISplitViewController derived class in order to get ShouldHideViewController to be called.

If you can't do 5.0 then MTSplitViewController is probably your best alternative, although the author says it has some resizing bugs.

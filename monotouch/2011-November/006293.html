Yup, I'm told the GC in MT5 is a bit more agressive, so this isn't going to
work anymore:

using (var alert = UIAlertView(etc))
{
  //hook up the alert view
  alert.Show();
}

just move the decl of alert outside of the method (ie, at class level)

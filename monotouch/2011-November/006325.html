If I use the following code in my a ViewDidLoad event to add two navigation
buttons to the NavigationItem object, all works as expected:

*UIBarButtonItem btnPreviousPoint = new UIBarButtonItem ("<",
UIBarButtonItemStyle.Plain, btnPreviousPoint_Click);
UIBarButtonItem btnNextPoint = new UIBarButtonItem (">",
UIBarButtonItemStyle.Plain, btnNextPoint_Click);

NavigationItem.RightBarButtonItems = new UIBarButtonItem[] {btnNextPoint,
btnPreviousPoint};*

However, I find that if I add the buttons to the NavigationItem using the
following code, the app completely crashes randomly (in both the simulator
and the iPhone) when clicking on the buttons:

*NavigationItem.SetRightBarButtonItems (new UIBarButtonItem[] {btnNextPoint,
btnPreviousPoint}, true);*

Am I doing something wrong? Any ideas as to why it's crashing with the
second approach?

Secondly, is it possible to set the size for the UIBarButtons?

in? Currently, the buttons fade in instead!?

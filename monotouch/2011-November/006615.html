Betreff: [MonoTouch] Detecting if an app has been updated on a device

I need to be able to determine whether or not my app has been updated on a
device (i.e. v1.0 was installed previously, but the user is now running
v1.1).

The SQLite database I use is built into the app as content, as it's read
only and will be updated with each new release of the app, so I can't use
that.

I'm thinking of dropping a small text/xml file in the personal folder, that
contains the version number of the app the last time it ran. When the app
runs, it checks this against the version running and will perform any
necessary update tasks. It will then update the text/xml file to the reflect
current version.

Or is there a more "iOS" way of achieving this?

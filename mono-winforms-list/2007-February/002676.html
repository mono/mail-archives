On Tuesday 06 February 2007 11:15, Andrus wrote:
> Any idea how make it to work with MONO ?

If you haven't done so already download the Mono Migration Analyzer
(http://www.mono-project.com/Moma) and run it against the RDLDesigner. The 
report should then be able to give you a clue where the app should be 
rewritten to achieve a greater portability to mono.

For the two missing methods: You could check if they are already in SVN or if 
someone is trying to implement them. If nobody is implementing them, then you 
could try to implement them yourself and then send a patch to this list.

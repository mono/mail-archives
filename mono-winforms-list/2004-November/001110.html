Hi all,

Just a heads up that if you are building gdi+ from cvs, you need to update 
to the cvs HEAD version of Cairo as well. We made some speed improvements to 
gdi+ that seem to expose a bug in older versions of Cairo. (The HEAD version 
of Cairo also requires the latest libpixman, which can be pulled from the 
same repository as Cairo itself)

Wayne.Arenz@gd-ais.com wrote:
 > How does one fix the overwrite if one did it :^)
 >

- rpm -e mono-wine (and rpm -e mono-wine-devel)
- rpm -e wine (and rpm -e wine-devel)
- rpm -e winelib
- Remove all the *32.dll.so wine libs in /usr/lib and remove 
/usr/lib/wine (I'm assuming that <prefix> is /usr, might be /usr/local 
for wine and/or mono)

That should get things clean, then you can install wine, wine-devel and 
winelib again.

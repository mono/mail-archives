You should either use WinForms or GTK#, not both.
If you want to use WinForms, then you should draw using GDI+, the same way you would in .NET.
If you want to use GTK#, then you should draw using GDK (http://docs.go-mono.com/?link=T%3aGdk.Pixbuf).

There is a separate cvs project, 'winforms', that holds all winforms
samples. The goal of test apps is to test every aspect of a particular
control, if possible even automated, to some point. Also, the test app
should, when used, indicate to the user what results are to be expected, so
the tester can verify that these results are indeed happening.

Feel free to send me your code, I can commit it to 'winforms' for you.

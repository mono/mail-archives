> LD_LIBRARY_PATH is not set in FC3 by default is it part of any

It is and isn't. It is as it points to /usr/local/lib;/usr/lib and isn't
as if you type echo $LD_LIBRARY_PATH it shows nothing.

Unless you've set the install to be somewhere odd (such as /opt) then
you don't need to do anything to LD_LIBRARY_PATH.

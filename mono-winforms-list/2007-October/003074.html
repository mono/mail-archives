Hi all,

Last night we just start to use the field "whiteboard" to define the
control related to bug, in some cases we can use special values like:
MDI, Keyboard and Strip (any strip control). In others we just use
control name like: Control, ComboBox, ToolBar... With this values we
can have a better view of MWF bug list and sort or group by control.

Also it is a good time to start using field OS instead of [Win32/X11]
in summary, OS can be showed as report column and can be sorted and
grouped.

On next days we will try to publish an clear rules about the use of
"severity" and "priority".

If you are a MWF contributor or reporter please try to use OS and
whiteboard fields and use change columns options to select this fields
in your default report view.

Here you can found a sample report with my bugs:
https://bugzilla.novell.com/buglist.cgi?bug_status=NEW&bug_status=ASSIGNED&bug_status=REOPENED&bug_status=NEEDINFO&email1=ecanuto%40novell.com&emailtype1=exact&emailassigned_to1=1&emailinfoprovider1=1

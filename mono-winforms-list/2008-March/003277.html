Asunto: Re: [Mono-winforms-list] .NET 3.0

It is probably a long ways off (many years), if ever.  I don't think 
we've decided if we will be doing WPF.

Whereas winforms has a large installed base that we want to help move to 
linux, WPF does not seem to have gathered much developer support.  It 
would be a very large effort to write WPF for Mono, and if the payoff is 
low, we probably won't do it.

However, the rest of 3.0 (WCF, Infocard, and WF) already have partial 
implementations for Mono, and will be completed, though I am unaware of 
the timeframe.  (see http://www.mono-project.com/Olive)

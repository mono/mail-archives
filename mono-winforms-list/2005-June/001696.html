Hi,

The monoupdater script has been updated to version 0.9c

http://www.all-the-johnsons.co.uk/mono/monoupdater.sh

Hopefully, I've vanquished the bugs that appeared in 0.9b and as a thank
you for the bug reports, I've added more functionality to it.

The script now allows you to do the following

monoupdater.sh <command> 

clean -> runs make clean before compiling
rebuild -> runs make distclean before compiling
build -> builds only (doesn't checkout from svn)
noldcon -> doesn't run the second ldconfig
help -> um...
-> checkout/update sources and builds

The script is released under the GPL vsn 2.

>B. you can code up the OnPaint function to paint the correct ONE of your
images.  Then you can make a Timer object on your form.  When the timer
fires, you can call the static Graphics.FromHwnd to get a graphics object
and directly >paint the newer image.  Also update a member variable with the
current image so that if your window is covered and uncovered it will paint
the correct thing.

Instead of directly drawing the image, in this case use Invalidate
(ImageRectangle) which will cause OnPaint to be called, thereby avoiding
Graphics.FromHwnd.

How to compile the latest snapshot in a few easy steps (assuming you've
never done it before, but have had mono installed from rpm and you've
hosed it).

1. Grab mono from svn (head version)
2. cd mono
3. ./autogen.sh --prefix=3D/usr
4. make; su;
5. make install

IMO, the instructions on the mono website are not very good for building
mono, hence why I have my page on doing it which seems (by looking at
the logs) very popular.

Stefan,

Others have already responded, but I'll take the chance and list the 
official URLs:

Mono Project Homepage, WinForms section:
http://www.mono-project.com/contributing/winforms.html

MWF Weblog, tracking daily progress:
http://svn.myrealbox.com/blog/

MWF Control and Dialog implementors and assignment status:
http://svn.myrealbox.com/mwf/owners.html

MWF class status:
http://svn.myrealbox.com/mwf/class-status-System.Windows.Forms.html

MWF overview page: Currently almost the same as the page on the Mono 
homepage, but will grow into more support/help oriented stuff over time:
http://svn.myrealbox.com/mwf/index.html

There are no packages or RPMs out there right now that contain the managed 
SWF version, you need to build from source. See the above links.

Cheers,
  Peter

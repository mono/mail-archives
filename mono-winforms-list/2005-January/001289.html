Hello,
you're trying to use an old implementation of winforms which used
wine. The new implementation do not use it anymore. You're probably
using it because there was no new release since Managed.Windows.Forms
was made the default, you should compile mono from SVN, you can find
more information about it on the website.

Hope that helps.

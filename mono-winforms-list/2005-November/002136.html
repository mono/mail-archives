Hello,

the attached patch for X11DesktopColors adds support for KDE colors.

It reads the color settings from ".kde/share/config/kdeglobals" file in
the users home dir.

Default Desktop support is still Gtk. If it finds the environment
variable "DESKTOP_SESSION" with value "KDE" or the environment variable
"KDE_FULL_SESSION" then "desktop" is set to Desktop.KDE.

If nobody complains I'll check it in...

Cheers

Hi.

Some explanation. System.Windows.Forms-winelib is broken due to some
structural changes atm. If you have cvs and want to play around with
swf, try swf-gtk. It is in mcs/class/System.Windows.Forms/Gtk

It can already be used for quite a few controls. It is in a better state
than swf-winelib, but will not have the completeness of swf-winelib in
the long run. But give swf-gtk a try if need to play around with swf. 

But if you want to create gui apps, you should be using Gtk#, not swf.

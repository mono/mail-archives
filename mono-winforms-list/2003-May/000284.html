Colleagues,

I am happy to announce that the patches to wine required to build shared
libraries for mono-winforms development under GNU/Linux (or whatever you
like calling it) are now available at http://www.openlinksw.com/mono

When the patches are applied, you can use the source to produce both
standard wine (make wine) or the shared objects (make wine-shared.) Both
targets can live harmoniously together in same environment so you do not
need to break/downgrade an existing Wine installation to use the mono
patches.

A hack to initialize the mono runtime is also available and included in
the patch tarball.

Please see the web page and READMEs in the tarball for further
instructions. 

I'd like to take this opportunity to thank my distinguished colleague
Vladimir Kaluzhny for his efforts. Now we can use Linux as the
development platform like it should be.

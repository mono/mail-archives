Hi,

=20

I have read many discussions about implementing SWF with Wine or GTK #. =
Dotgnu.org is implementing SWF using X11.=20

=20

It seems to be a better option than Wine/GTK# because:

=20

1.	Does not require additional library files from Wine/GTK#

=20

2.	Runs on Mac OSX, Linux, Windows, etc. whereas at present neither Wine =
nor GTK# runs on Mac OSX.

=20

I wonder why implementing SWF using X11 has not been discussed in the =
mono project. Or did I miss those messages? :-)

=20

Akhilesh Agarwal

The only way I know to make this work is to use a browser component.
There may be an easier way as the method I used was a little
convoluted.  I started by using SWT through IKVM to create the window,
and the SWT Browser component.  The SWT browser on linux ended up
depending on xulrunner, so I had to make sure that xulrunner was setup
correctly to use the flashplayer plugin (possibly using
nspluginwrapper if you are on a 64bit system).

I know that the winforms webbrowser control has recently been
receiving some great improvements, so that may be an easier route (at
least on linux, I don't think it works on osx yet).

Another alternative would be GTK, and either the Gecko control, or the
WebKit control.

I haven't tried anything other than the SWT route, so you may have to
dig further on the other alternatives to get anywhere. HTH.

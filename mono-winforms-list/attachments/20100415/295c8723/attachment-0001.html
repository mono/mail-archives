Use&nbsp;myBrowser.DocumentText<BR>
&nbsp;<BR>
but you'll also need to put the full HTML:<BR>
&nbsp;<BR>
string value = "this is a test";<BR>
myBrowser.DocumentText = &nbsp;"&lt;html&gt;&lt;body&gt;" + value + "&lt;/body&gt;&lt;/html&gt;";<BR>
&nbsp;<BR>
Then you can amend the CSS etc.. as well.<BR>
&nbsp;<BR>
use myBrowser.BodyHtml to retrieve the inner html if you have it in edit mode.<BR>

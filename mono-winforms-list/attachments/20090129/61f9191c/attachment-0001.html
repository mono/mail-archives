If mysqldupm.exe is a managed exe you need to call mono passing it as parameter (exe files aren&#39;t executable files in Linux), you can also create a shell script that call mono passing the exefile, and use the script name in your Process call.<br>
<br>If mysqldump.exe is a native windows program it can&#39;t be executed in linux, but you can see if there is some equivalent program for linux and call it instead (it might be called &quot;mysqldump&quot; without the &quot;.exe&quot; part for instance).<br>

Well, three years ago I tried MWF on Solaris. It worked when using it
through X11 forwarding with an X11 server running on Windows (which does not
help MWF in anyway). However the X11 server shipped with Solaris was unable
to make MWF work. At lot of things might have changed since, but maybe you
could inspect your X11 server. Also I guess that it should be a lot easier
with Nexenta (Debian packages on Solaris kernel), but I have never tested
that.

Just my 2 cents.

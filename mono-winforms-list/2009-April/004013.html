Ubuntu seems to have messed up.
The dotnet.pc file references some dll's that were not installed

I fixed these errors by installing these packages:
libmono-microsoft8.0-cil
libmono-oracle2.0-cil
libmono-ldap2.0-cil
libmono-ldap1.0-cil
libmono-system-ldap2.0-cil
libmono-cscompmgd7.0-cil

You could also edit the dotnet.pc file but you might get into trouble later
and then you don't remember that you edited that file.

Hi All,

The merge is complete. You can find the latest libgdiplus at 
svn://svn.myrealbox.com/source/trunk/libgdiplus. Those who already have a 
checked-out copy can simply 'svn up' to get the latest version.

The two major improvements are:
* Performance. Cairo has gotten pretty significant performance improvements, 
some quick tests show that basic rendering speed has improved more than 30% 
and image blitting has gotten over 50% faster.

* Leaks: We've fixed many places where libgdiplus leaked memory or cairo 
objects.


Known issues:
* Image color matrix on big-endian systems (Mac). There is still a 
byte-swapping issue when applying a color matrix to an image.


Cheers,
  Peter

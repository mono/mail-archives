Yes, Winforms on OSX uses the native driver by default since 1.9. 
However, the native driver does not mean that you app will look like OSX 
or use OSX widgets.

The old non-native driver required you to install X11 and run your 
application using X11.  The native driver uses native OSX libraries and 
does not require you to run X11.

Hopefully some point in the future we will have a more native 'look' for 
winforms, on both Linux and OSX.

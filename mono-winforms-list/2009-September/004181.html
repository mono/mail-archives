It appears you are relying on the TLP.  If you were setting the column 
explicitly, it would look roughly like this:

TableLayoutPanel tlp = new TableLayoutPanel ();
Label l1 = new Label ();
Label l2 = new Label ();

tlp.SetColumn (l1, 1);
tlp.SetRow (l1, 1);

tlp.SetColumn (l2, 2);
tlp.SetRow (l2, 1);

The status page says that DataVisualization is missing 4,439 members.
You have to drill down to see that basically no part of it is implemented.

If you are scanning with the latest MoMA (2.8), there should be "Test Against Mono Version: Mono 2.8 (4.0 Profile)".  Mono did not support 4.0 before version 2.8, so earlier versions will not report on anything missing for 4.0.

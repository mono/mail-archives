The only way to capture mouse left button down events for sliders is to add a
handler as follows:

SliderControl.AddHandler(MouseLeftButtonDownEvent, new
MouseButtonEventHandler(SliderControl_MouseLeftButtonDown), true);

This works with Silverlight 4.0 but not with Moonlight 3.99.0.1.


Reproducible: Always

Steps to Reproduce:
1. Create a slider
2. Attached a handler using the code shown above
3. Add a MessageBox.Show call inside the handler
4. Run the code and click on the slider - no messagebox is shown under
Moonlight but it is for Silverlight.
Actual Results:  
No handler call.

Expected Results:  
The handler should be called. This allows application code to tell apart drags
from clicks on the slider bar.

Adding 5000 PathFigure objects to a Path and then displaying them on a canvas
takes 500ms with Silverlight 4.0 under Windows 7 64-bit but takes 8,300ms under
Moonlight 3.99.0.1 and Ubuntu 10.10 32-bit.

This is about 17 times slower.

For reference Moonlight 3.99.0.1 and Ubuntu 10.10 32-bit under VMWare Player
with Windows 7 as the host takes 13,000 milliseconds.


Reproducible: Always

Steps to Reproduce:
1. Unpack and build the attached Visual Studio 2010 project
2. Click on the Refresh button to redraw the Path and observe the timing
information.

Live demo at: http://www.hullblitz.org/testmap.html
Actual Results:  
Moonlight 3.99.0.1, Ubuntu 10.10 32-bit = 8300ms
Silverlight 4.0, Windows 7 64-bit = 500ms

This is the time to execute the Render() function.

Expected Results:  
Moonlight should have a similar level of performance to Silverlight. Using Path
objects is a high performance and flexible way of drawing to the screen.

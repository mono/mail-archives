A simple app has a textbox and a button. When the button is clicked, an attempt
is made to get the textbox selected text. However, it seems that the selected
text is reset to empty when the button is clicked and the textbox loses focus.
Sample code is attached. This is with Moonlight 3.99.02.

Reproducible: Always

Steps to Reproduce:
1. Using the attached code, start the app.
2. Select text in the text box (e.g. using Ctrl-A).
3. Click the button. A message box should appear showing the selected text.
Actual Results:  
The message box always reports an empty string with Moonlight 3.99.02.

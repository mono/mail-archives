Description of Problem:
The ITV Player does not recognise a properly installed Moonlight plugin, and
will not play content.

Steps to reproduce the problem:
1. Go to www.itv.com/itvplayer
2. Select a programme to view
3. Click on the "Install Microsoft Silverlight" button to go to
http://go-mono.com/moonlight/Default.aspx
4. Install Moonlight, so that http://go-mono.com/moonlight/Default.aspx says
"Congratulations, you're running the current release of moonlight!"
5. Go back to www.itv.com/itvplayer
6. Select a programme to view
7. Notice that the "Install Microsoft Silverlight" button still appears, and no
content plays - Moonlight is not recognised.

Actual Results:
No content played in ITVplayer

Expected Results:
All content properly rendered by ITVplayer

How often does this happen? 
Every time - 100%

Additional Information:
I have reproduced this problem in Debian Lenny on two 32-bit Intel x86 machines
and a 64-bit AMD64 machine, using Firefox 3.0.6

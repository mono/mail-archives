Serialize method returns a System.NotSupportedException when passing a
XmlWriter attached to a XDocument.

Reproducible: Always

Steps to Reproduce:
The following snippet shows the code that throws the exception:

XmlSerializer oSerializer = new XmlSerializer(objType);

XmlWriterSettings ws = new XmlWriterSettings();
ws.OmitXmlDeclaration = true;

#if SILVERLIGHT
   ws.NamespaceHandling = NamespaceHandling.OmitDuplicates;
#endif

if (ns == null)
  ns = new XmlSerializerNamespaces();
ns.Add("", "");

XDocument xDoc = new XDocument();
XmlWriter oWriter = XmlWriter.Create(xDoc.CreateWriter(), ws);

oSerializer.Serialize(oWriter, obj, ns);
oWriter.Close();
Actual Results:  
System.NotSupportedException

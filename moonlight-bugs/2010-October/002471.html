Application will blow up into pieces when executing the following code:
            var _wAppMsg = new RadWindow();

            _wAppMsg.WindowStartupLocation =
WindowStartupLocation.CenterScreen;

            _wAppMsg.Header = "Header";

            _wAppMsg.Content = new ErrorMessage();

            System.Windows.MessageBox.Show("About to ShowDialog");

            _wAppMsg.ShowDialog();

            System.Windows.MessageBox.Show("Shown dialog");


If _wAppMsg.WindowStartupLocation = WindowStartupLocation.CenterScreen;, then
the application will only crash when closing the popup.

Reproducible: Always

Steps to Reproduce:
1. Compile attached application (you will need Telerik RAD Controls for
Silverlight 2)
2. Run and press the button.

Actual Results:  
Firefox blown to pieces.

Expected Results:  
The popup window to be displayed.

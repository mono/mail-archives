Description of Problem:


Steps to reproduce the problem:
1. Use git version 7795670c36b15ce9e60b783d2696b7fe85de0bc4 to build a
moonlight plugin.
2. make test-plugin to install the plugin
3. Follow instruction on
http://code.google.com/p/chromium/wiki/LinuxBuildInstructions to build a linux
GTK version of chromium browser
4. Verify the plugin is enabled in about:plugin
5. Visit www.silverlight.net 

Actual Results:
 The moonlight plugin process crash and xmessage popup. Check the additional
information section.

Expected Results:
 The page is loaded correctly.

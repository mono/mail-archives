Now here's an odd one.

Using Moonlight 1.0b1, visiting the following SL1 site:
http://silverlight.net/samples/1.0/Sprawl/default.html

All other letters work fine, but not "D". Odd.

Jackson Harper said on IRC that it seemed to work on MS SL on Mac OS for him,
but not Moon.

Reproducible: Always

Steps to Reproduce:
1. Visit http://silverlight.net/samples/1.0/Sprawl/default.html with Moon
2. Try to enter a name like, say, "directhex"

Actual Results:  
"irecthex"

Expected Results:  
"directhex"

Test 143 in the ms harness is failing and after digging into it we realized
that the problem was that mozilla was compressing motion events and because the
logic it is using is flawed it not only compresses but reorders and removes
motion events.  This breaks the test and is also potentially a source of other
mouse related bugs.

This is a tracker bug that will be updated with details as we work on a mozilla
patch etc.

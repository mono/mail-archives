Description of Problem:

I am using:
Moonlight build 1.99.1.99 Revision number 140549

Here is another of our Silverlight 2 applications: 
Http://www.propserv.otago.ac.nz/imap/ 
This is an interactive online campus map application.

When i goto the above url i see the loading "Splash" screen i have setup, then
when it is complete nothing is shown. See the error below

When it loads it should display a blue border with the university logo and a
menu at the top left, and the right should contain an outline of New Zealand.
Above the map is a title and a status message as to whether you are signed-in
or are a guest.

It appears Moonlight is haveing an issue with the the ImapSignup control which
inherits from SignUpDialog in a seperate referenced project.

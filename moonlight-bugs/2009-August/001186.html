Description of Problem:

The canadian CTV.ca TV has developed a hulu-like site and streams lots of TV
series and such. 

The URL is: http://watch.ctv.ca

The site's media player is built on Silverlight. Sadly it doesn't work with
moonlight yet; I've tested all moonlight versions available this far (latest is
1.9.6 as of this writing) with no luck.

I'm using Ubuntu 9.04, Firefox/shiretoko 3.5.2, AMD64


Steps to reproduce the problem:
1. Load shiretoko, open http://watch.ctv.ca
2. click on any video on the bottom panel


Actual Results: the media player frame stays empty


Expected Results: a media player interface should show up inside the media
player frame, and depending on the video and on where in the world you are, a
video will start playing or a copyright message should show up.


How often does this happen?  all the time

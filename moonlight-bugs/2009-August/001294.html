When showing the XAML hierarchy, it should be possible to copy values out of
the list box (e.g. URLs).

Reproducible: Always

Steps to Reproduce:
1. Right click, Show XAML Hierarchy
2. Find an interesting value
3. Try to copy it
Actual Results:  
Nothing happens.

Expected Results:  
Item should be copied to the clipboard.

This problem exists in both Moonlight 1.0.1 and 1.99.1.

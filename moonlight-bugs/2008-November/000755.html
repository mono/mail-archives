Not all people using Moonlight will be using x86/AMD64 processors. SLES is
available for three further arches, for example (POWER, S/390, Itanium).

Whilst the Microsoft Media Pack is not available for these architectures, users
should be informed of this fact at the point where they would usually be
prompted to install it. Something along the lines of "The Microsoft Media Pack
is not available for your processor type"

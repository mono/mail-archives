I have written a Silverlight demo to playback WavPack audio files. The code
contains an entire implementation of a WavPack audio codec decoder and uses
MediaStreamSource to playback.
The demo works fine on XP/Silverlight.

However on Ubuntu with Firefox 3.6.12, Moonlight 2.99.0.9 there is silence for
a couple of minutes after selecting the file to play, then it starts to play
normally (for, say, 30 seconds) then it plays a metallic/screeching sound.

Reproducible: Always

Steps to Reproduce:
1. Access the demo page at
http://members.upc.ie/p.mc.quillan/silverlight/demo.html

2. Load a WavPack audio file from your local PC (use the 'Open' button)
If you don't have a suitable file you can download one from
http://samples.mplayerhq.hu/A-codecs/lossless/

file to choose is luckynight.wv
Actual Results:  
Silence for a few minutes, then it plays normally, then part way through the
song you get a screeching sound.

Expected Results:  
Song should play normally within a couple of seconds of selecting it.

This is what happens on XP/Silverlight.

The source code (if required) can be downloaded from

http://www.wavpack.com/downloads.html

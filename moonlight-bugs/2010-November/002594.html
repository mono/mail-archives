--- Comment #1 from Rolf Bjarne Kvinge <rkvinge at novell.com> 2010-11-29 10:58:07 CET ---
This is actually by design - moonlight 2.3 needs the mono version it builds
with installed into its prefix, and that mono version is 2.6 (actually a
specific revision of the 2.6 mono branch, the exact info is in the README
file).

For moonlight 3 this has changed, you won't need moonlight's specific mono
version installed (you do still need it checked out next to moon though).

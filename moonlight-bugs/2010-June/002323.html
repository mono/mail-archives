If I attempt to use the Chart control from the April 2010 Silverlight Toolkit
release, the application fails to load due to a TypeLoadException.

Although Silverlight 4 is not fully supported in Moonlight 3, it would be a big
help if the toolkit was pretty well supported.

I tested with Moonlight 3 preview 7.

Reproducible: Always

Steps to Reproduce:
1. Create a Silverlight 4 application
2. Add a Chart control from the toolkit to the main page
3. Run the application from Moonlight
Actual Results:  
A TypeLoadException is thrown and the application fails to load.

Expected Results:  
If support for the April 2010 toolkit is added, then the expected result would
be that the chart is correctly rendered and the application loads properly.

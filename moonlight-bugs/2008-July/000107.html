Inlined XAML didn't work: the control was instantiated but without a source.
External XAML worked as expected.

The problem was observed on Ubuntu 8.04 64-bit with Firefox 3 from the Ubuntu
channel: "Mozilla/5.0 (X11; U; Linux x86_64; en-US; rv:1.9) Gecko/2008061017
Firefox/3.0". The moonlight control was installed from the official web site.
It was version 0.7 supporting silverlight 1.0 spec.

kangaroo in the #moonlight IRC channel helped me to troubleshoot and fix the
problem. It turned out that libssl3.so was missing, yet it was required for the
control. This problem was fixed when libnss3-dev was installed from Synaptic
package manager.

Actually you didn't run with gdb, not sure why. Looks like the firefox script
is broken, because you did the right thing. The crash output is because
moonlight detected that it crashed, and asked gdb to produce a stack trace.

try

gdb --eval-command=run --args /usr/lib64/firefox/firefox <url>

(that's were the firefox binary is located on my machine, if you're on x86 and
not x64 you need to use /usr/lib instead of /usr/lib64. Besides different
distros may put firefox in different places - and since we're skipping the
firefox initialization script it might not detect the moonlight plugin either,
but please try ;-)

This is a bug in either firefox (most likely) or gecko-mediaplayer (unlikely -
this happens with other plugins too).

There is already a bug open for firefox:
https://bugzilla.mozilla.org/show_bug.cgi?id=532666.

Note that you should be able to install both gecko-mediaplayer and moonlight,
then remove pluginreg.dat. Firefox will then regenerate a good version of
pluginreg.dat.

I noticed that moonlight is scaling a lot of audio streams when running
quakelive. I noticed looking in the gnome audio properties, where you can see
the different audio streams. 
I'm using libmoonloaderxpi.so version    3.0.40818.0


Reproducible: Always

Steps to Reproduce:
1.Using Firefox goto http://www.innoveware.com/ql3/QuakeLight.html (I'm using
Ubuntu 9.10 ia32e)
2.In Gnome, goto System, Preferences, Sound and select "Applications tab"
3.While Quakelite runs you should see scaling sound streams.  
Actual Results:  
In gnome sound preferences you'll notice many sound streams from moonlight. 

Expected Results:  
Should only see one stream from Moonlight.

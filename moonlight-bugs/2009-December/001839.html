--- Comment #2 from Rohde Fischer <krdata at gmail.com> 2009-12-21 20:50:20 UTC ---
The exception you're getting is because you're not calling
Deployment.Current.InitializeDeployment, and in fact you're not able to either
due to its protection level.

There is no intention of exposing this method, so you should use XAML in stead
to get it started. If you really intend to fill this programatically you should
be able to do something like this in XAML:

<Canvas x:Name="MyCanvas" />

and then add the grid to the canvas with:

MyCanvas.Children.Add(g);

You could also do it with the usercontrol.

Also as a side note you should be aware what you're trying to do, since you're
adding a circle on top of a button, if you intend to have the circle either
above or besides the button you should define row or column definitions.

launch the application from the desktop, I recieve the following message:

There was an error launching the application.

"/home/rupert/.mozilla/firefox/kei6jniv.default/extensions/moonlight at novell.com/plugins/moonlight/lunar-laucher"

1. Install an out-of-browser application.

3. Launch the application from the desktop.

The application should be launched out of browser.

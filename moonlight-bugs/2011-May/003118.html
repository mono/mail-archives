Description of Problem:
Moonlight does not open any video on www.rai.tv, the Italian national public
broadcast company website.

Steps to reproduce the problem:
1. Install moonlight latest version (3.99.0.3, april 2011) for either Firefox
4.0 and Chrome on Ubuntu Lucid LTS
2. Go to www.rai.tv and try to open any of the latest videos


Actual Results: 
Buffering is shown as expected, but video does not start. The "loading circle"
keeps circling in the middle of the screen.  

Expected Results: 
Video should start


How often does this happen? 
Always

Additional Information:
This is a regression with respect to Moonlight 3.99.0.2 . Streaming used to
work with previous version.

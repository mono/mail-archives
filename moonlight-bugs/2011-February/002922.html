The well known AgDataGrid does not work using moonlight 4 (preview one). The
application starts but the screen remains empty. There is no error reported in
the "error console" of Firefox. I'm aware that this is a commercial product but
in my understanding it should work using moonlight. Maybe this bug report can
be helpful to detect a problem in the current moonlight version. Please find
attached a simple demo application to reproduce the problem.

A free trial of the required assemblies for the 3rd party grid can be
downloaded from <https://www.devexpress.com/ClientCenter/Downloads/#Trials>.
Choose "Visual Studio Controls". After installation the assemblies can be found
in the folder "DevExpress 2010.2\Components\Bin\Silverlight".

BTW: Several of the demo applications (e.g. Book Calendar) available from 
<http://demos.devexpress.com/DXControlsSilverlightDemos> are not working
correctly under moonlight 4.

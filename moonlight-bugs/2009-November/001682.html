When moonlight plugin is embedded with windowless="true", plugin gets no mouse
events. This seems to even include mouse motion, since none of the "mouse over"
states of any controls are activated as the mouse is moved around over the
plugin. Clicks don't work either. I've only tested so far on our own
silverlight application since I don't know where to find public apps on the web
that run windowless. If there's any trouble reproducing this then I will be
happy to create a little testbed app that reproduces the problem and send it
over.

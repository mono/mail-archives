--- Comment #9 from Hongli Lai <honglilai at gmail.com>  2009-02-13 02:32:40 MST ---
I'm on x86 Ubuntu 8.04 with Firefox 3.0.5, Moonlight 1.0, Mono 1.2.6. I confirm
this bug. It happens every time.

But I've found a workaround. The following works:
1. Install Moonlight.
2. Go to http://www.go-mono.com/moonlight/
3. Rightclick on the "Congratulations, you're running the current release of
Moonlight!" banner.
4. Click "Install Microsoft codecs".

This installs the codecs without crashing Firefox.

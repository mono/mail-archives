I am having a problem with XmlSerializer since Moonlight commit 59db9aca72.
However, This commit is not the culprit itself, as it basically bumps Mono to
commit cbeaf9a. This catches *MONO* commit efb1035d49a0a, which is the one
introducing the bug. If I git revert this commit, everything is back in working
order.

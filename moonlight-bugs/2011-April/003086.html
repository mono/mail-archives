The colorado state netfile site has moved to silverlight/moonlight this year
and it does not work on linux. Instead, it gets to 100% and starts over, in an
infinite loop. I am trying to load this page for several months now
https://www.colorado.gov/mytaxes/netfile.htm without success. I am testing on
ubuntu 10.10 with chromium and firefox, moonlight version 3.99.0.3 32bit

Reproducible: Always

Steps to Reproduce:
1. Go to https://www.colorado.gov/mytaxes/netfile.htm
2. Install silverlight/moonlight as instructed
3. Restart browser and try to load the page
Actual Results:  
Page reloads infinitely, after reaching 100% it automatically starts over.

Expected Results:  
Page loads fine and I am able to use the site to file my state taxes.

Another user reported this problem happening to Utah state tax site as well. He
tried building latest moonlight but it reportedly still failed the same way.
Please fix this bug.

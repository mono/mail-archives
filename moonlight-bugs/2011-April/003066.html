This bug report is for the Moonlight 4 preview (version 3.99.0.2.99 stable or
latest nightly build as of Mar 31).

Description of Problem:

Try to load the following page:

http://066019d7fee74d0cabac1fe69eeba3e3.cloudapp.net/Tutorials.aspx

The Silverlight app never appears because an error occurs as the app is
loading. 

The same app loads fine using Silverlight on Windows and Macs but fails to load

incorrect format but I am not sure how that could be.

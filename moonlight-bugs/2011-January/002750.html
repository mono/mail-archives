I have a scenario under which the header of a RadGridView is still displayed
even if scrolled out of visibility.

I am attaching a test-case. I do mention the steps to reproduce below, but I am
also attaching a PDF demonstrating visually how to get there. It is not
complicated, but it's kind of visual.

Reproducible: Always

Steps to Reproduce:
1. Run the TestPage.html. Please be patient as it takes some time to be loaded.
2. Scroll the grid one line down (one is enough).
3. Scroll the grid horizontally totally to the left (displaying what is on the
right).
4. Scroll the grid horizontally totally to the right (displaying what is on the
left).
5. The rightmost header that should be partially hidden now is still being
shown  in full, except where it is being overlapped by the scrollbar.
Actual Results:  
Header that should not be fully displayed, being fully displayed.

Expected Results:  
Header being only partially displayed.

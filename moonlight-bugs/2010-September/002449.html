Chrome supports a value in the plugin API enabling plugins to respect privacy
in incognito mode. The latest Flash plugin already makes use of this and so
should Moonlight.

In particular, Moonlight should disable Isolated Storage in incognito mode. (If
you doubt this is really needed, take a look at this:
http://samy.pl/evercookie/ - Silverlight-support is on the TODO list and will
probably be added, so all kinds of people will make use of this.)

For information on the Chrome API, please take a look at this comment:
http://code.google.com/p/chromium/issues/detail?id=158#c27

Reproducible: Always

Steps to Reproduce:
Use IsolatedStorage in Chromes Incognito mode.
Actual Results:  
I guess data will be stored on disk but have not actually tried. ;-)

Expected Results:  
IsolatedStorage should be disabled.

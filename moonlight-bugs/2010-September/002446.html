By default the content of items in a ListBox is left-aligned. In Silverlight
you can make the items stretch horizontally by defining an ItemContainerStyle
with its HorizontalContentAlignment set to "Stretch". In Moonlight this has no
effect, i.e. the items remain left-aligned.

Reproducible: Always

Steps to Reproduce:
1. Define the ItemContainerStyle on a ListBox to set the
HorizontalContentAlignment to Stretch
2. Add some items to the ListBox
Actual Results:  
The items appear left-aligned in the ListBox

Expected Results:  
The items should stretch horizontally across the ListBox, as is the case in
Silverlight

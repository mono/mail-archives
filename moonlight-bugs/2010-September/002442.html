Components from the November '09 Silverlight Toolkit
(http://silverlight.codeplex.com/) fail to load in a project that also
references version 1.0.2.5 of the Reactive Extensions for .NET
(System.Reactive).

Reproducible: Always

Steps to Reproduce:
1. Create a project that references both System.Windows.Controls.Toolkit and
System.Reactive using the versions specified above
2. Place any toolkit control on the main page
3. Run the project in Moonlight
Actual Results:  
A TypeLoadException is thrown and the toolkit component fails to load. The
output of "MOON_TRACE=E:All firefox" is attached.

Expected Results:  
In Silverlight the component succeeds in loading without issues, so I would
expect that Moonlight should do the same.

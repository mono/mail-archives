Calling method Add(string prefix, string ns) of class XmlSerializerNamespaces
(namespace System.Xml.Serialization) always throws the following exception in
Moonlight 1.99.6 running on SUSE:

Type: System.MissingMethodException
Message: Method not found: 'System.Xml.XmlConvert.ToUri'


Here's a simple code snippet. It works fine in Silverlight 2.05 for
Windows-based browsers:

System.Xml.Serialization.XmlNamespaces ns  = new
System.Xml.Serialization.XmlSerializerNamespaces();
ns.Add("e", "http://www.something.com/");

Reproducible: Always

Steps to Reproduce:
Code example:

System.Xml.Serialization.XmlNamespaces ns  = new
System.Xml.Serialization.XmlSerializerNamespaces();
ns.Add("e", "http://www.something.com/");

Actual Results:  
An exception is thrown:
Type: System.MissingMethodException
Message: Method not found: 'System.Xml.XmlConvert.ToUri'

Description of Problem:
When trying to access a web application (Netflix Watch Instantly) which
requires Silverlight 2.0, application gives error that Silverlight needs to be
upgraded when Moonlight 2.0 should be accepted.


Steps to reproduce the problem:
1. Login to Netflix
2. Choose any "Watch Instantly" video


Actual Results: Netflix gives error that Silverlight needs to be upgraded to
2.0.


Expected Results: Video plays normally.


How often does this happen? Every time.


Additional Information: Admittedly, this was done with a false User Agent
(spoofing Firefox 3.0.10 on Windows Vista). However, the fact remains that an
application looking for Silverlight 2 should accept Moonlight 2.

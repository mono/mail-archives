Steps to reproduce:
1. Create a simple SL app in Visual Studio.
2. In the App class constructor, throw an exception: throw new Exception
("testXYZ");
3. Compile the app.
4. Run it on Linux.

Expected results:
The JavaScript generated by the SL template tells the correct information in
Silverlight:
Unhandled Error in Silverlight 2 Application 
Code: 4004    
Category: ManagedRuntimeError

   at TableTest.App..ctor()     

Current results:

not shown, the stacktrace is not shown.

I have attached a patch to support demuxing using FFMPEG.  It should allow
playback of any media format supported by FFMPEG.

We attempted to follow the same coding style that is used in the rest of
moonlight, but it may not be exactly right.

We haven't done much testing on this patch, but it seems to work well for
all of the files we've tried (mainly some mp3's and some mov files with h264
and aac content, I think we also tried it as a replacement for the existing
demuxer for wmv playback).

Hopefully this will be useful to the community, and will be useful when
adding support for mov/h264/aac support when silverlight 3 comes into play.

Do you guys need any copyright assignment or anything like that?  Let me
know what you think.

Dan

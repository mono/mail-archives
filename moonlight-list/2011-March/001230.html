Hi,

 

You need to have a normal mono installed from git master in addition to the
mono moonlight uses. It's by design that moonlight's mono uses the system
mono/mcs to build (mostly to cut down on our build time). Unfortunately it
also currently requires a system mono from git master (the idea was to be
able to use 2.10, but it's broken right now and nobody has looked into
fixing it yet).

 

Rolf

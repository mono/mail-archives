You either need to install gtk-sharp in your parallel prefix (from source,
here: https://github.com/mono/gtk-sharp), or when configuring moon, pass
--disable-desktop-support to not require gtk-sharp (this will remove support
for desklets, but the browser plugin will still work fine).

 

In my experience gtk-sharp has been slightly painful to build, so I tend to
disable desktop support since I don't need it anyway.

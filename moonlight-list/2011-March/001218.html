Hi, I've developed a Silverlight application that runs great in Windows 
7. However performance in Ubuntu is pretty terrible, almost unusable.

I tried to find out what could be causing this. Using bubblemark.com 
Silverlight 2 test (both on the same dual-booting PC):

Windows 7, Firefox 3.6.15, Silverlight 4.0 = 325 frames per sec

Ubuntu 10.10, Firefox 3.6.10, Moonlight 3.99.0.1 = 19 frames per sec

I then tried the Silverlight 3 hardware accelerated version in Ubuntu 
and it was 46 frames per sec. A bit better but still less than 1/6 of 
the Silverlight performance.

I tried setting EnableGPUAcceleration to true and it stopped Moonlight 
from even loading my application. Under Silverlight it had no visible 
effect.

Ubuntu is using the ATI FGLRX proprietary driver for an ATI Radeon HD 3450.

Any ideas on how I can improve Moonlight performance or at least work 
out if it is Ubuntu, graphics driver, Moonlight or something else?

Hey folks

Moonlight 1.0.1 has been released.  You can get the latest Firefox
plugin at http://go-mono.com/moonlight or by updating the plugin in
Firefox. This release includes some very important bug fixes for users
in non-English locales, as well as a bug fix in the codec downloader.

As always, bugs can be logged at http://www.mono-project.com/Bugs

The source tarball is located at
ftp://ftp.novell.com/pub/mono/sources/moon/moon-1.0.1.tar.bz2

On 03/24/2009 10:39 AM, Jeffrey Stedfast wrote:
> * build mono with `./autogen.sh --enable-moonlight=yes&&  make&&  make
> install` and DO NOT `make clean` afterward (the moon build depends on
> object files in the mono and mcs directories)
>
>    

Ensure that you stuff the new mono $prefix into PKG_CONFIG_PATH so the 
moon configure process finds the newly built mono.
I would suggest building everything required into a seperate prefix (I 
use /opt/mono).

You will also need to install Gtk# before moving on to the moon build, 
other than that these instructions are bang on.

Hey

The packages in that repository are *only* for testing the ability to
build, debug and deploy moonlight applications. This build contains no
video or audio decoders so it's incapable of playing video. This is
expected for now. When we start building the SDK as part of our
regular build and release cycle the you will not have this problem.

Right now if you want a fully functional moonlight plugin, use the one
from the official website. If you want to test building, debugging and
deploying moonlight applications, use the ones from that repository.

Moonlight should mostly work in Chrome, but media streaming is broken.

We're working on support for Chrome and Firefox 3.6+. Currently we have
different plugins for Firefox 2.0, 3.0, and 3.5 because we need to be
able to set specific headers for the media streaming GET requests and
each Firefox version was changing their APIs for doing this. We've found
a way to do it via javascript and so we've been working on a new plugin
to use that technique instead, then it should work in all browsers
because it won't be dependent upon any browser-specific APIs.

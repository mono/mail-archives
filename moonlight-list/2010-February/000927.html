Hi everyone,

I've just committed a new bridge using curl to support video streaming
and have an http backend for non-firefox browsers. Please note it's
not complete, doesn't deal with cookies yet and some parts might not
be exacly right, I haven't tested everything yet - I'm sure there's
threading issues to be fixed. If you want to test it, you have to
build with libcurl, and activate it on runtime with
MOONLIGHT_OVERRIDES="curlbridge=yes"

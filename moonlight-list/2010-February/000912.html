Ah, you're hitting a known bug which has already been fixed in SVN - it's
caused by one of the new SL 3.0 features which has been implemented in the
last few weeks. This fix will be in the next preview release. Unfortunately
the Olympics site is a Silverlight 3.0 site and so will not work with
Moonlight 2.0.

This bug dealt with the issue:
https://bugzilla.novell.com/show_bug.cgi?id=580246

Regards,
Alan.

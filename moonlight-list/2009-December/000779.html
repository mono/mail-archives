> can I download video?

It depends on the source, if it's just a plain video file on the server you
can, by executing firefox like this:

MOONLIGHT_OVERRIDES=keepmedia=yes firefox <url>

You will get a line printed to the terminal notifying about a file in /tmp
not being deleted which is the media file. If the media is streamed (mms://
link), then Moonlight doesn't write the data to the disk at all, it's kept
in memory.

> for example url:
> 
> http://www.rai.tv/dl/RaiTV/programmi/media/ContentItem-04d5069c-335b-
> 48d5-ab6e-5f4cc332d6a8-ctcf.html
> 

This page refers to an asx playlist file (which is downloaded and stored in
/tmp), which refers to a mms stream (which isn't downloaded.) You might be
able to use other programs to fetch the video though once you have the mms
link.

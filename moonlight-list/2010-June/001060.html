Hi,

Since it took me a while to figure out how to debug with gdb in chrome, I
thought I'd share it:

You need to create a script with this (/tmp/dgb-cr in my case, remember to
make it executable):

#/bin/bash
if [[ "x`echo "$@" | grep libmoonloader`" != "x" ]]; then
	echo "Attaching gdb: $@"
	# adding --eval-command=run seems to break stdin
	gdb --args "$@"
else
	echo "Executing without attaching gdb: $@"
	$@
fi

and then run chrome like this: 

google-chrome <url> --plugin-launcher=/tmp/dbg-cr

Note that if chrome decides to spin up more than one process for
libmoonloader (>1 plugin in a page maybe?), this script will spawn several
gdbs, and your terminal will look like a holy mess. The test for
libmoonloader is always required, since we also load libshocker and we'd end
up with several gdbs.

Hope this helps.

Rolf

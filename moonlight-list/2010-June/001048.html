Hello,

I think I once posted similar patch here, but anyways.
This patch tries to add the following SDK assemblies to moonlight_raw build:

     - System.ServiceModel.Syndication.dll
     - System.Runtime.Serialization.Json.dll
     - System.ServiceModel.Web.Extensions.dll

There's however an issue: I couldn't find out how to correctly
attribute InternalVisibleTo on System.ServiceModel.Web.dll which now
has some internal methods (which are public with FIXME comment so far)
to be used by System.ServiceModel.Web.Extensions.dll. Any suggestions
for fix are appreciated.

I noticed that System.Xml.Serialization.dll is already there.

I'm not really sure if those assemblies are built to be usable yet,
but it should be good starter.

They would also build on monotouch. I haven't tried yet, but I made
a comstic change around -r:System.Xml.Serialization (which does not
apply to monotouch).

System.ServiceModel.Extensions.dll and System.ServiceModel.NetTcp.dll
are not done yet.

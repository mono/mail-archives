We need to be API compatible with Storyboard so you can't just add
whatever you want to Storyboard. As storyboard is sealed, you can't
subclass it either. You could write an extension method which looks
something like this:

public static class StoryboardExtensions
{
    public static void Begin (this Storyboard storyboard,
FrameworkElement target)
    {
        Storyboard.SetTarget (storyboard, target);
        storyboard.Begin ();
    }
}

Maybe that'd do what you want.

Alan.

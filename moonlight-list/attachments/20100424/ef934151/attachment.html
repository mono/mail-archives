<div style="font: 1em arial;">

<div>
Montage project is represented by Timeline. Timeline is a single output channel by which movie is render. Timeline consist of channels which contains lists of clips.
Clip is an object which places particular media resource with added effects on channel.
</div>

<br/>

<div>
<img alt="Classes" src="http://dl.dropbox.com/u/681495/MoonComposer_Classes.jpg" />
<div>

<br/>
<h3><font style="color: purple;">Timeline</font></h3>
<div>
An object which describes whole project as final output channel. Contains list of channels for media.
</div>
<ul>
<li><b>StartTime</b> - time when the earliest clip starts on whole timeline</li>
<li><b>EndTime</b> - time when last clip ends on whole timeline</li>
<li><b>SelectionStartTime</b> - time when start selection marker is set</li>
<li><b>SelectionEndTime</b> - time when end of selection is set</li>
<li><b>SelectChannel()</b> - click on channel coused selection and other operations concern this selected channel</li>
<li><b>MoveChannelUp()</b> - move up channel timeline on timeline control</li>
<li><b>MoveChannelDown()</b> - move down channel timeline on timeline control</li>
</ul>

<br/>
<h3><font style="color: purple;">Clip</font></h3>
<div>
Connection between media resource, effects and timeline. Clip object describe single media resource. Each effect should be an another clip above them and playing
simultanouesly with source media.
</div>
<ul>
<li><b>Name</b> - name of clip displayed on clip block</li>
<li><b>ClipStartTime</b> - time of resource when clip starts</li>
<li><b>ClipEndTime</b> - time of resource when clip ends</li>
</ul>

<br/>
<h3><font style="color: purple;">MediaResource</font></h3>
<div>
Describe media file on server and its metadata (tags and previews).
</div>
<ul>
<li><b>Name</b> - resource file name</li>
<li><b>SourcePath</b> - resources address</li>
<li><b>Duration</b> - resource time in miliseconds</li>
</ul>

<br/>
<h3><font style="color: purple;">PreviewMedia</font></h3>
<div>
Its media resources which describes some media resource on server.
</div>
<ul>
<li><b>Name</b> - resource file name</li>
<li><b>SourcePath</b> - resource address</li>
<li><b>Duration</b> - resource time in miliseconds</li>
<li><b>Description</b> - compression method</li>
<li><b>PreviewStartTime</b> - time of source resource when preview starts</li>
<li><b>PreviewEndTime</b> - time of source resource when preview ends</li>
</ul>

<br/>
<h3><font style="color: purple;">MediaEffect</font></h3>
<div>
Is a clip which describes some modifications on media resource.
</div>
<ul>
<li><b>EffectStartTime</b> - </li>
<li><b>EffectEndTime</b> - </li>
</ul>

<br/>
<h3><font style="color: purple;">TreeviewItem</font></h3>
<div>
Visual representation of media resource on Project Treeview Control.
</div>
<ul>
<li><b>Name</b> - Media item name</li>
<li><b>Type</b> - media type (video, audio, image, text, other...)</li>
<li><b>Childs</b> - list of TreeviewItems</li>
</ul>

<br/>
<h3><font style="color: purple;">User</font></h3>
<div>
An object contains information about user, his projects and uploaded media resources.
</div>
<ul>
<li><b>Login</b> - name of user in system</li>
<li><b>Role</b> - describe user</li>
</ul>

<br/>
<h3><font style="color: purple;">Channel</font></h3>
<div>

</div>
<ul>
<li><b>Name</b> - channel name</li>
<li><b>Type</b> - audio or video channel</li>
<li><b>Background</b> - default media for channel</li>
<li><b>Priority</b> - describe importance of channel and its place on play sequence</li>
<li><b>MoveClip()</b> - set position of clip on channel</li>
<li><b>SelectClip()</b> - selected clip couse showing particular information about it</li>
<li><b>SplitClip()</b> - divide single clip into two clips, first ends in place when second starts</li>
</ul>

<br/>
<h3><font style="color: purple;">Renderer</font></h3>
<div>
</div>
<ul>
<li><b>OutputFormat</b> - describe parameters for FFmpeg encoding</li>
<li><b>OutputPath</b> - describe file path for rendered project</li>
<li><b>SaveRenderingTemplate()</b> - places parameters set for FFmpeg on server</li>
<li><b>LoadRenderingTemplate()</b> - downloads parameters set for FFmpeg from server</li>
</ul>

<br/>
<h3><font style="color: purple;">Tag</font></h3>
<div>
</div>
<ul>
<li><b>Kay</b> - tag name</li>
<li><b>Value</b> - optional tag value</li>
</ul>

<br/>
<h3><font style="color: purple;">Marker</font></h3>
<div>
</div>
<ul>
<li><b>Name</b> - name of point on timeline</li>
</ul>

<br/>
<h3><font style="color: purple;">Project</font></h3>
<div>
Describes entire set of clips on timeline and is serialized to file.
</div>
<ul>
<li><b>Title</b> - movie title</li>
<li><b>Description</b> - movie description</li>
<li><b>Author</b> - author of project</li>
<li><b>ProjectFilePath</b> - path of file contains project</li>
<li><b>ProjectResourcesPath</b> - path of workspace for project</li>
</ul>

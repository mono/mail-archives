<br>I was just thinking about how best to solve this and I think if the generator emitted a partial method based on the current property, it&#39;d be trivial to add this extra validation where it&#39;s needed and have no penalties where it is not needed. i.e.<br>
<br>public partial class TextBox {<br>&nbsp;&nbsp;&nbsp; public string Text {<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; get { return GetValue (TextProperty); }<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; set {<br>&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; ValidateText (value);<br>&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; SetValue (TextProperty, value);<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; }<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; partial void ValidateText (string value);<br>}<br><br>If no body is defined for the partial method, it&#39;ll just vanish. <br><br>Alan.<br>

The only streaming format Silverlight supports internally is http streaming
(by using an mms:// url, see
http://msdn.microsoft.com/en-us/library/cc189080(VS.95).aspx#anchor_mediafor
mats_streaming)

The documentation for http streaming can be found here:
http://msdn.microsoft.com/en-us/library/cc251059(PROT.10).aspx

However it is possible to roll your own streaming support (google for
'smooth streaming silverlight' and you'll get an example), but only over
http and the media has to be in the Windows Media format.

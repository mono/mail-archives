Has anyone been able to make the SL2/moonlight plugin run on OSX/ppc?

 

With a few minor changes to the source I have a build for OSX 10.5.5 that
results in the .xpi listed below but it seems the plugin installation is not
successful because Firefox 3.0.3 claims not to recognize the mime types that
I know moonlight supports, e.g.:

 

                <object data="data:application/x-silverlight,"
type="application/x-silverlight-2-b2" width="100%" height="100%">

                        <param name="source"
value="ClientBin/HelloWorld.xap"/>

                        <param name="onerror" value="onSilverlightError" />

                        <param name="background" value="white" />

                </object>

 

The next step is clearly to debug why firefox is not loading the plugin but
if anyone has gone down this path before I'd be more than happy to hear
about it.  And if anyone else is interested I'm happy to hand off the minor
changes I've made to get the build going.

 

Kevin

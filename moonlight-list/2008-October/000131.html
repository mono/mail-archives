To mark a bug to be fixed in 1.0, set the "Fixed in Milestone" field == 
1.0.0

We mark bugs to be fixed in 1.0 based on how important we think the bug 
is. All bugs marked as such should be reviewed by the team.
We can remove any bugs from that list if we think they're not that 
important.

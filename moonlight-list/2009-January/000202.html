> If there aren't any plans, what would need to be worked on to add support?


To add ffmpeg support for the codecs, you just have to add support for the
different fourccs.

However just adding support for the codecs would enable playback of asf
files (the container format used by wmv/wma files) containing media encoded
with those codecs.

Given that the container format is also different, a demuxer would have to
be written for each new container format to properly read the file and
properly split it up into packets of encoded media a codec can understand.

Hi Thomas,

Sorry for the delay, you can download binaries+source from here: 
http://utopia.duth.gr/~akyrtzi/moon.7z 
<http://utopia.duth.gr/%7Eakyrtzi/moon.7z>

Visual studio files are inside "moon/msvc".
"moon-dep" contains the dependencies, most of them were from 
http://www.gtk.org/download-windows.html

To try out a couple of stuff go to "moon/msvc/Debug" and run
-"mopen tiger.xaml" to diplay the vector tiger model
-"mopen calculator.xap.dir" to run the calculator desklet

The moonlight source is from a rather old revision, not the latest. I've 
done some ugly hacky stuff to get it compiled and working (the intention 
was choosing the quicker way and with the fewer modifications to 
moonlight sources), i.e. there are "include files" hacks to avoid name 
clashes with windows headers.
These stuff should be properly resolved (for example, one idea is 
moonlight getting its own namespace to avoid any name clash).

Let me know if you have any questions.

-Argiris

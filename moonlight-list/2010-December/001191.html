later exists in mono/mcs (since it's part of mscorlib.dll) and not in
moon (System.Windows.dll). IOW you need to look at:

No, ML3 won't support everything from SL4 (and I'm not sure how it can
be disabled and unlikely to spend time disabling features for SL4 ;-).

make this return true for Moonlight 3. That should enable your code to
work unless you depend on other SL4 features not present in ML3.

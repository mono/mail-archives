In the last four months I have to worked with moonlight desktop, so I have to
built it several times, and any of them it was a hell.

But, focus us on the last one.

I have decided to follow the instruction of 
https://github.com/mono/moon/blob/master/README README  file, and this is
the result.

Enviroment:
 - Ubuntu 10.4 updated to today morning with mono 2.6.7
 - virtual machine VMWare
 - installed some package by
     apt-get -y install gitk libx11-dev libxrender-dev g++ libglib2.0-dev
libgtk2.0-dev libdbus-glib-1-dev libssl-dev gtk-sharp2 libwnck1.0-cil-dev
libnss3-dev autoconf libtool

So:
 1) created a subfolder in the home: 
        mkdir moon
 2) enter to the new folder: 
        cd moon
 3) followed the readme's instruction
        git clone git://github.com/mono/moon.git moon
        git clone git://github.com/mono/mono.git mono
        cd mono && git reset --hard cbeaf9a && cd ..
        git clone git://github.com/mono/mono-basic.git mono-basic
        cd mono-basic && git reset --hard b104419 && cd ..

4) I don't care about pixel shader support so I have skipped it
5) Executing the "recommended" Automatic Mode Compiling procedure
        cd moon
        ./autogen.sh

  goes all good, but executing the make command I obtain this error:

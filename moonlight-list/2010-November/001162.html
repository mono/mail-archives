For the moment Moonlight is Linux-only, it does not work on Windows (neither
32 nor 64 bit). Given that Silverlight works on Windows (albeit not on 64
bit), making Moonlight work on Windows is not a high priority (though if
someone wants to contribute, we do of course accept patches).

http://www.mono-project.com/Moonlight/Moonlight2Tasks where we can start
putting down different tasks, so we can get them on our radar, figure out
who's doing what, and figure out ordering of them, in the face of so many
interconnected pieces.

possible, the names of other tasks on the same page so we can easily keep
track of things.

If you installed the Moonlight preview via the Firefox xpi, then the
plugin should update itself.
(Tools -> Addons -> Novell Moonlight -> Find Updates)

If you built from source, then you will need to grab the latest
sources and rebuild.

If you installed Moonlight from a Ubuntu repo, you'll need to contact
the maintainer of that repo.

I wasn't very clear, sorry. 

1) Using FF6, download the Add-on Compatibility Reporter (i.e., ACR) from https://addons.mozilla.org/en-US/firefox/addon/add-on-compatibility-reporter/.  It should install to your FF6 browser automatically.
2) Restart FF and hit cntrl-shift-A and go to the Extensions tab.  Make sure the ACR is enabled.

3) Go to the Moonlight page.  You should be able to download and install it.  If you get a message that ML is incompatible with FF6 /after/ it's installed, then ignore it.  The message has nothing to do with whether or not ML will actually work.  If you get a message that prevents the installation process from starting, then you have other issues.

4) After you've got it installed, go to a page that's ML-friendly and ensure that it's working.

Good luck!

I would love to see this happen as well. Unfortunately, I think Apple's
reasoning for not allowing Flash is precisely because it doesn't want

biggest players). Also, I believe that Windows Phone 7 (or was it Kin?)
uses a Silverlight UI toolkit, so that would auto-magically bring
cross-platform support there, too.

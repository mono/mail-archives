Moonlight requires the mcs directory to have the prebuilt binaries. So
you need to build mono first before you can build moonlight.
You will probably want to use a parallel mono install (See
http://www.mono-project.com/Parallel_Mono_Environments )

$ cd mono-2.5
$ ./autogen.sh && make && make && sudo make install
$ cd ../moon
$ ./autogen.sh --with-mcspath=<path> && make && sudo make install

The latest  (3.99.0.3-i586.xpi) has <em:maxVersion>4.0.*</em:maxVersion>
in install.rdf
Firefox 5 was released 21/06/11, Firefox 6 on 16/08/11 so Moonlight is
quite late.
I found an article on editing install.rdf to allow back-porting but,
since Firefox v4, XPI's will not install unless the MD5 and SHA1 digests
in META-INF/manifest.mf are exactly correct.
No doubt building from source would include that facility but I cannot
get past the Package Manager objections of ./configure in my Ubuntu
11.04 - Moonlight seems to always expect older versions of everything.
Sadly, it is easier to boot into Windows to watch Silverlight streams.

Does anyone have easier ways of installing in Ubuntu 11.04 and Firefox 6 ?

BTW the Chromium plugin "almost" works with streams showing percentage
loading but never any video.

Hi,

If you only want the moonlight plugin, and not the desklets/mopen, you can
try configuring --without-desktop-support.

If you do need the desklets/mopen, you need to install gtk-sharp into your
parallel prefix (/opt/mono from what I see).

Note that you'll want the moonlight's mono/mono-basic tarballs, not the 2.6
ones (as explained here:
http://www.mono-project.com/Moonlight#Working_from_tarball_releases)

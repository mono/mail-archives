I believe the correct fix for this error is to run autogen again on the mono
sources. Something is stale in your build.

However if you're building from source we have moved to git now, so you'll
need to check out mono and moon from there:

git://github.com/mono/mono.git
git://github.com/mono/moon.git

If you have any further build issues, join us on irc in the #moonlight
channel (http://www.go-mono.com/moonlight/chat.aspx)

Alan.

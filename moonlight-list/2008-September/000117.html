Hello folks,

   I read the minutes for today's meeting and wanted to comment on the
desired approach on installing the codecs.

   We will have URLs provided by Microsoft for both the EULA and for the
downloadable codec binaries.   The idea will be that once we hit a page
that requires media we would notify the user that there are no codecs,
and ask him if he would like to download the Codecs from Microsoft.

   If the user accepts, we need to fetch the codec EULA, display that
EULA, and if the user accepts the terms, then we fetch the binary
codecs, uncompress them in whatever location we look for codecs, and
dynamically load them and activate the video.

   Additionally, we could do this during the installation of our plugin,
at the end of the plugin installation, we could ask the user if he wants
to install the media codecs at that point, and follow the process above.

Miguel.

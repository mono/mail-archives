Montage project is represented by Timeline. Timeline is a single output
channel by which movie is render. Timeline consist of channels which
contains lists of clips.
Clip is an object which places particular media resource with added effects
on channel.





http://dl.dropbox.com/u/681495/MoonComposer_Classes.jpg 



Timeline

An object which describes whole project as final output channel. Contains
list of channels for media.


StartTime - time when the earliest clip starts on whole timeline
EndTime - time when last clip ends on whole timeline
SelectionStartTime - time when start selection marker is set
SelectionEndTime - time when end of selection is set
SelectChannel() - click on channel coused selection and other operations
concern this selected channel
MoveChannelUp() - move up channel timeline on timeline control
MoveChannelDown() - move down channel timeline on timeline control



Clip

Connection between media resource, effects and timeline. Clip object
describe single media resource. Each effect should be an another clip above
them and playing
simultanouesly with source media.


Name - name of clip displayed on clip block
ClipStartTime - time of resource when clip starts
ClipEndTime - time of resource when clip ends



MediaResource

Describe media file on server and its metadata (tags and previews).


Name - resource file name
SourcePath - resources address
Duration - resource time in miliseconds



PreviewMedia

Its media resources which describes some media resource on server.


Name - resource file name
SourcePath - resource address
Duration - resource time in miliseconds
Description - compression method
PreviewStartTime - time of source resource when preview starts
PreviewEndTime - time of source resource when preview ends



MediaEffect

Is a clip which describes some modifications on media resource.


EffectStartTime - 
EffectEndTime - 



TreeviewItem

Visual representation of media resource on Project Treeview Control.


Name - Media item name
Type - media type (video, audio, image, text, other...)
Childs - list of TreeviewItems



User

An object contains information about user, his projects and uploaded media
resources.


Login - name of user in system
Role - describe user



Channel




Name - channel name
Type - audio or video channel
Background - default media for channel
Priority - describe importance of channel and its place on play sequence
MoveClip() - set position of clip on channel
SelectClip() - selected clip couse showing particular information about it
SplitClip() - divide single clip into two clips, first ends in place when
second starts



Renderer



OutputFormat - describe parameters for FFmpeg encoding
OutputPath - describe file path for rendered project
SaveRenderingTemplate() - places parameters set for FFmpeg on server
LoadRenderingTemplate() - downloads parameters set for FFmpeg from server



Tag



Kay - tag name
Value - optional tag value



Marker



Name - name of point on timeline



Project

Describes entire set of clips on timeline and is serialized to file.


Title - movie title
Description - movie description
Author - author of project
ProjectFilePath - path of file contains project
ProjectResourcesPath - path of workspace for project

Hello,

I'm planning to collect some moonlight custom codecs
in our svn as mooncodecs module, namely:

	- Olivier's csdirac with some changes from Geoff,
	- csvorbis,
	- (IMA) adpcm

(I'm right now in trouble to verify dirac stuff yet.)

My basic ideas (not fully done though) are:

	- each of them could be built for both 2.0 and
	  2.1 profiles, with *.sln file for each. For
	  example, there is csdirac.sln (untouched from
	  Olivier's) and moondirac.sln.
	- They should load in Visual Studio too.
	- On linux we can simply run make to build
	  sample pages. But not "make install" as there
	  is no definite way to "install" them.
	- We could also use mdtool build on those .sln files.

For existing projects:

	- csvorbis can be moved into this mooncodecs, as
	  it d oes not lose full 2.0  (precisely 1.0) build.
	- csdirac too (unless Geoff's changes does not block)
	  as it also does not lose 2.0 build.

I put the files here (it is more 100K so I didn't attach it)

If no problem, I'll check them in svn later.

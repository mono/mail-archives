On Wed, 2009-04-08 at 17:23 -0400, Jeffrey Stedfast wrote:
> I'm probably going to be coming back to this very soon now, so I've been
> wondering:
> 
> How would I handle lowercasing of unicode filenames?

IIRC it's better to use uppercase with unicode since more than one
(uppercase) character can be mapped to the same (lowercase) character -
leading to different file names representing the same "normalized" file.

As for the other points let's talk about them on IRC next week. There's
a bunch of stuff that needs to keep being operational (non-rooted) like
some Silverlight features[1], the current tooling (like smcs) and
probably quite a few "internals" as well.

[1] http://www.mono-project.com/Moonlight/SecurityStatus#Local_I.2FO

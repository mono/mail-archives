Work is under way to support the new API exposed with Xcode 9.0 for iOS 11, macOS 10.13, tvOS 11 and watchOS 4.

This is all happening in our `xcode9` branch [1], the binding status is available from the wiki [2] and a draft of the release notes is also on the wiki [3].

The final date for Xcode 9.0 is unknown. To allow a “same day release” all our work is based on the branch that will be stable when Xcode 9.0 releases (`d15-3`) so, other than the new APIs, everything should be identical to the build currently released in the `Beta` channel.

If/when you’re curious to try those new API you can download [4] our most recent(s) continuous builds that comes out of our bot farm.

Please file bug reports [5] if you find any issue with those builds.

Thanks!

--Chris H

<p class="MsoNormal"><span style="font-size:11.0pt">Bonjour,</span><o:p></o:p></p>
<p class="MsoNormal"><span style="font-size:11.0pt"> </span><o:p></o:p></p>
<p class="MsoNormal"><span style="font-size:11.0pt">Today Apple released Xcode 8 beta 2. Our bots have been updated with the new version and we can start binding the latest API.</span><o:p></o:p></p>
<p class="MsoNormal"><span style="font-size:11.0pt"> </span><o:p></o:p></p>
<p class="MsoNormal"><span style="font-size:11.0pt">The first step will be to fix the existing tests (results should be available in the next hour [1]) so that future pull requests can be quickly reviewed and without regressions. Once we’re back to green we’ll
 be able to resume merging PR for `xcode8` bindings.</span><o:p></o:p></p>
<p class="MsoNormal"><span style="font-size:11.0pt"> </span><o:p></o:p></p>
<p class="MsoNormal"><span style="font-size:11.0pt">Vincent is preparing the documents to track the beta1 to beta2 changes. Only the frameworks already bound for beta 1 can be updated with this information. Missing bindings from beta 1 [2] must be done first
 and then the changes from beta 2 can be applied (but that can be part of the same PR).</span><o:p></o:p></p>
<p class="MsoNormal"><span style="font-size:11.0pt"> </span><o:p></o:p></p>
<p class="MsoNormal"><span style="font-size:11.0pt">Also we’ll soon have `xcode8` based packages for download. This will allow more people to try the new bindings without having to build everything from source code. Stay tuned!</span><o:p></o:p></p>
<p class="MsoNormal"><span style="font-size:11.0pt"> </span><o:p></o:p></p>
<p class="MsoNormal"><span style="font-size:11.0pt">If you have questions about how to participate (in the bindings or elsewhere) in our open sourced Xamarin.iOS and Mac SDK then please join us on gitter [3] and ask.</span><o:p></o:p></p>

<p class="MsoNormal"><span style="font-size:11.0pt">Bonjour,<o:p></o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt"><o:p> </o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt">Work is under way to support the new API exposed with Xcode 8.3 for iOS 10.3, tvOS 10.2 and watchOS 3.2.
<o:p></o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt"><o:p> </o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt">This is all happening in our `xcode8.3` branch [1], the binding status is available from the wiki [2] and a draft of the release notes is also on the wiki [3].<o:p></o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt"><o:p> </o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt">The final date for Xcode 8.3 is unknown but the changes in beta 2 are few, so this might be a smaller/quicker release than most. To allow a “same day release” all our work is based on our current stable branch
 (`cycle8-xi`) so, other than the new APIs, everything should be identical to our existing XI 10.3.1 release. Note that we are close to release 10.4 (RC5 out today) so this work will likely be rebased on `cycle9` soon.<o:p></o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt"><o:p> </o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt">If/when you’re curious to try those new API you can download [4] our most recent(s) continuous builds that comes out of our bot farm. Please file bug reports [5] if you find any issue with those builds. Note
 that a newer/unreleased version of XS is currently needed to deploy from device. The link to download it is inside the release notes.<o:p></o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt"><o:p> </o:p></span></p>

Hello,

We're in the middle of WWDC and some of you might already have noticed a
few PR [1] with bindings for Apple latest APIs.

All the new work on new SDKs, iOS 10, tvOS 10, watchOS 3.0 and macOS 10.12
Sierra, is all being done inside our `xcode8` branch [2]. That will allow
us to continue improving and fixing Xamarin.iOS and Xamarin.Mac before the
new Xcode is released publicly (using other branches and master).

This `xcode8` branch will be kept in sync with our latest stable release
(C8 is our target) so we can do a same-day-release next fall whenever Apple
release the SDKs.

If you are interested in following up the work, built it yourself or
participate in the binding of the new API sthen look at our tracking
document [3] and come chat with us on gitter [4] to learn more.

Note: automated tests will soon be run on every pull requests. They will
verify most of the common binding mistakes that happens while binding
(either our own or Apple's). That makes writing bindings a lot easier than
it looks like and we also got very good documentation on how to create them
on Xamarin's website.

Thanks!
Sebastien

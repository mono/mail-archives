Hey guys,

    A new monodoc feature has been checked into CVS: we now have
incremental searching for topics on the Index.   To use, run Monodoc
like this:

	monodoc --make-index

    If you install new documentation sources, you need to run the above
command to update the index.

    Now, you can use the monodoc "Index" tab.

Is it a problem?  Yes.  The GNU FDL is incompatible with the GPL.  This
means that you cannot quote from GPL code in a FDL document, nor can you
quote a FDL document in GPL code, unless you're the author of both.

Even worse, the GNU FDL doesn't count as Free Software, at least
according to the Debian Free Software Guidelines.  As such, GNU FDL
documentation cannot ship with Debian/GNU Linux.

See: http://people.debian.org/~srivasta/Position_Statement.html

I have a few feature requests for MonoDoc, particularly for the
documentation editor.

First, I would like the Right and Left arrow keys to work.  It's odd the
Up, Down, Begin, and End work for me, but not Right and Left.  Add in
Ctrl+Right and Ctrl+Left for moving by words, and we'd start to have a
decent text interface.

It would also be nice if we had <para/> and <note/> buttons in the
editor.  <para/> is simple enough to remember, but I still had to look
it up in other docs to make sure it was valid...

More pie-in-the-sky would be Copy and Paste support.  I can
Select+Middle-Click text with the mouse, but the "normal" keyboard
copy/paste should also work.

 - Jon

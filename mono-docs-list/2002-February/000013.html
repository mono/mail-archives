I found the download and install instructions for the
mono sources confusing.  Here is a rough outline of
how I would lay out the page:


1  Requirements

   Microsoft .Net Framework
   Cygwin
   Glib
   Pkg-config
   Either the mono nightly snapshot or the very
     old stable release

   Make sure you machine can install and run
   Microsoft .Net Framework and Cygwin before you
   download and install any software.


2  Install Microsoft .Net Framework.


3  Install Cygwin with the development tools.
   Choose the stable releases of Autoconf and other
   development tools


4  Install the precompiled glib and pkg-config.


5  Unpack the mono snapshot.


6  Change to the mcs directory


7  Execute the command 'make windows'

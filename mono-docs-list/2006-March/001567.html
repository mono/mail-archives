On Sun, 2006-03-19 at 01:32 +0100, László Monda wrote:
> 1) How often does the update happen?  On update, I mean updating the
> content of http://www.go-mono.com/docs/

They are updated for every "major" micro release -- 1.1.13, 1.1.14,
1.1.15, etc.

> 2) Is there any ways to download the updated documentation?  I don't
> know about any such possibilities, but I think there should be a way to
> update the documentation locally.

Build and run the monodoc and mono-tools/docbrowser modules on your
local machine. :-)

You don't need to build mono-tools/docbrowser if you don't want an
updated browser; to just update the docs, build & install monodoc into
the same prefix as your existing mono installation.

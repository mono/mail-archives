On Sat, 2003-01-18 at 17:45, Fawad Halim wrote:
> Hi,
> 	I'm trying to run the browser.exe from the monodoc module from CVS on 
> Redhat 8. I get errors on stderr like:
> 
> ** (<unknown>:27426): WARNING **: Failed to load library 
> libgtkhtml-3.0.so (gtkhtml-3.0): libgtkhtml-3.0.so: cannot open shared 
> object file: No such file or directory
> 

Please check out the following modules:

	cvs co -r gal-2 gal
	cvs co -r gnome-2-port gtkhtml

and install them. They will get your libgtkhtml-3.0, which is gtkhtml
(not gtkhtml2) ported to the gnome2 platform.

Hello!

In case no-one responded privately yet, there is much work on
documentation to be done.  There is essentially two main tasks:
1. The Mono Handbook (monkeyguide in cvs) a tutorial for mono and
related projects.
2. Api documentation for the class libraries.

More info at http://go-mono.com/documentation.html

Working on either will be very helpful, and a native english speaker can
always review the grammar at a later time.  So the first step is to
choose what you wish to work on, then you can get the sources and go to
work.  Let us know if you have any questions or need more info on how to
start.

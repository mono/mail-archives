Hello everybody,

my Monodoc-Browser was again not working because of some missing
unmanaged dependencies. Monodoc was looking for libgtkhtml-3.8.so.15
while 3.14.so.19.1 was present in my system.

While this is relatively easy to fix for it has been hard to diagnose
since the browser does not show a detailed error message. A "The
Renderer could not be initialized" is all you get by now.

Therefore I attached a small path to show more error output when the
renderer could not be started.

This is however more a hack than a real solution. But unfortunately I do
not know how to fix the real issue, the unmanaged dependency which could
not be found.

There is an error in the glade-3.pc, in wich if you get the routes to
install modules, pixmaps and catalogs they will be installed in a wrong
directory this is:

catalogdir=@datadir@/glade-3/catalogs
pixmapdir=@datadir@/glade-3/pixmaps
moduledir=@libdir@/glade-3/modules

catalogdir=@datadir@/glade3/catalogs
pixmapdir=@datadir@/glade3/pixmaps
moduledir=@libdir@/glade3/modules

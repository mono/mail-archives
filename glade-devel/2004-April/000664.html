I've just built glade3 cvs to see if my answer (which was using glade-2
rather than -3) was still appropriate.

There is a new clipboard feature that describes exactly what you ask.
The followign procedure can be used to grab the log: 

$ cvs -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome login
$ cvs -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome co glade3
$ cd glade3
$ cvs log > <a temp file>
$ <read/search your temp file>

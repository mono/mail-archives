I've put Glade 2.5.1 up at:

   http://ftp.gnome.org/pub/GNOME/sources/glade/2.5/

This is the second beta release on the way towards 2.6.0.
I think it is practically ready for 2.6.0. Let me know of any problems
with it.

NOTE - you need GTK+ 2.4, and GNOME 2.5.x if you want GNOME support.
Also, interfaces created with Glade 2.5.1 will need GTK+ 2.4/GNOME 2.6.


Glade 2.5.1     (Mar 29 2004)
===========
 
 o Fixed bug migrating old toolbar widgets which meant that widgets
   other than the normal/toggle/radio buttons were being lost when
   loading.
 o Use the new GtkFileChooseDialog throughout Glade.
 o Load the gail-gnome module as well as gail.
 o If there is no .gladep file check the <requires> tags to see if
   it is a GNOME or GTK+ project. Previously it would assume its a
   GNOME project.
 o Set up GtkFrame according to the HIG.
 o A few other minor fixes.

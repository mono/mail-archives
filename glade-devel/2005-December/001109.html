Alessandro wrote:
 > but my question is: the glade result file has to be launched using libglade 2
 > or using the include files installed by glade 3? If the latter is true,
 > what are the functions that i have to use?

No glade-3 uses the same glade format as glade-2 & libglade,
the headers installed by glade-3 are for use of the libgladeui API
which is the core of glade-3 itself, this can be used to integrate
the glade tool into an IDE.

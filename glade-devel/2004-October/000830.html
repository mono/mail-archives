I've put Glade 2.6.2 up at:

   http://ftp.gnome.org/pub/GNOME/sources/glade/2.6/

I'll announce it in a few days if noone spots a problem.

Changes since 2.6.0:

 o Save window positions between sessions.
 o Rewrote selection drawing code so it works better on win32.
 o Support "translatable", "context" and "comments" attributes for text
   properties to help with translation.
 o Bug fixes.


I've also put Glade 2.0.2 up at:

  http://ftp.gnome.org/pub/GNOME/sources/glade/2.0/

That is for people still using GTK+ 2.0.x or 2.2.x. (It won't compile
with 2.4.x.) It just has a few bug fixes.

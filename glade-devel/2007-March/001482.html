What is Glade ?
===============
Glade is a RAD tool to enable quick & easy development of user
interfaces for the Gtk+ toolkit and the GNOME desktop 
environment. The user interfaces designed in Glade are stored 
in XML format, enabling easy integration with external tools.
In particular libglade can load the XML files and create the 
interfaces at runtime.

About 3.2
=========
I should mention that the embedded workspace has been working
out quite well in most regards, events are generally better
interpreted by the core, thus cancelling out some weird bugs
related to interactions with project widgets, also with
drag/resize now enabled by the SHIFT modifier - interaction
is much cleaner than before. I should warn about one particular 
bug (http://bugzilla.gnome.org/show_bug.cgi?id=415021) that 
is causing a crash when redisplaying filechooserdialog, 
ideally we should be working towards a solution in gtk+ that 
allows us to reparent toplevels using supported codepaths.

There is also a bindings framework in the oven, so far we
know we can load widgets written in python, not all
languages will work, api still unstable goes without saying.

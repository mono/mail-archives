Since I missed the source code generation feature found in older
versions of glade a made a small utility for this purpose. It's
supposed to be compatible to how version 2 of glade works.
But it is a comand line utility that doesn't even depend on GTK+.

This first release is mostly a proof of concept. It doesn't yet work
with all widgets supported by glade and a few features are incomplete.

Anyone interested can have a look at it here:
http://glade2c.sourceforge.net/

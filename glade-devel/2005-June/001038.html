     The latest patches in glade-3 now reflect that we are targeting
the latest gtk+, glade-3 now uses GKeyFile to save a
"recently opened projects list". On older versions of GTK+-2.6, glade-3
will segfault when quitting due to a bug in g_key_file_remove_group (),
so people might have to upgrade there development environment to
try/test/use glade-3.

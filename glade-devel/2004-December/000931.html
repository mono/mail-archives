   For those of you who couldn't make it to the meeting, Here is
the irc log (in gz form this time, my previous message was rejected for
its size ;-).

The general immediate game plan is to attempt a test phase release
asap, marking dialogs as generaly broken, fixing current bug list
and cleaning up multiple glade file support (or "getting it to work").

Also here is the list of showstoppers blocking the 1.0 release:

   - Multiple project support (glade files)
   - Multiple selection for cut/copy/paste/delete
   - Custom editors (Support in plugin API + implementation
      of toolbar & menu editors)
   - Cleanup plugin api
   - i18n'ness of generated projects
   - Fix up dialog hell (all kinds of ugly specialcasing to do here)

Maybe there are more, this is what we came up with during the meeting.

Cheers,
                                            -Tristan

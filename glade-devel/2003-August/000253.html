Package: libgtk2.0-0
Version: 2.2.1-4
Severity: wishlist

(Crossposted to glade-devel@ximian.com)

Please add the possibility to specify by command line (with gtk_init())
and/or with environment additional RC files (among ~/.gtkrc-2.0) to be
parsed.

E.g. "app --gtk-rc-file=~/.my-additional.rc --gtk-rc-file=one-more.rc"
or "GTK_RC_FILES=~/.my-additional.rc:one-more.rc"

It is useful among other purposes for Glade for it to look like the actual
application.

It should be in GTK to be common feature for all applications.

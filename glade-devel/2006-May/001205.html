> I want to add widgets to the Glade palette.  I have read through the

> discussion followed.  My main question is whether there is some set of
> APIs to do this or if there is some configuration, i.e. some set of

take a look at widgets/adding-widgets.txt

Do you whant to add an existing gtk+ widget or a custom one?

If you want the later you will have to create a new catalog (xml file)
and probably a plugin library (support code for the catalog)
Otherwise gtk widgets are defined in widgets/gtk+.xml.in

Hi!

I think I found out the problem and the page has been update:
http://gladewin32.sourceforge.net

iconv.dll in libiconv-1.9.1.bin.woe32.zip provided somewhere is linked with
MSVCR70.dll, which is the C runtime library of .NET
I recompiled it using MSVCRT.dll and you can get it here

http://prdownloads.sourceforge.net/gladewin32/libiconv-1.9.1.bin.woe32.zip?download
- or -
http://gladewin32.sourceforge.net/iconv.dll

If you used the package "Glade 2.0.1 for Win32 with GTK+2.2.4/win32
Installer", here's the revision:
http://prdownloads.sourceforge.net/gladewin32/glade-2.0.1-win32_with_gtk_r1.exe?download

With Rgrds,
Ivan Wong.

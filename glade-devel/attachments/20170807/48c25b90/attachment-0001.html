<br>
What is <span class="gmail-m_8100735099215530450gmail-il">Glade</span>?<br>
============<br>
<span class="gmail-m_8100735099215530450gmail-il">Glade</span> is a RAD tool to enable quick & easy development of user<br>
interfaces for the GTK+ toolkit and the GNOME desktop environment.<br>
The user interfaces designed in <span class="gmail-m_8100735099215530450gmail-il">Glade</span> are saved as XML, and by using<br>
the GtkBuilder GTK+ object these can be loaded by applications<br>
dynamically as needed.<br>
<br>
By using GtkBuilder, <span class="gmail-m_8100735099215530450gmail-il">Glade</span> XML files can be used in numerous<br>
programming languages including C, C++, C#, Vala, Java, Perl,<br>
Python, and others.<br>
<br>===========<br>Glade 3.8.6<br>===========<br><br>    - Added -Wno-format-nonliteral flag to make it build on modern compilers<br>    - Use new documentation infrastructure<br>    - GladeWidgetAdaptor: use a direct hash for the adaptor hash table by stuffing the GType into a pointer with GSIZE_TO_POINTER()<br>    - Bug 589306 - Clicking in ... swap order of accelerators (Roberto Guido)<br>    - Add displayable values for properties bug 749796 (Ingo Brückl)<br>    - Fix bug 704600 "Cannot add items to menu hierarchy - assertion `GLADE_IS_WIDGET_ADAPTOR (adaptor)' failed"<br>

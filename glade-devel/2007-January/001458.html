Hi all,

We have committed a major glade3 reorganisation to svn.

Here are the intended outcomes that we hope this reorganisation will
introduce:

  * Increased modularity

  * For new and occasional developers, better understanding of the package
    and it's various components.

  * Increased tolerance for new source files and expansion in general. 

  * More opportunities for refactoring and rearchitecturing.

The new structure:

 data ................... miscellaneous data files
 data/icons ............. application icons
 gladeui ................ libgladeui
 plugins ................ plugins directory
 plugins/gtk+ ........... GTK+ plugin
 plugins/gtk+/icons ..... GTK+ plugin icons
 plugins/gnome .......... GNOME plugin
 plugins/gnome/icons .... GNOME plugin icons
 bindings ............... bindings directory
 bindings/python ........ Python bindings
 src .................... glade-3 executable
 help ................... GNOME user manual
 doc .................... API documentation

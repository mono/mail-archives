<p>I'd like to talk about adding a feature for playing animated image such as GIF files.<br>

<p>But unfortunately I couldn't find such APIs in Xamarin.Forms.<br>
So, to be able to use animated image in app, my proposal is to add below properties to Image class like UWP.<o:p></o:p></p>
<p>1) IsAnmatedImage - Boolean value indicating whether the image is animated.<o:p></o:p></p>
<p>2) IsPlaying - Boolean value indicating whether the animation is playing. If this value is true the animation will start.<o:p></o:p></p>
<p>I think that it will reduce the time to play animated image for App developers.<br>
What do you think about this?<o:p></o:p></p>

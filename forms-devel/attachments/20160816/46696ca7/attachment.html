<p dir="ltr">I'd like to talk about adding a feature for playing animated image such as GIF files.<br>
Since Animated images are used increasingly, many platforms already provide displaying or making animated image in a different way.</p>
<p dir="ltr">UWP - <a href="https://msdn.microsoft.com/en-us/library/windows/apps/windows.ui.xaml.controls.image.aspx">https://msdn.microsoft.com/en-us/library/windows/apps/windows.ui.xaml.controls.image.aspx</a><br>
          <a href="https://msdn.microsoft.com/en-us/library/windows/apps/windows.ui.xaml.media.imaging.bitmapimage.aspx">https://msdn.microsoft.com/en-us/library/windows/apps/windows.ui.xaml.media.imaging.bitmapimage.aspx</a><br>
Aandroid - <a href="https://developer.android.com/reference/android/graphics/drawable/AnimationDrawable.html">https://developer.android.com/reference/android/graphics/drawable/AnimationDrawable.html</a><br>
iOS - <a href="https://developer.apple.com/library/ios/documentation/UIKit/Reference/UIImageView_Class/">https://developer.apple.com/library/ios/documentation/UIKit/Reference/UIImageView_Class/</a></p>
<p dir="ltr">But unfortunately I couldn't find such APIs in Xamarin.Forms.<br>
So, to be able to use animated image in app, my proposal is to add below properties to Image class like UWP.</p>
<p dir="ltr">1) IsAnmatedImage - Boolean value indicating whether the image is animated.</p>
<p dir="ltr">2) IsPlaying - Boolean value indicating whether the animation is playing. If this value is true the animation will start.</p>
<p dir="ltr">I think that it will reduce the time to play animated image for App developers.<br>
What do you think about this?</p>

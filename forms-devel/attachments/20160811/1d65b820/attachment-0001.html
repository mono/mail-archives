<p dir="ltr">This CheckBox will have the following properties.<br>
- FontAttributes : A value that indicate whether the font of the checkbox is bold, italic or neither.<br>
- FontFamily : A value that indicate font family to which the font for checkbox text belongs.<br>
- FontSize : A value that indicate font size of checkbox.<br>
- IsChecked : A value that indicate whether this checkbox is checked or not. <br>
- Text : A text displayed as the content of the checkbox.<br>
- TextColor : A value that indicate the color for the text of checkbox.</p>
<p dir="ltr">Also, this CheckBox will have an event. <br>
- Checked : This event occurs when checkbox is checked. </p>

I find no particular issue for this change other than it being a completely
hidden API. Users who know about it can use it, but there is nothing which
would enable a new user to find it. In the future, if we ever can
(currently not looking likely), we could then change the event signature to
expose the new event args.

Unless objections are raised, I am going to approve this review.

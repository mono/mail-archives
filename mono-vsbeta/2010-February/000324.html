On 2/23/2010 7:38 AM, Dave Glick wrote:
> I’m getting a crash, but only in debug mode. My app runs fine when using
> the non-debug mode from Visual Studio and also when I run it myself from
> the /tmp folder. However, when running in debug mode it crashes suddenly
> without any warning or feedback. I do see several messages being output
> to the terminal (I have the “Run in Terminal” option checked) but the
> terminal window disappears too fast to read it. Is there any way to keep
> the terminal open or does its output get logged anywhere? I’ve looked at
> the “Log Viewer” but there are no entries. Thanks.

You can close the tray icon on Linux, open a terminal window, and run 
"monovs-server".  Then run your app again and the output should stay in 
that terminal window.

Yeah, we are trying to figure out what to do about the terminal window closing when your application terminates.<br>
<br>
When you run locally on Windows, it makes your press a key to close the terminal window.  I am afraid if we set it like that on Linux, people will never check it and end up with hundreds of opened terminal windows.<br>
<br>
For now, you can:<br>
- Not use &quot;Run in Terminal&quot;, however you will not be able to get input from a Console.Read, etc.<br>
- Close the monovs-server icon, open a terminal, and run monovs-server.  This will run a terminal based client.<br><font color="#888888">

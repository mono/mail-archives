Hello,<div>     You may need to install the VirtualBox Guest Tools, if they are not installed in the image already. You can probably google better directions, but:</div><div><br></div><div>Try the menu, Devices-&gt;Install Guest Additions. This will mount an iso image to the machine (you should see a cd-rom icon on the desktop). </div>
<div><br></div><div>Double click cd-rom image to open.</div><div>Then, right click on Desktop and choose Open Terminal.</div><div>In the terminal type &quot;sudo &quot; (but don&#39;t hit enter)</div><div>Drag the 32-bit version of the installer from the cd-rom browser into the terminal.</div>
<div>Hit enter. This should install the guest tools.</div><div><br></div><div>You should now be able to adjust the screen resolution to something larger.</div><div><br></div><div>Two quick notes:</div><div><br></div><div>
1. You may have issues if you are missing things needed for the installer as it does (or previously) compiled code. You (used) to need things like the kernel headers, gcc, etc.</div><div><br></div><div>2. It&#39;s been my experience that anytime you update the kernel, you&#39;ll need to reinstall the Virtual Guest Tool.</div>

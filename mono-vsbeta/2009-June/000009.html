Hi all

We've just released an update to the Mono Tools for Visual Studio, and the associated VMware and Virtual PC images.

You can find the latest downloads for Windows installer msi and virtual image zip files of the on the monovs site ( http://go-mono.com/monovs/ ), or access the download directory directly here:
http://go-mono.com/monovs-download/latest/

Username: mono-vsbeta
Password: VfSpW8lIPQ

The major changes in this release are:

* The VM is now updatable so it won't have to be re-downloaded for each 
release.  The image can be updated from the "Update MonoVS Server" shortcut on the desktop.

* Expression Evaluation has been added.  This enables the following features:
   - Tooltips
   - Immediate window

Please report any issues found against the "Visual Studio Integration" component in Bugzilla:
http://preview.tinyurl.com/monovs-newbug 

Known issues can be searched here:
http://tinyurl.com/monovs-bugs

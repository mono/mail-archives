Hi all

We've just released an update to the Mono Tools for Visual Studio.  The virtual image files have not been updated, instead please use the updater tool on the desktop of the most recent image (0.2.2541).

* Close the monovs server gui by right-clicking on the icon in the panel and selecting "Quit"
* Double-click on the "Update MonoVS Server" icon on the desktop and follow the prompts
* Re-start the server gui by double-clicking on the MonoVS Server icon on the desktop

You can find the latest downloads for Windows installer msi and virtual image zip files of the on the monovs site ( http://go-mono.com/monovs/ ), or access the download directory directly here:
http://go-mono.com/monovs-download/latest/

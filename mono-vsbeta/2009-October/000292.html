aspnet_compiler is Microsoft's program that ships with .Net framework 
and precompiles websites.  We use it to:

- Work around any code that Mono might not compile.
- Provide a better debugging experience.

Unfortunately, we don't have any control over how fast it is.  For 
future versions, we are exploring adding an option that allows you to 
turn off precompiling.

I don't think installing a new version of MonoVS or .Net 4 would affect 
this.  We find the location of aspnet_compiler from:

HKEY_LOCAL_MACHINE\Software\Microsoft\ASP.NET\2.0.50727.0\Path

so I doubt .Net 4 overwrites that.

MonoVS attempts to precompile the website so it can provide better 
information.  If it cannot, it simply goes with what it has.

Microsoft's aspnet_compiler.exe is used to precompile the website, so 
nothing from Mono is used for this step.  (You can see the command line 
used.)

You would likely see the same error in Visual Studio if you attempt to 
"Publish" the website with precompiling.

Jonathan

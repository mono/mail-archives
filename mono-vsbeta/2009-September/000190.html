Yes, this change made it into beta 2.  (Assuming we are still talking 
about running locally on Windows.)

You can see the script LaunchMono.bat that is used to launch processes 
in your Program Files\Mono Tool for Visual Studio.

Check your Tools->Options->Mono->Debugging Settings to make sure the 
option to automatically close windows is set to off.

It gets placed in:
/tmp/monovs-{random-guid}

However, it is only there until you stop running it.  Once you close it 
via the tray icon on windows, it will delete this directory (or else you 
would fill your hard drive).

So if you want to manipulate it, you need to leave it "running" or make 
a copy of it.

It is using xsp2 to host your site.

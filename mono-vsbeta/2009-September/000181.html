Matthias Fuchs wrote:
> I have the same problem. But it can't be that the program has finished. 
> On dotNET the program window is shown and the app quits when closing the 
> window.
> On Mono thoug, the window doesn't appear at all ...
> 
> Could it be, that Mono isn't found?

Generally, this means that an error is being hit while Mono is running 
your application.  For the next beta, the window will not autoclose, so 
you can see if there is an error message.

If Mono isn't found, you will receive a message telling you so, and it 
will not attempt to run it.

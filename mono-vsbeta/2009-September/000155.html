After having a play with MonoVS it is looking really nice, great work guys.
Is there a roadmap or a plan for what will be implemented for the first
version?

One of the features that I think would be really useful is a ReSharper unit
test runner plugin that would allow running/debugging unit tests in linux;
this would make it much easier to fix issues in libraries which occur only
under linux.

The other feature that I think would be nice is to allow the use of x11
forwarding with ssh (for console apps) rather than a terminal window in the
VM; this would provide an development experience closer to running it on
Windows because your keyboard focus would be set directly in that window.

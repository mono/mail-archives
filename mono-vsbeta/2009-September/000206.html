Federico Degrandis wrote:
> Hi, sometime when I click "Run remotely on mono" I get a hosts' list 
> where the ipaddress is 127.0.0.1
> Why?

I think this is occasionally caused by the way the VM is using NAT to 
report the VM's IP address.

I have not come up with a fix for it yet.

For now, you will need to type in the IP address manually if this 
happens.  You can get it by running "/sbin/ifconfig" in a terminal 
window in the VM.

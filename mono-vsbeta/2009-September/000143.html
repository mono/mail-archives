Unfortunately this is explicitly not allowed in the VisualStudio Shell
licence agreement. You can't write a package to support any of the
standard Visual Studio languages. I guess Microsoft don't want anyone
competing with Visual Studio using their own shell as a platform.

You can extend Visual Studio Express using the Tools menu and MSBuild.
Add-in like functionality is explicitly disallowed as of Express 2008.

All the files should be noarch, so they should run on arm.  However, 
debugging will not work as the Mono debugger does not run on arm I think.

monovs-server.exe or mono-gui-server.exe
RemotingContract.dll
Mono.Packager.dll
Mono.Ssdp.dll
Mono.TextTemplating.dll
monovs-web-server.exe
monovs-debug-server.exe
the /templates directory

Basically, take the .rpm, extract the files in it, and put them somewhere.

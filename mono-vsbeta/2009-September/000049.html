> One question: it seems to me that it only supports packaging as .rpm
> files on Suse Linux. Are there any plans to extend that support to
> Debian (.deb files)?

We're only providing .rpm's in this preview because we're all using some
sort of SuSE in our team, so it was easier for us to write and test the
packaging code with .rpm's, but that doesn't mean that we'll only
support .rpm's in the final product.

There seems to be some demand for other Linux distributions like Debian
or Ubuntu from other people as well, so we'll certainly look into
supporting this.

Jonathon Rossi wrote:
> After having a play with MonoVS it is looking really nice, great work 
> guys. Is there a roadmap or a plan for what will be implemented for the 
> first version?

The first version is pretty much what's already in the beta.  We hope to 
polish what's already there, and fix the bugs that people find before 
releasing.  However, we are definitely interested in hearing what 
features people would like for 2.0!

> One of the features that I think would be really useful is a ReSharper 
> unit test runner plugin that would allow running/debugging unit tests in 
> linux; this would make it much easier to fix issues in libraries which 
> occur only under linux.

That's interesting.  I'll have to check out this functionality of ReSharper.

> The other feature that I think would be nice is to allow the use of x11 
> forwarding with ssh (for console apps) rather than a terminal window in 
> the VM; this would provide an development experience closer to running 
> it on Windows because your keyboard focus would be set directly in that 
> window.

This would be neat as well.  The dependencies needed to do this on 
Windows might be an issue.  Will add it to the feature request list!

Thanks for the suggestions!

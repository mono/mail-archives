Hello,     You may need to install the VirtualBox Guest Tools, if they are
not installed in the image already. You can probably google better
directions, but:

Try the menu, Devices->Install Guest Additions. This will mount an iso image
to the machine (you should see a cd-rom icon on the desktop).

Double click cd-rom image to open.
Then, right click on Desktop and choose Open Terminal.
In the terminal type "sudo " (but don't hit enter)
Drag the 32-bit version of the installer from the cd-rom browser into the
terminal.
Hit enter. This should install the guest tools.

You should now be able to adjust the screen resolution to something larger.

Two quick notes:

1. You may have issues if you are missing things needed for the installer as
it does (or previously) compiled code. You (used) to need things like the
kernel headers, gcc, etc.

2. It's been my experience that anytime you update the kernel, you'll need
to reinstall the Virtual Guest Tool.

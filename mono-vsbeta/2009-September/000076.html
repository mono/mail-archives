Yunior Pupo Peña wrote:
> Hello,
> 
> There is any way to run/debug an application remotely on a Windows host?

It has not been tested, but you could probably take the 
monovs-server.exe and related files from Linux and run it on a Windows 
host.  That would let you run an app.  You will not be able to debug the 
app as Mono currently does not have a debugger on Windows.  (Though I 
think someone may be working on one.)

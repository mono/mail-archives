The code you have there to download the file is synchronous...are you running it on a background thread when you call it? Hi,

Anybody ever observed the following behavior when trying to download an image from the internet?
The following code:

private Bitmap LoadThumbnail(Uri eventImageUri)
{
    Bitmap result = null;

    try
    {

        var request = HttpWebRequest.Create(eventImageUri);
        var response = request.GetResponse();

        if (response.ContentLength != 0)
        {
            Stream responseStream = response.GetResponseStream();
            result = BitmapFactory.DecodeStream(responseStream);

            responseStream.Close();
        }
    }
    catch (Exception e)
    {
        DeviceContext.Current.Log.WriteError(e);
    }

    return result;
}

Causes the GC to start collecting chunks of 2K and the app to stop responding. I'm not doing doing anything weird, so I can't really relate to the problem I'm seeing :S

Its fairly easy, but not intuitive.

  1.  Bring up the project properties.
  2.  There should be an option displayed about adding AndroidManifest.xml.  Do it.
  3.  You can edit the resulting .xml file manually, or via the project properties options.  I would recommend doing this via the project properties.
  4.  The .xml file that you see there is a starting point, not the final file, however, its not a big deal.

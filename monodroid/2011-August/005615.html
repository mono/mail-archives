I'm afraid I don't understand your example, as I don't know what Main_Class is, much less what Main_Class.Start(Handler) is.

As for creating a Handler, you'd just need to introduce an explicit type:

	class MyHandler : Handler {
		public override void HandleMessage (Android.OS.Message msg) {...}
	}

then pass an instance of MyHandler to Main_Class.Start().

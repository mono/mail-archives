On Aug 23, 2011, at 6:14 AM, SRI wrote:
>         I have renamed the sqlite db extension to mp4 to avoid the
> compression and hence able to handle big asset files.

An alternative solution is to use the mandroid -0 (zero) option [0] in the $(MandroidExtraArgs) MSBuild property [1]. The -0 option disables compression for all files with the specified file extension. This would allow you to continue using a .db extension (or whatever you prefer).

 - Jon

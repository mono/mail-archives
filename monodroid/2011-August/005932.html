> 
> So the problem is that I was wrong before; you don't want to use the
> `package` line from obj/Debug/android/src/R.java`, you want to use the
> /manifest/@package attribute from obj/Debug/android/AndroidManifest.xml:
> 
> 	<manifest
> 		xmlns:android="http://schemas.android.com/apk/res/android"
> 		android:versionCode="1"
> 		android:versionName="1.0"
> 		package="MonoAndroidApplication2.MonoAndroidApplication2">
> 
> Which results in:
> 
> 	int id = Resources.GetIdentifier("chrysanthemum", "drawable",
> "MonoAndroidApplication2.MonoAndroidApplication2"); 
> 
> (Note the case changes in the package name.)
> 
> This results in the correct value of 2130837504 on my device, instead of
> the 0 that you were seeing.

On Aug 26, 2011, at 5:12 PM, chobo2 wrote:
> Sorry I am still confused. How do I take a shape and change it's color threw code?

You should be able to use the code previously discussed, except use a GradientDrawable instead of a ShapeDrawable, e.g.:

	var bgShape = (GradientDrawable) Resources.GetDrawable(Resource.Drawable.danger_box);
	bgShape.SetColor (Resources.GetColor (Resource.Color.test));

Sorry, my bad. Toast can only be done from the UI thread, not from a
thread pool thread, which is why things go badly.

Since you're dealing with a Service and there's no
Service.RunOnUiThread() method, you need to do things the hard way:

	// within background_DoWork()
	var h = new Handler (context.MainLooper);
	h.Post(() => Toast.MakeText(context, "Toast 3",
ToastLength.Long).Show());

You can also store `h` as a class member (instead of recreating within
the method).

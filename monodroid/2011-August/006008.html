Mono.Data.Sqlite.SqliteConnection.Close()? I don't see that. :-(

If you're using a custom System.Data.SQLite build, ensure that SQLITE_STANDARD was defined when you built, so that SQLiteBase.CloseConnection() would use UnsafeNativeMethods.sqlite3_close() and not UnsafeNativeMethods.sqlite3_close_interop().

On Sep 2, 2011, at 8:05 AM, Stuart Johnson wrote:
> String pwPath = Path.Combine(System.Environment.SpecialFolder.Personal.ToString(), "pw.txt");

Environment.SpecialFolder is an enumeration, not a string constant. You turn the SpecialFolder constant into a path by using Environment.GetFolderPath():

	string pwPath = Path.Combine(
		System.Environment.GetFolderPath(System.Environment.SpecialFolder.Personal),
		"pw.txt");

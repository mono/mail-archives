After this step, you need to "swipe to unlock" the emulator screen by dragging the green "unlock" icon to the right.

Then on step02.jpg the Start Emulator window should be gone, and you'll be able to select the emulator in the Select Device dialog.

If you don't see the emulator listed, see the "IDE does not display target device" section at:

	http://support.xamarin.com/customer/portal/topics/62117-android-troubleshooting/articles

Once you click OK in the Select Device dialog, your app should (eventually) appear in the emulator.

I would strongly suggest that you set the Skin: Built-in: value to QVGA, not HVGA. This will result in a smaller screen size, which should make for a more responsive emulator.

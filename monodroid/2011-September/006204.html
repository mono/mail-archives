Hi,
          If you save the files in /data/data/files
          [Environment.GetFolderPath(Environment.SpecialFolder.Personal)]
          then it will not get deleted.

          But you have to first time transfer the files using
Assets.Open to this folder first time after you package the files as
AndroidAssets in the Assets directory.

          Also configure the Android Options[Tools/Options/Mono for
Android] to not clear the data files at deployment.

          Hope this helps.

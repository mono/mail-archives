Hi,

I finally forked out for monodroid on Thursday using the mega generous
discount scheme for academics and students (it is seriously worth using
it if you're a student or teacher).

I've been developing apps for use by my Forensic science students in C#
using winforms, but want to go that step further and make them available
on Android phones (if work wants them on Apple, they'll need to shell
out for monotouch!).

I have not developed for Android before now as I hate Java with a
vengence (the object model sucks, mainly as it's the C++ model messed
about with). Installed monodroid and within 80 minutes had ported my
first application from winforms to Android. Most of that time was down
to playing with the event system and the UI.

All I can say is a million thank yous for developing this piece of kit.
Now developing for Android will mean a minimal amount of messing and
swearing at the computer.

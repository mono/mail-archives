On Sep 18, 2011, at 9:21 PM, riteshsahu wrote:
> Looks like we've got more missing static methods. I can't find the parseColor method

You'll need to use JNI:

	IntPtr Color = JNIEnv.FindClass ("android/graphics/Color");
	IntPtr Color_parseColor = JNIEnv.GetStaticMethodID (Color, "parseColor", "(Ljava/lang/String;)I");
	int c = JNIEnv.CallStaticIntMethod (Color, Color_parseColor, new JValue (new Java.Lang.String ("darkgray")));

 - Jon

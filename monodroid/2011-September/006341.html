Hey Everyone! 

We are happy to announce a new version of Mono for Android is available!

Some changes included in 1.2.0: 

- Several bugs fixed in Garbage Collection.
- Unhandled exceptions should now print a full stack trace on adb logcat,
instead of silently killing the process.
- Android Device Log for Visual Studio has been beefed up.

Full release notes: 
http://android.xamarin.com/Releases/Mono_for_Android_1/Release_1.2.0

MonoDevelop users should be prompted to upgrade next time they open
MonoDevelop, or can use Help->Check for Updates. 

Visual Studio users can download and install
http://c227137.r37.cf1.rackcdn.com/monoandroid-1.2.0.msi.  You do not need
to uninstall the previous version before upgrading.

Happy hacking! 
Jonathan

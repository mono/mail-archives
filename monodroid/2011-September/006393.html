You would have to modify the schemas that we ship.

In C:\Program Files (x86)\MSBuild\Novell, there is:

- android-layout-xml.xsd
- schemas.android.com.apk.res.android.xsd

Note these will get overwritten every time you upgrade Mono for Android, 
so I doubt this is a very viable solution.

I know there is a push to move from magic ints to enums when possible.
 I recall reading somewhere that we should speak up if we find a
places that are still taking an int when it would be cleaner to take
an enum.  Here are some spots I've found recently:

Android.Widget.EditText.Gravity => Android.Views.GravityFlags
Android.Widget.EditText.SetRawInputType(...) => Android.Text.InputTypes
Android.App.Activity.GetPreferences(...) => Android.Content.FileCreationMode

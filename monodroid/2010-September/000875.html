I like the new preview release, it seems to have cut the launch time down a
lot(but still 45mb apps:-[ ). One suggestion I've got is at that at present,
if the [IntentFilter] attribute is missing then no intent-filter is added
for an activity in the Manifest file, which makes the Activity unreachable.
Would it be worth generating the intent-filter if it doesn't exist to have
an equivalent of:

[IntentFilter(new string[] { "myApp.MySecondActivity" },
Categories = new string[] { "android.intent.category.DEFAULT" })]

And if there's no other activities, then change this to:

         [IntentFilter(new string[] { "android.intent.action.MAIN" },
Categories = new string[] { "android.intent.category.LAUNCHER" })]


An alternative is generate a warning somehow

This has become a bit of a FAQ, so I've added some overloads for the
Intent API which take a System.Type instead of a J.L.Class.  I also
added:

public static Java.Lang.Class Java.Lang.Class.FromType (System.Type);

FWIW, the above code you pasted will probably fail, since you are using
the C# casing of the type name.  You will need to use the java casing of
the package name in order to use Class.ForName or Intent.SetClassName.

The new helper method and overloads will be included in the next
preview.

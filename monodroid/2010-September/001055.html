I looked into this the other day. Besides DroidDraw, Google's Eclipse 
ADT plugin has some basic abilities.

WireframeSketcher is a commercial (US$75) Eclipse-based tool.
http://wireframesketcher.com/

Andrdev-EM-GUI-Designer is an Eclipse-based solution.
http://andrdev.blogspot.com/2010/01/gui-designer.html
http://code.google.com/p/andrdev-emf-gui-designer/

Android GUI PSD Vector Kit are sample UI elements for Photoshop prototypers.
http://www.smashingmagazine.com/2009/08/18/android-gui-psd-vector-kit/
http://www.matcheck.cz/androidguipsd/

And then there's Google's App Inventor, but not generally available. :-(
http://appinventor.googlelabs.com/about/
http://www.designerandroid.com/?p=165

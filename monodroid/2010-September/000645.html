You create a separate class library project for each platform, but you put
each project file in the same folder and they all reference the same code
files.  You should do your development in one primary project, and you only
have to modify the other projects when you add/remove code files or
references.  This "sync" process can be done manually or you can write
scripts to do it . I have been using this technique for years with .NET/CF
development.

>     The MonoDroid build tasks generate a new signing key for each
>     project, in the obj/$(Configuration) folder. It will be regenerated
>     if you ever clean or rebuild the project.
>
>
>     To avoid using this "volatile" debug key, you could explicitly
>     specify a key in the project file:
>     http://monodroid.net/Documentation/Build_Process#Signing

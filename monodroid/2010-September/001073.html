Good news everyone!

We are happy to release another preview of MonoDroid (r8196),
addressing some of the issues encountered and reported so far.

http://go-mono.com/monodroid-download/monodroid8196.msi

Username: monodroid-preview
Password: jsuj96Gh

NOTE: This release has a new shared runtime.  You must manually remove 
the old runtime from your emulator/device for the new one to get copied 
over.  (See release notes for instructions.)

The primary new features of this release are debugging support for 
physical devices and armeabi-v7a support.  Read about them and more here:

http://monodroid.net/Releases/Previews/Preview_4

Enjoy!

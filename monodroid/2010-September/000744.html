Thanks for the reply.

I found part of the answer:

The exact error "E/PackageManager(   59): Package com.example.mapViewDemo
requires unavailable shared library com.google.android.maps; failing!", was
because you have to use an AVD that was built using the Google API's. Then
the PackageManager will deploy the apk, since map.jar is not actually put
into the apk.

Now for the remaining problem. My activity needs to extend MapActivity. In
java you would use:

import com.google.android.maps.MapActivity;
import com.google.android.maps.MapView;

 and use the map.jar from the sdk to provide the interfaces. Since I need to
use "using" statements, if you could tell me how to translate imports to
using statements then I think I can get it to work.

Anthony

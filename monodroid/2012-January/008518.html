The situation is that 4.0.3 is a beta, and your settings for VS to 
prompt you of updates is set to only use stable versions.  Therefore it 
is trying to help you get back onto a stable release.

Tools->Options->Mono for Android->Check for updates

I did not realize this was the intended behavior of our update service. 
  In the future, I won't put the msi link for beta updates into the 
release notes, so that people only get it if they have asked for beta 
updates.

If you do indeed want to go back to 4.0.1, you will need to remove 4.0.3 
first in Add/Remove Programs.

Hope this helps!
Jonathan

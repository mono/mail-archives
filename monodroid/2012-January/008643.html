Sounds like your addin somehow got uninstalled.  Go to Add/Remove 
programs, remove "Mono for Android XX" if it exists, and then download 
the latest msi and install it.  (Make sure VS is closed when you install 
it.)

http://download.xamarin.com/MonoforAndroid/Windows/monoandroid-4.0.3.msi

Hope this helps!

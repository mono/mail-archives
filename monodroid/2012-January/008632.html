> 4.0.3 was "promoted" from beta to stable. As it happens, there was no rebuild involved (this time; things may change in the future).
> 
> We're attempting to migrate to a model where we push a release out into the beta channel, see what bugs appear, and if none/no "major" bugs appear the beta release will be "promoted" to stable status. We'll see how this works out. :-)
> 
> We'll also (eventually) have Alpha builds which have hardly any QA, but will allow testing new functionality. We'll be releasing an Alpha preview for OpenGLES support in this fashion before the next beta release.

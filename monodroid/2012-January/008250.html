Yes, in your activity where you set Label and other parameters for it,
you can set the theme.

I.e.:

    [Activity(Theme = "@style/Theme.MyAwesomeTheme", Label =
"@string/MyAwesomeActivity")]
    public class MyThemedActivity : Activity
    {
        ...
    }

You need to make sure you either start the emulator from within the IDE 
so it gets passed the correct flags, or you can start it from the 
command line if you want to pass the flags yourself.  By default, 
emulators do not have enough space to hold the entire debug runtime.

http://docs.xamarin.com/android/troubleshooting#There_is_not_enough_storage_space_on_the_device_to_deploy_the_package

Hope this helps!
Jonathan

Yup.  Should be in the fragments doc and the fragments walkthrough.
Basically what you need to do is this:

1. Copy the android-support-v4.jar into a folder in your project.
Make sure it's build type is set to AndroidJar ( or some such).

2. Add a reference to Mono.Android.Support.v4.dll.

You'll also need MfA 4.1 or higher.

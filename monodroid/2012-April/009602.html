The Mono for Android team is extremely happy to announce the first alpha 
(4.1.0) in a series leading up to our next big stable release (4.2).

This release has many features people have requested:
- Compatibility Library Bindings
- Java Binding Tool
- API Level 15
- Support for C#'s "dynamic"
- Many productivity enhancements in the VS extension

Full release notes are here:
http://docs.xamarin.com/android/Releases/Mono_For_Android_4/Mono_for_Android_4.1.0

Please note that this is an ALPHA level preview, potentially containing 
bugs.  If you're feeling adventurous, please try it out and file bugs!

If you need to downgrade back to 4.0.6, you can use the IDE updater or 
the Windows .msi:
http://download.xamarin.com/MonoforAndroid/Windows/monoandroid-4.0.6-p2.msi

Enjoy!
The Mono for Android Team

>       Hello Everyone
>       As Jonathan just announced, the latest Mono for Android 4.2
>       preview (Mono for Android 4.1.1) has been released to our alpha
>       channel.  This preview includes designer updates, including
>       designer support in Visual Studio.  It also includes support for
>       working against Android x86 emulators.
>
>       To install Mono for Android 4.1.1, you'll want to switch to the
>       Alpha update channel.
>
>       In Visual Studio, the Alpha update channel can be selected from
>       the drop down in:
>       Tools..Options..Mono for Android..Settings
>
>       In MonoDevelop on Windows, the Alpha update channel can be
>       selected from:
>       Help..Check for Updates
>
>       In MonoDevelop on Mac, the Alpha update channel can be selected
>       from:
>       MonoDevelop..Check for Updates
>
>       Enjoy!

Here's an older version of the installation instructions before we had the unified installer (warning: may have bitrotten...):

	http://android.xamarin.com/index.php?title=Installation/Visual_Studio&revision=26

If you have the JDK and Android SDK installed, all you should need to do is install the Mono for Android SDK and configure your emulators. Presumably you've already configured emulators if you're an Android/Java developer, so that leaves the Mono for Android SDK installation.

You can download the Mono for Android SDK installer from:

	https://store.xamarin.com/account/products

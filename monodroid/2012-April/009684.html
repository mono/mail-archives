Got it!  HTCSync has had a recent update that breaks communication.

My solution:
Connect the device to the computer in Disk Drive mode and find the folder
"HTC Sync" in the device's root folder.
Uninstall HTCSync on the computer and reinstall using the original setup
file found on the device, thus undoing the recent upgrade.

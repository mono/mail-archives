If you just want to display Chinese characters on the console use.  

System.Console.WriteLine(s);

This works because c# treats all strings as UTF16. 

I think System.Console.Out.WriteLine(s) uses a Stream that probably utilises
the default system encoding which in your case doesn't support Chinese
characters.  To make this work you'll have to set the encoding of the stream
yourself, something like - 

StreamWriter writer = new StreamWriter(new FileStream("log.txt",
FileMode.Append), Encoding.UTF8);
Console.SetOut(writer);

Also selecting 'CJK' in the 'Additional supported encodings' in the
Application tab of your project properties might help, although I'm working
with Chinese characters at the moment and it doesn't seem to make any
difference.  At least not in Windows.

I used this to hide the keyboard when a tab changed 

Perhaps you can adapt it ?

 

  TabHost.TabChanged += (IntentSender, e) => Tchanged();

 

public void Tchanged()

        {

 

            InputMethodManager imm =
(InputMethodManager)GetSystemService(Context.InputMethodService);

            imm.HideSoftInputFromWindow(TabHost.ApplicationWindowToken, 0);

            

        }

        

Hope this helps

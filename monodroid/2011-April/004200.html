How did you install your app? Installing via the IDE is supposed to install the necessary packages (in this case Mono.Android.Platform.ApiLevel_8). If you're installing by yourself (from the command line), you'd need to either build & install a Release build (Release builds don't require any other dependencies) or find & install the appropriate Mono.Android.Platform.apk (buried somewhere in the Mono for Android installation).

If you are installing via the IDE, then this is some form of bug and Pobst will need to ask for additional information. :-)

On Apr 10, 2011, at 7:18 AM, Karl Heinz Brehme Arredondo wrote:
> There is some (real world) tutorial about to work with many activities?

	https://github.com/mono/monodroid-samples/tree/master/ApiDemo

Specifically:

	https://github.com/mono/monodroid-samples/blob/master/ApiDemo/ApiDemo.cs#L52

In short, you use Context.StartActivity(Intent) to launch the Activity which the Intent specifies:

	StartActivity (new Intent (this, typeof (ActivityToLaunchNext)));

Note, however, that StartActivity() really just adds a message to the message loop, and won't be executed until "later." Specifically, if you call this in e.g. Activity.OnCreate(), you won't see the activity until _after_ Activity.OnCreate() returns (and ditto for any other context that you call StartActivity() from -- you need to return to Android before it'll actually be launched).

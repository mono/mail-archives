Hi Carlo,

One of our test devices was some el-cheapo generic Android tablet and we had
no USB cables suitable for this device.

We were able to deploy Mono for Android and the associated apk file using a
USB key by copying the files to the memory device and afterwards plugging
that into the USB port on the tablet.

Perhaps this is a suitable option for you to consider.

Can you somehow add "C:\Program Files (x86)\Reference Assemblies\Microsoft\Framework\MonoAndroid\v1.0" (or equivalent) to Sandcastle? This directory contains the Mono for Android System.Core.dll (and System.dll, and mscorlib.dll, and...), which (presumably) Sandcastle needs in order to generate documentation.

If that doesn't work (which I'd consider to be a crime -- how does Microsoft do Silverlight documentation?), there's always mdoc... ;-)

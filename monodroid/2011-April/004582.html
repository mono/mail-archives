All I can suggest is the Android documentation:

	http://developer.android.com/guide/topics/ui/declaring-layout.html

And, of course, reading (far too much) Android source.

Also note that DroidDraw is imperfect; we've had prior reports on this mailing list that XML which DroidDraw generated would result in runtime exceptions when executed on the emulator. DroidDraw should be taken as a guide, and not a "proper" UI editor.

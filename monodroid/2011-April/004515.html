On Apr 26, 2011, at 12:46 AM, Jeff Alstadt wrote:
> I was trying to run the Hello World example on VS 2010 Ultimate SP1  on Windows Vista.
> I wanted to evaluate the functionality of MonoDroid for possible use in project.
> I can start up emulator in 2.2 and it loads up fine. The solution compiles fine as well. However the emulator does not show up in Select Device.

Find the location of `adb.exe` (somewhere in the Android SDK location), and in CMD.EXE run:

	adb devices

Does your device show up? If it does, this sounds like a Mono for Android bug. If it doesn't, then `adb` has gotten confused. Kick it with:

	adb kill-server
	adb start-server

Then try the deployment from Visual Studio again.

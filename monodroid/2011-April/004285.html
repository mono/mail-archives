On Apr 12, 2011, at 1:14 AM, Tamer Hatoum wrote:
> I created a webview but My html files are attached to the application, 
> I create a folder inside resources and put all my web files inside , so 
> how can I put the url to my index.html ???

I'm not sure that it's possible to populate a WebView from use or assets; at least, I see no such functionality described at:

	http://developer.android.com/reference/android/webkit/WebView.html

It appears to either want the actual HTML to load (e.g. or WebView.LoadData(string)) or a URI to load (WebView.LoadUrl()).

You could thus store your HTML in assets, fully read the asset contents into a string, and use WebView.LoadData()...but that's not what I'd call "direct" support for assets or resources.

> 2- How can I make my application autorun when the android starts???

Yes; you need to provide a BroadcastReciever and register the BroadcastReceiver for the android.intent.action.BOOT_COMPLETED broadcast:

	[BroadcastReciever]
	[IntentFilter(new[] {Android.Content.Intent.ActionBootCompleted},
			Categories = new[]{Android.Content.Intent.CategoryDefault})]
	public class OnBootBroadcastReceier : BroadcastReceiver {
		// ...
	}

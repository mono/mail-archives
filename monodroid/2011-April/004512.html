The MFA documentation covers AsyncTask here with examples of how to use it:

http://docs.mono-android.net/index.aspx?link=T%3aAndroid.OS.AsyncTask`3

<http://docs.mono-android.net/index.aspx?link=T%3aAndroid.OS.AsyncTask`3>The
main thing you should keep in mind is whether you're doing something on the
UI thread or the background thread. As the name implies, you can only make
UI changes (ie: showing/hiding a progress dialog) on the UI thread. I have a
little example of doing that a different way here:

https://github.com/gshackles/Sample-Projects/blob/master/MonoDroid/MonoDroidSamples/MonoDroidSamples/DemoActivities/TextNotify/TextNotifyActivity.cs#L51

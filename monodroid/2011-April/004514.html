> 1) adb install -r FILENAME runs OK, IF I have the new APK. But on
> MonoDevelop Build option I do not have Package option.

This is a missing feature in MonoDevelop which should be provided in a future release.

In the meantime, you can create the package manually by using MSBuid/xbuild:

	xbuild Project.csproj /t:PackageForAndroid

You may want to add /p:Configuration=Release when packaging a release build.

Behold the power of LINQ!

	var rootele = new XElement ("Root",
			myCrew.Select((e, n) => new XElement ("crew", 
				"Ui",
				 e.Ui[n] == null ? UID.ToString() + n.ToString() : e.Ui[n].ToString()));

XElement.Add(object) will check to see if the value implements IEnumerable, and if it does it will enumerate each element, recursively (in a depth-first fashion) until it hits a string, XObject, or other non-IEnumerable value. This allows simple transforms from lists to XML.

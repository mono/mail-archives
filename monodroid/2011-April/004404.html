We check the following keys for the Java version:

HKLM\SOFTWARE\JavaSoft\Java Development Kit\CurrentVersion
HKLM\SOFTWARE\Wow6432Node\JavaSoft\Java Development Kit\CurrentVersion

With the CurrentVersion we find, we look for the path in:

HKLM\SOFTWARE\JavaSoft\Java Development Kit\<currentVersion>\JavaHome
HKLM\SOFTWARE\Wow6432Node\JavaSoft\Java Development 
Kit\<currentVersion>\JavaHome

These keys should have been created by the Java SDK installer.  Check 
and see if they exist on your system.

Jonathan

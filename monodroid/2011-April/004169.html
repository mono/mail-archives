Hello Wally,

This issue is basically a bug showing up when the android is installed but
Mono for Android itself hasn't been installed already -in this case, MD
should *not* display any error message nor the option to create a new Mono
for Android solution-.

If you have Mono for Android already installed, it should work just fine.

You need to wait until the emulator is completely loaded before attempting
to deploy to device ("device offline").

1)      Start without debug (ctrl - F5)

2)      Start emulator image if no active emulators or real devices
available. "Start emulator image"

a.       Then wait until the start screen for android OS version selected
Visible inside the emulator. (takes some time)

3)      Deploy application by selecting device.

a.       Will auto launch app if you wait long enough

4)      Once deployed there is a debug > debug option in visual studio as
listed in article but all I have is F5 "start debugging" and haven't yet
successfully hit breakpoints yet.

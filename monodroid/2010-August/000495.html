I haven't looked at the issue much, but one potential problem is that 
resources are only 'findable' after you have called SetContentView.

So this works:
SetContentView (R.layout.main);
var button = FindViewById (R.id.myButton);

This does not:
var button = FindViewById (R.id.myButton);
SetContentView (R.layout.main);

Also, these methods are instance methods on the Activity class, so they 
can only be called if you have an instance of Activity, not in a 
generated resource class.

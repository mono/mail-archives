The N1 is a great phone. But if you're doing App programming (not
system level stuff), any Android phone would be fine, methinks. I've
got a G1 and DroidX here, and soon to have a BeagleBoard-xM available
to me for Android hackery.

Now, you may want a phone that can handle NEON for the SIMD stuff,
which I think the Cortex-based set of chips would be best, which I
think the N1 and the recent OMAP-based hardware can handle. However, I
don't think Mono.SIMD does NEON yet (I smell a weekend project for
me....). Of course, I don't think MonoDroid supports Mono.SIMD
either...but it's food for thought.

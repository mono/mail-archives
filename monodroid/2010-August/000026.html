MonoDroid Users,

  We have identified some potential issues with the initial release of MonoDroid and issued r7755 to address the following issues:

- Fixed several important crashers in the debugger
- Certain usages of the debugger inspecting locals could have resulted in a timeout
- Certain installations required the JDK to be in $PATH, this should now be resolved
- Modifying the androidMinSdk variable could have resulted in an invalid AndroidManifest.xml
- Fixed a possible issue in Android.Runtime.JavaTypeConverter that could have generated a Unexpected type conversion

We have posted the new release to:

http://go-mono.com/monodroid-download

Please let us know if there are any futher issues via bugzilla at http://monodroid.net/Support

Thanks

Geoff Norton

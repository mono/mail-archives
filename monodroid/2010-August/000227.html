If you are following the Tutorials, some tell you to create the folders
while I'm pretty sure the first "Hello World" made it sound as if they are
already there.

* TIP *
At the moment the "Add New Item" dialog will not include an XML File as an
option but you can type "XML" into the search box and will be able to create
one right from there without creating something random and renaming the
file.

* TIP*

Once you get the XML File into your project, go into the Properties and set
it's Build Action as "AndroidResource".

This is an early beta so I'm sure an "Android Resource" item will be added
in to the "Add Item" box later to set all this for us.

On 8/25/2010 4:35 PM, Matthew Baxter-Reynolds wrote:
> Does the "no space left on device" sound likely?  (could be sd card?)

You likely did not start the emulator from within Visual Studio in the 
Device Chooser, so it didn't get passed the magic barely-documented flag 
that allows the emulator to start with more than ~40 MB of free internal 
space.

If you want to start the emulator image yourself, pass the 
-partition-size flag to it:

emulator.exe -partition-size 512 -avd <emulator image>

Or close the one you have open, try to run from within Visual Studio, 
and choose "Start emulator image".

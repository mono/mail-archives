The tutorial is wrong.  There is currently no "AVD Manager" link under 
Tools.

When you bring up the device chooser, there is "Start emulator issue" 
which launches a dialog that has "Create new emulator image".  This is 
the "AVD Manager".

You can add it as an external tool currently if you want a menu item:
C:\android-sdk-windows\tools\android.exe

That *may* require you to put a reference to your Java SDK bin in your 
PATH environment variable.

Jonathan

> versions of Java SDK (thanks Nate). I guess that's because Android SDK 
> searches for x86 and MonoDroid deployment looks for x64 version.

Hi,

The MonoDroid tools can detect either the x64 or x86 JDK automatically. 
Unfortunately the Android SDK tools invoked by MonoDroid can only detect 
the x64 JDK when on Windows x64. This has been fixed for the next build 
we will release - MonoDroid adds the JDK to PATH when invoking Android 
tools.

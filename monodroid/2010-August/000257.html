- Are you sure you installed the JDK and not the JRE?
- 32 bit or 64 bit Windows?
- 32 bit or 64 bit Java?
- What are the values of the following registry keys?

HKLM\SOFTWARE\JavaSoft\Java Development Kit\CurrentVersion
HKLM\SOFTWARE\JavaSoft\Java Development Kit\<CurrentVersion>\JavaHome
(where <CurrentVersion> is the value of the previous key)

HKLM\SOFTWARE\Wow6432Node\JavaSoft\Java Development Kit\CurrentVersion
HKLM\SOFTWARE\Wow6432Node\JavaSoft\Java Development 
Kit\<CurrentVersion>\JavaHome
(where <CurrentVersion> is the value of the previous key)

Does the path listed in the registry key match the path you installed 
Java to?

Jonathan

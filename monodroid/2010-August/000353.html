etc etc.... In the Java world we write HTML, XML, CSS etc directly
ourselves! Look at the Java world... JavaFX only got a designer in

a) Wait for Novel team to provide one;
b) Write one yourself or;
c) write XML layout yourself.
d) Use DroidDraw even if its seen as "abandonware".

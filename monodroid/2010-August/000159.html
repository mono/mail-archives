Hi all

Just wanted to share a tip in terms of not having to install 32 bit Java and
stay with 64 bit Java

I ran into this issue earlier just getting the Android SDK working, I traced
the error to a findjava.bat file where it looks in your path as well as some
hard coded directories for the java executable
Once I added the path for my JRE's bin path to my environment variables, the
Android SDK would succeed in locating it and thus start working.

So then I installed the monodroid install file, and proceeded to use it
without any problems at all and no special monodroid configuration needed
and no 32 bit install of the JDK or even the JRE needed

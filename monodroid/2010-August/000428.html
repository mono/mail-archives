I think Miha's point is that the logic is still there, it's just harder to
read. If you have the logic you can use software to make it more readable or
just step through it yourself.

 

Obfuscation isn't any type of real protection but sometimes it can be used
for mild performance increases depending on the type of obfuscator you use.

 

Personally, I don't use it. It makes it a million times easier when I look
at someone else's code to see if they are "borrowing" my logic because they
can do it either way.

On 8/20/2010 1:59 PM, Tom Opgenorth wrote:
> Noticed that MonoDroid install a 15 MB Monodroid Runtime application
> on my phone.  I was wonder:  with older versions of Android, one
> couldn't install apps on the SD card.  15MB is a fair bit of space to
> consume, especially on older versions of Android where one can't move
> apps to the SD card.  Does anybody have any thoughts on how to
> possibily mitigate the impact of this 15MB runtime?
>

We are extremely aware of the current size requirements.  So far, we 
have done zero work in this area in favor of getting it working first.  :)

We intend to continue slimming the profile as well as provide linking 
out unused code, just as we eventually did in MonoTouch.

Hopefully when we are done, our MonoDroid packages will end up at 
roughly the same size as MonoTouch.

Jonathan

I found a problem with Monodroid’s DrawLine method. Plotting a line with a
slight gradient on it using a Paint with  StrokeWidth = 1 doesn’t get
plotted, for example:

 

      Paint pen = new Paint();

      pen.Color = Color.Red;

      pen.StrokeWidth = 1;

      pen.SetStyle(Paint.Style.Stroke);

 

      canvas.DrawLine(100, 100, 150, 110, pen); //doesn't work

      //canvas.DrawLine(100, 100, 150, 90, pen); //doesn't work

      //canvas.DrawLine(100, 100, 150, 100, pen); //works

      //canvas.DrawLine(100, 100, 150, 190, pen); //works

any workaround

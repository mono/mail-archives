In Tools->Add-in Manager, expand the Mobile Development node. You should have a "MonoAndroid development" node, version 2.4.2.4.

If you don't, then click the Install Add-ins button, expand the Mobile Development node, and select MonoDroid development version 2.4.2.4. Press the Forward button twice, click OK when installation is complete, Close the Add-in Manager, and Restart MonoDevelop.

If that doesn't work, please let us know.

 - Jon

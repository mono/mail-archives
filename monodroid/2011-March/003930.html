We have ported UnitDriven (http://unitdriven.codeplex.com) to MonoDroid.

 

UnitDriven is the unit test framework/runner we created originally for
Silverlight 2 (when there was no unit tester for SL), and then ported to
WP7, and now to MonoDroid.

 

I use it to run all the .NET unit test code for CSLA .NET
<http://www.lhotka.net/cslanet/>  - so I have the same unit tests for .NET,
SL, WP7, and now MonoDroid.

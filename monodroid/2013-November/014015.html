I believe it is a runtim error with the Xamarin version of Amazon AWS SDK. 
The SDK uses XSLT files at runtime to process the XML responses and the
Xamarin version of Amazon AWS SDK does NOT include the XSLT in the project
as EmbeddedResources.  Set all XSLT files as EmbeddedResources and rebuild
the Xamarin version of Amazon AWS SDK and it should work.

Just to follow up, I was able to get the Google licensing service to work
from Mono for Android by creating a Java Android library using Eclipse. 
This way I could reference the market licensing files, but package them up
into a single java library.  I then included that .jar file into my Mono for
Android project and set the build action to AndroidJavaLibrary.  At this
point I used JNIEnv to access my wrapper Java class in the .jar file to call
the licensing service.  Thanks!

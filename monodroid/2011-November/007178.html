So you say you do not get any information or stack traces in your
output window in Visual Studio?

If you run the latest beta there should be improvements on that part
catching uncaught exceptions and presenting them in VS - that is ofc.
if you actually use the debugger.

Otherwise you can either use adb or ddms to show the device log and
see if there is any stack traces there.

You can just paste the XML here on the mailing list so that others can
help as well, or put it in a gist or on pastebin.

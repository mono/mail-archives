Just to close up this question, this occurs if you have multiple projects and
do not reference EVERY single project in your main Mono project.  For
example, in a typical .NET project you would try to keep your main project
references down to minimum to reduce dependencies, but this doesn't work
with Mono for Android.  You must have everything referenced, I am assuming
this is for the Linker to properly do its job.  

Anyways, I solved this a long time ago but I thought I would just answer my
own question for others knowledge.

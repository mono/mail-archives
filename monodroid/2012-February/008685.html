I've made some home use apps for the Nook Color with Mono for Android. It
works pretty good. If you are working with nook that haven't been rooted
you have to be a registered developer through Barnes & Noble.

B&N has a submittal process and access to more documentation via there
knowledge base/developer forums once you're approved.

Before being approved you won't be able to get a standard nook to be
recogonized by adb since the developer mode is disabled.

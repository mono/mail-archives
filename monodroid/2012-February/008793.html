Jonathan Pobst from Xamarin support provided a working solution:

"You can try setting the Project Settings -> Application target to Android
2.2, and then opening the AndroidManifest in your projects properties
folders and manually adding targetSdkVersion = "13".

If targetSdkVersion is set, it will be used rather than the one set in
Project Settings -> Application."

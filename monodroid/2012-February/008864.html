A few thoughts, that my/not be helpful.
I've got the Android sdk installed on my mac and MD working with MfA.  I had MD installed first due to mono touch development.  I'm not quite sure what the issue is, but if you have the SDK installed in multiple places, that's a formula for problems, at least on Windows.  On Windows, the only solution was to uninstall everything and then install the sdk correctly.  I've found that the android sdk is problematic at best, so anything is possible.  Not seeing the emulator is most likely an MD issue as you mention.  I wonder if there is any hope for getting a Xamarin installer for the mac.  The Jonathans would have to respond to that.  Playing video in MfA is nearly a 100% port from the Java world.  I have a magazine article coming out in March on this very subject in VSM.I have no idea about the MSBUILD issue on the mac is.  My guess is that there is an install problem on the mac.  I would suggest uninstalling and reinstalling.  I had MD installed first on my mac before I installed the Android SDK.  Perhaps order matters.  Maybe installing MD, the Android SDK, and finally MfA in that sequence would help.  I'm just talking out my a** there, so it may not be accurate.

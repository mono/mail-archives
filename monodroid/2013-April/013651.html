What version of Xamarin.Android are you running?

Xamarin.Android 4.6 introduced the free Starter edition which allows device deployment, but only from Xamarin Studio. You're within Visual Studio, with an "Evaluation version" message, which suggests you're using Mono for Android 4.4 or earlier.

I would suggest upgrading to Xamarin.Android 4.6:

	http://xamarin.com/download

 - Jon

OK, this may be a stupid question, but here goes. Are you launching the
application in release mode from within Visual Studio or directly on the
device? I found during some testing of my own that if you launch an
application built in release from within studio the device still attempts to
start a remote debugging session which causes the application to crash
almost immediately. When I launched the application from the device
everything worked as expected.

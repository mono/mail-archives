Yes you Can in the AVD Manager from google you can Create a new Emulation
Device and set the resolution of the Emulator Device.

Start -> Programs -> Android SDK Tools -> AVD Manager
or 

AndroidSDK\AVD Manager.exe

select a virtual Device use the Edit option and you can choose a individual
Resolution.

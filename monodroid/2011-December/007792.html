On Dec 9, 2011, at 5:31 AM, Miljenko Cvjetko wrote:
> It seems that VS and MD on the same laptop (machine) are two different machines.

I think I need to elaborate somewhat. A "unique machine" is the OS + the hardware, not just the hardware itself. Thus, installing Mono for Android under Windows and under OS X on the same physical hardware will result in two activations.

Within Windows, both Visual Studio and MonoDevelop should use the same activation. You should not need to re-activate Mono for Android between both Visual Studio and MonoDevelop/Windows.

If this isn't the case (VS throws errors, while deploying from MonoDevelop/Windows works), then something is wrong and we'll need to collect more information from you.

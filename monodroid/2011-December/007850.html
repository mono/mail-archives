On Dec 14, 2011, at 1:08 PM, Neal Culiner wrote:
> I checked this option (and the other one, basic armeabi) but did not see my .apk size grow so I’m not sure it’s packaging both in 4.0.1.  How do I know?

Run `unzip -l your.apk`, it should contain the file lib/armeabi-v7a/libmonodroid.so.

You can also check your .csproj and ensure that it has e.g.

	<AndroidSupportedAbis>armeabi;armeabi-v7a</AndroidSupportedAbis>

On Dec 19, 2011, at 8:05 PM, mattbrand wrote:
> Taking your advice, I got the emulator to start up correctly using the AVD Manager.

That's probably the problem. When you start an emulator via the AVD Manager, the resulting emulator has a _tiny_ /data partition (64MB), which isn't enough to store the Mono for Android packages.

You need to start your emulator either within the IDE, or launch the emulator from the command line and provide the -partition-size parameter:

	emulator -partition-size 512 -avd AVD_NAME

 - Jon

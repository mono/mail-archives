Today we are releasing the next preview of the Xamarin.Mobile library. This
version contains our cross-platform Contacts APIs for both Mono for Android
and MonoTouch (Windows Phone 7 will be added in a future release).

This version of the cross-platform contacts API is read-only. We expect an
upcoming release will include support for adding and modifying existing
contacts.

You can read the API documentation and download the binaries here:
http://betaapi.xamarin.com/

We are continuing to run a survey to better understand which are the most
important APIs and platforms for our community:
https://docs.google.com/a/xamarin.com/spreadsheet/viewform?hl=en_US&formkey=dFBlbEtBZmZaT3pIYnBTRHZOQWV4cFE6MQ#gid=0

If you want to send us feedback or report bugs, please File a Bug Report in
our bug tracking system:
http://bugzilla.xamarin.com/enter_bug.cgi?product=Xamarin%20Mobile%20API%20Preview

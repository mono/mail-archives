Also the inflation exception might be because of you not correctly referencing a custom view you made or something. I.e. mixing uppercase and lower case letters when Java spec only allows lower case letters for package names. Like described here:
http://docs.xamarin.com/android/advanced_topics/using_custom_views_in_a_layout

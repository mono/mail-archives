There is an issue with your adb.exe:

-- Starting Process - 18.12.2011 12:01:54 --
Executable: G:\Android\ANDROI~1\PLATFO~2\adb.exe
Arguments: -s 3630F086D76300EC shell cat /data/system/packages.xml
[STDOUT] adb server is out of date.  killing...

In the past, I've seen this caused by HTC Sync software and something 
called Android Explorer.  You might want to verify that you don't have 
these installed.

You might also try killing the adb.exe process as well.

If that doesn't help, you'll need to Google the error and try to find a 
solution.  Mono for Android relies on Google's adb.exe in order to 
communicate with the device.

Hope this helps,
Jonathan

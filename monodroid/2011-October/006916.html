It's going to have something to do with one of the following attributes:

[Register]
[Activity]
[Application]
[Service]
[BroadcastReceiver]
[ContentProvider]

Without seeing any code or test case, I don't think I can tell exactly 
what the issue is, but maybe you can try commenting out usage of these 
attributes in your app to narrow down which one is causing a problem.

Hope this helps!

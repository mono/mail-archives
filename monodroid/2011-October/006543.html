How are you installing, via the IDE or otherwise?

If installing via the IDE, this sounds like a bug. If installing via `adb`, then you need to also install the per-API level support package, e.g.:

	adb install /Developer/MonoAndroid/usr/lib/mandroid/platforms/android-12/Mono.Android.Platform.apk

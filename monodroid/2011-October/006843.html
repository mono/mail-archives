Just wanted to share with You, that last weekend I have managed to work 
with 2
devices. Prior to that I had some problems with usb drivers for HTC 
sensation and
Sony Ericsson Xperia X10...

Tried generic google drivers did not work and then went to OEM/vendor 
specific....

writeup FYI:

USB Drivers installation (loose less time)
Docs and references
- Google Windows USB drivers http://developer.android.com/sdk/win-usb.html
- OEM (Vendor specific) drivers 
http://developer.android.com/sdk/oem-usb.html

Prepared everything for deployment with Android SDK and AVD Manager.
Checked/downloaded: Available Packages =+ Google USB Drivers.

Tryed to install 2 devices on Windows Vista:

HTC Sensation
Sony Ericsson Xperia X10
in Computer =+ Manage =+ Device Manger drivers were reported

- for HTC Sensation as "Android Device"
- Sony Ericsson Xperia X10 as "SEMC HUSB"

Downloaded drivers for:

- HTC Sensation no drivers, but HTC Sync Application
http://www.htc.com/www/help/
- Sony Ericsson Xperia X10 drivers
http://developer.sonyericsson.com/wportal/devworld/search-downloads/driver?cc=gb&lc=en

Driver installation for X10 failed, but for XTC Sensation after 
installation of Sync application
everything worked as charm. Next step was to give it a try and download 
similar
application for Sony Ericsson and got PC Companion
http://www.sonyericsson.com/cws/support/mobilephones/downloads/subject/pccompanion/1.689743?cc=gb&lc=en 
.
Again after installation eveything worked like a charm...

Conclusion
It seems that it makes sense to install appropriate syncing applications 
for devices,
because most likely that they have up-to-date USB drivers for device.

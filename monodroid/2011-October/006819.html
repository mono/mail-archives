This is a public service announcement for anyone who is using Mono for
Android installed from our new one-click installer.

The new Android 4.0 Ice Cream Sandwich release changed update URLs and
repository file formats in a way that have made it impossible to install
new
Android updates from the version of Android that our installer provides.

We released a corrected one-click installer yesterday that should be more
resilient against these kinds of changes.  You can grab the corrected
installer
from our public download page:
http://xamarin.com/trial

For avoidance of doubt, the current version of the installer is 1.0.2.2, and
if
you are running an installer that says anything less than 1.0.2.1-beta2 in
the
bottom left corner, you need to grab a new installer to update the Android
SDK.
You may need to use the Windows Control Panel to uninstall the old installer
if you get an error while installing from the new installer.

Sorry for the interruption.  We now return you to your regularly scheduled
Mono for Android programming.

--Joseph

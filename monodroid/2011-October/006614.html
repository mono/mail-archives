There did not seem to be much API change between and 12 and 13, and 11 
had very little market share (and shrinking), so we went with 12.

Each level we add takes up an additional ~50 MB, so we're not sure if we 
want to add them by default.  We probably need to come up with a way to 
allow people to install additional targets that they are interested in.

Or maybe it will all be a moot point once Ice Cream Sandwich comes out.  ;)

Jonathan

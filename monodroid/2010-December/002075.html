On Dec 15, 2010, at 7:04 AM, Carlo Bolz wrote:
> How can I access an embedded resources.

If it's a normal .NET resource (e.g. `CSC /resource:RESOURCE ...`), then you'd use the norma .NET mechanism, e.g. Assembly.GetManifestResourceStream().  If it's marked as an AndroidResource, you need to use the Android APIs, as Miljenko Cvjetko mentioned.

NET resources are still supported.

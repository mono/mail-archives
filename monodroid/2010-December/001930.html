On Dec 5, 2010, at 7:46 PM, Wally McClure wrote:
> What does mono.data.tds.dll do for me?

Mono.Data.Tds.dll is used by System.Data.dll to implement Microsoft SQL Server TDS protocol support for System.Data.SqlClient:

	http://docs.monodroid.net/index.aspx?link=N:System.Data.SqlClient

(i.e. Yes virginia, we should support TDS access to Microsoft SQL Server Out Of The Box...assuming you don't have some firewall in the way blocking TDS access to your DB...)

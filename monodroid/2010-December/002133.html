On Dec 24, 2010, at 2:58 PM, Nikolai Sander wrote:
> Will System.Drawing be part of the final release? It is part of MonoTouch so I wanted to know if it's part of MonoDroid too.

System.Drawing.dll was never part of MonoTouch.  There is a System.Drawing _namespace_ in OpenTK.dll which contains Color, KnownColor, Point, PointF, Rectangle, RectangleF, Size, and SizeF.

The same is true in MonoDroid -- OpenTK.dll contains a System.Drawing namespace with those types.

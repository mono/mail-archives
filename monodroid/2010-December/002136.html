This could happen if there are problems with forwarding ports for the
debugger - the android server doesn't tend to report problems. Does it
happen every time, or just the first time? Does the application start
normally, or hang waiting for the debugger to connect? Did you already have
an older version of the shared Mono runtime on the device, and if so, have
you uninstalled it so that the P10 version can be installed?

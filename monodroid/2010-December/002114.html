Jorge,

When you run (Ctrl-F5) or debug (F5) an application, the runtime is deployed
to the device if it is not already there.

To remove it from the device, go to:
Settings -> Applications -> Manage applications -> MonoDroid Runtime ->
Uninstall

Best regards,
Dan

I was able to connect to a remote sql server 2008 database and run "select * from table."  This is provided by Mono.Data.Tds.dll which is a part of Monodroid.  IIRC, the dataset support is provide in the assembly and within the System.Data namespace.  I think the sql server support is labeled as experimental.
 
As for other remote databases, I read that firebird has an assembly that runs in monodroid and monotouch.  As for other databases, I am not aware of any support for them.  I wonder if there might be some type of layer somewhere that would work through JDBC drivers.

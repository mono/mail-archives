I can't talk about the app. But it's not a desktop app ported to a mobile.
It was envisioned, designed, and built already as a mobile app, for
another mobile platform.

Haven't touched it in MonoTouch as of yet. I hope to at some point. Likely
will just be able to use the stuff we did for MDR, since it's the same
profile.

It calls SQLite directly, no java layer.

FAK's ORM is great, I use it, I love it, but it's nowhere near as powerful
as nhibernate. It's usefulness is limited to small, non complicated
schemas with straight forward mappings.

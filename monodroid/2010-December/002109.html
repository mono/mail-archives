Hello,

- Make sure you are using Preview 9.
- Make sure you have the correct SDK tools installed:
   - Android SDK Tools, revision 8
   - Android SDK Platform-tools, revision 1
- Check that there is an adb.exe in C:\android-sdk\platform-tools.

Jonathan

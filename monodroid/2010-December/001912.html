It's ok to install MonoDroid before the Java SDK and Android SDK, it 
doesn't check for them on install.

Because you installed while VS was running, the project and item 
templates probably won't show up.  You can fix this with 
uninstall/reinstall or by running:

devenv.exe /setup

(with no copies of VS running)

This will prompt VS to rebuild the template cache.

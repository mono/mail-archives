Onderwerp: Re: [Monodroid] pass params to activity

I use the Intent.PutExtra() in the current activity and Intent.GetExtra() in the new activity...

Anyone know of anything better?  I don't know how to pass actual objects, this just allows basic items


current activity

        private void listView_ItemClick(object sender, ItemEventArgs e)
        {
            Intent intent = new Intent(this.BaseContext, typeof(ListItem));
            intent.PutExtra("selected_list_item", e.Position);

            StartActivityForResult(intent, 1);
        }


new activity

        private int SelectedListIndex
        {
            get { return Intent.GetIntExtra("selected_list_item", 0); }
        }

The focus has been on getting something on MacOS for the existing
MonoTouch base to use.  I think we are probably closer to an MD addin
for win32 than we are to a fullblown linux product since the VS addin
did most of the heavy lifting for us on the packaging front.  

Some of us on the development team have been helping to dogfood the MD
support on linux, and things are looking good there.  Unfortunately,
it's a lot of work to go from a few hackers building from source to a
full-blown installer for linux.  I'm not sure how many people are plying
us with dumptrucks full of cash for a linux product, either.

Mike

> > On Fri, Dec 24, 2010 at 6:28 AM, Michael Hutchinson
> > <mhutchinson at novell.com> wrote:
> >         We are pleased to announce a preview of MonoDevelop support
> >         for MonoDroid on Mac, based on MonoDroid Preview 10.
> >         
> >         To install it, please follow the following steps:
> >         
> >         1. Ensure MonoDevelop is fully updated to build 20401003.
> >         This is the version available on the MonoDevelop updater.
> >         
> >         2. Update the Mono Framework to 2.8.1 from
> >         http://www.go-mono.com/mono-downloads/download.html
> >         
> >         3. Install the MonoDroid SDK from
> >         http://go-mono.com/monodroid-download/monodroid-preview-r8978.pkg. The username and password are the same as for the Windows Preview 10:
> >          Username: monodroid-preview
> >          Password: jsuj96Gh
> >         
> >         4. Install the Android SDK Version 8 from
> >         http://developer.android.com/sdk/index.html
> >         
> >         5. In the Android SDK tool, install the "platform tools" and
> >         at least one platform API version. If desired, configure an
> >         emulator image. This is similar to the Windows installation:
> >         http://monodroid.net/Installation
> >         
> >         6. From the MonoDevelop Addin Manager, install the MonoDroid
> >         addin from the Alpha repository. This is like the install
> >         process for MonoMac:
> >         http://www.mono-project.com/MonoMac#First_Time_Installation
> >         
> >         7. In MonoDevelop Preferences->Other->MonoDroid SDKs, set
> >         the location of the Android SDK.
> >         
> >         8. You can now create, build, run and debug MonoDroid
> >         projects on your Mac.
> >         
> >         Enjoy!

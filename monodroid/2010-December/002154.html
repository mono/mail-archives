In a lot of cases, Java-style setXXX and getXXX functions have been replaced
by properties to line up with the .NET way of doing things.  A good rule of
thumb is that whenever the Java API says getXXX/setXXX, first look for a
property named XXX.  (I can't look at the ListActivity class right now but
I'm guessing that's the case here).

- Greg

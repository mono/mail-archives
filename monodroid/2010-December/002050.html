Oh Byron,

One more thing is if you are a professional .Net developer then Monodroid is
the obvious choice as you would already be familiar with the stack and
working with importing library's.

In effect I prefer working with Monodroid to the Eclipse environment on
anything overtly complex. The small overhead of the runtime gives far more
flexibility in coding than plain Java based development imho.

Hi David,

I had the same Problem/Error-Message. In VS2010 select "Tools -> Options 
-> Monodroid -> Settings". There you can enter the path to the Android SDK.
After entering the path you need to reboot VS2010. Then double check if 
the path is correct.

That solved the problem on my side...

Good luck,
Oliver

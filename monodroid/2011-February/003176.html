After thinking this over, I believe that a lot of the information you would
be looking for in go to definition could be displayed easily displayed in 
tooltips in the IDE by putting the information into XML documentation 
comments for the types generated in Resource.Designer.cs

I had already filed an enhancement request to include more information 
in the Resource.Designer.cs XML comments.  I have gone back and added
some more ideas based on this suggestion:
https://bugzilla.novell.com/show_bug.cgi?id=665396

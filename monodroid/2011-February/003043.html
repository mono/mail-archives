I've used Sqlite on the local Android system through ADO.NET and the namespace for Sqlite.  I've also connected up over the Mono.Data.Tds namespace and run some simple queries against a sql server database.  I've read where someone has connected to firebird as well.
 
If you want to connect to other databases, you need to do something else.  I've been connecting up over Json and that works, but you have to build lots of code to do it.

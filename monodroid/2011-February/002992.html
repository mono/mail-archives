We have the same issue, but it looks like you are a bit further down the
pipe than us.  

I suspect with .NET now becoming a viable platform for iOS,Android (and
clearly WP7) more and more people will be looking to build more commonality
beyond the language and the core .net libraries.  

I've not seen any projects as of yet, but I do know that Robert Kozak has
started on an implementation of MVVM for MonoTouch
https://github.com/RobertKozak/MonoTouch.MVVM .  this may have some
applicability as an approach for Android.

There are probably also options to improve the user design experience for
these different platforms - I did wonder how difficult it would be to use
expression blend to generate XAML (with dummy controls / views) and then us
a T4 template to convert this XAML to code to create the UI. I've not got
around to looking at this in great detail.

If you find a project let us know ; either way I'm happy to contribute from
here to improve the Mono/C# environment.

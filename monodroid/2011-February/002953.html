You can specify the callback either by an anonymous delegate (which is what
you're doing there with the delegate keyword, except you're not using the
arguments), or by specifying an event handler for it. The mock callback you
specified is also using the wrong signature - looks like that one is for a
TimePickerDialog.

Here is how to do it with an anonymous delegate:

new AlertDialog.Builder(this)
    .SetTitle("select airfield")
    .SetItems(Resource.Array.fields_array, (sender, args) =>
    {
        Toast
                .MakeText(this, args.Which.ToString(), ToastLength.Short)
                .Show();
    })
    .Show();

---

Here is how to do it by specifying a function as the callback:

new AlertDialog.Builder(this)
    .SetTitle("select airfield")
    .SetItems(Resource.Array.fields_array, new
EventHandler<DialogClickEventArgs>(alertItemClickHandler))
    .Show();

private void alertItemClickHandler(object sender, DialogClickEventArgs args)
{
    Toast
        .MakeText(this, args.Which.ToString(), ToastLength.Short)
        .Show();
}

---

In those, args.Which gives you an index into the array, so you can use that
to retrieve the selected value. Hope this helps!

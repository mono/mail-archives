There are some samples that may help you out:

https://github.com/mono/monodroid-samples

GLTriangle and GLCube are based on the game loop provided by OpenTK.

JetBoy and Snake are based on the Java.Lang game loop.

There currently aren't any examples using a .Net threading game loop.

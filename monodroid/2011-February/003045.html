Hey George,

Thanks for coming to my session, and for letting me know about this! I will
take a look at the issues more in depth tomorrow, but I have a few quick
observations:

1) You mentioned that you installed the 2.2 SDK. If I'm not mistaken,
MonoDroid actually requires v2.3 now. It looks like the documentation for
the install page (http://monodroid.net/Installation) is out of date with
that, so maybe one of the MD guys can go in and fix it? :)

2) I'm assuming that you installed the latest preview (Preview 12, build
9381). This got released the night before the talk so I haven't tested the
samples on it yet. It looks like some things might have changed in this
release to cause those build errors, so I'll look into updating my samples
for P12 tomorrow. If you're impatient, you could install Preview 11.2 (build
9301) in the meantime, but I'm guessing it won't be much effort for me to
get things updated. Older preview versions are available here:
http://go-mono.com/monodroid-download/archive/

3) I'm not sure the mailing list is the best forum for discussing this, so
I'll continue this with you privately to get you up and running.

Thanks again for the feedback!

- Greg

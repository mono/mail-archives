On Feb 18, 2011, at 6:42 AM, Gustavo Carlos wrote:
> I'm starting to develop for Android in C # with Monodroid.
> When I create a new project, and compile the program returns the following error:
> "MonoDroid does not support running the previous version. Please Ensure, builds your solution before running or debugging it."

My guess is that you're using the Android SDK r8 release; you need to use the r9 release. Download the updated SDK at:

	http://developer.android.com/sdk/index.html

If on Windows, make sure you download the .exe installer and not the zip, as the .exe creates some registry entries that MonoDroid uses to lookup the SDK location.

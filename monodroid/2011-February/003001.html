Hi to all,

to solve this problem I use this:

                    AlertDialog.Builder builder = new AlertDialog.Builder(this);
                    AlertDialog alert = builder.Create();
                    alert.SetMessage("Message to the users?");
                    alert.SetButton("Yes", delegate { this.Finish(); });
                    alert.SetButton2("No", delegate { });
                    alert.Show();

In this way is the user press "Yes" the dialog is closed, otherwise no ....
Obviously you can place any code in the delegate ....

This can help you?

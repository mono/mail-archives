You can start the emulator from the command line.

MOTODEV Studio allows you to specify settings from their Eclipse plugin.
However, you can start the emulator from the command line:

[path-to-sdk]\tools\emulator -avd XOOM -scale 0.75

where XOOM is the name of the device image.

>From VS, you can select the already running emulator when you debug.

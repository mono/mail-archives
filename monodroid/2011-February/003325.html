On Feb 25, 2011, at 5:24 PM, Paul Camacho wrote:
> I understand in VS2010 you can set an option to turn off compression for assets and resources that get packed in the APK.
> 
> Is there a way to do this using MDr on Mac OS?

Project Properties -> Build -> MonoDroid Build -> Extra arguments textbox, then enter:

	-0 .file-extension

(that's dash-zero, not dash-cap-O.)

So not quite as nice as VS, but still possible.

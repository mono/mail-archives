Hi John,

The ManagedQuery() method returns an ICursor, not a list. You have to
iterate over the cursor row by row to retrieve the data.

Try this:

var contacts = ManagedQuery(ContactsContract.Contacts.ContentUri,
null, null, null, null);
if (contacts.MoveToFirst())
{
    do
    {
        int nameColumn =
contacts.GetColumnIndex(ContactsContract.ContactsColumnsConsts.DisplayName);
        string st = contacts.GetString(nameColumn);
        Toast.MakeText(this, st, ToastLength.Short).Show();
    } while (contacts.MoveToNext());
}

Best regards,
Dan

On Feb 1, 2011, at 2:59 PM, Justin Creech wrote:
> Does someone have an example of how to set the text in a TextView?

The trivial example is:

	TextView tv = new TextView (this); // assuming 'this' is e.g. an Activity
	tv.Text = "Hello, world!";

A more complicated example can be found in ApiDemo [0]:

	// Programmatically retrieve a string resource with style
	// information and apply it to the second text view.  Note the
	// use of CharSequence instead of String so we don't lose
	// the style info.
	IEnumerable<char> str = GetText (Resource.String.styled_text);
	TextView tv = (TextView)FindViewById (Resource.Id.text);
	tv.Text = str;

In the above, the Resource.String.styled_text [1] is the value:

	<string name="styled_text">Plain, <b>bold</b>, <i>italic</i>, <b><i>bold-italic</i></b></string>

However, (again) ToString() will "skip"/lose any extra encoding information:

	Log.Debug("demo", tv.Text);

prints "Plain, bold, italic, bold-italic", so you don't want to use += on TextView.Text unless you really mean it. :-)

 - Jon

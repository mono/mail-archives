On Feb 5, 2011, at 8:46 AM, Jeffrey Alstadt wrote:
> What features will be supported for the production release?

It'll look like Preview 11.2 + bug fixes + API breaks. We're currently only targeting `android.jar`, though Android.Runtime.JNIEnv should be usable for some interop scenarios. The basic feature set has largely been in place since Preview 10 (binding of android.jar, tooling, linker), and the GC is the last remaining major feature (and the inevitable bug fixes and API binding improvements).

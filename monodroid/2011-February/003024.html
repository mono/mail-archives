Since MonoDroid mimics the same namespaces, classes, and mechanisms from
Android development, you have the same tools for databinding that you would
find in the Android docs.

Check out the docs on Adapters and Adapter views for more info:

http://developer.android.com/guide/topics/ui/binding.html

<http://developer.android.com/guide/topics/ui/binding.html>They are a pretty
nice way of handling things.  Feel free to ping me directly if you have any
issues.

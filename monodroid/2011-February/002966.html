On Feb 9, 2011, at 3:27 PM, Aaron Knabb wrote:
> Can I ask what you mean by deriving from Java.Lang.Object?

In general, any time you need to implement IJavaObject, you should instead inherit from Java.Lang.Object.  See also:

	http://monodroid.net/Documentation/Architecture/Android_Callable_Wrappers

Thus, instead of:

	public class PageAnimationListener : IAnimationListener {
		// ...
		public IntPtr Handle { get; set; }
	}

do:

	public class PageAnimationListener : Java.Lang.Object, IAnimationListener {
		// do NOT implement IJavaObject
	}

 - Jon

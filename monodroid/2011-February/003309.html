On Feb 24, 2011, at 4:08 PM, Chris Erickson wrote:
> I'm doing a presentation next week at a developers conference on using
> monotouch and monodroid, it would be *really* nice to actually be able to
> run :-).
> 
> Can you give me any clues as to what I need to do to debug on mac?  I.e.
> What are the filename limitations...?

It's not so much a filename limitation as a file pattern limitation. The Add-in expects filename A when filename B is the one actually created, so things go south.

Fortunately, it looks like Preview 13 fixes most of these issues (though you may need to Clean before you Build, and/or kill the bin directory), and Preview 13 should be out soon. :-)

As a workaround in the mean time, create a new (empty) project and see if that works (often it does), then move code into the new project...

Hello,

Instead of LibraryProjectZip, you might rather want to add 
"project.properties" file (of LibraryProjectProperties build action) 
into your binding project, which pulls the resulting resources from the 
relevant Eclipse project directory.

(It will pick up resources from "bin" directory, so make sure to build 
eclipse projects every time you add or remove resources.)

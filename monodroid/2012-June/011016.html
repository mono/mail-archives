http://www.droiddraw.org/ is a good place to start with trying to create your
layouts for the different activities.

Take a look at http://docs.xamarin.com/android/tutorials/User_Interface <--
for Layouts and
http://docs.xamarin.com/android/tutorials/User_Interface/list_view <-- for
ListView example (very basic)

You can do what you want (simple colored rectangle on left and text on
right) with linear layout. The one layout (row.axml) will be used by the
Listview code to be created as many items as you have it to populate.

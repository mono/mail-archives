On Apr 30, 2012, at 2:03 PM, Saeid Yazdani wrote:
> Sorry if this is a noob question. How can I find the current installed version of mono for android?

Find mandroid.exe [0], then within a console run:

	mandroid --version

> I heard about a new version with designer abilities I want to check if it is the version that I downloaded or not.


The designer abilities are within MonoDevelop, not the Mono for Android SDK; knowing the Mono for Android version won't help.

 - Jon

[0] On OSX, use /Developer/MonoAndroid/usr/bin/mandroid.

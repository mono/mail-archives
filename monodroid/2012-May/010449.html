Hello,

If you have a look at the build output, there should be many lines that 
warn unknown types such as:

         Unknown return type 
java.util.concurrent.Future<java.awt.image.BufferedImage> found in 
method Apply in type Org.Imgscalr.AsyncScalr

java.awt.image is used but it does not exist in Android, so I don't 
think this library supports Android.

Atsushi Eno

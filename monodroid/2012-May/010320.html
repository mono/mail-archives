Within Visual Studio:

1. Click Tools > Options
2. In the left tree view, select Mono for Android > Settings
3. In the right, is the [ ] Preserve data/cache between application deploys checkbox selected?

If the checkbox isn't selected, select it, click OK, and let me know if that fixes it.

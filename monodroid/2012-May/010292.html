Hello,

One thing that can potentially cause this is if you have the Android 
preview for the HONEYCOMB api level that was released before Honeycomb 
was made final.

ie: if you have a directory like this in your Android SDK, delete it:

...\android-sdk\platforms\android-honeycomb

There are a few other issues that can cause this that cannot be worked 
around. We should release an update shortly that fixes these issues.

Hope this helps!
Jonathan

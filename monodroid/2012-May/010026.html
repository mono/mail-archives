> Because of designer I'm using MonoDevelop more and more and came accross deploying/licensing question again.
> 
> According to
> http://support.xamarin.com/customer/portal/articles/177037-can-i-use-my-license-on-more-than-one-machine-
> 
> It is a bit unclear whether the same license can be used from VS and MD...

Registration is done per-machine+OS:

	http://docs.xamarin.com/android/getting_started/installation/offline_activation#Offline_Activation_on_Windows

Note that the license file is within %ProgramData%, so it will be shared between MonoDevelop and Visual Studio, along with any other users on that machine. On OS X, it is per-user.

Note that registration is not shared across OS installations on a single machine, e.g. between an OS X and BootCamp Windows installation. This would be detected as two separate installations.

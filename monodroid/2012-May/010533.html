I had the same problem. Downloaded the setup.exe to install Mono for Android.
It scanned my system and said I had everything installed (which I didn't)
and closed.  I did a few things, I'm not sure which one fixed my problem
though.  First I had an open instance of visual studio. I closed that before
running the setup.exe. Next I installed MonoDevelop and GTK# manually.
Finally I launched setup.exe as administrator.  This time the scan came back
correctly and installed the Mono SDK and updated Java for me.  I hope this
helps.

On May 15, 2012, at 1:05 PM, Steve Sharrock wrote:
> I use TextView.TextColors.DefaultColor property which is an int.

Darn, I missed a member. :-/

> I've been using TextView.SetTextColor to restore this "int" default color. Can the int from DefaultColor be converted to an Android.Graphics.Color easily? 

Yes, use the Color(int) constructor:

	http://androidapi.xamarin.com/?link=C%3aAndroid.Graphics.Color(System.Int32)

To go from Color to an int, you can use the Color.ToArgb() method or the implicit Color->int conversion operator.

Hey everyone,
I just wanted to let everyone know about a book I recently wrote that was just published by O'Reilly: Mobile Development with C#. As the name implies, it's an intro to MonoTouch, Mono for Android, and Windows Phone. It goes into the basics of each platform, some approaches/techniques for sharing code across them, and then how to do common tasks such as storing data, accessing the network, etc. 

The e-book was made available yesterday in O'Reilly's shop, and the print version will be shipping shortly. I'm told the Kindle version will be up within a couple days as well.

http://shop.oreilly.com/product/0636920024002.do
http://www.amazon.com/Mobile-Development-Building-Android-Applications/dp/1449320236/

> I have a problem in the run android virtual machine.
> I cannot from android virtual machine the webview connect network.
...
> But i just cannot from app the webview show webpage.

This is usually because the emulator doesn't have networking enabled. You can double check this by launching the Browser app within the emulator and trying to view a web page such as google.com.

If Browser cannot connect to a URL, you need to enable networking. F8 will toggle network connectivity within the emulator.

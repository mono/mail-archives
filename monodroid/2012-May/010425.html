I just ran through an interesting install sequence.  This is on my system that I use for video recording, demos, and such for MfA.  This system is running Windows 7 x64.  I am having problems with getting VS to recognize the emulator, so I have switched over to MonoDevelop.  MD recognizes the emulator.  The emulator session is running Android 2.3.  When I start the first deployment of the application, after the installation of the Mono framework, MD reports an error saying that the installation failed at some point.  If I go into the settings, I can see that the framework has been installed.  I then restart the deployment of the application, the device framework is then installed.  I then get a message that the installation of the device framework has occurred.  I go into the emulator and see that both the mono framework and the device framework have been installed.  I deploy the application a third time and the application deploys and runs.

Yeah, I know, strange problem.  Thought I would see if anyone had heard of this.

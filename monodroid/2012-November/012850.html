On Nov 15, 2012, at 6:02 AM, Narcís Calvet <narcis at steema.com> wrote:
> I have found a new issue in Mono for Android 4.2.8.

This is a bug; it will be fixed in the next version: https://bugzilla.xamarin.com/show_bug.cgi?id=7934

In the meantime, you can workaround the issue by setting the ApplicationAttribute.Name property: https://bugzilla.xamarin.com/show_bug.cgi?id=7934#c3

	[Application(Name="my.App")]
	public class MyApplication : Application {
		// ...
	}

Okay seems we do have a fix for this after all.  The issue is because the
Java interface TrackerHandler isn't visible. The fix is to add a metadata
fix-up to expose this interface.Add this to your MetaData.xml :

<attr
path="/api/package[@name='com.google.analytics.tracking.android']/interface[@name='TrackerHandler']"
name="visibility">public</attr>

Then things will compile.  :)

(thanks to Chris Hardy for the tip)

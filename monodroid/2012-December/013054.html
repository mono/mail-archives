We are pleased to announce a stable release of Mono or Android 4.4.41. This release improves Android API 17 Jelly Bean support, changes the default ARM ABI to armeabi-v7a, and Android Binding Projects now embed .jar files into the binding .dll, simplifying their use in referencing projects.

Well, to be clear, I'm not sure that is what is tripping up the designer -
that is just a guess on my part. It might be worth-while to log a bug at
bugzilla.xamarin.com.

What Goncalo suggested is probably your best bet. This is a personal
preference, but you could also edit the layout files by hand as well, and
not worry about using the designer.

That said, adding controls programatically can be a bit tricky, but heres
an example of some code that would be in OnCreate:

var layout = new LinearLayout(this);  // this is the Actitivy;
var button = new Button(this);
layout.AddView(button);
var textView = new TextView(this);
layout.AddView(textView);
SetContentView(layout);

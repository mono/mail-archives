I dont have an example i can give you, but we are working as you deschribed.
For WinCE, Android, WP7 und iOS. The basic idea is to share the code
beetween different projects. Create one Visual Studio Project for each
platform you want to target. Then develop as usual in opf of the projects
and add all the files to the other project als link. Use only code that
works on both plattforms or in case this is not possible write a solution
for each platform and don't share the files between the two projects.

On Jul 19, 2011, at 5:45 AM, gabriel.bico at gmail.com wrote:
> Does the monodroid define any compilation diretive that I can use to difere in the linked files ?

The build process defines __ANDROID__ and a set of __ANDROID_X__ for X from 1 through the API level you're building against, inclusive (thus if you're targeting API level 4 -- Android 1.6 -- you'll have __ANDROID__, __ANDROID_1__, __ANDROID_2__, __ANDROID_3__, and __ANDROID_4__).

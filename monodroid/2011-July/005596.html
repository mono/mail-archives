The market definitely takes some time to properly update. I was able to link
to my app directly using the package name almost immediately, but it took
about a day for it to fully start showing up in searches for both the web
and devices. Give it some time and everything will start showing up.

Also, be aware that the stats in the developer console about your app update
slowly as well. The install count updates maybe a few times a day, but the
"active" install count updates once a day starting a few days in so you end
up with some skewed numbers. After enough time it ends up being ok, but it's
disorientating at first.

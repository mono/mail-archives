It ocurrs in Monodevelop? I got the same error also. The reason is because
GTK# installed Mono.Posix.dll version 2.0.0 in Modevelop installation, but
4.0.0 is required.

The solution is to install Mono 2.10.12 with GTK# and copy manually
Mono.Posix.dll to the folder /Local Settings\Application
Data\MonoDevelop-2.6\Addins\MonoDevelop.MonoAndroid.2.6.4/ and it should
work.

It's possible that you have to install Monodevelop 2.6 beta 1 (trunk build
2.5.90.3) instead beta 3, because you will get other error related to the
loading of the project.

At least it worked for me.

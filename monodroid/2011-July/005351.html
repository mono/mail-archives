The graphics are available as part of the SDK you installed. Also, as noted
by Google:

*Warning:* Because these resources can change between platform versions, you
> should not reference these icons using the Android platform resource IDs
> (i.e. menu icons under android.R.drawable). If you want to use any icons
> or other internal drawable resources, you should store a local copy of those
> icons or drawables in your application resources, then reference the local
> copy from your application code. In that way, you can maintain control over
> the appearance of your icons, even if the system's copy changes. Note that
> the grid below is not intended to be complete.


http://developer.android.com/guide/practices/ui_guidelines/icon_design_menu.html

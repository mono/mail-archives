DO NOT stick an Activity into a static variable -- that way leads to memory leaks.

If you need a static Context instance, stick an Android.App.Application into a static varaible, e.g. via the Context.ApplicationContext instance property:

	SomeStaticContext = this.ApplicationContext;

Alternatively you can use a custom Application and the [Application] custom attribute to store a static Context:

	[Application]
	class MyApplication : Application {
		public MyApplication(IntPtr handle)
			: base (handle);
		{
			Context = this;
		}

		public static Context Context;
	}

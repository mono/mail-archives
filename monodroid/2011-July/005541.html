Make sure the app your are building are targeting the same Android SDK (ie
2.2) as your emulator supports.

Also, the emulator takes ages to start, so make sure it is running when you
wanna test your app.

I start it and let it run, and when I build and run from monodevelop, I can
select it in the dialog box and continue.

Hope this helps.

Hi g4b0,<div><br></div><div>To get the latest Mono-for-Android Add-in for MonoDevelop, launch MonoDevelop and in the &quot;MonoDevelop&quot; menu (assuming you are on Mac?), select &quot;Add-in manager...&quot;. In the Add-in manager dialog window, click the &quot;Updates&quot; tab and it should be available. If it isn&#39;t, switch to the &quot;Gallery&quot; tab and select the &quot;Repository&quot; drop-down menu and select &quot;Manage repositories&quot;. In the dialog that pops up, make sure you have the &quot;Beta&quot; channel checked. Once you do that, close the &quot;Manage repositories&quot; dialog and in the &quot;Add-in Manager&quot; dialog hit the refresh button to the right of the &quot;Repository&quot; drop-down menu. This will refresh the add-in list below. Once that is done, you should be able to find the latest Mono for Android Add-in under &quot;Mobile Development&quot;. If you select that, there will be an &quot;Install&quot; button at the bottom-right side of the dialog.</div>

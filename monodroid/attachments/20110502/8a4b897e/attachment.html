<br>
</div>Use an intermediate wrapper type:<br>
<br>
        class ObjectRef&lt;T&gt; : Java.Lang.Object {<br>
                public ObjectRef (T value)<br>
                {<br>
                        Value = value;<br>
                }<br>
<br>
                public T Value { get; private set; }<br>
        }<br>
<br>
Then use this in your GetGroup() method:<br>
<div class="im"><br>
        public override Java.Lang.Object GetGroup (int groupPosition)<br>
        {<br>
</div>                return new ObjectRef&lt;DeviceGroup&gt;(Groups [groupPosition]);<br>
        }<br>
<br>
You will of course need to use ObjectRef&lt;DeviceGroup&gt; in other methods instead of DeviceGroup.<br>
<br>
You may also consider filing an enhancement request for a BaseExpandableListAdapter&lt;T&gt; which does the value wrapping for you.<br>

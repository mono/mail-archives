<div>I think the MWC example approach is probably suitable (as someone already pointed out), since using the built-in Android multi-select behavior for ListView is really all about the 'click' performing the 'selection' and not really having dual functions.</div>
<div><br></div><div>Having said that, I think you can force the ListView to do most of what you want (whether it is a good UI choice is another matter). Some things to try:</div><div><br></div><div>* To turn multiple selection mode on:</div>
<div><font class="Apple-style-span" face="'courier new', monospace">myListView.ChoiceMode = 2 // Multiple (or Android.Widget.ChoiceMode.Mulitple enum for API11+)</font></div><div><br></div><div>* To show the multiselect checkbox, use this as your cell layout (in your Adapter)</div>
<div><font class="Apple-style-span" face="'courier new', monospace">Android.Resource.Layout.SimpleListItemMultipleChoice</font></div><div><br></div><div>* To pre-select mulitple items when you create the list, call this as many times as required</div>
<div><font class="Apple-style-span" face="'courier new', monospace">myListView.SetItemChecked (1, true); // pass position in</font></div><div><br></div><div>* OnListItemClick still gets called (if you are using a ListActivity), in it you can access CheckedItemPositions:</div>
<div><font class="Apple-style-span" face="'courier new', monospace">var sparseArray = FindViewById<ListView>(Android.Resource.Id.List).CheckedItemPositions;</font></div><div><font class="Apple-style-span" face="'courier new', monospace">for (var i = 0; i < sparseArray.Size(); i++)</font></div>
<div><font class="Apple-style-span" face="'courier new', monospace">   Console.Write(sparseArray.KeyAt(i) + "=" + sparseArray.ValueAt(i) + ",");</font></div><div><font class="Apple-style-span" face="'courier new', monospace">Console.WriteLine();</font></div>
<div>---- note that the sparseArray contains an element for each item that had its value changed - only the ones where ValueAt(i)==true are _currently_ selected</div><div><br></div><div><br></div><div>HTH?</div><div><br></div>

    You have properties and/or events with the same name as class and
    you have to rename those<br>
    in Transforms\Metadata.xml<br>
    with following:<br>
    <br>
    Renaming class:<br>
      <attr<br>
       
    path=<a class="moz-txt-link-rfc2396E" href="mailto:/api/package[@name='org.achartengine.tools']/class[@name='Pan']">"/api/package[@name='org.achartengine.tools']/class[@name='Pan']"</a><br>
        name="managedName"<br>
        >PanWrapper</attr><br>
      <br>
    <br>
    Renaming event by adding eventName attribute:<br>
      <attr<br>
       
path=<a class="moz-txt-link-rfc2396E" href="mailto:/api/package[@name='org.achartengine.tools']/class[@name='Pan']/method[@name='addPanListener']">"/api/package[@name='org.achartengine.tools']/class[@name='Pan']/method[@name='addPanListener']"</a><br>
        name="eventName"<br>
      >Panned</attr><br>
      <br>
    <br>
    Renaming property by adding propertyName attribute:<br>
      <attr<br>
       
path=<a class="moz-txt-link-rfc2396E" href="mailto:/api/package[@name='org.achartengine.tools']/class[@name='Pan']/method[@name='addPanListener']">"/api/package[@name='org.achartengine.tools']/class[@name='Pan']/method[@name='addPanListener']"</a><br>
        name="propertyName"<br>
      >Panned</attr><br>
      <br>
    <br>
    To prevent binding of property/event - set the event/property name
    to the empty string.<br>

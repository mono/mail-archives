<div>

<p class=MsoNormal>Go to the Extension Manager, search the Online Gallery for
&quot;Project Linker&quot;.&nbsp;<br clear=all>
<o:p></o:p></p>

</div>

<div>

<p class=MsoNormal>So right when you start, with your empty project, create the
two, three, whatever platform-specific projects you need. All of them empty.
Add whatever references each one needs, etc.<o:p></o:p></p>

</div>

<div>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

</div>

<div>

<p class=MsoNormal>Now, on each of the &quot;secondary&quot; projects (the ones
that will have linked files), right-click, select &quot;Add project
link...&quot; and select the &quot;main&quot; project.&nbsp;<o:p></o:p></p>

</div>

<div>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

</div>

<div>

<p class=MsoNormal>From that point on, that extension will track item
add/rename/delete and replicate the action in the linked projects, but over
*linked* files :). Yes, that means it will preserve also the folder structure
as you create it on the main project.<o:p></o:p></p>

On Jul 26, 2011, at 10:46 AM, Tim Kelly wrote:<br>
&gt; Does anyone have an example of starting an application automatically?<br>
I've built an application that has a service but I need to start it<br>
automatically when the device starts. &nbsp;Can this be done using the Mono<br>
environment?<br>
<br>
Yes, this can be done. You need to create and register a<br>
BroadcastReceiver which listens for the Intent.ActionBootCompleted<br>
broadcast. You also need to have the<br>
Android.Manifest.Permission.ReceiveBootCompleted permission.<br>
<br>
Wally has a sample on this, iirc.<br>

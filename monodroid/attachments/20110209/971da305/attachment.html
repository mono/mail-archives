You can specify the callback either by an anonymous delegate (which is what you&#39;re doing there with the delegate keyword, except you&#39;re not using the arguments), or by specifying an event handler for it. The mock callback you specified is also using the wrong signature - looks like that one is for a TimePickerDialog. <div>

</div>Granted, it&#39;s using a WithCommand() helper, but the intent should be clear. For example, to get an item count:<br>
<br>
        // <a href="https://github.com/xamarin/monotouch-samples/blob/master/MonoCatalog-MonoDevelop/MonoDataSqlite.xib.cs#L211" target="_blank">https://github.com/xamarin/monotouch-samples/blob/master/MonoCatalog-MonoDevelop/MonoDataSqlite.xib.cs#L211</a><br>


        WithCommand (c =&gt; {<br>
                c.CommandText = &quot;SELECT COUNT(*) FROM [Items]&quot;;<br>
                var r = c.ExecuteReader ();<br>
                while (r.Read ()) {<br>
                        count = (int) (long) r [0];<br>
                }<br>
        });<br>

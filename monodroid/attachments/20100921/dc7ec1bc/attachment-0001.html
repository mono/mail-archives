Oh, before I forget, just realized I forgot to point out the &quot;(no authors)&quot; issue related to the authors file used by git-svn and svn2git (definitely go with svn2git) when dealing with Google Code (and other) SVN repositories which has rev 0 set as (no author).  You need to add an entry to the top of the authors file for the repository you&#39;re importing to avoid getting and &quot;Author: (no author) not defined in .. file&quot; error.<br>
</blockquote><div><br>I should have just written a script and posted that instead to ensure that all contributors and variations thereof (for some reason Google Code SVN repositories seem to have a lot of variations in usernames representing the same contributor) are found and a proper author file generated. You may want to edit the resulting file to better match the desired user name and email on git, but this provides the basic starting template:<br>
<br>PROJECTNAME=protobuf-net mkdir -p ~/.svn2git &amp;&amp; echo &quot;(no author) = name-of-replacement-for-no-author <a href="mailto:email-for-no-author@example.com">email-for-no-author@example.com</a>&quot; &gt; ~/.svn2git/authors &amp;&amp; for name in `svn log http://$<a href="http://PROJECTNAME.googlecode.com/svn">PROJECTNAME.googlecode.com/svn</a> | grep -E &quot;r[0-9]+&quot; | grep &quot;|&quot; | grep -v &quot;(no author)&quot; | awk &#39;{print $3}&#39; | sort | uniq`; do echo &quot;$name = $name &lt;$<a href="mailto:name@gmail.com">name@gmail.com</a>&gt;&quot; &gt;&gt; ~/.svn2git/authors; done<br>
<br>.. will generate ...<br><br>(no author) = name-of-replacement-for-no-author <a href="mailto:email-for-no-author@example.com">email-for-no-author@example.com</a><br>marc.gravell = marc.gravell &lt;<a href="mailto:marc.gravell@gmail.com">marc.gravell@gmail.com</a>&gt;<br>
xmlhacker = xmlhacker &lt;<a href="mailto:xmlhacker@gmail.com">xmlhacker@gmail.com</a>&gt;<br><a href="mailto:xmlhacker@gmail.com">xmlhacker@gmail.com</a> = <a href="mailto:xmlhacker@gmail.com">xmlhacker@gmail.com</a> &lt;<a href="mailto:xmlhacker@gmail.com">xmlhacker@gmail.com</a>@<a href="http://gmail.com">gmail.com</a>&gt;<br>

<br>Also, be aware that the stats in the developer console about your app update slowly as well. The install count updates maybe a few times a day, but the &quot;active&quot; install count updates once a day starting a few days in so you end up with some skewed numbers. After enough time it ends up being ok, but it&#39;s disorientating at first.<br>

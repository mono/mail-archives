    ARM specific details.<br>
    <br>
    Howto:<br>
    <ol>
      <li>grab androidx86 ISO (version 2.2) not sure something like 60
        MiBs</li>
      <li>install in VirtualBox</li>
      <li>boot AndroidX86 and disable Mouse Integration (VB menu)</li>
      <li>change port from 5555 to 5554 (mono4android)</li>
      <ol>
        <li>on VM (section with NATed VM change port 5555 to 5554)<br>
          <a href="http://www.android-x86.org/documents/debug-howto">http://www.android-x86.org/documents/debug-howto</a><a
href="http://stackoverflow.com/questions/2604727/how-can-i-connect-to-android-with-adb-over-tcp"><br>
              <br>
http://stackoverflow.com/questions/2604727/how-can-i-connect-to-android-with-adb-over-tcp</a>
          <br>
        </li>
      </ol>
      <li>connect developer box with VM<br>
      </li>
      <ol>
        <li>enter console Alt+F1 <br>
        </li>
        <li>cli&gt; netcfg<br>
          to see IP_of_VirtualBox_VM</li>
        <li>go back to UI Alt+F7</li>
        <li>on developer box: adb connect $IP_of_VirtualBox_VM:5554</li>
      </ol>
    </ol>
    I repeat it will break when installing shared runtime!<br>

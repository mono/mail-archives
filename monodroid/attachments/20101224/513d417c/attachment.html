<blockquote class="gmail_quote" style="margin:0 0 0 .8ex;border-left:1px #ccc solid;padding-left:1ex;">We are pleased to announce a preview of MonoDevelop support for MonoDroid on Mac, based on MonoDroid Preview 10.<br>
<br>
To install it, please follow the following steps:<br>
<br>
1. Ensure MonoDevelop is fully updated to build 20401003. This is the version available on the MonoDevelop updater.<br>
<br>
2. Update the Mono Framework to 2.8.1 from <a href="http://www.go-mono.com/mono-downloads/download.html" target="_blank">http://www.go-mono.com/mono-downloads/download.html</a><br>
<br>
3. Install the MonoDroid SDK from <a href="http://go-mono.com/monodroid-download/monodroid-preview-r8978.pkg" target="_blank">http://go-mono.com/monodroid-download/monodroid-preview-r8978.pkg</a>. The username and password are the same as for the Windows Preview 10:<br>

  Username: monodroid-preview<br>
  Password: jsuj96Gh<br>
<br>
4. Install the Android SDK Version 8 from <a href="http://developer.android.com/sdk/index.html" target="_blank">http://developer.android.com/sdk/index.html</a><br>
<br>
5. In the Android SDK tool, install the &quot;platform tools&quot; and at least one platform API version. If desired, configure an emulator image. This is similar to the Windows installation: <a href="http://monodroid.net/Installation" target="_blank">http://monodroid.net/Installation</a><br>

<br>
6. From the MonoDevelop Addin Manager, install the MonoDroid addin from the Alpha repository. This is like the install process for MonoMac: <a href="http://www.mono-project.com/MonoMac#First_Time_Installation" target="_blank">http://www.mono-project.com/MonoMac#First_Time_Installation</a><br>

<br>
7. In MonoDevelop Preferences-&gt;Other-&gt;MonoDroid SDKs, set the location of the Android SDK.<br>
<br>
8. You can now create, build, run and debug MonoDroid projects on your Mac.<br>
<br>
Enjoy!<br>

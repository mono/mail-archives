To have the same size (for a button for exemple) you have to use dp instead of px. Then your UI does not depend of the density.<br><br>But if you try to change the resolution, you'll find that your view is too long compare to the screen (or too small).<br>
To do this you have to create different layouts for different screen size (look at this link to understand <a href="http://developer.android.com/resources/dashboard/screens.html">http://developer.android.com/resources/dashboard/screens.html</a>)<br>
<br>And here is a sample for the recent android versions: <a href="http://developer.android.com/resources/samples/MultiResolution/index.html">http://developer.android.com/resources/samples/MultiResolution/index.html</a>.<br>
And here is the documentation: <a href="http://developer.android.com/guide/practices/screens_support.html#qualifiers">http://developer.android.com/guide/practices/screens_support.html#qualifiers</a>.<br><br>Hope it'll help,<br>

Sometimes it can take the emulator a while to start up (especially on older hardware). It&#39;s a general Android complaint, not specific to M4A. Try starting up the emulator first, and then when it is running run your application.  If you disable animations and scale down the size of the emulator it might load up a bit faster.<div>

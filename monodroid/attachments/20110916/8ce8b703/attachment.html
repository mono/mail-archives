<br>
</div>Add your file to the project, and set the Build action to &quot;Embed as resource,&quot; and optionally set the Resource ID.<br>
<br>
At runtime, you can do:<br>
<br>
        var destPath = Path.Combine (<br>
                        System.Environment.GetFolderPath(System.Environment.SpecialFolder.Personal),<br>
                        &quot;items.db3&quot;);<br>
        using (System.IO.Stream source  = System.Reflection.Assembly.GetExecutingAssembly().GetManifestResourceStream(&quot;resource name&quot;))<br>
        using (var dest = System.IO.File.Create (destPath)) {<br>
                source.CopyTo (dest);<br>
        }<br>
<br>
Note, however, that if the resource will be sizable, it&#39;ll likely take a noticeable amount of time. Don&#39;t do this in the UI thread. :-)<br>

My understanding is that classes that derive from Android/Java classes have to implement Android.Runtime.IJavaObject (<a href="http://docs.monodroid.net/monodoc.ashx?link=T%3aAndroid.Runtime.IJavaObject%2f*">http://docs.monodroid.net/monodoc.ashx?link=T%3aAndroid.Runtime.IJavaObject%2f*</a>) which defines the handle for object on the Java side. If you have your class inherit from Java.Lang.Object (<a href="http://docs.monodroid.net/index.aspx?link=T:Java.Lang.Object">http://docs.monodroid.net/index.aspx?link=T:Java.Lang.Object</a>) instead of IJavaObject you get that for free, since it implements IJavaObject for you.<div>

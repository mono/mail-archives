      <div dir="ltr">Hey Vladimir,<br>
        <br>
        I've got the Navigation Drawer working on 2.3 with both
        ActionBarSherlock and the support library ActionBar.<br>
        <br>
        If you are using ActionBarSherlock you just need need to
        subclass ActionBarDrawerToggle and provide an implementation of
        OnOptionsItemSelected that takes the ActionBarSherlock version
        of IMenuItem.
        <div>
          Fortunately it's pretty straightforward:<br>
          <div><br>
          </div>
          <div>
            <div>    public class MainActivityDrawerToggle :
              ActionBarDrawerToggle</div>
            <div>    {</div>
            <div>        private const decimal IdHome = 0x0102002c;</div>
            <div>        private readonly DrawerLayout _drawerLayout;</div>
            <div><br>
            </div>
            <div>        public MainActivityDrawerToggle(Activity
              activity, DrawerLayout drawerLayout, int drawerImageRes,
              int openDrawerContentDescRes, int
              closeDrawerContentDescRes)</div>
            <div>            : base(activity, drawerLayout,
              drawerImageRes, openDrawerContentDescRes,
              closeDrawerContentDescRes)</div>
            <div>        {</div>
            <div>            _drawerLayout = drawerLayout;</div>
            <div>        }</div>
            <div><br>
            </div>
            <div>        public bool OnOptionsItemSelected(IMenuItem
              item)</div>
            <div>        {</div>
            <div>            if (item != null && (item.ItemId ==
              IdHome || item.ItemId == Resource.Id.abs__home))</div>
            <div>            {</div>
            <div>                if
              (_drawerLayout.IsDrawerVisible(GravityCompat.Start))</div>
            <div>                   
              _drawerLayout.CloseDrawer(GravityCompat.Start);</div>
            <div>                else</div>
            <div>                   
              _drawerLayout.OpenDrawer(GravityCompat.Start);</div>
            <div>                return true;</div>
            <div>            }</div>
            <div>            return false;</div>
            <div>        }</div>
            <div>    }<br>
              <br>
              If you want to use the V7-AppCompat library I have some
              bindings here: <a href="https://github.com/jkells/xamarin-android-support-binding" target="_blank">https://github.com/jkells/xamarin-android-support-binding</a> there
              are instructions in the readme but you need to use the
              revision 18 version of the v4 support library from the
              component store and you need to build the v7-appcompat
              library in eclipse first.</div>

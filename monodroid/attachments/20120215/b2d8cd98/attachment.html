Are you doing this heavy lifting on the UI thread? You can't do anything like that there; the OS will kill you after a certain period of unresponsiveness. If you're doing it on a background thread, are you accessing the UI thread from your background thread without using RunOnUIThread? Mono for Android crashes if you do that without any warning (see the bug list). Also, the debugger has major issues with catching exceptions in background threads. You're best to put in some breakpoints there or move your code temporarily to the UI thread whilst resolving issues with it.<br>

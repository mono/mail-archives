Slightly longer answer: where is your convertView coming from? :-)<br>
<br>
Normally you&#39;re using LayoutInflator.Inflate() to create a new View, so only register the event when you create:<br>
<br>
        public override View GetView (int position, View convertView, ViewGroup parent)<br>
        {<br>
                if (convertView == null) {<br>
                        var inflator = (LayoutInflater) Context.GetSystemService (Context.LayoutInflaterService);<br>
                        convertView = inflator.Inflate (resourceId, null);<br>
                        convertView.FindViewById&lt;Button&gt;(Resource.Id.art_qte_minus).Click += art_qte_minus_Click;<br>
                }<br>
                // ...<br>
        }<br>

in which ViewHolder is:<br>
<br>
        static class ViewHolder {<br>

so this appears to be a design pattern rather than a concrete type, and the C# equivalent would be:<br>
<br>
        class ViewHolder : Java.Lang.Object {<br>

I had the same build problem &quot;could not find android sdk&quot; and figured out the problem.<div><br></div><div>It looks like the ResolveSDK task which resolves the Android SDK directory is currently hardcoded to the %ProgramFiles(x86)%\Android directory. This is not where my Android SDK is installed. To temporarily fix this on my machine I created a symbolic link from my existing SDK directory to the location where monodroid was expecting it to be.</div>
<div><br></div><div>The commands I used to do this on my machine look like:</div><div><div>&gt;mkdir &quot;%ProgramFiles(x86)%\Android&quot;</div><div>&gt;mklink /D &quot;%ProgramFiles(x86)%\Android\android-sdk-windows&quot; &quot;&lt;Your actual install directory&gt;\android-sdk-windows&quot;</div>

I wanted to create an app that would take advantage of API level 11, but still run in a reduced functionality mode down to API level 8. <div><br></div><div>In my projects settings, I tried setting "Application/Minimum Android to Target" to "Android 2.2" and "Android Manifest/Target API level" to 12 (closest available to 11).  Unfortunately, my application wouldn't compile since it was taking advantage of API level 11.  If I set  "Application/Minimum Android to Target" to "Android 3.1", it would compile but wouldn't run on an Android 2.2 machine.</div>

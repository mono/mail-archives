<p class="MsoNormal">If 2.3s is too slow for startup, would a splash screen work?<br>
<br>
Unfortunately we have no samples for what to do for custom splash screens, but it involves writing a Java activity (the &quot;real&quot; activity) which displays the splash screen (google for Java code) then launches the managed activity through the generated Android
 Callable Wrapper. You'll want to investigate e.g. obj/Debug/android/src to see the generated wrappers (for types &amp; names) and obj/Debug/android/AndroidManifest.xml (which also provides the types &amp; names you can use with Activity.startActivity).<o:p></o:p></p>
<div>
<p class="MsoNormal" style="margin-bottom:12.0pt"><br>
&gt; I've been running the app on a HTC Sensation and the startup speed isn't much better on that device.<o:p></o:p></p>
</div>
<p class="MsoNormal">If your app is taking far longer that 2.3s to launch, then you're doing to much work in your Activity.OnCreate() method. Try doing more work on a background thread.<br>

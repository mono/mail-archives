<div>Well, some weird news, all is now working. I did nothing except:

</div>
<div>I first tried updating PostSharp to version 2.1.7.6. This made no changes.</div>
<div>Then, I created the sample in VS 2012:  <a href="http://www.sharpcrafters.com/postsharp/documentation#getting-started">
http://www.sharpcrafters.com/postsharp/documentation#getting-started</a>. No work...</div>

<div>So I tried the VS 2012 project again. Still no go.</div>
<div>Then, the brainwave. The VS 2010 project works, so lets open that in VS 2012. That worked. The only difference was the project files and the VS version.</div>
<div>I assumed that as the same project file (the one from VS 2010) worked in both VS versions, then it must be the project file. Wrong! Both the code and the project file were exactly the same. The only difference was the project GUID - one line.</div>
<div><br>
</div>

<div>I opened the original project (the one with the errors). It now builds fine :)</div>
<div><br>
</div>
<div>Don't ask me what happened. I haven't touched that project since yesterday. Maybe it was the upgrade slowly kicking in or maybe it was the first build with VS 2010. Or maybe it was the incessant building and re-building. (Or the hand waving).</div>

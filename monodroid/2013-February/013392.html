I'm looking at your code. In the original you had a sharing violation 
exception. Pretty much this means you're trying to create a file that 
already exists. You are also not closing off the

StreamWriter stw = new StreamWriter (path, true);

line.

Golden rule - when you open something you have to close it.

The next set of code you're using the method that I suggested a while 
back. The retrieveXml method you have should do the trick, however, your 
save method is at fault still

By removing the StreamWriter line and changing the XmlTextWriter to

XmlTextWriter xtw = new XmlTextWriter (path, System.Text.Encoding.Default);

the code will compile and not fall over dead on each save.

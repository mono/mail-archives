On Fri, 2010-10-29 at 23:23 -0400, Mono-Droid at bobf.frankston.com wrote:
> Google TV is Android platform. Am I correct in assuming that Monodroid
> will work on it - at least eventually? What the issues?

It won't work on it now, but it should work eventually.  The main holdup is 
ABI support: Google TV is x86, not ARM, so we'll need to provide an
x86 mono build with the shared runtime/apps, or better determine how we want 
to support multiple ABIs.

In itself, this isn't difficult (we have it building), but afaik the x86 
Android build itself is unstable, so we haven't put any resources into 
actually testing it.

So the bust I see is that you're not running your UI related operations
(FindViewById, setting the Adapter) on the UI thread.  This is probably the
source of the crash.

However, once you fix the code to use RunOnUiThread() you'll notice it still
crashes.  Using RunOnUiThread from threadpool threads is currently broken in
Preview 6 so you're kind of screwed until Preview 7 comes out.

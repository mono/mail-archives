On Mon, 2010-11-01 at 14:45 -0400, Mono-Droid at bobf.frankston.com wrote:
> I presume the x86 is a run time issue or do you precompile?

Yes. :-)

libmono-2.0.so is a native shared library built specifically for the
target ABI, as it contains the GC, JIT (for runtime code generation),
and other essential services.  We currently ship with armeabi and
armeabi-v7a libraries; x86 support would require shipping a 3rd lib.

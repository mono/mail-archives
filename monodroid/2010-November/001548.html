What will the app store experience for the user be? Will the runtime come
down with my app, or will it need to be pre-installed?

 

You will be given an option between using the shared runtime (so the user is
directed on first launch to get the shared Mono runtime) or a fully embedded
Mono runtime in your application with both linking and stripping applied to
the result.

 

The compressed version for the embedded version (linking+stripping) should
be about 4 megs.   This is just from comparing the equivalent sizes with
MonoTouch.

 

You will never get to something like 24k as Android already ships the
JavaVM, but does no ship the Mono VM.   You do have to pay the price for
using the MonoVM in your app.

 

For simple apps like a simple ToDo app, I am not sure there is much value in
using Mono.

 

Miguel

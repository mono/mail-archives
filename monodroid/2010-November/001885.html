You might want to check the Android documentation for this:
http://developer.android.com/guide/topics/ui/menus.html

I think this refers to what you are trying to do:

Note: Menu items in the Icon Menu (from the Options Menu) cannot display 
a checkbox or radio button. If you choose to make items in the Icon Menu 
checkable, you must manually indicate the checked state by swapping the 
icon and/or text each time the state changes.

In preview 6 there is a generic BaseAdapter<T> class which removes the
need for you to use Java.Lang.Object.  It proxies the underlying GetItem
method as:

public abstract T this [int position] { get; }

So no J.L.Os are needed to use BaseAdapter<T>.

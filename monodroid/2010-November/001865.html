On Nov 25, 2010, at 1:51 PM, Willem Meints wrote:
> Today I installed preview 8.1 and got a fatal application crash. Created a clean emulator device and tried to run my application after a clean rebuild (with delete of bin and obj folder).
> It looks like the monodroid runtime service isn’t running.

The problem isn't that the runtime service isn't running; the problem is that the shared runtime hasn't been installed.  Please install the shared runtime, or rebuild your app so that it doesn't require the shared runtime (it's an option in Project Properties).

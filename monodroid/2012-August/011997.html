> hello, the emulator 14 is installed, but my applications do not work on it.

Are you able to install it? If not, within Visual Studio enable Tools > Options > Mono for Android > [X] Extension debug logging (writes monodroid.log to your desktop).

Attempt to install your app, and then attach monodroid.log to an email.

If using MonoDevelop/Windows, please provide:

	%LOCALAPPDATA%\MonoDevelop-3.0\Logs\AndroidTools-DATE.log

If using MonoDevelop/OS X, please provide:

	$HOME/Library/Logs/MonoDevelop-3.0/AndroidTools-DATE.log

If your app is installing but crashes, please provide the contents of the Android Debug Log:

	http://docs.xamarin.com/android/advanced_topics/android_debug_log

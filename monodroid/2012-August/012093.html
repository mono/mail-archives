A minor clarification: R.java is generated by MfA when you are building 
your *application*, and it is for every package for each library that 
needs R class.

This is how Android SDK deals with (Java) Android Library Project and we 
follow that manner (in 4.2.5; in 4.2.4 we were not really aware of that).

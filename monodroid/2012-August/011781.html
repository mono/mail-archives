Never mind, found it out myself.

Apparently you need to add android:icon="@drawable/icon" to the
<application> element in your AndroidManifest.xml file. So in the end it
looks like this:

<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android">
  <uses-sdk android:targetSdkVersion="15" />
  <application android:label="..." android:icon="@drawable/icon">
  </application>
</manifest>

I'd like to make a tiny followup on it: what has changed in 4.2.5 is, 
now we use -S option in Android aapt command. It takes a path to a 
directory that points to specific resources directory. To my 
understanding, it is also used in Eclipse ADT to handle multiple library 
resources (which I believe -S option is primarily designed for that 
purpose).

https://github.com/OESF/OHA-Android-4.0.4_r1.0/blob/master/sdk/eclipse/plugins/com.android.ide.eclipse.adt/src/com/android/ide/eclipse/adt/internal/build/builders/PreCompilerBuilder.java#L735

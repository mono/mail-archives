Base columns is an abstract type with no public/protected abstract members to override. It cannot be used as a base class:

	http://androidapi.xamarin.com/?link=T%3aAndroid.Provider.BaseColumns
	http://androidapi.xamarin.com/?link=T%3aAndroid.Provider.BaseColumns%2f*

It exists solely for "source-like" compatibility when porting Java code:

	http://docs.xamarin.com/android/advanced_topics/api_design#Interfaces

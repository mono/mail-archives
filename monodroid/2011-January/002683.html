On Jan 25, 2011, at 8:03 AM, Stefan Wywior wrote:
> I currently tried to launch the "HelloWorld" monodroid application on my 
> android 1.5 device, and it crashed.
> Is android 1.5 not supported by monodroid?
...
> [STDOUT] E/dalvikvm(  747): Could not find method 
> android.content.Context.getApplicationInfo, referenced from method 
> mono.MonoRuntimeProvider.attachInfo

This method is in API Level 4, which was added in Android 1.6:

	http://developer.android.com/reference/android/content/Context.html#getApplicationInfo()

Unless we can find an alternate mechanism of getting the package.apk for the launching app, this looks like a hard lower limit of Android 1.6.

However, there may be an alternate; ApplicationInfo.sourceDir is listed as API level 1:

	http://developer.android.com/reference/android/content/pm/ApplicationInfo.html#sourceDir

so perhaps there's some way to obtain this information using only API level 3 members.

Please file a bug so we don't forget this.

 - Jon

The current runtime includes *everything* in the framework and debug 
symbols.  Before we ship the final version, we will enable linking which 
will only ship exactly what your application requires instead of everything.

There will always be some overhead to using MonoDroid over pure Java, 
but it will be much smaller than the current ~35 MB.

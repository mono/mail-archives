wrote:

You mean where to change the build action?

 

All you need is to right click on your new xml file and select Properties.
Then on the top of the Properties window there is the Build Action property.
Change that to AndroidResource and build your project. Your new xml file
should now be added to the Resource.Designer.cs file.

ARMEABI is a compiler standard for ARMv6-based CPUs (ARM11, ARM9, etc),
while the ARMEABI-V7 is the standard for ARMv7-based CPUs (A8, Hummigbird,
etc). This will decide what phones can run your application. Like, if you
run armeabi compiler, the old phones (HTC G1, Hero, Legend, ZTE Blade,
Huawei phones) can run your app, while the armeabi-v7 compiler will output a
binary for the new phones (Samsung Galaxy S, LG Optimus 2X, Nexus S).

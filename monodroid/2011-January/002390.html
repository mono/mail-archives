I don't know of any books per se, but you can start by visiting the Android
Developers website.

Application Fundamentals:
http://developer.android.com/guide/topics/fundamentals.html

Samples:
http://developer.android.com/resources/samples/get.html. I'd recommend
downloading the samples since the have a lot of common use cases. To get the
most out of the samples, you'll need to set up the Eclipse environment as
well.

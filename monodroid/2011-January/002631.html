After installing and setting up the environment starting to deploying
a small test app, I see the following items as items that could be
greatly improved.

First, the deploy to device dialog is modal and blocking any access to
other parts in VS.NET while deploying... This could maybe be
implemented like the Windows Mobile deployment is done, in the
background using the Output window and the status bar to report about
progress etc allowing the developer to continue to code or what ever
in VS.NET while deploying.

Second, the time it takes to deploy a really small app (50 lines) is
about 5-7 minutes currently when I do it, is this currently a known
issue or have I missed out on something really important in the setup
/ configuration process?

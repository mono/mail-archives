To answer my own question...again, first you have to get the supported
preview format for your phone. This can be achieved by calling
p.SupportedPreviewFormats.

 

It seems that HTC Desire doesn't support Jpeg preview format as the list
only contains NV21 image format. I tried using the YuvImage.CompressToJpeg
method but it comes out really weird.

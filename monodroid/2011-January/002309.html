What that poll is missing is the scenario under which you are willing to 
pay what.

In example:

If i want to use Monodroid to write commercial software then the price 
from 300-500€ is still ok.

But if i only want to play at home with some ideas, then i am not 
willing to pay more then 50€ for it. But in this scenario it would be ok 
to have some limitations. In example that i could run my app only on 3 
devices (based on device id).

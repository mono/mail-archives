Hey all,

Another question:  Working w/ some content providers and kept getting a
permission error.  Found that, when using the properties window in VS the
following uses permission was being generated:

<uses-permission
android:name="android.browser.permission.READ_HISTORY_BOOKMARKS" />


The correct generation should have been:

<uses-permission
android:name="com.android.browser.permission.READ_HISTORY_BOOKMARKS"
/>

Pretty sure this is a bug.  Anyone else run into this?

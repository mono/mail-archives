On Dec 24, 2010, at 6:42 PM, Peter Garafano wrote:
> This is probably a dumb question, but I'm trying to add a google maps object to my android app but I cannot find any documentation on how to do it.

I believe that the maps API isn't part of android.jar, and thus can't be easily used at this time.  You can either write some custom .java code, or use Android.Runtime.JNIEnv to invoke the maps API (and/or your custom .java code).

Hi,

Cocos 2D is a game library / framework which has spun off three
different versions

* Cocos 2d, PC, (Python, the orignal one)

* Cocos 2d iOS (Objective-C)

* Cocos 2d Android (Java)

Even though API remains largely same, porting between different
platforms needs complete code rewrite due to incompatible programming
languages. Internally mobile ports use OpenGL ES and PC port uses
OpenGL.

Now, my question is, could it be possible to write yet another Cocos
2D implementation using .NET / C# and make this one codebase portable
through Android/iOS/OSX/PC less painfully?

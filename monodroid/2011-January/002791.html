On Jan 31, 2011, at 10:23 PM, Wally McClure wrote:
I'm running p11.2.  I'm getting the following error:
> on the following code:
> EditText ev = FindViewById<EditText>(Resource.Id.Name);
> string message = ev.Text; //error is here
...
> Shouldn't I be able to just use ev.Text? 

Use `ev.Text` to do...what, exactly?

Append to it?  This works:

	ev.Text += "\n\nSome Constant String";

If you need to get it into a System.String for "complicated" manipulations, you'll need to copy (via Convert.ToString(), or `new string(ev.Text.ToArray())`, etc.), but we expect the most common operation to be string concat, and IEnumerable<char> permits that without any additional effort.

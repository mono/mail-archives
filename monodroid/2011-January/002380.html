You can share some code, but the UI is all native - so Android vrs
CocoaTouch. You will be rewriting a load of code there, not to mention
rethinking how the UX works, as the phone's are different in how they
are used (back button, anyone?)

That said, if you wanted to use (for eg) Sqlite, it's about the same.
Same if you ported JSON.NET over to MT, it should work fine in MD as
well. But thats 100% Non-Visual.

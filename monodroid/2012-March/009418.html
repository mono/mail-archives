On Mar 22, 2012, at 11:03 AM, andyjohnson0 wrote:
> I was using AsyncTask<> beccause it is safe to access the user interface in
> the post execute method [1]. This this also true for the .net task library
> classes?

The TPL runs on threadpool threads, so NO, it isn't safe to access the user interface directly. So you invoke on the UI thread:

	Task.Factory.StartNew(() => {
		// ...
		Android.App.Application.SynchronizationContext.Post(() => { /* update UI element here */ });
	});

 - Jon

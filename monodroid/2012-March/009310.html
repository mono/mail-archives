On Mar 14, 2012, at 7:13 AM, mcleodia wrote:
> Can anyone tell me whether the mono profiler is available on Mono for Android?

For most intents and purposes we don't have a profiler yet. (It is present, but there's no proper way to interact with it short of environment variables, assuming there are any that control it. You can set the debug.mono.env system property to set environment variables [0].)

We will be providing this in a future release, though I can't provide a specific timeline.

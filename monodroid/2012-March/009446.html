Hi, 

Do you have Fast Deployment activated? Are you on VS or MD?

There were a few changes to the deployment process that may delay things a bit on devices version 3.0 or higher. It should only happen the first time you deploy, though, subsequent deploys shouldn't be affected.

The slowest part of the deploy process is sending the apk package. With Fast Deployment on, the only times that happens are when you 1) add or change resources 2) add new subclasses or override methods that require new java binding code to be generated.

On Mar 27, 2012, at 9:46 AM, javigu wrote:
> JNIEnv.CallVoidMethod(PruebaHandler_Class, PruebaHandler_B, new JValue(webView1));

The signature for JNIEnv.CallVoidMethod is:

	// http://androidapi.xamarin.com/?link=M%3aAndroid.Runtime.JNIEnv.CallVoidMethod(System.IntPtr%2cSystem.IntPtr%2cAndroid.Runtime.JValue%5b%5d)
	public static void CallVoidMethod(IntPtr jobject, IntPtr jmethod, params JValue[] parms);

However, the first parameter you're providing is PruebaHandler_Class, which is a java.lang.Class, hence the ClassCastException. You should instead be doing:

	JNIEnv.CallVoidMethod(instance, PruebaHandler_B, new JValue(webView1));

 - Jon

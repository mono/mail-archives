Hello Iain,

i had also some problems with memory and found the solution in the DDMS
(Dalvik Debug Monitor Server). This tool is included in the android SDK
under tools.

You can see there the current allocated memory and if you destroy your view
you can see some Objects which are not released and where is the function
which uses it.

More information about the DDMS by android developers:
http://developer.android.com/guide/developing/debugging/ddms.html

I hope it helps too.

Thanks. I created a brand new project, and it compiled fine. I tried tweaking
some of the settings in my existing solution (API level, etc) and couldn't
get it to work. Additionally, a brand new project within that solution with
no references also failed with the same problem.

Comparing with my previous machine, the only difference was the location of
the Android SDK (that machine had started from MfA 1.0, which didn't have
the fancy bootstrap installer). So I downloaded the Android SDK and let it
install to the default location (C:\Program Files\...). I repointed the MfA
settings to the new SDK location and everything works.

Thanks for the help.

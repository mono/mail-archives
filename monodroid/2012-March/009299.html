On Mar 12, 2012, at 6:00 PM, johnHolmes wrote:
> I've already developed an application which use MonoTouch and SQLite with FTS support.
> I was wondering if I can do the same in Android? Does M4A support SQLite?

Maybe. Mono for Android does provide Mono.Data.Sqlite.dll, just as MonoTouch does. The real question relates to the libsqlite.so included with Android. Only Android 2.2+ has libsqlite.so.

Fortunately, the AOSP (open-source) Android source _does_ include FTS support within libsqlite.so:

	https://github.com/android/platform_external_sqlite/blob/master/dist/Android.mk#L9

You'll need to test to confirm, but it certainly looks like Android 2.2+'s SQLite includes FTS support.

You probably need to switch to csharp-sqlite
http://code.google.com/p/csharp-sqlite/

It has SqlCipher built in
http://code.google.com/p/csharp-sqlite/wiki/ENCRYPTION

Xamarin's MWC Sample project shows you how to use csharp-sqlite
https://github.com/xamarin/mobile-samples/tree/master/MWC

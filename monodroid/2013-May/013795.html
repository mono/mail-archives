I had this same problem in Visual Studio, after a few months picked up a
project and debugging stopped working. Managed to solve it here without
creating a new project.

- Removed all mono related stuff from device (shared runtime and app's).
- Did a clean on the solution and removed all bin and obj folders.
- Enabled 'Use shared runtime' in project file (was disabled).
- Set linking to 'None'.
- Build and F5.

I think i changed the runtime en linking settings when releasing a beta to
some colleagues...

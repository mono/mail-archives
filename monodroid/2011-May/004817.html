> Also the test just echoes "activated" - what's it supposed to do?

That's weird.

`mandroid --activated` is used by Visual Studio and IDE to determine if the product has been registered/activated. If the exit value is 0, it's activated; otherwise it hasn't been. Using `mandroid --activated && echo "activated"` should only execute the `echo` if the exit value is 0, so I find this rather odd...

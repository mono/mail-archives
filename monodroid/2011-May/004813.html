On May 8, 2011, at 9:10 AM, milop wrote:
> I have an app called Tutorial1. When I run it without debugging it works
> fine. When I try to run it with debugging I get the following error (Note: I
> was able to debug the app all day and then this started happening):
...
> 05-08 09:05:13.852: ERROR/AndroidRuntime(9211): Caused by: android.content.pm.PackageManager$NameNotFoundException:
> Mono.Android.Platform.ApiLevel_9

Are you trying to build, without recompilation, a project from Preview 13 (or earlier)? Preview 14 dropped support for API level 9, replacing it with API level 10 (primarily because I couldn't even download API level 9 from Google anymore; API level 10 is Android 2.3.3):

	http://mono-android.net/Releases/Previews/Preview_14#API_Changes

Make sure that your API level (in Project Options somewhere) is API level 10, rebuild your project, and then try debugging.

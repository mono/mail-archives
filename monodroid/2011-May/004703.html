On May 5, 2011, at 10:10 AM, Neal wrote:
> I see the code from Jon and Craig on getting DeviceID etc.  I have a class
> named Utils that I use to centralize information for the app such as
> retrieving Device information.  When I use code such as Jon suggested it
> errors on the ContentResolver.  How do I resolve this?
> 
> var id = Android.Provider.Settings.Secure.GetString(ContentResolver,
> Android.Provider.Settings.Secure.AndroidId);

ContentResolver is Context.ContentResolver:

	http://docs.mono-android.net/Android.Content.Context.ContentResolver

You need to use that property within a Context subclass (Activity, Service, etc.) or use a Context variable, e.g. `context.ContentResolver`.

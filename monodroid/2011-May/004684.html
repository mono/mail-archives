On May 4, 2011, at 10:06 AM, tomschrot wrote:
> Im using WebView for creating an GUI in my Activity. 
> 
> The problem now is, that the HTML will use some CSS like
> 
> <link rel="stylesheet" type="text/css" href="Default.css" > 
> 
> thus the css file should be on a path somewhere. I cant find out how to integrate the css-files to my project and how the WebView can read them in?

As per [0, 1], you need to use a URL starting with `file:///android_asset`, which will read from your Assets directory:

	<link rel="stylesheet" type="text/css" href="file:///android_asset/Default.css">

 - Jon

[1] http://developer.android.com/reference/android/webkit/WebView.html#loadDataWithBaseURL(java.lang.String, java.lang.String, java.lang.String, java.lang.String, java.lang.String)

System.Json comes from Silverlight, so you can use the Silverlight documentation:

	http://msdn.microsoft.com/en-us/library/system.json(VS.95).aspx

To convert a JSON string into a DateTime, it looks like you'd use JsonValue.Load()[0] and then use the implicit conversion operator [1].

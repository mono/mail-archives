As a follow-up, I have a sample that demonstrates using a MonoTouch library from a Mono for Android app and using a Mono for Android library from a MonoTouch app:

	http://www.jprl.com/tmp/ScratchMonoMobile.zip

There are limitations (e.g. the MonoTouch library can't reference monotouch.dll to be used by Mono for Android, and the Mono for Android library can't reference Mono.Android.dll to be used by MonoTouch), and this only works for assembly references (NOT project references), but it does work (tested in MonoDevelop on OSX with Mono for Android and MonoTouch installed).

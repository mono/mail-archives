On Jul 20, 2012, at 2:12 PM, Tsofatse Emmanuel <itsofa.xxl at gmail.com> wrote:
> i'm wondering if there's any documentation offline for mono for android.

Yes. We ship two forms of offline documentation: Mono.Android.xml and monodoc files.

To read the Mono.Android.xml content, you need to use the Visual Studio object browser.

To read the mondooc files,  you need to use the monodoc help browser, which is included with recent versions of MonoDevelop (> 3.0.2; I'm not sure when exactly, but 3.0.2 didn't include it). Within MonoDevelop, click Help > Help.

- Jon

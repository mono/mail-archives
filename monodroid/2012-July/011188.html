Hi,

First, you should try intel x86 emulator. With it the shared runtime 
deploys within a minute on my core i5+hdd machine.

Our doc team created a dedicated instruction doc:
http://docs.xamarin.com/android/getting_started/installation/Configuring_the_x86_Emulator

Second, in case you didn't, turn on the fast deployment option in the 
project properties ("Mono for Android Build" section). With this option, 
only assemblies and resources that were updated should be uploaded.
I assume you have huge resources in the apk (unless you wrote highly 
complicated app that reaches 20MB). If you are embedding it in the 
assemblies, you might want to change it to load in different manner 
(file resources, android resources etc.).

On Jul 18, 2012, at 9:53 PM, Darren Abbott wrote:
> I am trying to make use of the X and Y components of the span in a ScaleGestureListener.  Per the Android and the Xamarin documentation, I should be able to query CurrentSpanX, CurrentSpanY, PreviousScanX, and PreviousScanY properties from a ScaleGesterDetector object.  However, those fields don't seem to exist -- VS2012's autosense doesn't provide them

What the documentation giveth the documentation taketh away:

	http://androidapi.xamarin.com/index.aspx?link=P%3aAndroid.Views.ScaleGestureDetector.CurrentSpanX
	http://developer.android.com/reference/android/view/ScaleGestureDetector.html#getCurrentSpanX()

Read the above links carefully, and you'll find that those members were added in API level 11. You need to be targeting Android v3.0 or later to use them.

 - Jon

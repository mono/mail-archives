Hello,

Aes.Create() returns null, but you could use new AesManaged() instead.
This worked:

                 Console.WriteLine ();
                 var ms = new MemoryStream ();
                 var cs = new CryptoStream (ms, new AesManaged 
().CreateEncryptor (), CryptoStreamMode.Write);
                 cs.Write (new byte[] {0, 1, 2, 3, 4, 5}, 0, 6);
                 cs.Close ();
                 Console.WriteLine (Convert.ToBase64String (ms.ToArray ()));

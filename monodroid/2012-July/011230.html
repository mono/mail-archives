Unfortunately you can't do that in current MfA. Only application project 
can handle it (to be fair Android Java tools had long history of not 
supporting library resources correctly either).
We plan to support that in our future versions, but no actual schedule 
yet. We will go ahead and work on it when other couple of issues gets fixed.

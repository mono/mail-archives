On Thu, 2010-10-14 at 19:52 +0400, Alexey Nesterov wrote:
> i've tried to build default project - with click count button and
> start it at the emulator - and everything works great. But how can i
> now deploy app on device? I've tried just to install apk - but got
> error "Application not installed" without any additional explanations.
> Does it require Mono runtime itself and if it does where can I get apk
> with Mono runtime?

Yes, it requires the Mono Runtime, which is included with MonoDroid --
install the MonoRuntimeService-debug.apk file located at:

        %VSINSTALLDIR%\Common7\IDE\Extensions\Novell\MonoDroid for Visual Studio\1.0.8305\bin

Then install your application.

Alternatively, you can build a "static" application:

        http://monodroid.net/Releases/Previews/Preview_5#Static_Runtime_Support

though this will result in larger application packages.

 - Jon

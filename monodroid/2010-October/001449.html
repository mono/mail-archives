> The connection works fine but an exception happens in the Close()

> System.EntryPointNotFoundException: sqlite3_next_stmt

That doesn't look like the Close() method. :-)

Regardless, the problem is that you're running this on a platform
earlier than Android 2.2.  Only Android 2.2 has a SQLite recent enough
for Mono.Data.Sqlite to properly interact with it (as previous versions
are lacking necessary exports, as shown by all the
EntryPointNotFoundExceptions).

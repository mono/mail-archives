example?

 

I found this code for Java

editText.addTextChangedListener(new TextWatcher() {

                        public void afterTextChanged(Editable s) {

                                //XXX do something

                        }

                        public void beforeTextChanged(CharSequence s, int 

start, int count,

int after) {

                                //XXX do something

                        }

                        public void onTextChanged(CharSequence s, int start,


int before, int count) {

                                //XXX do something

                        }

        });

 

Or this way by implementing an interface

public class TabChangeListener implements OnTabChangeListener {
        public void onTabChanged(String tabID) {
                // XXX do something
        }
}
 
TabChangeListener listener = new TabChangeListener();
tabHost.setOnTabChangeListener(listener);

 

 

Obviously .NET events are far easier to use if you can adapt them to
MonoDroid.

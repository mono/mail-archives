bug?  In the meantime...

> I see there's an Android listener. Can someone provide a simple code
> example? 
> 
> I found this code for Java
> 
> editText.addTextChangedListener(new TextWatcher() {

C# doesn't support anonymous inner classes, so you just need to
translate this example into a regular class:

        class MyTextWatcher : Java.Lang.Object, ITextWatcher {
                public void AfterTextChanged(IEditable s) {}
                public void BeforeTextChanged(Java.Lang.CharSequence, 
                                int start, int count, int after) {}
                public void OnTextChanged(Java.Lang.CharSequence,
                                int start, int before, int count) {}
        }
        
        // ...
        editText.AddTextChangedListener(new MyTextWatcher());

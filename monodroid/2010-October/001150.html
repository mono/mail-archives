I got this to work using this:
            button.Click += (s, e) => {
                Intent newScreen = new Intent(this, typeof(Screen2));
                newScreen.SetFlags(ActivityFlags.NewTask);
                this.StartActivity(newScreen);
            };

But that, couple with the previous conversation, begs the question, should this really be that hard?

Seriously, loading screens is an incredibly common task. Shouldn't we offer an overload that makes this much simpler? Or maybe a new way to do it altogether?

Also, how does the GC manage this new activity? Does it need to be declared at a class level somehow to keep a reference to it?

Hey all,

In some places, the Java API expects a date/time that is expressed in the number of milleseconds from 1970.01.01. .net has datetimes that start at 01.01.01ad

Here's an extension method to convert:

    public static class DateTimeExtensionMethods
   {
        /// <summary>
       /// Java starts their dates from 1970, this method returns the number of milliseconds
       /// from a given date in java milliseconds from 1970
       /// </summary>
       /// <param name="from"></param>
       /// <returns></returns>
       public static long ToJavaMilliseconds(this DateTime from)
        {
            return (long)(from - new DateTime(1970, 1, 1)).TotalMilliseconds;
        }

    }

I feel the same way.  I download a preview, find a show stopper, report bug,
then sit on my hands until the next release.

The only real downside i can see to this is people not keeping up on
versions, and reporting bugs that have already been fixed in a newer
nightly, however this probably happens to some extent now with people
duplicating bug reports, and realistically if people are searching for
similar bugs before posting a new one it shouldn't happen so much.

Also, packaging could be an issue, I don't know how much effort goes into
packaging a release.  If it's a lot, maybe even just a zip of the nightly
build of assemblies would be nice?

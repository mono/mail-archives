The Camera.Open(int) and Camera.NumberOfCameras properties were added in API level 9 (Android v2.3):

	http://androidapi.xamarin.com/?link=M%3aAndroid.Hardware.Camera.Open(System.Int32) 
	http://androidapi.xamarin.com/?link=P%3aAndroid.Hardware.Camera.NumberOfCameras

Notice the "Since: API Level 9" in their documentation. Thus, you need to set your TargetFrameworkVersion to v2.3.3 (API 10) or later in order to use these members.

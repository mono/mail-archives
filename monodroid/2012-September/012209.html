Hi all

Re: https://github.com/MarkEaton1/Monodroid-InApp-Billing

The above link is all the in-app billing code converted from Java to C#.
This is my first time using GitHub and I was surprised how easy it was.
Makes TFS look complicated ;-)

Anyway this has been tested with real money for managed in-app products
only. All the code is there for unmanaged products and subscriptions, it
just needs to be tested.

I have tested the following with managed in-app products:
- billing supported (have tested this part for subscriptions too and works)
- purchased
- cancelled
- restore transactions
- refund
- item unavailable
- database updated

This has been designed to work for Android 2.0 and up. However someone can
make it work for below if interested.

As I do this part-time outside of my regular full-time job I just wanted to
get this out there for the community.

The Activity file is rough and can certainly be made more elegant and user
friendly.

I don't much spare time on my hands, but if you can make the code more
elegant and user friendly please feel free to do so.

In fact doing so will be great for the MonoDroid community!!


Kind regards

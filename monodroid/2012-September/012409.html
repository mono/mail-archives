Hi,

      If you are interested only In Game Development using C# and
Mono, then you can also use

      Unity3D (It will cost the same or more as Mono Touch or Mono for
Android License) for each Platform.

      It is cross platform (PC/Mac/Native Client/Android/IOS) with
tons of tutorial and battle tested.

      Unfortunately, even though both use the same technology, you
cannot mix Mono Touch and Mono for Android even if you buy licenses
from both Xamarin and Unity3D.

      Only Mono Touch and Mono For Android allow easy access to the
Underlying Platform API.

      Choose based on the Use Case of your Application.

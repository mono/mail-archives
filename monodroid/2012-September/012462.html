got it. simply add permissions in:
public void Init(Activity activity, Facebook fb)
        {
            Init(activity, fb, new String[] { "publish_checkins",
"publish_stream", "read_stream", *"email", "user_birthday"* });
        }

here *new String[]{ } parameter* is for permissions; to get email add
"email" and to get birthday add "user_birthday" and so on.... I still have
to get the friend list.

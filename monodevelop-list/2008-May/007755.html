Using svn trunk and msbuild project files GTK widget libraries don't
work, no toolbox entries, widgets not found in existing widgets that use
them.  The attached patch fixes this by checking for
"gtk-gui.objects.xml" / "gtk-gui.gui.stetic" instead of just
"objects.xml" and "gui.stetic" as the resources are named when built
with monodevelop project files.

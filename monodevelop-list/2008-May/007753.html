>> As of MonoDevelop 1.0, the BooBinding used a version of GTKSourceView
>> that didn't work on the Mac. However, I think it'll work if you build
>> MD and the BooBinding from trunk.
>
> But it still requires GtkSourceView, right?


Yes. It's used for the boo pad, to simulate a shell but with syntax
highlighting. Hopefully this can be ported to the the MD managed text
editor, but that'd require work on the managed editor to support
uneditable regions. Alternatively, it could be ported to the gtk
textview with minimal effort, but would lose syntax highlighting in
the shell pad.

The database addin also requires gtksourceview, but I believe this
could be ported to the managed text editor with very little effort.

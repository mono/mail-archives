Hi:

Two issues in Monodevelop 1.0 on Hardy Herron 8.04:

When adding new files to the project, they work in the solution explorer
until reopening the project, then they only appear in the files tab and have
to be copied and readded to the project to be part of the solution again.  I
saw someone report this in the forum, but there were no replies. It makes it
tough when I have to readd a ton of class files.

Also, when I add a .dll reference it works as long as I use project->build.
If I use project->rebuild the reference won't be found and the .dll will
actually get deleted from whereever it was referenced from.

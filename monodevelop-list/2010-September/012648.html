Check the YUM repositories (RPM).  On Ubuntu you can install from the
command line with:

sudo apt-get install monodevelop

I believe it is there in the RPM repos too, so install with:

sudo yum install monodevelop

It may be named differently so run:

yum search mono

To see what all is available in your repo.  If all else fails, then download
the RPM from the MonoDevelop page.

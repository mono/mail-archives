One solution I've found to run recent Monodevelop on Ubuntu 10.04 or .10 is
to install VirtualBox, openSUSE 11.2 then add software repositories for mono
http://mono.ximian.com/monobuild/preview/download-preview/openSUSE_11.2/ and
monodevelop http://monodevelop.com/Download/Trunk_Builds. Do the 1click
install here
http://mono.ximian.com/monobuild/preview/download-preview/openSUSE_11.2/ and
everything gets a bit vague here... I futzed with some installs and forced
upgrades to 2.8 and now xsp4 is present!

monodevelop let me create a asp.net application, set a break point in the
code behind on a button push and run the application...

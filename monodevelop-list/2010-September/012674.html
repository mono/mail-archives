>
> Does someone know a way to build monodevelop from github without
> gladeui-sharp and the missing libgladeui error?

Properly install all the necessary packages.

Debian (and therefore also Ubuntu) break Mono and related stuff up
into many tiny packages. Although we'd normally expect Glade# and its
pkgconfig file to be installed with GTK#, on Ubuntu you probably need
to install a package called libglade-cil-dev or something.

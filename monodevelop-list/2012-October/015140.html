If you do not want to resort to the Branch Master, you can adjust the
solution file in visual studio 2012, changing a single line.

Basically you need to edit the file. Sln in a text editor and change the
following line:
Microsoft Visual Studio Solution File, Format Version 12:00

for
Microsoft Visual Studio Solution File, Format Version 11:00

This causes the Mono recognize and open the solution.

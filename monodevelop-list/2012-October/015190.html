>> I am relatively new to Ubuntu and MonoDevelop. Is it possible to install MD
>> 2.x and 3.x into the same machine in a side-by-side manner?
>
> If you install from source it is placed usually at /usr/local/bin. This
> does not conflict with the system installed version at /usr/bin. Start
> the version you need by using an absolute path.

Another option is to compile from from source and run from there and
do not install . This will allows you have more than 2 versions of MD.
I normally follow this process.

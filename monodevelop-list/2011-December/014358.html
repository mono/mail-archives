I had the same problem, but the circumstances were somewhat different. See
the post (towards the end)
"GTK# version for dotNet - MonoDevelop under Windows XP" in this
(Mono-MonoDevelop IDE) forum.

The circumstances seem different, but your error message is because you
installed "mono-2.10.8-gtksharp-2.12.11-win32-1.exe" . The MonoDevelop
installer then gives you the error message about the GTK# version number.
This is likely a bug, but I have not reported it formally. I don't think
there is a workaround.

Note that had you installed the 3 items for MonoDevelop. You would likely
have an error message about the Net Framework version. That is a recorded
bug.

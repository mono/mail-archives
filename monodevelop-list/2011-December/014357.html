Hello,

I am trying to install Monodevelop 2.8.5
(http://download.xamarin.com/monodevelop/Windows/MonoDevelop-2.8.5.msi)
under Windows 7 (Enterprise, x86) and am encountering the following
error when I click through the installer:

  "Gtk# version 2.12.9 or greater must be installed."

Before trying to install Monodevelop, I installed the following
packages, in order:

  Mono 2.10.8: http://download.mono-project.com/archive/2.10.8/windows-installer/1/mono-2.10.8-gtksharp-2.12.11-win32-1.exe

  .NET Framework 4.0:
http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=8279

  GTK# for .NET 2.12.10:
http://ftp.novell.com/pub/mono/gtk-sharp/gtk-sharp-2.12.10.win32.msi

(Note: These packages were referenced on the Monodevelop download page!)

As for the Mono installation, I DID NOT select the Gtk# installation.
The only Gtk# on my system was from ftp.novell.com as shown above.  I
also tried GTK# 2.12.9 from ftp.novell.com but the error during
installation was the same.

I double checked my registry settings with regedit and the following
exists in my registry:

  HKEY_LOCAL_MACHINE->SOFTWARE->Novell->GtkSharp
    - InstallFolder->C:\Program Files\GtkSharp\2.12\
    - Version->2.12.10

All of the directories and files noted above exist.

I'd Googled around a bit to see if anyone else had seen this error and
the consensus was that this error should only appear if there's an
issue with a non-existent (or incorrect version of) Gtk# for .NET.
OK, but since I installed everything from the Monodevelop site,
shouldn't this mean that things should just work for me?

Please let me know if you have any ideas how I can get Monodevelop
installed on my system.

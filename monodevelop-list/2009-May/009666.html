> Apparently there is now optional Python support for MonoDevelop, but it must

Nope, it can be built against an existing MonoDevelop.

> Could this be done for the OS X version?

It's not included in the OS X app bundle because it's completely
untested on Mac. In fact, a release has never formally been made on
any platform -- it was a weekend hack that got contributed and
abandoned. that said, I've heard many complimentary things about it.

You could try downloading my rpm trunk build, extracting the addin's
dll, and placing that into ~/.config/MonoDevelop/addins
http://download.opensuse.org/repositories/home:/MJHutchinson/openSUSE_11.1/noarch/monodevelop-python-0.2.133630-1.1.noarch.rpm

Be aware that it's a *python* addin, not an *ironpython* addin.

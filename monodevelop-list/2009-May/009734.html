To build MD on windows you need to:
     1. Install gtk# (using the installer you can get from the mono
        website.
     2. Manually install other assemblies used by MD. IIRC those are
        Mono.Posix, ICSharpCode.SharpZipLib and Mono.Addins. You'll have
        to register them so that Visual Studio can find them.
     3. Open the Main solution in Visual Studio and build it. You can
        directly use msbuild if you want.

We are building an installer to do step 2. It will be ready soon.

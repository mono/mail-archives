Hi *,

lately I had to do a little bit of Flex 3 programming and Eclipse + Flex
Builder is... well... horrible. :) So I started a Flex Binding addin
that right now do the basic stuff:

* Create a Flex project (with nice FX icon overlay)
* Provide basic support for .mxml and .as files
* Compile your MXML or AS3 project into SWF on F7

In the future I hope to have enough time to support:

* Better syntax highlight
* Run SWF in standalone player or firefox on F5
* Auto-completion
* Debugging (?!)
* Any other ideas?

The addin is available from:

git://luna.dndg.it/public/flexbinding

ActionScript 3.0 syntax highlight requires a patch to MonoDevelop that I
still have to send. ;) 

Have fun,
federico

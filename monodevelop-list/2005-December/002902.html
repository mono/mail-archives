Thanks, it works now.

I have committed the ChangeLogAddIn.

To use it you must open a ChangeLog file. Then in the ChangeLog sub menu of
the Edit menu there are two items:

Insert Header: Inserts a header with name and email (taken from the
configured values in Preferences).
Insert Entry: Will insert a filename entry provided that you have marked a
file in the solution explorer. It will use the path relative to the
ChangeLog file and the selected file must be in the same directory or a sub
directory of the ChangeLog file.

Comments are welcome. Have fun! :-)

/Jacob

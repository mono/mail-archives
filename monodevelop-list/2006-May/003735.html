Hi,

> Can somebody help me, how can I enable the mono debugger from
> monodevelop?

Unless I'm mistaken, the current debugger needs to be built into
monodevelop (in other words, when MD is compiled, you include
--with-package=mono-debugger [or similar]). The last time I tried to do
that (a week or so back) using the tarballs from the go-mono.com/sources
website, MD complained that the api had changed for mono-debugger and so
it couldn't be compiled in.

As I say, I could be wrong and MD 0.11 has this fixed.

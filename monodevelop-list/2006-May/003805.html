I've been updating Nemerle binding to use the new Nemerle's compiler
integrated completion engine. Now MD compiles with gmcs, I think it's
time to add it to the main MD branch. It updates the previous version
(it continues being in C#), but has completion and class tree features.

neccessary in configure.in.

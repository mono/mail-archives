Yes, I understand that it compiles and runs. But there really needs to be
proper MonoDevelop integration:
 - project templates for MVC 3 that includes the necessary assemblies
already (not sure if this is possible with MVC's license)
 - scaffolding view wizards to create CRUD views in Razor
 - syntax highlighting and code completion for Razor

I know that Razor completion is probably the real monster feature there, but
even having everything else would be a really good start.

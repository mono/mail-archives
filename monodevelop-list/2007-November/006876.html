Actually, I didn't write that page, but have found it very useful in
the past, and wanted to pass the info on.

The ./configure command does apply directly to release tarballs. Some
confusion arises because ./autogen.sh runs ./configure automatically
after running the autotools commands, but you can quite happily
"re-./configure" afterwards. I've edited the wiki page to mention
./autogen.sh.

gtkhtml-sharp-2.0 is included in the gtk-sharp-2.0 package.

If you are building gtk-sharp from sources, make sure it says
"gtkhtml-sharp.dll: yes" when you run the configure script. If it says
"no", you're probably missing the libgtkhtml-devel package or something
like that (check the config.log to see what was not found).

Lluis.

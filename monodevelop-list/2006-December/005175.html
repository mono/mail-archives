After talking about it with Alan and Jo in #mono, I've written a tiny  
little easter egg patch for MonoDevelop.

This little patch allows for some "interesting" status messages to be  
displayed in the splash screen as opposed to the boring old  
"Initializing MonoDevelop" "Loading Boo Binding Add-in" etc.

Instead, it will display "Releasing Hamsters..." or "Burninating the  
Peasants..." etc. The behavior is enabled by setting the "RETICULATE"  
environment variable, for example:

tyler at onion:~/sources/mono-project/monodevelop> RETICULATE=1 make run

Whereas a simple "make run" will display the normal status messages,  
a screenshot of the "feature" can be found here: http://flickr.com/ 
photos/agentdero/327245643/

Hilarity Ensues...


The patch is against SVN, and is located here: http:// 
www.bleepconsulting.com/patches/monodevelop/ProgressTracker.cs.diff


I hope this patch is accepted, I'd love to set my MonoDevelop to  
start from the KDE bar with this enabled :)

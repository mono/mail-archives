I actually just got to responding to your email submitted via my blog,
however disregard that one... As this one contains more information.

Your problem is likely simple, You never installed X11. X11 is installed
as part of your operating systems install, under the custom button...
Apple only means it for part of academia; not as something for
everyone... In any case you will need to do this and in this specific
order.

1. Locate and install the X11 package from your Tiger install disc.
Located in /System/Installation/Packages on Disc Tiger install 1 (Or
factory restore Disc 1).

2. Run Software Update and run the update for X11.

3. Reboot.

4. Re-run the MonoDevelop Installer. This is needed because X11 has to
be patched (fontconfig that comes with OS X is screwy).

5. Try MonoDevelop.

Remember these packages are PPC-ONLY. If you have an Intel Mac they will
not work. Id continue development but do not currently have a Macintosh
at all; a real issue if you're trying to develop something on an OS that
requires specific hardware huh?

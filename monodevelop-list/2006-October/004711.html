Hi All

Please find attached a patch that will allow the Welcome Page Xslt be translated using the standard GettextCatalog.GetString method. I have added XsltArgumentList containing an extension object for the MonoDevelop.Core.GettextCatalog class to the XslTransform. This allows you to call the GetString method from within the xslt. I had to change the private constructor for GettextCatalog to public to implement this solution. I have also changed the xsl to use the GetString function.

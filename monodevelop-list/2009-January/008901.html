> Currently Novell's Bugzilla is down for registration, so I couldn't open a
> ticket for this. I believe that a hotkey should be available for fullscreen,
> and I would recommend F11 for this, because many other applications using

F11 is already used by the debugger's "Step Into" command -- this is
the same as Visual Studio. FWIW, I think it's useful to have a single
key for the most common stepping operations (instead of a combo).

Maybe we could use alt-shift-enter like VS does. Fortunately it's not
hard to bind a new key using Edit->Preferences->General->Key
Bindings->View. If people agree that Alt-shift-enter is a good
default, then I can add it.

It might also be nice to tweak this command to hide the toolbars and
autohide the pads.

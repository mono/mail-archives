Hi all,

This is just a warning to folks living on the edge, that after upgrading 
to r125208, I experienced some problems with the ChangeLog integration.

1) The old setting is ignored, and now integration is off by default:

https://bugzilla.novell.com/show_bug.cgi?id=471285

2) The formatting of the ChangeLog seems to be messed up:

https://bugzilla.novell.com/show_bug.cgi?id=471286

I'm sure the developers will fix these soon, but I was caught off-guard 
by these changes and I wanted to save others the headache.

That being said, the ChangeLog and other policy updates are pretty cool!

> But I can not find any templates for AddIn project. Did I missed

There aren't any templates for addin projects yet. This will be
supported via the "addin addin", described here:
http://foodformonkeys.blogspot.com/2007/05/monoaddins-keeps-growing.html

However, it's still not complete or usable. Hopefully it will be
available as a preview for MD 2.0.

> Also I'd be interested in Gendarme plugin but I can not find how to
> invoke it from monodevelop. I've compiled and installed CodeAnalysis
> folder.

That is the Gendarme addin, yes. Unfortunately it lacks a maintainer,
so it's bitrotted a little. Simple analysis commands (context menu on
projects) should work, but it uses local copies on gendarme and
smokey.

There are several discussions on the gendarme mailing list about
future development of this addin.

Actually, the default depends on your own default policy (see
Edit->Default policies) until a policy is saved to the solution.

I'm considering an automatic settings migration, but it would be
somewhat fragile, and I'm not sure how many people it would affect.

> https://bugzilla.novell.com/show_bug.cgi?id=471285
>
> 2) The formatting of the ChangeLog seems to be messed up:

Yes, this is certainly a bug and should be fixed ASAP. However, note
that you can edit the actual changelog entry and the commit message in
the commit dialog. Also, in the policies you can customise the commit
message and changelog formats, including using .NET date formatting
specifiers as custom date formats.

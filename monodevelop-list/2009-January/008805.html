First off, kill the mono repo. It's never a good idea to mix things from
different packagers. Stick to the fedora one.

Next, use

yum -y install monodevelop* mono-*

Though the version in f9 is somewhat old. You may wish to do

yum --enable-repo=rawhide -y install monodevelop* mono-*

which will grab the latest versions of mono and monodevelop.

For future readers: the Debian packages can be installed in this way:

"
Alternatively, you can install Mono 1.9.1 and MonoDevelop 1.0 by using apt
or aptitude from the debian.meebey.net/etch-backports repository. This
repository is driven by the official package maintainer of the Debian Mono
packages.

Add this line to your /etc/apt/sources.list file:

deb http://debian.meebey.net/etch-backports /

And then install or upgrade your Mono and MonoDevelop using:

apt-get update && apt-get dist-upgrade

or

apt-get update && apt-get install monodevelop
"

These instructions are found on: http://www.mono-project.com/Other_Downloads

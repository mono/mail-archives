I broadly agree with that, but MD 2 now requires Mono 2, which is
somewhat more challenging to build/install than MD, and Ubuntu doesn't
have it yet. For this reason, I'd recommend using the Mono 2.0 VMWare
image from http://www.mono-project.com/Downloads.

>From this VM you should be able to install my trunk packages with the
1-click installer at
http://software.opensuse.org/ymp/home:MJHutchinson/openSUSE_11.0/monodevelop.ymp

These packages have not been updated for a few weeks because of
instability in trunk, but I will start updating them very soon.

If you do decide to build Mono from source, please read
http://mjhutchinson.com/journal/2007/11/08/how_not_break_mono and use
a parallel environment.

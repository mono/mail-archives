Hi Vincent,

The XML Editor add-in that you can install using MonoDevelop's Add-In
Manager does not work with the latest MonoDevelop from the Subversion
repository. Uninstall the add-in and if you need it you can download
the XML Editor code, which works with the latest MonoDevelop, using:

svn co https://svn.sourceforge.net/svnroot/md-xed/trunk md-xmleditor

Regards,

Matt

Hi Uwe,

You might be interested in the new Boo language
(http://boo.codehaus.org/):

"Boo is a new object oriented statically typed programming language for
the Common Language Infrastructure with a python inspired syntax and a
special focus on language and compiler extensibility."

By being statically typed Boo keeps most of the concise Python syntax,
but blends better with the CLI (mono/.net) environment.

It's creator (Rodrigo 'Bamboo' Oliveira), already took the time to
create the syntax mapping files that gtksourceview uses for syntax
coloring, so at least you could use MonoDevelop to edit it's files, but
deeper integration is only in the planning stage just now.

Hope it is of some interest for you folks,

MonoDevelop 2.4 hasn't even begun development yet.  Perhaps you mean Mono
2.4.

Ubuntu does not yet support monodevelop 2.2 in the package repositories (and
therefore Python as well) since 2.2 has not yet been officially released.
You can install it from source packages which are available on the website.
Use --select with configure to enable python support.

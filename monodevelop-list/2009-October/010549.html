Hi All!

I'm glad to announce the first official release of DeveelDB, the
.NET/Mono embedded relational database management system. This is the
first fully working and functional version of the project: you can
download a copy at the following addresses

Linux:
http://deveeldb.googlecode.com/files/deveeldb-0.9.6-bin-linux.tar.gz
Windows:      http://deveeldb.googlecode.com/files/deveeldb-0.9.6-bin-win32.zip

If you wish to start contributing to the project, please feel free to
refer to the website http://db.deveel.com, where you will find all the
information concerning the project.

I wish you will enjoy using DeveelDB, as much as I enjoyed doing it!


Cheers!
Antonello

Windows does provide an easy way to switch between languages. Read up about the "language bar" in Windows help. This is in recent versions of Windows, XP and later I think.

It is there specifically to address the problem of different people using the same machine needing different interface languages. The bar is only available if you have multiple languages set up in Regional Settings in control panel.

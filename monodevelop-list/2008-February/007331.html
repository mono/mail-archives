The segfault means that there is some problem with the unmanaged
libraries, probably in the Mono runtime itself or GTK. From your
description of installation, I'd guess that you have somehow mixed up
bits and pieces of different Monos which -- becuase they're different
versions -- don't quite work correctly together.

Please read http://mjhutchinson.com/journal/2007/11/08/how_not_break_mono
for info on this.

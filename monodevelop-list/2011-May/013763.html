Okay so there is something wrong with the MonoDevelop 2.6 beta 2 GIT add-in.

I used the command line without issue and got my solution add to my repo.
Then MonoDevelop picked up all the settings and is working.  

So two issues:

1) it doesn't appear to work on setting up a repo via the UI to or ability
to commit a solution for the first time to a repo basically. This is a major
issue, I shouldn't have to go command line on it to get it started.

2) it doesn't use the Mac Keychain to access the SSH passphrase, so I had to
enter it the first time, after that it seems to remember it.  Very minor
issue, can live with this, just would be nice if it could ask permission to
use keychain.

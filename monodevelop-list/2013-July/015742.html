> under Ubuntu. That was why I installed Mono.
>
> I've read that since Mono is a"core" element in Ubuntu and that other
> apps depend upon it that it should not by upgraded. In addition there is
> no V4.0 package available for Ubuntu.

In general - if the problem is directly Mono or monodevelop related (say 
there is a problem in the compiler or MD explodes and eats your hard 
drive), then you should report it on Xamarin's bugzilla where they can 
fix the issue.

This problem sounds like an Ubuntu issue specifically. It may be worth 
putting this into the Ubuntu bugzilla for the package maintainer to try 
and solve.

The strings you are asking for are in 

src/Core/MonoDevelop.Core.Gui/Base.glade
(a glade file made by the GUI editor)

but this file is not (yet) included in the localization files list.

You can do it manually by adding this path to
po/POTFILES.in

Now update your translation file:
intl-tool-update ru

and you get new strings from Base.glade into ru.po .


I do not know if there is a way to do this automatically.
Maybe someone with SVN write access can correct this in the repository.

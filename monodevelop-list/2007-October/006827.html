Hello everybody,

in my quite large SW projekt (ERP system) I have created a handful of subdirs, 
which respresent certain types of classes (forms, reports, and the like).

the new class a member of some sub-namespace <project>.<subdir>. However, MD 
seems only to know the namespace <project>...and that's it.

But, in order to get the various classes into some order, MD should, for 
instance, create a namespace like myproject.forms for all classes the subdir 
$MYPROJECT/forms. Some Java IDEs (namely NetBeans) do that. Is this also 
possible in MD?

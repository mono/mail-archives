Maybe you got same problem like me again.

Once installed Monodevelop crashed at startup while loading the splash
screen.
The message was :

System.EntryPointNotFoundException: Unable to find an entry point named
'gtk_widget_set_tooltip_text' in DLL 'libgtk-win32-2.0-0.dll'

That happens when there are other gtk-based software installed (mine was
Gimp).
Once uninstalled it, Monodevelop was able to run without problem.

> MonoDevelop on mac does not use Xwt.Mac. If you do need to build
> Xwt.Mac you will have to check out git://github.com/mono/monomac and
> git://github.com/mono/maccore, then compile your own MonoMac.dll and
> change the Xwt solution to compile against this binary. Generally
> speaking this is the only way you'll get Xwt.Mac to compile as the API
> of monomac changes constantly and the version shipped with MonoDevelop
> will nearly always be incorrect as xwt from git tracks monomac from
> git.

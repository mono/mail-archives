You would need to install a custom GTK theme into ~/.themes/ or into
Mono's bundled copy of GTK, then it would appear in the visual style
preferences dropdown. Note that the only theme engine provided with
Mono's bundled GTK is clearlooks, so you'd have to use a theme that
uses that engine.

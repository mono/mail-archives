On Mon, Aug 9, 2010 at 5:14 PM, Eric Dunstan <e_dunstan at hotmail.com> wrote:
> Hello,
>
> I was wondering what version of Silverlight can applications be developed
> for on Mac OS X using MonoDevelop?

Currently, 2.0.

> I know that Microsoft provides the Silverlight plugin for Mac OS X, so it is
> as up-to-date as Windows Silverlight.

Yes, but they do not provide an SDK, so MonoDevelop uses the Moonlight
SDK. Also, we do not have access to their Silverlight debugger
protocol so we can only debug Moonlight (on Linux).

> If Monodevelop can support parts of Silverlight 2 and 3 and 4, what parts
> does it support? And what parts does it not support?

It supports 2.0. It could support 3.0 or possibly 4.0 if someone were
to update the bundled version of the Moonlight SDK to the current
Moonlight previews.

> Also, what differences do silverlight or moonlight applications developed in
> Monodevelop have compared with those developed with Visual Studio?

None, AFAIK.

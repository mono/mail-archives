The various released versions are tagged in the SVN repository:
http://svn.myrealbox.com/viewcvs/tags/monodevelop/

You can find out about Mono's SVN here: http://www.mono-project.com/AnonSVN

You'll probably want to run a command like
svn co svn://svn.myrealbox.com/source/tags/monodevelop/0.9/

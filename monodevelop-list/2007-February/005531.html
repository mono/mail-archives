On 2/23/07, Paul <paul at all-the-johnsons.co.uk> wrote:
> Well done on this release however, when I try to run it, I'm getting the
> following (using mono-1.2.3 from Fedora Extras)
>
> no implementation for interface method
> MonoDevelop.Projects.Gui.Completion.ICompletionDataProvider::get_DefaultCompletionString() in class MonoDevelop.XmlEditor.XmlCompletionDataProvider

This has come up on the mailing list several times. It happens when
externally-maintained AddIns such a sthe XML editor are
binary-incompatible with changes in MonoDevelop. Remove the XML editor
addin manually from ~/.config/MonoDevelop/addins, or use "sudo mdtool
gsetup" to uninstall it.

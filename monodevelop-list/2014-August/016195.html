Using "make install" with a user without privileges should have given 
you errors (so yeah, that's weird).

However, using "root" as the user to install is actually not the 
recommended way to do things. The encouraged way is to configure and 
compile mono using your normal user, and then use "sudo" for the 
installation: "sudo make install" instead of "make install".

Well, MD 1.0 is the latest stable release.  There are alphas of 2.0
(which are somewhat old).  I would recommend installing trunk from the
svn repository and compiling it yourself.  I do this and rarely run
into issues, and when I do I can either revert to a known working
revision or file a bug report, which usually gets dealt with in a few
days.

See <http://monodevelop.com/Download#Getting_MonoDevelop_from_the_Subversion_repository>.

> If I have a file called foo.bar and it has no registered MIME type, but it
> happens to contain XML markup, I would like to get pretty colors and all the
> other goodies.

Yes. If GnomeVFS/GIO recognises it as xml, it will get picked up.

Alternatively, you can force it (and assign a default schema) using
edit->preferences->other->xml schemas.

> Is there instruction on how to build Mono and MonoDevelop on Windows.

Regarding Mono,
http://www.mono-project.com/Compiling_Mono_on_Windows
It's quite difficult due to the use of Cygwin. I believe someone's
working on MSVC (no Cygwin) builds, but I'm unsure of the status of
that. For now, I suggest using binaries.

Regarding MonoDevelop, although it can be built and run on Windows,
it's not yet usable. This is something we will fix after 2.0. The best
instructions you'll currently find are at:
http://mjhutchinson.com/journal/2009/02/07/monodevelop_vista64
IIRC, SharpDevelop 2.x has the ability to target Mono, so this would
be the best option right now.

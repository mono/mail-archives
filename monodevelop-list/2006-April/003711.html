my workaround for this issue is using a symbolic link, so the gmcs binary
points to mcs, or to set an alias.
MD will throw you some exceptions (because of the syntax stuff you
mentioned), but its usable. Also make sure you're using the 2.0 assemblies,
so you could take full advantage of it.

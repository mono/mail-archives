Tracy,

I don't really know much about Mono I too have only just installed the
1.1.14 release but I have gotten the visual gui designer to work. If you
go into the solution section on the left hand side then double click on
the MainWindow.cs file you will see that below the code window there is 2
buttons 1 says source the other says designer if you click the designer
button then you will be able to design the window using the gui designer.

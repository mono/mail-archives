It seems rather that you have a very old or trimmed down Gnome.

Some distros run KDE and other alternative desktops and so install a
minimal gnome just enough to run some gnome apps.

Try to install full gnome support including metacity (but there's no
need to switch your desktop to Gnome), so that MonoDevelop and other
gnome apps may be able to run correctly.

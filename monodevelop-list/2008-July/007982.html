You were probably running an old MonoDevelop installation.  Whenever you 
want to run MD 1.9, make sure you launch MD from a terminal that is 
running in the parallel environment (by source'ing the script on that 
wiki page).

Also, I'm sure you figured out that --enable-gtksourceview2 uses the old 
code editor that doesn't have region folding, etc.  Don't pass that flag 
if you want the new editor features.

Sandy

Probably worth checking this thread:

http://lists.ximian.com/pipermail/monodevelop-list/2014-July/016110.html

Basic answer is (paraphrased) "MonoDevelop effectively doesn't exist in
its own right in the way it used to, because Xamarin Studio is MD plus
plugins and the only way to get it without plugins is to build from
source". Which is kinda awkward when Xamarin Studio Starter has build
size limits on it and all the other licenses cost money…

If you run Linux then you'll still have to use a vanilla MD from the
repos (because Xamarin have largely ignored their Linux heritage since
they were founded and don't yet make an installer available), but other
than that then it is Xamarin land all the way.

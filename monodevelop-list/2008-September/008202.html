Workspaces were added after 1.0, so yes, this API is only present in
1.9 and SVN.

In 1.0, you would have to do

IdeApp.ProjectOperations.CurrentOpenCombine.GetAllProjects ()

In 1.9+ you can do

IdeApp.Workspace.GetAllProjects ()

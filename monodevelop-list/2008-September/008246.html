Specifically, Andy, any code you find with an MIT/X11 license at the top
of the file can be used under those terms.

MonoDevelop as a whole is only licensed under the GPL because of certain
GPL-licensed code.  If you only use MIT/X11 code, you are not bound to
the GPL.

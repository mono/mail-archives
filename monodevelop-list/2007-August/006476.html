Hi,

If you get MD from SVN, you'll need to do a clean rebuild and install,
that is:

* make clean
* make
* rm -rf /usr/lib/monodevelop (or wherever you install it)
* make install

Surprisingly enough, ~/.config/MonoDevelop/addins* doesn't need to be
deleted.

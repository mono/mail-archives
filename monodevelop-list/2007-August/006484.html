A clean install did the trick..it seems that  yum did not removed some sharp
packages.
Even at clean install i did:
>yum remove *sharp*
>yum install sqlite2
>yum remove mono*
>yum install monodevelop --enablerepo=Mono --disablerepo=fedora
--disablerepo=updates

After this i had installed MD 0.15.
Hope it helps.
Thanks,
Adrian.

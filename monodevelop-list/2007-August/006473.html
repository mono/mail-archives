Christian,

You can try to use Kugar (or the Gnome equivalent, if exists).
Kugar requires a data file (.xml or other) and a template file (.ktf).
You can use it as shell to preview or print reports; or access to Kugar's 
classes to report; or create a QT designer plugin. . The last two options are 
usual practices for KDE programmers but to make them work on Mono you need to 
use Qyoto and know some of QT/KDE, However I think that using a Kugar shell 
is the best option to keep things easy and to stay in C#/Mono territory.

I never used on Mono but should work, or if someone know the equivalent of 
Kuguar on Gnome or other Linux or FOSS alternative, please let us know.

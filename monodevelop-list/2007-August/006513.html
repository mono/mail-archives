Hi Jimmy!

Welcome to mono and monodevelop!

You can see installation instructions using the binary file
here<http://www.mono-project.com/InstallerInstructions>
.

As you can see there's a sentence saying: "The installer contains
MonoDevelop, but it requires that you have gtk 2.4 or greater installed, as
well as the Gnome libraries."

So I think you are getting that problem because you don't have gtk
installed... Try to do a search in your distro package manager for 'gtk' and
install the most recent version and then try again using the directives from
the link above.

Hope this helps!

Vitor Hugo Barros

Hi,

I've worked on the designer in the past year. It isn't included in mono's
repos, because it is not stable enough and it hasn't reached the desired
functionality. Like getting the drag and drop components from the toolbox
to work and assigning handlers to events using the property grid.

The parts of it that do work are: the selection service, the document
outline, editing properties through the property grid, and there is a
protocol for communicating with the designer surface.

If you are interested I can walk you through the code.

Cheers,
Petar

Hello all,

	MonoDevelop now has experimental support for building and installing
using the auto* tools like many of the other projects out there.  It is
_not_ required that you install it.  You can continue to use it in the
self-contained fashion as before. To do this run: make run

However, you can now install monodevelop so it will appear in the GNOME
menu's, be in your path, etc.  This currently works for me, but may have
some bugs in it. So please report any problems to the list. (especially
those of you that use wacky prefixs)

Thanks,
John

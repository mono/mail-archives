Hi,

> **Error**: You must have `intltoolize' installed to compile MonoDevelop.
> [tracy at nec monodevelop]$
>
> I'm running Mandriva Linux 2006.0 with a KDE desktop, there does not
> appear to be any packages that include anything like autogen.sh

Autogen.sh is a script file that comes with monodevelop. It run
several autotools programs, one of which -- as this error message from
autogen.sh says -- is intltoolize. I would be very surprised if you
cannot get an intltoolize package for Mandriva. You will also need
automake and autoconf, and probably some other stuff too.

On 3/29/06, Gideon de Swardt <gdeswardt1978 at yahoo.co.uk> wrote:
> UNMANAGED_DEPENDENCIES_MONO... checking for
> UNMANAGED_DEPENDENCIES_MINT... configure: error: Please install mono
> version 1.1.10 or later to install MonoDevelop.
> Please see http://www.mono-project.org/ to download latest mono sources
> or packages

This error occurs when the configure script can find neither mono's
nor mint's autopackage information. You don't need both.

The file that is missing is /usr/lib/pkgconfig/mono.pc. The Dapper
package you need is libmono-dev.

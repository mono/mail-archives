The MonoDevelop team is proud to announce the release of MonoDevelop
0.8.

Release notes are here:
http://www.monodevelop.com/Release_notes_for_MonoDevelop_0.8

We now have a new web site (http://www.monodevelop.com) which is based
on MediaWiki, so now everybody can contribute (specially in the Articles
and Screenshots sections). If you have some content you want to add to
the wiki, just drop me an email.

There is also a TODO page that can be a good start for people willing to
start hacking on MD. Take a look at: http://www.monodevelop.com/TODO.

Enjoy!

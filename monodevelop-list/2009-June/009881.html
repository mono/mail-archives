Hey,

I created a simple Addin which adds a pad which integrates a Vte
terminal into MonoDevelop. Look at a screenshot here:
http://files.getdropbox.com/u/182844/md-terminal.png
or download the code:
http://files.getdropbox.com/u/182844/MonoDevelop.Terminal.tar.bz2


It's created for MonoDevelop 2.0. If you want to use it with the svn
version you need to change the dependency version in *addin.xml to
2.1.0. (thanks to lluis for this hint) 

However I have 2 problems:
- There is no icon. I tried to use 'terminal' or 'gnome-terminal' but
for a reason I don't know nothing is displayed.
- I have many crashes of MD when using this plugin. All stacktraces are
exactly like this one: http://paste.ubuntuusers.de/395576/

Any ideas on how to fix that?

You are confused. Loading the project into MonoDevelop on Windows won't
convert your project to Suse. MonoDevelop is not Mono. MonoDevelop is
just a .NET IDE, which runs on MS.NET on windows.

If you want to port your application to Linux, this guide will show you
how to do it:

http://www.mono-project.com/Guide:_Porting_Winforms_Applications

You can keep using Visual Studio for developing, or use MonoDevelop if
you want.

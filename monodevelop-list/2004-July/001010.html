What I am saying is that we will (potentially) be saying 'Existing code
has license GPL, new code written by MonoDevelop contributors will be
licensed as MIT X11', and I would like to do it retroactively, which
would mean securing every contributors agreement to allow this to
happen.

If we do go this route, I will have NO issue ripping out and rewriting
code that people will not allow to be covered under this license.

Right now, the new code in MonoDevelop is under basically NO license,
which is totally unacceptable.

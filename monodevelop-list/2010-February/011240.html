Debugging certainly works if you are application is a pure csharp
application, otherwise you have to write an addin to debug an embedded mono
app. You can find additional information searching in this forum

   If you are setting break points in cshap files that are not part of the
solution / project , please ensure that you have the option to "debug
framework code" checked.

  Mono debugging on OS X only works 2.6 or later

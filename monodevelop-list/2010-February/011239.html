Since you get this problem, i guess you try to install MonoOSC whit
"make" from source, so either you download source of
syntaxhighlighting and ./configure --prefix=/usr && make && make
install then you should have it for MonoOSC, if not you can ensure it
with the gacutil command to register syntaxhighlighting in the gac.
Or a more easy way should be to use the build package of
syntaxhighlighting and syntaxhighlighting-devel

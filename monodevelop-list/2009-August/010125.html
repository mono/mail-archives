Ok I have changed it so the code templates are loaded from 
resources(opposed to messing around with copying them across). I have 
also added an options panel to the project options to allow the ip and 
port for the test server to run on. There is also the ability to add new 
django apps to a projects app folder by using the right click menu -> 
Add -> Add New Django Application. This creates the same file structure 
as performing "python manage.py startapp [appname]"

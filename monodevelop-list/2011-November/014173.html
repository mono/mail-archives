On Tue, Nov 01, 2011 at 07:54:17PM +0000, gustavo.rubio wrote:
> Please don't make this a distro flame war. I'm just asking if someone knows
> what's needed to do in order for MD to be built with GTK/Gnome components
> available as references because they are not found on the GAC for some
> reason.

When building mono components from most source tarballs we use pkg-config which
specifies exact paths to dependant assemblies. We supply the -noconfig option 
to the compiler so it doesn't go off and look in the GAC.

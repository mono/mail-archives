Hi,

This issue indicates that neither the trial nor full version of Mono for
Android  has been installed. To verify that it was actually installed, you
should check that it exists at /Developer/MonoAndroid. If that directory
does not exist, you will need to re-install Mono for Android as described
in the error message. The Mono for Android addin is bundled within
MonoDevelop so it will always show up.

Hope that helps,
Alan.

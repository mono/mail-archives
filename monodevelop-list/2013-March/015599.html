Xamarin Studio is MonoDevelop with a few extras for Android and iOS 
development. It should do everything MonoDevelop always did, but be 
shiny, new, and supposedly improved.

If you go to the Downloads page and click on the Windows icon then you 
get an "Installer" link: 
http://download.xamarin.com/studio/Windows/XamarinStudio-4.0.msi

Linux users currently aren't so lucky - we don't get new packages (or 
support for modern versions of the OS in some cases).

I think this is because gtk style by default disabled icons on button.

On windows you can edit
$PROGRAMFILES\GtkSharp\2.12\share\themes\MS-Windows\gtk-2.0\gtkrc
and change "gtk-button-images = 1"

On Linux you can also change settings from org.gnome.desktop.interface
buttons-have-icons

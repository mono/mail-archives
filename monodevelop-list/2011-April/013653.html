Right now, there isn't. MonoDevelop add-in are usually installed through
the add-in manager. If the code of your add-in is public, I would
recommend publishing it through addins.monodevelop.com. If not, you can
create your own add-in repository and ask users to register to it.

See http://monodevelop.com/Developers/Articles/Publishing_an_Addin.

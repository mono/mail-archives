On Fri, Apr 22, 2011 at 2:28 AM, Rodney Foley <rsfoley at gnomesoft.com> wrote:
> I know there is a GIT add-in in the beta of MD 2.6.x but I cannot use that
> with MonoTouch yet so I am stuck in 2.4.2, and I current have to use GIT via
> command line which is a pain in the arse when you are working in a GUI IDE.
>
> So is there plans to back port the add-in or is there any 3rd party add-in
> for this? I googled around and couldn't find anything but talk about the GIT
> Add-in for the beta 2.6 nothing about 3rd party ones.

There isn't a git addin for 2.4, nor are there any plans to make one.
The git addin depends on many improvements in the VCS core in MD 2.6,
and we'd rather use our resources to get 2.6 released sooner.

MonoTouch should work fine with MD 2.6 b2 (and the hotfixed b1). It's
not officially supported but you can keep MD 2.6 and 2.4 side by side
in case you need to fall back to the supported version.

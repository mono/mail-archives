Hi all,

 

I'm currently working on integrating NBug library (as the author) to many of
our projects. Once integrated, the library logs all unhandled exceptions,
generates minidumps and gathers detailed build & system information about
the crashing application and submits a bug/crash report to the selected bug
trackers (it currently supports Redmine/Trac/Bugzilla as of v1.1). You can
find the library here: http://www.nbusy.com/projects/nbug

 

As I am also using Mono Develop for my application, I would like to
integrate this nice feature to the IDE to help it get even more stable. I
see that this is already in the todo list
(http://monodevelop.com/Developers/Tasks/Build_and_Distribution/Error_Report
ing) and I would like to know if there is an already ongoing effort for this
feature. If not, I would like to integrate NBug into Mono Develop sometime
during next month once the stable v1.1 is released. Looking forward to hear
your opinion, on this.

 

Teoman Soygul.

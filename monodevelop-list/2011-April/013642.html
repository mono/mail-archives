Hi Michael, 

I couldn't just let this go, so I went back to MonoDevelop and starting
tweaking stuff on solution and project configuration, until I finaly got it
solved. 

Code completion for core libs doesn't work if you set the Default Runtime to
Mono in Tools > Options > .NET Runtimes. 

http://mono.1490590.n4.nabble.com/file/n3442829/Code_Completion_Bug.png 

Setting it to Mono breaks code completion, setting it back to Microsoft.NET
(and restarting monodevelop ide) brings code completion into working state.

Oh, and debian/control needs some build-depends adding, really - as-is,
it'll fail to compile on a minimal environment like a PPA because you
need libgtk2.0-cil-dev, mono-devel, etc - but these requirements aren't
part of the Build-Depends: stanza. The "pbuilder" package can help you
set up an on-demand minimal build environment - "pbuilder build foo.dsc"
will try to compile foo.dsc given the packaging information in the
package, and is a useful debugging tool.

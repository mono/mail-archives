Yes, but actually I see two options:

* Wrap search: if search starts from the middle of a text, wrap the search
when the end of the text is reached and continue searching till the old
start point.
* Wrap search loop: when wrap searching is enabled continue to find matches
(looping forever) or stop when all matches have been found.

I think this would be useful. As long as a project looks like a decent
amount of coding work, and someone isn't already working on it, pretty
much anything is fair game for SoC. Some people have in the past
proposed combinations of several small-medium tasks rather than one
big one.

IMHO an important feature for this would be to have comfigurable
"keybinding profiles" with the settings from other popular IDEs.
VS2005-style "chords" might be nice as well, though I don't know if
those are patented.

Do you have to present a password when using svn in the console?

I normally use ssh+svn so my personal ssh keys automatically open the
repository access without asking for passwords.

With https, what seems to be your case, you probably need to checkout
using the MonoDevelop dialog, giving him the password to compose the
url (this presents a security risk as the password appears in the
screen in the url field, although it is protected in the password
field).

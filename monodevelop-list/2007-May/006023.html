Hi guys.

I've created a new Addin for MD which integrated Gendarme framework.

First release can be downloaded at:
http://gendarmeaddin.googlecode.com/files/gendarmeaddin-0.1.tar.gz

I am open to suggestions/bug reports/comments.

Right now it does the following:


   - A dockable pad for running the tests, which shows the problem,
   details and it's solution
   - A configuration panel for specifying the rules file and the rule set
   - Ability to run tests on the whole solution or on the current project
   - Ability to jump directly to problem source, if a rule violation was
   found


Hope someone will find it useful!

Regards,

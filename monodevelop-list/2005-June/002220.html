Anywho, it's checked in now.  autogen needs to be rerun to get the 
Makefiles, and then run make in Extras/VersionControl to build/enable it.

Here's a brief overview of what it does:
   It 'just works' if files are already versioned by svn.
   It adds overlay icons to the solution pad to indicate modified files 
(very cool that this is possible).  The icons are 'imported' from 
Tortoise, the Windows subversion client.
   For the project and file nodes in the solution pad, it adds Diff 
(files only), Log, Update, and Status (projects only) context menus.  To 
commit changes, go to Status first.  Log lets you see changes or the 
contents of files at any revision.

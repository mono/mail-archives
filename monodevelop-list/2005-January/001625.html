Hi,

When running a project, MonoDevelop now redirects the application output
into a output panel in the IDE. This will eventually be replaced by an
embedded terminal pad, but right now it is just a text buffer, which
means that the application input is not redirected.

If you are running a console application that needs console input, you
can set the "Run on external console" project option.

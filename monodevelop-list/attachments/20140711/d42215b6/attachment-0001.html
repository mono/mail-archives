    public partial class XamarinStudio : MonoDevelop<br>
    {<br>
        XamarinAndroid xa = new XamarinAndroid();<br>
        XamariniOS xi = new XamariniOS();<br>

        }<br>

    <br>
    Xamarin Studio is MonoDevelop with commerical addins which are free
    if user has Starter edition. In MonoDevelop (Xamarin Studio)
    everything is based on AddIns, so XamarinAndroid, XamariniOS,
    XamarinMac are commercial addins.<br>
    <br>
    There is no XamarinStudio on Linux and this might be source of
    confusion. <br>

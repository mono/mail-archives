Do you just want to run the project on mono or continue developing it?<br>
<br>
If you just want to run the project you just copy over the directory
containing the *.aspx files and bin directory into some location on you
Linux box.<br>
<br>
You can then use the xsp webserver to serve the pages have a look on google to find out how to use xsp, it is very easy.<br>
<br>
A reasonably simple project will just work with little or not changes.
remember that Linux is Case Sensitive so links image name, etc must
have the correct Case.<br>
<br>

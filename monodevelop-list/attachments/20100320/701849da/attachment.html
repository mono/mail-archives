<p>The MonoDevelop commandline build tool, mdtool build, depends indirectly on X because it uses MD libraries that depend on GTK#. Our &quot;fix&quot; for this has been to work on completing xbuild, which will also be integrated into MD as the new MSBuild engine.</p>

<p>Although you could use a recent version of xbuild on the server, I would suggest investigating using MD&#39;s Web Deploy feature, which can publish the compiled site to a directory on your server using sshfs from within MD.</p>

    <div id="smartTemplate4-template">Add it as additional .NET runtime:
      in MonoDevelop, go to Edit -> Preferences, section
      Projects/.NET Runtimes and click the Add button. Select /tmp as
      the mono installation prefix (assuming your mono executable is
      /tmp/bin/mono).<br>
      <br>
      Now when you have a project open in MD you can use mono from /tmp
      by selecting it from Project -> Active Runtime.<br>

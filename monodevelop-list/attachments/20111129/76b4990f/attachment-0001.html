      I don't know how you did it, I had no problems installing
      monodevelop 2.8.1 on Ubuntu 11.10.<br>
      Either you use an older version, or you haven't properly compiled
      the mono runtime of the 2.10.x branch, or you used the sources
      from git.<br>
      <br>
      I made a tutorial on how to install mono (&amp; monodevelp) on
      ubuntu from sources.<br>
      It's for 2.8, but just replace 2.8 with 2.10.x and it will work.<br>
      See here:<br>
      <a href="http://ubuntuforums.org/showthread.php?p=9943196">http://ubuntuforums.org/showthread.php?p=9943196</a><br>
      <br>
      PS: On Ubuntu 11.10, just do <br>
      apt-get install libglib2.0-cil libglib2.0-cil-dev<br>
      to have it glib installed.<br>
      <br>
      For mono 2.8 on Ubuntu 11.10, all you need to do is to download
      and compile the latest version of mono-addins.<br>
      <br>
      <br>
      PS2: <br>
      I see you're using kUbuntu.<br>
      If you're going to install Ubuntu 11.10, you'll fine gnome 3
      pretty unusable.<br>
      KDE is much too slow (and that althought I have 8 GB RAM, and a
      1.6&nbsp; GHz AMD Phenom Quadcore, and 1 GB of graphics RAM...), and
      XFCE too limited.<br>
      <br>
      As a resolution, I installed enlightenment:<br>
      sudo add-apt-repository ppa:hannes-janetzek/enlightenment-svn<br>
      sudo apt-get update<br>
      sudo apt-get install e17<span class="Apple-style-span"

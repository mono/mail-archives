<p dir="ltr">Hi, Steps to build and debug monodevelop. <br>
1. Fork the repository in Github <br>
2. git clone forked repository. <br>
3. git submodule update --init<br>
4. Open solution file and build whole solution. You may get an error but it is not a problem. <br>
5. Press Debug and Enjoy. </p>
<p dir="ltr">P. S. If you make a change in extension project build it (Shift + F6) before presing F5 </p>

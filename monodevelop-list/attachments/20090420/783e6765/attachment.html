<div><br></div><div>My suggestion: provide 3 zips.</div><div>1. Windows x86, with your game, SDL.Net and the unmanaged SDL library compiled for x86.</div><div>2. Windows x64, with your game, SDL.Net and the unmanaged SDL library compiled for x64.</div>
<div>3. Linux (both x86 and x64) with your game and SDL.Net. The user is expected to install SDL through his package manager. Optionally, you can include a simple executable script that calls &#39;mono MyGame.exe&#39;, in case the user hasn&#39;t registered executable assemblies to open with mono automatically.</div>

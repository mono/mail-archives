      You don't add a reference to unamanged dlls. Instead you need to
      create (or copy) a *.dll.config file that maps the dll name to the
      shared library (see
      <a class="moz-txt-link-freetext" href="http://www.mono-project.com/Interop_with_Native_Libraries">http://www.mono-project.com/Interop_with_Native_Libraries</a>).<br>
      <br>
      However, in your case it looks more like it can't find <i>libclntsh.so</i>.
      Do you have that file on your system, and is it in your PATH?<br>

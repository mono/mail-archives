True, you can use Glade to create your GUI in GTK.<br>
Glade saves your GUI as an XML file . Eg: window.glade.<br>
You can use this file in a Mono project as a resource.<br>
<br>
For this you have to include the line in the source code od the project:<br>
<br>
using Glade;<br>
<br>
and you have to include the package glade-sharp.<br>
<br>
A small sample application to create a GTK window and use it is given in this url:<br>
<a href="http://www.codeproject.com/cpnet/googleSearch.asp">http://www.codeproject.com/cpnet/googleSearch.asp</a><br>

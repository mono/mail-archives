</div></div>You can use the NUnit runners. Mono ships with some of these (e.g.<br>
nunit-console4) but unfortunately it ships with a really old version.<br>
This might be the version you're already using in which case you can<br>
do<br>
<br>
```<br>
$ nunit-console4 /path/to/your/tests.dll<br>
```<br>
<br>
If you want to use a newer version you can use nuget to get a newer version<br>
<br>
```<br>
$ mono nuget.exe install -Version 2.6.4 NUnit.Runners<br>
$ mono NUnit.Runners.2.6.4/tools/nunit-console.exe<br>
```<br>
<br>
There's also a GUI NUnit client in the NUnit.Runners package<br>
<br>
```<br>
$ mono NUnit.Runners.2.6.4/tools/nunit.exe<br>
```<br>
<br>
Hope that helps.<br>

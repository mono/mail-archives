I think you're going about this the wrong way.  :-)  If you're using Red
Hat Linux, you probably want to install Mono for Linux, not Mono for
Windows, right?  Wine shouldn't be involved at all.

Poke around for where to get packages, or build it from source.

> After installation, When i try to open, i can't find the mono IDE.

MonoDevelop (the IDE) is not bundled with Mono.  It is a separate
package, and it is only available for Linux and Mac OS X right now.  So
you won't be able to run it under Wine.

Again, poke around a bit to find packages for your distro.  It would
help to know what version of Red Hat you're running.  You'll probably
find that you're better off using the latest Fedora (or openSUSE, or
Ubuntu) if you want to use the latest Mono and MonoDevelop.

> So any idea on how can i resolve this issue?

related to the MonoDevelop IDE, please post them on mono-list instead.

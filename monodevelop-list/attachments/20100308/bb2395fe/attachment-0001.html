you can do:<br>

2) file type definitions<br>
3) IDotNetLanguageBinding for project &amp; compilation support<br>
4) file &amp; project templates<br>
5) option panels for setting language-specific project options<br>
6) a syntax highlighting definition<br>
7) code templates<br>
8) a parser, to provide outlining, folding, class list, quick finder,<br>
and fill the database that code completion uses<br>
9) a text editor extension subclassing CompletionTextEditorExtension,<br>
to provide code completion and smart indentation etc.<br>
10) add Refactorer to the IDotNetLanguageBinding implementation to<br>
support things like find refs, rename, insert member etc<br>
11) other things like code formatter etc.<br>
<br>

Hi Marc<br>
<br>
This might help - as an exercise, I've written a simple RSS news feed
reader that will display the web page for the news item - code snippets
as below:<br>
<br>
My Monodevelop project has 3 bits of code<br>
<br>
1) Main.cs which just invokes RSSreader;<br>
<br>
2) then, in RSSreader.cs this code:<br>
<br>
<br>
using WebKit;<br>
<br>
namespace myRSS<br>
<br>
<br>
// then in whatever method you want to invoke the browser:<br>
<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; // view webpage for news item (or 'home' page for this RSS
feed)<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; ErrorMessageLabel.Text = "";<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; if (Active_GUID != "") {<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; // invoke the WebKit browser view (cool!)<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; new NewsWebView (Active_GUID);<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; } else {<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; ErrorMessageLabel.Text = "no GUID url associated with
this item";<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; }<br>
<br>
<br>
3) and in NewsWebView.cs the complete code is:<br>
<br>
using System;<br>
using Gtk;<br>
using WebKit;<br>
<br>
namespace myRSS<br>
{<br>
&nbsp;&nbsp;&nbsp; public partial class NewsWebView : Gtk.Window<br>
&nbsp;&nbsp;&nbsp; {<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; public NewsWebView (string GUIDurl) :
base(Gtk.WindowType.Toplevel)<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; {<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; this.Build ();<br>
&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; <br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; // System.Console.WriteLine("GUID url is : " + GUIDurl +
"&lt;&lt;");<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; <br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; WebView webView = new WebView ();<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; webView.Open (GUIDurl);<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; WebViewScrolledWindow.Add (webView);<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; this.ShowAll ();<br>
&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; }<br>
&nbsp;&nbsp;&nbsp; }<br>
}<br>
<br>
That's it. And, of course, as I think you've figured out, you need to
add Webkit to your project's References and you need to design a simple
GUI (scrolled) window for the code that actually calls Webkit.<br>
<br>

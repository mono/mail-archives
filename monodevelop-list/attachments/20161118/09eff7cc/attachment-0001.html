      shows the available options. You can run MD with <tt>flatpak run
        --filesystem=host com.xamarin.MonoDevelop</tt> to see the host
      filesystem. This works if you have files in e.g. /opt. However, it
      won't help if you have libraries and files under /usr because the
      sandbox provides it's own version of /usr and doesn't merge it
      with the host filesystem.<br>

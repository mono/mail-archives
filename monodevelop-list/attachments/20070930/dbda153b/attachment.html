Hello,<br><br>The past week I&#39;ve been working on profiling support for MonoDevelop, and it&#39;s in a fairly usable state right now (though never tested on other systems, so it&#39;s still alpha software)<br>Supported features:
<br>&nbsp;* Profile applications and running processes<br>&nbsp;* Open existing profiling snapshots<br>&nbsp;* Graphically inspect snapshot data (screenshots: <a href="http://monodevelop.com/Profiling_support">http://monodevelop.com/Profiling_support
</a>)<br>&nbsp;* Both heap-shot and heap-buddy are supported<br><br>To try it out, simple add &quot;<a href="http://users.telenet.be/ben.motmans/md/" class="external" rel="nofollow">http://users.telenet.be/ben.motmans/md/</a>&quot; to your monodevelop addin repositories and select the 3 new addins
<br>Requirements:<br>&nbsp;* HeapShot (<a href="http://www.mono-project.com/HeapShot">http://www.mono-project.com/HeapShot</a>) or HeapBuddy (<a href="http://www.mono-project.com/HeapBuddy">http://www.mono-project.com/HeapBuddy
</a>)<br>&nbsp;* MonoDevelop 0.16 (beta 1) or the current svn version<br>&nbsp;* mono 1.2.3 (lower version could work, but never tested)<br><br>Please let me know if you have any problems / crashes / suggestions / ...<br><br>thx<br>

I just published a beta version of what is going to be a community<br>
add-in repository for MonoDevelop. This web site is based on the Cydin<br>
project [1], developed during the last hack-week at Novell. The<br>
repository is available at the following url:<br>
<br>

<br>
The idea is to have a web site where developers can publish their<br>
add-ins and make them available to all MonoDevelop users. The site works<br>
like a build bot: it pulls the source code of add-ins from version<br>
control repositories, builds and packages them, and then publishes them<br>
in an add-in repository, to which MonoDevelop will be subscribed.<br>
<br>

people to start publishing add-ins to see how it works. If you have any<br>
question or problem, feel free to contact me directly or through the<br>
MonoDevelop mailing list.<br>

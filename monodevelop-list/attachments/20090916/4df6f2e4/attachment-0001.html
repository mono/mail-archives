The MonoDevelop team is proud to announce the release of MonoDevelop 2.2<br>
Beta 1 (2.1). This is the first release of a series of releases that<br>
will lead to MonoDevelop 2.2, hopefully around the end of October.<br>
<br>
MonoDevelop is a GNOME IDE primarily designed for C# and other .NET<br>
languages. MonoDevelop enables developers to quickly write desktop and<br>
<a href="http://ASP.NET" target="_blank">ASP.NET</a> Web applications on Linux, Windows and Mac OSX. MonoDevelop<br>
makes it easy for developers to port .NET applications created with<br>
Visual Studio to Linux and to maintain a single code base for all<br>
platforms.<br>
<br>
This release contains lots of new features and improvements. Here is a<br>
summary of the new features:<br>
<br>
      * Windows and Mac OSX are now officially supported platforms<br>
      * Support for multiple target runtimes<br>
      * Per mime-type formatting settings<br>
      * Custom execution modes<br>
      * Text Editor improvements:<br>
              * Support for auto-save<br>
              * Improved code templates, which now can have<br>
                placeholders<br>
              * Block selection<br>
              * C# and XML formatters<br>
              * On-the-fly formatting<br>
              * Acronym matching in code completion<br>
              * Dynamic Abbrev command<br>
              * Code generation window<br>
      * New code refactoring engine, with plenty of new operations<br>
              * Resolve namespace, remove and organize &#39;using&#39;<br>
                declarations<br>
              * Rename refactoring with preview. In-line rename for<br>
                variables and parameters.<br>
              * Extract method<br>
              * Declare Local Variable and Integrate Temporary Variable<br>
              * Introduce constant<br>
              * Move type to own file<br>
              * Create/remove the backing store of a property<br>
      * Debugger improvements:<br>
              * Immediate window<br>
              * Win32 debugger<br>
      * <a href="http://ASP.NET" target="_blank">ASP.NET</a> MVS add-in<br>
      * IPhone add-in for MonoTouch development.<br>
      * Moonlight add-in<br>
      * Python binding<br>
      * Version Control improvements:<br>
              * Show annotations command<br>
              * Create patch command<br>
      * Many other fixes in the C/C++ binding, database support, Vala.<br>

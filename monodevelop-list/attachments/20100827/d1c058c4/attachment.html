<p>As installed, MonoDevelop needs .NET 3.5 to be installed to run.</p>

<p>If you include System.Core.dll in the MonoDevelop folder it should work on .NET 2.0 as well.  I run MonoDevelop on Windows 2000 this way.</p>

<p>Of course, you also need GTK to be installed so it is not entirely plug-and-play to put it on a USB stick and go.</p>

Hey, I&#39;ve found a fix for this. <br><br>You simply have to define an environment variable called MOZILLA_FIVE_HOME pointing to the root directory of firefox or mozilla. <br><br>In fedora when the firefox package gets updated the old one&nbsp; can remain in the /usr/lib directory so you must use the one for your current version.
<br><br>I&#39;m using this script to get the directory of the current firefox:<br><br>-----------------------------------------------<br>#/bin/sh<br>#get-fedora-geckoembed<br><br>#Find out the right firefox directory in fedora
<br><br>for directory in `locate -e libgtkembedmoz.so | grep firefox` ; do<br>&nbsp; directory=`dirname $directory`<br>&nbsp; #The right directory have the firefox-bin executable<br>&nbsp; if [ -x $directory/firefox-bin ] ; then<br>&nbsp;&nbsp;&nbsp; echo $directory
<br>&nbsp;&nbsp;&nbsp; break;<br>&nbsp; fi<br>done<br>-----------------------------------------------<br><br>And i call the script in my .bashrc to set the var<br><br>export MOZILLA_FIVE_HOME=`get-fedora-geckoembed`<br><br>Hope this fixes your problem too.

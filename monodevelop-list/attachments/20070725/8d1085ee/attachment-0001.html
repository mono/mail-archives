<p>Looks like the compiler is not aware of the Clicked Signal and/or the Label &amp; Entry widgets. Try the following:</p>
<p></p>
<p>1.- On Stetic you need to initialize the Clicked signal for the button1. On the Stetic designer, select the button on the &lt;Widget Properties&gt; go to the tab &lt;Signals&gt; (There are only two tabs, &lt;Properties&gt; &amp; &lt;Signals&gt;).  Expand &lt;Button Signals&gt; and then select &lt;Clicked&gt;, then double click to initialize the Signal, and again double click to create the template:</p>
<p></p>
<p><span style="color:#0000ff">protected virtual void OnButton1Clicked(object sender, System.EventArgs e){ }</span></p>
<p></p>
<p>On MainWindow.cs in the view Source Code (MainWindow.cs have two view &lt;Source Code&gt; &amp; &lt;Designer&gt;). Here you can start coding.</p>
<p></p>
<p>2.- Probably you are not declaring the label1 &amp; entry1 widgets. So you need to declare them as Gtk widgets right after the declaration of the public class and before the constructor:</p>
<p></p>
<p><span style="color:#0000ff">public class MainWindow: Gtk.Window</span></p>
<p><span style="color:#0000ff">{        </span></p>
<p><span style="color:#ff0000">        Label label1; // or Gtk.Label </span></p>
<p><span style="color:#ff0000">        Entry entry1; // or Gtk.Entry</span></p>
<p><span style="color:#0000ff">        public MainWindow (): base (&quot;&quot;)</span></p>
<p><span style="color:#0000ff">        {</span></p>
<p></p>

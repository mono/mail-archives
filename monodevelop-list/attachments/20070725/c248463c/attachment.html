<p>Try this:</p>
<p></p>
<p><span style="color:#0000ff">sudo apt-get install mono mono-gmcs mono-gac mono-utils monodevelop monodoc mono-xsp monodoc-http</span></p>
<p></p>
<p>If you are going to develop/host ASP.NET applications you may want to try this:</p>
<p><span style="color:#0000ff">sudo apt-get install mono-xsp mono-xsp2</span></p>
<p></p>
<p>or if you are going to use Apache Server try this;</p>
<p><span style="color:#0000ff">sudo apt-get install mono-apache-server2 libapache2-mod-mono</span></p>
<p></p>
<p>you can install some development manuals:</p>
<p><span style="color:#0000ff">sudo apt-get install monodoc-ipod-manual monodoc-njb-manual monodoc-nunit-manual monodoc-gtk-manual monodoc-gtk2.0-manual monodoc-gecko2.0-manual</span></p>
<p></p>
<p>If some of this packages are already installed, apt will ignore them.</p>
<p>Those are the most used dependencies. Probly this will fix your problem.</p>

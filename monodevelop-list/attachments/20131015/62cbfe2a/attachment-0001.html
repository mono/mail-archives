    <br>
    Just in case it helps here is a set of instructions I created when I
    built mono, gtk-sharp (2.12 and 3) as well as monodevelop (including
    some monodevelop dependencies). Since you've already installed a
    recent mono you probably don't have to build everything, but it
    might be worth installing the dependant libraries for everything.
    For example gtk-sharp-3 depends on libgtk-3-dev so install that.<br>
    <br>
    <blockquote type="cite">Instructions:<br>
      ==== mono ====<br>
      $ sudo apt-get install build-essential automake libxml-parser-perl
      git libtool<br>
      $ git clone <a class="moz-txt-link-freetext" href="https://github.com/mono/mono.git">https://github.com/mono/mono.git</a><br>
      $ cd mono<br>
      $ git checkout mono-3.2.3-branch<br>
      $ git submodule update<br>
      $ ./autogen.sh --prefix=/usr/local<br>
      if you have line ending issues run: $ find . -type f -exec
      dos2unix {} \;<br>
      $ make get-monolite-latest<br>
      $ make EXTERNAL_MCS="${PWD}/mcs/class/lib/monolite/gmcs.exe" -j5<br>
      $ sudo make install<br>
      <br>
      ==== gtk-sharp 2.12 ====<br>
      $ sudo apt-get install libgtk2.0-dev libpango1.0-dev libglade2-dev<br>
      $ git clone <a class="moz-txt-link-freetext" href="https://github.com/mono/gtk-sharp.git">https://github.com/mono/gtk-sharp.git</a><br>
      $ cd gtk-sharp<br>
      $ git checkout gtk-sharp-2-12-branch<br>
      $ git submodule update<br>
      $ ./bootstrap-2.12 --prefix=/usr/local<br>
      $ make -j5<br>
      $ sudo make install<br>
      <br>
      ==== gtk-sharp 3 ====<br>
      $ sudo  apt-get install libgtk-3-dev<br>
      $ git clone <a class="moz-txt-link-freetext" href="https://github.com/mono/gtk-sharp.git">https://github.com/mono/gtk-sharp.git</a><br>
      $ cd gtk-sharp<br>
      $ git submodule update<br>
      $ ./autogen.sh --prefix=/usr/local<br>
      $ make -j5<br>
      $ sudo make install<br>
      <br>
      === gnome-sharp ===<br>
      $ sudo apt-get install libgnomecanvasmm-2.6-dev
      libgnomeuimm-2.6-dev<br>
      $ git clone <a class="moz-txt-link-freetext" href="https://github.com/mono/gnome-sharp.git">https://github.com/mono/gnome-sharp.git</a><br>
      $ cd gnome-sharp<br>
      $ git submodule update<br>
      $ ./bootstrap-2.24 --prefix=/usr/local<br>
      $ make -j5<br>
      $ sudo make install<br>
      <br>
      === mono-addins ===<br>
      $ git clone <a class="moz-txt-link-freetext" href="https://github.com/mono/mono-addins.git">https://github.com/mono/mono-addins.git</a><br>
      $ cd mono-addins<br>
      $ git submodule update<br>
      $ ./autogen.sh --prefix=/usr/local<br>
      $ make -j5<br>
      $ sudo make install<br>
      <br>
      === monodevelop ===<br>
      $ sudo apt-get install libvala-0.20-dev<br>
      $ git clone <a class="moz-txt-link-freetext" href="https://github.com/mono/monodevelop.git">https://github.com/mono/monodevelop.git</a><br>
      $ cd monodevelop<br>
      $ git submodule update<br>
      $ git checkout monodevelop-4.1.12-branch<br>
      $ LIBVALA_CFLAGS="-I/usr/include/vala-0.20 -I/usr/include/glib-2.0
      -I/usr/lib/x86_64-linux-gnu/glib-2.0/include"
      LIBVALA_LIBS="-lvala-0.20 -lgobject-2.0 -lglib-2.0" ./configure
      --prefix=/usr/local<br>
      - select main, debugger and addins<br>
      $ make -j5<br>
      $ sudo make install<br>
    </blockquote>
    <br>
    Hope that helps you out a bit.<br>

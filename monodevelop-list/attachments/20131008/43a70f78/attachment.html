    If you have a monodevelop addins project that you want to debug, you
    can create a custom Execute command and run another instance from
    MonoDevelop there: In the Project Options, go to Run/Custom
    Commands, select Execute and then enter the path to your MonoDevelop
    executable. <br>
    <br>
    Additionally you'll have to make sure that the addins dll is in the
    right location so that the new MD instance will pick it up, e.g. by
    setting the Output Path accordingly.<br>

Hi,<br><br>I wrote a small addin for MonoDevelop which adds a little option to send the contents of a selection or an entire file to Pastebin. This should make combined debugging a lot more easier :). It&#39;s easy enough to use, allows you to specify a different subdomain for pasting (pastes by default to <a href="http://csharp.pastebin.com">csharp.pastebin.com</a>) and you can specify how long you want the snippet to stay.<br>

<br>I have put up a screenshot of the addin here - <a href="http://imgur.com/WxLQ2.png">http://imgur.com/WxLQ2.png</a> .I have also attached a .dll which can be placed in your addins directory and start using it. It&#39;s stable enough for regular use, and you can get the code for the pastebin addin from here:<br>

<a href="http://github.com/ninjagod/MonoDevelop.EditorTools/tree/master">http://github.com/ninjagod/MonoDevelop.EditorTools/tree/master</a><br clear="all"><br>To build, check out MonoDevelop.EditorTools into extras, and build against a built copy of MonoDevelop (and make sure that the newer version of the assemblies are present in your search path). This has been tested against trunk. It builds an addin directly into main/build/addins. I hope to merge this into the MonoDevelop trunk though it&#39;s a really really small feature. Looking forward for your feedback.<br>

I just wanted to add that I built an application using the new GUI designer that I've tested on multiple platforms, and it looks identical, and works very well.&nbsp; The trick is to using HBoxes and VBoxes.&nbsp; I've even looked at some of the code the GUI designer generates (Stetic), and it's not nearly as overbloated as the Windows Forms designer in Visual Studio.NET (there may be one or two things here and there to clean up), but overall, it's an excellent tool that does great work.&nbsp; I'm ecstatic to have a GUI designer in MD; I think that was the single most important thing to get a lot of people to jump over.<BR>

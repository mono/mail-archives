One of the changes we want to do this year is to shorten the time<br>
between stable releases, so that we can make 2 or 3 releases per year.<br>
The first release following this schedule will be MonoDevelop 2.4, which<br>
we plan to release next month.<br>
<br>
The roadmap for the release is the following:<br>
<br>
      * April 26th: branch 2.4 and feature freeze.<br>
      * May 5th: MonoDevelop 2.4 Preview 1, string freeze<br>
      * May 17th: MonoDevelop 2.4 Preview 2<br>
      * May 24th: MonoDevelop 2.4 Release<br>
<br>
The roadmap is also available in:<br>
<a href="http://monodevelop.com/Developers/Roadmap" target="_blank">http://monodevelop.com/Developers/Roadmap</a><br>
<br>
We are now switching to bug-fix mode, so if you find any bug, please<br>
report it through <a href="http://monodevelop.com/Bugs" target="_blank">http://monodevelop.com/Bugs</a>.<br>

&nbsp;&nbsp;&nbsp; Add support&nbsp;to new database should be easy, you could see Mysql addin&nbsp;[0], you should implement:<BR>
&nbsp;&nbsp; * Connection Provider: It's in charge of create a new connection. [1]<BR>
&nbsp;&nbsp;&nbsp;*&nbsp;Db Factory: It's the main db class, with a IDbFactory you could have access to Schema Provider, Gui, Provider, and other objects. [2]<BR>
&nbsp;&nbsp;&nbsp;* Dialect: a dialect class to use for diferent language syntaxis. [3]<BR>
&nbsp;&nbsp; * Gui Provider: Represent the gui part, it has all the operations like: Create Table, Database, and others. [4]<BR>
&nbsp;&nbsp;&nbsp;* Schema Provider: Implements all of the needed DB functions operations. [5]<BR>
&nbsp;&nbsp; * Syntax Mode (Xml): If has the Syntax Mode highlighting file&nbsp;(it depends on the provider). [6]<BR>
&nbsp;&nbsp; * PooledDbConnection: an internal (used by the pool manager) pooling class.<BR><BR>&nbsp;&nbsp; With the previous implementations you will have almost all done, but you could need to implemen new dialogs (database creation, and others).<BR><BR>

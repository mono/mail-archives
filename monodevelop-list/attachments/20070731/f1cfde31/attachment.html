In the IDE you can create external tools where you can specify the command, arguments and the working directory. Additionally you can use variables such as ${ProjectDir} etc. and there is a nice button to suggest possible variables.
<br><br>When you create a custom command for a project you only get to specify the command and you can&#39;t specify the working dir nor use variables. This patch makes the custom commands similar to the external tools in that you can specify command, arguments, working dir (defaults to project&#39;s base dir) and you can use variables plus it adds buttons to suggest available variables.

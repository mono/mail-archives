We started on a designer as a student summer of code project a few years
ago.   It is called LunarEclipse.

It was built as a separate tool from MonoDevelop.   The idea was that we
would turn LunarEclipse into a standalone tool, eventually allowing
LunarEclipse to be a Silverlight designer that could be used *inside*
Silverlight itself.

By building the designer on Silverlight the idea was that we could get
developers on OSX interested as well, since OSX developers do not have
access to Blend either.

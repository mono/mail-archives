Hey,

The functionality won't work because there are no packages in the
update service for any linux distro, nor are there any plans to do
this. The fact that you can see an update icon on linux is a bug and
we should fix it.

For the unit test issue, it would be great if you could file a bug so
we can start diagnosing the problem. MonoDevelop (and supporting
libraries) have a testsuite of several thousand tests which we can use
to test this issue. I can't say I've noticed a substantial difference
in runtime though, so it'd be fantastic if you could share the exact
code demonstrating the problem.

Alan

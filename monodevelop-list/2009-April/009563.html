Hi guys,

I want to announce an unofficial support for mono/monodevelop packages 
for Fedora 10, currently those packages are under review on bugzilla, 
but I can tell you that are currently working just fine on several 
machines here at my Univ.

Good things included are:

Monodevelop 2.0 final release.
Mono 2.4 release
Monodevelop Integrated GDB debugger addin
Monodevelop Integrated MDB debugger addin
Mono and Monodevelop dependencies updated for the mono stack...

We also have the database integration addin for monodevelop and 
java/ikvm and vala support to if someone need it (currently not in the 
repo).

So, just copy the uach.repo file to you repo directory (usually 
/etc/yum.repos.d) and execute on a terminal:

yum install mono-* monodevelop* --enablerepo=ic-uach,ic-uach-updates

Hope that this help...

Cheers,

Mauricio

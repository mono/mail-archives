On Wed, 2009-04-15 at 14:03 -0400, Justin Holewinski wrote:
> Is it possible to adjust the number of spaces that are used for tabs
> in MonoDevelop 2.0?  Specifically, I want MonoDevelop to use 2 spaces
> instead of 4 for tabs.
> All of the information I have found on-line (pre-2.0) seems to suggest
> that this is an option under Edit > Preferences > Text Editor, but I
> do not see any such option.  The only options I have under Text Editor
> are Coding, Code Folding, and Font, none of which specify tab spacing.
> I have looked through the rest of the Options dialog without finding
> anything.
> I am using MonoDevelop 2.0, built from source (non-SVN).

Select project / right click: Opions / Source Code / Text Style
See the "Whitespace" section.

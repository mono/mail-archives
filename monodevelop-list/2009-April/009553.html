>
> the curious and sad thing is that the file need to be "myfile.xml" (with
> the xml extention) to have the xml tools enable, if my file IS xml base
> but for example is called myfile.ipx the menu is still disable...

Edit->preferences->other->xml schemas, and add the ".ipx" extension to
"default file associations". This means MD will treat files with that
extension as XML. (You can set a blank schema).

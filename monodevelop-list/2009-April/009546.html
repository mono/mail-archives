If you're just looking at deploying to ubuntu based systems (includes
Kubuntu, Xubuntu etc) then you really should package it into a .deb file.
This allows Ubuntu to automatically pull in the dependencies (mono, mono
bidnings, etc) for you.


Additionally Ubuntu lets you register for a ppa on launchpad so you can
distribute and update your application online via the apt-get mechanism.

Having said that - I have not the faintest idea how to create a deb package

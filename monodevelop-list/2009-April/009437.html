Hello,


What is in MonoDevelop's download link is the APT link. This essentially
does the "apt-get install monodevelop" commmand automagically.

So, it will install the version which is in the distro's repository. If
you're using Ubuntu 8.10 or Linux Mint 6, that will be Monodevelop 1.0.

Wait for Ubuntu 9.04 (or Mint 7 I believe...) and you can install
MonoDevelop 2.0 easily just by clicking the "Install with APT link".

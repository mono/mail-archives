A simplistic explanation is here:
http://bugzilla.ximian.com/show_bug.cgi?id=54911

What I would like to see is whoever is going to be writing it
(presumably you) think about how you want to do it, and write up a short
proposal of how it will work to this list and after (short) discussion
you can go write it.  Include any questions that you may have about
MonoDevelop, AddIns, coding style, etc.  It might also be worth
researching if SharpDevelop has existing code for this, will you use
SharpCvsLib, etc.

Of course you can always write the code without a proposal also.

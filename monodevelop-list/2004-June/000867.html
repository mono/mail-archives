I've got 0.4 installed and working, but I use this .profile (bash),  
which may make a difference:

source /sw/bin/init.sh;
QTDIR=/usr/local/qt
MANPATH=$QTDIR/doc/man:$MANPATH
DYLD_LIBRARY_PATH=$QTDIR/lib:/sw/lib:/usr/lib:/usr/local/ 
lib:$DYLD_LIBRARY_PATH
PKG_CONFIG_PATH=/sw:/sw/lib/pkgconfig:/usr/local/lib/ 
pkgconfig:$PKG_CONFIG_PATH
DISPLAY=:0

PATH=/opt/local/bin:$PATH;
PATH=/usr/local/bin:$PATH;
PATH=$QTDIR/bin:$PATH

export QTDIR PATH MANPATH DYLD_LIBRARY_PATH PKG_CONFIG_PATH DISPLAY

I run X11 apps from the Terminal, but you can also

source .profile

in xterm

To use the debugger integration you have to get MD from SVN and compile
the MonoDevelop.Debugger.Mdb add-in. It requires Mono and the Mono
Debugger from SVN.

It currently has basic support for breakpoints, step-by-step execution,
call stack pad, and watch pad. It is still a bit unstable though.

The SLED/10.2/10.3/Factory versions build require the official Mono
Project OBS packages, which contain the newest stable Mono release.
They'll get pulled in automatically if you use the 1-click installer.

Right now the oS11 packages just use what's shipped with oS11.
However, in future they'll also depend on the Mono OBS packages.

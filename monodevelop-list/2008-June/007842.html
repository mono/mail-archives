Hello,

I've been working on a .NET code organization tool and have recently
extended it to support processing of MonoDevelop solution and project files.

Here's a description of the tool:
"NArrange is a stand-alone, configurable .NET code beautification tool that
automatically organizes code members and elements within .NET classes. It
allows developers to easily sort class contents according to their style and
layout conventions. NArrange works by parsing source code files into a
document object model, arranging the elements then rewriting the arranged
source code."

The documentation and link for the latest release are available 
http://narrange.sourceforge.net/ here .

I think you should apply this patch to CBinding before the release (I can't
commit it myself because I lost commit access because of the openssl bug).

It makes it so that CBinding no longer parses local variables by default,
the reason for this is that since the local variable parsing is not context
sensitive it really shouldn't be used in any real project IMO.

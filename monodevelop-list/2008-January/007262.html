0.18 added a dependency on mono-addin-0.3

Best way to do a clean removal in to remove the debs properly :)

Failing that, remove /usr/lib/monodevelop and ~/.config/MonoDevelop/addin*

The errors that prevent it from running usually help in diagnosing
these problems... start MD from a terminal if necessary to get error
output.

It would be great if you can update the translations for the next
release. Strings are frozen, so that's pretty much what MD 1.0 will

are several .po files that need to be updated:

      * extras/JavaBinding/po/*.po: The Java binding.
      * extras/MonoDevelop.Database/po/*.po: Database support.

All files are up to date, so you can just grab them from SVN (trunk) and
start working them.

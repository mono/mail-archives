Major features imho: (while writing this mail I'm actually starting to
wonder why these features aren't listed on the home page of the website)
* easy to use GUI designer
* Integrated support for version control
* Support for lots of languages: C#, VB.net, boo, C, C++, java, ...

other cool features:
* integrated unit testing
* asp.net support
* integrated database explorer (and database editor, but it needs some more
polishing)

-- Ben

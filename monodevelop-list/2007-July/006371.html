Hi,

Please, don't use 'sudo' to build the software, you don't need it and
you may end up with permission problems.

Try this:
      * In a *new* directory, get MD from SVN again.
      * rm -rf ~/.config/MonoDevelop
      * ./autogen.sh --prefix='pkg-config --variable=prefix mono'
      * make
      * make run

Which output do you get when running?

> Hello ppl,
> 
> I read that a NAnt addin was on the Roadmap to MD 1.0. What are the
> features that it will probably need ? Kindly add to the list.
> 
> * Ability to create a new project from a exist NAnt build file
> 
> * Abilty to create a NAnt build file for a project..
> 
> What else?

I think everything could be summarized in implementing an IFileFormat
for NAnt files, because by only implementing that interface you would
support a bunch of features such as:

      * opening a NAnt project in MD as if it were a native project,
      * support exporting/importing between NAnt and other supported
        file formats,
      * support creating packages based on NAnt

Support for NAnt is not planned for 1.0, unless somebody contributes it.

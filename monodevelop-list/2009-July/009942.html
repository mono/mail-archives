> Hello MonoDevelopers,

> I'm a Linux user and developer. I'm using MonoDevelop at work and home. I
> wold like to join the project and to contribute with some code too, to
> improve this wondefull tool.
>
> How can I help?

My suggestion is to find a feature that interests you, and hack on
that. Maybe the TODO page (http://monodevelop.com/TODO) has some ideas
that interest you, or the enhancements filed in bugzilla, or a feature
you've seen in another IDE, or some new idea that you have. Mail the
list to let us know that you're working on it, so that your work
doesn't conflict with someone else.

You may find the following pages useful:
http://monodevelop.com/Developers/Development:_Getting_Started
http://monodevelop.com/Developers/Articles

If you have questions about how to implement something, please email
the monodevelop-devel mailing list or ask on IRC.

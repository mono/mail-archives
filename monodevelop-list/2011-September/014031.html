Jim,

The upgrade to 2.6 crippled testing for me. Tests run, but the
indication of pass / fail only works at the aggregate level. I have to
run individual tests to see which test passed and which test failed in
the event of a failure. I posted a bug report here:

http://bugzilla.xamarin.com/show_bug.cgi?id=385

and someone changed the "Target Milestone" but that is all. For now I
grit my teeth as this makes testing quite painful. It is somewhat
surprising to me that this is not affecting more people.

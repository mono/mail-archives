Hello AspNetEdit users

Just a quick notice that in MD SVN, AspNetEdit's toolbox now sources
the default ASP.NET 2.0 webcontrols internally (with icons!). If you
have already added the System.Web.dll assembly to the toolbox then
each control will appear twice. To fix this, you should delete
~/.config/monodevelop/Toolbox.xml while MD is not running. This will
empty your custom toolbox.

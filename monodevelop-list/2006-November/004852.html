> from
> /usr/lib/monodevelop/AddIns/MonoQuery/Mono.Data.Sql.dll could not be

mono-data-sqlite package is missing. What you need to to is:
1. Either turn off MonoQuery Addon or install mono-data-sqlite
2. File a bug to fedora.
3. File another bug to fedora's rpm-build, saying that their build  
system should calculate dependencies automatically (as it's made in  
Deb/ALTLinux).

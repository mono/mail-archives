I have not used the packages from launchpad myself.

With my packages (on OBS), I know that you need to run a script

right Mono is picked up (MonoDevelop is installed in different
directory than the default directory).
It might be that xsp4 is installed in /usr/bin, and monodevelop is
installed to /opt/something.

If you install monodevelop-opt and mono-xsp-opt from my repository on
the OBS, it might work.
Though I have not tried running an xsp4 app from within Monodevelop, I
usually start it on the console.

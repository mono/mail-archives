Hello,

If you have your .sln file you can run:

$ mdtool build mysolution.sln

If you just want to invoke the compiler manually you can run the mono
compilers directly.

* mcs - old .Net 1.0 
* gmcs - .Net 2 -> 3.5
* smcs - moonlight

there is another one for .Net 4.0 but I can't remember what it is :)

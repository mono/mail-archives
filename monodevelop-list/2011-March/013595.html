Hello everyone,

Another year, another Summer of Code, and the Mono Project is once again
participating as a mentoring organization!

Google Summer of Code (GSoC) is a global program that offers student
developers stipends to write code for various open source software projects.
The Mono Project as been participating in GSoC since the very first year,
and some of our best core team members and contributors started out as GSoC
students.

The proposal submission period has begun, so check out our Ideas
Page<http://www.mono-project.com/Gsoc>[0] and our Organization
Page <http://www.google-melange.com/gsoc/org/google/gsoc2011/mono> [1],
decide on a project and submit your proposals!

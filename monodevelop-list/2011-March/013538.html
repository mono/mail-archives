openSUSE 11.4 is out, so you could always just install that instead of 
Ubuntu ;)

If you go further down the page then there is a "source code tarballs" 
section. As JMalcolm said, you'll need to download those and compile 
them yourself if no-one has put it in an Ubuntu PPA.

You'll just need to run "./configure && make && sudo make install" in 
the source code folder, then pick your options on what you want to 
compile and make sure that you've got the appropriate "-devel" packages 
installed (or whatever Ubuntu calls them).

In Edit -> Preferences -> .NET Runtimes, click on Add and select the
path to the prefix where your parallel mono is installed. Once the
runtime is registered, you can select it in the configuration combo box
available in the build toolbar, or by using the Active Runtime submenu
in the Project menu.

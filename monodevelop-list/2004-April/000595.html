Hello,
   C# uses a CompilerError.ErrorNumber so it looks in task list like:
   "error description(CS0165)"

   But many langugaes don't have "error number", like Nemerle ;)
   So my errors looks like:
   "error description()"

   Ugly. This patch fixes that.

Pawel

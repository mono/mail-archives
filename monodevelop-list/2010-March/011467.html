If you have Mono installed, in the configuration combo box (in the
toolbar) you should see an entry for selecting Mono as target runtime
(also in the Project->Active Runtime menu). When you set the active
runtime, all projects will be built and run using that runtime. You can
also use the "Run With" menu (and context menu) to run a project using a
specific runtime, regardless of what the active runtime is.

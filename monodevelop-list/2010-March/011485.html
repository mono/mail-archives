Trunk MD also supports custom tools, like VS, that can be set using
the "Custom tool" property on the property pad and are then run
automatically whenever the file is saved.

The extension point for adding such custom tools is
/MonoDevelop/Ide/CustomTools. For an example see the T4 generator in
main/src/addins/TextTemplating/MonoDevelop.TextTemplating

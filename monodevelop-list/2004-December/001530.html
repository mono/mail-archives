Hello,

Anyone interested in a start page should look at the existing partial
port that is disabled in svn.  It does most of what you would want
without requiring xsp.  You are basically limited by the info MD can
provide you anyways so aspx does not buy you anything.

The main question is: Do people really want a start page? and what
useful info would it provide that doesn't fit better somewhere else?

I think at the point where MD tutorials and howtos existed inside MD a
start page more like eclipse _might_ be useful.  Currently, what would
it show except for recent projects, which is already available in the
menus.

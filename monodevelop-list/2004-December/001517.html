I've just finished my first MonoDevelop Plugin, an interface to 
NAnt 0.85-rc1.

It enables you to use NAnt build files and shows compile errors in the
task list. More information is in the readme file. you can get it at

 https://www.lausch.at/svn/public/monodevelop/nantplugin/trunk

It's not yet finsihed but it works so that you can use a NAnt.build file
and execute the "build" target.

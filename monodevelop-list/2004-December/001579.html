I just released a new version of the NAnt Plugin for monodevelop
You can it access via SVN from the URL
https://www.lausch.at/svn/public/monodevelop/nantplugin/tags/alpha-0.2

News:
o  GUI for selecting a buildfile and a target from this buildfile
o  Run the NAnt build task in a seperate thread, not blocling the 
   MD GUI
o  Two ways of invocation: 
   o   Find a default buildfile with the same name as the project
       and execute the default target from this file
   o   let the user choose from a treeview which target from  which
       file to execute.

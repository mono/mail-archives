Most developers don't use the feature since their projects can be opened by Visual Studio also.  Visual Studio doesn't support having multiple solutions in the same window at the same time.  Based on the fact that you're developing a MonoTouch app, this doesn't apply to you.

If you want to have multiple solutions opened in multiple windows, just launch another MonoDevelop instance and create/open the solution.

Sadly, MD has no resx editor.

Note that for GTK#, the usual localization mechanism is Gettext, and
MD has good support for creating and editing Gettext translations. For
MonoMac/MonoTouch, the standard mechanism is lproj directories and
there are a number of tools available for handling these. So AFAIK
resx is usually only used by people porting Windows Forms apps.

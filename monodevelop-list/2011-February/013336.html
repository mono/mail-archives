On Sat, Feb 5, 2011 at 9:23 AM, David Carlton <dcarlton at cfl.rr.com> wrote:
> Hello:
> I am attempting to create a MonoMac project.  However, when I create the
> project, it will not save.  The error message reads:
> "DUPLICATE PROPERTY 'GtkDesignInfo' IN CLASS MONODEVELOP.PROJECTS.PROJECT
> Can you give me advice on how to clear this error?

This happens if you install the "experimental new GTK# designer" from
the addin repository - it conflicts with the old built-in GTK#
designer addin. Using the addin manager, either uninstall the
experimental one, or disable the built-in one.

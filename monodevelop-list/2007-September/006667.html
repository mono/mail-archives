Hello,

The past week I've been working on profiling support for MonoDevelop, and
it's in a fairly usable state right now (though never tested on other
systems, so it's still alpha software)
Supported features:
 * Profile applications and running processes
 * Open existing profiling snapshots
 * Graphically inspect snapshot data (screenshots:
http://monodevelop.com/Profiling_support)
 * Both heap-shot and heap-buddy are supported

To try it out, simple add "http://users.telenet.be/ben.motmans/md/" to your
monodevelop addin repositories and select the 3 new addins
Requirements:
 * HeapShot (http://www.mono-project.com/HeapShot) or HeapBuddy (
http://www.mono-project.com/HeapBuddy)
 * MonoDevelop 0.16 (beta 1) or the current svn version
 * mono 1.2.3 (lower version could work, but never tested)

Please let me know if you have any problems / crashes / suggestions / ...

thx
-- Ben

Each release of MonoDevelop is built against a particular version of NUnit.
You may be using a different version. NUnit itself will detect tests that
are built against older versions of the framework, but I believe that
MonoDevelop may be doing the test detection itself. However, I'm an NUnit
developer, not a MonoDevelop developer, so I could be wrong.

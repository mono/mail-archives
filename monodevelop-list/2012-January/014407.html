Hi,

Mono is growing very fast, a lot of new stuff is being added and also
some existing stuff is being replaced with new implementation.
When a new contributor looks at the bugzilla and documentation, he
find it difficult to decide from where he should start. It would be
easier for him if we put novice tag to bugs which are of low
complexity bug so he can start from it.

Same approach is adapted by Drupal and they find it useful. Although
Mono is much bigger and more complicated, I'm sure it will help to
attract new contributors.

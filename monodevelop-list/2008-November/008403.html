> What is the status of running MonoDevelop under Win32?

The current status is that nobody is working to make MD run on Windows.
There have been attempts from contributors to make it work, but I can't
tell if it would work now.

We are regularly getting requests for getting MD properly work on
Windows and Mac. We plan to put some effort on that at short term.
However, the core MD team is right now focused on the 2.0 release, and I
can't promise much progress in the following 2/3 months.

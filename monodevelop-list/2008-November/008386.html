I just got MonoDevelop from SVN up and running on Intrepid along side
the 1.0 install. This PPA contains all Mono 2.0 dependencies, just add

http://ppa.launchpad.net/firerabbit/ubuntu intrepid main

to your sources list and install all of the Mono 2.0 stuff. Then check
out the MonoDevelop source and build. As mentioned, it can be run
without being installed with make run. I added a shortcut that linked
to a file containing something like

cd <path to MonoDevelop source tree>
make run

I'm assuming this setup will work with the Alpha 2 release, we'll find
out in a couple days.

If it is any similar to the same error message on OSX, you might be  
able to solve it by first installing libglade. When gtk-sharp detects  
the availability of libglade, it will install glade-sharp. Like I  
said, this was how to solve this issue in the past on OSX, using  
MacPorts. Your mileage might differ.

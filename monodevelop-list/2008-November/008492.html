Ok I've made some headway.  Type each line into a commandline and it will
download the latest subversion.

You either make the build using "sudo bash autogen.sh" or "./configure",
then "sudo make", then "sudo make install", within each of the directories
installed.  Most likely those directories will be in your home folder.

svn checkout http://svn.gnome.org/svn/glib/trunk glib
svn checkout http://svn.gnome.org/svn/gtk+/trunk gtk
svn co http://svn.gnome.org/svn/pango/trunk pango

my command that I used to compile moonlight 2 from sources:

./configure --prefix=/usr --with-cairo=system
--with-mcspath=../mono-2.6.1/mcs --with-mono-basic-path=../mono-basic-2.6/

you will need download mono 2.6.1 sources and mono-basic 2.6.1 sources and
if you have cairo installed in your linux distro you can to use the
--with-cairo=system configure parameter

On Wed, Dec 9, 2009 at 5:06 AM, Yoav HaCohen <yoavhacohen at gmail.com> wrote:
> OK, Is it possible to change MonoDevelop to always write in English (even if
> the current system's input language is not)?

You need to set an environment variable:
export LANG=en

If you don't want to set this systemwide, you can edit it into the MD
startup script: MonoDevelop.app/Contents/MacOS/monodevelop

We have a fix planned for MD 2.4:
http://monodevelop.com/Developers/Tasks/General/General_Improvements#_task_a_General.LanguageOption

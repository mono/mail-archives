I was thinking of an extra dialog while importing a VS.Net project (see mockup).

In the VS.Net .csproj files, you can get the names of the references
<Reference Name = "gtk-sharp" ...../>
Use the name and some code based on gacutil to create a list of all
possible versions of the dll, and add those to a dialog.

Hey,
I'm building daily binary snapshots from MD Svn. They include
gecko-sharp.dll and gtksourceview-sharp.dll as the bindings are pure
managed code and the latest version (2.0) is not included in most
distributions yet. You will need the usual stuff installed like
gtk-sharp, monodoc and so.

The build options

   * mono-debugger: no
   * MonoQuery Providers:
   *   PostgreSQL:   yes
   *   MySQL:        yes
   *   SQLite:       no
   * java support: no


I've only a few KB of upload speed, so be patient ;).

The thing:
http://rubiojr.no-ip.org/~rubiojr/MDSnapshots/mdsnapshot-daily.tar.bz2

If you have any problems, please, mail me.

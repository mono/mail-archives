It looks like you're using the experimental MSBuild build engine, try
disabling it.

I suspect this is a bug with building heterogeneous solution configurations
(e.g. solution config "Debug|x86" mapped to the "Debug|AnyCPU" project
configuration) with the MSBuild engine that was fixed in MD a while back,
what MD version are you using?

Hi Robbie,

I don't think we support Mac OS X 10.5.x anymore.

The download links for older versions of MonoDevelop aren't listed on the
site, but you can easily find them by modifying the download link for the
current version like so:

http://download.xamarin.com/monodevelop/Mac/MonoDevelop-VERSION.dmg

I'm not sure what the last version released was that supported Mac OS X
10.5. I might have been one of the 2.8.x series (I think the last 2.8.x was
2.8.6.5).

As far as Mono goes, the docs on the Mono download page for Mac OS X really
need to be updated since I don't think the latest releases of Mono 2.10
work with Mac OS X 10.5 (I think up to 2.10.8 worked, not sure about
2.10.9). Here's the list of 2.10.9 downloads:
http://origin-download.mono-project.com/archive/2.10.9/macos-10-x86/
(you'll probably want the MDK and not the MRE)

I would highly recommend upgrading your OS to 10.7 or 10.8 if you can as
the newer versions of Mono & MonoDevelop's are much better.

Hope that helps,

Jeff

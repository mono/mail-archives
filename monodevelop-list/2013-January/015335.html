That's because there was never a MonoDevelop-2.10.8

MonoDevelop version that you probably want (2.8.6.5).

The MonoDevelop 3.x versions we've released so far are:

MonoDevelop-3.0.6.dmg

releases. Give them a try first. If they work, great - if not, just replace

> Just as important, I'd like to develop software that can run on older Macs.
> Will the software written with MonoDevelop on OSX 10.6, for example, run on

You can target Mac OS X 10.6, I don't know if it'll work on older versions
or not. Realistically, you'll still want MonoDevelop 3.1.1 or newer in
order to take advantage of all of the functionality that has been
added/improved for writing Mac OS applications.

Not that you can't do things the hard way with older versions of
MonoDevelop, but you'll never be able to target the App Store nor
distribute your apps to Mountain Lion users (Mountain Lion requires that
apps be signed which can only be done if you, the developer, is using 10.7
or newer).

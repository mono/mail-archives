Hi,

Were you running MonoDevelop as a 64bit process? If so, how? There are
a number of native libraries which MonoDevelop requires which are
typically only available as 32 bit libraries and MonoDevelop itself is
compiled as a 32bit only .NET application. If you truly were running
MonoDevelop as a 64bit process, that is definitely a problem.

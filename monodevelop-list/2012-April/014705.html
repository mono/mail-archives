Does this happen over time or very quickly?
Also what version of the mono runtime are you using?
The latest stable mono runtime has memory leaks when callbacks are
invoked from native code (which happens everywhere in GTK# and IO
libs). This leaks memory over time but I would be surprised to see
more than a couple hundred megs per hour.

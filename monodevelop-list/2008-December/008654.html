The designer doesn't support ascx files. It wouldn't be hard to add this 
(maybe trivial), but the designer is still very much an alpha and in need of 
major work in other areas. The main task is to port it to use WebKit as an 
editor instead of Gecko - this will break our dependence on Mozilla's 
fragile embedder support, and allow much tighter integration between the 
editor and its host. It'll then be possible to implement other important 
things like XHTML support.

My work on the designer is therefore on hold until WebKit-GTK has full DOM 
and editing hooks.

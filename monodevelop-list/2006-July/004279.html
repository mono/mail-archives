Hi All

I have managed to get the WebReference AddIn in some sort of working
condition and thought I will attached the dll, and addin.xml file for
those who want to play with it. 

The AddIn is far from complete but at least it create e web service
proxy class in any C# or VBNET project or project folder. It currently
does not support SSL, as well as I want to enhance it to create a "Web
Reference" node for storing all the web references under. This will
allow me to refresh etc. 

As you can see I am still far from done yet, but learning with every
step of the way. The most difficult part for me so far was that none of
the MonoDevelop API is documented, and I had to hunt around in the Core
code to figure out how things work.

Please play with the AddIn and let me know what you think.

Enjoy your weekend.

Regards

Gideon de Swardt

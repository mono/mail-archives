Hi,

MonoDevelop doesn't currently I think support Web projects per se (its being worked on though) but it is currently possible to implement web projects as a Class library both in MonoDevelop and/or in VS 2003
You won't really have debugging ability but you can use log4net to log output to help with troubleshooting.

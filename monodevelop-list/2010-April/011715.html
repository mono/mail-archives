Hi!

One of the changes we want to do this year is to shorten the time
between stable releases, so that we can make 2 or 3 releases per year.
The first release following this schedule will be MonoDevelop 2.4, which
we plan to release next month.

The roadmap for the release is the following:

      * April 26th: branch 2.4 and feature freeze.
      * May 5th: MonoDevelop 2.4 Preview 1, string freeze
      * May 17th: MonoDevelop 2.4 Preview 2
      * May 24th: MonoDevelop 2.4 Release

The roadmap is also available in:
http://monodevelop.com/Developers/Roadmap

We are now switching to bug-fix mode, so if you find any bug, please
report it through http://monodevelop.com/Bugs.

Thanks!
Lluis.

Does these special characters appear when using another gui text
editor like Kate or gedit? It sure looks like some wrong configuration
of the X keyboard handler, what would affect every program and not
only MD. Also beware of the encoding of source files you are using, if
they don't match the default encoding your system uses (normally UTF-8
in most recent distros).

I ask that as MD, just doesn't do anything special with the keyboard,
and uses the default encoding for editing source files.

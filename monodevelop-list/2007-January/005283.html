On 1/15/07, Kimble, Gary <gary.kimble at goldenven.com> wrote:
> > I would check to make sure the VB compiler is installed....
> That's an excellent idea! How do you do that?

Try to run "mbas" from the command line. If it's not recognised, you
don't have a VB.NET compiler.

There have been various VB.NET compilers for Mono. AFAIK none of them
has been feature-complete or stable. How you install depends on your
distro; look for package names like mono-basic, mono-mbas etc.

There's a very promising new compiler that's been developed recently,
but I don't think it's packaged yet:
http://tirania.org/blog/archive/2007/Jan-09.html You could try
building it from SVN.

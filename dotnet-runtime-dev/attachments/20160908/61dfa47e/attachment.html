<p class="MsoNormal">Hello, <o:p></o:p></p>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>
<p class="MsoNormal">With the growing love for UTF8 in the .NET world, I would love to see C# support implicit conversions from string constants to byte[] and byte*.   This should help things like
<a href="http://asp.net">ASP.NET</a> Core and it should help interoperate with the new Utf8String type [1].<o:p></o:p></p>
</div>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>
<p class="MsoNormal">The idea is to encode the string constant into a UTF8 encoded byte stream and add it as a binary blob to the assembly.<o:p></o:p></p>
</div>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>
<p class="MsoNormal">Then, we provide two implicit conversions on string constants:<o:p></o:p></p>
</div>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>
<ul type="disc">
<li class="MsoNormal" style="mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1">
one to byte [], which initializes the byte [] with the contents of the bytes<o:p></o:p></li><li class="MsoNormal" style="mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo1">
one to byte*, which initializes the pointer to the address containing the encoded null-terminated string.<o:p></o:p></li></ul>
</div>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>
<p class="MsoNormal">A proof of concept for Mono’s C# compiler is available here:<o:p></o:p></p>
</div>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>
<p class="MsoNormal"><a href="https://gist.github.com/migueldeicaza/96c2bc52bfc30aa47a3ba34ce135e6f1">https://gist.github.com/migueldeicaza/96c2bc52bfc30aa47a3ba34ce135e6f1</a><o:p></o:p></p>
</div>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>
<p class="MsoNormal">Apply the patch to your local Mono build, and enjoy:<o:p></o:p></p>
</div>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>
<p class="MsoNormal">byte [] foo = “Hello, world”;<o:p></o:p></p>
</div>
<div>
<p class="MsoNormal">byte *bar = “Hello there”;<o:p></o:p></p>
</div>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>
<p class="MsoNormal">In an ideal world, we should also support an implicit conversion to the new Span<byte> type:<o:p></o:p></p>
</div>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>
<p class="MsoNormal">Span<byte> span = “Hey there”<o:p></o:p></p>
</div>
<div>
<p class="MsoNormal"><o:p> </o:p></p>
</div>
<div>

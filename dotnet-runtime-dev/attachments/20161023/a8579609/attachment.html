<p class="MsoNormal">Hi,<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal">Assuming that safe Span<T> stackalloc exists, the verifier (and compiler) must track Spans that aren’t allowed to be returned. That also enables adding a constructor to Span that takes a ref T (resulting in a span to a single item). Combined
 with a safe cast this can be used to safely make fixed fields accessible as a span.<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal">(It would also be nice If the needless C# restriction that fixed fields are unsafe is removed.)<o:p></o:p></p>
<p class="MsoNormal"><o:p> </o:p></p>

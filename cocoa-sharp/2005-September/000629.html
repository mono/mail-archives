Last night I commited some new things to Cocoa#:

     samples: There are 4 new demos in here showing a couple things.   
NSOpenGLView subclassing (2 demos).  NSView subclassing (2 demos).   
The NSView subclassing also shows off the newly commited Cocoa#- 
 >System.Drawing bridge (you need X11 installed for fonts as per MWF)

     System.Drawing: Cocoa#->System.Drawing bridge :)

     BUGFIX: Methods that marshalled structs should have been calling  
objc_msgSend_stret rather than objc_msgSend and Marshaling the memory  
back to an instance of the struct.  This wasn't happening so things  
like drawRect: wouldn't work.  This is now fixed (this was a long  
standing and very annoying bug; kind of a pain to fix too).

   To play with all this stuff you will need Cocoa# from SVN (Cocoa#  
release will NOT run the new samples).  To use the System.Drawing  
bridge demos you will need System.Drawing from SVN.

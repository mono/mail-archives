You're missing the name of a framework after the first "-framework" switch. I
have no idea what's going wrong with the build, but, based solely on the
error statement and the incorrect syntax, I would suggest trying the
following:

gcc -bundle  -flat_namespace -undefined suppress \
-o .libs/libCocoaSharpGlue.0.dylib  .libs/Glue.o \
-L/Library/Frameworks/Mon\o/Versions/1.1.9/lib -framework Cocoa

If that works, then continue the build from there.

Hope this helps,

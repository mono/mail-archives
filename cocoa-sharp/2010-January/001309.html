There is no one answer, but I believe Cocoa# is not maintained at all.
 Other bridges you can use with Mono on OSX are:
http://code.google.com/p/mobjc/ and http://code.google.com/p/mcocoa/
http://www.monobjc.net/

Due to issues with Snow Leopard I migrated my app from monobjc to mobjc.  It
works very well.  I embed mono within a native xcode app.

I also suggest when this topic comes up that you should use InterfaceBuilder
to define the UI on Mac.  Do not use WinForms.  The only way to get a native
looking app is to use that OS's native tools.  This is also the strategy
MonoTouch uses for the iPhone.

Best of luck.
Duane

Uh, uninstall?   What is uninstall?  Mac OS X has never had an 
uninstaller.  This is supposed to be fixed "soon"  But we have been 
hearing that since Mac OS X Public Beta.  This commands (as root) 
should uninstall it manually

rm -r /Library/Frameworks/Mono.framework
rm -r /Library/Receipts/MonoFramework-1.0.pkg
cd /usr/bin
for i in `ls -al | grep Mono | awk '{print $9}'; do
rm ${i}
done

note, I typing quickly as my battery is dying so you might want to 
double check this.

The Cocoa Sharp Examples topic had a working link to the example.tgz 
archive when I last used it... The Makefile was outdated but using the 
command sequence detailed on that page it worked. I used the Cocoa Sharp 
Browser example as a starting point because I am writing my GUI in C#. 
For a simple Hello World app, it's easiest to create a .nib file (menu 
and window) in Interface Builder (choose Cocoa > Application), then 
write a C# file in TextEdit (or XCode) with a static Main method that 
loads it as in Thomas Richon's post - where MyNib.nib would be the name 
of your .nib file. Example:
using Apple.AppKit;
class YourClass {
    static void Main() {
       Application.Init();
       Application.LoadNib("your.nib");
       Application.Run();
    }
}
Compile it with mcs as on the examples page, then use macpack to create 
a .app bundle, which you can execute by doubleclicking on it.

There are some issues with this approach but you don't need to worry 
about them for Hello World. :-) See the archive for possible 
workarounds, Gareth Baker sent me some helpful code.

Tim,

The console is in Applications/Utilities (if you didn't already know :-).
Just fire it up before you run the Cocoa# app and you should see the output
appear. However if you run an app and it fails you can usually still get the
output by firing up the Console after the app has run/crashed (assuming it
did not totally hang the system!).

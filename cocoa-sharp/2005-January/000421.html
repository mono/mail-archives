As a learning expeience, I recently ported Apple's Objective-C AppKit example
OutlineView (located at /Developer/Examples/AppKit/OutlineView/ if you have
the Apple developer tools+docs installed) to Mono/Cocoa#. It's
feature-for-feature identical to Apple's example, though I ended up making a
couple under-the-hood improvements to Apple's code. For example, my
implementation supports multiple roots in the NSOutlineView. I'm willing to
donate it to the Cocoa# example code cause. It doesn't utilize any code from
Apple, though I obviously was looking at their code very closely when I wrote
it, as the classes and most of the method names are named identically.

Adhamh asked me to e-mail it to the mailing list, so that the code can be
reviewed. I hope this gets through....

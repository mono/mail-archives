Hi Andrew,

So Cocoa# 0.2 is done.  We are just waiting on it getting posted....

The build scripts in release/macosx should allow you to build from SVN 
source trees of Mono.  You'll need to have already done the checkout 
though.  I think I documented it pretty well, here's an example:

sudo buildMono.sh -v nightly -s <directory to checkedout mono>

Libgdiplus is now built by default, but you will really need the 
winforms module to do anything with it.  The buildMono.sh script should 
download all the tarballs for you. 

I'm pretty sure it "just works" at this point, but feel free to send 
patches to the list.  The scripts are a wee bit messy and could probably 
use a rewrite...

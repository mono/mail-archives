I'd just do it on the mac. You may need to install developer tools.  
You'll also need a subversion client (google subversion client os x,  
or get it from fink/DP) and the Mono OSX package installer (which you  
probably already have).

mkdir mono
cd mono
svn co svn://svn.myrealbox.com/source/trunk/cocoa-sharp
cd cocoa-sharp
export ACLOCAL_FLAGS="-I  /Library/Frameworks/Mono.framework/Versions/ 
Current/share/aclocal"
./autogen.sh --prefix=/Library/Frameworks/Mono.framework/Versions/ 
Current
make
make install

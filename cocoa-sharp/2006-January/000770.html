Sheets have been implemented for Window, Alert, OpenPanel and SavePanel.

If you can get the CocoaDoc.app sample in svn running you should be  
able to look at the Cocoa# documentation which contains examples of  
how to use them. If CocoaDoc.app doesn't work, you can always just cd  
into the cocoa-sharp/docs folder and type:

monodocs2html -source=en -dest=/path/to/where/you/want/the/docs

This will produce browsable documentation for what's been done on  
Cocoa# so far.

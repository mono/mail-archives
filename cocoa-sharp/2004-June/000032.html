Urs,

I'm not on that list so feel free to forward this to it.

It would be better to place the mono package in 
/Library/Frameworks/Mono.framework because that it much more native 
than the /opt location.  Jordan (jkh) would package the tarball into a 
form that DarwinPorts can use.  DarwinPorts is a system to get system 
installed on Darwin (Mac OS X), and /opt was chosen as the install 
location because it currently wasn't being used for anything by Darwin.

If the goal is to have the widest possible acceptance of Mono on OS X 
then the downloadable dmg that we have produced is the way to go.  If 
there are fears about stepping on Jordan's toes then I can contact him 
and work that out.  I think he'd be happy to have it off his plate 
though.

Adhamh

Some of the entries in the apple docs have a - or a + beside  
them....what does that signify?

Example..
– mouseDownFlags
Returns the modifier flags for the last (left) mouse-down event, or 0  
if tracking hasn’t occurred yet for the cell or no modifier keys  
accompanied the mouse-down event.

+ prefersTrackingUntilMouseUp
The default implementation returns NO, so tracking stops when the  
cursor leaves the NSCell; subclasses may override.

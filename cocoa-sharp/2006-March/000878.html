Here is a suggestion for adding default constructors to miscellaneous Cocoa
classes.

My apologies if this have been discussed elsewhere.

 

The default constructor could be implemented using the code pattern:

 

    public TableView () {

      if (GetType() ==
System.Reflection.MethodBase.GetCurrentMethod().DeclaringType)

        Initialize();

    }

 

In this way only the init would be called only once by the topmost subclass.

Also: this code can be copy and pasted into all the places where a default
constructor is needed.

decided that using makefiles to build Cocoa# applications was probably 
not the easiest way to go about things so I basically created a new 
Xcode template with mcs set up as the compiler. I have an alpha version 
available on my web site if anyone is interested, check it out: 
http://home.comcast.net/~pat2man/CocoaSharp.zip

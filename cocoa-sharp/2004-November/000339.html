Hi All,

Just a "heads up" about my last email - setting the environment  
variable within XCode only works within XCode. To achieve this globally  
the environment variable has to be set as described in this apple  
technical document:

http://developer.apple.com/qa/qa2001/qa1067.html

You may have to use the console to create the directory. Then your  
favorite console editor (vi, emacs...) to create the plist file  
containing:

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist SYSTEM  
"file://localhost/Syatem/Library/DTDs/PropertyList.dtd">
<plist version="0.9">
<dict>
         <key>DYLD_LIBRARY_PATH</key>
          
<string>/Library/Frameworks/Mono.framework/Versions/Current/lib</ 
string>
</dict>
</plist>

Obviously if this already exists you just need to add the  
<key>,<string> pair :-)

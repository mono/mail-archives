Not really, there is not much left, but both Kangaroo and myself have been
busy lately.

- The mono pinvoke bugs discovered have been fixed by Kangaroo.
- We will remove most glue (we don't know about libGlue.dylib) and use
pinvoke to objc_sendMsg directly
- We are able to parse Mach-O files to extract interfaces and methods, but
we have to merge with header parsing, since for instance BOOL is converted
into a 'char', and can't be distinguished between real char @ Mach-O level.

So we expected having 0.2 done by now, but we have not; so saying when it
will arrive is hard.  We are all working on free time on this.

THE INSTALLER INSTALLS THE PLUGIN FILES IN THE WRONG LOCATION

The installer puts the files under Library/Application
Support/Apple/Developer Tools/ The Files need to be under
Developer/Library/XCode/

If you simply move the files to the correct location, XCode will funtion
with Cocoa Sharp correctly.

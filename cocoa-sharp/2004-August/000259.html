Tim,

FYI:  Apple is not affiliated with this project in anyway.  I'm doing 
this on my own time.

We had a BIG debate about this.  In the end we decided it was better to 
not drop the NS because that would be changing the API and other 
bridges like Java and PyObjc didn't do this.  Also it makes looking up 
the API docs a little more difficult.

Basically we decided to follow what Apple does and to not make changes 
where we didn't have to.

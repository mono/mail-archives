There are a couple of elements here.

The first one would be to make MonoDevelop use the accessibility features.
  While it is a relatively large project, it does not have to be an
all-or-nothing issue, so it can be evolved to support it.

The second issue would be to map Atk to the underlying OSX APIs, and that I
have no idea how much work would be.

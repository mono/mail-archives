</div></div>You must create a simple addin to register the .xft.xml file. The<br>
addin doesn&#39;t need to have any code, it only needs an .addin.xml<br>
manifest.<br>
<br>
For some information on creating addins, see:<br>
<a href="http://monodevelop.com/Developers/Articles/Addin_development_basics" target="_blank">http://monodevelop.com/Developers/Articles/Addin_development_basics</a><br>
<br>
For an example of an addin that does not have code but only templates,<br>
see the OpenOddiceSamples addin:<br>
<a href="https://github.com/mono/monodevelop/tree/master/extras/OpenOfficeSamples" target="_blank">https://github.com/mono/monodevelop/tree/master/extras/OpenOfficeSamples</a><br>
<br>

source tree.<br>

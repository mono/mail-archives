<div dir="ltr">I think this has something do do with the compiler name changes, i.e. "mcs" is no longer the 1.0 framework's compiler, it can compile for any framework and defaults to 4.5. This means you're building gnome# with the 4.5 framework, when it expects to be built with the 1.0 framework. And somehow it's using the 2.0 Mono.Cairo too. Maybe ART# got compiled against the system GTK# and GNOME# is compiling against the locally built GTK#.<div>

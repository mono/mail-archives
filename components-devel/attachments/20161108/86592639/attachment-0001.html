<br>
For example, from pod Firebase/Database it will generate a set of nuget packages:<br>
<br>
["GoogleToolboxForMac/Defines"; "GoogleSymbolUtilities"; "FirebaseInstanceID";<br>
 "GoogleToolboxForMac/NSData+zlib"; "GoogleInterchangeUtilities"; "FirebaseCore";<br>
 "FirebaseAnalytics"; "FirebaseDatabase"; "Firebase/Core"; "Firebase/Database"]<br>

The question:<br>
<br>
I have a binding project "FirebaseCore" with lib FirebaseCore.a and API definition (exposing FIRApp class, for example) that has reference to "GoogleSymbolUtilities" project.<br>

<br>

```<br>
FIRApp.bla.blabla //calling some code from FirebaseCore package<br>

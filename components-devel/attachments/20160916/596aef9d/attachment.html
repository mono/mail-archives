<p>*Xamarin Auth*</p>
<p><br>
</p>
<p>We have started investigating what changes we need to move logins to native web browsers from embedded native web views.  This is in response to the world of providers recommending this workflow and some planning to disable apps using embedded views.  <a href="https://9to5google.com/2016/08/22/google-oauth-third-party-disable/" class="OWAAutoLink" id="LPlnk939276">https://9to5google.com/2016/08/22/google-oauth-third-party-disable/</a></p>
<p><br>
</p>

</p>
<p>*Android Support*</p>
<p><br>
</p>
<p>The update for Android Support v24 is very much in progress.  It is a top priority for the team.  There are many changes in this version of Android Support and it has uncovered a bug in our binding generator.  This defect was resolved recently and has unblocked
